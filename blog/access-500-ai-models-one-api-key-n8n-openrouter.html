Step-by-Step: Connecting n8n to OpenRouter

Integrating OpenRouter with n8n takes just four steps, transforming your automation capabilities:

Step 1: Create Your OpenRouter Account

Visit openrouter.ai and sign up (free tier available). Navigate to the API Keys section under your profile.

Step 2: Generate Your Master Key

Click "Create Key" and set your preferred rate limits. Unlike individual model keys, this one credential will work across all supported models.

Step 3: Configure the HTTP Request Node in n8n

Add an HTTP Request node with these parameters:

  • URL: https://openrouter.ai/api/v1/chat/completions
  • Method: POST
  • Headers: Authorization: Bearer YOUR_KEY

Step 4: Specify Your Target Model

In the request body, include the model identifier (like anthropic/claude-3-sonnet) from OpenRouter's model list.

Pro Tip: Store your OpenRouter key as an n8n credential variable rather than hardcoding it into workflows. This makes key rotation easier and improves security.

Common Issues and Troubleshooting

When implementing this integration, watch for these potential challenges:

Model Availability

Some models may experience temporary outages. Build fallback logic into your n8n workflow to automatically retry with alternate providers.

Rate Limiting

OpenRouter enforces global rate limits. If hitting thresholds, consider:

  • Implementing request queuing in n8n
  • Distributing load across multiple models
  • Upgrading your OpenRouter tier

Output Formatting

Different models return slightly different JSON structures. Use n8n's JSON transformation nodes to normalize responses before processing.
