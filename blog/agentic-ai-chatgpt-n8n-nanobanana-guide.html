Setting Up the MCP Connection

The Model Context Protocol (MCP) acts as a bridge between ChatGPT and N8N, allowing them to share context and act on contextual information. Think of it as giving ChatGPT the ability to press buttons in other applications.

Setup involves three simple steps:

Step 1: Enable Developer Mode in ChatGPT

Navigate to Settings → Apps and Connectors → Advanced Settings to toggle on Developer Mode. This unlocks the custom MCP connection option.

Step 2: Activate MCP in N8N

In your N8N instance, go to Settings → MCP Access and toggle on the feature. Copy the server URL provided.

Step 3: Create the Connection

Paste the N8N server URL into ChatGPT's MCP connection dialog and authenticate. The systems are now linked.

Note: Currently this feature requires ChatGPT Plus subscription and works best with the latest N8N version (2.8++).

Enabling Workflow Access

Not all N8N workflows are automatically available to ChatGPT. You need to explicitly enable MCP access for specific workflows and configure appropriate triggers.

The video demonstrates two critical workflows:

  1. Nanobanana Pro image generation: Takes text prompts and returns branded marketing images
  2. Instagram posting: Accepts images and captions publishes them automatically

For each workflow you want to expose:

  1. Set the workflow to "Active"
  2. Enable "Available in MCP" in workflow settings
  3. Use a form, webhook or chat trigger (required for initial compatibility)

Adding Brand Context

The real magic happens when you teach ChatGPT about your specific brand. This transforms generic AI outputs into on-brand content automatically.

We achieve this through:

  • Brand book: PDF or webpage defining colors, typography, and voice
  • Product references: URLs to key product images
  • Style guidelines: Tonal examples and voice characteristics

These are added via ChatGPT's Custom Instructions in a dedicated Project (shown at 9:30 in the video). The system will reference these automatically when generating content.

Pro tip: You can have AI help create your brand book by providing examples and letting Nanobanana Pro generate the initial design (template available in our resources).
