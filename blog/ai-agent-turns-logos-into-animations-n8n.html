Configuring Google Sheets and AI API Connections

The workflow needs three key API connections to function:

1. Google Sheets API

This lets you submit logos via a simple spreadsheet. Setup involves:

  • Creating a Google Cloud project
  • Enabling the Sheets API
  • Generating OAuth credentials
  • Adding these to n8n's credential manager

2. AI Animation Service API

The specialized logo animation AI requires an API key, which you can obtain with a $5 credit purchase. This key gets added to n8n's HTTP Request node.

3. Google Drive API

Similar to Sheets setup, this allows automatic saving of finished animations to your preferred Drive folder.

Security note: All credentials are stored encrypted in n8n and only accessible to your specific workflow.

Running Your First Automated Logo Animation

With APIs connected, animating a logo becomes a 3-step process:

  1. Upload your logo to Google Drive and set sharing permissions
  2. Add the Drive link to your Google Sheet in column B
  3. Set status to "production" in column C to trigger the workflow

The system will then:

  • Fetch your logo file automatically
  • Analyze its visual elements using AI
  • Generate multiple animation concepts
  • Select the best version based on design principles
  • Save the final MP4 to your Drive
