Why Carousels Dominate Social Engagement

Carousel posts generate 3-5x more engagement than single-image posts across Instagram and LinkedIn. Yet most businesses struggle to create them consistently due to the time-consuming design process. The blank canvas syndrome hits hard when you need to produce educational content daily.

This automation solves that by turning simple topic lists into professionally designed carousels. At 2:15 in the video, you'll see how the AI maintains brand consistency across all slides while adapting to different content themes.

Engagement boost: Automated carousels see 87% higher completion rates than manually created ones because the AI optimizes content flow between slides.

The 3-Part Automation Workflow

The system breaks down into three connected components that handle different aspects of the carousel creation process. Each part works seamlessly with the others to deliver complete posts ready for publishing.

First, Google Sheets serves as the content input hub where you store topic ideas. Second, OpenAI processes these topics into structured carousel content. Finally, specialized modules handle platform-specific posting requirements for Instagram and LinkedIn.

Step 1: Google Sheets Topic Management

The workflow begins with a simple Google Sheet containing your content topics. As shown at 4:30 in the video, you only need a single column titled "Topic" where you add ideas like "Life of an AI Journalist" or industry-specific subjects.

Make.com's Google Sheets module watches for new rows added to this sheet. You can configure it to process one topic per day or batch multiple ideas at once. The system pulls each new topic into the next phase of the workflow automatically.

Step 2: AI-Powered Content Creation

The magic happens when OpenAI's GPT processes each topic into a complete carousel structure. At 6:45 in the tutorial, you'll see the precise system message that ensures brand-consistent output:

  • Specifies slide count (default 5)
  • Defines visual style rules
  • Sets typography guidelines
  • Maintains logo continuity

The AI outputs a JSON structure for each slide containing the title, content, and image generation prompt. This structured data feeds directly into the next automation steps.

Step 3: Automatic Image Generation

Freepik's AI image generator creates visuals for each slide using the prompts from the previous step. The workflow handles the entire process:

  1. Iterates through each slide's JSON data
  2. Submits the image prompt to Freepik
  3. Waits 20-25 seconds per image generation
  4. Compiles all images into a single array

At 9:10 in the video, you can see how the system manages the waiting period between image requests to avoid API limits while maintaining efficiency.

Step 4: Cross-Platform Posting

For Instagram, the workflow uses Make.com's native integration to create carousel posts directly. The system:

  • Takes the array of generated images
  • Formats them for Instagram's API
  • Posts to your connected business account

You'll see this in action at 10:30 in the tutorial, where the completed carousel appears on the test Instagram account with all slides perfectly formatted.

The LinkedIn PDF Workaround

LinkedIn requires a different approach since it doesn't support native carousel posts from APIs. The solution:

  1. Creates a Google Slides presentation
  2. Adds each generated image to a slide
  3. Exports as PDF
  4. Uses custom LinkedIn modules to post as a document

At 11:45 in the video, you'll see the specialized LinkedIn modules that handle the PDF upload and post creation process seamlessly.
