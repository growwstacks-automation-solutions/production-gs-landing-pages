How the AI Analyzes Three Data Streams

The system's power comes from merging technical and fundamental analysis automatically. Most traders focus on one or the other, but combining both creates a more robust signal.

Three parallel API requests pull different timeframes (1m, 15m, 1h) from Twelve Data, while NewsAPI gathers recent headlines. The merge node combines these into a single dataset the AI evaluates holistically.

Step 1: Setting Up Twelve Data API

Technical analysis begins with clean candlestick data. Twelve Data offers a free tier perfect for testing this workflow.

  1. Create a free Twelve Data account and generate your API key
  2. In n8n, add an HTTP Request node configured to api.twelvedata.com/time_series
  3. Set parameters: symbol (dynamic from WhatsApp), interval (1m/15m/1h), output_size (100 candles), and your API key

Pro Tip: Clone the node twice to pull all three timeframes simultaneously rather than sequentially. This cuts analysis time by 66%.

Step 2: Adding News Sentiment Analysis

Markets move on news as much as charts. At 4:30 in the tutorial, you'll see the NewsAPI integration that scans headlines for your ticker.

  1. Sign up for a free NewsAPI account
  2. Configure the HTTP Request node with q=[ticker] and from=last 24 hours
  3. Pipe the output to OpenAI for sentiment scoring (sample prompt in description)

The AI evaluates whether news is bullish ("Tesla unveils new battery tech") or bearish ("SEC investigates Elon Musk"), weighting this against the technical signals.

Step 3: Programming the AI Trading Rules

The magic happens in the AI Agent node where you define your trading strategy. The sample prompt includes:

  • Technical filters (RSI thresholds, volume spikes)
  • News sentiment weights (how much to trust headlines)
  • Risk parameters (1:2 reward/risk minimum)

At 6:45, the video shows how changing one line in the system message can shift the AI from conservative to aggressive trading styles.

Step 4: WhatsApp Integration for Signals

The final step delivers signals where you already are - your phone. The WhatsApp Business Cloud node sends structured messages with:
