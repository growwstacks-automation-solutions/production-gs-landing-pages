How the AI Photo Workflow Solves It

This n8n automation creates complete product photo sets with just one iPhone snapshot as input. The system handles everything: generating realistic models wearing your product, creating varied scenes, and delivering ready-to-use images formatted for ecommerce platforms.

The Meta-Prompting Secret

Traditional AI image workflows require manually writing prompts for each product type. This system uses meta-prompting - having AI write the perfect prompts for your specific product. Gemini 2.5 Pro analyzes your photo and generates tailored prompts that Nano Banana then executes.

At 8:45 in the tutorial, you'll see how the system automatically creates prompts for different shot types: studio angles, detail close-ups, and lifestyle scenes. For fashion brands, this means getting back images of your product being worn in relevant contexts - gym wear in fitness settings, formalwear in elegant environments.

Meta-prompting adapts to any product category - simply modify the base prompt to emphasize different features for electronics, home goods, or other verticals while keeping the core workflow identical.

Step-by-Step n8n Build

The complete workflow contains 14 nodes in n8n, but the core logic breaks down into four phases:

Phase 1: Input Processing

1. Form Trigger: Accepts file upload via webhook or direct upload
2. File to Base64: Converts image for API processing
3. Google Drive: Optional backup of original image

Phase 2: Model Generation

4. Set Field: Creates prompt for model generation
5. HTTP Request: Calls Nano Banana API
6. Set Field: Extracts generated model image
7. Base64 to File: Converts back to usable format
8. Google Drive: Stores model reference

Phase 3: Scene Creation

9. Set Field: Meta-prompt for scene generation
10. HTTP Request: Gemini 2.5 Pro analyzes product+model
11. Set Field: Extracts generated prompts
12. Split Out: Separates prompts for batch processing

Phase 4: Image Generation

13. HTTP Request: Nano Banana executes each prompt
14. Google Drive: Stores final images in organized folder

Pro Tip: Add a 5-second delay between image generations to avoid API rate limits while maintaining fast processing times.
