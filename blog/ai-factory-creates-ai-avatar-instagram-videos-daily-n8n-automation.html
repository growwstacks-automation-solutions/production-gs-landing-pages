Complete System Overview

The full automation handles every step from script generation to social posting. At 9:00 AM daily, the system wakes up and:

  1. Selects the next content pillar from Airtable (e.g., "Forgiveness", "Letting Go")
  2. Chooses a different avatar setting from your library (temple, lake, olive grove etc.)
  3. Generates a unique 200-word script using Claude AI
  4. Creates the video in HeyGen with your avatar reading the script
  5. Adds captions automatically via Creatomate
  6. Posts to Instagram/TikTok through Blot
  7. Logs everything in Airtable for tracking

The only manual work required is the initial setup and occasionally updating your content pillars. After that, the system produces and posts videos indefinitely with zero daily effort.

Creating Your AI Avatar

The foundation of your automated content is a consistent yet varied AI character. The viral monk account cycles through different outfits and settings while maintaining core recognition factors. To replicate this:

Step 1: Generate Base Avatar Images

Using Midjourney or Stable Diffusion, create your core character. For our example, we developed a "wise shepherd" figure through iterative prompting:

Prompt example: "A wise, elderly Christian man with kind eyes and a long beard, wearing simple robes, sitting peacefully outside an ancient church at golden hour, photorealistic style"

Step 2: Create Variations

Generate 6-10 versions of your avatar in different settings (church, garden, desert etc.). Maintain identical facial features and clothing style while changing backgrounds. These variations prevent audience fatigue from seeing the exact same image daily.

Step 3: Build Talking Avatars in HeyGen

For each variation:

  1. Upload the image to HeyGen
  2. Create or select a voice (use ElevenLabs for custom voices)
  3. Save each as a separate avatar with unique ID

The system will automatically rotate through these avatar IDs to keep content fresh.

n8n Automation Breakdown

The workflow connects all components through a series of logical steps:

Daily Trigger

A schedule trigger initiates the workflow each morning. This can be adjusted to your optimal posting time.

Content Selection

Two code nodes:

  1. Selects the next content pillar (topic + message)
  2. Chooses an avatar setting that hasn't been used recently

Script Generation

Claude AI receives:

  • Your character description
  • Selected topic and message
  • Instructions for video length (typically 90 seconds)

It outputs both the video script and social media caption.

Video Production

The script and avatar ID feed into HeyGen through an HTTP request. The system then polls HeyGen until video generation completes (typically 11 minutes).

HeyGen Video Generation

The HeyGen node requires three key configurations:

1. Avatar ID

Pulled dynamically from your Airtable based on the selected setting.

2. Voice ID

Your custom or selected voice ID from HeyGen's library. For maximum authenticity:

  • Record 30+ seconds of clear speech if creating a custom voice
  • Match voice tone to your character (calm for spiritual guides, energetic for coaches)

3. Video Dimensions

Set to 9:16 aspect ratio (1080x1920) for Instagram Reels and TikTok. The system automatically:

  1. Submits the generation request
  2. Waits for completion
  3. Downloads the final video URL
  4. Logs it in Airtable

Automated Caption Creation

While optional, captions significantly boost engagement. Our system uses Creatomate ($54/month) to:

1. Apply Consistent Styling

All videos receive:

  • White background text boxes
  • Black, easy-to-read fonts
  • Timed captions matching the audio

2. Template Configuration

The JSON template includes:

  1. Positioning rules for captions
  2. Font styles and sizes
  3. Background opacity settings

You can customize this template in Creatomate's interface after importing.

3. Automatic Processing

The n8n workflow:

  1. Submits the HeyGen video to Creatomate
  2. Applies your template
  3. Waits for processing
  4. Saves the final captioned video

Social Media Auto-Posting

Posting Logic

Status Tracking
