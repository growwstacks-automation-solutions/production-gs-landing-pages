Complete Workflow Overview

This system connects 7 different services through n8n to automate UGC ad creation from start to finish. Here's how data flows through the pipeline:

Key Insight: The workflow uses Telegram as an image processing intermediary because Airtable's attachment URLs expire. Telegram provides permanent access to uploaded product images throughout the automation chain.

Step 1: Setting Up Airtable

Your Airtable base serves as both the input trigger and final output repository. Follow these exact steps:

Create Your Base Structure

  1. Create new base named "Photo/TikTok Shop Affiliate Ads"
  2. Add table called "Media Entities" with these exact columns:
    • Name (Single line text)
    • Type (Single select)
    • Create Video (Button)
    • Image File (Attachment)
    • Product Description (Long text)
    • Video URL (URL)
    • Status (Single select with options: Not Started, Ready to Process, Video Processing, Done)
    • Create Time (Created time)

Configure API Access

Generate a personal access token in Airtable's developer settings with full permissions to your new base. This allows n8n to read and write records automatically.

Step 2: Configuring Telegram for Image Processing

Telegram handles image URL stability - a critical but often overlooked step. Here's how to set it up:

  1. Create a new bot via @BotFather (free)
  2. Note your bot token and chat ID
  3. In n8n, add these credentials to both Telegram nodes

This ensures product images remain accessible throughout the multi-step AI processing pipeline, preventing failed workflows from expired URLs.

Step 3: AI Services Setup (Gemini, OpenAI, Veo)

The workflow leverages four specialized AI services, each handling a different task:

Google Gemini (Image Analysis)

Generates detailed product descriptions from images. Get your API key from AI Studio.

OpenAI (Prompt Generation)

Creates scene descriptions and dialogue. Use your existing OpenAI API key.

Kaiber (Image Generation)

Produces professional model images. Sign up at kaiber.ai for an API key.

Veo 3.1 (Video Generation)

Transforms prompts into video clips. Currently in beta - apply for access.
