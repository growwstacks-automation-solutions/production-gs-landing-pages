Setting Up Your Vapi Assistant

Creating your car sales agent begins in the Vapi dashboard. After signing up, you'll:

  1. Click "Create Assistant" and name it (e.g., "Car Dealer Assistant")
  2. Customize the first message (the greeting callers hear)
  3. Configure the system prompt that defines the AI's behavior

The system prompt is especially crucial - it establishes the agent's role (car sales specialist), personality (friendly but professional), and critical rules like never hallucinating about inventory. A well-crafted prompt includes:

  • Clear instructions about only answering based on actual inventory data
  • Guidance on how to handle unavailable vehicles
  • Test drive booking protocols
  • Brand voice and tone guidelines

Pro tip: Start with a proven prompt template rather than building from scratch. The demo uses a sophisticated prompt with guardrails against hallucinations that you can adapt for your dealership.

How to Prevent AI Hallucinations

Hallucinations - when AI makes up false information - are particularly dangerous in sales contexts. Imagine an agent telling customers about non-existent vehicles or incorrect pricing. The demo shows two key prevention methods:

1. Prompt Engineering: The system prompt includes strict instructions like "Never provide information about vehicles not in the inventory sheet" and "If unsure, say 'Let me connect you with a salesperson who can check that for you.'"

2. Model Selection: While stronger models like GPT-5 reduce hallucinations, they're slower and more expensive. The demo uses GPT-4 with prompt safeguards for optimal balance of accuracy and performance.

The difference is dramatic - when the hallucination prevention snippet is removed, the AI confidently describes imaginary BMWs in stock. With it enabled, the agent properly defers questions it can't answer from the inventory data.

Connecting Live Inventory Data

The magic happens when connecting your vehicle inventory to the AI agent. The system uses:

  1. A Google Sheet containing all available vehicles with columns for make, model, year, price, mileage, features, and notes
  2. An n8n workflow that serves as the bridge between Vapi and Google Sheets
  3. Vapi's "Custom Tool" feature to query inventory through the n8n API

When a customer asks "Do you have any Toyota RAV4s from 2024?", the AI:

  1. Sends the query to n8n through the custom tool
  2. n8n fetches the current Google Sheet data
  3. The workflow filters for matching vehicles
  4. Results return to Vapi for the AI to formulate a natural response

This entire process happens in under 2 seconds, making the interaction feel instantaneous to callers.

The n8n Automation Workflow

The n8n workflow is the engine that powers the inventory lookup. Here's how it works:

  1. Webhook Trigger: Receives queries from Vapi when customers ask about vehicles
  2. Google Sheets Node: Connects to your inventory spreadsheet and fetches current data
  3. Item Lists Node: Consolidates all vehicle entries into a single queryable dataset
  4. Response Node: Returns the filtered results to Vapi for the AI to respond

Setting this up requires:

  • Creating a free n8n account
  • Importing the pre-built car inventory workflow template
  • Connecting your Google Sheet (make a copy of the template sheet first)
  • Adding your Google API credentials for access

Implementation note: The workflow handles up to 400 inventory items efficiently. For larger dealerships, we recommend connecting to a proper dealership management system instead of Google Sheets.

Customizing the Agent's Voice

The voice makes a huge difference in customer perception. Vapi offers multiple options:

  1. Built-in Voices: Several natural-sounding options across genders and accents
  2. ElevenLabs Integration: For premium, ultra-realistic voices (used in the demo)
  3. Custom Voice IDs: Clone specific voice profiles using ElevenLabs voice IDs

To change voices:

  1. Navigate to Voice Configuration in your Vapi assistant
  2. Select a provider (Vapi or ElevenLabs)
  3. Choose from available voices or enter a specific voice ID
  4. Click Publish to update your live agent

The demo uses a female voice named "Kylie" from ElevenLabs - calm, professional, and slightly upbeat, perfect for sales conversations. You can test different voices to match your dealership's brand personality.
