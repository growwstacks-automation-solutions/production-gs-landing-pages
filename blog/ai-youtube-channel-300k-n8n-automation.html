4-Step Automation Workflow

The n8n workflow automates content creation through four connected phases. Each step hands off to the next, requiring zero manual intervention after initial setup.

  1. Create Story: AI generates scripts based on viral templates
  2. Create Images: Consistent character art using NanoBanana
  3. Create Videos: AI video synthesis with dynamic scenes
  4. Combine Videos: Final assembly with FFmpeg automation

At 4:30 in the tutorial, you'll see how Google Sheets acts as the control panel - simply adding a row with "status: create" triggers the entire workflow. This makes it easy to queue up weeks of content in advance.

Step 1: AI Story Creation

The magic begins with prompt engineering. At 6:10, the tutorial shows how the AI Agent node combines:

  • Reference thumbnails from top-performing videos
  • Analyzed image descriptions (from OpenAI's vision API)
  • Story templates proven to engage viewers

This creates a feedback loop where each successful video improves future content. The system outputs both image prompts (for character consistency) and video scene descriptions in JSON format.

Key insight: Adding "system message" instructions (shown at 7:45) dramatically improves output quality by guiding the AI's creative process.

Step 2: Consistent Image Generation

Character consistency is the holy grail of AI content. At 9:20, you'll see how the workflow uses:

  • Kaiber AI's NanoBanana model (selected for character stability)
  • Reference images to maintain visual continuity
  • Batch processing of 5-10 scenes per video

The tutorial demonstrates a clever technique at 10:15 - using the "wait" node to pause execution while images render. This prevents API timeouts when running the full workflow automatically.

Step 3: AI Video Synthesis

VideoGen's API (shown at 12:30) transforms static images into dynamic scenes. The JSON configuration includes:

  • Camera movements (zooms, pans for cinematic effect)
  • Scene transitions matching the narrative rhythm
  • Automatic lip-sync for any added voiceover

At 13:45, notice how the workflow handles errors gracefully. If one scene fails, the system continues with others rather than stopping the entire job.

Step 4: Final Video Assembly

The last phase (shown at 15:00) uses FileAI's FFmpeg API to:

  • Combine individual scenes into a cohesive narrative
  • Add standardized intro/outro bumpers
  • Apply audio normalization for consistent volume

The finished video automatically uploads to a Google Sheet with "status: done" - ready for review or direct publishing. At 16:20, you'll see the final rendered output with perfect scene transitions.
