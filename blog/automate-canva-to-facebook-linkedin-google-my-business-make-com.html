How the Automation Works

The Make.com workflow acts as a bridge between your content (Google Sheets), designs (Canva), and publishing platforms (Facebook, LinkedIn, Google My Business). Here's the step-by-step flow:

  1. Checks your Google Sheet for unpublished posts (where status column is empty)
  2. Pulls the next post content from the designated column
  3. Retrieves the matching Canva design based on row number
  4. Publishes to all connected platforms simultaneously
  5. Updates the Google Sheet status to "done"

The magic happens through Make.com's visual workflow builder, which connects these services through their APIs without requiring any coding. At 2:15 in the video tutorial, you can see how the modules connect in a logical flow.

Setting Up Google Sheets

Your Google Sheet serves as the content queue for the automation. The tutorial uses a simple structure with three key columns:

  • Column B: Post title (optional)
  • Column C: Status (blank = unpublished, "done" = published)
  • Column D: Post content (what actually gets published)

The Make.com scenario first searches for rows where Column D has content but Column C is empty. This finds unpublished posts waiting in the queue. After successful posting, it updates Column C to "done" so the same post isn't published again.

Pro Tip: Add a date column to schedule posts in advance. Modify the search filter to only select rows where the date matches today.

Connecting Canva Designs

Canva integration brings visual consistency to your automated posts. The tutorial shows how to:

  1. Create a multi-page Canva design where each page matches a Google Sheet row
  2. Connect your Canva account to Make.com
  3. Configure the "Export a Design" module to pull the correct image

The key is matching your Canva page numbers to Google Sheet row numbers. Row 2 in your sheet corresponds to Page 2 in your Canva design. This automatic pairing ensures the right image goes with each post.

At 4:30 in the video, you'll see how to set the export quality (100% JPEG recommended) and map the design URL to be used in the social media posts.

Configuring Facebook Posting

Facebook Page posting requires:

  • Admin access to a Facebook Page (not just a personal profile)
  • Approved Facebook Developer account
  • Make.com Facebook connection

The tutorial shows how to use the "Create a Photo Post" module to publish to your Page. You'll map the Canva image URL to the photo field and the Google Sheet content to the post caption. The module handles all the API calls behind the scenes.

Note: Facebook's API sometimes changes requirements. If you encounter errors, check our Make.com automation service page for updated connection guides.

Automating LinkedIn Posts

LinkedIn Company Page integration follows a similar pattern to Facebook but with different module settings:

  1. Connect your LinkedIn account in Make.com (requires Page admin access)
  2. Select "Share on LinkedIn" module
  3. Map the Canva image as the post image
  4. Pull the Google Sheet content for the post text

Unlike Facebook, LinkedIn doesn't separate image URLs and captions - everything goes into a single rich media post. The tutorial demonstrates how to format this at 6:45.

For businesses that post frequently, this automation can save 3-5 hours per week just on LinkedIn content distribution.

Google My Business Integration

Google My Business (GMB) posting helps local businesses appear more active in search results. The automation adds GMB posts with:

  • Your business location selection
  • Post title from Google Sheets
  • Content from the designated column
  • Canva image as the post photo
  • Call-to-action button (optional)

The tutorial mentions that GMB connection requires additional setup not shown in this video. At 8:10, it recommends checking the description for a link to a dedicated GMB connection guide.

Scheduling the Automation

The final step is setting when your automation runs. Make.com offers several scheduling options:

  1. Daily: Post at the same time every day (ideal for most businesses)
  2. Specific days: Only post on weekdays or selected days
  3. Custom interval: Every X hours/days for irregular schedules

The tutorial sets a daily schedule at 9:00 AM, but you can choose any time that matches when your audience is most active. The automation will check your Google Sheet at this time and publish any queued posts.

Advanced Tip: Create multiple scenarios with different schedules if you need to post more than once per day.
