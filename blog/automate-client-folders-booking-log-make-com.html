How the Automation Works

This workflow connects three core business apps to create a seamless onboarding system. When a new event appears in your Google Calendar (presumably from a booking system like Calendly or Acuity), Make.com automatically:

  1. Creates a dedicated Dropbox folder for the client
  2. Generates a shareable link to that folder
  3. Logs all booking details in a Google Sheet

The entire process happens in seconds without any manual intervention. At 2:15 in the video tutorial, you can see how the test booking "Bloom Studios" triggers the workflow perfectly on the first try.

Step 1: Google Calendar Trigger

The automation begins by watching for new events in your Google Calendar. This is configured as the "watch events" trigger in Make.com, specifically monitoring your bookings calendar.

Key configuration points:

  • Connect your Google account to Make.com
  • Select your specific bookings calendar (not just the primary calendar)
  • Set the limit to 1 event per trigger
  • Choose "Run scenario once" for initial testing

When testing, the tutorial shows how "Bloom Studios" appears as the first test event, confirming the Google Calendar connection works correctly.

Step 2: Create Dropbox Folder

For each new booking, the workflow creates a dedicated client folder in Dropbox. The folder name comes from the event summary in Google Calendar.

Configuration details:

  • Connect your Dropbox account to Make.com
  • Select "Create a folder" action
  • Set folder name to the Google Calendar event summary
  • Choose your existing "Clients" folder as the parent directory

In the tutorial at 3:52, you can see how "General Architects" becomes the folder name, matching the calendar event exactly. This ensures consistent naming across all systems.

Step 3: Generate Share Link

After creating the folder, the workflow generates a shareable link. This link will be included in the Google Sheets log for easy access.

Implementation notes:

  • Use the "Create or update a share link" action in Dropbox
  • Set the file path to the newly created folder path
  • No additional settings needed for basic sharing

The tutorial demonstrates at 4:30 how "Awesome Arts" becomes the third test event, with all modules processing successfully. The share link is created but won't be used until the Google Sheets step.

Step 4: Log in Google Sheets

The final step adds a row to a Google Sheet with all booking details and the Dropbox folder link. This creates a searchable log of all client bookings.

Key configuration elements:

  • Connect Google Sheets and select your booking log spreadsheet
  • Map calendar fields to sheet columns (client name, date, etc.)
  • Format the date using: formatDate(Start; "MM/dd/yyyy hh:mm a")
  • Create a clickable folder link using: =HYPERLINK("[URL]", "[Folder Name]")
  • Set "Overwrite" in advanced settings for clean formatting

At 6:15 in the video, you can see the completed log entry for "Apple Interiors" with properly formatted date and clickable folder link.
