OpenAI Affirmation Setup

The heart of this automation is OpenAI's text generation capability. In Make.com, we use the "Generate a response" module with carefully crafted instructions that ensure high-quality, varied affirmations.

The prompt parameters include:

  • Addressing the reader directly as "you"
  • Uplifting but grounded tone
  • 5-15 word length
  • Themes of happiness, self-worth, and resilience
  • No religious or medical references

At 3:12 in the video tutorial, you'll see the exact prompt structure that produces messages like "You greet crisp fall light with grounded confidence and kind curiosity." This specificity ensures meaningful content rather than generic platitudes.

Text-to-Speech Conversion

Converting the text affirmation to speech creates a more impactful experience. Make.com's OpenAI integration includes text-to-speech capabilities with multiple voice options.

The key configuration points:

  1. Select the voice (Shimmer used in example)
  2. Set output filename with date for uniqueness
  3. MP3 format for compatibility
  4. Normal speech speed (1.0)

Pro Tip: Test different voices during setup to find one that resonates with you. The vocal tone significantly affects how you receive the message each morning.

Telegram Bot Configuration

Telegram provides the perfect delivery channel because of its robust bot API and voice message capabilities. Setting up the bot involves:

  1. Creating a new bot with BotFather
  2. Generating API token
  3. Creating private channel
  4. Adding bot as admin
  5. Getting chat ID via IDBot

At 7:45 in the video, you'll see the exact steps to forward a message to IDBot and retrieve your channel's unique identifier. This chat ID is crucial for Make.com to know where to send your daily messages.

Scheduling Your Daily Messages

The final piece is setting the automation schedule in Make.com. The scheduler allows you to:

  • Choose specific times (e.g., 9:06 AM)
  • Set daily recurrence
  • Activate the scenario

This ensures your affirmation arrives at the perfect moment in your morning routine. The system will automatically generate a new message, convert it to speech, and deliver it without any ongoing effort from you.
