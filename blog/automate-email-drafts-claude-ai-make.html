Step 1: Setting Up the Make.com Scenario

The foundation is a Make.com scenario that creates email drafts in Gmail. Here's how to build it:

Step 1: Create New Scenario

In Make.com, click the "+" button to create a new scenario. Search for and select "Gmail" as your first module.

Step 2: Add Create Draft Action

Choose "Create a draft email" from the Gmail module actions. This will be the endpoint that receives the AI-generated content.

Step 3: Set to On-Demand

Critical step: Change the scenario trigger to "On-demand" so Claude can activate it through MCP. Name it something clear like "Create Email Draft."

Pro Tip: At 4:30 in the video, you'll see how to properly configure the on-demand setting which is essential for MCP integration.

Step 2: Configuring Inputs and Outputs

For Claude to properly communicate with Make.com, you need to define the data structure:

Required Inputs:

  1. Recipient Email (Text, Required)
  2. Email Subject (Text, Required)
  3. Email Body (Text, Required)

Output:

Simple status confirmation like "Done" to let Claude know the draft was created.

At 5:45 in the tutorial, you'll see exactly how to set these up in Make.com's input/output configuration panel. The key is marking all three fields as required so Claude knows what information to provide.

Step 3: Connecting via MCP (Make Connection Protocol)

The bridge between Claude and Make.com requires an API token:

  1. Go to your Make.com profile → API Access
  2. Click "Add Token" and select all permissions (temporarily)
  3. Copy the generated API key
  4. Construct your MCP connection URL: 
    https://eu1.make.com/api/v1/mcp/[YOUR_ORG_ID]?token=[YOUR_API_KEY]

At 7:50 in the video, you'll see where to find your organization ID in the Make.com URL. This ensures Claude only accesses your specific organization.

Security Note: After setup, you can refine the API token permissions to only include MCP access rather than "all" permissions.

Step 4: Integrating with Claude AI

The final step connects everything in Claude:

  1. In Claude, click the "+" button → Connectors → Manage Connectors
  2. Add a Custom Connector named "Make" or "Email Drafts"
  3. Paste your MCP connection URL
  4. Save the connector

Now when you ask Claude to create email drafts, it will:

  • Analyze your request and past communications
  • Structure the data according to your Make.com inputs
  • Trigger the scenario through MCP
  • Return confirmation when drafts are created
