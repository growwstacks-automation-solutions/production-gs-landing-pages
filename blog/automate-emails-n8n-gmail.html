Setting Up Your Google Sheets Log

Before building the n8n workflow, we need to prepare our logging system. The Google Sheet serves three critical functions:

  • Permanent record of all support interactions
  • Quality control dashboard for AI performance
  • Audit trail for customer service metrics

Create a new Google Sheet with these exact column headers in row 1:

date | from | subject | category | priority | sentiment | summary | needs_human | auto_replied

The timestamped log becomes invaluable for tracking response times, identifying common questions to improve your FAQ, and monitoring customer sentiment trends over time.

Configuring n8n Step-by-Step

With our Google Sheet ready, we'll now build the automation in n8n:

Step 1: Gmail Trigger

Create a new workflow named "Gmail AI Support Agent". Add a Gmail trigger node configured to:

  • Poll every minute
  • Filter for unread emails only
  • Include full message content

Step 2: Data Formatting

The code node transforms raw Gmail data into a clean prompt for the AI. It extracts:

  • Sender's email address
  • Email subject
  • Message body (plain text)
  • Thread ID
  • Message ID

Step 3: AI Analysis

The AI agent node uses OpenAI's Gemini 1.0 Mini to analyze each email and return:

Structured JSON containing: category, priority, sentiment analysis, needs_human flag, suggested reply, and one-sentence summary

AI Response Handling & Safety

AI responses need careful handling to ensure reliability. The parse node acts as a safety net:

  • Strips any markdown or extra text
  • Validates JSON structure
  • Flags problematic responses for human review
  • Merges AI data with original email content

At 8:45 in the video, you'll see how the system gracefully handles a malformed AI response by automatically routing it to human support rather than sending a potentially incorrect reply.

Human Escalation Pathway

The IF node splits the workflow based on the needs_human flag:

Auto-Reply Path (needs_human = false)

  • Gmail node sends the AI-generated reply
  • Response appears in the original email thread
  • Google Sheets logs the interaction as auto-resolved

Human Path (needs_human = true)

  • Sends alert to your support team
  • Includes all analysis data for context
  • Can be configured for Slack/Discord instead of email

Pro Tip: Add conditional formatting in Google Sheets to highlight human-review rows in red and high-priority items in orange for at-a-glance triaging.
