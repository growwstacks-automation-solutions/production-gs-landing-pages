Method 1: From Template (Recommended)

This is our most-used approach for client reports and standardized documents. You create a Google Doc template with placeholder variables (like {{client_name}}), then n8n replaces these with actual data from your spreadsheet or CRM.

How It Works:

  1. Create your Google Doc template with mustache-style variables ({{variable_name}})
  2. Set up a Google Sheet with client data (or connect your CRM)
  3. Use AI to analyze the data and generate insights (optional)
  4. n8n copies the template and replaces all variables with actual values
  5. The finished doc gets saved to Drive and shared automatically

Pro Tip: Use Google's built-in templates as starting points. Their report templates already have professional formatting you can customize with your variables.

At 4:32 in the video, you'll see how we replace images in the template by locating their element IDs - crucial for maintaining branded documents with logos.

Method 2: From Markdown

When you need more flexibility in document structure or want human review before finalizing, generating from markdown is ideal. The workflow creates a draft document from markdown text, which you can then paste into a Google Doc with proper formatting.

Key Advantages:

  • Easier to generate from AI systems that output markdown
  • Allows for manual review and tweaks before finalizing
  • Simpler than HTML for basic documents

The secret sauce is enabling markdown paste in Google Docs (Tools > Preferences > Enable Markdown). This converts markdown formatting like headings, lists and tables into proper Google Docs styling with one click.

Method 3: From HTML

For maximum control over document styling and layout, generating from HTML is the most powerful approach. This method shines when you need:

  • Complex table structures
  • Precise image positioning
  • Custom styling for different sections
  • Documents generated from web-based tools

While more technical to set up, the HTML method provides pixel-perfect results. At 14:20 in the video, you'll see how we use the Google Drive API to convert HTML directly into a formatted Google Doc - no manual steps required.
