Setup Requirements

Before building your Instagram DM automation, gather these three essential components:

  1. n8n account: Use either their cloud version (free tier available) or self-hosted instance
  2. Facebook Developer account: Required to connect to Instagram's API (must be linked to an Instagram business profile)
  3. OpenAI API key: Get this from platform.openai.com (requires billing enabled, but costs are minimal)

The entire setup takes about 30 minutes if you follow our step-by-step configuration guide below. Most of that time is spent in the Facebook Developer Portal, which has the steepest learning curve.

Facebook Developer Configuration

Facebook's recent interface changes have broken many existing Instagram automation tutorials. Here's the updated process that works as of :

  1. Go to developers.facebook.com and create a new app (select "Business Messaging" as the use case)
  2. Add your Instagram account as a test user under App Roles → Roles → Add People
  3. Navigate to Use Cases → Instagram → Configure and generate an access token (save this securely)
  4. Enable webhook subscriptions and prepare to enter your n8n webhook URL

Pro Tip: Facebook now requires a privacy policy URL and app logo before publishing. Use a simple Google Doc for the policy and Canva to create a basic logo - we show examples at 8:15 in the video.

n8n Webhook Setup

With Facebook configured, switch to n8n to establish the connection:

  1. Create a new workflow and add a Webhook node (HTTP method)
  2. Copy the webhook URL and paste it into Facebook's developer settings
  3. Add an IF node to handle the verification challenge from Facebook
  4. Connect a "Respond to Webhook" node that returns the hub.challenge parameter

At this stage (12:40 in the video), you'll test the connection by sending a verification request from Facebook. The workflow should respond successfully, establishing the webhook for live messages.

AI Agent Configuration

The AI agent node transforms incoming messages into intelligent responses. For best results:

  1. Add an AI Agent node after the webhook
  2. Define the user message from the webhook's message.text parameter
  3. Create a detailed system prompt explaining your business and response style
  4. Select the gpt-3.5-turbo model (or gpt-4 for complex use cases)

At 18:22, we demonstrate a prompt template for a course business that handles inquiries, provides details, and follows up automatically. The key is including examples of ideal responses and conversation flows.

Sending Responses Back to Instagram

The final piece connects the AI's output back to Instagram:

  1. Add an HTTP Request node configured to POST to Instagram's API
  2. Use the graph.instagram.com endpoint (not facebook.com - a common mistake)
  3. Insert the page access token you saved earlier
  4. Map the recipient ID from the webhook's sender.id
  5. Pass the AI's response as the message text

We include error handling at 24:50 to manage special characters and undefined values that might break the JSON formatting.

Going Live With Your Automation

Before activating your workflow:

  1. Switch from the test webhook URL to your production n8n URL
  2. Publish your Facebook app (requires privacy policy and logo)
  3. Activate the workflow in n8n
  4. Test with real messages from your Instagram account
