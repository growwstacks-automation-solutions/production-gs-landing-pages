How the System Works

The Social Wizard automation follows a precise 6-step sequence that runs daily without human intervention:

  1. Form Trigger: Client submits brand details once via GoHighLevel form
  2. Data Storage: Information saved to custom fields in GoHighLevel
  3. AI Caption Generation: GPT-4 creates post text based on brand parameters
  4. AI Image Creation: GPT Image 1.5 generates matching visual content
  5. Media Processing: Image converted to JPEG and uploaded to Cloudinary
  6. Instagram Posting: Caption + image published automatically via Facebook API

At 2:15 in the video tutorial, you'll see how the form captures all necessary brand elements - from voice tone to target locations - that make each post feel personalized rather than generic AI content.

GoHighLevel Setup

The foundation of the automation begins in GoHighLevel with three key components:

1. Custom Fields Creation

Navigate to Settings → Custom Fields and create these essential fields:

  • Brand Name
  • Business Niche
  • Primary Offer
  • Tone of Voice (professional/comedy/casual)
  • Target Audience
  • Call to Action URL

2. Form Building

Under Sites → Forms, create a new form that includes:

  • All custom fields from above
  • Email field (required for submission)
  • Platform selection (Instagram/Facebook/LinkedIn)
  • Wizard Status dropdown (Configured/Paused)

3. Workflow Triggers

Set up three automation triggers:

  1. Schedule: Daily at midnight to initiate posts
  2. Form Submitted: When client completes the setup form
  3. Contact Changed: To pause/resume posting via form

Pro Tip: The "Wizard Status" field acts as an emergency stop button - clients can pause automation without needing to contact you (shown at 5:42 in the video).

Make.com Configuration

The Make.com workflow handles the technical heavy lifting with these key nodes:

Webhook Receiver

Creates the connection point from GoHighLevel with these parameters:

  • Method: POST
  • Content Type: application/json
  • Data Structure: {"caption":"{{chatGPTresponse}}"}

AI Image Generation

Uses OpenAI's GPT Image 1.5 model with:

  • Prompt: The received caption text
  • Size: 1024x1024
  • Quality: Standard

Image Processing

Critical steps to prepare images for Instagram:

  1. Convert PNG to JPEG (Instagram compatibility)
  2. Upload to Cloudinary (reliable public URL)
  3. Store secure URL for posting

At 8:30 in the tutorial, you'll see why we abandoned Google Drive - Instagram now blocks Drive-sourced images even with public sharing enabled.

AI Content Generation

The secret to authentic automated posts lies in the GPT-4 prompt engineering. Our optimized prompt:

"You are a social media manager for [Brand]. Write a single engaging caption about [Primary Offer] for [Target Audience] in [Location]. Use a [Tone] voice. Include a call-to-action to [CTA URL]. For geotagging, mention [Location] specifically. Return only the caption text on a single line without special characters or quotes."

This structure ensures:

  • Brand-consistent messaging
  • Location-relevant hashtags
  • Proper voice and tone
  • Clean formatting for API processing

Instagram Posting

The final step connects to Instagram via Facebook's API with strict requirements:

Authentication

Must use:

  • Facebook Business account
  • Connected Instagram profile
  • Page admin permissions

Data Formatting

Each post requires:

  • Public image URL (Cloudinary)
  • Caption text (from GPT-4)
  • Proper line breaks for readability
  • Hashtags (auto-generated from location/niche)
