Setting Up Your Make.com Account

Make.com (formerly Integromat) serves as the automation hub connecting your forms to Pipedrive. The platform offers a free tier that handles up to 1,000 operations monthly - perfect for small teams.

After creating your account, navigate to the Scenario builder where we'll construct the automation logic. The interface uses a visual workflow designer with drag-and-drop modules for each step in the process.

Connecting Your Lead Capture Forms

Whether you use Typeform, Google Forms, or another platform, the integration follows the same pattern. In Make.com, add your form tool as the first module in the scenario.

For Typeform users (shown at 3:42 in the video): authorize the connection, select your form, then map all response fields you want to capture. The system will automatically trigger whenever someone submits the form.

Pro tip: Create a dedicated "Lead Source" field in your form to track which campaigns, ads, or pages generate the highest quality leads.

Integrating Pipedrive CRM

Pipedrive's API allows Make.com to search, create, and update records automatically. Add the Pipedrive module after your form connection, then authenticate using OAuth.

The first Pipedrive action should be a "Search Person" operation using the submitter's email address. This becomes the foundation for duplicate prevention - if the email exists, we'll update rather than create.

Building Duplicate Prevention Logic

At 5:18 in the tutorial, we add a router that branches based on whether the email search returns a match. This simple if/else logic prevents duplicate contacts while ensuring all new information gets captured.

For existing contacts: map the form responses to update relevant Pipedrive fields. For new leads: create a complete person record with all submitted data. The system handles both cases automatically.

Automating Company/Organization Mapping

Professional leads often include company information. The workflow can automatically create or link Organization records in Pipedrive, then associate the person with their company.

Add a second Pipedrive module to search for the organization by name. If none exists, create it using details from the form. Then reference this organization when creating/updating the person record.

Testing and Refining Your Workflow

Before going live, use Make.com's test mode to verify each step. Submit test forms with both new and existing email addresses to confirm the correct Pipedrive actions trigger.

Common refinements include adding field mappings for custom CRM properties or setting lead scores based on form responses. The system grows with your sales process.
