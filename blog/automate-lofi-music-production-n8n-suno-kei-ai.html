Complete Automation Workflow Overview

The system transforms a simple Google Sheet into a music production factory. You specify themes like "rainy cafe vibes" or "study beats" - the AI handles the rest on a schedule.

Here's the 4-stage automation pipeline (watch 2:15 in the video for a live demo):

  1. Music Generation: Suno AI creates original LOFI tracks via Kei.ai's API
  2. Artwork Creation: Local AI generates matching album covers
  3. Video Production: FFmpeg combines audio+visuals with animated text
  4. Social Publishing: Automatic uploads to YouTube/TikTok on schedule

Step 1: AI Music Generation with Suno

The workflow starts by querying your Google Sheet for pending track requests. Each row specifies:

  • Music description (e.g. "chill piano with rain sounds")
  • Overall theme
  • Number of variations to generate
  • Instrumental flag (true for LOFI)

n8n sends these parameters to Kei.ai's Suno API endpoint. For just 12 credits per track ($5 = 1,000 credits), you get:

  • Professional-quality LOFI composition
  • Two version variations (A/B testing built-in)
  • MP3 files saved to your S3/Minio storage

Pro Tip: The system automatically filters out poor-quality generations by analyzing the API response metadata before proceeding to image creation.

Step 2: Automated Album Art Creation

For each approved track, the workflow triggers a local Stable Diffusion instance to generate matching artwork. The AI agent:

  1. Analyzes the music's mood and theme
  2. Creates a detailed prompt (e.g. "anime-style rainy window scene")
  3. Generates high-res 4K images
  4. Names files consistently for the video pipeline

All artwork saves to your image bucket with the same naming convention as the audio files. The entire process runs locally, costing nothing beyond your existing hardware.

Step 3: Video Production & Branding

The magic happens when n8n orchestrates FFmpeg to combine assets:

  • Loops the artwork for the song's duration
  • Adds animated text showing the track title
  • Applies consistent branding (font/positioning)
  • Renders final MP4 files ready for upload

This all occurs through the NCA Toolkit's Dockerized FFmpeg instance. The workflow automatically:

  • Handles different aspect ratios for platforms
  • Adds your channel watermark
  • Generates both landscape and vertical formats

Step 4: Automated Social Media Posting

The final stage publishes content across platforms. Using Pallyy's API, the workflow:

  1. Uploads videos to YouTube as "unlisted"
  2. Formats perfect captions with hashtags
  3. Schedules posts for optimal times
  4. Updates the Google Sheet with publication status

You maintain complete control through the "man-in-the-middle" approval system. Before anything posts:

  • All content saves to Google Drive first
  • You can delete poor-quality tracks
  • Only approved videos proceed to publishing
