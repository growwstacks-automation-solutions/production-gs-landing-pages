Retell AI Agent Setup for Meat Stores

Configuring the Retell AI agent requires three key components:

1. The Conversation Prompt

A detailed prompt teaches the AI how to handle meat store orders, including:

  • Standard menu items and cuts
  • Common customer questions
  • Order confirmation flow
  • Pickup time options

2. Order Placement Tool

This function triggers when an order is complete, sending data to:

  • Customer name and phone number
  • Order items and quantities
  • Pickup time
  • Special instructions

3. Call Handling Logic

Rules determine when to:

  • Take an order
  • Transfer to human staff
  • Handle common questions

Pro Tip: Start with 10-15 common menu items, then expand as the AI learns your customers' ordering patterns.

The Order Numbering Logic Workflow

The automated order numbering system uses this Make.com workflow:

Step 1: Search for Today's Orders

The system queries Google Sheets for all orders with today's date.

Step 2: Count Existing Orders

A count module tallies how many orders already exist for the current date.

Step 3: Increment Order Number

The next order number becomes (current count + 1), resetting daily at midnight.

Step 4: Log Complete Order

The system adds the new row with all details plus the generated order number.

Example: If 3 orders exist for January 29, the next becomes #4. At midnight, the count resets and the first January 30 order becomes #1.
