The Manual Narration Problem

Real estate professionals know video tours convert 4x better than photos alone—but creating narrated versions eats valuable time. Between scripting, recording takes, editing audio, and syncing with video, a single 60-second narration can consume over an hour of an agent's day.

The workflow bottleneck becomes painfully clear when scaling to multiple listings. At 3:27 in the tutorial video, you'll see how this automation eliminates seven manual steps:

Before automation: Write script → Record voiceover → Edit audio → Source B-roll → Edit video → Sync narration → Upload final version

Workflow Overview

This n8n automation connects four specialized tools to handle each step of the narration process:

  1. Form Input: Collects property details and video URL
  2. OpenAI: Generates professional narration script
  3. ElevenLabs via FileAI: Converts text to natural voiceover
  4. FFmpeg via FileAI: Merges audio with source video

The entire process runs automatically after submitting basic property information. At 5:12 in the video, you can see the workflow in action—taking raw inputs and producing a polished video with AI narration.

Setup Steps

Getting started requires three credentials configured in n8n:

Step 1: Import the Template

Download the workflow template JSON file, then import it into your n8n instance via the three-dot menu (shown at 1:15 in the video). The imported nodes will show red X's until you configure the external service connections.

Step 2: Configure API Keys

Create credentials for:

  • OpenAI (script generation)
  • FileAI (voice and video processing)
  • Gmail (optional completion notifications)

Step 3: Customize Defaults

Modify the AI agent's system prompt to match your preferred narration style (professional, conversational, etc.) and select your default voice from ElevenLabs' library.

AI Script Generation

The workflow uses GPT-4 to transform basic property details into compelling narration scripts. At 7:45 in the video, you'll see how the AI agent:

  1. Takes property description, location, and listing type
  2. Follows instructions in the system prompt
  3. Generates a natural-sounding script

Pro Tip: Feed your best existing listing descriptions into the system prompt as examples to train the AI on your preferred style.

Voice Synthesis

ElevenLabs' multilingual v2 model converts the AI-generated script into natural speech. The workflow handles:

  • Voice selection (40+ professional options)
  • Automatic polling for completion
  • Error handling if processing delays occur

At 10:30 in the tutorial, you can hear the difference between raw text and the final AI voiceover—nearly indistinguishable from human recording.

Video Merging

FFmpeg seamlessly combines the source video with the new AI narration:

  1. Downloads both files from their URLs
  2. Matches audio to video length
  3. Applies professional audio ducking (lowers music during speech)
  4. Outputs a polished MP4 file

The workflow automatically retries if processing takes longer than expected (shown at 14:20 in the video), ensuring reliable completion.

Notification & Delivery

When processing finishes, the workflow can:

  • Email you the final video URL
  • Save all details to an n8n data table
  • Post to Slack/Teams
  • Upload to YouTube/Vimeo

At 16:45, the tutorial shows how to customize the completion email with property details and direct video links.

Customization Options

This workflow offers multiple adjustment points:

Voice Selection

Choose from ElevenLabs' voice library (demonstrated at 11:10) by copying voice IDs into the configuration.

Script Style

Modify the AI agent's system prompt to produce different narration tones—from ultra-professional to friendly and conversational.

Output Formats

Adjust FFmpeg settings for different video resolutions, aspect ratios, or file sizes depending on your distribution channels.
