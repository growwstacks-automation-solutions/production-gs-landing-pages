How Make.com Solves It

Make.com (formerly Integromat) provides a visual automation platform that connects Google Calendar and Slack. This workflow watches for new calendar events starting soon and sends customized Slack messages before they begin.

The automation runs continuously in the background, requiring zero maintenance once setup. Unlike manual calendar checks or last-minute scrambles to join meetings.

Key benefit: This automation takes just 15 minutes to setup but saves hours of missed work and stress every month.

Google Calendar Setup

The first component watches your Google Calendar for upcoming events. Here's how to configure it:

Step 1: Add Google Calendar Module

In Make scenario, search for and add the "Watch Events" module from Google Calendar. This will monitor your calendar for new events.

Step 2: Configure Calendar Settings

Select your primary calendar (or specific calendar you want to monitor). Set the trigger when events are created, modified, or about to start.

Step 3: Set Time Filter

Add a filter to only trigger for events starting within the next 15 minutes. This ensures you get the reminder with enough time to prepare.

Pro tip: If you manage multiple calendars, duplicate this module for each one or use the "Calendar List" module to watch them all.

Slack Configuration

Next, we'll setup the Slack portion to receive the reminders:

Step 1: Add Slack Module

Search for and add the "Send Message" module from Slack. Connect it to your Slack workspace if you haven't already.

Step 2: Customize Message

Create a clear reminder message including the event title, time, and any other relevant details from the calendar event.

Step 3: Select Recipient

Choose whether to send the reminder to yourself, a channel, or multiple team members of the meeting.

Customization idea: Add emojis or urgency indicators like ⏰ or 🔔 to make the reminders stand out in Slack.

Testing The Workflow

Before relying on the automation, it's crucial to test it works as intended:

Step 1: Create Test Event

Add a meeting to your Google Calendar starting in 20-30 minutes to give yourself time to test.

Step 2: Run Scenario

Trigger the Make scenario manually to process the test event and verify the Slack message.

Step 3: Verify Timing

Confirm the reminder arrives exactly 15 minutes before the event start time.

Testing tip: Schedule several test events at different times of day to ensure reliability.
