Setting Up n8n (3 Critical Steps)

Before building the workflow, you need to properly configure n8n:

Step 1: Update to Latest Version

Navigate to Admin Panel → Manage and enable verified community nodes. Click "latest stable" and save changes. This ensures access to all required nodes.

Step 2: Install Bloatato Nodes

Search for "Bloatato" in community nodes and install. These nodes handle the automated social media posting.

Step 3: Authenticate Claude API

Create an API key in Anthropic's console and authenticate in n8n. This powers the AI agent that generates video concepts.

Pro Tip: Set your schedule trigger first (the very first node). This determines how often your system will create and post new videos.

AI Agent Configuration

The AI agent is the creative brain of this operation. Here's how to set it up:

  1. Add an AI Agent node connected to your schedule trigger
  2. Select "Define Below" for both prompt and system message
  3. Attach Claude Sonnet 4.5 as the chat model
  4. Add simple memory with a random key (set to remember 10 messages)
  5. Include the "Think" tool for reprocessing when needed

The system message should include:

  • JSON output format requirements (title, description, prompt)
  • List of viral video types to rotate through
  • Specific instructions about maintaining bodycam aesthetic

SORA 2 Video Generation

With concepts generated, the system automatically creates videos:

  1. HTTP request node configured with KeyAI's SORA 2 API
  2. Prompt from AI agent inserted into generation parameters
  3. Wait node set to 450 seconds (7.5 minutes) for generation
  4. Second HTTP request to retrieve completed video

Cost Efficiency: At 10 cents per video, you could generate 100 videos (potential millions of views) for just $10.

The retrieved video URL then feeds into the Bloatato nodes for processing and posting.

Automated Posting Setup

Bloatato handles the final publishing stage:

  1. Upload Media node processes the SORA 2 video
  2. Create Post node formats the content for each platform
  3. AI-generated title and description are automatically inserted
  4. "Contains synthetic media" flag enabled for compliance

You can connect multiple platform accounts (YouTube, TikTok, Instagram) and the system will post to all simultaneously. For accounts where you want manual review first, add a Telegram node to send the video to your phone before publishing.

Scheduling Optimization

The schedule trigger determines your posting frequency:

  • High-volume: Every 4-6 hours (4-6 posts/day)
  • Moderate: Every 12 hours (2 posts/day)
  • Conservative: Once daily

Community testing shows optimal results with 4-6 posts daily, but this depends on your niche and platform. The system's low cost per video makes experimentation easy.
