Biteable API Overview

Biteable's video API accepts JSON-formatted requests containing your content, style preferences, and output parameters. The system handles all rendering, voiceovers, and music synchronization automatically.

Key features accessible via API:

  • Automatic text-to-speech voiceovers in 50+ languages
  • Template-based visual styles matching your brand
  • Royalty-free music synchronization
  • Multiple output formats (landscape, square, portrait)
  • Webhook notifications when rendering completes

The API uses a simple authentication model - just generate an API key in your Biteable account dashboard and include it in your request headers.

Customizing Automated Videos

While the API handles initial rendering, you maintain full creative control. Every API-generated video opens in Biteable's storyboard editor where you can:

These customizations become templates for future API videos, creating a consistent brand experience at scale.
