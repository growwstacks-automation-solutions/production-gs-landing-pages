How the n8n Workflow Works

At its core, this automation solves three critical problems content creators face: coming up with ideas, producing quality content, and maintaining consistent posting schedules. The n8n workflow connects specialized AI tools to handle each step without human intervention.

The process begins with a script agent that generates video concepts based on proven templates. It then creates detailed prompts for Sora 2 (through Key AI) to generate the actual videos. Finally, it uses Blotato to distribute the content across platforms with optimized captions and hashtags.

Step 1: Concept Generation

The script agent (using Anthropic's Claude) selects from pre-defined video types that have historically performed well. In Stephen's case, it chose "hyperrealistic Minecraft world with reflections" based on a template proven to generate 1.4M views.

Step 2: Video Creation

Key AI receives the detailed prompt and generates the actual video using Sora 2's capabilities. Each 10-second clip costs just $0.15 to produce, with the system automatically checking for quality before proceeding.

Step 3: Multi-Platform Distribution

Blotato handles the complex API connections to Instagram, TikTok and YouTube, posting each video with platform-specific captions and formatting. This avoids the need to manually manage multiple social media APIs.

Automation advantage: The entire workflow runs on a schedule (typically 3x/day) with error handling that retries failed steps automatically.

The 3 Key Components

This automation combines three specialized tools through n8n's workflow builder. Each serves a distinct purpose in the content creation pipeline:

1. n8n Workflow Engine

The backbone that connects all components. n8n handles the logic, scheduling, error recovery, and data passing between services. Its visual interface makes complex automations manageable without coding.

2. Key AI + Sora 2

Responsible for turning text prompts into high-quality videos. Key AI provides API access to Sora 2's advanced capabilities at just $0.15 per 10-second clip. The system automatically retrieves completed videos and checks for errors.

3. Blotato

Simplifies social media posting by providing unified API access to multiple platforms. Unlike manual API setups, Blotato maintains trusted relationships with platforms to ensure deliverability (critical for Stephen's 30M views).

Why this combination works: n8n orchestrates specialized tools that each excel at one part of the process, creating a system greater than the sum of its parts.
