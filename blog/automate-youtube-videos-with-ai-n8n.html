Complete Automation Workflow Overview

The system transforms spreadsheet rows into published YouTube videos through seven automated stages:

Step 1: Scheduled Trigger

The n8n workflow activates daily via schedule trigger, checking Google Sheets for new video projects marked "Ready". This eliminates manual initiation.

Step 2: Data Retrieval

Each row contains complete video specifications - title, theme, visual style, voice preferences, and scene count. The workflow pulls this structured data for processing.

Step 3: AI Content Generation

Gemini AI generates scene-by-scene scripts, image prompts, and voiceover text based on the specifications. This creates all raw content assets automatically.

Step 4: Media Production

Kai AI renders images for each scene while voice synthesis APIs create matching audio tracks. These components are stored in Google Drive.

Step 5: Video Assembly

JSON to video compiles all media assets into polished videos using your custom template, complete with transitions and branding.

Step 6: YouTube Upload

Finished videos upload directly to your channel with metadata pulled from the original spreadsheet entry.

Step 7: Status Updates

The workflow marks completed videos in Sheets and stores final URLs for tracking. The entire process runs unattended.

Key architecture: By separating content generation (Gemini), media production (Kai AI), and assembly (JSON to video), the system achieves professional quality while maintaining flexibility to swap components.

Google Sheets: Your Video Control Center

Google Sheets serves as the workflow's brain - a simple interface for managing hundreds of video projects without technical complexity. Each column controls a specific aspect of production:

  • Title: Final YouTube video title
  • Caption: Video description text
  • Story Idea: Core concept/theme
  • Character: AI persona/style
  • Visual Style: Cinematic/animated/etc
  • Scenes: Number of segments (3-10)
  • Production Status: Ready/Processing/Complete
  • Final Output: YouTube URL when done

To add new videos, simply fill a row and set status to "Ready". The workflow handles the rest. For batch creation, use ChatGPT to generate dozens of pre-formatted rows at once based on your channel's niche.

Pro tip: Create separate sheets for different video formats (tutorials, vlogs, etc) with columns tailored to each style. The workflow can process multiple sheets simultaneously.

AI Content Generation Process

Once the workflow retrieves a video project from Sheets, AI agents handle all creative work:

Script Generation

Gemini AI analyzes the title and story idea to produce a complete scene breakdown. Each scene includes:

  • Visual description for image generation
  • Voiceover script
  • Suggested duration
  • Transition notes

Image Creation

Kai AI's C Dream model transforms the visual descriptions into high-quality images. The workflow automatically:

  • Sets aspect ratio (16:9 landscape)
  • Applies consistent style across scenes
  • Stores renders in designated Drive folders

Voice Synthesis

Text-to-speech APIs convert scripts into natural narration using the specified voice profile. Options include:

  • Gender and age selection
  • Emotional tone adjustments
  • Multiple language support

Quality control: The workflow includes validation steps to ensure all generated assets meet minimum duration/clarity thresholds before proceeding to assembly.
