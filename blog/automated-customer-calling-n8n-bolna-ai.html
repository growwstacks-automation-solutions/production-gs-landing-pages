Bolna AI Voice Agent Integration

Bolna AI provides the voice agent platform that makes the actual phone calls. The n8n workflow serves as the brain that decides when to call, who who to call, and what to say. Integration happens through a simple HTTP request to Bolna's API endpoint.

The integration requires three key pieces of information:

  1. Agent ID: Identifies which pre-configured voice agent to use (you set these up in Bolna's dashboard)
  2. Recipient Number: The customer's phone number pulled from the payment data
  3. Your Bolna Number: The caller ID number you've purchased through Bolna's platform

Razorpay Webhook Setup

The entire automation starts with a Razorpay webhook trigger. When configured correctly, Razorpay will send real-time payment data to your n8n workflow the moment a transaction completes. Here's how to set it up:

Step 1: Create the Webhook in n8n

Use n8n's Webhook node as your trigger. Copy the unique URL it generates - this what you'll provide to Razorpay.

Step 2: Configure Razorpay Webhooks

In your Razorpay dashboard, navigate to Webhooks under Settings. Add new webhook with:

  • URL: Your n8n webhook address
  • Events: Select "payment.captured"
  • Secret: Optional but recommended for security

Step 3: Add 5-Minute Delay

Insert a Wait node after the webhook trigger. This creates buffer period to account for potential payment reversals buyer's remorse.

Pro tip: Test your webhook with Razorpay's test mode payments before going live. This verifies the data structure matches what your workflow expects.

API Configuration Steps

Connecting to Bolna AI's API requires careful setup of authorization headers and request parameters. At 4:50 in the video, you can see the exact configuration:

1. HTTP Request Node Settings

  • Method: POST
  • URL: https://api.bolna.ai/call (from their documentation)
  • Authentication: Bearer Token

2. Headers Configuration

Add these headers exactly:

  • Content-Type: application/json
  • Authorization: Bearer [your_API_key]

3. JSON Body Setup

The request body contains all call parameters:

  • agent_id: Your Bolna agent ID
  • recipient: Customer phone number
  • from: Your Bolna-purchased number

Finding your API key is simple - it's in the Developers section of your Bolna AI dashboard. The video shows exactly where to look at the 5:30 mark.

WhatsApp Group Follow-up System

When the Bolna agent detects that customer hasn't joined the required WhatsApp group (through their verbal response or lack of), the workflow automatically sends follow-up message with join link. This happens through separate WhatsApp Business API integration in n8n.

The complete sequence:

  1. Voice asks if they've joined group
  2. If "No" response or timeout triggers follow-up
  3. n8n sends templated WhatsApp message
  4. System logs the action for reporting
