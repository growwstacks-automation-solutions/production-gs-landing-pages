The Manual Ad Audit Problem

Every creative agency knows the drill - you land a discovery call with a potential client, then spend hours manually pulling their active ads from Meta Library, analyzing creative performance, and building a custom pitch deck. While effective, this process creates a major bottleneck in your sales pipeline.

The breaking point comes when you realize you're spending 3-4 hours per audit, often for prospects that don't convert. At 10 audits per week, that's 30-40 hours of non-billable work - nearly an entire work week lost to manual processes that could be automated.

Key insight: Agencies using manual ad audits typically cap out at 15-20 new client pitches per month. Automating this process removes the bottleneck, allowing you to scale to 50+ pitches with the same team.

Workflow Overview

This n8n automation transforms the manual audit process into a streamlined system with two distinct phases. The entire workflow triggers from just two inputs: the brand's website URL and their Meta Ads Library link.

Phase 1 - Context Building: The system scrapes the brand's website using Firecrawl API to extract branding elements (colors, fonts, messaging) and scrapes their Meta Ads Library via Appify to collect all active creatives. This establishes the foundational context for the audit.

Phase 2 - AI Transformation: Gemini 3.1 Pro analyzes the collected ads against the brand context, identifying creative strengths, weaknesses, and opportunities. This analysis then feeds into Gamma's API, which generates a polished, brand-aligned sales presentation automatically.

Phase 1: Context Building

The workflow begins with two parallel scraping operations that happen simultaneously. First, Firecrawl API extracts critical brand information from the company website - not just text content, but visual branding elements that will inform the presentation design.

Simultaneously, the Appify scraper pulls all active ads from the Meta Ads Library URL provided. This includes both image and video creatives, which the system processes differently based on format. The scraper handles pagination automatically, collecting every active ad regardless of quantity.

Technical note: The system uses temporary file storage for ad creatives to avoid broken links during processing. Facebook's ad URLs can expire during long-running automations, so we upload them to a temp service with 60-minute persistence.

Phase 2: AI Analysis

With all context collected, Gemini 3.1 Pro performs the creative audit. The key advantage of Gemini 3.1 is its multimodal capability - it can analyze both images and videos within the same evaluation framework.

The audit evaluates five key dimensions: creative mix balance (are they over-relying on one format?), messaging consistency with website positioning, visual branding alignment, creative fatigue indicators, and competitive differentiation opportunities. This generates a structured report that feeds directly into the presentation builder.

Accuracy benchmark: In testing, Gemini's analysis matched human strategists' conclusions 85% of the time when provided with detailed brand context. The remaining 15% typically involved nuanced creative choices that require subjective judgment.

Gamma Presentation Generation

The final phase transforms the audit report into a client-ready presentation. Gamma's API accepts the structured analysis along with brand visuals and ad creatives, then generates a PowerPoint-style deck automatically.

The template includes standard sections like Creative Performance Overview, Strengths & Opportunities, and Recommended Next Steps - all populated with the AI's findings. Crucially, it maintains brand consistency by using the scraped color scheme, fonts, and logo from the website.

At 4:15 in the video, you can see the final output - a professionally formatted deck that would typically take 2+ hours to create manually. The system even inserts actual ad creatives into the presentation for visual reference during the pitch.

Customization Requirements

While the demo shows a generic version, real implementation requires tailoring two key components. First, the Gemini prompt needs your specific audit framework - what metrics and dimensions you evaluate when analyzing client ads.

Second, the Gamma presentation template should mirror your agency's sales process. This includes your unique value proposition, service packages, and call-to-action flow. The demo provides the technical framework, but the strategic content must come from your playbook.

Implementation tip: Start by documenting your manual audit process step-by-step, then translate those steps into the Gemini and Gamma prompts. This ensures the automated version aligns with your existing successful approach.

Implementation Tips

For agencies implementing this system, three optimizations can dramatically improve results. First, enrich the brand scraping with additional sources like LinkedIn or Crunchbase to get fuller context about the company's positioning and competitors.

Second, integrate the output with your CRM - have completed presentations automatically save to the prospect's record in HubSpot or Salesforce. This creates a seamless handoff between marketing and sales.

Finally, build a library of presentation templates in Gamma for different service offerings (UGC, performance creative, full-funnel strategy). The workflow can then select the appropriate template based on the prospect's needs.
