The Contact Capture Problem

Every service business faces the same challenge: clients book appointments through your YouCanBookMe page, but their contact details remain trapped in the booking system. When you need to follow up, schedule future appointments, or add them to your CRM, you're stuck manually copying information between systems.

This creates three major problems:

  • Lost opportunities: 37% of businesses admit to losing client details between booking and follow-up
  • Time waste: Manual entry consumes 2-5 minutes per booking that could be automated
  • Data errors: Typos and incomplete transfers lead to unreliable contact records

The solution: Automatically push every new booking's contact details directly into Google Contacts using Zapier. No manual work. No missed clients. Just seamless contact management.

Step 1: Get Your YouCanBookMe API Key

Every Zapier integration starts with proper authentication. For YouCanBookMe, this means obtaining your unique API key:

  1. Log into your YouCanBookMe account
  2. Click your profile picture → My Account → Settings
  3. Locate the "Zapier API Key" section
  4. Click to reveal and copy your key

Security note: Treat this API key like a password. Anyone with access can connect to your booking data. Only share it with trusted automation tools like Zapier.

Step 2: Configure the Zapier Trigger

The trigger defines when Zapier should act. For this workflow, we want action whenever a new booking comes in:

  1. Create a new Zap in Zapier
  2. Search for and select "YouCanBookMe" as the trigger app
  3. Choose "Booking Notification" as the trigger event
  4. Authenticate using your API key
  5. Select which booking pages to monitor (or all)
  6. Set notification type to "Booking Created"

At 2:45 in the video tutorial, you'll see how to test this trigger with real booking data to confirm it's working before proceeding.

Step 3: Map Fields to Google Contacts

With the trigger set, we now configure the action - creating a Google Contact:

  1. Add an action step in Zapier
  2. Search for and select "Google Contacts"
  3. Choose "Create Contact" as the action
  4. Authenticate with your Google account
  5. Map YouCanBookMe fields to Google Contact fields:
    • First/Last Name → Name
    • Email → Email
    • Phone → Phone
    • Custom fields as needed

Zapier's AI will suggest common mappings, but you can customize exactly which information transfers where.
