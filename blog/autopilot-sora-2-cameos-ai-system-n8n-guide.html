Setting Up Your Sora 2 Cameo

The most crucial step is properly configuring your Sora 2 cameo. At 5:18 in the video, we show exactly how to:

  1. Download the Sora 2 app (US/Canada only without VPN)
  2. Complete the cameo recording (rotating head while saying numbers)
  3. Set cameo visibility to "Everyone" in settings
  4. Note your Sora 2 username (shown at top of app)

Pro tip: The system will fail if your cameo isn't set to public. Double-check this in the "Edit Cameo" section before proceeding.

Building the n8n Workflow

The n8n workflow consists of 7 key nodes that handle the automation from start to finish. At 9:40 in the tutorial, we walk through each one:

  1. Schedule Trigger: Controls how often the workflow runs
  2. Random Number Generator: Selects which video concept to use
  3. AI Agent (Claude): Generates the video script and caption
  4. HTTP Request (Key AI): Creates the Sora 2 video
  5. Wait Node: Pauses while video renders (10 minutes)
  6. HTTP Request (Key AI): Retrieves the finished video
  7. BloTato Nodes: Posts to social platforms

You'll need to enable "Verified Community Nodes" in n8n's admin panel (shown at 7:12) to access the BloTato integration.

AI Script Generation Process

The magic happens in the AI Agent node using Claude. At 11:25, we show the exact system prompt that:

  • Generates a catchy title ("Skydiving With My Howling Dog")
  • Creates an engaging description for social posts
  • Outputs the detailed Sora 2 prompt featuring your cameo

The key is replacing "primary person" in the template with your Sora 2 username. When done correctly, you'll get prompts like: "GoPro footage of [your_username] laughing while being chased by robotic bees in a sunflower field."

Automated Video Creation

Key AI's Sora 2 API handles the video generation at just 15 cents per video. The setup process (shown at 15:30) requires:

  1. Creating a Key AI account
  2. Adding API credits ($5 minimum)
  3. Configuring the HTTP request node with your API key
  4. Setting aspect ratio to "portrait" for mobile viewing

The wait node is crucial here - videos take 4-5 minutes to render. We recommend setting this to 600 seconds (10 minutes) to be safe.

Multi-Platform Posting with BloTato

BloTato solves the biggest headache in social automation - reliable API connections. At 22:10, we show how to:

Important: Always disclose AI content as required by platform policies. BloTato makes this easy with checkboxes in the post configuration.
