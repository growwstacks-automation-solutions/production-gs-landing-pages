Behind the Scenes: Prompt Engineering

The magic happens in the carefully constructed prompt that guides the AI's behavior:

Chain-of-thought prompting breaks down complex tasks into sequential steps the AI follows exactly - like a flowchart for conversations.

The prompt establishes:

  • Role: "You are a professional, friendly AI receptionist for accounting firms"
  • Rules: Never guess answers, always check knowledge base
  • Processes: How to handle scheduling, transfers, and Q&A
  • Tools: When to call calendar checking vs booking functions

Dynamic variables automatically insert:

  • Current time in the firm's timezone
  • Caller's phone number for SMS confirmations
  • Business-specific knowledge base references

Smart Calendar Availability Checking

The system uses a two-phase approach to calendar management:

Phase 1: Exact Time Request

When a caller asks "Do you have 3pm tomorrow?":

  1. AI confirms the request: "Let me check that for you"
  2. Sends time to Make.com workflow
  3. Checks Google Calendar via API
  4. Returns "Available" or "Unavailable" response

Phase 2: Alternative Suggestions

If unavailable, the system:

  1. Scans the same day for open slots
  2. Filters out non-business hours
  3. Uses AI to select 3 optimal alternatives
  4. Presents options conversationally: "We have 5pm, 6pm or 7pm available"

Pro Tip: The timezone handling is critical - the prompt ensures all times reference the firm's local timezone, not UTC.

Automated SMS & Email Confirmations

Once booked, the system triggers two confirmation paths:

SMS Confirmation

  • Uses caller's number automatically detected from the call
  • Includes Google Meet link for virtual appointments
  • Sent via Twilio within 30 seconds of booking

Email Reminder

  • Sent to the address collected during booking
  • Includes all appointment details
  • Uses firm's branded template
  • Scheduled to arrive 1 hour before the meeting

The entire confirmation system eliminates:

  • Manual calendar invites
  • Copy-paste of meeting links
  • Follow-up calls to confirm details
