n8n Basics: The Visual Automation Builder

n8n (pronounced "n-eight-n") is an open-source workflow automation tool that connects apps and services using a drag-and-drop interface. Unlike simpler automation tools, n8n supports complex workflows with conditional logic, error handling, and AI integration.

The core concept is simple: you create "nodes" for each step in your workflow (like fetching data, processing with AI, or sending emails), then connect them to define the flow. n8n handles all the technical details behind the scenes.

Two hosting options: You can use n8n Cloud for $20/month or self-host for free/cheaper. Self-hosting on Hostinger gives you more control and lower costs while maintaining reliability.

Setting Up Google API Credentials

To connect services like Gmail to your workflows, you'll need to set up OAuth credentials in Google Cloud Console. While this is the most technical part of the process (shown at 9:20 in the video), the steps are straightforward:

  1. Create a new Google Cloud project
  2. Enable the Gmail API
  3. Configure OAuth consent screen
  4. Create OAuth client credentials
  5. Connect to n8n

The n8n interface provides direct links to documentation that walks you through each step. Once set up, these credentials work for all your future workflows.
