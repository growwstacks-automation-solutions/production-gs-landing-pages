Workflow Overview: How the Pieces Connect

The complete system uses just 5 n8n nodes but delivers enterprise-grade functionality. At its core is an AI agent that makes decisions based on the conversation context and available tools:

  1. Chat Trigger: Starts the workflow when a new message arrives
  2. AI Agent Node: Processes conversations using GPT-4 with custom tools
  3. Code Tool: Validates dates and prevents weekend meetings
  4. Google Sheets: Logs all lead details and conversation summaries
  5. Google Calendar: Books meetings and sends invites automatically

What makes this powerful is how the components interact. When a lead mentions scheduling a meeting (like at 18:30 in the video), the agent uses the code tool to verify the date isn't a weekend, then triggers the calendar integration while simultaneously updating the lead record in Sheets.
