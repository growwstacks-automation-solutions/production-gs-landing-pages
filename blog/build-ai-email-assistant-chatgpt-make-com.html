How the AI Email Assistant Works

This automation acts as your first line of defense against email overload. When a new message arrives in your Gmail inbox, Make.com detects it instantly and sends the content to ChatGPT. The AI analyzes the message and drafts a professional response based on your instructions, then creates a Gmail draft with the reply ready for your review.

The magic happens in three stages:

  1. Monitoring: Make.com watches your Gmail inbox for new messages
  2. Processing: ChatGPT generates context-aware responses
  3. Drafting: Replies are formatted as Gmail drafts with original message included

Unlike fully automated systems that risk sending inappropriate replies, this workflow keeps you in the loop while handling the heavy lifting. At 2:15 in the video tutorial, you'll see how the draft appears exactly as you'd want it - with sender info preserved and professional tone maintained.

Step 1: Set Up Gmail Monitoring

The foundation of your AI assistant is the Gmail connection in Make.com. This allows the system to detect new emails and retrieve their contents for processing.

To configure:

  1. Create a new scenario in Make.com
  2. Add the Gmail "Watch Emails" module
  3. Connect your Gmail account (requires one-time authorization)
  4. Set the filter to monitor your inbox
  5. Enable "Mark as read" to prevent duplicate processing

Pro tip: For sales inquiries, set a filter like "label:inquiry" to only process qualified leads rather than all incoming mail. This prevents wasting AI resources on spam or newsletters.

Step 2: Connect ChatGPT

With Gmail configured, the next step is adding ChatGPT's intelligence to your workflow. This requires an OpenAI API key, which you can generate in your account settings.

Key configuration points:

  • Use the GPT-3.5-turbo model for cost efficiency
  • Set temperature to 0.7 for balanced creativity/consistency
  • Provide clear instructions in the prompt (shown at 3:42 in the video)

The prompt is where you define your assistant's personality. A sample structure:

 You are an assistant that writes professional email replies. Draft a polite and concise email response to the following message. Return only the email body text - no subject or greetings.

This ensures ChatGPT generates just the reply content, which we'll combine with the original message details in the next step.

Step 3: Automate Draft Creation

The final piece connects ChatGPT's output back to Gmail as a properly formatted draft. The "Create Draft" module handles this by combining:

  • The original sender's email address (for "To" field)
  • The original subject line (with "Re: " prefix)
  • ChatGPT's generated response
  • The original message text (for reference)

At 5:18 in the video, you'll see how this creates a complete draft that looks exactly like one you'd compose manually - just without the typing time. The workflow preserves all the original context while adding the AI-generated reply above it.

Important: Always include the original message text in drafts. This maintains email threading and gives you full context when reviewing the AI's response.
