Step-by-Step: Image Generation Workflow

Step 1: API Setup

Create your Higgsfield account and generate an API key from the dashboard (shown at 1:30 in the video). This single key provides access to all 150+ models.

Step 2: Upload Reference

Select a high-quality headshot of your influencer. The system will use this to maintain consistency across all generated content.

Step 3: Configure Settings

Choose your desired aspect ratio (square for Instagram, landscape for blogs) and resolution (up to 4K).

Step 4: Enter Prompt

Describe the scene using natural language. Example: "Professional influencer at beach resort, sunset lighting, vacation vibes, photo-realistic."

Step 5: Generate & Download

Results appear in 8-10 seconds. Download the high-res image or use it as input for video generation.

In summary: Upload reference → Set parameters → Describe scene → Generate → Download. The entire process takes under 30 seconds per image.

Step-by-Step: Video Generation Workflow

Step 1: Start from Image

Use either your original reference photo or a generated image as the first frame.

Step 2: Describe Action

Specify camera movements and character actions. Example: "Slow zoom out to reveal office environment, influencer nods approvingly."

Step 3: Set Duration

Choose between 5-60 seconds. Longer videos consume more credits but maintain quality.

Step 4: Generate Preview

The system first creates a 3-second preview you can approve before full rendering.

Step 5: Render & Download

Final video delivers in 30-40 seconds at 1080p resolution with cinematic motion.

Pro Tip: Combine multiple short clips into longer sequences using simple video editing software for polished final results.
