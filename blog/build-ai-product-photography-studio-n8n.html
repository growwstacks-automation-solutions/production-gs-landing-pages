The Product Photography Problem

Product photography is one of the most expensive and time-consuming aspects of running an eCommerce business. Traditional studio shoots require renting space, hiring photographers, and paying for post-production editing - often costing hundreds per product. Even then, you're limited to the angles and lighting setups you captured during the shoot.

This n8n workflow solves that by creating an AI-powered photography studio that runs automatically whenever you upload new product images. For just $1-2 per run, you get multiple professional angles, lighting variations, and even panning videos - all generated from a single smartphone photo.

90% cost reduction: Traditional product photography averages $150-300 per product. This AI workflow delivers comparable results for under $2.

How the AI Studio Works

The workflow combines n8n's automation power with Foul.ai's image generation API to transform basic product photos into professional studio shots. Here's the step-by-step process:

Step 1: Upload Trigger

When you upload a product photo to a specific Google Drive folder, n8n automatically detects the new file and begins processing.

Step 2: AI Image Generation

The workflow sends your photo to Foul.ai's API with customized prompts to generate 1-4 studio-quality variations at different angles.

Step 3: Video Creation

The same API creates a 5-8 second panning video showing your product from multiple perspectives.

Step 4: Organized Output

All generated assets are automatically saved to a new Google Drive folder with your original image and the AI-enhanced versions.

Key benefit: The entire process runs automatically in the background - no manual editing, no photography skills required.

Setting Up Credentials

You only need two credentials to get this workflow running:

1. Foul.ai API Key

After creating a Foul.ai account, navigate to Settings → API Keys to copy your unique key. In n8n, create a new HTTP Request credential with Header Authentication type, using "Authorization" as the header name and your API key as the value.

2. Google Drive Access

The Google Drive credential uses standard OAuth. Simply sign in with your Google account when prompted during setup. This allows n8n to monitor your intake folder and save generated assets.

These two connections power the entire automated photography studio - no additional services or accounts needed.

Configuring Google Drive

The workflow uses Google Drive as both the trigger point and output destination. Here's how to set it up:

Intake Folder

Create a dedicated folder in Google Drive where you'll upload product photos. Copy the folder ID from the URL (the long string after "folders/") and paste it into the Google Drive trigger node in n8n.

Output Folder

Create another folder for the AI-generated assets. Add this folder ID to the Set node in the workflow. Each time the workflow runs, it will create a new subfolder here containing:

  • The original product photo
  • 1-4 studio-quality variations
  • A professional panning video

This automatic organization means you'll never lose track of which assets belong to which product.

Customizing AI Outputs

The workflow includes several adjustable parameters to fine-tune your AI-generated photos:

Number of Images

Set between 1-4 variations per product. More images give you more options but increase the cost slightly per run.

Aspect Ratio

Choose between square, portrait, or landscape orientations depending on your eCommerce platform's requirements.

Prompt Guidance

The CFG scale (0-1) controls how strictly the AI follows your prompts. Higher values produce more predictable results, while lower values allow more creative interpretations.

Pro tip: Start with 2 images per product and a CFG of 0.7 for balanced results. Adjust based on your specific products' needs.
