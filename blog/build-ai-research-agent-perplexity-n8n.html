5-Minute n8n Workflow Setup

The workflow follows a simple input-output structure with three key components:

Step 1: Create the Agent Framework

In n8n, add an "AI Agent" node as your first step. This creates the container that will hold your research instructions.

Step 2: Configure the Brain

Connect to OpenAI's API (the tutorial shows exactly where to get your API key). This provides the reasoning framework.

Step 3: Add Research Tools

Connect Perplexity AI as your research tool. The video demonstrates how to get your Perplexity API key in under 60 seconds.

Pro tip: Set memory length to 5 interactions so your agent maintains context during complex research chains.

Connecting OpenAI and Perplexity APIs

Both services require API keys, but the setup is simpler than most people expect:

  • OpenAI: Get your key from platform.openai.com → API keys → Create new secret key
  • Perplexity: Sign up at perplexity.ai → Account → API keys → Generate new key

The workflow costs pennies to run - $0.006 per Perplexity query and about $0.03 for OpenAI's processing. At 5:30 in the video, you'll see the exact billing dashboard showing these micro-transactions.
