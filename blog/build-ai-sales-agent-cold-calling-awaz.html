Getting Started With Awaz

Setting up your first AI sales agent takes just 30 minutes. Follow these essential first steps:

1. Account Setup

After signing up, add your payment method (required for purchasing phone numbers). New accounts receive $10 in free call credits.

2. Phone Number Acquisition

Purchase a local number or connect an existing one via Twilio/Telnyx. This becomes your AI's caller ID.

3. Calendar Integration

Connect Google Calendar so the AI can book appointments during calls. Set your availability windows to prevent scheduling conflicts.

4. Initial Knowledge Base

Even a basic Q&A document helps the AI sound informed. Start with 10-15 common prospect questions and approved responses.

Voice Agent Setup Walkthrough

Creating your first AI sales agent involves these key configuration areas:

Voice Selection

Choose from natural-sounding male/female voices across multiple languages. Preview each voice's tone and pacing to match your brand.

Agent Personality

Adjust emotion levels, speech speed, and interruption sensitivity. Enable "back-channeling" for natural conversational flow.

Call Initiation

Set whether the AI or prospect speaks first. Customize the opening greeting with dynamic fields for personalization.

Response Guidelines

Define how the AI should handle objections, stay on script, and steer conversations toward booking appointments.
