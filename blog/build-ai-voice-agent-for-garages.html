Prompt Engineering for Auto Shops

The magic happens in the Vapi prompt that defines the agent's behavior. Key components include:

  1. Role definition: "You are Tom, Gearshift Garages' receptionist" sets the identity
  2. Time awareness: References Vapi's "now" variable to avoid date hallucinations
  3. Service menu: Lists available services (MOT, oil change, etc.) with alternate names
  4. Conversation flow: Structures information collection in logical steps
  5. Calendar rules: Defines how to check availability and create events

At 4:20 in the tutorial, the creator explains why including timezone rules prevents scheduling errors when customers say "tomorrow" or "next Monday." The prompt ensures the agent always calculates dates relative to the current time.

Tool Configuration in Vapi

Setting up the calendar tools requires:

  1. Naming consistency: Tool names in Vapi must match references in the prompt
  2. Calendar ID: Unique identifier for your Google Calendar
  3. Timezone settings: Must match your business location

The tutorial at 7:15 demonstrates creating the "Check Availability" tool first, followed by the "Create Booking" tool. Both pull from the same calendar but serve different functions during customer interactions.

Getting Your Calendar ID

To find your Google Calendar ID:

  1. Go to Google Calendar settings
  2. Select your business calendar
  3. Click "Settings and sharing"
  4. Scroll to "Integrate calendar"
  5. Copy the Calendar ID string

At 8:40, the video shows exactly where this appears in the Google Calendar interface. This ID gets pasted into both Vapi tools to connect your booking system.

Pro tip: Create a dedicated calendar for customer appointments separate from internal meetings.
