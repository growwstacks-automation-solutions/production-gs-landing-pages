Building Your First Voice Agent

Creating an agent begins with SigmaMind's blank canvas or pre-built templates. The towing notification example uses these core components:

Step 1: Trigger Node

Sets the conversation starting point. For outbound calls, configure the AI to initiate the conversation.

Step 2: Send Response Node

Delivers the initial message - in this case, politely asking if it's a good time to talk.

Step 3: Wait Node

Pauses the flow until receiving customer input, creating natural conversation pacing.

Step 4: Analyze Customer Message

Interprets the response to determine next steps (confirm vehicle details, explain fees, etc.)

Pro Tip: Connect nodes by dragging between them to create conversation branches based on different customer responses or intents.

Crafting Effective AI Prompts

The secret to natural-sounding agents lies in well-structured prompts divided into four sections:

1. Identity

"You are Anna from Lead Gen Man Towing Company" - establishes the agent's role

2. Context

Business address, hours, fees - provides necessary operational details

3. Style Guardrails

"Be conversational but empathetic when delivering bad news" - controls tone

4. Goals

Step-by-step conversation flow from greeting to resolution

This structure gives the AI clear parameters while allowing natural variation in each conversation. At 4:32 in the video, you'll see how breaking the prompt into these categories produces more reliable results than a single paragraph.

Adding Call Transfer Functionality

Even well-designed AI agents sometimes need human assistance. SigmaMind makes call transfer seamless:

  1. Add an Analyze Customer Message node after key interaction points
  2. Configure triggers for phrases like "speak to supervisor"
  3. Connect to Voice Action node set to "Transfer Call"
  4. Specify the destination phone number

The transferred call maintains context from the AI conversation, so human agents don't need to start from scratch. At 12:15 in the video, you'll see this in action when the frustrated Tesla owner requests human assistance.
