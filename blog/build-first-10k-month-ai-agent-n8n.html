Creating Your First AI Agent

The simplest AI agent to start with is a daily email summarizer. It solves a universal pain point - information overload - while demonstrating all the core concepts you'll use in more advanced agents.

In N8N, create a new workflow and add a Schedule Trigger node. Set it to run daily at 8 PM (or whenever makes sense for your schedule). This becomes the "when" of your automation.

Pro tip: Start with manual triggers while testing. Click "Execute Workflow" to run your agent on demand instead of waiting for the scheduled time.

Building the Email Summary Workflow

After your trigger, add an AI Agent node. This is where the magic happens. The node connects to OpenAI (or other models) and coordinates all the intelligent behavior.

Double-click the AI Agent node to configure it. Here's the exact prompt we use for email summaries (timestamp 4:30 in the video):

"Act as my personal assistant. Give me a brief summary of what emails I received today. You have access to my latest emails in tools. Send me the summary to my email which you also have access to in tools. Do not include sender/subject header. Only include a summary of the information in the emails. Make the message fun and engaging."

This prompt is specific enough to produce great results while leaving room for the AI to be creative with the formatting.

Connecting OpenAI to Your Agent

For the AI Agent node to work, it needs access to an AI model. N8N provides free OpenAI credits with your trial, or you can connect your own account.

To use your own OpenAI account (recommended for production):

  1. Get your API key from platform.openai.com
  2. In N8N, create new credentials under the AI Agent node
  3. Paste your API key and select GPT-4 (or newer)

The video shows this process at 5:10, including where to find your API key and usage statistics in the OpenAI dashboard.

Adding Memory and Tools

Basic AI agents act on each trigger independently. Adding memory lets them reference previous interactions for more contextual responses.

In your AI Agent node, click the plus button to add Simple Memory. The default context window of 5 past interactions works well for email summaries.

Next, add tools - these are the integrations that let your agent actually do things. For our email summarizer:

  • Gmail (Get): Fetches your latest emails with filters (e.g., only today's messages)
  • Gmail (Send): Emails the summary back to you with AI-generated subject and body

At 7:45 in the tutorial, you'll see how to configure the Gmail tools with the right permissions and filters.

Testing and Refining Your Workflow

With everything connected, test your agent by clicking "Execute Workflow." N8N will show each step executing in real time.

Check your email for the summary. If it's not perfect, refine:

  1. Adjust the prompt to be more specific about format
  2. Limit the number of emails processed (start with last 5)
  3. Add filters to exclude newsletters or promotions

The example in the video (9:20) shows a real summary generated by this exact workflow - including how it handles different email types.
