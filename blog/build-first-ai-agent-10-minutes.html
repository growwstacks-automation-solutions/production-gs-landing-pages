Step 1: Setting Up Your RSS Feed

The foundation of your AI agent is the RSS feed that provides fresh content to analyze. Here's how to configure it in Make.com:

  1. Create a new scenario in Make.com
  2. Search for and add the "RSS" module
  3. Paste your RSS feed URL (from Google Trends or any news site)
  4. Set to return maximum 10 items (prevents information overload)

Pro Tip: Use Google Trends RSS feeds for trending topics in your country or industry. The video shows exactly how to find and copy these URLs at 4:30.

Step 2: Connecting Gemini AI

At 5:50 in the tutorial, you'll connect Google's Gemini to transform raw news into content ideas:

  1. Add the "Gemini" module after your RSS feed
  2. Create a free API key at ai.google (shown at 6:15)
  3. Select the Gemini 1.5 Flash model (fast and affordable)
  4. Paste your custom prompt to generate specific content formats

The key is crafting a prompt that tells Gemini exactly what to create. The example in the video generates:

  • Video titles optimized for click-through
  • 10-second hooks to grab attention
  • Detailed angles explaining the content approach
  • Relevant keywords for SEO

Step 3: Parsing the AI Output

Gemini returns structured JSON data that needs processing:

  1. Add a "Parse JSON" module after Gemini
  2. Define your data structure (title, hook, angle, keywords)
  3. Map Gemini's output to these fields

This step (shown at 8:10) ensures clean data flows to your spreadsheet and email. The JSON structure makes it easy to add new fields later if you want more detailed outputs.

Step 4: Updating Google Sheets

At 9:25, you'll connect to Google Sheets to create a content repository:

  1. Add "Google Sheets" module
  2. Connect your Google account
  3. Create or select a spreadsheet
  4. Map parsed JSON fields to spreadsheet columns

Each time the agent runs, it adds new rows with fresh content ideas. Over time, you build a searchable database of proven topics and angles.

Step 5: Configuring Email Alerts

The final piece (10:40 in video) sends you a daily digest:

  1. Add "Gmail" module
  2. Connect your email account
  3. Set recipient (usually yourself)
  4. Create a template using parsed JSON data

The email includes direct links to the original articles and all generated content ideas. You can review everything in 2 minutes and decide what to create.

Scheduling Your Agent to Run Automatically

A true AI agent works autonomously. At 11:50, you'll learn to schedule it:

  1. Convert your scenario to a Make.com AI Agent
  2. Set a schedule (daily, hourly, etc.)
  3. Add multiple scenarios to the same agent for expanded functionality

Advanced Tip: The same framework can power customer research agents, social media managers, or lead qualification systems. Just change the RSS source and AI prompt.
