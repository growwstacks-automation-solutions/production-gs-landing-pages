Accessing the Agent Builder in Copilot

Finding the agent creation feature isn't intuitive - it's buried under the "Chat" interface in Copilot. The workflow varies slightly depending on whether you access through Teams, the M365 app, or web browser.

For fastest access (shown at 3:48 in the video):

  1. Open Copilot in Teams or at m365copilot.com
  2. Select the "Chat" tab
  3. Click "Create an agent" in the right sidebar
  4. This opens the Agent Studio interface

The first time you access the studio, you'll see an empty dashboard. All your created agents will appear here for easy management.

Integrating Company Knowledge

While Copilot agents can't access all internal systems by default, the video reveals clever workarounds (18:30):

  • File prompts: Configure your agent to request needed files during conversations
  • SharePoint links: Add relevant company documentation URLs in the instructions
  • Pre-attach: Upload frequently referenced documents during agent setup

The resume screening demo shows how to add sample job postings and candidate resumes during creation (19:45), making the agent immediately useful without requiring users to find and upload materials.

Sharing Agents Across Your Organization

Created agents can be shared at three permission levels (21:10 in video):

  1. Private: Only visible to you
  2. Organization: Available to all employees
  3. Specific groups: Targeted sharing with departments or teams

Shared agents appear in recipients' Copilot interface within 24 hours. There's no limit to how many users can access an agent simultaneously, making this ideal for enterprise-wide tools like:

  • HR onboarding assistants
  • IT help desk troubleshooters
  • Sales playbook coaches
