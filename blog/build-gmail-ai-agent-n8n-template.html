What This AI Agent Can Do

Most business owners waste hours every week manually processing emails - reading the same messages multiple times, forgetting to follow up, or letting important communications get buried. This AI agent changes everything by giving you a smart assistant that understands natural language requests and takes action in your Gmail account.

At 2:15 in the demo video, you'll see the agent in action summarizing emails with perfect accuracy. Setup Requirements

You'll need just three things to get started: an n8n instance (cloud or self-hosted), an OpenAI API key, and Google API credentials for Gmail access. The total setup time is under 30 minutes, and ongoing costs are minimal.

The OpenAI API costs pennies per day - about $5 will last months with regular use. n8n offers a free tier that can run this workflow, or you can self-host n8n for free on your own computer if you're comfortable with basic technical setup.

Cost breakdown: OpenAI API (~$5/month for heavy use) + n8n (free tier available) + Google Cloud (free tier covers email API usage). No expensive SaaS subscriptions required.

Building the n8n Workflow

The workflow starts with a simple chat interface node that accepts your natural language requests. This connects to ChatGPT which interprets your intent and determines which Gmail actions to take. The magic happens in the conditional logic that routes different types of requests to the appropriate Gmail API endpoints.

At 8:42 in the video, you'll see how we configure the AI agent's "brain" - the system prompt that instructs ChatGPT how to behave. This includes professional response templates, action decision logic, and error handling instructions.

Step 1: Create a new n8n workflow

Name it "Ultimate Gmail Agent" and add a Webhook trigger node that will receive your chat messages.

Step 2: Add the ChatGPT node

Configure it with your OpenAI API key and select the gpt-4-1106-preview model for best performance.

Step 3: Set up the system prompt

This defines how the AI should behave and what tools it has available.

Pro tip: Use HTML formatting in your system prompt for more structured responses. The AI will automatically format replies with proper headings, lists, and professional styling.

Connecting OpenAI

Getting your OpenAI API key takes just 2 minutes. Log into your OpenAI account, navigate to the API keys section, and create a new secret key. Copy this into n8n's credential manager where you'll create a new OpenAI connection.

The free template uses gpt-4-1106-preview which offers excellent performance at lower cost than standard GPT-4. For testing, you could use gpt-3.5-turbo, but we recommend gpt-4 for production use as it handles complex email tasks much better.

Cost saver: The gpt-4-1106-preview model is 3x cheaper than standard GPT-4 for prompts and 2x cheaper for completions, while offering nearly identical quality for this use case.

Setting Up Gmail API

The Google Cloud setup takes about 10 minutes. Create a new project in Google Cloud Console, enable the Gmail API, and configure OAuth consent screen credentials. You'll need to specify authorized redirect URIs that match your n8n instance's URL.

At 14:30 in the video, we walk through the exact OAuth scope configuration needed for the agent to access your emails. These scopes allow reading messages, sending drafts, modifying labels, and other essential functions while maintaining security.

Step 1: Create Google Cloud project

Go to cloud.google.com and create a new project specifically for this integration.

Step 2: Enable Gmail API

Search for "Gmail API" in the library and enable it for your project.

Step 3: Configure OAuth consent

Add your email as a test user and specify the required scopes.

Security note: The OAuth scopes are carefully selected to only grant the minimum necessary permissions. The agent can't delete emails or access your entire mailbox indiscriminately.
