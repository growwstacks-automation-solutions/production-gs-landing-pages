Building the Telegram Command Interface

The Telegram bot serves as the store owner's control panel. Instead of logging into multiple dashboards, they send simple messages like "Fulfill today's orders" or "Generate social post about our new sneakers".

In n8n, we create this with the Telegram trigger node set to "Message" events. The key configuration points:

  • Set operation mode to "Webhook" for real-time responsiveness
  • Configure allowed user IDs for security
  • Add phrase matching to detect command intent ("fulfill", "audit", "post")

Pro Tip: Add a 2-second delay before responses to make the AI feel more human. At 4:30 in the video, notice how the slight pause makes the interaction feel conversational rather than robotic.

Creating the AI Agent Brain

The AI agent transforms simple commands into intelligent actions. When a store owner asks "How many blue shirts do we have left?", the agent:

  1. Understands the request requires inventory data
  2. Queries Shopify's API for current stock levels
  3. Formats the response in natural language

We implement this in n8n using the OpenAI node with these settings:

  • Model: GPT-4 (or Claude 3 for larger context windows)
  • Temperature: 0.7 for balanced creativity/accuracy
  • System prompt: "You are a Shopify operations assistant..."
  • Session ID: Unique per user for conversation history
