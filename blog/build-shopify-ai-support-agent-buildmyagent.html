Shopify API Integration

Connecting your Shopify store involves creating a dedicated app in your Shopify admin (Settings → Apps and sales channels → Develop apps). The critical steps shown at 4:30 include:

  1. Creating a new app called "BuildMyAgent"
  2. Configuring API scopes (permissions)
  3. Installing the app on your store
  4. Copying the admin API access token

Security note: The API token is only visible once during setup. Store it securely as you'll need to paste it into BuildMyAgent to complete the connection.

Configuring API Permissions

At 6:45, the video demonstrates the importance of granular permission settings. While you can enable all API access initially for testing, production deployments should follow the principle of least privilege:

  • Essential: read_orders, read_customers, read_products
  • Optional: read_price_rules, read_locations
  • Risky: write permissions (create/update/delete)

The agent shown in the demo intentionally disables product creation and order modification capabilities to prevent accidental changes to the store.
