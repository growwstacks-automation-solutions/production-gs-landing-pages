Gemini Live API Integration Details

The magic happens in the invite agent route (app/api/invite/route.js) where we implement the withMLLimb method. At 5:30 in the tutorial, Mason explains each configuration parameter:

 new GeminiLive({   model: "gemini-3.1-flash-preview",   apiKey: process.env.GEMINI_API_KEY,   url: "wss://google-websocket-server",   inputModality: "voice",   outputModality: "voice",   greeting: greetingPrompt,   prompt: systemPrompt })

Key advantages of this approach:

  • Eliminates 3 separate API calls (STT → LLM → TTS)
  • Supports real-time language switching (demonstrated at 7:20)
  • Enables multimodal responses combining voice with other outputs
