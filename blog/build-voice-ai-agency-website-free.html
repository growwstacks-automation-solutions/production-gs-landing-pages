Make.com Webhook Setup

The magic happens when website form submissions trigger instant demo calls. This requires connecting your site to Make.com (formerly Integromat) - the automation platform that orchestrates the entire process.

Setting up the webhook takes just three steps:

  1. Create a new scenario in Make.com with a "Custom Webhook" module
  2. Copy the unique webhook URL generated
  3. Paste it into your AI-generated website code where indicated

Pro tip: Always test your webhook by submitting the form and verifying the data appears in Make.com before connecting expensive services like voice calling.

Domain Configuration

Namecheap offers domains starting at $0.99/year - the only required expense in this setup. After purchase:

  1. Go to Advanced DNS settings in Namecheap
  2. Add a CNAME record pointing to Vercel
  3. Update the A record with Vercel's IP address
  4. Add your domain in Vercel's project settings

DNS changes typically propagate within minutes, giving you a professional domain like youragency.ai instead of a generic Vercel subdomain.
