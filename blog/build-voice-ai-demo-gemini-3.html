Step 1: Researching Your Target Client

The most effective demos are hyper-personalized to the prospect's business. Start by identifying a real company in your target industry - we use a Boston plumbing company in our example.

Look for businesses with:

  • An outdated website that could benefit from modern AI features
  • Clear service offerings that involve phone interactions
  • Brand colors and styling that can be replicated

Capture their domain name and take a screenshot of their site. This visual reference ensures your demo matches their existing branding for maximum impact.

Step 2: Extracting Brand Information with AI

With your target client identified, use ChatGPT or Gemini to extract key brand elements:

  1. Paste the company URL into ChatGPT with instructions to extract brand name, services, contact info, and color scheme
  2. For accuracy, supplement with a screenshot of their actual website
  3. Request the output in markdown format for easy copying

Pro Tip: At 6:15 in the video, you'll see how correcting the AI's color interpretation makes the demo match the client's actual branding perfectly.

This process takes about 2 minutes but results in all the information needed to recreate their branding in your demo.

Step 3: Creating the Demo in Gemini 3

With your brand information ready, head to Gemini 3 and create a prompt that includes:

  • Instruction to create a one-page app for the specific business type
  • Request for two voice agents (e.g., front desk and emergency dispatcher)
  • The brand information you extracted
  • Specification to place the voice agents "above the fold"

Gemini 3 will generate the complete website code automatically. The process typically takes 2-5 minutes, during which you can watch the code being written in real-time.

At 8:47 in the video, you'll see the exact moment the working demo becomes available with fully functional voice interaction.

Customizing the Voice Agents

The generated demo includes two voice agents by default, but you can easily customize them:

  1. No-code method: Simply tell Gemini 3 in natural language what changes you want (e.g., "Make Sarah female and Mike male")
  2. Code method: Edit the JSON configuration directly to change names, voices, and behaviors

Key customization options include:

  • Agent names and genders
  • Voice tones and styles
  • Response patterns and personality
  • Color schemes matching the brand

At 12:30 in the video, we demonstrate both customization methods side-by-side.

Step 4: Deployment Options

Gemini 3 offers multiple deployment paths for your demo:

Google Cloud (Recommended): One-click deployment with automatic API key security through Google's proxy system. No risk of exposing your Gemini API key.

Alternative options include:

  • Downloading the code for self-hosting (requires additional security measures)
  • Using GitHub Pages for simple static hosting
  • Exporting to various cloud platforms

The video at 14:20 explains the security implications of each approach in detail.

Adding a Custom Domain

For maximum impact, map your demo to a custom subdomain that matches your client's brand:

  1. In Google Cloud, navigate to Domain Mappings
  2. Add a new mapping with your desired subdomain (e.g., demo.clientdomain.com)
  3. Add the provided CNAME record to your DNS provider
  4. Wait for SSL certificate provisioning (typically 5-60 minutes)
