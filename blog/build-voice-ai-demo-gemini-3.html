Step 1: Researching Your Target Client

The process begins by identifying a real business that could benefit from Voice AI. In the tutorial, we search for "plumbers in Boston" to find a target client (shown at 3:42). The ideal candidate has:

  • A website with clear branding (colors, logo, services)
  • Phone-based customer service (front desk/dispatcher)
  • An online presence but potentially outdated technology

This research takes about 60 seconds. The goal isn't perfection — it's finding a representative example where Voice AI could clearly add value. For agencies, this means selecting clients in industries where voice interactions are common (healthcare, legal, home services, etc.).

Step 2: Extracting Brand Information with AI

At 4:30 in the video, we use ChatGPT to automatically extract all necessary brand information from the target website. The prompt:

"Take the following website URL and extract the brand name, services they offer, contact information, color profile, and layout style. Return it in concise markdown format."

This generates a complete brand profile including:

  • Company name and services
  • Phone numbers and contact details
  • Primary and accent colors
  • Website structure and hero section text

If colors aren't accurate (as happened at 5:18), simply upload a screenshot and ask for corrections. The entire extraction process takes about 90 seconds.

Step 3: Generating the Voice AI Website

With the brand profile ready, we move to Gemini 3 (7:02 timestamp). The generation prompt:

"Create a one-pager app for a plumber based on the given brand information. Include two voice AI agents — one as a front desk and one as an emergency dispatcher — available in the above-the-fold section."

Gemini 3 then:

  1. Designs a matching website layout
  2. Creates two functional voice agents
  3. Generates all necessary code
  4. Provides a live preview

The generation typically takes 2-8 minutes. You can watch the code being written in real-time (7:45) or simply wait for the finished product.

Step 4: Customizing Voice Agents

At 9:50, we customize the voice agents' names and roles. There are two approaches:

No-code method: Simply tell Gemini 3 in natural language: "Make Sarah a female voice and Mike a male voice for the dispatcher role."

Code method (for advanced users): Edit the JSON configuration to modify:

  • Agent names and voice types
  • Role descriptions and responses
  • Color schemes and positioning

This level of customization ensures the demo perfectly matches your client's specific use case and brand voice.

Step 5: Deploying Your Demo

Deployment to Google Cloud (shown at 12:30) is a one-click process:

  1. Click the rocket icon in Gemini 3
  2. Select or create a Google Cloud project
  3. Click "Deploy App"

Within 60 seconds, you'll have:

  • A live URL for your Voice AI demo
  • Automatic HTTPS security
  • Google-managed infrastructure
  • Built-in API key protection

Costs are minimal — typically under $10/month for multiple demos. At 13:15, we test the live demo with actual voice interactions.

Step 6: Adding Custom Branding

For maximum impact, deploy the demo on a custom subdomain (15:40):

  1. In Google Cloud, go to Domain Mappings
  2. Add a CNAME record like "plumber.yourdomain.com"
  3. Point it to Google's hosting address
  4. Wait for DNS propagation (up to 48 hours)

This creates a white-labeled experience that looks completely professional. As shown at 17:20, you can now send clients to a URL like "bostonplumber.youragency.com" where they'll interact with their branded Voice AI demo.

Security Considerations

At 10:30 in the video, we address a common concern: API key exposure. Key points:

Google Cloud hosting: Your Gemini API key is automatically protected by Google's proxy system. No additional security measures are needed.

External hosting: If downloading the code to host elsewhere, you would need to:

  • Set up a backend service to protect the API key
  • Implement proper authentication
  • Handle streaming between frontend and backend

For most agencies, using Google Cloud is recommended as it handles all security automatically with minimal cost.
