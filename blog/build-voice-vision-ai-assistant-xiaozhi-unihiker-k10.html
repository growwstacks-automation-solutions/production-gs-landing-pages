What Makes Xiaozhi AI Special

Most AI assistants today are either cloud-based services with privacy concerns or simple voice-only interfaces. Xiaozhi breaks this mold by offering a fully open-source ESP32-based framework that combines real-time voice conversations with computer vision capabilities - all while keeping processing local to your hardware.

The magic happens through Xiaozhi's MCP protocol, which connects powerful language models (like DeepSeek or Quinn) to both physical devices and cloud services. This creates a unified assistant that can control your smart home lights while simultaneously solving math equations from photos and answering complex questions.

Key capability: Xiaozhi supports over 70 ESP32 development boards and offers multilingual out-of-the-box support for English, Chinese, and Japanese. The system uses efficient Opus audio encoding for crisp voice responses and can recognize different speakers for personalized interactions.

Hardware You'll Need

For this tutorial, we're using the UNIHIKER K10 development board which features an ESP32-S3 microcontroller, built-in microphone, and 2.4-inch touchscreen. However, Xiaozhi supports many ESP32 variants including:

  • M5Stack Core S3
  • ESP32-C3/C5/C6
  • Newer ESP32-P4 boards
  • Any ESP32 with camera/mic capabilities

The complete setup requires:

  1. UNIHIKER K10 board (or compatible ESP32 device)
  2. USB-C cable for power/flashing
  3. Windows PC for flashing tool (Mac/Linux users can use esptool.py)
  4. Smartphone for Wi-Fi configuration

Downloading the Firmware

Xiaozhi provides pre-compiled firmware binaries that make deployment simple for beginners. Here's how to get the right version:

  1. Visit the DF Robot community forum (link in resources)
  2. Navigate to the Xiaozhi AI installation guide
  3. Download the latest firmware (v1.8.5 at time of recording)
  4. Extract the .bin file from the ZIP archive

Pro tip: The firmware package includes all necessary AI models and voice data, eliminating the need for separate downloads. The entire system fits in the UNIHIKER's 16MB flash memory.

Flashing to UNIHIKER K10

The flashing process requires putting the UNIHIKER into bootloader mode and using Espressif's Flash Download Tool:

  1. Press and hold the BOOT button on the UNIHIKER's back
  2. While holding BOOT, connect the USB cable to your computer
  3. Verify bootloader mode (screen stays black)
  4. Open Flash Download Tool as Administrator
  5. Select ESP32-S3 as chip type
  6. Browse to select the Xiaozhi .bin file
  7. Click Erase → Start to flash the firmware

At 2:15 in the video, you can see the exact flashing process with progress indicators. The entire process takes about 90 seconds.

Wi-Fi Configuration

After flashing, the UNIHIKER will boot into Wi-Fi configuration mode:

  1. Connect your phone to the Xiaozhi hotspot (SSID shown on screen)
  2. Visit the configuration URL (192.168.4.1)
  3. Select your 2.4GHz Wi-Fi network (5GHz not supported)
  4. Enter credentials and connect
  5. Note the 6-digit verification code displayed

The board will reboot and announce its connection status. You're now ready to configure the AI personality.
