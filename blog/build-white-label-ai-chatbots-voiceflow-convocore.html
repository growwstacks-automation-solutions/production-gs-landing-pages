Step 1: Setting Up Your Voiceflow Account

Creating your foundation in Voiceflow takes less than 5 minutes. Navigate to voiceflow.com and sign up for a free account - no credit card required. The platform immediately guides you through creating your first "agent" (their term for chatbot instances).

At 2:30 in the tutorial, we demonstrate how to:

  1. Name your agent according to its purpose (e.g. "Sales Qualifier" or "Support Assistant")
  2. Set the default language (English recommended initially)
  3. Choose a basic template to start from (we recommend "Blank Project")

Pro tip: Even on the free plan, Voiceflow allows unlimited agents - create separate ones for different departments or languages to keep your flows organized.

Step 2: ConvoCore Integration for Branding Magic

While Voiceflow handles the conversation design, ConvoCore (convocore.ai) brings the visual customization and deployment capabilities. Sign up for their free tier, then navigate to the API Keys section as shown at 5:45 in the video.

The critical connection steps:

  1. Copy your Voiceflow project ID from the settings menu
  2. Paste it into ConvoCore's integration panel
  3. Copy ConvoCore's primary API key back into Voiceflow

This two-way sync ensures all your Voiceflow designs automatically appear in ConvoCore's white-labeled interface. Changes made in either platform update in real-time.

Complete Visual Customization Guide

ConvoCore's branding options transform generic chatbots into seamless brand extensions. At 7:20 in the tutorial, we walk through uploading:

  • Company logo (recommended 200x200px transparent PNG)
  • Header image for the chat window
  • Optional background image (great for premium brands)

The color palette editor lets you match your exact brand colors using hex codes. We recommend setting:

  1. Primary brand color for buttons and accents
  2. Secondary color for backgrounds
  3. Contrasting text colors that remain readable

Font selection: Choose from hundreds of Google Fonts - but stick to your website's existing font family for consistency. The tutorial shows how to preview fonts before committing.

Implementing Multilingual Support

ConvoCore's automatic language detection handles most European languages out of the box. At 12:30 in the video, we demonstrate how to:

  1. Enable "Automatic" language detection in settings
  2. Add supported languages (start with 2-3 your customers use)
  3. Test detection by simulating different user locations

For right-to-left languages like Arabic or complex character sets, manual language setting works more reliably. The system remembers each user's preference across sessions.

Warning: Automatic detection can sometimes misidentify languages early in conversations. We recommend keeping English as the fallback option.

Configuring Human Handoff Features

The true power emerges when AI meets human assistance. At 18:45 in the tutorial, we configure:

  1. Handoff triggers (when the chatbot reaches its limits)
  2. Notification sounds for human agents
  3. Automatic conversation history transfer

This creates a seamless transition where customers never need to repeat themselves. The system even suggests potential responses based on the chat history, reducing agent training time.

Implementation tip: Set business hours for live handoff and customize the offline message. The chatbot handles after-hours queries until morning.
