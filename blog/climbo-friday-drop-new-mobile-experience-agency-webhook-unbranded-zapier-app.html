Climbo Friday Drop: New Mobile Experience, Agency Webhook & Unbranded Zapier App

Managing client reviews just got easier with Climbo's latest updates. Discover how the redesigned mobile interface, client-facing Zapier integration, and automated agency webhooks can streamline your reputation management workflow and save hours of manual work each week.

Redesigned Mobile Experience

Business owners struggling to monitor their online reputation on the go now have a powerful new tool. Climbo's completely redesigned mobile interface (shown at 0:45 in the video) features a bottom navigation bar with three key sections that make managing reviews effortless from any smartphone.

The home section now displays critical metrics at a glance - average rating, total reviews, private feedback counts, and platform-specific goals. For example, clients can see they need 25 five-star reviews to improve their Facebook rating from 4.5 to 4.6. The right side shows AI-generated sentiment analysis that will soon include competitor benchmarking.

Key improvement: The review feed now resembles a messaging app, making it intuitive to view, reply to, and manage reviews. Clients can filter by unanswered reviews or use advanced filters to quickly find specific feedback.

Unbranded Zapier App for Clients

Many businesses want to automate review requests but lack technical skills to work with APIs. Climbo's new unbranded Zapier app (called "AI Connector") solves this by letting clients create no-code automations through a familiar interface.

Currently offering three actions - add contacts, send campaigns, and send invites - the integration enables powerful workflows. At 4:20 in the video, you'll see how clients can automatically send SMS review requests whenever new rows are added to a Google Sheet, complete with dropdown menus for selecting locations and channels.

Coming soon: An embedded Zapier experience will let clients build automations directly within Climbo's interface, eliminating the need to visit Zapier.com. Agencies will control access to this feature through custom plans.

New Agency Webhook Feature

Agencies wasting time manually onboarding new clients now have an automation solution. The newly added webhook category in the client API documentation (shown at 7:15) sends real-time notifications when new clients sign up.

Currently limited to the "client created" event, the webhook delivers all relevant client data (email, plan ID, business name) to a specified URL. This enables automations like instantly sending welcome emails when a free trial begins or adding new clients to your CRM without manual data entry.

Client API Documentation

Technical clients now have access to comprehensive, unbranded API documentation through their account settings. Each client receives a unique API key (not the agency's key) to authenticate requests for managing their review campaigns.

The documentation covers all available endpoints including get business, add location, create campaign, and send invites. At 3:50 in the video, you can see how clients would insert their API key as a header credential to start automating their review management processes.

Automatic Language Detection

Agencies embedding Climbo demos on their websites can now showcase the platform in each visitor's native language. The embedded client mode automatically detects browser language settings (demonstrated at 9:30 with an Italian example).

This feature is particularly valuable for agencies serving international clients or multiple language markets. Visitors from France see the demo in French, German visitors see it in German, and so on - all without any additional configuration from the agency.

Automated Monthly Reports

Clients often forget to check their monthly sentiment analysis, missing valuable insights. The new optional email notification (mentioned at 2:40) automatically delivers this analysis on the first of each month when the data refreshes.

The email includes the regenerated sentiment summary, lists of what's working well, areas needing improvement, and action plans - giving clients regular, digestible insights without requiring them to log in to the platform.

Coming Soon to Climbo

The Climbo team previewed several exciting features planned for release before Christmas. The agency Zapier app (shown at 6:20) will include triggers like new client notifications and actions to manage client accounts programmatically.

Other anticipated updates include the embedded Zapier experience for clients, competitor analysis in the mobile home section, and expanded webhook events beyond just new client notifications. These additions will further automate reputation management workflows for both agencies and their clients.

Key Takeaways

Climbo's latest updates significantly enhance both the client experience and agency automation capabilities. The mobile redesign makes reputation management truly on-the-go, while the Zapier integration and webhooks open powerful new automation possibilities.

In summary: Clients get easier mobile management and no-code automation options, while agencies gain tools to automate onboarding and scale their services more efficiently through API and Zapier integrations.
