Step 1: Twilio Account Setup & Number Purchase

Twilio provides the telephony infrastructure that routes calls to your Retell AI agent. The setup begins with creating accounts in both platforms:

Creating Your Twilio Account

1. Visit Twilio.com and sign up using your business email

2. Verify your email and phone number (Twilio will send confirmation codes)

3. Important: When prompted, select "Programmable Voice" as your primary use case

Purchasing Your Phone Number

1. In Twilio Console, navigate to Phone Numbers → Manage → Active Numbers

2. Click "Buy a Number" and search for your preferred area code

3. Select a number from the available options (costs $1-$3/month)

4. Complete purchase using your $15 free trial credit

Pro Tip: Choose a local area code number even if you serve nationwide customers. Caller ID recognition increases answer rates by 22% according to Twilio data.

Step 2: SIP Trunk Configuration in Twilio

The SIP trunk acts as the bridge between your Twilio number and Retell AI. This is where most technical setup occurs, but we'll walk through each setting:

Creating the SIP Trunk

1. In Twilio Console, use the search bar to find "Elastic SIP Trunks"

2. Click "Create New SIP Trunk" and name it (e.g., "RetellAI_Connection")

Configuring General Settings

1. Under the General tab, enable these three critical options:

  • ✓ Enable Codec Transfer SIP
  • ✓ Enable PSTN Transfer
  • ✓ Enable Symmetric RTP

2. Click Save to apply these changes

Setting Up Termination

1. Navigate to the Termination tab

2. Create a termination URL (name it anything, like "retell_term1")

3. Leave all other termination settings at default values

Configuration Complete: After these steps, your Twilio environment is ready to connect to Retell AI. The next steps happen within Retell's interface.

Step 3: Connecting to Retell AI

With Twilio configured, we now establish the connection in Retell AI:

Linking Your Phone Number

1. Log in to your Retell AI account

2. Navigate to Phone Numbers → Add Number

3. Select "Connect via SIP Trunking"

Entering Twilio Credentials

1. Paste your Twilio phone number (format: +1XXXXXXXXXX, no spaces)

2. For Termination URL, copy from Twilio's Termination tab

3. Enter the username/password you created in Twilio's Authentication tab

4. Save the configuration

Assigning to an Agent

1. Create or select an existing voice agent in Retell AI

2. Under "Inbound Call Agent," select your agent

3. Save changes - your number is now live!

Testing Tip: Call your new number immediately to verify connectivity. The AI should answer within 2 rings. If you hear Twilio's default message instead, double-check your SIP trunk settings.

Step 4: Testing Your Call Flow

Proper testing ensures your AI receptionist provides a seamless caller experience:

Basic Functionality Test

1. Call your number from a different phone

2. Verify the AI answers within 2-3 rings

3. Test basic queries your customers might ask

Monitoring Call Logs

1. In Retell AI, check the Calls dashboard

2. Review conversation transcripts

3. Identify any unanswered questions to improve your agent

Twilio Debugging

1. In Twilio Console, navigate to Monitor → Logs → Calls

2. Check for error codes if calls fail

3. Common issues include authentication errors or incorrect termination URLs
