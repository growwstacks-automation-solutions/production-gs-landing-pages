Getting Started with Convayla

After creating your free Convayla account, you'll land on the dashboard that provides an overview of your AI voice agents. The interface is divided into clear sections showing your current balance, active plan, and listed agents with their key metrics. Navigation is handled through the left sidebar where you can access all major features.

The dashboard is designed to give you immediate visibility into your agents' performance while making it easy to jump into configuration. At 1:15 in the video tutorial, you can see how the navigation works with agents, knowledge base, widgets, and analytics all accessible from the main menu.

Pro Tip: Bookmark the Convayla dashboard URL after logging in for quick access to your voice agents. The platform doesn't currently support multiple user accounts, so team members will need to share login credentials.

Creating and Configuring Agents

Creating a new voice agent in Convayla involves several key configuration steps that determine how it will interact with users. From the Agents section, you can edit existing agents or create new ones by clicking the appropriate buttons. Each agent requires a name, description, and voice selection - with options to preview voices before committing.

The system prompt is perhaps the most important configuration as it defines the agent's personality and response behavior. Convayla provides an expandable input field for crafting detailed prompts that guide how your agent should respond in various situations. Below this, you'll set the greeting message - the first thing users hear when starting a conversation.

Key Configuration Elements: A well-configured agent combines a clear system prompt with appropriate tools (like knowledge base access) and a friendly greeting. Test different voice options to find one that matches your brand personality before deploying to customers.

Setting Up the Knowledge Base

Convayla's knowledge base feature allows your voice agents to access and reference your business information during conversations. The knowledge base section shows an overview of uploaded documents and their processed chunks - the smaller pieces of content the AI uses for reference. You can upload files in multiple formats or use the built-in web crawler.

The enhanced crawler offers three methods for importing web content: individual URLs for specific pages, full domain crawls for comprehensive coverage, or sitemap imports that use a website's own structure. At 3:45 in the video, you can see how to schedule recurring crawls to keep your knowledge base updated automatically - particularly useful for businesses with frequently changing information.

Widget Configuration and Testing

Convayla provides multiple options for embedding your voice agent on websites through customizable widgets. The basic widget generator creates a simple code snippet you can copy and paste, while the advanced options offer extensive customization. Placement, appearance, and security settings all help integrate the agent seamlessly with your site.

Widget themes allow you to match the agent's visual presentation to your brand with preset options and fine-tuned color controls. The security tab is particularly important for specifying which domains can embed your widget, preventing unauthorized use. At 5:20 in the tutorial, you can see the test agent feature that lets you preview changes before deploying to your live site.

Testing Tip: Always test your widget on multiple device types after configuration. Mobile responsiveness is automatic, but checking different screen sizes ensures optimal user experience.

Understanding Analytics and Metrics

Convayla's analytics dashboard provides valuable insights into how your voice agents are performing and being used. Key metrics include current balance, session counts, time used, and calculated costs - all viewable for custom date ranges. The platform breaks down performance by individual agent, showing which are most active and effective.

All conversations are stored for 30 days, allowing you to review transcripts and analyze interactions. This historical data helps identify common questions, areas where knowledge base improvements are needed, and opportunities to refine system prompts. At 7:10 in the video, you can see how to filter analytics by date range and export data for further analysis.

Billing and Account Management

Convayla operates on a pay-as-you-use credit system with packages starting at 5 euros. The billing section shows your current balance and payment history, with options to top up manually or set up auto-recharge when your balance falls below a threshold. Spending limits can be configured for daily, weekly, or monthly periods to control costs.

Payment processing happens through Stripe, with purchased credits available immediately after transaction completion. The invoices section maintains records of all transactions, useful for accounting purposes. At 8:30 in the tutorial, you can see how to set spending limits that prevent unexpected charges while ensuring your agents remain available to customers.
