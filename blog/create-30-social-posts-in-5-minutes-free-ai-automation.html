The Content Repurposing Problem

Content creators know they should repurpose videos across multiple platforms, but the manual work involved makes it impractical. Between formatting for different platforms, rewriting for different audiences, and maintaining brand consistency, what should take minutes ends up consuming hours.

This workflow solves three critical pain points: First, it eliminates the blank page problem by automatically generating draft content. Second, it ensures platform-specific formatting. Third, it maintains consistent messaging across all outputs while adapting tone for each platform.

Content repurposing accounts for 30% of marketers' time according to recent industry surveys. This automation can reduce that to less than 5% while increasing output volume.

Workflow Overview

The automation follows a simple but powerful sequence: YouTube URL → Video Transcription → AI Content Generation → Formatted Outputs. Here's how the pieces connect:

  1. You paste a YouTube URL into a designated Google Sheet
  2. Make.com reads the URL and sends it to RapidAPI
  3. RapidAPI extracts the video transcript
  4. Gemini AI processes the transcript into platform-specific content
  5. Outputs are saved to Google Docs in ready-to-use formats

The entire process runs automatically in about 5 minutes, requiring no manual intervention after pasting the URL. At 2:15 in the video, you can see the exact moment when all three content types generate simultaneously.

Step 1: Setting Up Google Sheets

The workflow begins with a simple Google Sheet that serves as your input interface. Create a sheet with these columns: Video URL, Status, and Timestamp. The Status column will automatically update when processing begins and completes.

In Make.com, add the Google Sheets module and connect it to your account. Search for your file by name in the spreadsheet ID field. Select the specific sheet name and set "table contains headers" to yes. Limit the rows processed to 1 to ensure the system only handles one URL at a time.

Pro Tip: Name your Google Sheet something descriptive like "Content Repurposer" to easily identify it in Make.com's module selection.

Step 2: Connecting RapidAPI

The secret sauce of this workflow is the RapidAPI connection that handles video transcription. After testing multiple APIs, we found one that offers reliable free transcription with accurate results.

In Make.com, add an HTTP request module configured with:

  • URL copied from RapidAPI settings
  • GET method
  • X-RapidAPI-Key header with your API key
  • X-RapidAPI-Host header from your API settings
  • Query parameter named "URL" that pulls from your Google Sheet

Set "parse response" to yes and save. This ensures the transcript comes through in a format Gemini can process.

Step 3: Configuring Gemini AI

Gemini AI does the heavy lifting of transforming raw transcripts into polished content. The key is crafting precise system instructions that govern the output format.

In Make.com's Gemini module:

  1. Connect your Google AI API key
  2. Select Gemini 1.5 Flash as your model
  3. Add system instructions specifying output formats
  4. Configure messages with role="user" and your content prompt

The prompt should specify:

  • LinkedIn post structure (intro, key points, conclusion)
  • Twitter thread format (concise points, thread sequence)
  • Newsletter style (conversational, value-focused)
  • Any brand voice requirements

At 4:30 in the video, you can see the exact prompt structure that produces perfectly formatted outputs for all three content types.

Step 4: Output to Google Docs

The final step saves all generated content to Google Docs for easy access and editing. Configure the Google Docs module in Make.com to:

  • Create a new document for each run
  • Include the video URL in the document name for tracking
  • Pull content from Gemini's output
  • Save to a designated folder in your Drive

Each document will contain three clearly labeled sections: LinkedIn Post, Twitter Thread, and Newsletter. The formatting preserves line breaks and structure specified in your Gemini prompt.

This creates a centralized repository of all repurposed content that's easy to share with team members or connect to other automation workflows.

Testing the Workflow

To test the workflow, simply paste a YouTube URL into your Google Sheet and run the scenario. Within minutes, you should see:

  1. The Status column update to "Processing"
  2. A new Google Doc appear in your designated folder
  3. Three perfectly formatted content pieces ready for review

The first test revealed an important lesson: Some API services struggle with longer videos. For optimal results, start with videos under 30 minutes until you confirm your API's capabilities.

Real-world result: One creator reported generating 30 social posts from a single 20-minute video, saving 4 hours of manual work per week.
