The Content Repurposing Problem

Content creators and marketers waste countless hours manually repurposing video content across platforms. A single YouTube video could become 30+ pieces of social content - if only someone had time to write all those posts.

The breakthrough came when we realized AI could analyze video transcripts and automatically reformat the content for different platforms while maintaining brand voice. This eliminates the blank page paralysis that stalls most content calendars.

Content repurposing consumes 37% of marketers' time according to recent surveys. This workflow reduces that to under 5 minutes per video while actually increasing output quality through AI optimization.

Workflow Overview

This automation transforms YouTube videos into multiple content formats through a simple 4-step process:

  1. Add video URL to Google Sheets
  2. Scrape transcript via RapidAPI
  3. Reformat content with Gemini AI
  4. Output to Google Docs

The magic happens in the AI prompt engineering. By giving Gemini specific formatting instructions for each content type, we ensure professional-quality output that maintains brand voice across platforms.

Setting Up Google Sheets

The workflow starts with a simple Google Sheet containing video URLs. At 1:22 in the tutorial, you can see the exact sheet configuration needed:

  • Name your sheet "Contents Repurposer"
  • Include headers: Video URL, Status
  • Connect to Make.com using the spreadsheet ID

This becomes your content intake system - just paste YouTube links and the automation handles the rest. The "Status" column automatically updates when processing completes.

Connecting to YouTube API

At 2:45 in the video, we configure the RapidAPI connection that extracts video transcripts. Key requirements:

Use the free tier which allows 50 requests per day - perfect for small businesses. The API settings must include:

  • X-RapidAPI-Key from your account
  • Correct host URL from API documentation
  • GET method with video URL parameter

The HTTP request module in Make.com passes the YouTube URL from your sheet and receives the full transcript text as response.

Configuring Gemini AI

The real magic happens when we feed the transcript to Gemini 2.5 Flash with specific formatting instructions (shown at 4:10 in the video):

System: Convert this video transcript into: 1. LinkedIn post (H1 heading format)    - Engaging opening    - 3 key takeaways    - Call-to-action 2. Twitter thread (3-5 tweets)    - Numbered points    - Hashtags 3. Newsletter section    - Friendly tone    - Bullet points Maintain professional tone throughout.

This prompt structure ensures consistent, platform-appropriate output every time. The AI handles all the writing while you focus on strategy.

Customizing Output Formats

While the tutorial shows LinkedIn, Twitter and newsletter outputs, you can modify the workflow for:

  • Instagram captions
  • Facebook posts
  • TikTok descriptions
  • Email sequences

Simply adjust the Gemini prompt to include your desired formats. The AI will adapt its writing style accordingly while maintaining consistency across all outputs.

Pro tip: Save different prompt versions as templates in Make.com to quickly switch between content strategies.
