Traditional video production costs between $500-$2000 per 30-second spot, making scalable campaigns prohibitively expensive for most businesses.

The breakthrough comes from combining three technologies: Make.com for workflow automation, large language models for script generation, and Sora 2 for AI video creation. Together, they eliminate 80% of manual work while maintaining brand alignment.

Key stat: Businesses using automated video production see 3x more ad variations tested and 40% higher engagement from personalized messaging.

Complete Workflow Overview

This Make.com automation transforms static website content into dynamic video ads through four seamless steps:

  1. Content scraping: Extracts service descriptions, value props, and customer pain points from your website
  2. AI script generation: Creates ad scripts complete with hook, visuals, voiceover, and CTA
  3. Sora 2 rendering: Converts scripts into 10-30 second video files
  4. Cloud delivery: Automatically organizes finished videos in Google Drive with metadata tagging

The entire process runs unattended once triggered, with optional human review points after script generation and before final rendering.

Step 1: Scraping Source Content

The workflow begins by extracting raw material from your website or knowledge base. Make.com's HTTP and parser modules extract:

  • Service/product descriptions
  • Customer pain points
  • Value propositions
  • Testimonials or reviews

At 1:15 in the tutorial video, you'll see how we configured the scraper to target specific page sections while ignoring navigation and footer elements. This ensures clean input for the AI models.

Step 2: AI Script Generation

The scraped content feeds into a large language model (we use GPT-4 but any model works) with these creative directions:

  • "You are an expert advertising creative director"
  • "Create scripts for 10-second television commercials"
  • "Include hook, visual direction, voiceover, call-to-action"

The AI outputs complete scripts like this example from the demo:

[OPEN: Busy professional drowning in paperwork] "OVERWHELMED by tasks? It's time to supercharge your workflow. Introducing Automation Superstars - elite Make.com and Zapier partners. Automate processes, slash costs up to 60%, focus on what really counts. Visit automationsuperstars.com today."

Step 3: Sora 2 Video Creation

Generated scripts automatically route to OpenAI's Sora 2 API which:

  1. Interprets the visual direction
  2. Generates appropriate scenes
  3. Adds smooth transitions
  4. Renders a complete video file

The 2:30 timestamp in the tutorial shows how we configured Make.com to handle Sora 2's asynchronous API - polling for completion status and automatically retrieving finished videos.

Step 4: Automated Delivery

Finished videos flow through these automated steps:

  1. Quality check (duration, aspect ratio)
  2. Metadata tagging (source URL, generation date)
  3. Google Drive upload to client-specific folder
  4. Notification sent to marketing team

The system maintains a complete audit trail of which website content became which video ads, enabling easy refreshes when services change.
