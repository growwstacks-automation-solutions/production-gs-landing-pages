The Voice AI Personality Challenge

Traditional voice assistants often sound robotic and impersonal, failing to engage users effectively. Businesses want conversational AI that reflects their brand personality - whether that's professional, friendly, or even humorously sarcastic like our demo agent Humphrey.

Speechmatics Flow solves this by combining accurate speech recognition with customizable LLM responses and natural-sounding text-to-speech. At 2:15 in the video, you'll see how we program Humphrey's snarky personality through simple instruction prompts rather than complex coding.

Key insight: Personality in voice AI isn't about complex programming - it's about crafting the right instructions for the LLM. A few well-chosen sentences can transform a generic assistant into a memorable character.

Getting Started with Speechmatics Flow

Accessing the Flow interface is simple - just navigate to the Agents section in your Speechmatics portal. The interface is divided into three main areas: agent management, creation tools, and testing.

You can start from scratch or use one of the preset templates for common use cases. The video shows how we selected "Start from agent" to create our custom Humphrey assistant. The entire setup process takes less than 3 minutes for a basic agent.

Creating Your First AI Agent

Naming your agent is the first step - we chose "Humphrey" to match the personality we wanted to create. The template ID (visible at 3:42 in the video) is the crucial piece you'll need for integration.

Basic configuration includes:

  1. Agent name and description
  2. Primary language selection
  3. Voice style and gender
  4. Initial greeting message

Pro tip: The greeting message sets the tone for the entire interaction. Notice how Humphrey's "If I must" introduction immediately establishes his reluctant, sarcastic personality.

Customizing Agent Personality

The magic happens in the instruction prompts section. Here's where you define how your agent should behave, respond, and interact with users. For Humphrey, we used prompts like:

  • "You are a friendly but snarky AI agent"
  • "You like to make jokes and are very quick with them"
  • "Respond with reluctant enthusiasm"

At 4:30 in the video, you'll see how these simple instructions create consistent personality throughout the conversation. The agent remembers its character even when handling unexpected inputs.

Language and Voice Options

Speechmatics Flow supports multiple languages with various voice options for each. We selected British Male English for Humphrey to complement his sarcastic personality with a dry delivery.

Key language features:

  • Multiple English variants (US, UK, etc.)
  • Gender options for most languages
  • Custom pronunciation dictionaries
  • Automatic speech adaptation

The custom dictionary feature (shown at 5:15) lets you add specialized vocabulary - we added "speech-eleometer" as an inside joke that Humphrey references later.
