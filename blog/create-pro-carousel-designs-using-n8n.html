The $100 vs 5-Cent Cost Problem

Most businesses know carousels perform exceptionally well on Instagram and LinkedIn - often getting 3-5x more engagement than single-image posts. But creating them is painfully expensive. Quality designers charge $50-$100 per carousel on platforms like Fiverr, and even basic Canva templates require hours of manual work.

This n8n workflow changes the economics completely. By combining Claude for creative direction, Gemini for image generation, and Buffer for publishing, it produces professional-grade carousels for just 5 cents each. The secret lies in three technical innovations:

1. The visual consistency loop - Each new slide references previous ones as images, forcing matching design elements

2. The banned elements list - 200+ specific AI clichés the system is prohibited from using

3. The phased creative process - First establishing a complete visual identity before generating any slides

Your Google Sheet Control Panel

The entire workflow runs from a simple Google Sheet where each row represents one carousel. This becomes your content calendar and creative brief combined. Here's what each column controls:

  • Brand Logo URL - Appears consistently on every slide
  • Product URL - Optional source for scraping descriptions/images
  • Product Description - Manual entry recommended for best results
  • Product Images - Comma-separated URLs of different angles
  • Specification - Creative direction like "dark cinematic luxury"
  • Post Date/Hour - Exact scheduling control
  • Socials - Instagram, Facebook, X or combinations

The workflow processes one row at a time, marking each as complete to prevent duplicates. For bulk production, you can schedule weeks of content in advance by filling multiple rows with future dates.

AI Creative Direction System

At 2,000+ words, the system prompt transforms Claude into an executive creative director. It works in three strict phases:

  1. Visual Identity Creation - Defines color palette, typography, lighting style before any slides
  2. Slide Strategy - Different compositional approaches for each slide (close-up, detail shot, full product)
  3. Image Prompt Template - 7-component structure ensuring consistent output format

The banned elements list is what eliminates the "AI look." It specifically prohibits: organic wave shapes, scattered botanical illustrations, soft pastel washes, flat backgrounds, and 200+ other generic AI tendencies. This forces unique, brand-appropriate designs.

The Visual Consistency Breakthrough

The workflow's secret weapon is its sequential processing of slides (not parallel) with a memory array that stores each completed slide. Before generating the next slide:

  1. Previous slides convert to base64 images
  2. They attach as reference images to the new prompt
  3. The text instruction explicitly says "match exact typography, color palette and lighting"

This dual reinforcement - visual reference plus written instruction - creates remarkable consistency. In testing, slides maintain identical design language even when generated hours apart.

Automatic Multi-Platform Publishing

The final stage handles social media posting through Buffer's API, chosen after testing multiple options. Key advantages:

  • Native support for multi-image carousels (many APIs only handle single images)
  • Reliable media uploads from n8n's server
  • Support for Instagram, Facebook and X from one workflow

The system publishes to all platforms specified in the Google Sheet simultaneously, then updates the sheet with live post URLs for tracking.
