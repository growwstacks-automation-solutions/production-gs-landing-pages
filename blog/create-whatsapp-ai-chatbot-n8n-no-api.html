Step 1: Getting Your API Keys

The workflow needs two API connections - one for AI and one for WhatsApp. Here's how to get both:

AI API Key (OpenAI/Jmini)

For OpenAI (more powerful but paid):

  1. Go to platform.openai.com
  2. Create API key under Account → API Keys
  3. Copy the key (you'll paste into n8n later)

For Jmini (free alternative):

  1. Visit jmini.io
  2. Sign up and generate your free API key

WhatsApp Bridge Token

At 4:30 in the video, you'll see how to:

  1. Sign up at WATI.io (free tier)
  2. Add your WhatsApp number under "Instances"
  3. Generate and copy your API token

Pro Tip: Use a business email for WATI signup even if connecting a personal WhatsApp number. This makes upgrading easier later.

Step 2: Setting Up the WhatsApp Bridge

The bridge service acts as a middleman between n8n and WhatsApp. Here's how to connect them:

  1. Log into your WATI dashboard
  2. Click "Add Instance" and name it (e.g., "Main Business Number")
  3. Enter your WhatsApp number in international format (+[country code][number])
  4. Scan the QR code with your phone's WhatsApp (just like web.whatsapp.com)
  5. Copy the Instance ID and API Token from the dashboard

At 6:10 in the video, you'll see the exact screens where to find these credentials. The connection stays active as long as your phone maintains internet connectivity.

Important: This method doesn't store messages on WATI's servers - all processing happens in your n8n workflow for maximum privacy.

Step 3: Importing the n8n Workflow

With your API keys ready, it's time to set up the automation brain:

  1. Download the pre-built JSON workflow file (link in description)
  2. Log into n8n and create new workflow
  3. Click "Import from File" and upload the JSON
  4. Paste your API keys into the credential fields
  5. Activate both the OpenAI and WhatsApp nodes

The workflow automatically handles:

  • Receiving incoming WhatsApp messages
  • Sending them to AI for processing
  • Returning thoughtful responses
  • Logging conversations for review

At 8:45 in the video, you'll see how to test the connection by sending a WhatsApp message to your number - you should get an instant AI reply!

Step 4: Customizing Your AI Prompts

The magic happens in the "System Prompt" section of the workflow. This tells the AI how to respond:

Example prompt for an eCommerce store: "You are a friendly customer support agent for [Business Name]. Answer questions briefly but helpfully in [Language]. For order status requests, ask for the order number first. Never make up product details - say 'I'll check' if unsure."

To edit:

  1. Find the "System Message" node in the workflow
  2. Replace the placeholder text with your instructions
  3. Add any specific FAQs you want pre-programmed
  4. Save and execute the workflow

At 10:20 in the video, you'll see how to create different responses for keywords like "price" or "support" by adding conditional logic nodes.
