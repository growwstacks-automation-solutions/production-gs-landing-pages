Before Lerty, agencies had three bad options: 1) Build custom front-ends for each client (expensive), 2) Train clients to use technical tools like n8n (unrealistic), or 3) Handle all execution themselves (not scalable). None of these approaches create the kind of sticky, recurring revenue that transforms an agency's business model.

The breakthrough: Lerty provides the missing operations layer that makes AI agents indispensable to daily business operations. It's not just another chatbot builder - it's a complete system for deploying, managing, and monetizing automations at scale.

How Lerty Works: Technical Overview

At its core, Lerty acts as a bridge between your automation workflows (built in n8n, Make, or custom APIs) and your clients' teams. The platform provides three key components:

  1. Webhook Integration: Connect any workflow by replacing your chat output node with a webhook to Lerty's API
  2. Organization Management: Create separate dashboards for each client with custom branding and user permissions
  3. Conversation Orchestration: Automatically handles message threading, user isolation, and mobile sync

What makes Lerty unique is its focus on the business operations layer. Unlike generic chatbot platforms, it's designed specifically for agencies selling automation services to non-technical clients.

Connecting Existing Agents in 2 Minutes

The most powerful feature of Lerty is how easily it connects to your existing automations. At the 4:30 mark in the video, you'll see a live demo of connecting an n8n workflow in under two minutes:

  1. Replace your chat output node with a webhook node
  2. Paste in Lerty's webhook URL from your agent dashboard
  3. Import the response curl command (auto-populates all fields)
  4. Map your workflow outputs to the conversation thread

Pro Tip: Use Lerty's conversation IDs to maintain isolated memory for each client user. This means different employees at the same company can have completely separate interactions with the same underlying agent.

Client Dashboards & Organization Management

Lerty's organization system (shown at 6:15 in the video) solves the biggest headache for agencies - managing multiple clients with different needs. Here's how it works:

  • Each client gets their own organization (dashboard)
  • Organizations can contain multiple agents (content generator, support bot, etc.)
  • Invite client team members with email - no technical setup required
  • Control permissions at both organization and agent levels

The system handles all the messy details of user management so you can focus on building valuable automations rather than admin work. Clients see only the agents you've assigned to them, with your branding and custom interfaces.

Human-in-the-Loop Functionality

At 12:40 in the tutorial, you'll see Lerty's killer feature for professional services - the human-in-the-loop payload system. This allows you to build workflows that pause for client approval or input at critical steps.

The system supports multiple interaction types - simple approve/reject buttons, form inputs, and even multi-step approval chains. All interactions sync automatically with your backend workflow, eliminating the need for custom status tracking.
