Configuring Core Agent Settings

Setting up a voice agent begins with defining its core identity and capabilities. Unlike rigid IVR systems, these settings create a flexible foundation that adapts to customer needs.

Step 1: Language & Availability

Select primary and secondary languages - the agent will automatically detect and switch based on customer speech. Set operating hours or enable 24/7 mode with after-hours messaging options.

Step 2: Personality & Opening Message

Craft an opening line that establishes rapport while setting expectations. Example: "Hi, I'm Sam from TechSupport. I can help with account questions, troubleshooting, or connect you to a specialist."

Step 3: Core Prompt Engineering

The agent prompt defines its knowledge boundaries, response style, and escalation protocols. Include:

  • Brand voice guidelines (formal, friendly, technical, etc.)
  • Types of inquiries it can handle independently
  • When and how to transfer to human agents
  • Safety protocols for sensitive information
