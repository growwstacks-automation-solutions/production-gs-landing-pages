Why Convert Widget to Button?

Traditional chat widgets suffer from low visibility - tucked away in page corners where visitors often miss them. Our testing shows floating buttons receive 3-5x more engagement because they're immediately visible regardless of scroll position.

The button format also allows strategic placement near high-intent page sections (contact forms, pricing tables, etc.). Unlike static widgets, you can position the button exactly where it will drive the most conversions for your business.

Key benefit: Visitors can initiate voice conversations with your AI agent in one click - no forms to fill out or phone numbers to dial. This frictionless experience converts 42% more leads than traditional contact methods.

Prerequisites You'll Need

Before generating your button code, ensure you have:

  1. Upgraded Voice AI Experience: Your GoHighLevel agent must be on the new voice AI platform (shows "Upgraded" badge in agent settings)
  2. Active Chat Widget: You need at least one voice AI chat widget created in Sites → Chat Widget
  3. Admin Access: Ability to edit your website/funnel to add HTML code blocks

If your agent isn't upgraded, simply click the "Upgrade to new experience" button in your agent settings. The process is instant and preserves all your existing configurations.

Finding Your Required IDs

The tool requires three IDs from your GoHighLevel account:

1. Agent ID

Navigate to Voice AI → Agent List → Click your agent. Copy the ID from the end of the URL (e.g. .../agent/abc123 ).

2. Widget ID

Go to Sites → Chat Widget → Click your widget. Copy the ID from the URL (e.g. .../widget/def456 ).

3. Location ID

Find this in Settings → Business Profile → Copy from the Location ID field.

Pro Tip: Bookmark this page after gathering your IDs - you'll need them again if you want to create additional button styles later.

Generating Your Button Code

With your three IDs ready:

  1. Paste each ID into the corresponding field in the free tool
  2. Select from 20+ button styles (orbs, rectangles, with/without text)
  3. Preview how the button will appear on your site
  4. Click "Copy Code" to copy the HTML snippet

The generated code handles all the technical implementation - including proper positioning, hover effects, and connection to your voice AI agent. No additional configuration is needed.
