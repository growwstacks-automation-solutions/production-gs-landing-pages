Kel.com Calendar Setup

The secret to reliable scheduling lies in Kel.com's API-first approach. Unlike Calendly, Kel.com gives programmatic control over availability - exactly what our AI needs to prevent overbooking.

Here's how to configure it (timestamp 1:15 in video):

  1. Create a free Kel.com account
  2. Navigate to Availability → Set your working hours (e.g. Saturdays 11am-2pm)
  3. Create a "Viewing" event type with 30-minute duration
  4. Under Limits → Set max events per day to 2-4 (prevents burnout)

Pro Tip: Grab your Kel.com API key from Settings → API Keys, and note the Event ID from your viewing event URL. These connect to Vapi in the next step.

Vapi AI Agent Configuration

Vapi provides the conversational intelligence that makes this system feel human. Their free tier includes $10 credit (about 80 minutes of talk time) - enough to test thoroughly.

Configuration steps (timestamp 3:22):

  1. Log in to Vapi → Create New Agent → Single Prompt Agent
  2. Select voice (Simo works well for professional female agents)
  3. Under Functions → Add Kel.com for both availability checking AND booking
  4. Paste your Kel.com API key and Event ID in both sections

The knowledge base feature (optional) lets you upload PDFs like pet policies or HOA rules. The AI will reference these documents when answering specific questions.

Spanglish Prompt Engineering

"You are a native bilingual speaker. Use New York-style Spanglish - mixing English and Spanish naturally within sentences as real people do. Never use textbook Spanish unless the caller insists."
