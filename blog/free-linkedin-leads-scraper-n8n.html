How the Free Scraper Works

The workflow uses n8n's HTTP Request node to query Google for LinkedIn profiles matching your ideal customer criteria. Unlike API-based tools that frequently break, this method taps into Google's reliable search infrastructure - the same results you'd get manually searching "CEO automotive Melbourne site:linkedin.com".

At 2:15 in the video, you'll see how the system automatically paginates through up to 10 pages of results (100 leads total per search query). The workflow includes smart error handling to prevent rate limiting and automatically resumes where it left off if interrupted.

Google Search Query Technique

The secret sauce lies in crafting precise search queries that surface your ideal prospects. For local businesses, include both location and industry terms (like "dental clinic Melbourne"). The workflow supports multiple simultaneous searches to cover different buyer personas.

Three key query components make this effective: 1) Job title/role, 2) Industry/niche keywords, and 3) The "site:linkedin.com" operator. This combination forces Google to return only LinkedIn profiles matching your exact criteria. You can further refine with location terms for geo-targeting.

Lead Extraction Process

For each search result, the workflow extracts: name, headline, profile URL, and industry. This data flows into Google Sheets where you can add custom fields like outreach status and notes. The extraction uses CSS selectors to parse Google's result pages, adapting automatically to layout changes.

At 5:30 in the tutorial, notice how the system handles pagination - it will continue scraping until either reaching 100 leads or detecting no more results. This ensures you get maximum value from each search without manual oversight.

Duplicate Lead Management

The biggest challenge with search-based scraping is duplicate leads across different queries. The workflow solves this by comparing new leads against your existing database using profile URLs as unique identifiers.

In practice, this means if you search both "CEO automotive" and "auto shop owner", the system will automatically filter any overlapping profiles. Our testing shows this reduces duplicates by 72% compared to manual methods.

Automated Outreach System

Once leads are captured, the workflow generates personalized icebreaker messages using AI. A simple prompt template combines the prospect's name, industry, and your value proposition to create relevant outreach.

At 8:45 in the video, you'll see how the system tracks outreach status in Sheets, marking contacts as "reached out", "responded", or "not interested". This creates a complete sales pipeline without needing a separate CRM.

Pro tip: The AI message generator works best when you provide 2-3 unique value propositions that can be mixed/matched based on the prospect's profile. This maintains personalization at scale.
