Security Considerations for Production

A common concern (addressed at 9:00 in the video) is API key exposure. Here's the essential security breakdown:

  • Google Cloud hosting: Uses a secure proxy that protects your Gemini API key automatically
  • External hosting: Requires additional backend infrastructure to keep keys secure
  • Demo purposes: Google Cloud provides the simplest, most secure option

For client demonstrations, we always recommend using Google's built-in deployment options. This eliminates security concerns while providing a professional, branded experience.

Step 4: Adding a Custom Domain

At 16:30 in the tutorial, we demonstrate how to create a branded subdomain for your demo:

  1. Click "Deploy" in Gemini 3 to generate a live URL
  2. Navigate to Domain Mappings in Google Cloud
  3. Add a CNAME record in your DNS provider pointing to Google's servers
  4. Wait for SSL certificate provisioning (typically 5-60 minutes)

This creates a professional demo URL like voice.bostonplumbingdemo.com that you can share with clients. The custom domain significantly increases perceived value and makes the demo feel like a finished product rather than a technical prototype.

Voice Agent Customization Options

At 12:45 in the video, we explore two ways to customize your voice agents:
