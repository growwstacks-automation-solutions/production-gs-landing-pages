Security Considerations Explained

At 10:45 in the video, we address the most common concern: API key exposure. Here's the breakdown:

Google Cloud deployment: Your Gemini API key remains secure through Google's proxy system. The frontend communicates with a backend service that holds the actual key, using temporary session tokens.

External hosting: If you download the code and host it elsewhere, you would need to implement proper backend security measures to protect your API key. For quick demos, we strongly recommend using Google Cloud to avoid these complexities.

Pro tip: Google Cloud deployment costs less than $0.10 per demo for typical usage patterns. The time saved on security implementation makes this the obvious choice for agency workflows.

Deployment Options for Client Demos

At 17:30 in the video, we walk through the deployment process. Gemini offers two primary options:

Basic deployment: With one click, your demo is live on a Google-provided URL. This works perfectly for quick internal reviews or initial client presentations.

Custom domain: For maximum impact, you can deploy to a branded subdomain (like demo.clientbusiness.com). This requires adding a simple CNAME record to your DNS settings (explained in the next section).

The deployment process typically completes in under a minute, making it possible to go from idea to live demo faster than scheduling a meeting with your development team.

Custom Domain Setup Process

At 19:45 in the tutorial, we demonstrate the exact steps to deploy your demo to a custom domain:

  1. Click "Open in Google Cloud" from AI Studio
  2. Navigate to Domain Mappings in the sidebar
  3. Click "Add Mapping" and enter your subdomain
  4. Add the provided CNAME record to your DNS
  5. Wait for propagation (typically 5-60 minutes)

Using Cloudflare as an example, we show how to:

  • Create a new CNAME record
  • Set the name to your desired subdomain
  • Point it to ghss.googlehosted.com
  • Disable proxy status during setup

This process works with any domain provider and ensures your demo looks completely professional when presenting to clients.
