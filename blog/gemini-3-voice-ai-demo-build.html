Finding Your Demo Client

The most effective demos target real businesses in your prospect's industry. For this example, we'll create a demo for a plumbing company to showcase how Voice AI can handle customer inquiries and emergency dispatch.

Start by identifying a local business with an existing website. Google Maps listings often provide the best targets - look for businesses with:

  • Clear service offerings listed
  • Contact information available
  • Some branding elements (colors, logo)

For our demo, we selected "Boston Budget Plumbing" - a mid-sized operation with enough online presence to extract branding but not so polished that our demo wouldn't impress.

Extracting Brand Information

Manually copying brand details is time-consuming. Instead, we leverage AI to scrape and summarize the key elements we need for our demo:

  1. Copy the business website URL
  2. Use ChatGPT to extract: brand name, services, contact info, color scheme
  3. Verify accuracy with a screenshot for color correction

Pro Tip: The prompt "Extract brand information including services, contact details, and color profile. Return in concise markdown format" reliably produces structured data for the next step.

For Boston Budget Plumbing, this process captured their navy blue and orange color scheme, service offerings, and phone number - everything needed to create a convincingly branded demo.

Generating the Website

With brand information prepared, we move to Google AI Studio to generate the demo:

  1. Paste the brand information into Gemini 3
  2. Request: "Create a one-pager app with two voice AI agents (front desk and emergency dispatcher) in the above-the-fold section"
  3. Watch as Gemini generates HTML, CSS, and JavaScript in real-time

The generation process takes 2-5 minutes, during which you can observe the code being written. Errors that appear are automatically corrected by Gemini - no developer intervention needed.

The result is a complete website with:

  • Brand-appropriate color scheme
  • Functional voice interface
  • Mobile-responsive design
  • All necessary backend connections

Customizing Voice Agents

The default demo includes two agents - "Sarah" (front desk) and "Mike" (dispatcher). These can be customized in two ways:

  1. No-code method: Simply tell Gemini to "Make Sarah use a female voice and Mike use a male voice"
  2. Code adjustment: Modify the constants in the generated code to change names, voices, and responses

For our plumbing demo, we adjusted the agents' system prompts to include industry-specific knowledge about common plumbing issues, service areas, and emergency response protocols.

Conversion booster: Customized agents that demonstrate industry knowledge convert 3x better than generic demos during client presentations.

Deployment Options

Gemini offers two deployment paths with different security implications:

Google Cloud Hosting

  • API keys protected by Google's proxy system
  • Deploys in under 2 minutes
  • Low cost ($5-20/month)
  • Supports custom domains

Self-Hosting

  • Requires backend service to protect API keys
  • More setup time
  • Greater control over infrastructure
  • Higher ongoing maintenance

For most agency demo purposes, Google Cloud hosting provides the perfect balance of speed, security, and cost-effectiveness.

Custom Domain Setup

While the auto-generated URL works, custom domains make demos more professional:

  1. In Google Cloud, navigate to Domain Mappings
  2. Add a subdomain like "demo.clientbusiness.com"
  3. Add the provided CNAME record to your DNS
  4. Wait for propagation (typically 1 hour)

For our plumbing demo, we set up "bostonbudgetplumbing.demo.agency.com" - creating a seamless experience when sharing with the potential client.

Pro Tip: Use Cloudflare for DNS management - their interface makes CNAME updates simple and provides visibility into propagation status.
