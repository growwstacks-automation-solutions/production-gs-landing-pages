Security Considerations for API Keys

One common concern with AI-powered demos is API key security. At 9:45 in the tutorial, we address this critical question: Does hosting these demos expose your Gemini API key?

The answer depends on your deployment method. When using Google Cloud (the recommended approach), your API key remains secure through an automatic proxy system. Google handles the security layer, preventing direct exposure of your credentials. For other hosting methods, additional precautions are necessary to protect your API key.

Important: Always deploy Voice AI demos through Google Cloud when possible. This maintains security while requiring zero additional configuration on your part.

Custom Domain Magic for Client Presentations

Generic demo URLs lack professionalism and memorability. At 14:20 in the video, we show how to map your Voice AI demo to a custom subdomain (like plumber.yourdomain.com) for maximum impact.

This process involves:

  1. Deploying the app through Google Cloud
  2. Adding a domain mapping in Google Cloud Console
  3. Creating a CNAME record with your domain provider

The result is a branded demo URL you can proudly share with clients, complete with SSL security. While DNS propagation can take up to 48 hours, it typically completes much faster — often within 30-60 minutes.
