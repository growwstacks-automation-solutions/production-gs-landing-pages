The Agent Skills Revolution

For months, developers using AI coding assistants faced a frustrating limitation - generic prompts that often missed the mark for specialized tasks. Claude's recent Agent Skills breakthrough showed what was possible, but terminal-based coders were left out... until now.

Google's open-source Gemini CLI v0.23 introduces Agent Skills - modular packages that teach the AI specific capabilities. Instead of hoping your prompt will work, you can now equip Gemini with the exact expertise needed for each task, just like adding tools to a belt.

Game-changing difference: Where generic AI might struggle with specialized coding tasks, Agent Skills provide structured, reusable knowledge that makes Gemini reliably competent in specific domains. One demo showed transforming hundreds of messy notes into an organized knowledge graph in under 30 minutes.

Getting Started with Agent Skills

Upgrading to access these powerful new capabilities takes just minutes. At 1:25 in the video tutorial, you'll see the exact steps to enable this game-changing feature.

First, update to the latest version (0.23+) using npm. Then open Gemini CLI and navigate to settings to enable preview features - this unlocks access to the Gemini 3 model series. Search for "agent skills" in settings and toggle it to true, then reset your CLI.

Now you can use the /skills command to manage your Agent Skills. The interface lets you list, enable, disable, and reload skills as needed. Creating new skills is surprisingly simple - you can do it directly in terminal or by editing the skills.md file with just name, description, and instructions.

Gemini Conductor Integration

While Agent Skills steal the spotlight, the full Conductor integration is equally transformative. At 5:30 in the tutorial, you'll see how Conductor moves project context out of volatile chat history and into persistent markdown files that live with your code.

This is particularly powerful for existing projects where most AI tools struggle. Conductor formalizes your project's goals, architecture, tech stack and workflows - and keeps this context updated as the project evolves. Installation takes just one command, and then you can start using all Conductor commands through the Gemini CLI interface.

New Usage Dashboard Features

At 7:10 in the video, you'll discover the new /stats command that provides an intuitive display of your Gemini CLI usage summary. This makes it easy to track your performance and interaction patterns.

For deeper insights, the mpx gemini-w command generates a condensed 365-day summary showing:

  • Total tokens used across all sessions
  • Message counts and session statistics
  • Your longest usage streak
  • Breakdown of models used
  • Primary language usage

Additional Updates in v0.23

While Agent Skills and Conductor are the headliners, version 0.23 brings several other quality-of-life improvements:

  • Windows clipboard image support: Paste images directly into CLI using Alt+V
  • Automatic terminal background detection: CLI now detects your terminal's background color to select compatible themes
  • Accessibility warnings: Surface potential accessibility issues automatically
  • Increased free tier usage: More generous limits with the Gemini 3 model
  • Numerous bug fixes: Smoother experience across all platforms

The full changelog is available in the video description, but these are the highlights that will impact most developers' daily workflows.
