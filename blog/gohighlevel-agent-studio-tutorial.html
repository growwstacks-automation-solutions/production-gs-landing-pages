Getting Started with Your First AI Agent

Creating your first AI employee in Agent Studio takes just minutes. Here's how to begin:

  1. Navigate to AI Agents → Agent Studio in your GoHighLevel dashboard
  2. Click Create Agent and name your AI employee (e.g., "Sales Assistant Lisa")
  3. Add a brief description of the agent's role and responsibilities
  4. Click Create Agent to enter the visual builder interface

The builder uses a node-based system similar to Make.com or n8n, where you drag and connect different capabilities to create your agent's workflow. At 4:32 in the tutorial video, you'll see how simple it is to set up the basic structure.

Pro Tip: Start with one specific task (like appointment booking) before building more complex agents. This lets you test and refine the workflow before scaling up.

Node-by-Node Breakdown of the Builder

Agent Studio's power comes from its specialized nodes - the building blocks of your AI workforce. Here's what each one does:

AI Agent Node (The Brain)

This is your agent's core intelligence. You define its personality, expertise, and decision-making parameters here. The prompt you write determines how it thinks and responds.

Sequential Node

Creates step-by-step processes where actions happen in a specific order. Essential for multi-step workflows like lead qualification → pricing research → appointment booking.

End Node

Marks the completion point of your agent's workflow. Like a "stop" in traditional automation, this tells the AI when its tasks are complete.

Search Knowledge Base

Enables your agent to reference your company's information repositories. Critical for maintaining accuracy in legal, medical, or technical fields.

Search the Web

Allows real-time internet research. At 12:45 in the video, you'll see how this works for checking industry averages and competitive pricing.

MPC Server Node

Connects your AI to other AI systems and servers, essentially giving it "superpowers" by integrating with specialized tools and databases.

API Call Node

Lets your agent interact with other software platforms. Sync data with CRMs, update spreadsheets, or trigger actions in external apps.

Generative AI Nodes

These handle content creation - generating audio, images, text, and videos on demand. We'll explore the video generation feature in detail next.

Building a Video Generation Agent (Step-by-Step)

One of Agent Studio's most impressive features is automated video creation. Here's how to build an agent that generates personalized sales videos:

  1. Start with a Sequential Node as your workflow container
  2. Add a Generate Video processing node
  3. Enter your video script/prompt (e.g., "Create video of a woman mowing a lawn saying...")
  4. Select your preferred AI model (VO or VO Fast)
  5. Choose aspect ratio and resolution
  6. Connect to your main workflow and click Save

At 9:15 in the tutorial, you'll see this in action as the AI generates a sample video in seconds. The quality is surprisingly good, especially when you provide detailed prompts about subject, setting, and messaging.

Real-World Application: Service businesses can show "before and after" transformations. Dentists can generate smile makeover videos. The key is triggering these personalized videos at strategic moments in your sales process.

Using the Web Research Feature

The web search capability takes AI agents beyond simple automation. Here's how to implement it:

  1. Add a Search the Web node to your workflow
  2. Enter the research query (e.g., "average cost for backyard renovation in [location]")
  3. Connect it to your conversation flow where this data would be relevant
  4. Test to ensure it pulls accurate information (shown at 13:20 in the video)

This feature is particularly powerful for:

  • Real estate agents comparing property values
  • Service businesses providing accurate quotes
  • Consultants offering data-driven recommendations
  • Any industry where current market knowledge builds trust

As the technology improves, this research capability will enable AI agents to handle increasingly complex sales conversations that previously required human expertise.
