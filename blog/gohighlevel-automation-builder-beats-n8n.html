Accessing the New Builder

Many GoHighLevel users don't realize the new automation builder exists or how to activate it. Unlike the clunky old interface, this version requires a simple but often-missed activation step.

To access it, navigate to Automations in your dashboard and click any workflow. Look for the Advanced Builder button—if it's not visible, you'll need to enable it in Labs under Settings by searching for "workflow" and activating all three related features for your subaccount.

Pro Tip: Don't immediately convert all existing automations—test the new builder first as some elements may rearrange differently.

Major UI Improvements

The old automation builder felt technical and intimidating. The new interface solves this with a clean, modern design that makes automation accessible to non-technical users.

Key upgrades include auto-save functionality (eliminating constant manual saves), intuitive node connections, and visual clarity that reduces cognitive load. The entire workflow is now easier to follow at a glance.

Time Saver: Auto-save alone prevents an average of 15-20 manual saves per workflow session, reducing frustration and lost work.

Drag-and-Drop Simplicity

Building automations used to require precise clicking through nested menus. The new builder introduces true drag-and-drop functionality that feels more like using design software than coding.

Simply click and drag triggers or actions from the sidebar, then connect them visually. The interface provides clear visual feedback about valid connections, reducing errors. At 2:15 in the video, you can see how effortlessly nodes can be rearranged.

Multiple Triggers Per Workflow

Previously, each automation could only have one trigger, forcing businesses to create separate automations for related processes. This created organizational nightmares.

The new builder allows unlimited triggers per workflow. You can now combine all related automations into one organized flow—for example, handling new leads from web forms, calls, and chat messages in a single workflow with different trigger paths.

Game Changer: Agencies report reducing automation clutter by 60-70% by combining previously separate workflows.

Powerful Sidebar Features

The new sidebar transforms automation from a technical tool to a collaborative workspace. Key features include:

  • Workflow Notes: Document processes and leave team instructions directly in the automation
  • Sticky Notes: Visually annotate complex workflows for team members
  • Error Detection: Automatic problem spotting before testing
  • Stats View: Real-time performance metrics for every automation element
  • Workflow Switcher: Jump between automations without returning to the dashboard

These tools make it easier for teams to collaborate on complex automations while maintaining clarity.

Professional Keyboard Shortcuts

Power users will appreciate the new keyboard shortcuts that mirror professional design and development tools. Essential shortcuts include:

  • Zoom in/out: + and - keys
  • Stats view: Command+Shift+S
  • Navigation: Arrow keys for panning
  • Quick actions: Various modifier key combinations

These allow experienced users to build automations up to 3x faster than clicking through menus.

Format Tree Organization

Complex automations often become visually messy during editing. The Format Tree button (shown at 8:32 in the video) automatically reorganizes your entire workflow with one click.

This feature intelligently arranges all nodes and connections in a clean, readable layout—saving countless hours of manual reorganization. It's particularly valuable when modifying existing automations or collaborating with team members.
