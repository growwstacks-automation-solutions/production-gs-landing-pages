Zapier solves this by creating instant connections between Squarespace and your other business tools. When properly configured, each form submission can automatically trigger actions like CRM updates, email sequences, or spreadsheet logging - all without writing code or hiring developers.

Step 1: Generate Your Squarespace API Key

The foundation of your automation is a secure API key that allows Zapier to communicate with your Squarespace site. Here's how to create one:

Step 1.1: Access Developer Settings

Log into your Squarespace account and navigate to Settings → Advanced → Developer API Keys. This menu is only visible to site owners, not contributors.

Step 1.2: Create New Key

Click "Generate Key" and give it a descriptive name like "Zapier Automation". This helps identify the key's purpose later if you need to revoke access.

Pro Tip: Generate separate keys for different automation purposes (forms vs commerce) for better security control.

Step 2: Configure Key Permissions

API keys follow the principle of least privilege - they only access what you explicitly permit. For form automation:

Step 2.1: Select Form Permissions

Check the "Forms" permission box. This allows Zapier to read your form submissions without accessing other site data.

Step 2.2: Add Commerce Permissions (If Needed)

If automating order processing, also check "Commerce". Avoid selecting permissions you won't use to minimize security risks.

Step 2.3: Generate and Copy Key

Click "Generate Key" and immediately copy the alphanumeric string. Squarespace won't display it again for security reasons.

Security Note: Treat API keys like passwords. Never email them or store in unsecured documents.

Step 3: Create a New Zap in Zapier

With your API key ready, switch to Zapier to build the automation workflow (called a "Zap"):

Step 3.1: Start New Zap

Click "Create Zap" in Zapier and select Squarespace as your trigger app. Search for "Squarespace Forms" as the trigger event.

Step 3.2: Choose Trigger Event

Select "New Form Submission" as your trigger. This tells Zapier to watch for completed forms on your site.

Time Saver: Naming your Zap clearly (e.g. "Contact Form to CRM") helps when managing multiple automations.

Step 4: Connect Your Squarespace Account

Now link Zapier to your Squarespace site using the API key:

Step 4.1: Add API Key

When prompted, paste your Squarespace API key into Zapier's connection settings. Click "Yes, Continue" to test the connection.

Step 4.2: Select Your Form

Zapier will display all forms from your Squarespace site. Choose the specific form you want to automate.

Step 4.3: Pull Test Data

Zapier retrieves sample submission data to verify the connection works. Review these fields to ensure all expected data appears.

Data Mapping Tip: The test data shows exactly how Zapier will receive future submissions - use this to plan your action steps.

Step 5: Test and Activate Your Zap

The final steps ensure your automation works reliably:

Step 5.1: Configure Action Steps

Add what happens after a form submission - like sending to Google Sheets, your CRM, or email service. Map each form field to its destination.

Step 5.2: Test with Live Data

Submit a test form on your Squarespace site. Verify Zapier processes it correctly before full activation.

Step 5.3: Turn On the Zap

Once testing succeeds, activate your Zap. It will now run automatically for every new form submission.

Monitoring Tip: Check Zapier's history tab periodically to catch any failed runs early.

Troubleshooting Common Connection Issues

Most Squarespace-Zapier connection problems stem from a few fixable causes:

API Key Expired or Revoked

If your Zap stops working, first verify the API key still exists in Squarespace's developer settings. Regenerate if needed.

Form Deleted or Modified

Changing form fields or deleting the form in Squarespace breaks existing Zaps. Update your Zap's field mappings after any form changes.

Permission Changes

If you added new form fields requiring additional permissions, regenerate your API key with updated permissions.

Zapier's error logs provide specific failure reasons - always check here first when troubleshooting.

Advanced Automation Tips
