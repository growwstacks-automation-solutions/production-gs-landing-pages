n8n Workflow Overview

This automated podcast workflow consists of just four core components in n8n, demonstrating how powerful simplicity can be when combined with the right AI tools:

  1. Trigger: Starts the workflow (manual chat input or scheduled)
  2. Topic Generator: AI agent that creates relevant podcast topics
  3. Script Writer: Transforms topics into engaging scripts
  4. Voice Generator: ElevenLabs converts text to studio-quality audio

The magic happens in how these nodes pass data seamlessly between them. The topic generator's output becomes the script writer's input, which then feeds directly into the voice generator - all without manual intervention.

Step 1: AI-Powered Topic Generation

The foundation of any great podcast episode is a compelling topic. Our n8n workflow starts with an AI agent specifically trained to generate podcast ideas tailored to your brand.

Using a customized system prompt (like the one at 4:12 in the video), the AI understands your niche, audience, and preferred episode format. For example, a prompt might specify: "Generate topics for a solo monologue tech podcast focused on AI ethics, targeting small business owners, with episodes under 15 minutes."

Pro tip: Add a memory node to your topic generator so it remembers previous episodes and avoids repetition. This creates a coherent podcast series rather than disconnected episodes.

Step 2: Automated Script Writing

With a topic selected, the next node generates a complete podcast script. This is where choosing the right AI model makes all the difference in quality:

  • GPT-4: Best for technical accuracy and structure
  • Claude: Excels at natural, conversational writing
  • Mixtral: Great for creative storytelling

The script writer uses its own system prompt (shown at 7:45 in the video) to maintain consistent tone and format across episodes. You can specify elements like:

  • Preferred episode length
  • Inclusion of calls-to-action
  • Level of humor or seriousness
  • Structural elements (intro, segments, outro)

Step 3: ElevenLabs Voice Generation

ElevenLabs transforms your AI-written script into natural-sounding audio. The HTTP Request node in n8n (configured at 12:30 in the video) handles the API connection with these key parameters:

  • Voice ID: Choose from 100+ pre-made voices or clone your own
  • Stability: Controls consistency vs. emotional variation
  • Clarity: Adjusts pronunciation precision
  • Style Exaggeration: For more dramatic delivery

The generated audio file (MP3 format) is then passed to the final delivery node. ElevenLabs' advanced models add natural pauses, emphasis, and even breath sounds that make the audio indistinguishable from human recording.

Step 4: Delivery Options

The completed audio file can be routed to multiple destinations based on your workflow needs:

  • Email: Simple notification with attachment
  • Cloud Storage: Automatic saving to Google Drive or Dropbox
  • CMS: Direct upload to your podcast hosting platform
  • Team Chat: Post to Slack/Teams for review

For fully automated publishing, connect n8n to your podcast host's API (like Buzzsprout or Captivate). The workflow can add metadata, schedule publication, and even generate show notes - creating a completely hands-off podcast production system.
