What Is an AI Agent in Make.com?

An AI agent in Make.com transforms your automation from simple if-then rules into intelligent decision-making. Unlike basic bots that follow rigid instructions, an AI agent can analyze context, make judgment calls, and determine the best course of action.

At 1:15 in the video, we see how the agent evaluates meeting requests differently based on the sender's importance, your current schedule, and the meeting purpose. This dynamic decision-making is what sets AI agents apart from traditional automation.

Key difference: Regular automation executes predefined steps. An AI agent evaluates options and chooses actions based on your goals and constraints.

Setting Up Your AI Agent

The foundation of any effective AI agent is proper setup. After your Google Sheets module (where meeting requests are collected), click the plus button and search for "agent". Select "Run an agent" and click "Create an AI agent".

You'll need to:

  1. Name your agent (e.g., "Scheduling Assistant")
  2. Select a model (choose the larger one for complex tasks)
  3. Connect your Google account
  4. Prepare your system prompt (more on this next)

The system prompt is where you define the agent's role, priorities, and boundaries. We provide a template you can customize for your specific needs.

Connecting Calendar Tools

For scheduling automation, your AI agent needs two key Google Calendar tools:

1. Get Free/Busy Information

Configure this tool to:

  • Set time range fields to "Let AI agent decide"
  • Select your specific calendar from the dropdown
  • Turn off mapping in calendar settings

2. Create an Event

For event creation:

  • Set most fields to "Let AI agent decide"
  • Enable Google Meet for video conferencing
  • Set default duration to 30 minutes (0030)
  • Allow agent to format the description

Pro tip: At 3:22 in the video, we show how to configure buffer times between meetings automatically.

Configuring Email Automation

Your AI agent can send polished follow-up emails by connecting Gmail:

  1. Add the "Send an email" tool module
  2. Connect your Google account
  3. Set subject and content fields to "Let AI agent decide"
  4. Change body type to "raw HTML" for formatting flexibility

The agent will compose emails using information from your spreadsheet and calendar event details. At 4:10 in the video, you'll see how it personalizes each message based on the meeting context.

Crafting the Perfect System Prompt

Your system prompt determines how the AI agent makes decisions. A good prompt includes:

  • Your preferred meeting hours and duration defaults
  • Priority rules for different request types
  • Email tone and formatting preferences
  • Boundaries on what the agent can decide autonomously

We provide a template with these key sections:

Example instruction: "For internal team meetings, always include the project name in the calendar event title and set reminders for 1 hour before. For external meetings, include the company name and send confirmation emails with Google Meet links."

Testing and Refining Your Workflow

Before going live:

  1. Save your agent and scenario
  2. Open your calendar and email in new tabs
  3. Run a test with recent data
  4. Verify the new event appears correctly
  5. Check the email formatting and content
