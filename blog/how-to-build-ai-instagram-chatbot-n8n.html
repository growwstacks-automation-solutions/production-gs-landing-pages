Step 1: Facebook App Setup for Instagram Messaging

Facebook's API is the gateway to Instagram automation. You'll need to:

  1. Create a Facebook Business Account at business.facebook.com
  2. Connect your Instagram account in Settings → Linked Accounts
  3. Register a new app at developers.facebook.com with "Manage messaging and content on Instagram" permissions

The critical permissions to enable are:

  • Instagram Basic Display
  • Instagram Manage Messages
  • Pages Read Engagement
  • Business Management

Pro tip: At 4:12 in the video, you'll see how to troubleshoot common permission errors during setup.

Step 2: n8n Webhook Configuration

The webhook is the bridge between Instagram and your automation. In n8n:

  1. Add a Webhook node with "Allow multiple HTTP methods" enabled
  2. Create a verification endpoint using a Code node (template provided in video)
  3. Connect to Respond to Webhook node for Facebook's verification check

This setup ensures Facebook can verify your app while passing incoming messages to your workflow. The verification token must match exactly between n8n and your Facebook app settings.

Step 3: Training Your AI Agent for Brand-Specific Responses

Generic AI responses sound robotic and hurt conversions. The system message defines:

  • Your brand voice (professional, friendly, etc.)
  • Product/service knowledge boundaries
  • Call-to-action structure for bookings

Example system message from the tutorial:

"You are an assistant for [Business Name]. Respond conversationally to Instagram messages. Answer only questions about our [products/services]. If someone asks about working with us, invite them to book a free strategy call at [link]. Keep responses under 3 sentences. Never output anything except the direct reply text."

Step 4: Building the Call Booking System

The most valuable part of this automation handles appointment scheduling without human intervention:

  1. Connect Google Sheets to track availability (template provided)
  2. Configure AI to check open slots before offering times
  3. Set up confirmation messages with booking details

At 18:30 in the video, you'll see how to troubleshoot common booking errors like duplicate entries or failed calendar updates.

Step 5: Long-Lived Token Management

Default Facebook tokens expire hourly. To maintain 24/7 operation:

  1. Generate token in Graph API Explorer
  2. Extend at developers.facebook.com/tools/debug
  3. Update n8n credentials with new 60-day token

This process needs repeating every 2 months. For fully hands-off operation, GrowwStacks can implement automatic token rotation.

Testing and Deployment

Before going live:

  1. Test with secondary Instagram account (not your business page)
  2. Verify response times under 5 seconds
  3. Check booking system creates calendar events

Once validated, publish your Facebook app and activate the n8n workflow. The automation will now handle all incoming messages with zero ongoing effort.
