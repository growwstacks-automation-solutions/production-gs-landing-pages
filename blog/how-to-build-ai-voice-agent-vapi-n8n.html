Step 1: Set Up OpenAI API

VAPI uses OpenAI's Whisper for speech-to-text and GPT for conversation handling. You'll need:

  1. An OpenAI account at platform.openai.com
  2. A $5 credit balance (actual usage is about $0.15/month for most small businesses)
  3. To create a new secret key under API keys

At 4:12 in the video, you'll see where to copy this API key into n8n's credentials manager. This connection powers the agent's ability to understand and respond naturally.

Step 2: Import n8n Workflow

The tutorial uses a pre-built n8n workflow that handles:

  • Call routing logic
  • Calendar integration
  • Data collection
  • Confirmation messaging

To import it (shown at 5:18):

  1. Download the workflow JSON file
  2. In n8n, click the dots menu → Import
  3. Select the file and confirm

Pro tip: The workflow includes error handling for cases where the AI doesn't understand a request, automatically asking the caller to rephrase or offering alternative solutions.

Step 3: Connect Calendar

At 6:45, the tutorial shows how to connect Google Calendar:

  1. Click credentials in n8n
  2. Select "Sign in with Google"
  3. Choose your business account
  4. Grant calendar access permissions

This allows the agent to check real-time availability and book appointments directly into your schedule. The workflow can be modified to sync with other calendars like Outlook or Calendly if needed.

Step 4: Configure VAPI Assistant

The VAPI dashboard (shown at 8:30) is where you:

  1. Create a new assistant from blank template
  2. Name it (e.g., "HVAC Booking Agent")
  3. Set the opening message ("Hello, this is Amy from Kamore Air...")
  4. Paste in your custom prompt (provided in the workflow files)

The prompt defines the agent's personality, knowledge base, and call handling rules. For HVAC businesses, it includes service descriptions, pricing ranges, and common troubleshooting questions.

Step 5: Add Custom Tools

At 10:15, the tutorial adds two critical tools:

  1. Webhook Tool: Connects VAPI to your n8n workflow URL
  2. Knowledge Base: Uploads your FAQ document (service areas, pricing, warranties etc.)

These tools enable the agent to:

  • Trigger real-world actions (booking appointments)
  • Answer complex questions using your business documentation
  • Escalate to humans when needed

Step 6: Activate Workflow

The final setup step (shown at 12:40):

  1. In n8n, click "Activate workflow"
  2. In VAPI, click "Publish"
  3. Test with a phone call to confirm everything works

That's it - your AI receptionist is now live. The entire setup from zero to functioning agent takes most businesses under 30 minutes following this workflow.
