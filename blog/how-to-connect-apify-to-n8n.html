Apify + n8n: The Perfect Automation Pair

Apify specializes in web scraping with pre-built "actors" (scrapers) for platforms like Google Maps, LinkedIn, eCommerce sites and more. n8n provides workflow automation to process and route that data. Together they create an end-to-end solution without coding.

Apify handles the complex scraping operations - dealing with pagination, JavaScript rendering, and anti-bot measures. n8n manages the business logic - scheduling runs, transforming data, and sending it to CRMs, sheets or other destinations. The integration gives you scraping power without infrastructure headaches.

Step 1: Create Your Apify API Token

Every n8n workflow connecting to Apify needs authentication. In your Apify console (1:15 in the video), navigate to Settings → API and Integrations. Create a new API token with a descriptive name like "n8n Integration".

This token allows n8n to trigger Apify actors and retrieve results. Treat it like a password - never expose it in client-side code or share publicly. The token only needs "Run actor and read dataset" permissions for basic integrations.

Pro Tip: Create separate tokens for development and production environments. Revoke compromised tokens immediately in the Apify console.

Step 2: Choose and Configure an Apify Actor

Apify's Actor Store (3:40) offers hundreds of pre-built scrapers. For lead generation, the Google Maps Scraper is particularly powerful. Search for "Google Maps" in the store, then configure your target:

  • Search terms: "yacht builder" (or your industry)
  • Location: "Amsterdam" (or your target city)
  • Max results: 100 (adjust based on needs)

The actor's configuration panel shows all available options. Most business use cases don't require advanced settings - the defaults work well for contact scraping, competitor monitoring and market research.

Step 3: Connect Apify to n8n

At 5:20 in the tutorial, we copy the Actor's API endpoint from Apify's "API" tab. In n8n, create a new HTTP Request node:

  1. Method: POST
  2. URL: Paste the "Run actor" endpoint
  3. Authentication: Basic Auth with your Apify token
  4. Body: JSON configuration from Apify

When executed, this node starts the Apify actor. Scraping jobs take time - we handle this in n8n either with a fixed delay (for predictable scrapes) or a loop checking completion status (for variable durations).

Note: The first execution may fail as n8n learns the JSON structure. Use the "Set manually" option to map fields correctly on subsequent runs.

Step 4: Process and Store Scraped Results

Once scraping completes (8:50), add another HTTP Request node to fetch results using Apify's "Get dataset items" endpoint. Configure it as:

  • Method: GET
  • URL: Dataset endpoint from Apify
  • Authentication: Same token as before

The tutorial shows storing in Google Sheets, but n8n supports 300+ destinations. For lead generation, consider enriching data with email verification or social profiles before sending to your CRM. For market research, analyze trends over time by timestamping each scrape.
