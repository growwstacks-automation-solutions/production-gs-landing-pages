Configuring Facebook Trigger

The first critical step is setting up the trigger - the event that will start your automation. For this integration, we'll use "New Lead" in Facebook Lead Ads as our trigger.

At 3:30 in the video, you'll see how to:

  1. Select "Facebook Lead Ads" as your trigger app
  2. Choose "New Lead" as the trigger event
  3. Connect your Facebook account (one-time authorization)
  4. Select the specific Facebook Page and Lead Form you want to monitor

This configuration ensures Zapier watches for new leads only from your selected form, preventing duplicate entries from other lead sources.

Mapping to Insightly CRM

With the trigger configured, we now set up the action - sending the lead data to Insightly CRM. This is where field mapping becomes crucial for accurate data transfer.

At 5:10 in the tutorial, we demonstrate how to:

  1. Select "Insightly" as your action app
  2. Choose "Create Lead" or "Create Contact" as the action event
  3. Connect your Insightly account
  4. Map Facebook form fields to corresponding Insightly fields

Critical Mapping: Always map these core fields at minimum - Name, Email, Phone Number. Custom fields from your Facebook form can be mapped to Insightly's custom fields for complete data capture.

Common Mistakes to Avoid

After implementing this integration for dozens of businesses, we've identified these frequent pitfalls:

  1. Incomplete Field Mapping: Missing key fields like phone numbers or custom questions
  2. Wrong Trigger Selection: Choosing "New Page Lead" instead of "New Form Lead"
  3. Testing Oversights: Not verifying all mapped fields with real test data
  4. Permission Issues: Facebook page admin rights required for form access
