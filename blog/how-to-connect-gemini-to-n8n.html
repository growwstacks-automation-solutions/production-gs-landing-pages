Why Connect Gemini to n8n?

Businesses today need to leverage AI to stay competitive, but integrating AI into existing workflows can be challenging. Google Gemini offers powerful natural language capabilities, while n8n provides flexible workflow automation. Connecting them creates a powerful combination.

With this integration, you can automate content creation, customer support responses, data analysis, and more - all within your existing n8n workflows. The setup takes just minutes but unlocks significant productivity gains.

Key benefit: Adding Gemini to n8n lets you automate complex language tasks that previously required human intervention, saving hours per week on repetitive content-related work.

Step 1: Create Your n8n Workflow

Begin by logging into your n8n dashboard. You can either open an existing workflow or create a new one specifically for your Gemini integration. For this tutorial, we'll create a new workflow from scratch.

At 0:45 in the video, you'll see how to start with a simple chat trigger node. This creates the foundation for your AI-powered workflow. The chat trigger will initiate the workflow whenever a new message is received.

Step 2: Add the Gemini Node

Next, you'll add the Google Gemini node to your workflow. In n8n, search for "Gemini" in the node panel and select the "Google Gemini Chat Model" node. This node handles all communication with Gemini's API.

The Gemini node appears at 1:10 in the video tutorial. You'll see how to position it after your trigger node and configure the basic settings before connecting the API.

Step 3: Get Your Google Gemini API Key

To connect Gemini to n8n, you'll need an API key from Google AI Studio. Visit ai.google.dev (shown at 1:30 in the video) and navigate to the API keys section.

Create a new API key specifically for your n8n integration. Give it a descriptive name like "n8n Integration" so you can identify it later. Google provides this key for free with usage limits.

Security tip: Never share your API key publicly. Store it securely in n8n's credential system and rotate it periodically for security.

Step 4: Configure the Connection

Back in n8n, paste your API key into the Gemini node's credentials section (demonstrated at 2:15 in the video). The host URL should remain as the default Google API endpoint - don't modify this.

After pasting your key, click "Save" and n8n will test the connection. You should see a "Connection tested successfully" message confirming everything is working properly.

Step 5: Test the Integration

With the connection established, you can now test your Gemini integration. At 2:50 in the video, you'll see a simple test where typing "Hi" generates a response from Gemini.

Try sending different prompts to verify the integration works as expected. You can also select different Gemini model versions (like Gemini 1.5 Flash or Pro) to compare their performance.
