How to Connect Calendar Booking in 15 Minutes

The real magic happens when your agent can book appointments directly into the business's calendar. Here's how to set it up:

1. Create a Cal.com Account

This free alternative to Calendly offers robust API access. Sign up with Google to automatically connect the business's existing calendar.

2. Set Up Appointment Types

Create event types matching the business's services (e.g., "Dental Cleaning - 60min"). Note the Event Type ID from the URL.

3. Generate API Key

In Cal.com settings, create a new API key labeled "Retail AI Integration" and copy it.

4. Connect to Retail AI

In your voice agent's Functions tab, add "Check Calendar Availability" and "Book Appointment" functions. Paste your API key and Event Type ID when prompted.

Testing Tip: Always simulate the full booking flow before delivering to clients. Call your agent, request an appointment, and verify it appears correctly in the calendar.

Testing and Optimizing Your Agent's Performance

A poorly tuned voice agent frustrates callers. Follow this checklist before client delivery:

1. Test Common Scenarios

Simulate 10-15 real calls: booking requests, service questions, after-hours calls, and complex inquiries needing human transfer.

2. Measure Response Time

Agents should respond within 2 seconds. Laggy interactions feel robotic.

3. Check Calendar Accuracy

Verify booked appointments have correct details (name, service, duration) and don't double-book.

4. Optimize Prompt Length

Agents should get to the point. If they ramble (like at 12:40 in the video), edit your prompt to be more concise.
