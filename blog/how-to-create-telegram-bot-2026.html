Building the Simple Version (Rule-Based)

We'll start with a basic rule-based bot that handles common questions through buttons and predefined responses. This version works well for businesses with predictable, repetitive inquiries.

At 4:32 in the video, you'll see how to create a welcome message with two options: "Return Policy" and "Return Status." When customers click a button, they receive the appropriate automated response.

Pro tip: Always include a text input option alongside buttons. About 30% of users prefer typing their questions naturally rather than clicking buttons.

Connecting ManyChat to Telegram

The first technical step is linking ManyChat to your Telegram account. Here's how it works:

  1. Create a new bot through Telegram's BotFather
  2. Copy the API token provided
  3. Paste it into ManyChat's Telegram integration settings

This connection allows ManyChat to send and receive messages through your Telegram bot. The entire setup takes about 5 minutes and requires no technical knowledge.

Creating Automated Response Flows

With ManyChat connected, we'll build two automated flows:

1. Return Policy Flow

Triggers when users ask about returns. Provides a predefined answer about your policy.

2. Return Status Flow

Asks for the customer's email and order number, then connects them to human support.

Important: Always include an escalation path to human support. About 15% of cases will require personal attention.
