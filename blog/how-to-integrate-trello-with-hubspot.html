Zapier Setup Overview

Zapier acts as the bridge between HubSpot and Trello, automatically passing data between them whenever specific events occur. The setup involves three key components:

  1. Trigger: What HubSpot event starts the automation (new contact, deal, etc.)
  2. Action: What happens in Trello (create card, update card, etc.)
  3. Field Mapping: Which HubSpot data goes where in Trello

Unlike API coding, Zapier provides a visual interface that lets non-technical teams build these automations in minutes. The platform handles all the technical complexity behind the scenes.

Step 1: Choose Your HubSpot Trigger

Your first decision is which HubSpot event should create Trello cards. The four most powerful options:

1. New Contact

Ideal for: Sales teams wanting immediate visibility on inbound leads. Creates a card whenever a new contact enters your CRM.

2. New Deal

Ideal for: Opportunity tracking. Automatically creates cards for new sales pipeline items with deal amount and stage.

3. Form Submission

Ideal for: Marketing-to-sales handoffs. Turns website form fills (demo requests, content downloads) into actionable Trello cards.

4. Property Change

Ideal for: Advanced workflows. Triggers when specific fields update (e.g., deal stage changes from Prospecting to Negotiation).

Pro Tip: Start with "New Contact" for simplicity, then add additional zaps for other triggers as your workflow matures.

Step 2: Connect Trello Action

After setting the HubSpot trigger, you'll configure what happens in Trello. While "Create Card" is the most common action, Zapier offers several options:

Create Card

The foundation action. Generates a new Trello card in your specified board/list whenever the HubSpot trigger occurs.

Update Card

Modifies existing cards when HubSpot data changes (e.g., updates deal amount when it increases).

Add Checklist Item

Builds out multi-step processes (e.g., adds "Schedule Discovery Call" to new lead cards).

Add Comment

Logs HubSpot activity (e.g., "Contact opened pricing email") directly on relevant cards.

For your first integration, stick with "Create Card" to keep things simple. You can always add more sophisticated actions later.

Step 3: Field Mapping Strategies

Field mapping determines which HubSpot data appears where in your Trello cards. Smart mapping transforms basic cards into powerful sales tools.

Essential Mappings

  • Card Title: Contact name + company (e.g., "John Doe - Acme Inc")
  • Description: Email, phone, deal amount, and notes
  • Labels: Deal stage or lead source for visual scanning

Advanced Mappings

  • Due Date: Set based on "Next Activity Date" in HubSpot
  • Checklists: Pre-populate onboarding steps for new customers
  • Attachments: Automatically add HubSpot contact profile links

Time-Saver: Use HubSpot contact tags to automatically set Trello labels. For example, tag "High Priority" contacts in HubSpot to give them red labels in Trello.

Advanced Automation Tips

Once you've mastered basic integration, these pro techniques unlock even more value:

Conditional Paths

Use Zapier's "Paths" feature to create if/then logic. Example: IF deal amount > $10,000 THEN create card in "Enterprise Deals" list ELSE use "SMB Deals" list.

Multi-Step Zaps

Chain actions together. Example: 1) Create Trello card, 2) Add checklist items based on lead source, 3) Assign to appropriate team member.

Two-Way Sync

Create complementary zaps that update HubSpot when Trello cards move (e.g., change deal stage when card moves to "Closed Won" list).

These advanced workflows require more setup but can transform your sales process from reactive to proactive.
