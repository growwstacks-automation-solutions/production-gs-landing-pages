The traditional solutions - manually scrubbing through videos or relying on auto-generated captions - are time-consuming and ineffective. That's why we built this automated summarization workflow using n8n, combining the power of Apify's transcript extraction with OpenAI's summarization capabilities.

Key insight: 72% of YouTube viewers abandon videos before reaching the key information point. This workflow surfaces those insights immediately.

How the Summarization Workflow Works

The workflow follows a simple but powerful sequence:

  1. You paste any YouTube URL into the n8n form
  2. Apify's YouTube Transcript Scraper extracts the full text
  3. OpenAI processes the transcript into a concise summary
  4. The formatted result is returned via email, Slack, or saved to your database

What makes this particularly valuable is the workflow's ability to handle videos of any length automatically. Unlike manual methods that require you to watch and take notes, this system processes hours of content in seconds with consistent quality.

Step-by-Step Setup Guide

Step 1: Import the Workflow Template

Begin by importing the JSON workflow template into your n8n instance. The template includes pre-configured nodes for:

  • YouTube URL input form
  • Apify API connection
  • OpenAI summarization
  • Output formatting

Step 2: Configure Apify Credentials

Create a free Apify account and locate the YouTube Transcript Scraper actor. Copy your unique API endpoint URL from the Apify dashboard - this includes your authentication token. Paste this into the designated HTTP Request node in n8n.

Step 3: Connect OpenAI

In the OpenAI node, add your API key. We recommend using GPT-4 for best results, though GPT-3.5 works for simpler videos. Set your preferred summary length and style parameters.

Step 4: Test and Activate

Run a test with a short YouTube video URL. Verify you receive both the raw transcript and formatted summary. Once confirmed, activate the workflow for ongoing use.

Our Testing Results
