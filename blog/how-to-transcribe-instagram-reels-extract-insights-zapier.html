How the Automation Works

This Zapier workflow acts as your personal content strategist - watching Reels so you don't have to. The entire process happens in 4 stages with zero manual intervention after setup:

  1. Trigger: Paste a Reel URL into your designated Slack channel
  2. Transcription: Apify extracts the video's text content (costs $2/month)
  3. Analysis: AI processes the transcript through your custom business lens
  4. Delivery: Cleanly formatted insights arrive back in Slack within minutes

Step 1: Slack Trigger Setup

The automation begins when you paste a Reel URL into Slack - the digital equivalent of handing a document to your assistant. Here's how to configure this starting point:

Step 1.1: Create Private Slack Channel

Set up a dedicated channel (we recommend naming it #reel-insights). This keeps all automation traffic separate from team communications.

Step 1.2: Configure Zapier Trigger

In Zapier, create a new Zap with:

  • Trigger: Slack → New Message in Channel
  • Channel: Select your #reel-insights channel
  • Trigger for Bot Messages: No

Pro Tip: At 6:45 in the video, notice how they test the trigger with sample URLs before proceeding. This catches 80% of setup errors early.

Step 2: Apify Instagram Transcription

Apify handles the heavy lifting of extracting text from Reels - a task that would normally require manual work or expensive APIs. Configuration requires:

Step 2.1: Apify Account Setup

1. Create free Apify account
2. Search store for "Instagram Transcript"
3. Add the $2/month transcription actor

Step 2.2: Connect to Zapier

In your Zap:
1. Add Action: Apify → Run Actor
2. Select your Instagram transcript actor
3. Map Instagram URL from Slack trigger
4. Set Run Synchronously: Yes

The video at 9:20 shows how to troubleshoot common API key issues during this step.

Step 3: AI Insight Generation

This is where generic transcription becomes valuable business intelligence. The AI step requires careful prompt engineering:

Step 3.1: Connect OpenAI

1. Add Action: AI by Zapier → Create Completion
2. Model: GPT-4 (or Claude if preferred)
3. Paste your OpenAI API key

Step 3.2: Craft Your Business-Specific Prompt

The magic happens in your instructions to the AI. Effective prompts include:

  • Your company's specific goals
  • Content formats you create
  • Metrics that matter to you
  • Examples of ideal outputs

At 12:30, the creator shares their exact prompt template that generates customized action items.

Step 4: Slack Output Configuration

The final step delivers polished insights back to Slack in an organized thread. Key configuration points:

Step 4.1: Transcript Message

1. Add Action: Slack → Send Channel Message
2. Channel: Same as trigger channel
3. Message Text: Apify transcript output
4. Reply in Thread: Use TS value from trigger

Step 4.2: Action Items Message

Duplicate the previous step, but:
1. Use AI output as Message Text
2. Add "Action Items:" header
3. Same thread TS value

The video at 15:00 demonstrates how threaded replies keep everything organized as you process multiple Reels.
