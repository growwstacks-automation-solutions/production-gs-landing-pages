The Image-to-Video Content Problem

Businesses today face a critical content gap - static product images dominate e-commerce, but video content drives 3x more engagement. The challenge? Professional video production requires hours of editing per clip, specialized software, and creative skills most teams lack.

This n8n workflow solves that problem by automating video creation from existing product images. Using AI-powered contact sheet prompting, it generates natural-looking animations showing products from multiple angles - all without manual editing.

Complete Workflow Overview

The system takes a simple product image as input and produces a professional marketing video ready for social media distribution. Here's the high-level process:

  1. Image Input: Pulls product images from Google Sheets or Drive
  2. Frame Generation: Creates 6 variations using contact sheet prompting
  3. Video Creation: Animates frames together using Klink API
  4. Enhancement: Upscales to 4K/8K with Nano Banana Ultra
  5. Distribution: Publishes to multiple social platforms via Bluetato

Image Processing with Contact Sheet Prompting

Contact sheet prompting is the secret sauce that makes this workflow powerful. Instead of manually creating multiple product angles, the AI generates them automatically from a single image.

The process:

  1. Upload source image to temporary hosting
  2. Send to Nano Banana Ultra with specific prompt
  3. Receive 6-frame contact sheet output
  4. Crop into individual frames using n8n image node

Video Generation with AI APIs

With our six frames ready, we now animate them into a seamless video sequence. The Klink API handles the animation while Nano Banana Ultra ensures high-resolution output.

Configuration highlights:

  • HTTP request node formatted for Klink API

  • Quality settings for 4K/8K output
  • Automatic saving to Google Drive

Note: The workflow includes error handling for API limits and retry logic for large file processing.
