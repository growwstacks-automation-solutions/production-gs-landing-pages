CGI Ad Workflow Overview

The complete automation processes product images through four intelligent stages:

1. Image Analysis

OpenAI's vision model examines the product image, identifying materials, shapes, and key features to inform the CGI process.

2. Description Generation

The AI creates a detailed product description highlighting selling points and technical specifications.

3. CGI Render Creation

RunwayML generates a high-quality 3D render of the product based on the analysis and description.

4. Video Compilation

The final stage produces a cinematic video with dynamic camera movements, lighting effects, and optional voiceover.

The entire process happens automatically after uploading a product image to your connected Dropbox folder.

Step 1: Dropbox Integration

Dropbox serves as the input channel for product images. The workflow monitors a specific folder, automatically processing new uploads.

Configuration Steps:

  1. Create a Dropbox developer account
  2. Generate an API key with file read/write permissions
  3. Add the key to three nodes in the n8n workflow

This connection enables the workflow to retrieve uploaded images and store final videos, creating direct download links for each completed ad.

Step 2: OpenAI Configuration

Four specialized OpenAI nodes power the intelligence behind the CGI generation:

Pro Tip: Use GPT-4 Vision for image analysis as it provides more accurate product understanding than standard models.

To connect:

  1. Create an OpenAI account with payment method
  2. Generate an API key in the developer dashboard
  3. Add the key to each OpenAI node in the workflow

The workflow uses approximately $0.02 worth of OpenAI credits per product analyzed - negligible compared to traditional CGI costs.

Step 3: RunwayML Connection

RunwayML handles the actual video generation with its advanced AI models. You'll need:

  1. A RunwayML account with payment method added
  2. At least $20 in credit (produces ~33 videos)
  3. The API key added to three workflow nodes

The workflow automatically selects optimal video styles based on your product category, but you can customize parameters for specific looks like:

  • 360-degree product spins
  • Cinematic reveal animations
  • Lifestyle context shots
