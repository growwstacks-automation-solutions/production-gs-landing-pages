Workflow Breakdown: How It Works

The automation handles video processing through a smart sequence:

Step 1: File Upload

The workflow generates a pre-signed URL for secure cloud storage upload, capturing all video metadata.

Step 2: Processing Trigger

n8n sends the video details to FFmpeg Micro's API with your specified editing parameters (trim points, overlay text, etc.).

Step 3: Status Polling

The workflow automatically checks processing status every 10 seconds until completion.

Step 4: Output Delivery

Finished videos are either downloaded or posted directly posted to destinations like YouTube or social platforms.

Typical processing time: Most edits complete in 20-40 seconds, though complex operations may take longer. The workflow handles all waiting automatically.
