Step 1: Getting Your Google API Keys

The only technical hurdle is obtaining your Google API credentials. Here's the foolproof method:

Step 1: Create a Google Cloud Project

Navigate to aistudio.google.com/app/apikey and click "Create Project." Name it something memorable like "NanoBanana-n8n."

Step 2: Generate API Key

Under "Credentials," click "Create Credentials" and select API Key. Copy this key immediately - you won't be able to view it again later.

Step 3: Enable Gemini API

Search for "Gemini API" in Google Cloud Console and enable it for your project. This gives access to NanoBanana Pro through the backdoor.

Security Note: Never commit API keys to public repositories. n8n's credential system stores them encrypted.

AI Image Generation: Prompt Engineering Tips

NanoBanana Pro responds exceptionally well to structured prompts. Here's the formula we've refined through hundreds of generations:

Prompt Structure: "[Age] [Gender] [Description] [Setting] [Style]"

Example from the tutorial: "23 year old female wearing a large sun hat shopping at a boutique dress store, photorealistic style"

Key observations:

  • Including age produces more realistic proportions
  • Specifying gender helps with clothing and hairstyles
  • Adding "photorealistic" prevents cartoonish outputs

The workflow includes a prompt template with these fields pre-configured as variables you can swap via n8n's UI.

Editing Existing Images Like a Pro

Where NanoBanana Pro truly shines is editing existing photos. The tutorial demonstrates changing a Yankees cap to a Red Sox cap while perfectly maintaining:

  • Lighting and shadows on the face

The secret is in the edit prompt structure:

  1. Upload the original image
  2. Describe the change precisely: "Change the baseball cap from Yankees to Red Sox"
  3. Specify what to preserve: "Keep all other clothing and background identical"

This works wonders for ecommerce - generate one base product image, then automatically create variants with different colors or accessories.
