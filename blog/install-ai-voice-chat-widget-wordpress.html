Step 1: Call Agent AI Dashboard Setup

Begin by logging into your Call Agent AI dashboard (if you don't have an account yet, you'll need to sign up first). The entire setup process takes place in two locations: the Call Agent AI web dashboard and your WordPress admin panel.

Inside the dashboard, create or select an existing AI assistant. Each assistant can be specialized for different functions - you might have one for sales inquiries and another for technical support. Navigate to the "Plugin Setup" tab where you'll find three critical pieces of information:

  1. User API Token: This authenticates your WordPress site
  2. Assistant ID: Tells the widget which AI to use
  3. Allowed Domains: Security measure restricting where the widget loads

At 1:15 in the video tutorial, you'll see exactly where to locate these credentials. The most common mistake at this stage is forgetting to add your website domain to the allowed list - without this, the widget won't appear even if properly installed.

Step 2: WordPress Plugin Installation

With your Call Agent AI credentials ready, switch to your WordPress admin dashboard. The official plugin is listed in the WordPress repository, making installation straightforward:

  1. Navigate to Plugins → Add New
  2. Search for "Call Agent AI"
  3. Click Install Now then Activate

Once activated, you'll find a new "Call Agent AI" menu item under Settings. This is where you'll paste the API Token and Assistant ID from the previous step. The plugin validates these credentials immediately - if you see a green checkmark, your connection is successful.

Pro Tip: Create a staging site first if you're implementing on a live business website. While the plugin is stable, testing changes in a non-production environment prevents any temporary glitches from affecting real customers.

Step 3: Widget Configuration Options

The Call Agent AI plugin offers two widget styles with different use cases:

  • Chat & Voice Widget: Persistent button in the corner supporting both text and voice
  • Voice Button Widget: Centered call-only button with a 7-minute cool down between calls

For most businesses, the combined chat and voice widget provides the best balance of accessibility and functionality. When configuring this option, you'll choose:

  1. Position: Bottom left or right corner (right typically converts better)
  2. Interaction Mode: Chat only, voice only, or both
  3. Initial State: Whether the widget starts open or minimized
  4. Branding Colors: Match your website's color scheme

At 3:42 in the video, you can see how these settings appear in the WordPress admin panel. The changes preview instantly, letting you experiment with different configurations before going live.

Step 4: Strategic Widget Placement

One of the plugin's most powerful features is flexible deployment options. Unlike many solutions that force you to put widgets on every page, Call Agent AI gives you precise control:

  • Global Mode: Widget appears sitewide (ideal for support/sales sites)
  • Shortcode Mode: Place [callagentai] only on high-value pages

For eCommerce sites, we recommend placing the widget on product pages and the checkout flow but excluding blog posts. Service businesses might enable it globally except for administrative pages. The shortcode can even be placed within specific content sections using WordPress blocks.

Conversion Tip: Add a custom greeting message that changes based on page context. For example: "Ask me about this product's availability" on PDPs versus "Need help with your order?" in the cart. This contextual approach increases engagement by 27%.

Step 5: Testing & Optimization

Before announcing your new AI capabilities, thoroughly test all widget functions:

  1. Verify voice calls connect properly (use different devices)
  2. Test chat responses for accuracy
  3. Check widget behavior on mobile vs desktop
  4. Confirm lead data flows to your CRM if integrated

The Call Agent AI dashboard provides conversation logs and analytics to monitor performance. Look for:

  • Resolution Rate: Percentage of inquiries handled without escalation
  • Average Handle Time: How long conversations typically last
  • Intent Analysis: What customers are asking about most

Use these insights to continuously refine your AI assistant's knowledge base and response patterns. The system learns from every interaction, becoming more effective over time.
