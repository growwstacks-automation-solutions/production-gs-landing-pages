Revolutionary Design View Feature

The new design view transforms how users interact with AI-generated images. Rather than relying solely on text prompts, this interactive canvas allows precise editing with point-and-click controls powered by cutting-edge models like Nano Banana 2.

Key capabilities include making local changes to specific image sections, adding or modifying text directly on images with high-quality rendering, and combining multiple images to create complex designs. In the Flappy Bird example, the user could highlight the bird and simply request "add a smile" to modify the design instantly.

Professional-grade editing: The design view provides fine-grained control comparable to traditional design software while leveraging AI's creative capabilities. This eliminates the frustration of regenerating entire images for minor changes.

Comparing the Different Versions

Manus AI offers three tiers to suit different needs and budgets:

1.6 Max

The flagship version for complex tasks requiring extended reasoning. It uses more credits but delivers higher accuracy, especially for multi-step projects like competitor analysis or feature comparison matrices.

1.6

The versatile general-purpose version suitable for most tasks. Available to Pro plan subscribers, it balances capability with resource efficiency.

1.6 Light

The free version that still handles basic tasks effectively. While limited compared to paid tiers, it's excellent for quick experiments and simple automation.

Flexible switching: Users can change between tiers as needed, allowing them to use Max for critical projects and lighter versions for simpler tasks.
