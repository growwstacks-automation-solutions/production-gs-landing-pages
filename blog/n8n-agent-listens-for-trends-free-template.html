How The Automated Trend Agent Works

The system combines three monitoring streams into one centralized dashboard. A scheduled n8n workflow runs daily (or hourly) to collect, analyze, and organize findings from Twitter, email newsletters, and target websites.

At 6:00 AM each morning, the workflow automatically:

  1. Scans Twitter for recent posts matching your configured keywords
  2. Processes incoming newsletters from your Gmail filters
  3. Scrapes updates from specified websites using browser automation
  4. Compiles all findings into a Google Sheets database with source attribution

The magic happens in the analysis phase - Claude AI processes the raw data to extract key insights while filtering out noise. This creates a digestible daily briefing of what actually matters in your industry.

Twitter Monitoring Setup

Twitter moves fastest for emerging trends, but manual monitoring misses crucial context. The workflow uses Twitter's API to track:

  • Keyword mentions from industry leaders
  • Hashtag trends in your niche
  • Threads with high engagement

Configuration involves three key steps:

  1. API Connection: Create a Twitter Developer account and generate API keys
  2. Query Setup: Define your search terms (e.g. "AI funding" OR "Series A" near:"San Francisco")
  3. Metadata Selection: Choose which tweet data to capture (username, text, URLs, timestamps)

Pro Tip: Set the "max_results" parameter to 20 initially - enough for daily monitoring without hitting rate limits.

Newsletter Processing

Industry newsletters contain curated insights - but inbox overload means most go unread. The system automatically:

  • Filters incoming newsletters by sender (e.g. @futuretools.io)
  • Extracts key content while stripping formatting
  • Identifies emerging topics across multiple sources

The Gmail node uses regular expressions to target specific senders while excluding irrelevant emails. For example:

 from:(futuretools.io|ben-evans.com|stratechery.com)

Each processed newsletter includes the sender, date, and cleaned text content in your trend database for easy reference.

Website Scraping Configuration

Direct website monitoring catches announcements before they hit social media or newsletters. The workflow uses Airtop.ai's browser automation to:

  1. Load target websites (e.g. openai.com/news)
  2. Extract new posts from the last 24 hours
  3. Capture headlines, URLs, and publication dates

Airtop's visual selector makes it easy to pinpoint which page elements to monitor. The system can track:

  • Blog post listings
  • News sections
  • Product update pages

Note: Start with 10,000 Airtop credits (about $10) - enough for months of daily monitoring before needing to refill.

Data Consolidation & Analysis

Raw data becomes powerful when organized and analyzed. The workflow:

  1. Deduplicates findings across sources
  2. Tags each entry with its origin (Twitter, newsletter, website)
  3. Passes content through Claude AI for insight extraction

The Google Sheets output includes:

  • Short name/title of each trend
  • Detailed description
  • Original source link
  • Date detected

This structure lets you quickly spot:

  • Which topics are gaining momentum
  • How stories develop across platforms
  • Emerging patterns in your industry
