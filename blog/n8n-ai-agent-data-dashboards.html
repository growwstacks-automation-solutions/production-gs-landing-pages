Step 3: Configure the AI Agent

The magic happens in the AI agent configuration (shown at 11:08):

  1. Memory: Add simple memory so the agent remembers past interactions
  2. Streaming: Enable response streaming for real-time updates
  3. Public Access: Make the chat publicly available for dashboard sharing

This creates an agent that:

  • Understands context from previous questions
  • Updates visualizations in real-time
  • Can be shared with team members without n8n access
