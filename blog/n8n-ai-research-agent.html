Setup Requirements: The 3 Services You Need

This automation connects three powerful services through n8n:

  1. Telegram - Acts as the user interface where you send queries and receive reports
  2. SERP API - Provides real-time Google search results without browser limitations
  3. OpenAI - Analyzes search results and generates structured reports

Each requires an API key (essentially a password that lets n8n access the service). The good news? All three offer free tiers suitable for testing:

Free tier availability: Telegram bots are always free, SERP API offers 100 free searches/month, and OpenAI provides $5 in free credit - enough to test this workflow thoroughly before any costs.

Step 1: Creating the Telegram Bot Interface

We'll use Telegram as our input/output channel because it's simple, secure, and available on all devices. Creating a bot takes 2 minutes:

  1. Search for "@BotFather" in Telegram
  2. Send "/newbot" and follow the prompts
  3. Copy the API token provided

In n8n, we add a Telegram trigger node that watches for new messages to this bot. When you send a research question, it triggers the workflow and passes your query to the next steps.

Pro tip: Name your bot something descriptive like "ResearchAssistantBot" so team members understand its purpose at a glance.

Step 2: Building the AI Agent Core

This is where the magic happens. Unlike a standard ChatGPT conversation, an AI agent in n8n can decide when to search for information and when to answer from its training.

We add an "AI Agent" node with two critical components:

  1. User Prompt: Mapped from the Telegram message so each query becomes a fresh research task
  2. System Message: The instruction manual that transforms GPT from a general chatbot into a research specialist

The system message does three key things:

  • Defines the agent's role ("You are a research assistant...")
  • Specifies when to use Google searches vs. existing knowledge
  • Formats responses for clarity in Telegram (sections, emoji, sources)

Step 3: Adding SERP API for Live Data

For current information, we connect SERP API (Search Engine Results Page API). This lets the agent perform real Google searches without browser limitations.

After getting a free API key from serpapi.com, we:

  1. Add a SERP API node in n8n
  2. Connect it to our AI agent
  3. Set the search query to use the user's question automatically

Now when the agent determines it needs fresh data (like for " e-commerce trends"), it fetches the latest results before analyzing them.

Why SERP API? It provides clean, structured search results without ads or JavaScript clutter, and handles CAPTCHAs and rate limiting automatically - crucial for reliable automation.

Step 4: Sending the Analyzed Response

The final step sends the AI's structured report back to Telegram. We:

  1. Add a "Send Message" Telegram node
  2. Map the chat ID from the initial trigger (so replies go to the right conversation)
  3. Connect the AI agent's output as the message content

The result? A complete loop where your question goes in, and a researched, analyzed answer comes back - all automated through n8n.
