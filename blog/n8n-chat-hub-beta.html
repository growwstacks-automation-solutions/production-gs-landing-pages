What is Chat Hub in n8n?

Chat Hub is n8n's revolutionary new interface that transforms your automation workflows into conversational AI experiences. Accessible from the sidebar in version 2.5 and above, it provides a familiar chat interface where you can interact with different types of AI agents.

The magic of Chat Hub lies in how it bridges the gap between complex workflow automation and simple chat interfaces. Where previously you needed technical skills to build chatbot workflows, now you can create and manage conversational agents through an intuitive UI.

Key benefit: Chat Hub lets non-technical users benefit from n8n's powerful automation capabilities through a simple chat interface, while giving developers the full power of workflows under the hood.

Personal Agents vs Workflow Agents

Chat Hub offers two distinct types of agents, each serving different needs:

Personal Agents are designed for simplicity. You can create one in minutes by just selecting a model and adding a system prompt that defines the agent's purpose and personality. These are perfect for individual assistants, basic customer support bots, or simple Q&A applications.

Workflow Agents unlock n8n's full potential. These are based on actual workflows that include a chat trigger and agent node with model and memory. They can incorporate complex logic, multiple AI steps, human-in-the-loop approvals, and integration with any of n8n's 400+ app connections.

Pro tip: Start with a personal agent for quick wins, then graduate to workflow agents when you need more sophisticated behavior or external integrations.

Creating Your First Chat Agent

Creating a personal agent is straightforward. From Chat Hub, click "New Chat" and select "Create Personal Agent" from the left menu. Choose your preferred AI model (after connecting your API keys in settings), then craft a system prompt that defines your agent's role and behavior.

For workflow agents, the process involves:

  1. Creating a new workflow with a Chat Trigger (make sure to enable "Available in Chat Hub")
  2. Adding an Agent node with your chosen model and memory settings
  3. Publishing the workflow to make it available in Chat Hub

The workflow approach means your chat agent can include any additional logic or integrations you need - from database lookups to API calls to human approval steps.

Powerful Debugging Features

One of Chat Hub's most valuable features is its debugging capability. Unlike traditional chatbots where conversations disappear into a black box, every message in a workflow agent is tied to a specific workflow execution.

By clicking the info button next to any message, you're taken directly to that execution in the workflow editor. This transparency makes it incredibly easy to:

  • See exactly what data was processed at each step
  • Identify where responses might be going wrong
  • Understand how the AI is interpreting inputs

This level of visibility is rare in chatbot platforms and makes n8n particularly powerful for developing reliable, production-grade conversational agents.

Team Collaboration with Chat-Only Users

Chat Hub introduces a new "chat-only user" type that changes how teams can collaborate on AI agents. This feature allows you to share workflow agents with team members who only need chat access, without exposing the underlying workflows.

For these users, n8n appears as a simple chat interface similar to ChatGPT. They can interact with the agents but don't see or need to understand the workflow logic behind them. This is particularly valuable for:

  • Customer support teams using pre-built agent workflows
  • Executives who need data from business systems without technical complexity
  • Departments sharing specialized knowledge agents

Note: While Chat Hub itself is free in the community edition, chat-only users are only available on n8n cloud business and enterprise plans.

Future Features Coming to Chat Hub

The n8n team has an exciting roadmap for Chat Hub that will make it even more powerful. Some of the planned enhancements include:

  • Knowledge management - Built-in RAG (retrieval augmented generation) capabilities
  • Chat artifacts - Special formats for running code or returning structured data
  • Mobile optimization - Better experience for using agents on smartphones
  • Prompt management - Tools for versioning and testing system prompts

These additions will position Chat Hub as a complete platform for building and managing all types of conversational AI applications, from simple assistants to complex business automation agents.
