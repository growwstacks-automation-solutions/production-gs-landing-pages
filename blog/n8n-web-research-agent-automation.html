How the Research Agent Works

The workflow follows a clear decision tree for each query received through Telegram (though it could connect to Slack, email, or webhooks):

Step 1: Query Analysis

The agent classifies incoming questions by type (fact-check, research, synthesis) and determines which sources to consult.

Step 2: Source Selection

Based on the query type, the agent routes to appropriate APIs - avoiding unnecessary API calls for simple questions.

Step 3: Information Retrieval

The agent gathers data from selected sources, with timeouts to prevent endless searching.

Step 4: Answer Synthesis

Retrieved information gets formatted according to predefined business rules about length, citations, and structure.

The entire process typically completes in under 2 minutes - compared to 15-60 minutes for manual research. For recurring queries (like daily competitor checks), the agent can be scheduled to run automatically.
