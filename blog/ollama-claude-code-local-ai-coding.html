The 3-Step Ollama + Claude Code Setup Process

Setting up this powerful combination requires just three straightforward steps that even non-technical users can follow:

Step 1: Install Ollama

Download and install Ollama for your operating system (Windows, Mac, or Linux). The lightweight application runs in the background and manages your local models.

Step 2: Configure Claude Code

Install Claude Code if you haven't already. The configuration involves exporting Ollama's environment variables to point Claude Code at your local models instead of the cloud API.

Step 3: Select Your Model

Choose from Ollama's library of open-source models. The tutorial demonstrates using GPT OSS 20B cloud model, but you can also run smaller models like Gemma 4B locally for even faster performance.

Pro Tip: The terminal command claude-model oss-20b-cloud switches between models instantly—you'll see this in action at 4:30 in the video.
