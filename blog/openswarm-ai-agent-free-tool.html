60-Second Installation Guide

Installing OpenSwarm requires just two terminal commands if you have NodeJS 20+:

 npm install -g openswarm swarm init

The setup wizard handles dependencies automatically, including Python if needed. You'll need either an OpenAI or Anthropic API key for core functionality, with optional keys for advanced features.
