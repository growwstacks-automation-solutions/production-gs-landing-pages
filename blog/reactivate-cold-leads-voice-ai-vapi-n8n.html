Step 2: Building the n8n Automation Workflow

The n8n workflow (shown at 7:45) handles four critical functions:

  1. Trigger: Runs daily to identify leads needing follow-up
  2. CRM Query: Pulls lead details from Airtable/Salesforce
  3. VAPI API Call: Initiates personalized calls with dynamic data
  4. Result Logging: Saves call outcomes back to your CRM

At 9:20, notice how the workflow filters for leads marked "No" in the "FollowedUp" column. This simple logic ensures each lead gets exactly one reactivation attempt.

Step 3: Connecting to Your CRM Data

The magic happens when VAPI personalizes calls using your CRM data. The tutorial uses Airtable (at 11:05), but the same principles apply to any CRM:

  • Required Fields: Name, phone, initial inquiry details, follow-up status
  • API Access: Ensure your token has read/write permissions
  • Data Mapping: Match VAPI's expected JSON structure to your CRM fields

At 12:30, watch how the n8n HTTP request node transforms Airtable data into VAPI's call parameters - making each conversation uniquely relevant to the lead.
