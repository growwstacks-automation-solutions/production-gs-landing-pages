Step 1: Google Cloud Setup (5 Minutes)

Before connecting to n8n, you'll need to enable two Google Cloud APIs and create credentials. This one-time setup takes less than 5 minutes:

Enable Required APIs

  1. Go to Google Cloud Console > APIs & Services > Library
  2. Search for "YouTube Data API v3" and enable it
  3. Search for "Gemini API" and enable it

Create API Key

  1. Navigate to APIs & Services > Credentials
  2. Click "Create Credentials" > API Key
  3. Restrict the key to your n8n IP if running cloud

Security note: For production use, always restrict API keys to specific IP addresses and enable application restrictions. The video tutorial shows the exact settings at 3:42.

Step 2: n8n Configuration (7 Minutes)

With Google Cloud setup complete, the n8n configuration involves just three main steps:

1. Credentials Node

Add your Google API key to a credentials node in n8n. The workflow needs this to authenticate both YouTube data access and Gemini analysis.

2. YouTube Analysis Node

Configure the YouTube node to extract video metadata and frames for analysis. The workflow automatically handles the optimal frame sampling rate.

3. Gemini Prompt Reconstruction

The magic happens in the Gemini node where we've pre-configured the prompt analysis template. No changes needed here unless customizing the output format.
