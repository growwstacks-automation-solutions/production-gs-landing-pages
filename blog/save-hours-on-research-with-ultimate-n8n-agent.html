How the Research Agent Works

The automation follows a streamlined three-step process that mimics how expert researchers work, but at machine speed:

Step 1: Question Input via Messaging

Users submit research questions through Telegram (or other connected platforms). The system captures the exact query and prepares it for processing.

Step 2: Deep Research with Perplexity AI

The Perplexity AI agent conducts comprehensive research, focusing specifically on extracting quantitative data, statistics, and verifiable facts from reliable publications.

Step 3: Professional Formatting

A second AI agent transforms the raw research into clear, structured responses with proper citations - making complex information easily digestible.

Key advantage: Unlike generic AI responses, this system always provides sources so you can verify findings. At 2:15 in the video, you'll see how the formatted response includes percentage data with citations.

Setting Up Superchat for Messaging

Superchat serves as the communication hub for this automation, providing an all-in-one inbox for Telegram, WhatsApp, Messenger and other channels. Here's how to configure it:

Step 1: Create Superchat Account

Sign up for Superchat (free trial available) and connect your preferred messaging platforms. The system consolidates all conversations in one interface.

Step 2: Generate API Key

Navigate to Settings → Integrations in Superchat to create your API key. This allows n8n to send and receive messages through your connected channels.

Step 3: Configure n8n Trigger

Set up the Superchat trigger node in n8n to watch for incoming messages. The demo uses Telegram, but you can configure any supported platform.

Pro tip: At 4:30 in the video, you'll see how Superchat's unified inbox makes it easy to manage research requests from multiple channels simultaneously.

Configuring Perplexity AI for Deep Research

Perplexity AI provides the research engine for this automation. Follow these steps to set it up:

Step 1: Get API Key

Log into Perplexity, go to Account → API Keys, and generate a new key. This authenticates your n8n workflow with Perplexity's research models.

Step 2: Choose Research Model

Select between Sonar Pro (faster) or Sonar Deep Research (more thorough) based on your needs. The demo shows Sonar Pro providing excellent results in most cases.

Step 3: Configure System Prompt

The critical component is the system message that instructs the AI to focus on evidence-based research. The template includes this pre-configured:

 "You're a deep research agent focused on producing evidence-based insights... return only hard facts such as statistics, percentages, numerical findings..."

Accuracy note: Using proper system prompts improves result accuracy by 40-50% compared to generic queries. At 7:15 in the video, see how the prompt ensures cited, quantitative responses.

The Secret Sauce: Response Formatting Agent

Raw research output often needs refinement to be truly useful. This is where the formatting agent adds tremendous value:

Step 1: Receive Raw Research

The formatting agent takes the Perplexity output - typically dense with citations and data - and analyzes its structure.

Step 2: Apply Formatting Rules

Using predefined instructions, it breaks long paragraphs into bullet points, highlights key statistics, and ensures proper source attribution.

Step 3: Deliver Polished Response

The final output presents complex information in a clear, scannable format that's ready to share with colleagues or clients.

Before/after example: At 9:45 in the video, compare the raw Perplexity output with the formatted version - the difference in readability is dramatic.

Automated Tracking with Google Sheets

The workflow includes automatic logging to Google Sheets for record-keeping and analysis:

Step 1: Copy Template Sheet

Use the provided Google Sheets template (available in the free resources) that includes columns for questions, answers, dates, and sources.

Step 2: Connect to n8n

Configure the Google Sheets node in n8n with your sheet ID and proper authentication.

Step 3: Map Data Fields

Ensure the question, formatted answer, timestamp, and other relevant data flow correctly into your sheet.

Analytics benefit: Over time, your sheet becomes a searchable knowledge base of all research conducted, saving even more time on recurring questions.

Complete n8n Workflow Walkthrough

Here's the end-to-end flow of the research agent automation:

  1. Trigger: Message received via Superchat (Telegram/WhatsApp/etc.)
