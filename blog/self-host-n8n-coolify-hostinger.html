Step 3: DNS Configuration (1 Minute)

This is the only slightly technical step, but we've made it foolproof. At 8:20 in the tutorial, we configure DNS records to give your n8n instance a professional web address (like n8n.yourdomain.com):

  1. In your domain registrar's DNS settings, create two A records:
    • cloud.yourdomain.com → [Your server IP]
    • *.apps.yourdomain.com → [Your server IP]
  2. Set TTL to 600 seconds
  3. Save changes (may take 5-10 minutes to propagate)

This wildcard DNS setup means Coolify can automatically create subdomains for every app you deploy later - no additional DNS changes needed.

Step 4: Deploy n8n (3 Minutes)

With DNS configured, deploying n8n is just a few clicks in Coolify (demonstrated at 10:05 in the video):

  1. Create a new project (name it something like "n8n Automation")
  2. Search the marketplace for "n8n"
  3. Select the official n8n Docker image (community edition)
  4. Leave most settings default, but set the service name to "n8n"
  5. Click "Deploy"

Coolify handles all the complex Docker setup, network configuration, and SSL certificate generation automatically. Within 2-3 minutes, your private n8n instance will be live at n8n.apps.yourdomain.com with HTTPS already configured.
