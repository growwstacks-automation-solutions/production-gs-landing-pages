Step-by-Step Setup Guide

Setting up n8n on Hostinger takes about 10 minutes. Here's the complete process:

Step 1: Create Your Hostinger Account

Visit Hostinger.com and sign up for a VPS hosting account. The interface walks you through:

  • Selecting your server location (choose closest to your users)
  • Picking Ubuntu as your OS
  • Setting a secure root password

Step 2: Deploy n8n via Docker

Hostinger's one-click Docker deployment makes this simple:

  • Navigate to VPS → Docker Manager
  • Search for "n8n" in the developer tools section
  • Click deploy and wait 2-3 minutes for installation

In summary: Sign up → Choose VPS plan → Select Ubuntu → Deploy n8n via Docker → Access your unlimited automation environment.

How to Update Your n8n Instance

Unlike cloud hosting, self-hosted n8n requires manual updates. The process is simple:

  1. Check for updates in n8n's web interface (notification appears near the help icon)
  2. Log into your Hostinger VPS dashboard
  3. Navigate to Docker Manager → n8n container
  4. Click the three-dot menu and select "Update"

The video demonstrates this process at 5:45, showing how updates complete in 2-3 minutes without affecting running workflows.
