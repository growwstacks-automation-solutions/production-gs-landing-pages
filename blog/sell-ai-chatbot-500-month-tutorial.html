Tools You'll Need (All Free to Start)

We'll use three main tools in this tutorial, all with free tiers that let you build your first chatbot at no cost:

  1. ConvaCode (formerly TextAgents) - The chatbot builder platform (free for basic use)
  2. Vapi - For adding voice AI capabilities (free for testing)
  3. Sitemap Generator - To import product data automatically (free online tools)

At 6:45 in the video, I show how to sign up for ConvaCode - you can use Google authentication for instant access. The platform has evolved since its TextAgents days, with a more intuitive interface that makes building complex chatbots accessible to non-developers.

Step 1: Setting Up Your Text Agent

Follow along starting at 7:20 as we create a new agent:

  1. Click "New Agent" in ConvaCode
  2. Name it (we used "Go Zero AI Agent")
  3. Select "Blank Template" (avoid the presets)
  4. Enable tabs under Widgets → Dark Configurations

Pro Tip: Enabling tabs (shown at 8:15) dramatically improves the user experience by organizing the chatbot's responses into clear sections. This small tweak makes your chatbot look 10x more professional.

Step 2: Importing Product Knowledge

The magic happens when your chatbot knows the store's products. At 9:30, I demonstrate two methods to import product data:

Method 1: Using the Store's Sitemap

For stores like Go Zero that have a sitemap.xml (found at 10:45), simply:

  1. Copy the sitemap URL
  2. In ConvaCode, go to Knowledge → Add Source → URL
  3. Paste the sitemap URL and click Fetch

Method 2: Generating a Sitemap

For stores without a sitemap (like Nika at 11:00):

  1. Use a free sitemap generator tool
  2. Enter the store's homepage URL
  3. Generate and download the sitemap
  4. Upload it to ConvaCode

By 13:00, you'll see the chatbot already pulling product data from Go Zero's website, ready to answer customer questions.

Step 3: Customizing the Chat Interface

At 14:30, we make the chatbot visually match the brand:

  1. Upload the store's logo
  2. Set brand colors (we used pink for Go Zero)
  3. Customize the welcome message
  4. Add quick-reply buttons ("I want to buy", "Track order")

The key is making the chatbot feel like a natural extension of the store. Notice at 16:00 how we changed the default "Track order" and "New arrivals" buttons to match Go Zero's product categories - small details that make a big difference in user experience.

Step 4: Adding Voice AI with Vapi

The game-changer comes at 17:45 when we integrate voice AI:

  1. Create a Vapi account (free tier available)
  2. Copy your API keys from Vapi dashboard
  3. In ConvaCode, go to Settings → Integrations
  4. Connect to Vapi using your API keys
  5. Enable "Show Vapi on load"

At 19:00, you'll see the stunning result - a chatbot that can now speak to customers naturally. The voice AI understands context, remembers previous questions, and even expresses emotion (notice how it says "That's so sweet of you" when asked about a gift for mom).
