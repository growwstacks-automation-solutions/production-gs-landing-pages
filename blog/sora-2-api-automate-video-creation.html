Sora 2 API Capabilities

Sora 2 represents a quantum leap in AI video generation. Unlike previous tools that simply animated existing elements, Sora 2 can create entirely new scenes and actions based on a single reference image and text prompt. The API accepts either:

  • A text prompt alone (generating video from scratch)
  • A reference image + text prompt (transforming the image into video)

In our workflow, we focus on the image-to-video transformation capability. This is particularly powerful for e-commerce because it allows businesses to breathe life into existing product photography. The AI interprets the prompt creatively - for example, turning a simple product shot into a commercial with animated elements, camera movements, and even added context like backgrounds or characters interacting with the product.

Make.com Workflow Breakdown

The complete automation requires just three Make.com modules:

Step 1: Source the Image

We use Google Drive's "Download a file" module, but this could connect to any image source - Dropbox, Shopify, Airtable, or even direct uploads through a webhook. The critical requirement is that the module outputs the image as binary data.

Step 2: Generate Video with Sora 2

The OpenAI module connects to your verified account (requires ID verification) and offers several configuration options:

  • Model selection (Sora 2 or Sora 2 Pro)
  • Video format (landscape or vertical)
  • Duration (up to 12 seconds)
  • Image reference (the binary file from step 1)
  • Text prompt (detailed instructions for the video)

Step 3: Save the Output

We store the generated video back to Google Drive, but this could just as easily upload to YouTube, Vimeo, or your CMS. The video arrives as binary data that any storage module can handle.

Pro tip: For production use, set the workflow to process videos asynchronously rather than waiting for each generation to complete. This allows batch processing of multiple images simultaneously.

Cost Considerations

While powerful, Sora 2 isn't cheap. Current pricing is:

  • Sora 2: $0.10 per second ($1.20 for 12 seconds)
  • Sora 2 Pro: $0.50 per second ($6 for 12 seconds)

The Pro version offers higher resolution but the same maximum duration. These costs can add up quickly when testing multiple prompts or processing large catalogs. We recommend:

  • Starting with shorter 5-8 second clips during development
  • Implementing approval steps before processing bulk jobs
  • Setting budget alerts in your OpenAI account

Despite the cost, this remains significantly cheaper than human-produced video at scale. For a 100-product catalog, $120 in API fees compares favorably to $5,000-$10,000 for professional video production.

Current Limitations

While impressive, Sora 2 has important constraints:

  • No human faces: Images containing recognizable people are rejected
  • 12-second maximum: Videos can't be longer without creative editing
  • Precise dimensions: Input images must be exactly 720x1280 or 1280x720
  • Creative interpretation: The AI may interpret prompts unexpectedly
  • Copyright restrictions: Known characters or logos trigger rejections

These limitations mean the technology works best for product-focused content rather than human-centric storytelling. However, for e-commerce applications, these constraints often align well with business needs.

Advanced Implementation Tips

For production deployments, consider these enhancements:

1. Dynamic Prompt Generation

Pull product details from your CMS or PIM to create tailored prompts automatically. For example: "Create a 8-second video showing [product name] being used in [use case]. Highlight [key feature] with close-up shots."

2. Multi-Platform Distribution

Extend the workflow to automatically upload finished videos to YouTube, TikTok, and your e-commerce platform with platform-optimized formats.

3. Human Approval Step

Insert a Slack or email approval step before processing large batches to control costs.

4. Performance Tracking

Tag videos with UTM parameters to measure which prompts drive the most conversions.

Implementation insight: The simplest version of this workflow takes 30 minutes to set up, but sophisticated implementations with these enhancements can become a complete video production pipeline.
