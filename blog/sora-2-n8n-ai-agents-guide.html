Handling Errors and Restrictions

Common Sora 2 generation issues and their solutions:

500 Internal Server Errors: Usually indicate temporary Sora 2 capacity issues. The workflow automatically retries up to 3 times with exponential backoff.

Content Restrictions: Certain prompts get rejected automatically. The workflow logs these for review and can substitute alternative content when possible.

JSON Breakage: Raw AI prompt outputs often contain newlines and quotes that break API requests. The workflow includes automatic sanitization:
