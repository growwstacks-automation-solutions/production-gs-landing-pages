Two-Part Workflow Architecture

The system uses a clever two-flow architecture in n8n to handle all processing steps:

Flow 1: Prompt Optimization

  1. Captures form data from Airtable via webhook
  2. Analyzes reference image using AI vision
  3. Optimizes basic prompt for Sora 2 video generation
  4. Structures final prompt with dialogue, emotion, action, character, setting
  5. Logs everything to Airtable for tracking

Flow 2: Video Generation

  1. Triggered automatically when Flow 1 completes
  2. Passes optimized prompt + image to Sora 2 via Kai AI API
  3. Waits 5 minutes for video generation
  4. Downloads finished video and uploads to Google Drive
  5. Updates Airtable with video URL and completion status

This architecture allows batch processing - you can queue dozens of videos overnight and wake up to a folder full of ready-to-use content.

Prompt Optimization Engine

The secret sauce is our AI-powered prompt optimization layer. A basic prompt like:

"10 second video of female holding product at kitchen table talking about benefits"

Gets transformed into:

Dialogue: "Product is so much better than alternatives. I actually enjoy this."
Emotion: Enthusiastic
Action: Female character holding up product package
Character: Woman in late 20s
Setting: Bright inviting kitchen

This structured format consistently produces high-quality UGC-style videos that perform well in ads and social content.

Smart Image Analysis

Before video generation, the system analyzes your reference image to extract brand-relevant details:

  • Identifies product vs character focus
  • Extracts color schemes and fonts
  • Recognizes text elements on packaging
  • Creates detailed visual description

This analysis gets incorporated into the final prompt to ensure video output accurately represents your product's physical attributes and branding.

Automated Video Generation

The video generation flow handles all technical steps:

  1. Creates organized Google Drive folder structure
  2. Uploads reference image to Cloudinary for public URL
  3. Submits optimized prompt + image to Sora 2 via Kai AI API
  4. Polls API every 5 minutes for completion status
  5. Downloads finished video and saves to designated folder
  6. Updates Airtable record with video URL and metadata

The entire process runs automatically once initiated - no manual steps required after submitting the initial form.
