The Problem With Manual Image Creation

Creating visual content is time-consuming. Most content creators and social media managers follow the same tedious process: think of an idea, type a prompt into an AI image generator, wait for the result, download it, and then upload it to their platform of choice. This manual workflow kills creativity and wastes valuable time.

The breakthrough came when we realized voice commands could bypass all these steps. By connecting Telegram's voice messaging with Make.com's automation capabilities, we created a system where speaking your idea instantly generates and delivers the finished image.

Content creators spend 3-5 hours per week just generating images manually. This automation cuts that time by 90%, letting you focus on strategy and creativity instead of repetitive tasks.

How the Voice-to-Image Workflow Works

The magic happens through a series of connected steps in Make.com. When you send a voice message to your Telegram bot, it triggers a chain reaction that handles everything automatically:

  1. Your voice message is converted to text using Whisper Turbo's speech recognition
  2. The text prompt is sent to Pollination's AI image generator
  3. The generated image is delivered to your chosen destination (Slack, email, etc.)

At 2:15 in the video tutorial, you can see the entire process in action - from speaking "a dinosaur walking through sunshine mountains" to receiving the finished image in Slack just moments later.

Step 1: Setting Up Your Telegram Bot

Every automation needs a trigger. In this case, we're using a Telegram bot to capture voice messages. Setting it up takes just a few minutes:

  1. Open Telegram and search for @BotFather
  2. Send /newbot command and follow the prompts to name your bot
  3. Copy the API token provided
  4. In Make.com, add the Telegram module and paste your token

Pro Tip: Name your bot something memorable like @YourNameImageBot so team members can easily find and use it.

Step 2: Voice Recognition with Whisper

The key to this workflow is accurately converting voice to text. We're using Block API's Whisper Turbo implementation because:

  • It's free for moderate usage
  • Supports multiple languages
  • Processes audio files up to 25MB

In Make.com, you'll add an HTTP request module configured to send the voice message to Block API's endpoint. The critical settings are:

  • Method: POST
  • Headers: Authorization with your API key
  • Body: The audio file from Telegram

Step 3: AI Image Generation

With the voice converted to text, we now generate the image using Pollination. This free service is perfect for automation because:

  • No API key required
  • No login needed
  • Generous free tier

The Make.com workflow takes the transcribed text and sends it to Pollination via another HTTP request. You can customize parameters like:

  • Image size (512x512 default)
  • Style (realistic, cartoon, etc.)
  • Number of variations

Step 4: Delivering the Final Image

The final step is getting your AI-generated image where you need it. While the tutorial shows Slack delivery, you can easily modify this to:

  • Email the image to yourself or team
  • Save to Google Drive or Dropbox
  • Post directly to social media platforms
  • Send back to Telegram as a reply

At 7:30 in the video, you'll see how to set up the Slack integration. The same principles apply to other destinations - just swap the Slack module for your preferred service.
