Step 1: Setting Up Meta Business Suite

The only technical part is connecting your WhatsApp through Meta's developer portal (timestamp 5:30). Here's the simplified process:

  1. Create a Facebook Business Page if you don't have one
  2. Go to Meta Business Suite → Settings → Apps
  3. Click "+ Create App" and select "Business" as the type
  4. Name your app (e.g., "WhatsApp Booking Agent")

Pro Tip: The client ID and secret key from your new app will be used to authenticate WhatsApp in n8n. Keep these secure but accessible during setup.

Step 2: Connecting WhatsApp Business Cloud

In n8n (timestamp 7:45):

  1. Create a new workflow and add the "WhatsApp Business Cloud" node
  2. Select "On Message" as the trigger
  3. Enter your Meta app's client ID and secret key
  4. Generate a test number in WhatsApp API Setup

This creates the foundation - any message sent to your WhatsApp number will now trigger the workflow automatically.

Step 3: Building Your AI Agent

The AI agent node (timestamp 12:10) is where the magic happens. You'll:

  1. Add an "AI Agent" node after the WhatsApp trigger
  2. Enable "Require Specific Output Format"
  3. Add a system message that defines your agent's personality and capabilities

The tutorial includes a complete system prompt template you can copy-paste and modify for your business. It structures the conversation flow and ensures professional, on-brand responses.

Step 4: Adding Conversation Memory

For multi-message conversations (timestamp 14:30), you'll add a memory node that:

  • Remembers previous messages in the chat
  • Maintains context throughout the booking flow
  • Prevents the AI from asking the same question twice

Set the memory limit to 10 messages - enough to handle a complete booking while preventing overload.

Step 5: Sending Automated Responses

The final step (timestamp 16:00) connects back to WhatsApp to send responses:

  1. Add a "Send Message" node from WhatsApp Business Cloud
  2. Use your Meta API access token for authentication
  3. Map the AI's response to the message body
  4. Use the original sender's number as the recipient

Test your workflow by sending a message to your WhatsApp test number - you should receive an instant AI response!
