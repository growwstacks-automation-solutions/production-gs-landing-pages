Step-by-Step Chatbot Setup

Connecting your TikTok account takes under 30 seconds through the platform's channels menu. The system uses TikTok's business API where available, falling back to alternative methods in restricted regions.

Create your first flow by naming it (keep titles under 40 characters) and selecting TikTok as the primary channel. The omni-channel option lets you reuse the same automation across platforms, but we recommend TikTok-specific flows for best performance.

Step 1: Initial Engagement Message

Start with a simple yes/no question like "Want to grow faster?" This binary choice creates an easy entry point while qualifying interest. At 4:30 in the tutorial, you'll see how to add follow-up messages based on their response.

Step 2: Data Collection

Add an email collection block with validation to ensure quality leads. Set the field as required and enable proper opt-in consent if you plan email marketing. The system automatically stores collected details in your connected CRM.

Pro Tip: Add a confirmation message like "Check your inbox in 2 minutes" to set expectations after email collection. This small touch increases lead magnet open rates by 32%.
