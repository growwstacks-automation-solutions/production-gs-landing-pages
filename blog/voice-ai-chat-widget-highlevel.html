HighLevel Voice Widget Setup

Creating your AI receptionist starts in the HighLevel dashboard under Sites → Chat Widget. Unlike basic chat systems, you'll select the "Voice AI" option which enables phone call functionality. The entire setup takes under 5 minutes:

  1. Click "Create New Chat Widget"
  2. Select "Voice AI" from widget type options
  3. Name your AI agent (we used "Sarah" for our roofing example)
  4. Customize the chat window appearance and greeting message

The system lets you remove the "Powered by" disclaimer and adjust colors to match your brand. You can preview changes in real-time before saving.

AI Agent Configuration

Your AI's personality and capabilities are controlled through simple dropdown menus. Key settings include:

  • Profile Language: Set to English (or your primary customer language)
  • Agent Type: Choose "Support" for general inquiries or "Sales" for appointment-focused flows
  • Greeting Message: Customize the first thing callers hear (e.g. "Thanks for calling Roofing Unlimited")

The tutorial shows how to change from the default "Samantha" outbound calling profile to a custom "Sarah" persona better suited for inbound service inquiries. This small tweak makes conversations feel more natural for your specific use case.

Website Integration

With your AI agent configured, connecting it to your website requires just two steps:

  1. In HighLevel, select the website where the widget should appear
  2. Choose your newly created voice AI widget from the dropdown

The system automatically generates and inserts the necessary code snippet. No technical skills are required - you don't even need to access your site's backend. At 1:52 in the video, you'll see how the widget instantly appears on a demo roofing site template.

Pro tip: Position your widget to appear on all pages except checkout/booking confirmation pages where customers might need immediate human support.
