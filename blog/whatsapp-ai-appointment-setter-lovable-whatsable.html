Configuring Lovable's AI Agent

The magic begins with a single prompt to Lovable describing the entire system. This includes:

  1. Core Functionality: Appointment scheduling via WhatsApp
  2. Technical Stack: WhatsAble API for messaging
  3. Database Schema: Contacts, conversations, appointments tables
  4. Business Rules: Working hours, maximum slots per response
  5. Edge Functions: Incoming message handler and AI router

Lovable processes this prompt and generates the complete application, including authentication system and admin interface. The only manual configuration required is connecting the WhatsAble API key and base URL.

Implementation Tip: Include links to WhatsAble's API documentation in your prompt so Lovable's AI can reference them when building the integration.

Connecting WhatsAble to Your WhatsApp Business Account

WhatsAble simplifies WhatsApp Business integration with three key features:

  1. One-Click Authorization: Connect your business number without Meta developer complexity
  2. Message Type Handling: Supports text, images, files and audio messages
  3. Team Collaboration: Multiple users can access the same business account

To complete the setup, you simply:

  1. Authorize your WhatsApp Business account in WhatsAble
  2. Copy your WhatsAble API key into Lovable
  3. Configure Lovable's edge function URL as the webhook in WhatsAble

This creates a two-way connection where WhatsAble forwards incoming messages to Lovable, and Lovable sends responses back through WhatsAble.
