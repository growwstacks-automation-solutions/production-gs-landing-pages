The Problem

Property developers, planners, and consultants face a significant challenge in monitoring planning applications across multiple local authorities. Manually checking each local authority's portal daily is time-consuming and prone to errors. This often leads to missed opportunities and delayed decision-making.

The lack of a centralized system for tracking planning applications means that valuable time is spent on repetitive data collection and analysis. This inefficiency can hinder project timelines and increase operational costs. Furthermore, inaccurate or incomplete data can lead to flawed assessments and poor investment choices.

The Solution

The solution is an automated workflow built using n8n that fetches and parses planning applications from multiple local authority portals daily. This workflow filters for recent submissions (last 30 days) and saves the data to Google Sheets, providing a centralized and easily accessible overview of planning activity.

n8n was chosen for its flexibility, scalability, and ability to handle complex workflows. Its open-source nature and wide range of integrations make it an ideal platform for automating this data collection process. Google Sheets was selected as the destination for the data due to its ease of use and accessibility for the client's team.

How It Works — Automated Data Consolidation

This n8n workflow automates the process of collecting and organizing planning application data from various local authorities into a single Google Sheet.

  1. Initial Trigger: The workflow is triggered daily to ensure the data is always up-to-date.
  2. Fetch Data from Portals: The workflow uses HTTP Request nodes to fetch planning application data from multiple local authority portals.
  3. Parse HTML Content: The HTML Parser node extracts relevant information from the fetched HTML content, such as application number, address, and description.
  4. Filter Recent Submissions: The workflow filters for recent submissions (last 30 days) to focus on the most relevant data.
  5. Transform Data: The Function node transforms the extracted data into a format suitable for Google Sheets.
  6. Save to Google Sheets: The Google Sheets node saves the transformed data to a specified Google Sheet, updating it with the latest information.
  7. Error Handling: The workflow includes error handling to ensure that any issues during data collection or processing are identified and addressed.
  8. Notification (Optional): Optionally, a notification can be sent via email or Slack to alert the team when the data has been updated.

💡 Centralized Data: By consolidating data from multiple sources into a single Google Sheet, the workflow eliminates the need for manual data collection and provides a comprehensive overview of planning activity.

Workflow automation involves using technology to automate repeatable tasks and processes. This reduces manual effort and improves efficiency. It's useful because it saves time, reduces errors, and allows businesses to focus on more strategic activities.

By automating tasks such as data entry, report generation, and customer communication, businesses can free up valuable resources and improve overall productivity. Workflow automation can also help to ensure consistency and compliance with established procedures.

n8n is a free, open-source automation platform that offers a wide range of integrations and customization options. Key benefits include its flexibility, scalability, and ability to handle complex workflows. It also allows for self-hosting, giving users more control over their data and infrastructure.

n8n's node-based approach makes it easy to visualize and modify workflows, while its fair-code license ensures transparency and community support. The platform also offers a variety of pre-built nodes and integrations, making it easy to connect to other applications and services.

The security of workflow automation depends on the platform and implementation. It's crucial to choose a platform with robust security features, such as encryption, access controls, and regular security audits. Properly configuring access rights and monitoring workflow activity are also essential.

Self-hosting options, like those offered by n8n, can provide additional security by allowing businesses to maintain complete control over their data and infrastructure. It's also important to follow best practices for data security, such as using strong passwords and regularly backing up data.
