Configuring the scenario

This automation scenario watches for your new posts on Instagram, downloads the files, and posts them to Telegram and Pinterest. The implementation takes approximately 20 minutes and requires intermediate technical skills.

Before starting, ensure you have accounts configured on all platforms with the necessary API connections established in Make.com. You'll need working knowledge of routers, filters, and array aggregators within the Make.com platform.

Pro tip: The Instagram module can only watch for your own posts, not other accounts. Make sure you're automating content from your own Instagram profile.

The scenario consists of several key modules that work together:

  • Instagram > Watch my account
  • Instagram > Download a media file
  • Array Aggregator
  • Instagram > Get media album
  • Telegram Bot > Send a document/image
  • Pinterest > Create a pin
  • Telegram Bot > Send a video
  • Telegram Bot > Send an album (media group)

Adding a router, Telegram and Pinterest modules - Top route

The router splits the scenario into three distinct routes, each handling different media types. The top route specifically processes single image files.

Configure the filter between the router and Telegram module to only allow image files by mapping the media type element and setting the condition to "Contains (case insensitive) IMAGE". This ensures only image posts proceed down this path.

Filter configuration to route only image files

For the Telegram module, you'll need the chat ID of your target channel or group. Map the caption from Instagram and configure the source file to come from the download media module. The Pinterest module similarly uses the downloaded image file.

Adding the Telegram module - Middle route

The middle route handles single video files with a similar filter setup as images, but configured for VIDEO media type instead. The Telegram video module configuration mirrors the image module setup.

This parallel processing ensures videos get optimized handling rather than being forced through an image processing pipeline that might not handle them correctly.

Pro tip: Test your video automation separately with different video formats to ensure compatibility with Telegram's requirements.

Adding the Instagram and Telegram modules - Bottom route

The bottom route processes carousel albums containing multiple images or videos. The filter checks for CAROUSEL_ALBUM media type and routes these complex posts differently.

The Instagram Get Album Media module retrieves all files from the post, while the Array Aggregator combines them into a single bundle for Telegram's media group functionality.
