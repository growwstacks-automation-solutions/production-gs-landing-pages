Configuring the Scenario

This automation scenario watches for your new posts on Instagram, downloads the files, and posts them to Telegram and Pinterest. The implementation requires intermediate skills and takes approximately 20 minutes to set up.

Before starting, ensure you have a Telegram account with a configured bot, Instagram and Pinterest accounts with all connections added, and basic understanding of how to use routers, filters, and array aggregators in your automation platform.

Pro tip: The Instagram module can only watch for your own new posts on your account, not other users' content.

The scenario consists of several key modules that work together to automate your social media posting:

  • Instagram → Watch my account
  • Router
  • Instagram → Download a media file
  • Array Aggregator
  • Instagram → Get media album
  • Telegram Bot → Send a document/image
  • Pinterest → Create a pin
  • Telegram Bot → Send a video
  • Telegram Bot → Send an album (media group)

Adding Router, Telegram and Pinterest Modules (Top Route)

The first route in our automation handles single image posts from Instagram. After the initial Instagram watch module, we add a router that splits the scenario into three routes, each with specific filters.

The top route filter only allows single image files to pass through to subsequent modules. You'll set this up by mapping the Media type element to the filter field, selecting "Contains (case insensitive)" as the operator, and typing "IMAGE" in the comparison field.

Filter setup to process only single image posts

After the filter, configure the Telegram Bot module to send the image to your channel. You'll need your Chat ID (available through Telegram bot commands) and to map the caption from Instagram. Select "Data" in the Send by field and choose the downloaded media file as the source.

Adding Telegram Module (Middle Route)

The middle route processes single video posts from Instagram. The filter setup is similar to the image route but uses "VIDEO" as the comparison value instead.

Configuration of the Telegram video module mirrors the image module setup - using the same Chat ID and caption mapping. The key difference is that this route doesn't include Pinterest posting, as Pinterest's API handles videos differently than images.

Pro tip: For platforms that don't support direct video posting, consider uploading to YouTube first and then sharing the link.

Adding Instagram and Telegram Modules (Bottom Route)

The bottom route handles Instagram carousel albums (posts with multiple images/videos). The filter uses "CAROUSEL_ALBUM" as the comparison value to identify these posts.

This route requires additional modules to process multiple media files. The Instagram Get Album Media module retrieves all files from the post, which are then downloaded and aggregated into a single array before being sent to Telegram as a media group.
