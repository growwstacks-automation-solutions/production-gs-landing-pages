What can Make's AI do for HR?

360-degree feedback is a gold mine of HR insights, yet its value often goes untapped. Why? Because processing and analyzing all that data is slow and prone to bias. Especially when it comes to open-ended comments.

This is where Make and AI-driven HR analytics step in. By combining automated workflows with AI analysis, you can turn raw performance evaluations into structured reports in minutes, eliminating days of manual work.

For HR leaders and department managers, AI-driven performance automation eliminates tedious manual work, allowing you to focus on your people and their professional growth.

The Make scenario for automated 360-degree feedback processing

The Make scenario we introduce in this guide will:

  • Be triggered when the 360-degree feedback questionnaire is submitted
  • Send the responses to AI for analysis
  • Extract employees' strengths, areas for improvement, and development recommendations
  • Save the response and AI analysis to your database
  • Generate ready-to-use performance review documents (PPTX, PDF, or image)
  • Instantly report to relevant managers and HR personnel

Pro tip: For best results, structure your feedback form with both quantitative ratings (1-5 scales) and qualitative open-ended questions. The AI analyzes both types of data differently to provide comprehensive insights.

Step-by-step: Build 360-degree feedback automation in Make

Here is how you can build your own HR automation in Make to transform feedback into actionable insights.

Step 1: Watch for form submissions

The scenario is triggered as soon as new feedback is received through your chosen form platform. Popular options include Plumsail Forms, Google Forms, or Typeform connected via webhooks.

Configure the trigger module to watch for new submissions and pass the complete response data to the next steps. Include all rating scores and text comments in the payload.

Example 360-degree feedback form structure

Step 2: Analyze feedback with AI

Add the Make AI Toolkit module to process the feedback. You'll need to craft a detailed prompt that instructs the AI how to analyze the responses.

A well-structured prompt should:

  • Define the AI's role (e.g., "You are an industrial-organizational psychologist")
  • Specify the analysis objectives
  • List all elements to extract from the feedback
  • Set the output format (JSON recommended)

Example prompt structure:

"Analyze this 360-degree feedback submission. Identify: 1) Three key strengths with supporting comments, 2) Two development areas with specific examples, 3) Recommended training/resources for each development area. Output as JSON with these fields: employee_name, reviewer_role, strengths[], development_areas[], recommendations[]."

Step 3: Parse JSON

The AI will return structured data in JSON format. Use Make's JSON parser to extract clean, typed fields that are easier to log, filter, and map into reports.

This step creates discrete data points for each insight (strengths, weaknesses, recommendations) that can be individually processed in subsequent steps.

Step 4: Log feedback to a database

Save both the original response and the AI analysis to your HR database. This creates an audit trail and allows for historical trend analysis.

Common integration targets include:

  • Monday.com HR boards
  • Airtable performance tracking bases
  • Zoho People or BambooHR records
  • Notion performance databases
Structured feedback data stored in Monday.com

Step 5: Create a report file

Use a document generator like Plumsail Documents to create professional reports from the analyzed data. Map the JSON fields into a standardized template that includes:

  • Employee and reviewer metadata
  • Quantitative ratings with visual charts
  • AI-analyzed strengths with supporting comments
  • Development areas with specific examples
  • Personalized development recommendations
  • Suggested goals for the next review period

Output formats can include PDF for distribution, PPTX for presentation in review meetings, or even image files for quick sharing in chat platforms.

Automatically generated performance report

Step 6: Send the report

Distribute the final report through your organization's preferred channels:

  • Email to the employee and their manager
  • Slack message to the HR team
  • Upload to Google Drive or SharePoint
  • Post in your HRIS system

Configure conditional routing to ensure reports are only visible to authorized personnel based on your HR policies.

Make it a consolidated report

For comprehensive analysis, create a consolidated report that combines feedback from all reviewers (self, peers, managers). Here's how:

Trigger a separate scenario when all feedback is submitted or when the review period closes. The process:

  1. Group all feedback for the employee
  2. Send the complete dataset to the AI for combined analysis
  3. Generate a master report showing:
    • Consensus strengths across all reviewers
    • Consistent development themes
    • Outlier perspectives that differ from the group
    • Prioritized development plan

This consolidated view eliminates the need to manually compare multiple individual reports and highlights the most important insights.
