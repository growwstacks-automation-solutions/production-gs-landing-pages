How It Works

This workflow operates in three main stages: AI content generation, data parsing, and spreadsheet organization. Each stage is handled by a dedicated module in Make.com, ensuring a smooth and efficient process.

Step 1: Set up the CometAPI + Make integration

First, you'll connect CometAPI to your Make.com account. This involves obtaining your API key from the CometAPI console and adding it to a new Make scenario to establish a secure connection. This foundational step allows Make to send requests to CometAPI and receive AI-generated content.

Creating a new scenario in Make.com.
Adding the CometAPI key to establish the connection.

Step 2: Configure the content generation workflow

This is the core of the automation. A three-module workflow is built: a CometAPI module generates structured JSON content based on an optimized prompt, a "Parse JSON" module extracts specific data fields (Question, Answer, Tag), and a Google Sheets module maps these fields to a new row in your spreadsheet. This ensures that the AI's output is immediately usable and organized.

Step 3: Test and deploy the automation

After configuring the modules, you'll run the scenario once to verify that the entire workflow functions correctly. This involves checking your Google Sheet to confirm that a new row with the structured AI-generated content has been successfully added. Once confirmed, the automation can be scheduled to run regularly, providing a continuous stream of content ideas.

Quick Setup Guide

  1. Sign up/Log in to CometAPI: Obtain your API key from the CometAPI console.
  2. Create a new Make.com scenario: Log in to your Make.com account and start a new scenario.
  3. Add CometAPI module: Search for CometAPI, select "Create a Chat" and add your API key to establish the connection.
  4. Configure CometAPI prompt: Use the provided optimized prompt to generate structured Q&A content.
  5. Add Parse JSON module: Connect it after the CometAPI module to extract "Question", "Answer", and "Tag" fields.
  6. Prepare Google Sheet: Create a new Google Sheet with "Question", "Answer", and "Tag" as column headers.
  7. Add Google Sheets module: Connect your Google account and map the parsed fields from the previous module to the corresponding columns in your spreadsheet.
  8. Test and activate: Run the scenario once to ensure it works, then schedule it to run at your desired intervals.
