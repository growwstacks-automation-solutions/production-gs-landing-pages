What This Workflow Does

This powerful workflow automates the entire process of collecting valuable web data, organizing it, and extracting actionable insights using AI. It addresses the common business challenge of acquiring large-scale web data reliably, especially from dynamic or protected websites. By combining Make.com's automation capabilities with Oxylabs' advanced web scraping technology, businesses can effortlessly gather product information, market trends, or competitor data without encountering common roadblocks like IP blocking.

How It Works

This workflow is designed for seamless, automated web data collection and analysis. Here's a step-by-step breakdown:

Step 1: Create a Scenario in Make.com

The journey begins in Make.com, your central automation hub. You'll start by creating a new scenario from scratch, providing a blank canvas to design your data workflow. This initial setup is straightforward, even for those new to automation, and lays the foundation for connecting all subsequent modules.

Step 2: Configure the Oxylabs Web Scraper Module

Next, you integrate the Oxylabs module into your Make.com scenario. This is where the magic of web scraping happens. You'll connect your Oxylabs Web Scraper API account and configure parameters such as the target website (e.g., Amazon), search queries (e.g., "Lenovo laptop"), geographical location for localized results, and the number of pages to scrape. After the data is scraped, a 'Set Variable' tool is used to consolidate the results into a single, manageable array. A simple formula flattens the nested data structure from Oxylabs into a clean array of items.

Step 4: Structure and Save Data to Google Sheets

To ensure the data is neatly organized, the workflow employs an 'Iterator' to handle individual data rows and an 'Array Aggregator' to bundle them. Finally, a Google Sheets 'Bulk Add Rows' module pushes this structured data into your designated spreadsheet. This allows for precise mapping of scraped fields to specific columns in your sheet, ensuring clarity and usability.

Step 5: Run the Scenario

With all modules configured, you can now run the entire workflow. Make.com will orchestrate the process: triggering the Oxylabs scraper, processing the data, and populating your Google Sheet with organized results. This step validates your setup and demonstrates the automation in action.

To add an intelligence layer, you'll create an AI Agent within Make.com. This agent is given a specific role and prompt (e.g., "expert e-commerce product assistant") to analyze the scraped data. A 'Router' module then branches the workflow, sending the collected data to both Google Sheets and the AI agent. The AI agent processes the data and generates a summary report, which is then saved to a separate tab in your Google Sheet, providing valuable analytical insights.

Quick Setup Guide

  1. Sign up for Make.com and Oxylabs: Ensure you have active accounts for both platforms.
  2. Create a new scenario in Make.com: Log in, navigate to Scenarios, and click "Create a new scenario."
  3. Add and configure the Oxylabs module: Search for "Oxylabs," select "Scrape Amazon Search" (or relevant module), and connect using your Oxylabs API credentials. Define your query, geo-location, and page count.
  4. Add a "Set variable" module: Use this to flatten the scraped data into a single array for easier processing.
  5. Integrate Google Sheets: Add an "Iterator" and "Array Aggregator" to prepare data. Then, add a Google Sheets "Bulk Add Rows" module, connect your Google account, and map the aggregated data to your spreadsheet columns.
  6. (Optional) Set up an AI Agent: Create an AI Agent in Make.com with a specific role (e.g., "expert e-commerce product assistant") and instructions for analysis.
  7. Add a Router for AI analysis: Insert a "Router" between your "Set variable" and "Iterator" to send data to both Google Sheets and your AI Agent. Configure the AI Agent to receive the flattened data.
  8. Save AI analysis to Google Sheets: Add another Google Sheets module ("Update a cell") to save the AI Agent's summary report to a dedicated tab in your spreadsheet.
  9. Test and schedule: Run the scenario to verify everything works. Once confirmed, schedule your scenario to run automatically at desired intervals.
