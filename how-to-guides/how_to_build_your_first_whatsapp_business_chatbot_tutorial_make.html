What This Workflow Does

This workflow provides a step-by-step guide to building a simple yet effective WhatsApp Business chatbot using Make.com. It's designed to automate responses to frequently asked questions, allowing your business to offer instant support to customers 24/7. By setting up predefined replies for common queries, you can significantly reduce the workload on your customer service team and ensure consistent, rapid communication.

The core function of this automation is to listen for incoming messages on your WhatsApp Business account, identify specific keywords or numerical selections from customers, and then automatically send the appropriate, pre-written response. This not only streamlines customer interactions but also collects valuable data on the most common inquiries, helping you understand your audience better and improve your services.

How It Works

This automation operates by monitoring your WhatsApp Business account for new messages and routing them based on customer input to deliver instant, relevant responses.

Step 1: Connect WhatsApp Business Cloud

The first step involves setting up a "Watch Events" module for WhatsApp Business Cloud in Make.com. This module acts as the trigger, constantly listening for new messages sent to your business WhatsApp number. You'll connect your WhatsApp Business account by providing a permanent token and account ID, ensuring that Make.com has the necessary permissions to receive and process messages.

Step 2: Add a Router for Conditional Responses

After receiving a message, a "Router" module is used to create multiple paths in your scenario. Each path corresponds to a different type of customer query. For example, if a customer types "1" for promotions, their message will be routed down a specific path designed to respond with promotional information. This allows the chatbot to handle various inquiries without getting confused.

Adding a Router to create multiple response paths.

Step 3: Configure Each Route with Responses and Tracking

For each route, you'll add a "Send a message" module from WhatsApp Business Cloud. Here, you'll define the automated text response for that specific query. Crucially, each route also includes an "Increment function" module and a "Microsoft 365 Excel > Update a Worksheet Row" module. The increment function counts how many times a particular query option is selected, and the Excel module records this data, providing valuable analytics on customer interests.

Quick Setup Guide

  1. Create a New Scenario in Make.com: Log into your Make.com account and start a new scenario.
  2. Add WhatsApp Business Cloud "Watch Events" Module: Search for "WhatsApp Business Cloud" and select the "Watch Events" module. Create a new webhook connection by providing your Permanent Token and WhatsApp Business Account ID (instructions for obtaining these are usually found in your WhatsApp Business API documentation). Set the event to "Messages".
  3. Enable Webhooks: Follow the instructions provided by WhatsApp Business Cloud to enable webhooks in your Facebook Developer account, pointing to the URL generated by Make.com.
  4. Add a Router: From the "Tools" menu, add a "Router" module to split your scenario into multiple paths, one for each type of query you want to automate.
  5. Configure Response Routes: For each path from the router, add a "WhatsApp Business Cloud > Send a message" module. Map the customer's WhatsApp ID to the "Receiver" field and enter your desired automated response text in the "Body" field.
  6. Set Up Query Tracking: In each response route, add an "Increment function" module (from the "Tools" menu) and a "Microsoft 365 Excel > Update a Worksheet Row" module. Configure the Excel module to connect to your spreadsheet and update a specific cell (e.g., a counter for that query type) with the output of the Increment function.
  7. Apply Filters: On the link between the Router and each "Send a message" module, add a "Filter". Configure the filter to check the incoming message text (e.g., "Message []: Text.Body") and only allow the scenario to proceed if it matches a specific input (e.g., "1" for the first option, "2" for the second, etc.).
  8. Save and Activate: Once all routes and modules are configured, save your Make.com scenario and activate it. Test it by sending messages to your WhatsApp Business number.
