What This Workflow Does

This automation streamlines the often tedious process of converting Google Docs into PDF files. Instead of manually opening each document, selecting "Download as PDF," and saving it, this workflow handles it all for you automatically. It monitors a designated Google Drive folder for new or updated Google Docs, converts them to the universally compatible PDF format, and then saves these new PDF versions to another specified folder in Google Drive. This eliminates repetitive clicks and ensures your documents are always in the correct format for sharing, archiving, or presentation.

For businesses and individuals who frequently work with Google Docs and require PDF versions for various purposes, this automation is a game-changer. It ensures consistency, reduces human error, and frees up valuable time that can be redirected to more strategic tasks. The workflow begins by actively watching a specific Google Drive folder for any new Google Docs that are created or added. This acts as the trigger for the entire automation. You define which folder should be monitored, ensuring that only relevant documents initiate the conversion process.

Quick Setup Guide

Setting up this automation is straightforward. Follow these steps to get your Google Docs converting to PDFs automatically:

  1. Create a New Scenario in Make.com: Log into your Make.com account and click "Create a new scenario."
  2. Add Google Docs "Watch Documents" Module: Search for Google Docs, select "Watch Documents," and connect your Google account. Configure it to watch the specific Google Drive folder where your source Google Docs will be placed. Set a reasonable "Limit" for documents to process per run.
  3. Add Google Docs "Download a Document" Module: Add another Google Docs module, select "Download a Document." Map the "Document ID" from the previous step and choose "PDF Document" as the output type.
  4. Add Google Drive "Upload a File" Module: Add a Google Drive module, select "Upload a File." Connect your Google account again. Specify the destination folder in Google Drive where you want the converted PDFs to be saved. Select the file output from the "Download a Document" module.
  5. Schedule and Test: Set the schedule for how often you want the scenario to run (e.g., daily, hourly). Run a test with a sample Google Doc in your source folder to ensure everything works as expected.
  6. Activate and Save: Once testing is successful, turn the scenario "ON" and save it.
