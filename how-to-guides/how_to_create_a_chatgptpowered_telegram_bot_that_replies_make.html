How to Create a ChatGPT-powered Telegram Bot That Replies

Step-by-step guide to building an AI-powered Telegram bot that automatically responds to messages using ChatGPT and Make.com automation.

  1. Step 1: Create a new scenario and install a Telegram Bot
  2. Step 2: Add the OpenAI module

Building a Telegram bot that automatically responds to messages with ChatGPT's intelligence solves numerous communication challenges. From answering FAQs to providing instant support, this integration combines Telegram's messaging platform with ChatGPT's conversational AI.

While Telegram doesn't natively offer AI-powered replies, Make.com provides the perfect automation platform to connect these services. The result is a responsive bot that handles inquiries 24/7 with human-like understanding.

Step 1: Create a new scenario and install a Telegram Bot

The foundation of our automation begins in Make.com's scenario builder. After logging into your account, initiate a new scenario which will serve as the workflow orchestrator between Telegram and ChatGPT.

The Telegram Bot app's "Watch Updates" module acts as our trigger point. This webhook-based module instantly detects incoming messages, whether from direct chats or channel posts. Configuration involves connecting your Telegram account through Make's interface.

Setting up the Telegram Bot module in Make.com

Webhook Configuration

Make simplifies webhook creation with a one-click setup. The platform automatically handles the technical complexities of establishing a persistent connection between Telegram's API and your automation workflow.

This webhook remains active as long as your scenario runs, requiring no additional maintenance. Its efficiency lies in triggering the workflow only when new messages arrive, optimizing resource usage.

Step 2: Add the OpenAI module

The intelligence layer comes from OpenAI's ChatGPT integration. The "Create a Completion" module allows us to harness ChatGPT's language capabilities while customizing its behavior for our specific use case.

Authentication requires your OpenAI API key, which you can generate in your account settings. The module offers configuration options that determine response quality, including model selection and temperature settings.

Pro tip: Use GPT-4 for more nuanced responses or GPT-3.5 for faster, more cost-effective replies depending on your needs.

Crafting the Perfect Prompt

The prompt engineering determines your bot's effectiveness. Our recommended structure handles several key functions:

  • Language detection and matching
  • Topic filtering to avoid off-subject responses
  • Context-aware reply generation
Configuring the OpenAI module with the response prompt

Step 3: Add the final Telegram module

Completing the circle involves sending ChatGPT's generated response back to Telegram. The "Send a Text Message or Reply" module handles this final step, requiring proper mapping of data between modules.

Key mapping points include connecting the correct chat ID (whether from direct messages or channels) and feeding the AI-generated response into the message content field. These connections ensure replies reach the right conversation with appropriate context.

Pro tip: For group chats, ensure you've granted the bot necessary permissions and consider adding @ mentions to clarify who the bot is responding to.

Activation and Testing

Before activating your scenario, thoroughly test all connections. Make.com provides a testing mode that lets you verify each step without sending actual messages. Once live, monitor initial interactions to refine responses as needed.
