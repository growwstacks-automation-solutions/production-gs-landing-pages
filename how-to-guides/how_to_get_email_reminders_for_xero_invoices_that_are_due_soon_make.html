How to Get Email Reminders for Xero Invoices That are Due Soon

Learn how to set up Xero invoice reminders and receive an email notification when an invoice is about to be due.

1. Create a new Make scenario and add the Xero app

Log into your Make account and create a new scenario. This will serve as the foundation for your Xero invoice reminder system. The scenario acts as a workflow that runs automatically on your specified schedule.

Then, add the Xero > Search for Invoices module to this scenario. This module connects directly to your Xero account and organization, allowing you to query invoice data. You'll need to authenticate your Xero account the first time you set this up.

Pro tip: Make sure you have admin-level access to your Xero account when setting up the connection, as some API endpoints require elevated permissions to access invoice data.

Set the Search by dropdown setting to "user-defined condition" - this allows you to write custom search logic using Xero's API query language. The where condition is where the magic happens for filtering invoices.

Configure the Xero module with your custom search conditions

The search condition filters invoices based on four key criteria: type (ACCPAY for bills), status (AUTHORISED for approved invoices), amount due (greater than zero), and due date (within your specified timeframe). Each condition is joined with && to require all must be true.

For the due date parameter, the formula calculates dates dynamically. The example searches for invoices due within 6 days (a week), but you can adjust this to any timeframe. The DateTime function converts dates into Xero's required format.

Customize the search condition to match your preferred reminder timeframe

The Maximum number of returned accounts field controls how many results the search returns per execution. Set this based on your typical invoice volume - too low and you might miss some, too high and you'll waste processing capacity.

Format your invoice list with the Text Aggregator module

After configuring the Xero module, add the Text Aggregator module to compile the invoice data into a readable format for your email. This module takes the raw invoice data and transforms it into a clean list with your specified formatting.

Add a filter to prevent empty notifications

The filter between modules ensures you only receive emails when there are actual invoices matching your criteria. Without this, you might get blank notifications when no bills are due.

Customize your email notification content and recipients

Finally, configure the Gmail module to send your formatted invoice list. You can customize the subject line, body content, and recipients. The HTML formatting ensures the invoice list displays cleanly in email clients.
