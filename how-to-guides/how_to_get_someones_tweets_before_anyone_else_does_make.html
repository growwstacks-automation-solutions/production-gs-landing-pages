Make.com Twitter Automation Notifications

How to Get Someone's Tweets Before Anyone Else Does

Automated Twitter notifications help you cash in on fresh information. Learn how to get tweets instantly with Make.com automation.

1. The Native Twitter Notification Solution

Twitter offers a built-in notification feature that allows you to get alerts when specific accounts tweet. This is activated by clicking the bell icon on a profile you follow. While simple to set up, this native solution has several limitations that make it less than ideal for professional use.

First, these notifications get mixed in with all other Twitter alerts - likes, replies, follows - making it easy to miss important tweets. Second, there's no filtering capability, so you'll get notified about every single tweet from an account, regardless of content. Third, web browser notifications are unreliable, only working consistently in Twitter's mobile app.

2. Automated Twitter Notifications with Make

Using Make.com (formerly Integromat) to monitor Twitter accounts provides significant advantages over the native solution. You can customize notification channels, filter content, and receive alerts faster than 99% of Twitter users. This method is particularly valuable for investors, journalists, marketers, and anyone who benefits from real-time information.

The Make.com automation works by continuously checking a specified Twitter account for new tweets. When a matching tweet is found (optionally filtered by keywords), it triggers an immediate notification through your preferred channel - email, Slack, SMS, or other integrated services.

Pro tip: Twitter is the only premium app on Make.com, requiring a paid account. The investment is worthwhile for professionals who rely on timely Twitter information.

3. Step-by-Step Implementation

Creating this automation involves setting up a scenario in Make.com with two main components: monitoring tweets and sending notifications. Below we'll walk through each step in detail.

Step 1: Create the Make Scenario

Begin by logging into your Make.com account and creating a new scenario. Select Twitter as the trigger app - this will check for new tweets at regular intervals. The scenario builder provides a visual interface where you'll connect different modules to create your automation workflow.

After selecting Twitter as your trigger, click "Continue" to access the scenario builder canvas. This is where you'll assemble all components of your automation. The Twitter module will be pre-loaded as your starting point.

Creating a new scenario in Make.com dashboard

Step 2: Configure the Twitter Module

The Twitter "Watch tweets" module is the foundation of your automation. Click on it to configure these key settings:

  • Connect your Twitter account to Make.com
  • Set "I want to watch" to "Someone else's tweets"
  • Enter the Twitter handle (without @ symbol) you want to monitor
  • Enable "Return full text of tweets"
  • Set an appropriate tweet limit based on how active the account is

For accounts that tweet frequently (like Elon Musk used in our example), set a higher limit to ensure you don't miss anything between checks. The module will only return new tweets since the last check.

Configuring the Twitter watch module settings

Step 3: Configure Notification Module

After setting up the Twitter monitoring, add your notification method. For this example we'll use Gmail, but Make.com supports numerous alternatives:

  1. Click the plus sign next to the Twitter module
  2. Search for and select Gmail (or your preferred notification service)
  3. Choose "Send an email" as the action
  4. Connect your Gmail account
  5. Configure email details including recipient, subject, and content

In the email content, include relevant tweet data like the text, URL, and timestamp. You can customize exactly what information each notification contains.

Setting up email notification details in Gmail module

Step 4: Add Filters (Optional)

To avoid notification overload from prolific tweeters, add filters between the Twitter and Gmail modules:

  1. Click the wrench icon between modules
  2. Add a "Text" operator under Condition
  3. Set to "Contains (case insensitive)"
  4. Enter keywords that must appear in the tweet

For our Elon Musk example, we filtered for "bitcoin" and "dogecoin." You can add multiple keywords and create complex filtering logic to match your specific needs.

Pro tip: When monitoring financial or market-moving accounts, consider adding variations of key terms (like "BTC" for bitcoin) to ensure comprehensive coverage.

Step 5: Schedule the Scenario

The final step is setting how often Make.com checks for new tweets:

  1. Click the clock icon on the Twitter module
  2. Select "At regular intervals"
  3. Set frequency to 1 minute for near real-time alerts
  4. Save and activate the scenario

At 1-minute intervals, you'll receive notifications faster than most Twitter users see the tweets organically. The scenario runs continuously once activated.
