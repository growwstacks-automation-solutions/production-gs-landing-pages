How to Repurpose TikTok Videos into LinkedIn Posts [Tutorial]

Automate the transformation of your TikTok video content into professional LinkedIn posts, saving time and expanding your reach with AI.

What This Workflow Does

This powerful automation streamlines the process of transforming your engaging TikTok video content into professional, text-based LinkedIn posts. In today's fast-paced digital world, content creators and social media managers often struggle to maintain a consistent presence across multiple platforms without significant manual effort. This workflow eliminates that bottleneck by automating the transcription of your TikTok videos and the generation of tailored LinkedIn posts, ensuring your valuable content reaches a broader, professional audience.

Imagine effortlessly extending the life and impact of your short-form video content. Instead of spending hours manually transcribing, writing, and formatting posts for LinkedIn, this solution takes care of it all. It's designed to save you precious time, reduce repetitive tasks, and ensure your brand's message is consistently delivered across different social media ecosystems, maximizing your content's return on investment.

Overview of the Make.com scenario for repurposing TikTok videos.

How It Works

Step 1: Triggering the Automation with Email

The workflow begins when you send an email containing a TikTok video link to a designated mailbox. A Gmail "Watch Emails" module in Make.com monitors this inbox for new, unread emails. This acts as the manual trigger, initiating the entire automation sequence.

Setting up the Gmail module to watch for new emails.

Step 2: Extracting the TikTok Video Link with AI

Once an email is received, an OpenAI (ChatGPT) module is used to intelligently extract the TikTok video link from the email body. This is crucial because emails might contain additional text. ChatGPT is prompted to return only the URL in a structured JSON format, making it easily usable for subsequent steps.

Configuring ChatGPT to extract the video URL from email content.

Step 3: Filtering for Relevant Emails

To ensure only relevant emails trigger the full workflow, a filter is applied between the Gmail and OpenAI modules. This filter checks for a specific subject line (e.g., "LinkedIn Post Request"), preventing the automation from processing every incoming email and ensuring efficiency.

Applying a filter to process only specific emails.

Step 4: Parsing the JSON Output

The JSON output from ChatGPT, containing the extracted video URL, is then processed by a JSON "Parse JSON" module. This step converts the JSON string into a usable data structure, allowing the URL to be easily mapped and utilized in the subsequent modules of the scenario.

Parsing the JSON output to extract the video URL.

Step 5: Fetching Video Information via HTTP

Two HTTP "Make a request" modules are used to interact with a TikTok video download API (e.g., via RapidAPI). The first module fetches metadata about the video, including the URI for its audio track. This is a critical step for preparing the content for transcription.

Making an HTTP request to retrieve TikTok video metadata.

Step 6: Downloading Video Audio

The second HTTP "Make a request" module then uses the audio URI obtained in the previous step to download the audio file from the TikTok video. This isolated audio is essential for the next stage of transcription.

Downloading the audio track from the TikTok video.

Step 7: Transcribing Audio with OpenAI Whisper

An OpenAI Whisper module is employed to convert the downloaded audio file into text. This advanced speech-to-text system accurately transcribes the spoken content, providing the raw material for the LinkedIn post. This step leverages AI to handle complex audio processing efficiently.

Using OpenAI Whisper to transcribe the video's audio.

Step 8: Generating the LinkedIn Post with ChatGPT

A final OpenAI (ChatGPT) module takes the transcribed text and generates a professional and engaging LinkedIn post. The prompt is carefully crafted to ensure the post is inspiring, uses appropriate emojis, maintains a professional tone, and is suitable for the LinkedIn audience. This step transforms raw transcription into polished, ready-to-publish content.

Crafting the LinkedIn post using AI based on the transcribed text.

Step 9: Delivering the Draft Post via Email

The last step involves sending the AI-generated LinkedIn post back to the original requester via a Gmail "Send an email" module. This allows for a final review and approval before the content is manually published to LinkedIn, ensuring human oversight and quality control.

Sending the final draft of the LinkedIn post for review.

Quick Setup Guide

  1. Create a New Make.com Scenario: Log in to your Make.com dashboard and start a new scenario.
  2. Add Gmail "Watch Emails" Module: Configure it to watch for new unread emails in your chosen inbox. Set it to mark emails as read and specify the maximum number of results (e.g., 10).
  3. Add OpenAI "Create a Completion" Module (for URL extraction): Connect your OpenAI account. Use the "Create a Chat Completion (GPT Models)" method with GPT 3.5 turbo. Map the email's text content and use the prompt: Extract the URL out of this Content and respond with a JSON. Content: {text} {"URL":"VALUE"} JSON
  4. Insert a Filter: Place a filter between the Gmail and OpenAI modules. Configure it to only allow emails with a specific subject line (e.g., "LinkedIn Post Request") to pass through.
  5. Add JSON "Parse JSON" Module: Map the content output from the OpenAI module to parse the JSON and extract the URL.
  6. Add First HTTP "Make a Request" Module (for video info): Configure this module to call the TikTok video download API via RapidAPI. Include your RapidAPI key in the headers and map the extracted URL to the query string.
  7. Add Second HTTP "Make a Request" Module (for audio download): Map the data:music:uri from the previous HTTP module's output to the URL field to download the audio.
  8. Add OpenAI "Create a Transcription" Module (Whisper): Connect your OpenAI account. Use a timestamp for the file name with a .mp3 extension and map the audio data from the previous HTTP module.
  9. Add Third OpenAI "Create a Completion" Module (for post generation): Use the "Create a Chat Completion (GPT Models)" method, preferably with GPT-4. Map the transcribed text and use a prompt like: Generate an inspiring LinkedIn post based on this TRANSCRIPT. Rules: Use Emojis. Use English even if the transcript is not in English. Sound inspiring and motivational. Focus on simple yet professional language. TRANSCRIPT: {text}
  10. Add Final Gmail "Send an Email" Module: Map the original sender's email address, a unique subject line (e.g., original subject + Email ID), and the AI-generated post content from the last OpenAI module.
  11. Schedule and Activate: Set your scenario's desired execution schedule and turn it ON.
