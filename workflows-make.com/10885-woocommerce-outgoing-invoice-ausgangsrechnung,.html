Automate WooCommerce to FreeFinance Invoicing

Eliminate manual data entry by automatically creating FreeFinance invoices for every new WooCommerce order

What This Workflow Does

This automation solves a critical pain point for WooCommerce stores using FreeFinance for accounting. Every time a customer places an order, the system automatically generates a corresponding invoice in FreeFinance without any manual intervention.

The workflow checks for new WooCommerce orders at regular intervals (default every 15 minutes), extracts all relevant order details including customer information, products, quantities, prices, taxes, and shipping costs, then creates a properly formatted outgoing invoice in FreeFinance with this data. This eliminates:

  • Manual copy-pasting of order details
  • Data entry errors that cause accounting discrepancies
  • Delays in invoice generation that slow down cash flow
  • Time wasted reconciling WooCommerce and FreeFinance records

How It Works

1. Poll for new WooCommerce orders

The scenario starts by checking your WooCommerce store for any new orders that haven't been processed yet. It uses WooCommerce's REST API to fetch order details in real time.

2. Extract and format order data

For each new order, the workflow extracts customer details (name, address, email), product information (SKUs, quantities, prices), taxes, shipping costs, discounts, and payment method. This data is formatted to match FreeFinance's invoice requirements.

3. Create FreeFinance invoice

The formatted data is sent to FreeFinance via API, where it creates a complete outgoing invoice with all line items, taxes, and totals. The invoice includes your company branding and follows your accounting templates.

4. Optional notifications

You can configure the workflow to send email confirmations to your accounting team or store admins when new invoices are created, or trigger additional actions like updating internal spreadsheets.

Pro tip: For high-volume stores, adjust the polling frequency to 5 minutes or less. For stores with fewer than 10 daily orders, hourly checks may suffice to reduce API calls.

Quick Setup Guide

  1. Connect WooCommerce: In Make.com, add your WooCommerce store URL and generate API keys in WooCommerce settings
  2. Connect FreeFinance: Enter your FreeFinance API credentials from the accounting software
  3. Map fields: Match WooCommerce order fields (like "product_name") to corresponding FreeFinance invoice fields
  4. Test: Run the scenario manually with a test order to verify invoice creation
  5. Schedule: Set your preferred polling frequency (default is every 15 minutes)
