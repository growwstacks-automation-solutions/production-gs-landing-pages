What This Workflow Does

This automation solves the time-consuming process of manually transcribing and summarizing Zoom meeting recordings. It automatically processes each new recording through OpenAI's Whisper for accurate transcription, generates concise summaries, creates audio highlight reels using ElevenLabs' text-to-speech, and shares everything with relevant team members via Slack.

Teams using this system report saving 5-10 hours per week on meeting documentation while improving information retention and accessibility. The automated transcripts ensure nothing gets missed or misremembered, and the audio summaries make key points accessible to team members who prefer listening over reading lengthy notes.

How It Works

1. New Zoom recording detection

The workflow continuously monitors your Zoom account for newly available recordings. When a recording is detected, it automatically downloads the audio file for processing.

2. OpenAI Whisper transcription

The audio file is sent to OpenAI's Whisper API, which converts speech to text with industry-leading accuracy. Whisper handles different accents, filters background noise, and automatically punctuates the transcript.

3. AI-powered summarization

The full transcript is analyzed to extract key points, decisions, and action items. You can customize the summary length and detail level to match your team's needs.

4. ElevenLabs audio creation

The text summary is converted into a natural-sounding audio file using ElevenLabs' advanced text-to-speech technology. This creates portable meeting highlights perfect for busy team members.

5. Slack notification delivery

The complete package - full transcript, text summary, and audio summary - is delivered to designated Slack channels. You can configure different channels for different meeting types or teams.

Pro tip: Add a Google Drive step to archive all transcripts and audio files with consistent naming conventions for easy search and reference later.
