What This Workflow Does

This automation solves the problem of manually checking Spotify playlists for updates. Whether you're a music curator, playlist collaborator, or just want to stay informed about changes to your favorite playlists, this workflow eliminates the need for constant manual checking.

Every time a new track is added to any playlist you're monitoring, the system automatically sends a push notification through Google Chrome. This means you'll get instant alerts on any device where you're signed into Chrome, whether you're at your computer or using your mobile device.

How It Works

1. Monitoring Spotify Playlists

The workflow continuously checks your specified Spotify playlists for changes using Spotify's API. It can monitor multiple playlists simultaneously and detects new additions in near real-time.

2. Detecting New Tracks

When a new track is added, the system captures all relevant details including the song name, artist, album, and who added it (for collaborative playlists). This data is then prepared for notification.

3. Sending Chrome Notifications

Using Chrome's notification API, the system pushes a customized alert to your browser. Notifications can include clickable links to the new track, album artwork, and other metadata you choose to display.

Pro tip: Configure different notification styles for different playlists by duplicating and modifying the scenario for each important playlist.
