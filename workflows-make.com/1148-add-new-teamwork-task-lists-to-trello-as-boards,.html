What This Workflow Does

This automation bridges two powerful project management tools - Teamwork for detailed task management and Trello for visual workflow tracking. Every time your team creates a new task list in Teamwork, this workflow automatically generates a corresponding board in Trello with the same name and structure.

Teams often struggle with maintaining parallel systems or manually recreating project elements across platforms. This solution eliminates duplicate work while giving team members the flexibility to work in their preferred environment. Project managers get a unified view while team members can choose between Teamwork's robust features or Trello's intuitive kanban interface.

How It Works

1. Detects new Teamwork task lists

The workflow constantly monitors your specified Teamwork projects for newly created task lists using Make.com's Teamwork integration.

2. Extracts relevant task list details

When a new list appears, the automation gathers key information including the list name, project context, and any configured metadata.

3. Creates matching Trello board

Using Trello's API, the workflow generates a new board with the same name as the Teamwork task list, placed in your designated Trello team workspace.

4. Configures board settings

The automation applies your predefined board preferences - background colors, default lists (To Do, Doing, Done), permissions, and labels matching Teamwork's priority system.

What You'll Need

  1. Active accounts with admin access in both Teamwork and Trello
  2. Make.com account (free tier sufficient for basic usage)
  3. API keys for both Teamwork and Trello (available in each platform's settings)
  4. Clear mapping between Teamwork projects and Trello teams/workspaces

Quick Setup Guide

  1. Install the template in your Make.com account
  2. Connect your Teamwork account via OAuth
  3. Connect your Trello account via OAuth
  4. Specify which Teamwork projects to monitor
  5. Set the default Trello team for new boards
  6. Configure board naming conventions if desired
  7. Test by creating a new task list in Teamwork

Pro tip: Create a naming convention prefix (like "TW-") for auto-generated boards so team members can easily identify synced content versus manually created Trello boards.
