Automate Resource Guru Booking Emails with Microsoft 365

Get instant, professional email notifications sent automatically every time a new booking is created in Resource Guru.

What This Workflow Does

Managing project resources efficiently requires clear, timely communication. When a team member is booked on a project in Resource Guru, multiple stakeholders—project managers, clients, and the resources themselves—need to be notified immediately. Manually sending these emails is time-consuming, prone to errors, and often gets delayed in the hustle of daily operations.

This automation solves that problem by creating a seamless bridge between your resource scheduling and your communication platform. Every time a new booking is created in Resource Guru, this Make.com workflow triggers automatically. It captures all the relevant booking details—like the project name, resource person, dates, and notes—and uses them to compose and send a professional notification email directly from your Microsoft 365 account. This ensures everyone is on the same page instantly, without any manual intervention.

How It Works

The workflow operates on a simple trigger-action principle, transforming a booking event into a communication action.

Step 1: Monitoring Resource Guru

The scenario starts by watching your Resource Guru account for a specific event: the creation of a new booking. Make.com polls the Resource Guru API at regular intervals, instantly detecting when a new booking record appears.

Step 2: Gathering Booking Data

Once triggered, the workflow retrieves the complete set of data for that new booking. This includes the resource's name, the client or project they are booked for, the start and end dates, booked hours, and any custom fields or notes added during booking.

Pro tip: Use Resource Guru's custom fields to capture specific project codes or cost centers. The automation can include this data in the email, providing richer context for recipients.

Step 3: Composing & Sending the Email

The core data is then mapped to a pre-designed email template in Microsoft 365. You have full control to customize the subject line, email body, and recipient list. The email is sent directly from your official company email address via Microsoft 365, ensuring deliverability and maintaining brand consistency.

Quick Setup Guide

You can have this automation running in under 15 minutes.

  1. Clone the Template: Click "Get This Workflow" to copy the template into your Make.com account.
  2. Connect Resource Guru: In the first module, authorize Make.com to access your Resource Guru account using your API credentials.
  3. Connect Microsoft 365: In the email module, authorize the connection to your Microsoft 365 account.
  4. Map Your Data: Review the email module. Ensure the fields from Resource Guru (like Resource Name, Project, Dates) are correctly placed in your email subject and body.
  5. Set Recipients: Define the "To," "CC," or "BCC" fields. You can use static emails or, for advanced use, pull email addresses from another connected app.
  6. Test & Activate: Create a test booking in Resource Guru. Turn the scenario on and verify that the email sends correctly. Once confirmed, your automation is live.
