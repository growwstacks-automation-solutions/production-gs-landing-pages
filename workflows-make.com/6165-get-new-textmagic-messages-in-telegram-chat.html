What This Workflow Does

For businesses using TextMagic for SMS communication, a major challenge is message visibility. Texts arrive on a separate platform, often checked intermittently, leading to delayed responses and potential missed customer inquiries. This fragmentation hurts customer service and team efficiency.

This automation solves that by creating a seamless bridge between TextMagic and Telegram. Every time a new SMS message is received in your TextMagic account, this Make.com workflow instantly captures it and posts the details—like the sender's number, message content, and timestamp—into a designated Telegram chat or group. It transforms isolated SMS into collaborative, real-time team notifications.

How It Works

The workflow is a simple, reliable chain of events powered by Make.com's no-code automation platform.

Step 1: Trigger on New TextMagic Message

The scenario starts by monitoring your TextMagic account. The "Watch Messages" module acts as a trigger, polling for new inbound SMS at regular intervals. When a new message is detected, the workflow instantly activates.

Step 2: Format the Message for Telegram

Make.com then processes the raw data from TextMagic. It extracts key information such as the sender's phone number, the message body, and the time it was sent. This data is structured into a clear, readable format suitable for a chat environment.

Step 3: Send to Telegram Chat

Finally, the formatted message is sent via the Telegram Bot API to a specific chat ID you configure. This could be a private chat with yourself, a channel for customer support, or a team group where everyone can see incoming customer texts. The notification appears in Telegram just like any other message, ready for action.

Pro tip: Use a dedicated Telegram group for customer SMS. This creates a shared inbox where your team can assign, discuss, and respond to texts collaboratively, turning SMS into a team-managed channel.

What You'll Need

  1. A Make.com account (free tier available).
  2. An active TextMagic account with API access enabled.
  3. Your TextMagic API credentials (Username and API Key).
  4. A Telegram account and a Telegram Bot created via @BotFather.
  5. The Bot Token from @BotFather and the Chat ID of the destination Telegram chat or group.

Quick Setup Guide

Follow these steps to deploy this template in under 10 minutes:

  1. Clone the Template: Click "Get This Workflow" and duplicate the scenario into your Make.com account.
  2. Connect TextMagic: In the first module, add your TextMagic connection using your API username and key. Test the connection.
  3. Configure Telegram: In the "Send a Message" module, connect your Telegram Bot using the token from @BotFather. Paste the target Chat ID.
  4. Customize the Message: Adjust the message template if desired. You can add emojis, format the text, or include/exclude specific data fields.
  5. Activate & Test: Turn the scenario ON. Send a test SMS to your TextMagic number and verify it appears correctly in your Telegram chat.
