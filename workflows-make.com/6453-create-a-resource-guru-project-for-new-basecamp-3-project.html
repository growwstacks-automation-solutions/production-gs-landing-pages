What This Workflow Does

For agencies, consultancies, and internal project teams, launching a new project often involves a frustrating shuffle between tools. A project manager creates a project in Basecamp 3 to manage tasks and communication, but then must manually open Resource Guru to create a corresponding project for scheduling people and tracking capacity. This duplicate entry is time-consuming, prone to typos or missed details, and creates a lag between project approval and resource planning.

This automation solves that disconnect. It acts as a digital bridge between your project management and resource scheduling software. Every time a new project is created in Basecamp 3, this Make.com workflow triggers instantly. It captures the key project details—name, description, dates, client—and uses them to automatically create a matching project in Resource Guru. This ensures your resource plan is always aligned with your active project portfolio, giving you accurate visibility into team availability from the moment a project begins.

How It Works

The workflow is a straightforward, reliable sequence that runs silently in the background, requiring zero manual intervention after setup.

Step 1: Monitor Basecamp 3 for New Projects

The scenario starts with a Make.com "Watch Projects" module connected to your Basecamp 3 account. It continuously polls Basecamp for new project creations. The instant a project is added, the workflow captures the event and retrieves all relevant data.

Step 2: Format and Prepare the Project Data

Make.com then processes the incoming data. It maps Basecamp fields (like project name, description, start date) to the corresponding fields required by Resource Guru's API. This step can include data transformation, such as reformatting dates or combining fields to create a specific project code.

Pro tip: Use this step to add custom logic. For example, you can route projects to different Resource Guru "portfolios" based on the Basecamp team name or client code, ensuring perfect organizational alignment.

Step 3: Create the Project in Resource Guru

The formatted data is sent to Resource Guru via its API. The workflow instructs Resource Guru to create a new project, populating all the predefined fields. Upon successful creation, Resource Guru returns a confirmation, and the project immediately appears in your resource schedule, ready for team assignments.
