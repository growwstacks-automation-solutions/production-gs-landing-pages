What This Workflow Does

For professionals and teams using Todoist for task management, a common challenge is the disconnect between a task list and an actual schedule. Tasks often live in isolation, making it easy to overcommit or forget to allocate realistic time for them. This leads to missed deadlines, last-minute rushes, and poor work-life balance.

This automation bridges that gap. It acts as a smart bridge between Todoist and Google Calendar, automatically pulling your incomplete tasks and creating dedicated events for them. By converting tasks into calendar blocks, you enforce time commitment, visualize your day realistically, and significantly increase the likelihood of task completion. It eliminates the tedious, error-prone process of manually copying items between apps.

How It Works

The workflow runs on a schedule you define (e.g., every hour, each morning) to keep your calendar dynamically aligned with your priorities.

Step 1: Fetch Incomplete Tasks from Todoist

The scenario triggers and connects to your Todoist account via the API. It retrieves a list of all tasks marked as incomplete. You can configure filters at this stage—for example, only fetching tasks from specific projects, with certain labels, or that have a due date.

Step 2: Process and Format Task Data

Each retrieved task is processed. The workflow extracts key details: the task name, due date/time, project, priority, and any notes. It then formats this information to create a structured event title and description suitable for Google Calendar.

Pro tip: Use Todoist labels like "#15min" or "#DeepWork" in your task names. The automation can parse these and set corresponding event durations in Google Calendar automatically.

Step 3: Create Events in Google Calendar

The formatted data for each task is sent to Google Calendar's API. A new event is created with the task name as the title. The due date/time from Todoist becomes the event's start time, and a default or calculated duration is applied. Task details are added to the event description for context.

Step 4: (Optional) Prevent Duplicates

A robust setup includes a check to see if an event with a similar title already exists in the calendar for that time slot. This logic prevents the workflow from creating multiple events for the same task on subsequent runs, keeping your calendar clean.

Quick Setup Guide

  1. Clone the Template: Click the "Get This Workflow" button to open the template on Make.com and create a copy in your own account.
  2. Authorize Connections: In the Make.com scenario editor, connect your Todoist and Google Calendar accounts by following the OAuth prompts. This is a one-time, secure process.
