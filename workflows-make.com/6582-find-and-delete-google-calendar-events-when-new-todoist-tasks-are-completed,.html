Auto-Delete Google Calendar Events from Completed Todoist Tasks

What This Workflow Does

For professionals and teams who block time in Google Calendar for specific Todoist tasks, a common problem arises: when the task is finished, the calendar event remains. This creates clutter, misrepresents your availability, and forces manual cleanup. This disconnect between your task manager and your calendar wastes time and causes scheduling confusion.

This automation solves that by creating a seamless bridge between completion in Todoist and cleanup in Google Calendar. When you mark a task as done in Todoist, the workflow instantly finds the corresponding calendar event using a stored Event ID and deletes it. This ensures your calendar always reflects your current workload and available time slots, turning two separate tools into one unified system.

Pro tip: This template works best as the second half of a powerful duo. First, use a template to create Google Calendar events from new Todoist tasks. Then, this template handles the cleanup upon completion, creating a perfect open-and-close loop for your scheduled work.

How It Works

The workflow is triggered in Make.com and performs a precise sequence of actions to keep your calendars in sync.

Step 1: Monitor Todoist for Completed Tasks

The automation watches your specified Todoist project or label for any task that changes status to "completed." It captures the task's content, which must be the unique Google Calendar Event ID.

Step 2: Extract the Google Calendar Event ID

The workflow parses the completed task's content to find the Event ID. This ID is the critical link that tells Google Calendar exactly which event to target, ensuring safe and accurate deletion.

Step 3: Find and Delete the Calendar Event

Using the extracted ID, the automation searches your connected Google Calendar. Once it finds the matching event, it executes a delete command. The old time block is removed, instantly freeing up your schedule.

Quick Setup Guide

Get your automated calendar cleanup running in under 10 minutes.

  1. Clone the Template: Click "Get This Workflow" to copy the template into your Make.com account.
  2. Connect Your Apps: Authorize the Make.com modules to access your Todoist and Google Calendar accounts.
  3. Configure the Trigger: Set the Todoist module to watch the correct project or filter for completed tasks.
  4. Map the Data: Ensure the module that parses the task content is correctly pointed to the field containing the Google Calendar Event ID.
  5. Select the Calendar: In the Google Calendar module, specify which calendar (e.g., "Primary") contains the events to be deleted.
  6. Test and Activate: Run a test by completing a Todoist task linked to a calendar event. Confirm the event is deleted, then turn the scenario on.
