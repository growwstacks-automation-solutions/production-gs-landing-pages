What This Workflow Does

For businesses operating in or with the Czech Republic, managing finances with accurate, up-to-date currency exchange rates is non-negotiable. Relying on manual checks of the Czech National Bank (ČNB) website is a time-consuming and error-prone process that can lead to pricing mistakes, inaccurate financial reporting, and wasted operational hours.

This automation solves that problem completely. It is a pre-built Make.com scenario that acts as a reliable bridge between the official ČNB data source and your Google Sheets. Once set up, it runs silently on a schedule you define—daily, weekly, or even multiple times a day—to fetch the latest exchange rates for the Czech koruna (CZK) against major currencies like EUR, USD, GBP, and more. It then structures this data and inserts it directly into your designated spreadsheet, creating a clean, timestamped historical record.

The value is immediate: your financial models, invoice calculations, cost reports, and pricing strategies are always powered by the correct, official rates without any human intervention. This transforms a routine administrative task into a flawless, automated system.

How It Works

The workflow follows a logical, reliable sequence to move data from a public API to your spreadsheet.

Step 1: Scheduled Trigger

The automation begins with a Schedule module in Make.com. You configure it to run at your preferred time—for example, every weekday afternoon after the ČNB publishes its official daily rates. This trigger kicks off the entire process automatically, ensuring you never miss an update.

Step 2: Fetch Data from ČNB API

Make.com sends a request to the Czech National Bank's public API endpoint. This API returns a structured data file (typically in XML or JSON format) containing the official middle exchange rates for the day. The workflow is built to parse this specific format reliably.

Step 3: Parse and Filter Currency Data

The raw API response is processed. The workflow extracts the relevant exchange rates you care about (e.g., EUR/CZK, USD/CZK). It can filter out unnecessary data, convert number formats, and prepare a clean dataset ready for insertion.

Step 4: Update Your Google Sheet

Finally, the workflow connects to your Google Sheets via the Google Sheets module. It locates your pre-configured sheet and adds a new row of data. This row typically includes the date, timestamp, and each currency pair with its corresponding rate. This builds a running log, perfect for creating charts or historical analysis.

Pro tip: Pair this automation with a simple Google Sheets formula or a connected dashboard tool like Looker Studio. You can instantly create a real-time currency dashboard for your team, showing trends and current rates at a glance.
