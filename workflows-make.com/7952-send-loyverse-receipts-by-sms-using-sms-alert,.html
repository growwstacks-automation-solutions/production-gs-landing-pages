What This Workflow Does

This automation solves the challenge of providing customers with immediate purchase confirmations from Loyverse POS systems. Instead of printing paper receipts or relying on customers to check email, it automatically sends transaction details via SMS within minutes of each sale.

The workflow periodically checks Loyverse for new receipts, formats the purchase information into a customer-friendly SMS message, and delivers it through SMS Alert's reliable messaging platform. This creates a seamless digital receipt experience that improves customer satisfaction while reducing paper waste.

How It Works

1. Scheduled Receipt Checking

The template runs every 15 minutes (configurable) to check Loyverse for new receipts since the last run. This ensures timely delivery without overloading either system.

2. Receipt Data Processing

For each new receipt found, the workflow extracts key details like transaction ID, items purchased, totals, payment method, and customer phone number from Loyverse's API.

3. SMS Message Formatting

The system formats the receipt data into a concise SMS message that includes store name, purchase summary, total amount, and optional thank-you message or promotional content.

4. SMS Delivery via SMS Alert

The formatted message is sent through SMS Alert's API to the customer's mobile number. Delivery status is logged for reference.

Pro tip: Customize your SMS template to include your store's branding, loyalty program details, or special offers to turn receipts into marketing opportunities.
