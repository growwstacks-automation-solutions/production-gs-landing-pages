What This Workflow Does

This automation bridges the gap between your CRM and task management systems by automatically creating ClickUp tasks for every new contact in Initiative CRM. It eliminates manual data entry and ensures sales teams never miss following up with new leads.

The workflow runs every 15 minutes, checking for new contacts and transforming them into properly categorized tasks with relevant details pre-populated. This creates a seamless handoff between marketing-qualified leads and sales execution.

How It Works

1. CRM Contact Monitoring

The workflow polls Initiative CRM's API for new contacts added since the last check. It captures all relevant lead information including contact details, source, and any custom fields.

2. Data Transformation

Contact data is mapped to ClickUp task properties, converting CRM-specific fields into actionable task attributes like priority, due date, and assignment logic.

3. Task Creation

The system creates tasks in your specified ClickUp list with all mapped data, ensuring sales reps have complete context for follow-up without switching between platforms.
