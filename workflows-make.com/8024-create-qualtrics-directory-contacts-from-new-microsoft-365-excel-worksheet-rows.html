What This Workflow Does

This automation solves the tedious manual process of transferring contact data from Microsoft Excel spreadsheets to Qualtrics directories. Every time new rows are added to your specified Excel worksheet, the workflow automatically creates corresponding contact entries in your Qualtrics directory with all mapped fields.

Businesses using Excel for contact management waste countless hours copying and pasting data between systems. This workflow eliminates that inefficiency while ensuring data accuracy. The automation handles field mapping, format conversion, and error checking - reducing manual errors by 90% according to user reports.

How It Works

1. New Excel Row Detection

The workflow monitors your specified Microsoft 365 Excel worksheet for new row additions. It can track either a specific table or the entire worksheet depending on your configuration.

2. Data Extraction & Validation

When new rows appear, the system extracts specified columns and validates data formats against Qualtrics requirements. Invalid entries trigger error handling workflows.

3. Field Mapping Transformation

Your predefined field mapping configuration converts Excel column data into the exact format Qualtrics expects for each directory field.

4. Qualtrics API Integration

The workflow uses Qualtrics' API to create new directory contacts with all mapped fields. Successful creations are logged for auditing.

5. Error Handling & Notifications

Any failed records trigger notification emails with detailed error reports. The system can be configured to retry failures or pause for manual review.
