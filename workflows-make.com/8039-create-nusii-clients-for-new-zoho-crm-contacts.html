What This Workflow Does

This automation solves a common pain point for sales teams using Zoho CRM with Nusii for proposals. Every time you add a new contact in Zoho CRM, you typically need to manually create a corresponding client record in Nusii before you can send them proposals. This workflow eliminates that redundant data entry.

The integration watches for new contacts in your Zoho CRM and automatically creates matching client records in Nusii with all relevant contact details. This ensures your proposal tool always has up-to-date client information without any manual copying and pasting between systems.

How It Works

1. New Contact Detection

The workflow continuously monitors your Zoho CRM for newly created contacts. You can configure it to watch specific contact types or apply filters based on deal stage or other criteria.

2. Data Mapping

When a new contact is detected, the system maps relevant fields from Zoho CRM to Nusii's client fields. Standard mappings include name, email, phone, company, and address fields.

3. Client Creation

The workflow creates a new client in Nusii using the mapped data. It checks for existing clients with matching email addresses to prevent duplicates.

4. Error Handling

If any issues occur during the sync (like missing required fields), the workflow can notify your team via email or Slack so you can address the problem manually.
