What This Workflow Does

This automation solves the tedious manual process of shortening URLs and maintaining tracking records. Marketing teams, content creators, and social media managers often waste hours copying links between their YOURLS dashboard and spreadsheets. The workflow automatically:

  • Detects new URLs added to a designated Google Sheets column
  • Shortens them through your YOURLS instance
  • Logs both original and shortened URLs with timestamps
  • Maintains a clean, searchable database of all shortened links

How It Works

1. New URL Detection

The workflow monitors a specific Google Sheet for new URL entries. You can designate any column as the input source, making it adaptable to existing spreadsheets.

2. YOURLS API Connection

Each new URL gets passed to your YOURLS installation via its API. The system can use custom shortcodes if specified or generate random ones automatically.

3. Data Logging

The shortened URL, original URL, creation timestamp, and any custom metadata get written back to your Google Sheet in a structured format. This creates an auditable history of all shortened links.

Pro tip: Add a "Campaign" column to your sheet to categorize links by marketing initiative. The workflow can include this metadata in your tracking records.
