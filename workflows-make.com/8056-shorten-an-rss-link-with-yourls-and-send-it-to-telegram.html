What This Workflow Does

This automation solves the time-consuming process of manually checking RSS feeds, shortening links, and sharing them with your Telegram audience. It monitors your specified RSS feed for new items, automatically creates branded short URLs using YOURLS, and posts them to your Telegram channel with customizable messages.

For content creators and marketers, this eliminates hours of repetitive work while ensuring your audience gets timely updates with professional-looking short links. The workflow maintains consistency in your messaging and provides valuable click-through data through YOURLS analytics.

Pro tip: Use custom slugs in YOURLS to create memorable short links that reinforce your brand (e.g., yourbrand.com/news-jan24 instead of random characters).

How It Works

1. RSS Feed Monitoring

The workflow continuously checks your specified RSS feed for new items using Make.com's RSS module. It can monitor multiple feeds simultaneously and only processes new entries.

2. Link Shortening with YOURLS

Each new RSS item's URL is sent to your YOURLS installation, which generates a branded short link. The template supports custom slugs and tracking parameters.

3. Telegram Message Formatting

The workflow constructs a clean message using the RSS item's title, your custom branding, and the shortened link. You can customize the message template to match your channel's style.

4. Automated Posting

Make.com sends the formatted message to your specified Telegram channel or group through the Telegram Bot API. The workflow includes error handling to retry if there are temporary issues.

What You'll Need

  1. A Make.com account (free or paid)
  2. A working YOURLS installation (self-hosted or managed service)
  3. A Telegram bot token (create via @BotFather)
  4. The Telegram channel or group ID where messages should be posted
  5. The RSS feed URL you want to monitor

Quick Setup Guide

  1. Copy the template to your Make.com account
  2. Connect your YOURLS account in the "HTTP" module
  3. Add your Telegram bot token in the "Telegram" module
  4. Specify your target channel/group ID
  5. Customize the message template if desired
  6. Set the RSS feed URL and polling interval
  7. Activate the scenario
