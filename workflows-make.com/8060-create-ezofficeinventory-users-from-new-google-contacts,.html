What This Workflow Does

This automation solves the common problem of manually creating user accounts across multiple business systems. When new employees, contractors, or partners are added to your Google Contacts, the workflow automatically provisions corresponding user accounts in EZOfficeInventory - your asset management platform.

By eliminating duplicate data entry, you ensure consistent user information across systems while saving HR and IT teams hours of administrative work each week. The integration maintains data accuracy and provides immediate access to company assets for new team members.

How It Works

1. New Contact Detection

The workflow monitors your Google Contacts for new additions using Make.com's instant trigger. When a new contact appears, the system captures all relevant details including name, email, phone number, and organization information.

2. Data Transformation

Contact information is formatted to match EZOfficeInventory's user schema. You can customize field mappings during setup to ensure proper data alignment between systems.

3. User Creation in EZOfficeInventory

The workflow authenticates with your EZOfficeInventory account via API and creates a new user with the transferred contact details. Optional steps can assign default roles, departments, or asset groups.

4. Confirmation & Error Handling

The system verifies successful user creation and can notify administrators via email or Slack if any issues occur during the process.
