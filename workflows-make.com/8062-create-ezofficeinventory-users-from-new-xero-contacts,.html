Create EZOfficeInventory Users from New Xero Contacts

Automatically add new Xero contacts as users in your asset management system, eliminating manual data entry and ensuring immediate access

What This Workflow Does

This automation bridges the gap between your accounting system (Xero) and asset management platform (EZOfficeInventory). Every time a new contact is created in Xero - whether it's a customer, vendor, or employee - the workflow automatically provisions them as a user in EZOfficeInventory with all relevant contact information.

Manual user creation in asset management systems often falls through the cracks, leading to delays when employees or clients need to check out equipment. This workflow ensures that happens instantly and accurately, with no human intervention required. The integration maintains data consistency across both platforms while saving your team hours of administrative work each week.

How It Works

1. New Contact Detection

The workflow monitors your Xero account for new contact creations using Make.com's Xero connector. It can be configured to watch for specific types of contacts (customers, suppliers, etc.) based on your business needs.

2. Data Formatting

Contact information from Xero is formatted to match EZOfficeInventory's user profile requirements. This includes proper name formatting, address standardization, and mapping custom fields between the two systems.

3. User Creation

The formatted data is sent to EZOfficeInventory via API, creating a new user account with appropriate permissions. The workflow can be customized to assign default groups, locations, or access levels in EZOfficeInventory.

4. Verification & Error Handling

The workflow confirms successful user creation and logs any errors for review. It includes safeguards against duplicate entries and missing required fields.

Pro tip: Extend this workflow to send welcome emails with EZOfficeInventory login instructions when new users are created, creating a complete onboarding automation.
