What This Workflow Does

This automation solves a common pain point for IT and operations teams managing both Zendesk for support and EZOfficeInventory for equipment tracking. Whenever a new employee or team member is added to your Zendesk account, this workflow automatically creates a corresponding user in EZOfficeInventory with all the necessary details.

Manual user provisioning between these systems often leads to delays in equipment access, duplicate data entry errors, and security risks when former employees retain access. This automated solution ensures perfect synchronization between your helpdesk and asset management systems.

How It Works

1. New User Detection

The workflow monitors your Zendesk account for new user creation events through real-time webhooks or periodic checks.

2. Data Extraction

Essential user information like name, email, department, and role is extracted from the Zendesk user profile.

3. EZOfficeInventory Account Creation

The workflow uses EZOfficeInventory's API to automatically create a new user account with the extracted data, applying the appropriate permissions and access levels.

4. Error Handling

If any issues occur during the sync process, the workflow can notify your IT team via email or Slack while automatically retrying the operation.

Pro tip: Extend this workflow to automatically deactivate EZOfficeInventory accounts when users are removed from Zendesk, completing the full user lifecycle management.
