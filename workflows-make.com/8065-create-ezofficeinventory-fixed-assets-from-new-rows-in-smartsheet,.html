What This Workflow Does

This automation bridges the gap between your procurement tracking in Smartsheet and fixed asset management in EZOfficeInventory. Every time a new asset is added to your designated Smartsheet (whether through purchases, transfers, or new deployments), the workflow automatically creates a corresponding fixed asset record in EZOfficeInventory with all relevant details.

Manual asset creation is error-prone and time-consuming, especially when managing hundreds of assets across multiple locations. This workflow ensures your EZOfficeInventory always reflects current assets without requiring staff to manually enter the same information twice. It's particularly valuable for IT departments, facilities managers, and operations teams managing equipment inventories.

How It Works

1. New row detection in Smartsheet

The workflow monitors your specified Smartsheet for new rows added, which typically represent newly acquired assets. It captures all column data including asset details, purchase information, and assignment data.

2. Data transformation and mapping

Make.com transforms the Smartsheet data into the format EZOfficeInventory requires, mapping spreadsheet columns to the appropriate asset fields in EZOfficeInventory. This includes handling date formats, currency values, and category mappings.

3. Asset creation in EZOfficeInventory

The workflow uses EZOfficeInventory's API to create a new fixed asset record with all the mapped data. It includes error handling to retry if the API is temporarily unavailable and alerts administrators if creation fails.

4. Confirmation and logging

After successful creation, the workflow can optionally update the Smartsheet row with the EZOfficeInventory asset ID or send confirmation notifications to relevant team members via email or Slack.

Pro tip: Extend this workflow to automatically generate asset tags/barcodes and attach them to the Smartsheet row for easy physical labeling reference.
