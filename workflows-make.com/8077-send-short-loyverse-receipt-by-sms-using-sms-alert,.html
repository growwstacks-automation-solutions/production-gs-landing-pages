What This Workflow Does

This automation solves the common retail challenge of providing customers with immediate purchase confirmations while reducing paper waste. The workflow connects your Loyverse point-of-sale system with SMS Alert to automatically send digital receipts via text message whenever a new transaction occurs.

Instead of printing paper receipts or manually sending messages, this system runs in the background every 15 minutes (configurable), checking for new Loyverse receipts and forwarding the key details to customers via SMS. It handles the entire process from data extraction to message delivery without any staff intervention.

How It Works

1. Scheduled Check for New Receipts

The workflow begins by querying your Loyverse account at regular intervals to detect any new receipts that haven't been processed yet.

2. Extract Key Transaction Details

For each new receipt, the system pulls essential information like customer phone number, purchase total, items purchased, and transaction timestamp.

3. Format SMS Message

The workflow compiles the extracted data into a concise, readable SMS format that fits within standard message limits (160 characters).

4. Send via SMS Alert

The formatted message is delivered to the customer's phone number through your SMS Alert account, with delivery confirmation.

Pro tip: Customize the SMS template with your store name and a thank you message to enhance brand recognition and customer experience.
