What This Workflow Does

This automation solves the problem of delayed awareness when new tasks are added to shared Checkvist lists. Traditional email notifications often get buried in inboxes, causing critical tasks to be overlooked until it's too late. The workflow instantly pushes desktop notifications through Google Chrome whenever any team member adds a new item to your monitored Checkvist lists.

The system works by connecting Checkvist's API with Make.com's automation platform and the Chrome Notifications extension. When configured, it eliminates the need for constant manual refreshing of task lists while ensuring no new item goes unnoticed. This is particularly valuable for distributed teams, time-sensitive projects, and managers who need to stay informed about task additions in real-time.

How It Works

1. Monitoring Checkvist for New Items

The workflow starts by regularly polling your specified Checkvist list through its API. Make.com checks for new items at intervals you configure (typically every 1-5 minutes for optimal performance). When a new task is detected, the system captures all relevant details including the task name, description, assignee if specified, and any custom fields.

2. Processing the Task Data

Make.com then processes the task information to create a customized notification message. You can configure which fields to include in the alert, set priority indicators, and format the message for clarity. The system can also apply filters to only notify about certain types of tasks based on your criteria.

3. Delivering Chrome Notifications

The processed message is sent to the Chrome Notifications extension installed in your browser. This triggers an immediate desktop alert that appears even when Chrome is minimized. The notification includes the task details and remains visible until dismissed or clicked.

Pro tip: Combine this with Make.com's mobile app to get notifications on your phone when you're away from your computer, creating a complete alert system for your Checkvist tasks.

Quick Setup Guide

  1. Install the Chrome Notifications extension from the Chrome Web Store
  2. Copy the template from Make.com to your account
  3. Connect your Checkvist account in Make.com using API credentials
  4. Specify which Checkvist list(s) to monitor for new items
  5. Customize the notification message format to include relevant task details
  6. Set your preferred polling interval (start with 2 minutes for testing)
  7. Activate the scenario and test by adding a new item to your Checkvist list
