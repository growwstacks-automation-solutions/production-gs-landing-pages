What This Workflow Does

This automation solves the tedious manual process of shortening URLs collected through Google Forms. When respondents submit links (like website references, document URLs, or social media profiles), the workflow automatically processes them through dlvr.it's URL shortener and logs both the original and shortened versions in your Google Sheets spreadsheet.

Marketing teams, researchers, and event organizers frequently need to collect and share URLs through forms. Long, unwieldy links look unprofessional in communications and make analytics tracking difficult. This workflow ensures every submitted URL gets consistently shortened while maintaining a complete audit trail in your spreadsheet.

How It Works

1. New form submission triggers the workflow

The scenario monitors your Google Form for new responses containing URL fields. Each new submission instantly initiates the shortening process.

2. URL validation and processing

The system checks that the submitted text is a valid URL and prepares it for shortening. Invalid entries get flagged in your spreadsheet with error details.

3. dlvr.it API shortening

The workflow sends the URL to dlvr.it's shortening service through their API, which returns a compact, trackable link (like dlvr.it/ABC123).

4. Google Sheets logging

Both the original URL and shortened version get recorded in your designated spreadsheet along with timestamps and any other form response data you want to preserve.

Pro tip: Enhance this workflow by adding columns for click analytics from dlvr.it. Schedule daily updates to show each shortened link's performance metrics right in your spreadsheet.
