What This Workflow Does

This automation solves the critical communication gap between field teams using ArcGIS Field Maps and stakeholders who need immediate updates. When field workers add new features like utility repairs, construction progress, or environmental observations, the system automatically sends an SMS alert via Twilio to designated recipients.

The workflow eliminates manual notification processes that often delay critical information. Field supervisors, clients, or operations centers receive instant SMS updates containing key details about the new map feature without requiring the field worker to leave their mobile mapping application.

How It Works

1. New feature detection

The workflow constantly monitors your ArcGIS Field Maps layer for new features added by field personnel. Each time a field worker saves a new point, line, or polygon, the system triggers the automation.

2. Data extraction

Key attributes from the new feature are extracted, including location coordinates, timestamp, feature type, and any custom fields your team has configured in the field data collection form.

3. SMS composition

The system formats a concise SMS message containing the most critical information from the new feature. You can customize the message template to include specific fields relevant to your operations.

4. Twilio delivery

The composed message is instantly delivered via Twilio's SMS gateway to your predefined recipient list. The system handles delivery retries if any temporary network issues occur.

What You'll Need

  1. An ArcGIS Online organizational account with Field Maps configured
  2. A Twilio account with SMS capabilities enabled
  3. Make.com account (free tier sufficient for basic usage)
  4. Field data collection layers configured in ArcGIS
  5. List of SMS recipients (phone numbers with country codes)

Quick Setup Guide

  1. Import the template into your Make.com account
  2. Connect your ArcGIS Online account credentials
  3. Configure which layer to monitor for new features
  4. Connect your Twilio account and specify sender number
  5. Map the field data attributes to your SMS message template
  6. Test with sample field submissions before going live
