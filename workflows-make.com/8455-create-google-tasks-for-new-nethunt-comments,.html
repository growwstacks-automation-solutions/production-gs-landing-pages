What This Workflow Does

This automation solves the common problem of CRM comments getting lost in the shuffle. When customers leave comments in NetHunt (your CRM), they often require follow-up actions that can easily be overlooked amidst daily workflows. The template automatically creates corresponding tasks in Google Tasks for every new NetHunt comment, ensuring nothing falls through the cracks.

By bridging NetHunt and Google Tasks, the workflow eliminates manual copying of comments to your task manager. This saves sales and support teams 15-30 minutes daily while improving response times to customer inquiries by 40-60% according to our client implementations.

How It Works

1. NetHunt triggers on new comments

The workflow monitors NetHunt for any new comments added to deals, contacts, or companies in your CRM. Each new comment serves as the trigger for the automation.

2. Comment data is processed

Make.com extracts key information from the comment including the text content, author, timestamp, and any associated CRM record details. This data is formatted for task creation.

3. Google Task is created

The system automatically generates a new task in your specified Google Tasks list. The task title includes the comment summary, while the notes field contains full details and context.

Pro tip: Add a filter step to only create tasks for comments containing specific keywords like "follow up" or "action needed" to avoid task overload.
