What This Workflow Does

This automation solves the tedious process of manually copying data from Google Sheets and creating Pastebin entries. It's perfect for teams that need to frequently share spreadsheet content as text snippets with clients, collaborators, or support teams without granting direct access to the spreadsheet itself.

The workflow monitors your specified Google Sheet for new rows. When detected, it automatically formats the content and creates a Pastebin paste with your preferred settings (private/unlisted/public, expiration time, syntax highlighting). This eliminates repetitive manual work while maintaining control over what gets shared.

How It Works

1. New row detection

The workflow constantly monitors your Google Sheet for new additions using Make.com's Google Sheets trigger. You can specify which columns to include and set filters to only process relevant rows.

2. Data formatting

Before creating the paste, the workflow formats your spreadsheet data into clean text. It handles line breaks, removes unnecessary formatting, and can apply custom transformations based on your needs.

3. Pastebin creation

Using Pastebin's API, the workflow creates a new paste with your configured settings. You can set default parameters like visibility and expiration, or dynamically control them based on spreadsheet values.

4. Link management

Optionally, the workflow can store the generated Pastebin URLs back in your spreadsheet, send them via email or Slack, or add them to a link management system.

Pro tip: Use a dedicated "Status" column in your spreadsheet to track which rows have been processed. This prevents duplicates and helps with troubleshooting.
