What This Workflow Does

Educational administrators and institutional researchers spend countless hours manually extracting data from Canvas Learning Management System (LMS) to create reports, track enrollments, or analyze course performance. This process is not only tedious but prone to human error, leading to inaccurate data for critical decisions like accreditation reporting or resource allocation.

This automation workflow solves that problem by creating a seamless, scheduled bridge between Canvas LMS and Microsoft Excel. It automatically retrieves user course data—including student information, enrollment status, and course details—and appends it as new rows in a designated Excel worksheet. The result is a live, centralized dataset that updates without any manual intervention, giving your team a single source of truth for all Canvas-related analytics.

How It Works

The workflow operates on a trigger-based model, ensuring your Excel sheet is always current with the latest data from Canvas.

Step 1: Trigger the Data Fetch

The automation can be initiated on a schedule (e.g., every night at 2 AM) or by a specific event. It connects to your Canvas instance using secure API credentials and requests a list of users and their associated courses based on your configured parameters.

Step 2: Process and Structure the Data

Make.com receives the raw data from Canvas API. The workflow then parses this information, extracting key fields like user ID, name, email, course ID, course name, and enrollment state. It formats this data into a clean, consistent structure suitable for a spreadsheet row.

Step 3: Write to Microsoft Excel

The formatted data is sent to your connected Microsoft 365 account. The workflow locates your specified Excel workbook and worksheet, then appends the new row of course information at the bottom of your existing data. This maintains a historical log of all syncs.

Pro tip: Use Excel's built-in "Table" format for your target worksheet. This ensures the automation always appends data correctly, even as you add filters or new columns, and makes creating PivotTables for analysis much easier.

Quick Setup Guide

You can have this automation running in under 20 minutes.

  1. Clone the Template: Click "Get This Workflow" and it will be copied to your Make.com scenario list.
  2. Connect Canvas: In the first module, add your Canvas API key and specify your Canvas instance URL (e.g., yourschool.instructure.com).
  3. Connect Microsoft 365: Authorize Make.com to access your Microsoft account and select the target Excel file and worksheet.
  4. Map Your Data: Match the Canvas API fields (like 'user.name') to the corresponding column headers in your Excel sheet.
  5. Set the Schedule: Configure the trigger to run hourly, daily, or weekly based on your reporting needs.
  6. Test & Activate: Run a single test cycle to ensure data flows correctly into Excel, then turn the scenario on.
