Automatically Send Coupons to New WooCommerce Customers

Turn first-time buyers into loyal repeat customers with a personalized welcome discount delivered instantly after their first order.

What This Workflow Does

Acquiring a new customer is expensive, but many e-commerce stores miss the golden opportunity to secure that second, crucial purchase. This automation solves that by instantly rewarding a customer's first order with a personalized discount coupon for their next shop. It transforms a transactional relationship into the beginning of a loyal customer journey.

The workflow listens for new "Order Completed" events in your WooCommerce store. It then automatically generates a unique coupon code—with your specified discount, expiration date, and usage rules—and emails it directly to the customer. This hands-free system ensures every new buyer feels valued and is incentivized to return, boosting your customer lifetime value (LTV) without any manual effort from your team.

How It Works

The automation acts as a silent, efficient marketing assistant, working in the background from the moment an order is placed.

Step 1: Trigger on New Order Completion

The Make.com scenario is triggered whenever a new order status in WooCommerce changes to "Completed" (or a status you define). It captures all the customer's details, including their email address and order ID.

Step 2: Create a Dynamic Coupon Code

Using WooCommerce's API, the scenario creates a new coupon. You can customize the coupon's value (percentage or fixed amount), set an expiration date (e.g., valid for 30 days), and define usage limits. The code can be unique per customer, often using their order ID or name for personalization.

Step 3: Send the Personalized Email

Finally, the system sends a beautifully formatted email to the customer. This email thanks them for their first purchase and presents their exclusive coupon code, with clear instructions on how to use it. The entire process happens within minutes of their order being finalized.

Pro tip: For even greater impact, modify the workflow to only send coupons for orders above a certain value. This targets your most valuable new customers first and improves your return on investment.

Quick Setup Guide

You can have this automation running in under 30 minutes.

  1. Clone the Template: Click "Get This Workflow" to open the pre-built template in your Make.com account.
  2. Connect WooCommerce: In the first module, add your WooCommerce store URL and API credentials to establish the connection.
  3. Configure the Trigger: Set the scenario to watch for the "Order Updated" event and filter it to only run when the status is "Completed".
  4. Set Coupon Parameters: In the "Create a Coupon" module, define your discount amount, type, expiry date, and any usage restrictions.
  5. Customize the Email: Edit the email content module with your brand's voice, logo, and a compelling message that delivers the coupon code.
  6. Test & Activate: Run a test with a recent order to ensure the coupon is created and emailed correctly, then turn the scenario on.
