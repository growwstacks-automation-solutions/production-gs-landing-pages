What This Workflow Does

For accounting firms and bookkeepers using Dext (formerly Receipt Bank), managing a clean, up-to-date client list outside the platform is a common headache. You might need the list for reporting, client segmentation, onboarding new team members, or integrating with other systems like a CRM. Manually exporting this data weekly or monthly is repetitive, error-prone, and eats into valuable billable hours.

This automation solves that by creating a seamless bridge between Dext and Microsoft Excel. It automatically pulls your client database—including names, IDs, contact details, and statuses—and writes it into a structured Excel worksheet in Microsoft 365. The workflow is scheduled to run daily, ensuring your Excel sheet is always current without any manual intervention. It transforms a tedious administrative task into a reliable, background process that delivers accurate data exactly where you need it.

How It Works

The workflow acts as a scheduled data pipeline. It connects to your Dext account, fetches the client list, formats the information, and pushes it into your designated Excel file.

Step 1: Scheduled Trigger

A schedule module initiates the workflow every day at a set time (default 10:00 AM). This ensures the export runs consistently without you having to remember to start it.

Step 2: Fetch Clients from Dext

The automation authenticates with your Dext account via API and retrieves the list of clients. It can filter by active status, date added, or other criteria you configure.

Step 3: Transform Data Structure

The raw client data is mapped and formatted. You can choose which fields to export (e.g., client name, email, business number, last activity date) and how they should be arranged.

Step 4: Write to Microsoft Excel Worksheet

The formatted data is sent to your Microsoft 365 Excel file. The workflow can create a new worksheet, append to an existing one, or update specific rows, keeping the file organized and ready for use.

Pro tip: You can extend this workflow to also send an email notification with a snapshot of the export summary, or trigger a secondary process in another app whenever a new client is added to Dext.

Quick Setup Guide

Getting the workflow live takes about 15-20 minutes. Follow these steps to configure it for your business.

  1. Copy the template: Click "Get This Workflow" to open the template on Make.com and create a copy in your own account.
  2. Connect your apps: In the Make.com scenario, connect the Dext module to your Dext account and the Microsoft 365 Excel module to your Microsoft account.
  3. Configure data mapping: Select the specific Dext client fields you want to export and match them to the column headers in your Excel worksheet.
  4. Set the destination: Specify the exact Excel file and worksheet name where the data should be written. You can choose to create a new sheet or update an existing one.
  5. Adjust the schedule: Change the trigger time from the default 10:00 AM to a time that suits your workflow (e.g., early morning before work starts).
  6. Test and activate: Run a test execution to verify the data flows correctly. Then activate the scenario to let it run automatically on schedule.
