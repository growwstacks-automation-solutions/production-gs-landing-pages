What This Workflow Does

Understanding why users drop off in your conversion funnels is critical, but manually correlating Smartlook session recordings with funnel data is time-consuming and inconsistent. This automation solves that by creating a seamless bridge between qualitative user behavior (Smartlook) and quantitative analysis (Google Sheets).

The workflow automatically extracts funnel event data—such as user progress through a sign-up sequence, checkout steps, or feature adoption clicks—and logs each event into a structured Google Sheets spreadsheet. This includes timestamps, user identifiers, session URLs, and event metadata. Instead of jumping between tools, your product, marketing, and UX teams have a single source of truth to analyze drop-off points, measure step-by-step conversion rates, and prioritize fixes based on actual user behavior.

How It Works

This Make.com scenario acts as a scheduled data pipeline, fetching new funnel events and appending them to your designated spreadsheet.

Step 1: Scheduled Trigger

The scenario is triggered on a schedule (default is daily at 10:00 AM). You can easily adjust this frequency to hourly, weekly, or any custom interval based on how fresh you need your data.

Step 2: Fetch Funnel Events from Smartlook

Make.com connects to your Smartlook account via API and requests all new funnel events that have occurred since the last run. It handles authentication and pagination to ensure no data is missed.

Step 3: Parse and Structure the Data

Each raw event is processed. Key details like the funnel name, event step, user ID, session timestamp, device information, and the associated recording URL are extracted and formatted into clear columns.

Step 4: Add Rows to Google Sheets

The structured data for each event is appended as a new row in your pre-configured Google Sheets spreadsheet. The template can be set to add data to a specific tab, creating a running historical log of all funnel interactions.

Pro tip: Use a separate tab in your Sheet for raw data and another for analysis. Create pivot tables or charts on the analysis tab that automatically update as new data flows in, giving you real-time visual reports.

Quick Setup Guide

You can have this automation running in under 15 minutes.

  1. Clone the template: Click "Get This Workflow" and duplicate the scenario in your Make.com account.
  2. Connect Smartlook: In the first module, authenticate with your Smartlook API key (found in your Smartlook project settings).
  3. Connect Google Sheets: Authenticate the Google Sheets module and select (or create) the spreadsheet and worksheet where data should be added.
  4. Map your data: Ensure the funnel event fields from Smartlook are correctly mapped to the column headers in your Sheet (the template provides a default structure).
  5. Set the schedule: Configure the schedule module to run at your desired frequency (e.g., daily, hourly).
  6. Run once & monitor: Perform a test run to ensure data flows correctly into your Sheet, then activate the scenario.
