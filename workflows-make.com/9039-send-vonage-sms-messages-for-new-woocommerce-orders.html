What This Workflow Does

When a customer completes a purchase on your WooCommerce store, there's a critical moment of uncertainty. Did the order go through? When will it ship? Traditional email confirmations often get lost in crowded inboxes, leading to anxious customers and increased support queries. This automation solves that problem by sending instant SMS notifications via Vonage the moment an order is placed.

The workflow connects your WooCommerce store to Vonage's SMS API through Make.com. It automatically detects new orders, extracts key customer and order information, and sends a personalized text message to the customer's phone number. This creates a seamless, professional post-purchase experience that builds immediate trust and reduces the administrative burden on your team.

How It Works

The automation runs on a scheduled basis, checking for new orders and triggering SMS notifications without any manual intervention.

Step 1: Monitor for New WooCommerce Orders

Make.com periodically polls your WooCommerce store's API (typically every 1-15 minutes, configurable based on your needs) to check for newly created orders. When it detects an order with a status like "processing" or "completed," it captures all relevant data including order ID, customer details, products purchased, and total amount.

Step 2: Format the SMS Message

The workflow processes the order data to create a personalized SMS message. It can include dynamic variables like the customer's name, order number, product names, total cost, and estimated shipping timeline. You can customize the message template to match your brand voice and include only the most important information customers need immediately after purchasing.

Pro tip: Keep SMS messages under 160 characters for single-message delivery and higher readability. Include the order number and one key next step (like "shipping in 24h") for maximum impact.

Step 3: Send via Vonage SMS API

The formatted message is sent to Vonage's SMS API, which delivers it to the customer's mobile number. Vonage provides reliable global delivery, real-time status updates, and fallback options if needed. The automation can handle success/failure responses and log them for your records.

Quick Setup Guide

You can have this automation running in under 30 minutes with these simple steps:

  1. Get the template: Click "Get This Workflow" above to open the template in your Make.com account.
  2. Connect WooCommerce: Create a connection to your WooCommerce store using the Store URL, Consumer Key, and Consumer Secret from WooCommerce → Settings → Advanced → REST API.
  3. Configure Vonage: Connect your Vonage account using your API Key and API Secret from the Vonage dashboard.
  4. Set the recipient number: Specify the phone number field from WooCommerce that contains customer mobile numbers (ensure international format, e.g., +441234567890).
  5. Customize the message: Edit the SMS template to include your brand name, preferred wording, and the order variables that matter most to your customers.
  6. Test and activate: Run a test with a recent order, check the SMS delivers correctly, then activate the scenario to run automatically.
