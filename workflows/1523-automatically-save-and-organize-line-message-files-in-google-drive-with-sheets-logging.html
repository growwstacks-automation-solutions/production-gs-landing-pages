What This Workflow Does

This automation solves the common problem of losing important files shared through LINE messages. Customer service teams, sales reps, and field staff often receive critical documents via LINE - contracts, receipts, ID copies - that need proper archival. Manual saving leads to lost files, disorganization, and compliance risks.

The workflow automatically detects file attachments in incoming LINE messages, saves them to designated Google Drive folders with proper naming conventions, and creates a detailed log in Google Sheets. This creates a searchable archive while eliminating manual file management work.

Automatically organized LINE message files in Google Drive with metadata tracking

How It Works

1. LINE Message Trigger

The workflow monitors your LINE channel for new messages containing file attachments. It captures the file along with sender details and timestamp.

2. File Processing

Each file gets renamed using a consistent format (e.g. "2023-11-15_ClientName_Document.pdf") and has its metadata extracted. The system checks for duplicate files to prevent unnecessary transfers.

3. Google Drive Upload

Files get saved to predetermined Drive folders, which can be organized by date, department, or project. The folder structure automatically creates subfolders as needed.

Automated tracking sheet showing all transferred files with clickable links

4. Google Sheets Logging

A row gets added to your tracking sheet with file details including: original message text, sender info, timestamp, file type, Drive link, and any custom tags. This creates an audit trail.

Pro tip: Add a column in your Sheet for status tracking to manage follow-up actions on received files.
