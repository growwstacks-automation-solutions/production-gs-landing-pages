What This Workflow Does

This AI-powered stock analysis agent automates the process of generating comprehensive market reports by combining technical indicators with news sentiment analysis. Designed specifically for Hebrew/RTL markets, it processes financial data through GPT-4o to deliver actionable insights in a format tailored for traders and analysts.

The system monitors multiple data streams simultaneously - price movements, volume trends, and news sentiment - then synthesizes them into coherent reports. It identifies potential trading opportunities where technical setups align with positive sentiment, or warns when negative news contradicts bullish chart patterns.

Sample output showing combined technical analysis and news sentiment scoring

How It Works

1. Data Collection Phase

The workflow pulls real-time stock data from financial APIs, including price history, volume, and key technical indicators. Simultaneously, it scans Hebrew and English news sources for relevant financial updates using specialized RTL text processing.

2. Sentiment Analysis

GPT-4o analyzes each news article, assigning sentiment scores and identifying key themes. The system weights sources by credibility and tracks sentiment trends over time to distinguish meaningful shifts from noise.

3. Technical Assessment

Concurrent with news processing, the system evaluates technical conditions using moving averages, RSI, and other indicators. It identifies chart patterns and calculates statistical probabilities for various outcomes.

4. Synthesis & Reporting

GPT-4o combines the technical and sentiment data into narrative reports, highlighting convergences where both factors align. Reports include risk assessments, potential price targets, and suggested watch levels.
