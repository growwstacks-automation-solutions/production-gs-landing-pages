What This Workflow Does

This automation solves the challenge of manually responding to Telegram messages with images or documents. Businesses often waste time sending the same product photos, instructional images, or promotional materials repeatedly to different customers. The workflow automatically detects incoming Telegram messages and responds with predefined images based on triggers or keywords.

By implementing this solution, teams can ensure instant responses 24/7 while maintaining consistency in visual communication. The template serves as a foundation that can be customized to handle various use cases from customer support to marketing engagement.

Automated image response example in Telegram chat

How It Works

1. Message Trigger Setup

The workflow monitors a specified Telegram channel or chat for new incoming messages. You can configure it to watch for specific keywords, phrases, or respond to all messages.

2. Image Selection Logic

Based on the message content or other conditions, the workflow determines which image to send. This can be a static image file or dynamically selected from a database or cloud storage.

3. Automated Response

The system sends the selected image back to the user through Telegram's API. Advanced configurations can include captions, buttons, or follow-up messages.

What You'll Need

  1. A Telegram account with admin access to the target chat/channel
  2. n8n instance (cloud or self-hosted)
  3. Images stored in accessible location (local storage, cloud, or URL)
  4. Basic understanding of Telegram bot creation

Pro tip: Start with simple keyword triggers before building complex conditional logic. Test with a private chat before deploying to customer-facing channels.
