Automatically monitor Twitter and deliver relevant tweets to your team's Mattermost channels in real-time

What This Workflow Does

This n8n workflow solves the challenge of keeping your team informed about relevant Twitter conversations without requiring constant manual monitoring. It automatically checks Twitter every minute for new tweets matching your specified search criteria (like mentions of your brand, competitors, or industry keywords) and delivers them directly to your Mattermost channels.

For marketing teams, customer support departments, or product managers, this automation means never missing important tweets while eliminating hours of manual social media monitoring. The workflow handles all the technical complexity of connecting Twitter's API with Mattermost, including proper formatting of tweets and handling rate limits.

How tweets appear when delivered to a Mattermost channel

How It Works

1. Minute-by-Minute Trigger

The workflow activates every minute using n8n's scheduler. This ensures you get near real-time updates without overwhelming your Mattermost channels.

2. Twitter Search Execution

Each run queries Twitter's API using your configured search parameters. The template includes examples for searching specific hashtags, mentions, or keywords, which you can customize.

3. New Tweet Filtering

The workflow checks each returned tweet against the last processed timestamp, ensuring only new tweets are forwarded to Mattermost.

4. Mattermost Notification

Each qualifying tweet is formatted with relevant metadata (author, timestamp, engagement metrics) and posted to your specified Mattermost channel with appropriate formatting.

Pro tip: Add sentiment analysis to automatically flag urgent customer complaints or positive mentions that need immediate attention.

Quick Setup Guide

  1. Download the template file and import it into your n8n instance
  2. Configure your Twitter API credentials in the Twitter node
  3. Set up a Mattermost incoming webhook and add the URL to the workflow
  4. Customize the search query parameters to match your monitoring needs
  5. Test the workflow with the "Execute Workflow" button
  6. Activate the workflow for automatic minute-by-minute execution
