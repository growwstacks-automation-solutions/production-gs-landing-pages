Send private welcome messages to new WhatsApp group members with Evolution API

Automatically send personalized welcome messages to new members joining your WhatsApp groups

Download Template JSON · n8n compatible · Free

What This Workflow Does

Managing WhatsApp groups can be time-consuming, especially when new members join and need orientation. This workflow automates the process of sending personalized welcome messages to new group members privately, creating a professional first impression while saving administrators hours of manual work.

The solution uses Evolution API (a WhatsApp Business API solution) integrated with n8n to detect new group members and automatically send them customized welcome messages with important group information, rules, or resources - all without cluttering the main group chat.

Example of personalized welcome message sent via Evolution API

How It Works

1. New member detection

The workflow monitors your WhatsApp group for new member additions through Evolution API's webhook capabilities. When a new participant joins, the system captures their phone number and name.

2. Message personalization

Using n8n's templating engine, the system generates a personalized welcome message that can include the member's name, group name, and other dynamic content from your database or CRM.

3. Private message delivery

The workflow sends the welcome message directly to the new member's private WhatsApp chat (not the group), creating a more personal experience and keeping group chats clean.

What You'll Need

  1. An Evolution API account with WhatsApp Business access
  2. n8n instance (self-hosted or cloud)
  3. Admin access to the WhatsApp group you want to monitor
  4. Your welcome message template (HTML/text)

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Connect your Evolution API credentials in the HTTP Request node
  3. Configure the webhook URL in your Evolution API settings
  4. Customize the welcome message template with your content
  5. Test with a new group member joining
  6. Activate the workflow
