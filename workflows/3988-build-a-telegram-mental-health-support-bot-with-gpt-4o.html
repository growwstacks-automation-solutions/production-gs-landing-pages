What This Workflow Does

This n8n workflow template creates an AI-powered mental health support bot for Telegram using GPT-4o. It provides immediate, compassionate responses to users seeking emotional support, while incorporating safeguards for crisis situations. The bot can handle common mental health inquiries, offer coping strategies, and escalate urgent cases to human professionals.

Mental health organizations, universities, and employee assistance programs can deploy this bot to extend their support services. It reduces wait times for non-critical cases while maintaining 24/7 availability. The template includes conversation flows for anxiety, depression screening, and stress management techniques.

How It Works

1. Telegram message reception

The workflow connects to Telegram's API to receive incoming messages. It processes both text and voice messages (converted to text using speech recognition). Each new conversation starts with a welcome message explaining the bot's capabilities and limitations.

2. Emotional tone analysis

GPT-4o analyzes the message for emotional content and potential risk factors. The system scores urgency levels based on keywords, sentiment, and contextual clues. High-risk triggers immediately route the conversation to crisis protocols.

3. Context-aware response generation

For standard inquiries, GPT-4o generates responses tailored to the user's stated concerns. The bot maintains conversation history to provide coherent, personalized support across multiple interactions. Responses emphasize active listening and evidence-based techniques.

n8n workflow interface showing the GPT-4o integration setup

4. Escalation protocols

When detecting potential crisis situations, the bot provides immediate resources (hotline numbers, emergency contacts) and can notify designated human responders. The workflow includes customizable escalation thresholds based on your organization's policies.

What You'll Need

  1. Self-hosted n8n instance (community nodes required)
  2. Telegram bot token from BotFather
  3. OpenAI API key with GPT-4o access
  4. Optional: Crisis support contacts for escalation
  5. Optional: Webhook URL for human responder notifications

Quick Setup Guide

  1. Import the JSON template into your n8n instance
  2. Configure Telegram node with your bot token
  3. Add your OpenAI API key to the GPT-4o node
  4. Set escalation thresholds in the Risk Assessment node
  5. Test conversation flows using the n8n UI
  6. Deploy the workflow and share your bot's Telegram link
