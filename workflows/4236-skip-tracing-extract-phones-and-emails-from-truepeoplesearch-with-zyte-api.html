What This Workflow Does

This automation solves the time-consuming challenge of manually searching for contact information across multiple sources. It automates skip tracing by extracting phone numbers and email addresses from TruePeopleSearch using the Zyte API, then formats the data for easy use in your CRM or outreach tools.

The workflow handles all the technical complexities of web scraping - managing proxies, bypassing CAPTCHAs, and parsing unstructured data - delivering clean, structured contact information ready for your business processes. What typically takes hours of manual searching can now be completed in minutes with consistent, reliable results.

Sample output showing extracted phone numbers and email addresses

How It Works

1. Input Preparation

The workflow accepts a list of names and locations (or last known addresses) in CSV format. This could be leads from your CRM, debtors list, or any other contact database.

2. TruePeopleSearch Query

Using the Zyte API, the workflow automatically queries TruePeopleSearch for each person in your list. The API handles all the technical aspects of web scraping while complying with TruePeopleSearch's terms of service.

3. Data Extraction

The workflow parses the HTML response to extract phone numbers, email addresses, and other available contact information. Zyte's advanced parsing ensures you get structured data instead of raw HTML.

Technical workflow showing the data extraction steps

4. Results Compilation

All extracted contact information is compiled into a clean, standardized format. The workflow can output results to your preferred destination: CSV file, Google Sheets, CRM, or any other system you use.

Pro tip: For best results, include as much identifying information as possible (full name, last known city/state) to improve match accuracy.
