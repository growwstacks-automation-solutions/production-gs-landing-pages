What This Workflow Does

This n8n automation solves the common challenge of manually tracking e-commerce orders across multiple platforms. It automatically captures every new order from your e-commerce system and logs it in a Google Sheets spreadsheet with intelligent organization by month.

The workflow creates a new tab each month automatically, ensuring your order history remains organized and accessible. It also tracks order status changes (like payment received, shipped, delivered) so you always have real-time visibility into your fulfillment pipeline without manual updates.

Automatically generated monthly tabs keep your order history organized

How It Works

1. Order Capture

The workflow connects to your e-commerce platform's API to detect new orders as they come in. It captures all relevant order details including customer information, products, quantities, prices, and shipping details.

2. Monthly Sheet Creation

At the start of each month, the automation checks if a tab for the current month exists. If not, it creates a new sheet with standardized column headers, maintaining consistency across all months.

3. Data Logging

Each order is appended to the current month's tab with all relevant details. The system timestamps each entry and can include calculated fields like taxes, discounts, or shipping costs.

Real-time status tracking for each order in your spreadsheet

4. Status Updates

The workflow periodically checks your e-commerce platform for status changes and updates the corresponding order rows in Google Sheets. This creates an audit trail of each order's journey from placement to fulfillment.
