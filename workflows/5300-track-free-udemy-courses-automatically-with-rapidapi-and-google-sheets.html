What This Workflow Does

This automation solves the problem of manually checking for free Udemy course offers, which are often time-limited and easy to miss. The workflow connects to Udemy's Coupons and Courses API through RapidAPI to fetch current free course offerings, filters them based on your criteria, and automatically updates a Google Sheet with the details.

By running hourly checks, you'll never miss a free learning opportunity again. The system can be configured to send email or Slack notifications when new free courses matching your interests become available, helping you build your skills without constantly monitoring Udemy's website.

Sample output showing free Udemy courses tracked in Google Sheets

How It Works

1. API Connection Setup

The workflow begins by authenticating with RapidAPI and connecting to the Udemy Coupons and Courses API endpoint. This provides access to current course offerings and discount information.

2. Data Filtering

The raw API response is processed to extract only free courses, filtering out paid offerings. Additional filters can be applied based on course categories, ratings, or other metadata.

3. Google Sheets Update

Filtered course data is formatted and appended to a designated Google Sheet, maintaining a historical record of all free courses discovered. The sheet includes course title, instructor, rating, URL, and expiration date.

What You'll Need

  1. A RapidAPI account with access to the Udemy Coupons and Courses API
  2. An n8n instance (cloud or self-hosted)
  3. A Google Sheets document set up with the appropriate column headers
  4. Google Service Account credentials for API access

Pro tip: Create a separate Google Sheet tab for each month to maintain organized historical data without creating an unwieldy single sheet.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure your RapidAPI credentials in the HTTP Request node
  3. Set up your Google Sheets connection in the Google Sheets node
  4. Adjust filters to match your course preferences (category, rating thresholds)
  5. Test the workflow and schedule it to run hourly
