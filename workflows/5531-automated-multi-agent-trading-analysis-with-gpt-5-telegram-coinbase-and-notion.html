What This Workflow Does

This advanced n8n workflow automates cryptocurrency trading analysis by coordinating multiple specialized GPT-5 agents that each focus on different market aspects. The system integrates real-time Coinbase market data with AI-powered valuation models, sentiment analysis scraped from news and social media, technical indicators, and macroeconomic factors - then synthesizes trading signals delivered via Telegram with supporting documentation saved in Notion.

Traders waste countless hours manually monitoring charts, news, and social sentiment while struggling to synthesize all this information objectively. Our multi-agent approach solves this by having specialized AI agents work in parallel to analyze different market dimensions, then convene to reach consensus recommendations. This eliminates emotional bias while ensuring no critical analysis dimension gets overlooked.

The workflow architecture showing how different AI agents specialize in distinct analysis types

How It Works

1. Data Collection Phase

The workflow begins by simultaneously pulling real-time market data from Coinbase's API, scraping news/social media for sentiment analysis, and retrieving macroeconomic indicators. These data streams feed into separate processing pipelines optimized for each data type.

2. Multi-Agent Analysis

Five specialized GPT-5 agents activate in parallel - Technical Analyst, Sentiment Tracker, Fundamental Valuator, Macro Observer, and Risk Manager. Each agent receives tailored data inputs and applies its unique analytical framework, outputting scored recommendations.

3. Consensus Engine

The workflow's consensus layer evaluates all agent recommendations using weighted scoring based on your trading strategy preferences. It resolves conflicts between agents (like bullish technicals vs bearish fundamentals) using predefined priority rules.

Example of how agent analyses are compiled into a consensus recommendation

4. Alert Generation

Approved trading signals trigger customizable Telegram alerts containing the trade rationale, risk parameters, and supporting data. All analyses and decisions automatically log to your Notion trading journal with rich formatting for easy review.

Pro tip: Configure different Telegram channels for various alert types (e.g., high-confidence signals vs watchlist observations) to prioritize your attention.
