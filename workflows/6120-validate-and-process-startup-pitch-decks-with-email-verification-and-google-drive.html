How It Works

1. Submission Intake

The workflow triggers when a new pitch deck is submitted via form, email attachment, or cloud storage. It immediately checks file type (PDF/PPT), size limits, and basic document structure.

2. Email Verification

Each submitter's email address undergoes multi-layer validation including domain checks, syntax verification, and deliverability testing. Invalid emails trigger automatic rejection notices.

3. Google Drive Processing

Approved decks are automatically filed in designated folders with standardized naming conventions. The system can add metadata, generate share links, and notify review teams.

Pro tip: Configure custom metadata fields for tracking vertical, stage, or geographic tags that match your investment thesis.
