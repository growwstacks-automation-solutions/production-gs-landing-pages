What This Workflow Does

This automation solves the time-consuming process of creating blog content from scratch. By combining GPT-4O's advanced language capabilities with Pixabay's image library, it generates complete WordPress post drafts including formatted text, SEO elements, and relevant featured images - all from a simple content brief.

The system transforms content creation from a multi-hour task to a matter of minutes. Marketing teams can maintain consistent publishing schedules without sacrificing quality or spending excessive time on repetitive writing tasks.

AI-generated blog post with automatically selected Pixabay featured image

How It Works

1. Form submission triggers the workflow

The process begins when a team member submits a content brief through a simple form. This includes the topic, target keywords, desired word count, and any specific instructions.

2. GPT-4O generates the article content

The workflow sends the brief to GPT-4O which returns a well-structured blog post draft with headings, paragraphs, and proper formatting. The AI follows your predefined style guidelines.

3. Pixabay provides matching images

Based on the article content, the system searches Pixabay's library for relevant, high-quality images. It selects the best match and properly attributes it.

4. WordPress draft creation

The completed package - formatted text and featured image - gets published as a draft in WordPress for final review and publishing.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure your WordPress API credentials in the HTTP Request node
  3. Add your OpenAI and Pixabay API keys to the respective nodes
  4. Test with a sample content brief to verify the workflow
  5. Create a simple form for your team to submit content requests
