What This Workflow Does

This automated workflow solves the challenge of consistently creating inspiring, poetic content without daily creative effort. Each morning at a scheduled time, the system generates a fresh haiku poem using OpenRouter AI, formats it in the traditional 5-7-5 syllable structure, saves it to a Google Docs archive, and emails it to your inbox.

For businesses and individuals who value creativity but lack time for daily writing, this automation delivers poetic inspiration on autopilot. The AI maintains high-quality output while eliminating the need to manually compose poems every day. The system creates a searchable archive in Google Docs while ensuring you never miss your morning dose of creativity.

Sample haiku automatically formatted and saved in Google Docs

How It Works

1. Scheduled Trigger

The workflow activates each morning at your specified time using n8n's scheduler. This ensures consistent daily generation without manual intervention.

2. AI Haiku Generation

OpenRouter AI receives a carefully crafted prompt requesting a haiku following strict 5-7-5 syllable structure. The AI model returns multiple options, and the system selects the best one.

3. Formatting & Validation

The workflow checks syllable count and structure, making minor adjustments if needed to ensure perfect haiku format before proceeding.

4. Google Docs Archiving

Each poem is appended to a master Google Docs file with date stamps, creating a permanent, searchable collection of your daily haikus.

5. Email Delivery

The formatted haiku is sent via Gmail to your specified address, arriving in your inbox as morning inspiration.
