What This Workflow Does

This automation transforms your email management by converting important Gmail messages into natural-sounding voice notes delivered directly to your Telegram account. The system intelligently processes incoming emails using GPT-5 to extract key information, then converts the summarized content into high-quality audio using Inworld's advanced text-to-speech technology.

Busy professionals save an average of 5-7 hours per week by listening to their inbox during commutes or downtime instead of manually reading each message. The AI-powered summarization ensures you hear only the most critical information from each email, filtering out fluff and signatures while preserving action items and deadlines.

The workflow extracts key content from emails and converts it to natural-sounding speech

How It Works

1. Email Trigger and Filtering

The system monitors your Gmail inbox for new messages matching your specified criteria (specific senders, labels, or keywords). You can configure filters to exclude newsletters or low-priority emails from conversion.

2. AI-Powered Summarization

GPT-5 analyzes the email content, identifying the main purpose, key points, and any action items. The AI removes signatures, disclaimers, and repetitive information while preserving critical details.

3. Voice Conversion

The summarized text passes to Inworld's TTS engine which converts it to natural-sounding speech. You can customize the voice characteristics (gender, tone, speed) to match your preferences.

Voice messages appear in your Telegram app ready for playback

4. Telegram Delivery

The final audio file gets sent as a voice message to your designated Telegram chat or channel. The message includes the sender's name and original subject for context.

Pro tip: Create a dedicated Telegram channel for your email summaries to keep them organized separately from personal messages.

What You'll Need

  1. A Gmail account with access to API connections
  2. A Telegram account and bot API token
  3. Access to GPT-5 (through OpenAI API or equivalent)
  4. Inworld TTS API credentials
  5. An n8n instance (self-hosted or cloud)

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Connect your Gmail account in the first node
  3. Configure your GPT-5 API credentials in the AI processing node
  4. Set up your Inworld TTS API connection
  5. Add your Telegram bot token and destination chat ID
  6. Test with a sample email and adjust summarization settings as needed
