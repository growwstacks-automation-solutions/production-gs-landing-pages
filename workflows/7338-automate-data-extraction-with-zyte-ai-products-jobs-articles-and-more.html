What This Workflow Does

This n8n workflow automates the extraction of structured data from websites using Zyte's AI-powered web scraping API. It solves the challenge of manually collecting product details, job listings, or article content from multiple sources - a process that typically consumes hours of repetitive work.

The template handles the entire data pipeline: from sending URLs to Zyte's API, processing the extracted data, to formatting the results for analysis. Unlike simple scrapers, it intelligently identifies key elements like product specifications, job requirements, or article bodies regardless of website layout variations.

Zyte AI automatically identifies and extracts product details from e-commerce pages

How It Works

1. URL Input Configuration

The workflow accepts a list of target URLs from a spreadsheet, database, or manual input. You can specify whether the pages contain products, jobs, or articles to optimize extraction accuracy.

2. Zyte API Processing

Each URL gets sent to Zyte's AI endpoint which analyzes the page structure, identifies relevant data fields, and returns clean JSON. The API handles JavaScript rendering, pagination, and anti-bot measures automatically.

3. Data Normalization

Extracted data gets standardized into consistent formats - prices converted to numbers, dates formatted uniformly, and empty fields handled gracefully. This ensures analysis-ready output regardless of source website variations.

4. Output & Integration

The workflow can save results to Google Sheets, Airtable, or databases. Optional steps include sending email alerts for price drops or new job postings matching your criteria.

What You'll Need

  1. An n8n instance (cloud or self-hosted)

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance (Settings > Workflows > Import)
  3. Add your Zyte API key in the credentials section
  4. Configure your input URLs (can be from a spreadsheet or database)
  5. Set your output destination (Google Sheets, Airtable, etc.)
  6. Test with 2-3 URLs to verify data extraction
  7. Schedule automatic runs as needed
