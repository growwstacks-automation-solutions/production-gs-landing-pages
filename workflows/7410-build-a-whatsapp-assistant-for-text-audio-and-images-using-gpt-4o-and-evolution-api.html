What This Workflow Does

This automation creates an intelligent WhatsApp assistant that handles customer inquiries through text messages, voice notes, and images. When a customer sends a message, the system uses GPT-4o to analyze the content (whether text, audio, or visual) and generates a contextually appropriate response. The Evolution API ensures reliable message delivery back to WhatsApp.

For businesses, this means 24/7 customer support without human intervention for common queries. The AI can answer product questions, process simple requests, and even interpret photos (like reading product codes or understanding support issues). During peak hours or overnight, it maintains consistent response quality when your team isn't available.

AI assistant handling mixed media WhatsApp messages

How It Works

1. Message Reception

The workflow triggers when a new message arrives via Evolution API's WhatsApp integration. It captures the message content, whether text, audio file, or image attachment.

2. Content Processing

For audio messages, the system transcribes to text using GPT-4o's speech recognition. Images are analyzed for visual content. Text messages are processed directly for intent and context.

3. AI Response Generation

GPT-4o generates a natural language response based on the processed input. The system can reference product databases or knowledge bases to ensure accurate information.

Multimodal message processing flow

4. Response Delivery

The generated response is sent back through Evolution API to the customer's WhatsApp. The system maintains conversation history for context in subsequent messages.

Pro tip: Configure response templates for common queries to ensure brand voice consistency while still allowing AI flexibility for unique questions.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. Evolution API account for WhatsApp integration
  3. OpenAI API key with GPT-4o access
  4. WhatsApp number (can be a new number specifically for automation)

Quick Setup Guide

  1. Download and import the JSON workflow into your n8n instance
  2. Configure Evolution API credentials in the WhatsApp node
  3. Add your OpenAI API key to the GPT-4o node
  4. Set up response templates for common queries (optional)
  5. Test with your WhatsApp number before going live

Pro tip: Start with a limited rollout to test response quality, then gradually increase volume as you refine the AI's knowledge base and response patterns.
