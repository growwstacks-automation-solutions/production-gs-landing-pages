What This Workflow Does

This automation solves the common productivity challenge of starting each day with an unorganized task list or forgetting to migrate unfinished items. The workflow runs two automated sequences every morning: First, it clears out any uncompleted tasks from the previous day at 5:00 AM, then at 5:10 AM it populates your Todoist with a fresh set of predefined daily tasks.

By automating this routine, you eliminate the mental overhead of manually managing your task list each morning. The system ensures consistency in your daily workflow while preventing task list clutter from accumulating over time. This is particularly valuable for professionals who follow structured daily routines or managers who need to track team tasks systematically.

Example of automated daily tasks appearing in Todoist

How It Works

1. Scheduled Trigger

The workflow activates automatically each morning using n8n's scheduler. The first trigger fires at 5:00 AM to initiate the cleanup process.

2. Task Cleanup

The system connects to your Todoist account via API and identifies all incomplete tasks from previous days. These are either deleted or moved to an archive project, depending on your configuration.

3. Fresh Task Creation

At 5:10 AM, the second sequence runs, creating a new set of tasks in your specified Todoist project. These can include recurring daily items, priority tasks, or templated task lists.

Pro tip: Combine this with other automations to pull tasks from your calendar, email, or project management tools for a comprehensive daily plan.
