What This Workflow Does

This n8n workflow automates the creation of professional AI avatar videos using VEED's Fabric 1.0 technology. It transforms spreadsheet data (like customer names, product details, or training content) into personalized talking head videos at scale. The workflow supports both OpenAI and ElevenLabs for text-to-speech conversion, giving you flexibility in voice selection and quality.

Businesses use this automation to generate hundreds of personalized video messages for marketing campaigns, onboarding sequences, product announcements, or customer support responses without manual video editing. The workflow handles everything from data processing to final video rendering and distribution.

Example output of an AI-generated talking head video from VEED

How It Works

1. Data Preparation

The workflow starts by reading rows from your spreadsheet (Google Sheets, Airtable, or Excel). Each row contains the script, avatar image reference, and any personalization variables needed for the video.

2. Text Processing

Using OpenAI or custom logic, the workflow processes the raw text into natural-sounding scripts optimized for voice synthesis. This includes handling variables like names, dates, or product details that need personalization.

3. Voice Generation

The workflow sends the processed text to either OpenAI's text-to-speech API or ElevenLabs for high-quality voice synthesis. You can configure different voices, tones, and speaking styles per video.

Configuration options for voice generation in the workflow

4. Video Rendering

VEED's API combines the generated audio with your selected avatar image to create the final talking head video. The workflow handles all API calls and monitors the rendering process.

5. Distribution

Completed videos can be automatically uploaded to YouTube, sent via email, posted to social media, or saved to cloud storage based on your configured rules.
