Automatically Export Google Sheets to Dropbox

Free n8n workflow template that transfers spreadsheet data to Dropbox as Excel files every 15 minutes

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the common challenge of manually exporting and sharing spreadsheet data between teams or systems. Many businesses use Google Sheets for collaborative editing but need Excel files in Dropbox for archival, reporting, or integration with other tools.

The workflow automatically reads data from your specified Google Sheet, converts it to XLSX format, and uploads it to your Dropbox account on a schedule you define (default: every 15 minutes). This eliminates repetitive manual exports while ensuring stakeholders always have access to the latest data version.

How It Works

1. Scheduled Trigger

The workflow activates automatically based on your configured schedule (default: every 15 minutes). n8n's scheduler is highly flexible, allowing minute-by-minute, hourly, or daily execution as needed.

2. Google Sheets Data Fetch

Using OAuth authentication, the workflow securely accesses your specified Google Sheet and reads all data from the designated worksheet. It handles pagination automatically for large spreadsheets.

3. Excel Conversion

The raw sheet data gets converted to XLSX format while preserving formulas, conditional formatting, and cell styling. You can customize the output filename to include timestamps or version numbers.

4. Dropbox Upload

The converted Excel file uploads to your chosen Dropbox folder path. The workflow includes error handling to retry failed uploads and avoid duplicate files.

Pro tip: Add a timestamp to filenames (like "Sales-Report-{DATE}.xlsx") to maintain version history without overwriting previous exports.

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance (Settings → Workflows → Import)
  3. Connect your Google Sheets account in the first node
  4. Connect your Dropbox account in the upload node
  5. Configure the spreadsheet ID and target folder path
  6. Adjust the schedule trigger if needed
  7. Test and activate the workflow
