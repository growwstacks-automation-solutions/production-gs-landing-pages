Automate social media content aggregation to a Telegram channel. This n8n template automatically aggregates and analyzes key updates from your social media platforms.

What This Workflow Does

This automation solves a common business problem: information overload from social media. Instead of manually checking Twitter/X (and other platforms) for updates, this workflow automatically collects, analyzes, and delivers curated content directly to your team via Telegram.

It's designed for marketers, brand managers, and analysts who need to stay informed about industry trends, competitor announcements, and relevant conversations without spending hours scrolling. The template uses BrowserAct to fetch posts, Google Gemini AI to summarize and deduplicate content, and Telegram to deliver clean, actionable intelligence.

The result is a private intelligence feed that updates automatically, giving your team a competitive edge with minimal effort. You get the key insights without the noise.

How It Works

Step 1: Scheduled Trigger & Content Collection

The workflow starts on a schedule you set (daily, hourly, etc.). It triggers BrowserAct to scrape the latest posts from specified Twitter/X accounts or search terms. This happens automatically, so you never miss important updates.

Step 2: Data Processing & AI Analysis

Raw posts flow into a series of conditional nodes that monitor the data processing job. Once complete, the content passes to Google Gemini. The AI agent summarizes lengthy threads, removes duplicate mentions, and extracts key points based on your instructions.

Step 3: Content Formatting & Decision Making

An "If" node checks each post for images. Posts with images are formatted as rich media messages; text-only posts become clean text updates. This ensures optimal presentation in Telegram.

Step 4: Delivery to Telegram Channel

The curated, analyzed content is sent to your designated Telegram channel. Your team receives a clean, organized digest of the most important social media updates, ready for review and action.

What You'll Need

  1. BrowserAct Account: For accessing and scraping Twitter/X content through their API.
  2. BrowserAct "Twitter/X Content Aggregation" Template: Pre-configured workflow within BrowserAct that this n8n template calls.
  3. Google Gemini API Key: For AI-powered content analysis and summarization.
  4. Telegram Bot Token & Channel: Create a Telegram bot via @BotFather and set up a channel where the bot can post.
  5. n8n Instance: Either self-hosted or n8n.cloud account to run the workflow.

Quick Setup Guide

  1. Import the Template: Download the JSON file above and import it into your n8n instance.
  2. Configure BrowserAct: Set up the "Twitter/X Content Aggregation" template in your BrowserAct account and note your Workflow ID.
  3. Add Credentials: In n8n, add your BrowserAct API key, Google Gemini API key, and Telegram bot token to the credential manager.
  4. Update Workflow ID: In the HTTP Request node (Run Node), replace the placeholder workflow_id value with your actual BrowserAct workflow ID.
  5. Test & Activate: Run the workflow once manually to verify everything works, then activate the schedule trigger for automatic operation.

Pro tip: Start by monitoring 2-3 key accounts before scaling up. This lets you fine-tune the AI prompts and Telegram formatting before adding more sources.
