What This Workflow Does

Brands and marketers spend hours manually searching Google for product reviews, reading through multiple websites to gauge general sentiment, and struggling to spot recurring customer complaints or praises without aggregating data. This process is tedious, inefficient, and often misses critical insights buried in scattered feedback.

This n8n workflow solves these problems by automating the entire review analysis pipeline. It instantly searches and scrapes review content from the web using Decodo, uses Google Gemini AI to read and score the sentiment of every review found, and generates a consolidated “Executive Summary” with key quotes and actionable advice. You get a clear, data-driven snapshot of public perception without manual labor.

The automation turns unstructured online chatter into structured business intelligence. It delivers sentiment scores, highlights common themes, and extracts direct customer voice—allowing you to respond to market feedback faster and make informed product or marketing decisions.

How It Works

Step 1: Trigger via Telegram

You send a company or product name (e.g., “XQ Pharma”) to your configured Telegram bot. This message acts as the workflow trigger, initiating the automated research process.

Step 2: Automated Web Search & Scraping

The workflow uses Decodo to perform a Google search for “[Name] reviews” and extracts the top URL results. Decodo then visits each review page, strips away the HTML, and returns clean text content ready for analysis.

Step 3: AI Sentiment Analysis Loop

Each scraped review text is passed to a Google Gemini AI Agent configured as a “Strict Data Analyst.” The AI reads the text, determines sentiment (Positive/Neutral/Negative), identifies key topics, and extracts notable customer quotes.

Step 4: Executive Summary Generation

A second AI Agent collects all the individual analysis pieces and compiles a final report. This summary includes an overall Sentiment Score, a “Customer Voice” section with direct quotes, and an “Actionable Verdict” with recommendations.

Step 5: Delivery Back to Telegram

The final executive summary is formatted and sent back to you as a Telegram message, completing the loop. You receive a concise, insightful report without ever opening a browser.

Pro tip: You can modify the search query to target specific platforms like “site:reddit.com [Product] reviews” for deeper niche community insights.

Quick Setup Guide

Import the template JSON file into your n8n instance. Then follow these steps to activate the automation:

  1. In the Telegram node, add your Bot token and set the trigger to watch for text messages.
  2. In the Decodo node, input your API credentials and verify the search query is set to “[Input] reviews”.
  3. In the Google Gemini AI Agent nodes, insert your API key. Review the pre-configured system prompts—you can adjust them to match your preferred analytical tone.
  4. Test the workflow by sending a product name to your Telegram bot. The workflow should return a structured sentiment report within minutes.

Pro tip: For ongoing monitoring, set the workflow to run on a schedule (e.g., weekly) instead of a Telegram trigger, so you receive automatic reports without manual initiation.
