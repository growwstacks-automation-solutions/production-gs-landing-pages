What This Workflow Does

If you run events, webinars, or conferences, you know that collecting feedback is easy—but analyzing it is painful. Sorting through hundreds of responses, identifying patterns, and extracting actionable insights can take days of manual work. This workflow solves that problem completely.

When attendees submit feedback via Google Forms, this automation instantly processes each response with AI. It analyzes sentiment, extracts key likes and improvement areas, identifies quotable testimonials, and calculates overall satisfaction scores. Then it automatically posts summarized insights to your Slack channel and appends detailed reports to a Google Doc—giving your team real-time visibility without lifting a finger.

The result? You go from raw feedback to structured insights in minutes instead of days. Event planners can immediately address concerns, marketing teams get testimonials for promotion, and leadership sees clear metrics on event success.

How It Works

1. Trigger: New Google Forms Submission

The workflow activates whenever someone submits your event feedback form. It captures all response data including ratings, open-ended comments, and attendee information.

2. AI Analysis with GPT

Each comment is sent to OpenAI's GPT model for intelligent analysis. The AI determines sentiment (positive/neutral/negative), extracts specific themes, identifies improvement suggestions, and pulls out powerful quotes that can be used as testimonials.

3. Data Processing & Scoring

The workflow calculates overall sentiment scores, aggregates numerical ratings, and structures the insights into clear categories. It can handle everything from simple satisfaction scores to complex NPS calculations.

4. Slack Notification

A formatted message is sent to your designated Slack channel with the key findings: sentiment score, top positive comments, main improvement areas, and any urgent issues that need immediate attention.

5. Google Docs Reporting

Every response is logged in a master Google Doc that serves as a living record of all event feedback. This creates a searchable history that shows trends across multiple events over time.

What You'll Need

  1. Google Forms with your event feedback questions already set up
  2. OpenAI API key (GPT-4 or GPT-3.5-turbo works perfectly)
  3. Slack workspace with a dedicated channel for event feedback
  4. Google Docs document to serve as your feedback log
  5. n8n instance (cloud or self-hosted) to run the automation

Pro tip: Start with GPT-3.5-turbo for cost efficiency—it provides excellent sentiment analysis at about 1/10th the cost of GPT-4. Upgrade to GPT-4 only if you need extremely nuanced understanding of complex feedback.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance.
  2. Connect your Google Forms trigger by authenticating with Google and selecting your specific feedback form.
  3. Add your OpenAI API key in the AI node settings and configure the prompt to match your feedback questions.
  4. Set up Slack credentials and specify which channel should receive the notifications.
  5. Configure the Google Docs node with your document ID and preferred formatting.
  6. Test with a sample submission to ensure everything flows correctly from form to Slack to Docs.
  7. Activate the workflow and start collecting intelligent feedback automatically!
