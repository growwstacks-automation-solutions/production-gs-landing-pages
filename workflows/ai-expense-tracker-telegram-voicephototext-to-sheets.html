AI Expense Tracker: Telegram Voice/Photo/Text to Sheets

What This Workflow Does

Manual expense tracking is a tedious, time‑consuming chore. Most expense apps require you to unlock your phone, find the app, navigate menus, and manually input details—leading to forgotten entries and inconsistent data.

This n8n workflow solves that friction by letting you log expenses wherever you already communicate: Telegram. Simply send a text message (“Coffee $5”), a voice note while driving, or a photo of a receipt. The workflow uses AI (Google Gemini) to transcribe, extract details, categorize, and log the expense automatically into your Google Sheets.

You retain 100% ownership of your data—it lives in your own spreadsheet, not a third‑party app. The system intelligently splits multi‑item messages, detects currencies, assigns categories, and can even top‑up budgets via a simple command like /add budget 500. It’s a seamless, low‑cost alternative to subscription‑based expense apps.

How It Works

Step 1: Trigger via Telegram

The workflow listens for incoming messages in a Telegram chat (text, voice, or photo). When you send an expense note, it captures the content and prepares it for processing.

Step 2: AI Processing & Extraction

Voice notes are transcribed, photos are OCR‑read, and text is parsed. Google Gemini AI extracts the item name, amount, currency, and date. It also automatically assigns a category (Food, Transport, Bills, etc.) based on context.

Step 3: Data Structuring & Validation

The extracted data is cleaned, formatted, and validated. If you send “Lunch 20k and Grab 50k”, the AI splits it into two separate rows. Currency symbols are normalized, and a default currency can be applied if needed.

Step 4: Logging to Google Sheets

The structured expense entry is appended to your Google Sheet template. Each row includes item, amount, currency, category, date, and source (text/voice/photo). The sheet becomes your central, real‑time expense ledger.

Step 5: Quiet Reporting & Budget Updates

To avoid notification spam, the system waits for 30 minutes of inactivity before sending a daily summary report via Telegram. Budget top‑ups are handled via the /add budget command, updating your sheet instantly.

What You'll Need

  1. A running n8n instance (cloud or self‑hosted).
  2. A Telegram Bot token (create via BotFather).
  3. Google Sheets API credentials (service account or OAuth).
  4. Google Gemini API key (free tier available).
  5. A copy of the provided Google Sheet template.
  6. A Data Table named ReportTokens in n8n for reporting debounce.

Quick Setup Guide

Follow these steps to import and run the workflow:

  1. Download the template JSON file using the button above.
  2. Import it into your n8n workspace.
  3. Create a Telegram bot via BotFather and add its token to the Telegram trigger node.
  4. Set up Google Sheets and Gemini credentials in the respective nodes.
  5. Copy the Google Sheet template and update the spreadsheet ID in the Config node.
  6. Create the ReportTokens Data Table in n8n and link it.
  7. Activate the workflow and start sending expense messages to your bot.

Pro tip: Customize the AI prompt in the Gemini node to better match your expense categories and local currency formats. You can also add additional validation rules or connect the sheet to accounting software.
