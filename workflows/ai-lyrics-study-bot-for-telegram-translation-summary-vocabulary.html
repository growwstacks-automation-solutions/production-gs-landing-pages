What This Workflow Does

This automation transforms a standard Telegram bot into a powerful educational tool for language learners, music teachers, and content creators. Instead of manually searching for translations or analyzing lyrics, users simply send a song URL to the bot and receive instant, AI-generated analysis.

The workflow solves the time-consuming process of lyric analysis by automating everything from web scraping to intelligent interpretation. It extracts lyrics from any public URL, cleans the text, then uses OpenAI to generate multiple educational outputs including translations, vocabulary lists, summaries, and poetic device analysis—all delivered back to the user within seconds.

How It Works

1. Telegram Command Reception

The workflow starts when a user sends a command like /get_lyrics [URL] to your Telegram bot. The webhook trigger captures this message and extracts both the command type and the lyrics URL for processing.

2. URL Processing and Content Extraction

An HTTP Request node fetches the webpage containing the lyrics. A custom Code node then cleans the HTML, removing scripts, styles, and irrelevant content to isolate just the lyrical text in a standardized format.

3. Intelligent Routing and Analysis

Switch and If nodes determine which type of analysis the user requested. The workflow then sends the cleaned lyrics to OpenAI with specific prompts tailored for translation, vocabulary extraction, summary generation, or poetic analysis.

4. AI-Powered Content Generation

OpenAI processes the lyrics based on the selected command, generating educational content. For translations, it provides line-by-line conversion with cultural notes. For vocabulary, it identifies key words with definitions and usage examples.

5. Formatted Response Delivery

The Telegram Send Message node delivers the AI-generated content back to the user, automatically splitting long responses into manageable chunks and formatting with Markdown for readability.

Pro tip: Configure your OpenAI prompts to match your target audience's language level. For beginners, focus on basic vocabulary and simple translations. For advanced learners, include cultural context and complex grammatical analysis.

What You'll Need

  1. Telegram Bot Token: Created through @BotFather on Telegram
  2. OpenAI API Key: From platform.openai.com with GPT-4 or GPT-3.5 access
  3. n8n Instance: Self-hosted or cloud version with webhook capabilities
  4. Public Lyrics URLs: From sites like Genius, AZLyrics, or MetroLyrics
  5. Basic Server: To host your n8n instance with a public URL for webhooks

Quick Setup Guide

  1. Create your Telegram bot via @BotFather and save the API token
  2. Import the downloaded JSON template into your n8n instance
  3. Add your Telegram and OpenAI credentials to n8n's credential management
  4. Configure the webhook trigger with a unique path (like /lyrics-bot)
  5. Activate the workflow and set your Telegram webhook using the API endpoint
  6. Test with /start followed by /get_lyrics [song-url]
