AI Movie Recommender on WhatsApp

Automate personalized movie suggestions via WhatsApp messages using AI analysis

What This Workflow Does

This automation transforms WhatsApp into a smart movie concierge, analyzing user requests to deliver personalized recommendations. When customers message preferences like "scary movies" or "something like Inception," the system intelligently interprets requests, checks multiple databases, and replies with tailored suggestions.

The workflow solves three key problems: 24/7 instant response to movie queries, elimination of manual research by staff, and consistent quality of recommendations. Businesses save 15-20 hours weekly on customer service while increasing engagement through hyper-relevant suggestions.

How It Works

1. Message Reception

The WhatsApp Business API triggers the workflow when new messages arrive. The system captures the raw request and prepares it for AI analysis.

2. Intent Recognition

An AI model classifies the message type (genre request, specific movie query, or "where to watch" question) and extracts key parameters like movie titles or preferred genres.

3. Database Query

The system checks TMDb for movie details or genre matches, then cross-references JustWatch for streaming availability when users ask "where can I watch this?"
