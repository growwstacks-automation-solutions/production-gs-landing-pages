What This Workflow Does

This n8n workflow creates an intelligent AI assistant that can answer questions about documents stored in your Nextcloud instance. Instead of manually searching through folders and files, team members can simply ask questions in natural language and receive accurate answers extracted directly from your document content.

The system combines a public chat interface with a LangChain-powered AI agent that maintains conversation context. When a user asks about information in a specific Nextcloud folder, the workflow automatically retrieves all readable files (PDFs, DOCX, Markdown), extracts their text content, and feeds relevant information to OpenAI's model to generate precise, context-aware responses.

This transforms your Nextcloud from a passive file repository into an active knowledge base that team members can "converse" with, dramatically reducing time spent searching for information while ensuring answers are always based on your latest official documents.

How It Works

1. Chat Interface & Trigger

The workflow begins with a public chat trigger that greets users and captures their questions. Each incoming message starts the automation flow, creating a conversational experience similar to ChatGPT but specifically trained on your Nextcloud documents.

2. AI Agent with Memory

A LangChain agent ("AI Nextcloud") processes the conversation using OpenAI's model with short-term memory. This allows follow-up questions to be understood in context, creating a natural dialogue rather than isolated Q&A sessions.

3. Document Retrieval System

When the AI determines it needs document information to answer a question, it calls the integrated Nextcloud Tool sub-workflow. This component connects to your Nextcloud instance via API, lists files in the specified folder, and filters for readable formats.

4. Multi-Format Text Extraction

The system automatically handles different file types: PDFs are processed with optical character recognition, DOCX files are parsed for structured content, and Markdown files are read directly. All extracted text is aggregated into a single context payload.

5. Intelligent Response Generation

The AI analyzes the combined document content alongside the user's question to generate accurate, relevant answers. The response is then delivered back through the chat interface, completing the automated document Q&A cycle.

What You'll Need

  1. n8n instance (self-hosted or cloud) with workflow permissions enabled
  2. Nextcloud instance with API access credentials (username/password or app token)
  3. OpenAI API key with GPT-3.5 or GPT-4 access
  4. Documents stored in Nextcloud in supported formats: PDF, DOCX, or Markdown
  5. Basic understanding of folder paths in your Nextcloud structure

Pro tip: Start with a dedicated "Knowledge Base" folder in Nextcloud containing your most frequently referenced documents. This keeps the AI focused and improves answer accuracy.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure Nextcloud credentials in the "Nextcloud Tool" sub-workflow with your server URL and authentication
  3. Add your OpenAI API key to the AI agent node with appropriate model settings
  4. Test the chat trigger by sending a question mentioning a Nextcloud folder path
  5. Deploy the workflow and share the webhook URL with your team or integrate with your preferred chat platform
