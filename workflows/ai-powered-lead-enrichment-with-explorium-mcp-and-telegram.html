What This Workflow Does

Manual lead research is a massive time sink for sales teams. Finding a company's size, tech stack, funding status, and key pain points can take 15-30 minutes per lead—time better spent on actual selling conversations.

This automation solves that by turning Telegram into your instant research assistant. Send a company name or domain to your bot, and within seconds, you receive a structured JSON profile containing enriched data pulled from Explorium MCP and web sources, analyzed by AI to highlight relevant insights.

The workflow transforms vague leads into qualified prospects ready for personalized outreach, ensuring your sales team starts conversations with context and relevance rather than generic cold emails.

How It Works

1. Trigger via Telegram

A sales rep sends a company name or website URL to a dedicated Telegram bot. This acts as the workflow trigger, kicking off the automated research process instantly from anywhere.

2. Multi-Source Data Collection

The workflow queries Explorium MCP for structured company data (industry, employee count, location) while simultaneously using Tavily Web Search to gather recent news, blog posts, and public announcements for contextual understanding.

3. AI Analysis & Synthesis

An OpenAI GPT model processes the collected data, identifying key pain points, technology usage, growth signals, and potential needs. It structures this into a clean, actionable summary.

4. Structured Output Delivery

The enriched profile is formatted into standardized JSON and sent back via Telegram for immediate review. Optionally, it can be logged to a PostgreSQL database for CRM integration or team sharing.

What You'll Need

  1. OpenAI API Key: For GPT-powered analysis and synthesis of collected data.
  2. Explorium MCP API Access: Provides reliable, structured company data feeds.
  3. Tavily Web Search API Key: For gathering recent, contextual information from the web.
  4. Telegram Bot Token: Create a bot via BotFather to serve as the input interface.
  5. PostgreSQL Database (Optional): For storing enriched profiles and creating a searchable history.
  6. n8n Instance: Self-hosted or cloud version to run the automation.

Quick Setup Guide

Get this automation running in under 15 minutes:

  1. Import the Template: Download the JSON file above and import it into your n8n instance.
  2. Configure API Credentials: Add your OpenAI, Explorium MCP, and Tavily API keys to the respective nodes in the workflow.
  3. Set Up Telegram Webhook: Connect your Telegram bot token and configure the webhook trigger node.
  4. Test with a Sample Company: Send a company name like "Stripe" or "Shopify" to your bot and verify the output format.
  5. Customize AI Prompts: Adjust the prompt in the OpenAI node to focus on specific industries, data points, or output structures relevant to your business.
  6. Add CRM Integration (Optional): Connect the output to HubSpot, Salesforce, or your CRM of choice using n8n's built-in nodes.
