What This Workflow Does

This automation solves the time-consuming problem of manually searching for relevant local events across multiple platforms. It combines AI-powered search tools to continuously monitor for events matching your specific criteria - including event type, location, date range, and interests.

The system intelligently processes search results from Brave Web Search, Brave Local Search, and Google Gemini, then uses Jina AI to extract key details from event pages. It outputs formatted event information ready for your calendar or CRM, saving hours of manual research each week.

How It Works

1. Criteria Input

The workflow accepts your search parameters including location radius, date ranges, and interest keywords. These can be provided manually or pulled from your CRM/calendar.

2. Multi-Tool Search

Simultaneously queries Brave Search (general web), Brave Local (geo-specific), and Google Gemini (semantic understanding) to cast the widest net.

3. AI Processing

The AI agent evaluates results for relevance, removes duplicates, and prioritizes events based on your historical preferences.

4. Data Extraction

Jina AI scrapes event pages to pull structured data like dates, locations, pricing, and descriptions.
