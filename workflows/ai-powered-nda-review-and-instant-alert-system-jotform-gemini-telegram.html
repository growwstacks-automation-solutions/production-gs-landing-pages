What This Workflow Does

Manual contract review is slow, expensive, and prone to human error. Legal teams waste hours scanning NDAs for standard clauses while business deals stall waiting for approval. This automation solves that by instantly analyzing contracts submitted through web forms.

When someone submits an NDA via your Jotform, this workflow automatically extracts the text, uses Google's Gemini AI to identify potential red flags and unfair clauses, and sends a structured risk assessment directly to your Telegram channel. It transforms days of legal review into minutes of automated screening.

The system acts as your first line of defense, filtering out problematic contracts before they reach your legal team. This lets lawyers focus on high-value negotiation rather than initial screening, dramatically accelerating deal cycles while maintaining compliance standards.

How It Works

This intelligent pipeline connects four key services to create a seamless contract review experience.

1. Form Submission Capture

The workflow triggers when someone submits your Jotform with an attached PDF contract. It captures the submission ID, user email, and file upload details, then prepares the document for analysis.

2. Document Processing

The system fetches the complete submission details via Jotform's API to obtain the direct download URL, then retrieves the actual PDF file. An extraction node converts the PDF into plain text, removing formatting while preserving all contractual language for AI analysis.

3. AI-Powered Analysis

Google Gemini AI receives the contract text with specific instructions to act as an expert contract analyst. It scans for concerning clauses like unlimited liability, unusual termination terms, excessive confidentiality obligations, and other red flags based on your risk parameters.

4. Instant Alert Delivery

The AI generates a concise risk assessment report formatted for Telegram's MarkdownV2. This formatted alert is sent directly to your specified Telegram chat or channel, providing immediate visibility into contract issues that need attention.

Pro tip: Customize the AI prompt to match your specific risk tolerance. You can instruct it to flag particular clause types, assign risk scores, or even suggest alternative language for problematic sections.

What You'll Need

  1. JotForm account with a form containing email and file upload fields
  2. JotForm API key from your account settings
  3. Google Gemini API key from Google AI Studio
  4. Telegram Bot Token created via @BotFather
  5. Telegram Chat ID for your personal or team channel
  6. n8n instance (cloud or self-hosted) to run the workflow

Quick Setup Guide

Get this automation running in under 15 minutes with these straightforward steps.

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure JotForm trigger by connecting your API key and selecting your form
  3. Set up Google Gemini with your API key in the AI Agent node
  4. Connect Telegram using your Bot Token and target Chat ID
  5. Test the workflow by submitting a sample NDA through your form
  6. Activate the workflow and monitor initial alerts for accuracy
  7. Refine the AI prompt based on your specific contract review criteria

Implementation note: Start with non-critical contracts to validate the AI's accuracy before processing sensitive agreements. The system improves as you refine the prompt based on its performance.
