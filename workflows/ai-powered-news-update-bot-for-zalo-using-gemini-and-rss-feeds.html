This automation solves the challenge of keeping Vietnamese audiences informed with timely, relevant news updates. Instead of manually monitoring news sources and sending updates, this workflow creates an intelligent Zalo chatbot that automatically fetches the latest articles from RSS feeds, uses Google Gemini AI to generate concise summaries, and delivers them directly to users through Vietnam's most popular messaging platform.

How It Works

1. User Request via Zalo Webhook

When a user sends a message to your Zalo bot (like "What's new today?"), Zalo sends this request to your n8n webhook endpoint. The workflow captures the user's message and prepares to fetch relevant content.

2. RSS Feed Monitoring

The workflow checks configured RSS feeds for the latest articles. You can monitor multiple news sources simultaneously—tech blogs, financial news, industry publications, or any content source that provides RSS feeds.

3. AI-Powered Summarization

Each article is sent to Google Gemini AI, which generates a concise, readable summary in Vietnamese or English. The AI extracts key points, maintains context, and creates engaging content suitable for messaging platforms.

4. Response Formatting & Delivery

The summarized content is formatted with emojis, clear headings, and a friendly tone, then sent back to the user through Zalo's messaging API. The entire process happens in seconds, providing instant value to users.

What You'll Need

  1. Zalo Bot Account: A Zalo account with bot creation privileges (bot names must start with "Bot")
  2. Google Gemini API Key: Access to Google's Gemini AI models for content summarization
  3. RSS Feed URLs: News sources or content feeds you want to monitor
  4. n8n Instance: Self-hosted n8n or n8n.cloud account with webhook capabilities
  5. Publicly Accessible URL: For Zalo to send webhook requests to your n8n workflow

Quick Setup Guide

1. Create Your Zalo Bot: Open Zalo, search for "Zalo Bot Creator," tap "Create Zalo Bot Account," and note your bot token.

2. Configure Webhook: Replace the placeholder webhook URL in Zalo Bot Creator with your n8n webhook URL from the downloaded template.

3. Add API Credentials: Enter your Google Gemini API key in the HTTP Request node labeled "Summarize AI News."

4. Customize RSS Feeds: Replace the default RSS URLs with your preferred news sources in the RSS Feed Read node.

5. Test & Deploy: Activate your workflow, send a test message to your Zalo bot, and verify you receive summarized news updates.

Pro tip: Start with 2-3 high-quality RSS feeds rather than many low-quality sources. Better content leads to better AI summaries and happier users.
