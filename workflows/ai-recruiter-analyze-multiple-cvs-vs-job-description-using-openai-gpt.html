AI Recruiter: Analyze CVs vs Job Description with OpenAI

Automate candidate screening with AI. Compare multiple CVs to a job description, score fit, and rank applicants in minutes.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Manual CV screening is a time-consuming, repetitive, and often biased process. Recruiters spend hours sifting through hundreds of applications, trying to match keywords and experience to a job description. This free n8n automation template transforms that manual grind into an intelligent, automated system.

This AI Recruiter workflow acts as your first-round screening assistant. You provide one Job Description (JD) and multiple candidate CVs (PDF or text). The system automatically extracts the text, uses an AI agent (like OpenAI's GPT) to compare each CV against the JD, and generates a structured report. It outputs a fit score (e.g., 85/100), highlights key strengths and weaknesses for each candidate, and provides an overall ranking.

The result? You move from a pile of resumes to a shortlist of ranked, analyzed candidates in minutes—not days. It standardizes your screening process, reduces unconscious bias, and lets your human recruiters focus on engaging with the best-fit applicants.

How It Works

The workflow is built with a logical sequence of nodes in n8n, creating a seamless pipeline from raw documents to actionable insights.

1. Trigger & File Input

The process starts with a Webhook node, allowing you to trigger the workflow manually or via an API call. You upload your Job Description and the batch of candidate CVs. A File Detector node automatically identifies which document is the JD and which are CVs.

2. Text Extraction & Data Preparation

An "Extract from File" node reads the text content from the uploaded PDFs or documents. The workflow then merges this extracted text into a clean, structured dataset, preparing each candidate's information for AI analysis.

3. AI-Powered Analysis & Scoring

This is the core of the automation. An AI Agent node (configured for OpenAI or a compatible LLM) receives the job description and each candidate's CV text. It performs a comparative analysis based on instructions you set—evaluating skills, experience, qualifications, and cultural fit. It returns a consistent score, a bullet-point list of strengths, a list of potential gaps, and a hiring recommendation (e.g., "Strong Match," "Consider," "Not a Fit").

4. Output & Integration

The final node formats all the analyzed data into a clear, structured JSON output. This can be sent to a dashboard, saved to a Google Sheet, posted to a Slack channel for the hiring team, or fed directly into your Applicant Tracking System (ATS). The workflow loops through all CVs provided, delivering a complete, ranked candidate report.

What You'll Need

  1. An n8n instance: Either n8n Cloud or a self-hosted version (v1.50+ recommended).
  2. OpenAI API Key (or compatible LLM): Credentials for an AI model service like OpenAI, Anthropic Claude, or Google Gemini. The workflow uses the AI Agent node.
  3. Input Documents: One Job Description (text or PDF) and multiple Candidate CVs (PDF or text format). English is ideal, but multilingual models can be configured.
  4. Basic n8n Knowledge: Familiarity with importing workflows and configuring node credentials is helpful.

Pro tip: For the most accurate results, ensure your job description is detailed and includes specific required skills, years of experience, and qualifications. The AI uses this as its benchmark.

Quick Setup Guide

Get your AI recruiter up and running in under 10 minutes:

  1. Download & Import: Click the "Download Template" button above to get the JSON file. In your n8n instance, go to "Workflows" > "Import from File" and select the downloaded JSON.
  2. Configure AI Credentials: Click on the "AI Agent" node. In the credentials section, add your OpenAI API key (or configure it for your chosen LLM provider).
  3. Test the Trigger: Execute the Webhook node manually. Upload a sample JD and 2-3 CVs (PDF or text) when prompted.
  4. Review & Customize Output: Check the output of the final node. You can modify the AI's prompt in the Agent node to emphasize certain criteria (e.g., "weight technical skills at 60%").
  5. Connect to Your Tools (Optional): Add a node after the final output to send results to Google Sheets, your ATS, or a notification channel like Slack or email.
