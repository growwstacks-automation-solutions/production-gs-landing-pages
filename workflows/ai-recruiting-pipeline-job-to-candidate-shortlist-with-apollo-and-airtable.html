AI Recruiting Pipeline: Job to Candidate Shortlist

Automate candidate sourcing and prequalification for recruiting teams

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation transforms recruiting workflows by turning job descriptions into ready-to-interview candidate shortlists. Designed for agencies and talent teams, it eliminates hours of manual sourcing while improving candidate quality through AI scoring.

The pipeline handles everything from title normalization and candidate searching to AI scoring, LinkedIn enrichment, and personalized outreach generation. It outputs structured candidate assessments directly to Airtable for easy tracking and collaboration.

How It Works

1. Job Title Processing

AI generates 5-7 normalized title variations from the input job description to broaden candidate search results while maintaining relevance.

2. Candidate Search

Queries Apollo (or your CRM) using the generated titles and location filters. Built-in deduplication ensures clean candidate lists.

3. AI Scoring

Scores each candidate 0-5 across experience, industry fit, seniority, and skills with concise reasoning for each assessment.

4. Profile Enrichment

Pulls missing data from LinkedIn including current title, location, experience, and profile images.

5. Structured Assessment

Generates recruiter-ready evaluations covering alignment, red flags, positive signals, and recommended next steps.

6. Outreach Generation

Creates personalized cold emails (<75 words) and LinkedIn DMs (<60 words) tailored to each candidate's background.

7. Airtable Integration

Logs all jobs and candidates with assessments and outreach messages for tracking and downstream automation.

What You'll Need

  1. n8n instance (cloud or self-hosted)
  2. Airtable account with API access
  3. Apollo API or CRM integration
  4. LLM provider (OpenAI/Anthropic)
  5. LinkedIn enrichment API (RapidAPI, Apify, etc.)

Pro tip: Store all API keys in n8n Credentials rather than hardcoding them in nodes for better security.

Quick Setup Guide

  1. Import the JSON template into your n8n instance
  2. Configure your Airtable base with Jobs and Candidates tables
  3. Connect your Apollo/CRM and LLM credentials
  4. Test with a sample job description
  5. Schedule regular runs or trigger via webhook
