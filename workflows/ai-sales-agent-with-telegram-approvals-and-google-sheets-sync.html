What This Workflow Does

This n8n workflow template creates a fully automated AI sales agent for food delivery businesses operating through Telegram. It handles the complete customer journey from initial order placement to payment processing and order fulfillment, while automatically syncing all transaction data to Google Sheets for record-keeping and analysis.

The system intelligently processes orders, sends approval requests to staff via Telegram, handles common customer service inquiries, and maintains real-time inventory tracking. By automating these repetitive tasks, food businesses can reduce operational overhead by 30-50% while improving order accuracy and customer satisfaction.

How It Works

1. Customer initiates order via Telegram

The AI chatbot greets customers and guides them through the ordering process with interactive menus. It can answer FAQs about ingredients, delivery times, and special offers.

2. Order validation & inventory check

The system checks Google Sheets for real-time inventory availability and validates order details before proceeding. If items are unavailable, it suggests alternatives.

3. Staff approval workflow

For large orders or special requests, the system sends approval requests to designated staff members via Telegram. They can approve/reject with a single tap.

4. Payment processing

Integrated payment gateways process transactions, with automatic receipts sent to customers. Failed payments trigger follow-up messages.

5. Order fulfillment tracking

The system updates order status throughout the fulfillment process and notifies customers at each stage (preparation, dispatch, delivery).

6. Data sync to Google Sheets

All order details, customer information, and transaction records are automatically logged in designated Google Sheets for reporting and analytics.
