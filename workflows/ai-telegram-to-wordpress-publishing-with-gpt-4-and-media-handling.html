AI Telegram-to-WordPress Publishing with GPT-4 & Media Handling

Automatically rewrite, enhance, and publish Telegram content as polished WordPress articles—including image/video upload and category mapping.

What This Workflow Does

Content creators and news publishers often share updates on Telegram but struggle to republish that content as formatted articles on their WordPress sites. Manually copying posts, rewriting them, uploading media, and assigning categories consumes hours each week.

This AI-powered automation solves that by monitoring Telegram RSS feeds, using GPT-4 to rewrite and enhance content into polished news-style articles, detecting and downloading images/videos, uploading them to WordPress, and publishing complete posts with proper categories automatically. It turns brief Telegram updates into full website articles without manual intervention.

How It Works

Step 1: RSS Feed Monitoring

The workflow regularly checks your Telegram channel RSS feed for new posts. It extracts the content, metadata, and any media links while implementing duplicate prevention to avoid republishing the same content.

Step 2: AI Content Enhancement

Each Telegram post passes through two-stage GPT-4 processing. First, a full rewrite adds structure and improves readability. Second, context enhancement expands on key points and ensures the article matches your brand's tone and style.

Step 3: Media Detection & Handling

The system analyzes content for image or video attachments, downloads them locally, detects file types, and prepares them for WordPress upload. It handles different media formats and ensures proper linking to the final article.

Step 4: WordPress Publishing

The enhanced article text, along with uploaded media, gets published to your WordPress site via its REST API. The workflow automatically maps content topics to appropriate WordPress categories and sets excerpts for SEO optimization.

What You'll Need

  1. A WordPress site with REST API access enabled
  2. WordPress Application Password for API authentication
  3. OpenAI API key (GPT-4 access)
  4. Telegram channel RSS feed URL
  5. n8n instance (cloud or self-hosted)

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n workspace.
  2. Configure the RSS Feed Trigger node with your Telegram channel RSS URL.
  3. Add your OpenAI API key in the GPT-4 node settings.
  4. Set up WordPress credentials in the HTTP Request nodes targeting your site's REST API.
  5. Adjust category mapping in the Switch node if needed.
  6. Activate the workflow and test with a recent Telegram post.

Pro tip: Schedule the workflow to run hourly for real-time publishing, or daily for batch processing. Adjust the GPT-4 prompt to match your specific brand voice—the template includes editable instructions.
