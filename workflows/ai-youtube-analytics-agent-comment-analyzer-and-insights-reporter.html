What This Workflow Does

This intelligent automation solves a common problem for content creators, marketers, and businesses: manually reading hundreds of YouTube comments to understand audience sentiment. Instead of spending hours scrolling through comments, this workflow automatically fetches comments, analyzes them with AI, and delivers professional email reports with actionable insights.

The system monitors a Google Sheets spreadsheet for new YouTube video IDs marked as "Pending," triggers automated comment collection via YouTube API, performs comprehensive sentiment and theme analysis using GPT-4, and generates formatted HTML reports emailed directly to you. It also updates spreadsheet status to prevent duplicate processing, creating a fully organized analytics pipeline.

By automating this process, you gain deep audience understanding without the time drain, enabling data-driven content improvements, better engagement strategies, and competitive intelligence.

How It Works

Step 1: Video Monitoring & Trigger

The workflow watches your Google Sheets for new YouTube video IDs. When a video is marked "Pending" in the status column, the automation triggers immediately, ensuring fresh comments are analyzed as soon as you request it.

Step 2: Comment Collection

Using YouTube API credentials, the system fetches up to 100 top comments per video, ordered by relevance. It captures comment text, likes, replies, and timestamps, creating a rich dataset for analysis.

Step 3: AI-Powered Analysis

GPT-4 processes the comments, detecting sentiment (positive/negative/neutral), identifying common questions, extracting feedback themes, and generating actionable insights. The AI looks beyond surface text to understand underlying audience needs and preferences.

Step 4: Report Generation & Delivery

A professional HTML email report is constructed with statistics, sentiment breakdown, key feedback points, and improvement recommendations. The report is formatted for easy reading and decision-making.

Step 5: Status Update & Completion

The spreadsheet status is updated to "Mail Sent," preventing duplicate analysis and maintaining a clean workflow history. The entire process runs unattended, from trigger to delivery.
