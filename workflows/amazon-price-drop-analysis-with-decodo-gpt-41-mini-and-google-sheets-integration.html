What This Workflow Does

This automation solves a critical problem for e-commerce businesses, affiliate marketers, and product researchers: manually tracking Amazon price drops is slow, inconsistent, and misses valuable insights. This workflow creates a complete price intelligence pipeline that runs automatically.

It scrapes daily deal pages (like CamelCamelCamel), extracts product details using Decodo's web scraping, enriches each item with AI-generated summaries and sentiment scores via OpenAI GPT-4.1-mini, and stores the structured analysis in Google Sheets. The result is a real-time, searchable database of discounted products with contextual insights—not just raw numbers.

You get automated market monitoring that identifies not just when prices drop, but why a deal might be attractive based on product descriptions and reviews. This enables smarter purchasing decisions, competitive analysis, and trend spotting without manual effort.

How It Works

The automation follows a logical sequence from data collection to actionable insights.

1. Trigger & Source Configuration

The workflow starts manually or on a schedule. You configure the target URL (default is CamelCamelCamel's daily drops page). This flexibility lets you monitor specific product categories, competitor listings, or deal aggregators.

2. Web Scraping with Decodo

The Decodo node visits the configured page and extracts raw product listings—including titles, prices, savings percentages, and direct Amazon links. This replaces hours of manual copying and pasting with seconds of automated data collection.

Installing the Decodo community node in n8n to enable web scraping capabilities.

3. AI-Powered Structuring & Enrichment

Each product URL is revisited by Decodo for detailed page content. OpenAI's GPT-4.1-mini then performs two key analyses: generating a concise summary of the product and its deal context, and calculating a sentiment score (positive/neutral/negative) based on description language.

4. Data Aggregation & Storage

All enriched data—product name, current price, savings, link, AI summary, and sentiment—is merged into a clean dataset. The Google Sheets node appends this structured information to a designated spreadsheet, creating a growing log of analyzed deals.

Pro tip: Schedule this workflow to run daily at 9 AM to get fresh deal insights before your team starts work. You'll have a ready-made report waiting in Sheets.

Quick Setup Guide

Get this price intelligence system running in under 30 minutes:

  1. Import the template into your n8n workspace using the downloaded JSON file.
  2. Install the Decodo community node via n8n's "Community Nodes" settings.
  3. Configure credentials for Decodo, OpenAI, and Google Sheets in their respective nodes.
  4. Update the price_drop_url in the "Set" node to your target deal page.
  5. Test the workflow with the "Execute Workflow" button to verify data flows to Sheets.
  6. Add scheduling (Cron node) for daily automated runs.
Configuring the Decodo node with API credentials to enable web scraping functionality.
