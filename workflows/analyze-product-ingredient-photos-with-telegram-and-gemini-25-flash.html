What This Workflow Does

This automation solves a critical problem for health-conscious consumers: understanding complex product ingredients instantly. When shopping for skincare, food, or supplements, people face walls of chemical names they can't decipher. Manual research takes time they don't have, and potentially harmful ingredients get overlooked.

The workflow creates a conversational AI assistant accessible through Telegram. Users simply send a photo of any product label, and within seconds receive a structured analysis highlighting safety concerns, benefits, and recommendations. For businesses, this transforms customer education from a burden into an automated service that builds trust and reduces returns.

By connecting Telegram's massive user base with Google Gemini's advanced vision capabilities, this solution makes expert-level ingredient analysis available to anyone with a smartphone. It's particularly valuable for parents checking children's products, individuals with allergies, and anyone pursuing cleaner living without becoming chemistry experts.

How It Works

1. Telegram Photo Reception

The workflow monitors a Telegram bot for incoming messages. When a user sends a photo (with or without a caption), the bot immediately captures the image file and user context. The system first checks if the message contains a photo—if not, it handles text queries or provides guidance.

2. AI-Powered Ingredient Extraction

The captured image is sent to Google Gemini 2.5 Flash, which performs optical character recognition to read all text from the product label. The AI then identifies and extracts the ingredient list specifically, filtering out marketing text, brand names, and other irrelevant information.

3. Safety Analysis & Categorization

Based on the user's query context (caption presence), the system branches. For photos with captions asking specific questions, it provides direct "Use/Do Not Use" recommendations with reasons. For photos without context, it generates comprehensive analysis covering advantages, disadvantages, recommended user groups, and warnings.

4. Structured Response Delivery

The analyzed information is formatted into clear, conversational responses and sent back through Telegram. The response includes emojis for quick scanning, bullet points for readability, and actionable advice that helps users make immediate purchasing or usage decisions.

What You'll Need

  1. n8n Instance: Either self-hosted n8n or n8n.cloud account with workflow execution capabilities.
  2. Telegram Bot Token: Created via BotFather on Telegram with message receiving permissions enabled.
  3. Google Gemini API Key: From Google AI Studio with vision capabilities enabled for image analysis.
  4. Basic Server Resources: For image processing and API calls, especially if expecting high volume.
  5. Optional Database: PostgreSQL or MySQL if you want to log queries and build an ingredient knowledge base over time.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n instance through the workflow import interface.
  2. Configure Telegram Credentials: Create credentials in n8n with your bot token from BotFather, then connect the Telegram trigger node.
  3. Set Up Gemini AI: Add your Google Gemini API key in n8n credentials and connect it to the AI model nodes within the workflow.
  4. Test with Sample Images: Activate the workflow and send test product photos through your Telegram bot to verify image capture and analysis.
  5. Customize Response Templates: Adjust the Telegram response nodes to match your brand voice and add any specific disclaimers required.
  6. Deploy & Monitor: Go live and monitor initial interactions to fine-tune AI prompts based on real user queries and photo quality.

Pro tip: Start with common products in your niche to build a reference library. The AI improves with context—providing examples of "good" and "concerning" ingredients specific to your industry yields more accurate analyses.
