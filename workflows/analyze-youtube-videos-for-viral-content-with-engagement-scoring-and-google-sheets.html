What This Workflow Does

Manual YouTube research is time-consuming and inconsistent. This n8n automation transforms hours of manual analysis into a fully automated process that systematically discovers, evaluates, and documents viral video opportunities.

The workflow performs bulk research on YouTube videos based on your search terms, analyzes engagement data (views, likes, comments, channel statistics), calculates performance scores, and exports everything to Google Sheets with visual indicators. It helps content creators, marketers, and agencies identify what's working in their niche, benchmark against competitors, and make data-driven content decisions.

Instead of guessing what might go viral, you get actionable intelligence showing exactly which topics, formats, and channels are gaining traction—with all data organized and ready for strategic planning.

How It Works

1. Input & Search Configuration

The workflow starts with a simple form or manual trigger where you specify your search term, desired video format (short, medium, or long), and the number of videos to analyze. This configuration determines the scope and focus of your research.

2. YouTube Data Collection

Using the YouTube Data API, the automation fetches video IDs based on your search criteria, then retrieves detailed metadata for each video including view counts, likes, comments, publication dates, and channel statistics.

3. Engagement Scoring & Analysis

Each video is automatically scored based on engagement rates and performance metrics. The system applies intuitive labels like 🚀 HOLY HELL (high viral potential), 📈 Growing, or 💀 Dead to help you quickly identify opportunities.

4. Google Sheets Export

All analyzed data—including video titles, URLs, thumbnails, metrics, and scores—is appended to your designated Google Sheet. This creates a searchable, sortable database of video intelligence that updates automatically with each run.

What You'll Need

  1. n8n instance (cloud or self-hosted)
  2. YouTube Data API v3 key from Google Cloud Console
  3. Google Sheets with edit permissions for service account
  4. Google Service Account credentials for Sheets API access
  5. Basic understanding of n8n workflow import/export

Pro tip: When setting up your YouTube API key, enable the "YouTube Data API v3" specifically. The free tier provides sufficient quota for most research needs unless analyzing thousands of videos daily.

Quick Setup Guide

Follow these steps to implement this YouTube research automation:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure YouTube API credentials in n8n's credential manager
  3. Set up Google Sheets connection with your service account credentials
  4. Update the Google Sheet ID in the workflow to point to your destination spreadsheet
  5. Test with a simple search term like "AI tutorials" to verify data flows correctly
  6. Schedule the workflow to run weekly or monthly for ongoing research

The workflow includes clear comments at each configuration point. Most users can have it running in under 15 minutes with basic API setup experience.
