Automatically Archive Your Spotify Discover Weekly Playlist

Never lose a great discovery again. This free n8n workflow automatically saves your weekly Spotify recommendations to a permanent archive playlist every Monday.

What This Workflow Does

Spotify's Discover Weekly playlist is a fantastic feature that delivers 30 personalized song recommendations every Monday. However, these recommendations disappear when the playlist refreshes the following week. If you don't manually save songs you like, they're lost forever.

This automation solves that problem by automatically archiving your entire Discover Weekly playlist to a permanent "Discover Weekly Archive" playlist. It runs every Monday morning, ensuring you never lose a potentially favorite track. The workflow handles authentication with Spotify's API, finds your current Discover Weekly playlist, locates or creates your archive playlist, and transfers all tracks seamlessly.

Beyond simple archiving, this template demonstrates how to work with Spotify's API for more complex music management automations. It's perfect for music enthusiasts, playlist curators, or anyone who wants to build a personal music discovery library over time.

How It Works

1. Scheduled Trigger

The workflow is configured to run at 8 AM every Monday using n8n's schedule trigger. This timing ensures it captures your fresh Discover Weekly playlist after Spotify updates it but gives you the weekend to listen to the previous week's recommendations.

2. Spotify Authentication & Playlist Discovery

The workflow authenticates with Spotify's API using OAuth credentials you set up once. It then retrieves your current Discover Weekly playlist by searching for the standardized playlist name and your user ID.

3. Archive Playlist Management

The system searches for your existing "Discover Weekly Archive" playlist. If found, it uses that playlist; if not, it can be configured to create one automatically (though the base template assumes you create it manually first for clarity).

4. Track Transfer & Error Handling

All tracks from your current Discover Weekly playlist are added to the archive playlist. The workflow includes basic error handling to manage API rate limits and connection issues, with options to extend it with notification systems if anything fails.

What You'll Need

  1. A Spotify Premium account (required for API access to modify playlists)
  2. n8n instance (cloud or self-hosted) with internet access
  3. Spotify Developer credentials (free registration at developer.spotify.com)
  4. Basic understanding of API authentication to set up OAuth in n8n
  5. The "Discover Weekly Archive" playlist created manually in your Spotify account (or modify the workflow to create it automatically)

Pro tip: Before running this automation in production, test it with a dummy playlist to ensure it works correctly with your Spotify account setup. Also consider setting up an error workflow that notifies you (via email, Slack, etc.) if the archiving fails for any reason.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance.
  2. Create a Spotify Developer application at developer.spotify.com and note your Client ID and Client Secret.
  3. Set up Spotify OAuth credentials in n8n for each Spotify node in the workflow.
  4. Create your "Discover Weekly Archive" playlist in Spotify (or modify the workflow to create it).
  5. Test the workflow manually first by executing it once to ensure it can find and archive your current Discover Weekly.
  6. Activate the schedule to run automatically every Monday at your preferred time.
