Archive Trending TikTok Hashtags to Airtable

Automate TikTok trend discovery. Scrape trending hashtags with Apify and save them to Airtable for content planning and market analysis.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Manual TikTok trend research is time-consuming and inconsistent. Social media managers, content creators, and marketers spend hours each week scrolling through TikTok to identify trending hashtags, often missing important data points or regional variations. This creates content calendars based on guesswork rather than data-driven insights.

This automation solves that problem by systematically collecting TikTok's top 100 trending hashtags for any country, complete with metrics like video views, publish count, and ranking. It transforms scattered social media observations into structured, analyzable data in Airtable, creating a searchable database of trends that informs content strategy, market research, and competitive analysis.

How It Works

1. Scheduled Trigger

The workflow activates automatically on a monthly schedule (configurable to weekly or daily). This ensures your trend database stays current without manual intervention.

2. TikTok Hashtag Scraping

Using the Apify community node, the workflow runs a specialized TikTok scraper that collects the top 100 trending hashtags for your specified country. You can customize parameters like country code and time period.

3. Data Retrieval & Processing

Once scraping completes, the workflow retrieves the dataset containing hashtag names, video view counts, publish frequencies, rankings, and other metrics. It processes this data into individual items for database insertion.

4. Airtable Database Population

Each hashtag becomes a new record in your Airtable base, organized with fields for name, country, industry, date added, metrics, and status. This creates a historical archive of trends for analysis.

  2. Apify account and API credentials for TikTok scraping
  3. Airtable account with a base containing a "Trending Hashtags" table
  4. Community node: @apify/n8n-nodes-apify installed in your n8n settings
  5. Basic understanding of how to import and configure n8n workflows

Quick Setup Guide

  1. Install the community node: In your n8n settings under "Community Nodes," add @apify/n8n-nodes-apify and restart your instance.
  2. Prepare your Airtable base: Create a table with fields for ID, Name, Country, Industry, Date Added, Publish Count, Video Views, Rank, and Status.
  3. Add credentials: In n8n, create credentials for both Apify and Airtable with appropriate access permissions.
  4. Configure the scraper: Open the "Run TikTok Hashtag Scraper" node and adjust the custom body parameters if needed (country_code, top100_period).
  5. Map Airtable fields: In the "Save Hashtag to Airtable" node, select your base and table, then ensure field mappings align with your Airtable structure.
  6. Activate and test: Save the workflow, activate it, and run a test to verify data flows correctly from TikTok to Airtable.
