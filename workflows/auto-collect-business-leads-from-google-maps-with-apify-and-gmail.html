What This Workflow Does

Manual lead generation from Google Maps is a tedious, time-consuming process that involves searching, clicking, copying, and pasting—often for hours with inconsistent results. This automation solves that problem by turning a manual research task into a hands-off, reliable system.

The workflow connects APIFY's powerful Google Maps scraping actor with n8n's automation logic. You define what you're looking for (e.g., "plumbers in Austin" or "marketing agencies in London"). The system then scrapes the public business listings, extracts key details like business name, phone, website, and email, filters out incomplete entries, and emails you a clean CSV file of ready-to-contact leads. It effectively acts as a virtual research assistant that works 24/7.

How It Works

The process is broken down into logical, automated steps that ensure data quality and delivery.

1. Input Collection

A simple form or manual trigger captures your search parameters: business type, city/country, and the recipient email address for the results.

2. APIFY Scraper Activation

n8n sends an HTTP request to APIFY's API, triggering their Google Maps Scraper "actor" with your search criteria. APIFY handles the complex task of interacting with Google Maps and extracting structured data.

3. Data Filtering & Cleansing

The raw results come back. The workflow uses Filter and IF nodes to remove listings without email addresses or other essential contact information, ensuring you only get actionable leads.

4. CSV File Generation

The filtered, clean data is converted into a standardized CSV file using n8n's "Spreadsheet File" node, making it easy to import into any spreadsheet or CRM.

5. Automated Email Delivery

Finally, the Gmail node (or your configured email service) sends an email to your specified address with the CSV file attached, delivering a batch of qualified leads directly to your inbox.

Pro tip: Start with APIFY's free $5 credit (approx. 1,000 leads) to test the system. Use very specific search terms and locations to improve lead relevance and reduce scraping costs.

Quick Setup Guide

You can have this automation running in under 30 minutes.

  1. Import the Template: Download the JSON file above and import it into your n8n instance.
  2. Configure APIFY Credentials: In the HTTP Request node, add your APIFY API token in the authentication settings.
  3. Set Up Gmail: Connect the Gmail node using OAuth2 or your SMTP credentials for your email service.
  4. Customize the Form/Trigger: Adjust the initial node to collect the search parameters you need (business type, location, your email).
  5. Test with a Small Search: Run the workflow once with a narrow search (e.g., "coffee shops in Brooklyn") to verify everything works and the email arrives with the CSV.
  6. Schedule or Trigger: Set the workflow to run on a schedule (daily/weekly) or trigger it manually when you need a new batch of leads.
