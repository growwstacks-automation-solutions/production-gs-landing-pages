What This Workflow Does

Manual company research is one of the biggest time-sinks in sales and business development. When a new company gets added to your CRM, someone has to spend 15-30 minutes Googling, reading articles, and compiling notes before meaningful outreach can begin. This workflow eliminates that manual step entirely.

Whenever a new company is created in your CRM (CentralStationCRM in this template), it automatically triggers a web research process using Tavily's search API and ChatGPT's analysis capabilities. The AI gathers relevant information about the company—industry, size, recent news, funding, leadership, and potential opportunities—then formats it into a clean, readable note that gets saved directly back to the CRM.

Your sales team gets instant context about every new prospect without lifting a finger. This means faster follow-ups, more personalized outreach, and better prioritization of high-value accounts.

How It Works

1. Webhook Trigger from CRM

The workflow starts when CentralStationCRM sends a webhook notification about a new company being created. This webhook contains the company name and any initial data captured during creation.

2. AI-Powered Web Research

The company name is passed to an AI Agent node configured with ChatGPT and Tavily's search tools. Tavily performs a comprehensive web search, gathering information from multiple sources including news articles, company websites, and business databases.

3. Intelligent Analysis & Summarization

ChatGPT analyzes the search results, extracts key insights, and structures them into a well-formatted markdown note. The AI focuses on information most relevant to sales teams: industry positioning, recent developments, potential pain points, and opportunities for engagement.

4. Automated CRM Note Creation

The generated research note is automatically posted back to the CRM via an HTTP Request node, creating a "protocol" (note) attached to the company record. The entire process happens in seconds, providing immediate value to your team.

What You'll Need

  1. n8n instance (cloud or self-hosted) with community nodes enabled
  2. CentralStationCRM account with API access (or another CRM with webhook capabilities)
  3. OpenAI API key for ChatGPT access
  4. Tavily API key for web search functionality
  5. Basic understanding of how to set up webhooks in your CRM

Pro tip: If you're using a different CRM like HubSpot or Salesforce, you can easily adapt the HTTP Request node at the end of this workflow. The AI research portion remains exactly the same—only the final API call needs modification.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance
  2. Configure the Webhook trigger by setting up a webhook in CentralStationCRM that fires when new companies are created
  3. Add your API credentials for OpenAI and Tavily in their respective nodes
  4. Test the connection by creating a test company in your CRM and verifying the research note appears
  5. Activate the workflow and let it run automatically for all new company additions
