Auto Expense Tracker from LINE Messages with GPT-4 & Google Sheets

Automatically log personal spending from LINE messages using AI. Send a text or photo, and GPT-4 extracts the details into your Google Sheets ledger—no manual entry needed.

What This Workflow Does

Forgetful about tracking daily spending? This automation solves the universal problem of expense amnesia by turning your LINE messaging app into a smart financial assistant. Instead of manually opening spreadsheets or finance apps, you simply send a LINE message—whether text like "Coffee $4.50" or a photo of a receipt—and the system does the rest.

The workflow uses GPT-4 to intelligently parse your messages, extracting key details: amount, category, date, merchant, and notes. It then automatically logs this structured data into your Google Sheets ledger, creating a real-time, searchable expense history. The system even includes duplicate detection to prevent accidental double entries, making your financial tracking both effortless and accurate.

How It Works

Step 1: Message Reception via Webhook

When you send a message to your dedicated LINE bot, it triggers a webhook that captures the message content, whether text or an image. The workflow immediately receives this data and prepares it for processing, maintaining the conversational ease of regular messaging.

Step 2: AI-Powered Expense Extraction

The core intelligence happens here. GPT-4 analyzes your message, understanding natural language to identify the expense amount, category (food, transport, entertainment), date, and any contextual details. For receipt images, it performs OCR and data extraction, transforming visual information into structured data.

Step 3: Duplicate Prevention Check

Before logging anything, the system checks your existing Google Sheets data for similar recent entries. It creates a unique identifier for each potential entry and compares it against past records, ensuring you don't accidentally log the same expense twice—a common manual tracking error.

Step 4: Automated Google Sheets Logging

Once verified as new data, the extracted expense information is automatically written to your designated Google Sheet. The system organizes it into proper columns with timestamps, creating a clean, sortable financial record without any spreadsheet navigation or manual typing on your part.

Step 5: Confirmation & Response

The workflow sends a confirmation message back to you on LINE, showing what was logged or alerting you if it detected a duplicate. This closed-loop communication ensures you always know the system is working and what's been recorded.

What You'll Need

  1. A LINE Developer account to create your bot and obtain API credentials
  2. Google Cloud Platform access with Sheets API enabled and OAuth configured
  3. An OpenAI API key with GPT-4 access for the AI processing component
  4. A Google Sheet prepared with column headers for your expense data
  5. An n8n instance (cloud or self-hosted) to run the automation

Pro tip: Start with a simple Google Sheet structure—Date, Amount, Category, Description. You can always add more detailed columns later as the system proves valuable. The AI can be instructed to extract whatever fields you define in your sheet.

Quick Setup Guide

  1. Import the template: Download the JSON file above and import it into your n8n instance using the workflow import function.
  2. Configure credentials: Set up connections for LINE (webhook URL), Google Sheets (OAuth), and OpenAI (API key) in n8n's credentials management.
  3. Update your Google Sheet ID: Replace the placeholder Sheet ID in the Google Sheets node with your actual spreadsheet identifier.
  4. Set up LINE webhook: Copy the webhook URL from n8n and paste it into your LINE Developer Console's webhook settings.
  5. Test with a message: Send a test expense message to your LINE bot and verify it appears correctly in your Google Sheet.
  6. Customize categories: Adjust the AI instructions in the workflow to better match your spending categories and preferences.
