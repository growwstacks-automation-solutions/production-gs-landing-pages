What This Workflow Does

This automation solves the common problem of cluttered Google Drive folders by automatically detecting and classifying uploaded files based on their type. It intelligently moves PDFs, images, and documents into designated folders, maintaining an organized file structure without manual intervention.

The workflow is particularly valuable for teams that receive frequent file uploads from multiple sources. It eliminates hours of manual sorting while ensuring consistent file organization across departments. The template preserves all file metadata during transfer and includes error handling for reliable operation.

How It Works

1. File Detection

The workflow monitors a specified Google Drive folder for new uploads. When files are detected, it retrieves their MIME types for accurate classification.

2. Classification Logic

Using conditional nodes, files are categorized into three main types: PDFs, images (JPG/PNG), and documents (JSON/DOCX). The system can be easily extended to handle additional file types.

3. Organized Transfer

Files are moved to their designated destination folders while preserving all metadata. The workflow includes batch processing to handle large volumes efficiently.
