What This Workflow Does

This automation solves the challenge of maintaining an active social media presence with fresh, engaging visual content. Many content creators and small businesses struggle to produce enough quality images to post regularly, leading to inconsistent posting schedules or repetitive content.

The workflow automatically generates unique AI images using Google Gemini for creative prompts and Pollinations AI for image generation, then posts them to your Facebook Page with optimized captions and hashtags. It runs 3 times daily (adjustable) to maintain content consistency without manual effort.

How It Works

1. Scheduled Trigger

The workflow activates at preset times (7:00, 11:00, and 17:00 by default) using n8n's scheduler.

2. AI Prompt Generation

Google Gemini creates imaginative image prompts in styles like retro-futuristic or cinematic based on your preferences.

3. Image Creation

Pollinations AI generates high-quality images from the prompts with customizable parameters for style and composition.

4. Facebook Posting

The workflow publishes the images to your Facebook Page with clean captions and relevant hashtags for maximum reach.

What You'll Need

  1. Facebook Page with admin access
  2. Facebook Graph API token (pages_manage_posts permission)
  3. Google Gemini API key
  4. n8n instance (self-hosted or cloud)
  5. Basic understanding of API authentication

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance
  3. Configure Facebook Graph API credentials
  4. Add your Google Gemini API key
  5. Adjust posting schedule if needed
  6. Test with a private post first
