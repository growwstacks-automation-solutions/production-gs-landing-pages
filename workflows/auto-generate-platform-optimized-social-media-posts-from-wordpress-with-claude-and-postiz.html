What This Workflow Does

This workflow solves the time-consuming challenge of manually repurposing WordPress blog content for social media. It automatically transforms each blog post into platform-optimized social media content across LinkedIn, Facebook, Instagram, and Twitter (X).

By leveraging AI (Claude and OpenAI), it generates unique captions tailored to each platform's style and audience expectations. The system then schedules these posts through Postiz, ensuring consistent social media presence without daily manual effort.

How It Works

1. Content Extraction

The workflow starts by fetching the latest WordPress post content including title, body text, and featured image.

2. AI Content Transformation

Advanced AI analyzes the blog content and generates:

  • Professional, insight-driven captions for LinkedIn
  • Creative, emoji-rich posts for Instagram
  • Community-focused updates for Facebook
  • Concise, hashtag-optimized tweets for Twitter

3. Visual Enhancement (Optional)

For platforms that benefit from custom visuals, the workflow can generate AI-created images based on the post content.

4. Cross-Platform Publishing

The system automatically schedules each platform-specific post through Postiz at optimal times, with tracking to prevent duplicate posts.

What You'll Need

  1. A WordPress site with API access
  2. Postiz account (free trial available)
  3. Access to Claude or OpenAI API
  4. Google Sheets for workflow control
  5. Self-hosted n8n instance (required for community nodes)

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Connect your WordPress credentials in the "Get Post" node
  3. Configure API keys for Claude/OpenAI and Postiz
  4. Link your social accounts in Postiz and note the integration IDs
  5. Test with a sample WordPress post ID
  6. Schedule the workflow to run automatically when new posts publish
