What This Workflow Does

Every Shopify store with a blog faces the same challenge: readers finish one article and leave, missing your other valuable content. Manually adding "related articles" sections is time-consuming and often results in generic, keyword-based recommendations that don't truly engage readers.

This automation solves that by using AI to understand the actual meaning of your articles. It automatically identifies and inserts the three most semantically relevant related articles at the end of every blog post. The system runs weekly to keep recommendations fresh as you publish new content, creating a self-improving content network that boosts engagement metrics and SEO performance.

How It Works

1. Content Collection & Processing

The workflow pulls all published articles from your selected Shopify blog using the Admin API. It cleans the HTML content by removing any existing related articles sections and extracts clean text for AI processing.

2. AI Embedding Generation

Using OpenAI's text-embedding-3-small model, the system converts each article's content into numerical vectors (embeddings). These embeddings capture the semantic meaning of your content in a way computers can understand and compare.

3. Similarity Matching

The workflow calculates cosine similarity between all article embeddings to identify which pieces of content are most semantically related. It selects the top 3 matches for each article based on configurable similarity thresholds.

4. Dynamic Content Updates

For each article, the system builds a custom HTML "Related Articles" section with links, titles, and optional excerpts. It then updates the article content on Shopify via API, seamlessly integrating the recommendations into your existing blog layout.

5. Scheduled Maintenance

The automation runs on a weekly schedule (configurable) to recalculate relationships as new content is added. It stores all data in n8n Data Tables for tracking and optimization.

What You'll Need

  1. Shopify Admin API Access Token with read/write permissions for blogs
  2. OpenAI API Key with access to the embeddings API
  3. n8n instance (cloud or self-hosted) with Data Tables enabled
  4. Three Data Tables created in n8n: articles, article_relations, and article_related_links_snapshot
  5. Basic configuration values including your Shopify blog ID, store name, and similarity threshold

Pro tip: Start with a 70% similarity threshold and adjust based on your content diversity. Niche blogs can use higher thresholds, while broad-topic blogs might benefit from lower thresholds to discover cross-topic connections.

Quick Setup Guide

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Create the three Data Tables in n8n with the specified schema
  3. Add your credentials for Shopify and OpenAI in the credential management section
  4. Configure environment variables in the Workflow Configuration node with your Shopify blog details
  5. Test with a single article first to verify the HTML output matches your blog's styling
  6. Adjust the schedule trigger to match your content publishing frequency
  7. Run the workflow and monitor the Data Tables to see relationships being created
