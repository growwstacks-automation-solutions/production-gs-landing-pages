What This Workflow Does

This workflow solves the problem of manual, repetitive email response handling. When a new email arrives in your Gmail inbox, the system automatically determines if it's a genuine enquiry using GPT-4o. If it is, it passes the email content to a LangChain agent that uses Dumpling AI to search your internal knowledge base—like FAQs, documentation, or product pages—for relevant information. The agent then crafts a personalized, context-aware reply and sends it back via Gmail.

Instead of spending hours each day answering similar questions, you get an AI assistant that responds instantly with accurate, helpful information. This reduces response time from hours to minutes, improves customer satisfaction, and frees up your team to focus on higher-value tasks.

The workflow is especially valuable for businesses with high-volume enquiry channels—customer support, sales inquiries, or solopreneurs managing multiple client communications. It ensures every enquiry gets a timely, intelligent reply without human intervention.

How It Works

Step 1: Gmail Trigger listens for new emails

The workflow starts with a Gmail Trigger node that monitors your inbox for new messages. It captures the email subject, body, sender, and any attachments.

Step 2: GPT-4o classifies the email intent

The email content is passed to a GPT-4o node with a custom prompt that classifies whether the message is a genuine enquiry, spam, or another category. This ensures only relevant emails proceed to the next steps.

Step 3: Filter passes only enquiries

A Filter node checks the classification result. If the email is marked as an enquiry, it continues; otherwise, the workflow stops, preventing unnecessary processing of non-enquiry emails.

Step 4: Dumpling AI retrieves contextual knowledge

For genuine enquiries, the workflow calls your Dumpling AI agent via an HTTP Request node. Dumpling AI searches your uploaded documents, help articles, or website content to find information relevant to the enquiry.

Step 5: LangChain Agent constructs the reply

A LangChain Agent node, equipped with memory and tools, combines the original email, Dumpling AI's retrieved context, and your predefined response guidelines. It generates a coherent, helpful reply that addresses the customer's specific question.

Step 6: Gmail sends the personalized response

The final Gmail node sends the AI-generated reply back to the original sender, maintaining the email thread. The response is personalized, accurate, and delivered within minutes of the original enquiry.

  2. An OpenAI API key (for GPT-4o classification and LangChain agent).
  3. A Dumpling AI account and an agent trained on your relevant documents (help docs, product pages, FAQs).

  5. Optional: A Notion, Google Sheets, or CRM integration if you want to log conversations.

Quick Setup Guide

Follow these steps to get your AI email responder running in under 30 minutes:

  1. Import the template – Download the JSON file and import it into your n8n instance.
  2. Connect Gmail – Set up the Gmail Trigger node with your account credentials.
  3. Configure Dumpling AI – Sign up at Dumpling AI, create an agent, upload your knowledge documents, and paste your agent ID and API key into the HTTP Request node.
  4. Add OpenAI credentials – Enter your OpenAI API key in the GPT-4o and LangChain Agent nodes.
  5. Test the workflow – Send a test email to your Gmail and watch the automated reply generated.
  6. Customize prompts – Adjust the classification prompt and agent instructions to match your business tone and categories.

Pro tip: Train your Dumpling AI agent on a wide range of documents—product manuals, pricing pages, blog posts, and support articles. This gives the AI more context to pull from, resulting in more accurate and comprehensive replies.
