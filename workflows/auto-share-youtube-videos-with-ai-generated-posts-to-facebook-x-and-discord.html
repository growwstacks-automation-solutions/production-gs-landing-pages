What This Workflow Does

Content creators and marketers spend hours manually sharing each new YouTube video across social platforms, often with inconsistent messaging and missed opportunities. This automation solves that by instantly detecting new YouTube uploads and automatically distributing them to Facebook Pages, X (formerly Twitter), and Discord channels with AI-optimized captions.

The workflow monitors your YouTube channel via RSS, extracts video metadata, uses an AI language model to generate engaging platform-specific posts with relevant hashtags, then posts to all connected social accounts simultaneously. It also sends notifications to your Discord community, ensuring your most engaged followers never miss new content.

How It Works

1. YouTube Video Detection

The workflow polls your YouTube channel's RSS feed at regular intervals (every 15-60 minutes). When a new video is detected, it captures the title, description, URL, and thumbnail for processing.

2. AI Content Generation

Using an AI model like DeepSeek or OpenAI, the system analyzes the video metadata and generates engaging social media posts. The AI creates platform-optimized content: shorter posts for X with trending hashtags, longer descriptions for Facebook, and community-focused announcements for Discord.

3. Multi-Platform Distribution

The generated content is automatically posted to your connected Facebook Page via the Graph API, to your X account via Twitter API v2, and to your Discord channel via webhook. Each post includes the video link and appropriate engagement elements.

4. Notification and Logging

The workflow sends a confirmation to Discord with posting results and logs all activities. Failed posts trigger retry logic, and you receive alerts if any platform connection issues occur.

What You'll Need

  1. YouTube Channel: Public channel with regular uploads (RSS feed enabled by default)
  2. Facebook Page Access: Page admin role and Facebook Developer App credentials
  3. X Developer Account: Elevated access for Twitter API v2 posting permissions
  4. Discord Server: Webhook URL from your desired notification channel
  5. AI API Key: Access to DeepSeek, OpenAI, or similar LLM service
  6. n8n Instance: Self-hosted or cloud version of n8n

Pro tip: Start with one social platform (like Discord for notifications) before connecting all three. This lets you verify the AI content quality before broader distribution.

Quick Setup Guide

  1. Download and Import: Click the download button above to get the JSON file, then import it into your n8n instance.
  2. Configure YouTube RSS: Replace the sample RSS URL with your actual YouTube channel RSS feed.
  3. Set Up API Credentials: Add your Facebook Page token, X API keys, and Discord webhook in their respective nodes.
  4. Configure AI Service: Connect your preferred AI model (DeepSeek recommended for cost-effectiveness).
  5. Test with Existing Video: Manually trigger the workflow with a recent video URL to verify all connections work.
  6. Schedule and Activate: Set the RSS poll to run every 30 minutes and activate the workflow.
