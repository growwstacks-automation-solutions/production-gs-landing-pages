Automate AI Video Ad Generation with Google Veo 3, Gemini, and Airtable

Transform static product images and creative ideas into dynamic video ads automatically. Complete workflow from concept to downloadable video file.

What This Workflow Does

This automation solves the time-consuming and expensive process of creating professional video ads for marketing campaigns. Traditional video production requires specialized skills, equipment, and weeks of work. This workflow enables businesses to generate high-quality video ads in minutes by combining AI creativity with automated workflows.

The system takes a simple product image and a creative idea, then orchestrates multiple AI services to produce a complete video advertisement. It handles everything from creative direction to final video rendering, tracking each project in Airtable for easy management. This transforms what was once a complex production process into a simple, repeatable automation that scales with your marketing needs.

How It Works

1. Creative Brief Submission

The workflow starts with a web form where users submit their creative idea (like "A soda can exploding into a disco party") and upload a corresponding product image. This initiates the creative process and creates a tracking record in Airtable.

2. AI Creative Analysis

Google Gemini analyzes both the text prompt and uploaded image, acting as an AI creative director. It generates a detailed video brief that interprets the static image according to the creative vision, ensuring the final video aligns with marketing objectives.

3. Video Generation with Veo 3

The detailed creative brief is sent to Google's Veo 3 AI video generation model. The workflow manages the long-running task, handling API calls, waiting periods, and status updates while the video is being created.

4. Final Delivery and Tracking

Once the video is ready, the workflow retrieves it from the Veo 3 API, converts it to a downloadable format, updates the Airtable record with the final video link, and makes it available for immediate use in marketing campaigns.

What You'll Need

  1. n8n Instance: Cloud or self-hosted n8n with the LangChain integration package installed
  2. Google Cloud Account: Project with Vertex AI API enabled, access to Gemini and Veo 3 models, and proper authentication credentials
  3. Airtable Base: Setup with tables for tracking video projects, including columns for image prompts, attachments, and status
  4. Basic Understanding: Familiarity with n8n workflows and API authentication methods

Pro tip: Start with a small Google Cloud budget to test video generation costs before scaling. Veo 3 pricing varies based on video length and resolution, so monitor your usage during initial testing.

Quick Setup Guide

  1. Import the Template: Download and import the JSON file into your n8n instance
  2. Configure Airtable: Update all Airtable nodes with your Base ID, Table ID, and Personal Access Token
  3. Set Up Google AI: Add your Gemini API key and Google OAuth2 credentials for Vertex AI access
  4. Update Project References: Replace placeholder Project ID and Location values in the Veo 3 API calls
  5. Test the Workflow: Activate the workflow and use the provided form URL to submit your first creative brief
