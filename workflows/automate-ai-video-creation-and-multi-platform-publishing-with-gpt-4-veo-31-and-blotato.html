What This Workflow Does

This automation transforms raw ideas into published videos across multiple social platforms with zero manual intervention. It solves the time-consuming process of video content creation by combining AI script generation with automated video production and distribution.

The workflow reads content ideas from Google Sheets, generates professional scripts using GPT-4, creates cinematic videos with Veo 3.1, and publishes them simultaneously to TikTok, Instagram, Facebook, LinkedIn, YouTube and more through Blotato - all while updating your tracking spreadsheet.

Veo 3.1 AI video creation interface processing reference images into finished video content

How It Works

1. Content Ingestion

The workflow monitors a designated Google Sheet for new video ideas and reference images. Each row represents a potential video with niche, concept, and visual references.

2. AI Script Generation

GPT-4 analyzes the input concept and generates a compelling script complete with scene descriptions, captions, and platform-specific optimizations.

3. Video Production

Veo 3.1 processes the script and reference images to create a polished video with proper pacing, transitions, and visual storytelling elements.

4. Multi-Platform Publishing

Blotato handles the platform-specific formatting and publishes the video simultaneously to all connected social channels with optimal settings.

5. Status Updates

The workflow updates the original Google Sheet with completion status, video URLs, and performance metrics as they become available.

What You'll Need

  1. OpenAI API access for GPT-4
  2. fal.ai account for Veo 3.1 API
  3. Google Cloud project with Sheets and Drive APIs enabled
  4. Blotato account for multi-platform publishing
  5. n8n instance (cloud or self-hosted)

Pro tip: Start with a small test batch of 5-10 videos to refine your prompt engineering before scaling up production.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure API connections for OpenAI, Veo 3.1, Google Sheets and Blotato
  3. Set up your Google Sheet with the required columns (template provided)
  4. Add 3 reference image URLs per video concept
  5. Test with one row before enabling automation
