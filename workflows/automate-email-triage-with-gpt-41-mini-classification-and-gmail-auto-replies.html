How It Works

The automation follows a logical sequence that mimics how an efficient human assistant would handle email, but with machine speed and consistency.

1. Email Detection & Capture

The workflow continuously monitors your Gmail account for new unread messages. When a new email arrives, it immediately captures the complete message including sender details, subject line, body content, and any attachments.

2. Sender Information Extraction

Before processing the content, the system extracts the sender's name and email address. This information is used later for personalization, ensuring automated replies don't feel generic or robotic.

3. AI-Powered Classification

This is where the magic happens. Using OpenAI's GPT-4.1-mini model, the email content is analyzed and classified into one of four predefined categories: Sales Inquiry, Support Request, General Feedback, or Other. The AI understands context and intent, not just keywords.

4. Automated Action Execution

Based on the classification, the system executes the appropriate action. Sales inquiries might trigger an immediate personalized reply with follow-up information. Support requests could create a draft response for your team to review. All contacts are automatically logged to Google Sheets for future reference.

What You'll Need

  1. A Gmail account with access to enable API connections
  2. OpenAI API key (GPT-4.1-mini access required)
  3. Google Sheets for contact logging (optional but recommended)
  4. n8n instance (cloud or self-hosted) to run the workflow
  5. Four Gmail labels created in your inbox matching the classification categories

Quick Setup Guide

You can have this automation running in under 15 minutes with these simple steps:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Connect your accounts by adding credentials for Gmail, OpenAI, and Google Sheets in the n8n credentials manager
  3. Configure the workflow by updating the Set node with your specific Gmail label IDs, Google Sheet ID, and sheet name
  4. Create Gmail labels in your inbox exactly as specified in the workflow (Sales, Support, Feedback, Other)
  5. Test the system by sending a test email to your connected Gmail account and verifying the classification and response
  6. Activate the workflow and let it run continuously to handle incoming emails automatically

Pro tip: Keep the setup instructions in the sticky notes inside the workflow for easy reference. They contain all the specific details about label formats and configuration values.
