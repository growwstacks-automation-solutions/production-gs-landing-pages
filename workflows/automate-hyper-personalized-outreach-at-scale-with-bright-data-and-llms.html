What This Workflow Does

Manual, personalized outreach is a massive time sink for sales and marketing teams. Researching each prospect, finding relevant conversation starters, and crafting individual messages doesn't scale. This leads to generic, low-response outreach or unsustainable manual effort.

This n8n workflow automates the entire research and personalization process. It takes a list of LinkedIn profile URLs, enriches them with detailed data using Bright Data's Dataset API, and then uses Claude AI to generate respectful, relevant ice breakers based on the prospect's recent activity and profile information. The results are automatically written back to a Google Sheet, ready for your outreach campaigns.

You move from hours of manual research per lead to a fully automated, scalable system that delivers hyper-personalized messaging at the click of a button.

How It Works

The workflow orchestrates data between Google Sheets, Bright Data, and Anthropic's Claude AI through a series of logical steps.

Step 1: Input & Preparation

The workflow reads from a Google Sheet where each row contains a LinkedIn profile URL and a unique row identifier. It formats this data and prepares batches for efficient API processing.

Step 2: LinkedIn Profile Enrichment

Each URL is sent to Bright Data's Dataset API. The workflow intelligently polls the API, waiting for the data snapshot to be ready, then retrieves enriched details like the person's name, location, current company, 'About' section, and their most recent posts.

Step 3: Data Consolidation

The retrieved profile data is mapped back to the correct row in your Google Sheet using the row identifier, populating columns for name, city, company, and recent activity.

Step 4: AI Ice Breaker Generation

The enriched profile data—especially the recent posts—is sent to Claude AI with a custom prompt. The AI analyzes the context and generates a concise, relevant, and respectful 1–4 line ice breaker designed to start a genuine conversation.

Step 5: Output & Completion

The AI-generated ice breaker is written back to the 'Ice Breaker' column in the original row. Your sheet is now populated with rich prospect data and a tailored opening message, ready for use in your email or LinkedIn outreach sequences.

What You'll Need

  1. A Google Account & Sheet: A Google Sheet formatted with at least two columns: Linkedin_URL_Person and row_number.
  2. Bright Data Account: Access to Bright Data's Dataset API with a valid API key and an active dataset configured for LinkedIn URLs.
  3. Anthropic API Key: An API key for Claude (like Claude 3.5 Haiku) to generate the ice breakers.
  4. n8n Instance: Access to an n8n environment (cloud or self-hosted) with the HTTP Request, Google Sheets, and necessary node credentials configured.
  5. Basic Automation Understanding: Familiarity with connecting API services and importing JSON workflows into n8n.

Quick Setup Guide

Follow these steps to get the workflow running in your own n8n environment.

  1. Download & Import: Click the "Download Template" button above to get the JSON file. In your n8n instance, go to Workflows > Import from File and select the downloaded JSON.
  2. Configure Credentials: In n8n's Credentials settings, set up OAuth2 for Google Sheets, add your Bright Data API key as a generic HTTP Auth header, and add your Anthropic API key.
  3. Connect Your Sheet: In the first Google Sheets node, select your copied template sheet and the correct worksheet (usually 'Sheet1').
  4. Set Bright Data Details: In the HTTP Request node, paste your specific Bright Data dataset ID into the request URL parameters.
  5. Optional: Tune the AI Prompt: Review the text in the Claude node. You can modify it to change the tone, length, or focus of the generated ice breakers.
  6. Test & Run: Use the Manual Trigger node to test with 1-2 LinkedIn URLs. Once confirmed, you can activate the Schedule Trigger node to run the workflow daily or weekly automatically.
