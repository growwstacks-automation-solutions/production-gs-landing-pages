What This Workflow Does

Creative professionals drowning in unorganized visual assets face a constant time drain: manually sorting, tagging, and archiving images. This workflow eliminates that burden by automating the entire process. When a new image lands in your Google Drive "Inbox" folder, AI instantly analyzes its content, determines its category (portrait, landscape, etc.), extracts dominant colors, generates descriptive tags, checks for inappropriate content, moves it to the correct folder, and creates a comprehensive entry in your Notion portfolio database.

The system acts as a digital assistant for your visual library, ensuring every asset is properly categorized, tagged, and archived without human intervention. It transforms chaotic storage into a structured, searchable portfolio where you can instantly find images by content, color, or theme—saving countless hours previously spent on manual organization.

How It Works

Step 1: Image Upload Trigger

The workflow monitors a designated "Inbox" folder in Google Drive. Any new image upload automatically triggers the process, capturing the file details and preparing it for analysis.

Step 2: AI Vision Analysis

GPT-4o analyzes the image content, generating a detailed description, extracting dominant colors, and determining the primary category (e.g., Portrait vs. Landscape). This AI-powered analysis replaces manual inspection.

Step 3: Safety Filter Check

An AI-based NSFW filter evaluates the image for inappropriate content. If detected, the process stops immediately and sends a warning notification to Slack, protecting your portfolio integrity.

Step 4: Smart Sorting & Tagging

Based on the AI-determined category, the image is automatically moved to the correct subfolder (Portraits, Landscapes, etc.). Contextual tags (like "smile, natural light" for portraits) are generated and added to the file metadata.

Step 5: Notion Archiving

A comprehensive entry is created in your Notion database with the image link, AI-generated description, tags, category, and date. This creates a centralized, searchable archive of your visual assets.

Step 6: Success Notification

A summary alert is sent to Slack confirming the asset has been processed, categorized, tagged, and archived—keeping you informed without requiring manual checking.

What You'll Need

  1. n8n instance (version 1.0 or later) to run the workflow.
  2. OpenAI API access with GPT-4o model availability for accurate vision analysis.
  3. Google Drive with folder structure (Inbox, Portraits, Landscapes, etc.) prepared.
  4. Notion account with a database created for portfolio archiving.
  5. Slack workspace for notification channels.
  6. Basic configuration skills to set folder IDs, database ID, and channel ID in the workflow's central configuration node.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance.
  2. Connect credentials for Google Drive, OpenAI, Notion, and Slack in n8n.
  3. Open the "Workflow Configuration" node and replace placeholder IDs with your actual Folder IDs (Inbox, Portraits, Landscapes), Notion Database ID, and Slack Channel ID.
  4. Prepare your Notion database with properties: Category (Select), Description (Rich Text), Image URL (URL), Tags (Rich Text), Date (Date).
  5. Test the workflow by uploading an image to your Google Drive Inbox folder and verifying the automated processing, sorting, tagging, and archiving.

Pro tip: Customize the AI analysis prompts in the workflow to match your specific tagging preferences—adjust language style, add custom categories, or emphasize certain visual elements relevant to your work.
