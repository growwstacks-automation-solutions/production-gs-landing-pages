What This Workflow Does

This automation solves a common HR bottleneck: manually copying job descriptions from your Applicant Tracking System (ATS) to LinkedIn. Every time a new vacancy is created in Recrutei, this workflow triggers instantly. It extracts all job details, cleans the data, and uses GPT-4o to generate a marketing-ready LinkedIn post.

The AI acts as your copywriter, transforming technical job descriptions into engaging posts that highlight key benefits, company culture, and candidate requirements. The final post is published automatically to your LinkedIn company page, ensuring immediate visibility.

This eliminates delays, maintains consistency, and frees your recruitment team from repetitive administrative tasks. You get faster posting, better content, and more time for candidate engagement.

How It Works

Step 1: Recrutei Webhook Trigger

When a new job vacancy is published in Recrutei ATS, it sends a webhook notification containing all job data—title, description, location, salary, remote options, etc. This triggers the workflow instantly.

Step 2: Data Cleaning & Preparation

Two Code nodes process the raw data. The first converts boolean fields (like remote=1) into readable text ("yes"/"no"). The second maps data keys to user-friendly labels and sanitizes HTML descriptions into clean Markdown.

Step 3: AI Content Generation

The structured job details are sent to GPT-4o configured as a "Marketing Copywriter." The AI generates a compelling LinkedIn post optimized for engagement, incorporating persuasive language and highlighting what matters to candidates.

Step 4: LinkedIn Publishing

The AI-generated text is automatically posted to your configured LinkedIn profile or company page via the LinkedIn node. The post appears immediately, with no manual intervention.

Step 5: Internal Logging

The workflow logs the event—job title and confirmation status—into a Google Sheet for tracking and auditing. This provides visibility into every automated posting.

What You'll Need

  1. Recrutei ATS: An active Recrutei account with webhook capabilities enabled.
  2. LinkedIn Company Page: A LinkedIn page where you want to post jobs, with API access configured.
  3. OpenAI API Key: Access to GPT-4o (or GPT-4o-mini) for content generation.
  4. Google Sheets: A Google Sheet for logging automation events (optional but recommended).
  5. n8n Instance: A running n8n environment (cloud or self-hosted) to import and run the workflow.

Quick Setup Guide

  1. Download the Template: Click the download button above to get the JSON file.
  2. Import into n8n: In your n8n dashboard, import the JSON file via the workflow import function.
  3. Configure Credentials: Set up credentials for OpenAI, LinkedIn, and Google Sheets in n8n.
  4. Update Webhook URL: Copy the webhook URL from the n8n workflow and add it to your Recrutei ATS settings.
  5. Test with a Sample Job: Create a test vacancy in Recrutei to trigger the workflow and verify the LinkedIn post appears correctly.
  6. Adjust AI Prompt: Fine-tune the AI prompt in the OpenAI node to match your brand voice and posting style.

Pro tip: Before going live, run a few test postings to ensure the AI-generated content meets your quality standards. You can adjust the prompt to emphasize specific company values or benefits.
