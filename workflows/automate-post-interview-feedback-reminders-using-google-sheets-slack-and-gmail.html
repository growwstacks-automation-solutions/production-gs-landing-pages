Automate Post-Interview Feedback Reminders

Free n8n workflow to automatically send follow-up reminders from Google Sheets to Slack (with email fallback), ensuring timely feedback from hiring teams.

What This Workflow Does

Collecting feedback after interviews is a critical yet frustrating bottleneck in the hiring process. Interviewers get busy, emails get buried, and recruiters waste hours chasing people down. This delay can cause you to lose top candidates to faster-moving competitors.

This n8n workflow solves that by fully automating the reminder process. It connects your single source of truth—a Google Sheet tracking all interviews—with your team's communication hub, Slack. It checks daily for pending feedback, sends personalized Slack DMs to interviewers, and automatically falls back to email if a Slack ID isn't available. Finally, it updates the sheet to prevent duplicate reminders, creating a closed-loop, hands-off system.

You eliminate manual follow-up, ensure consistent and timely nudges, and get feedback faster to make hiring decisions without delay. It turns a chaotic, manual process into a reliable, automated pipeline.

How It Works

The workflow runs on a schedule you set, acting as your automated recruitment coordinator.

Step 1: Scheduled Trigger & Data Fetch

A Schedule Trigger initiates the workflow daily (e.g., at 6 PM). It then connects to your designated Google Sheet via the Google Sheets node, reading all rows containing recent interview data.

Step 2: Filter for Pending Feedback

An IF node filters the data, isolating only the rows where the 'Feedback Submitted' column is not marked "Yes." This ensures the system only acts on interviews that still require follow-up.

Step 3: Determine Communication Channel

Another IF node checks each filtered row for a valid 'Slack ID.' If a Slack ID is present, the workflow proceeds to send a Slack DM. If the field is empty, it routes the reminder through the Gmail node as a fallback.

Step 4: Send Personalized Reminders

For Slack, the workflow uses the Slack node to send a direct message to the interviewer, mentioning the candidate's name and position. For email, the Gmail node sends a similar personalized nudge. The message templates are fully customizable.

Step 5: Update Tracking Sheet

Finally, a Google Sheets node updates the 'Reminder Sent' column to "Yes" for each processed row. This log prevents the same interviewer from receiving multiple reminders for the same interview in subsequent runs.

What You'll Need

  1. A running n8n instance (cloud or self-hosted).
  2. A Google Sheet with columns for: Candidate Name, Interviewer Email, Interviewer Slack ID (optional), Feedback Status (Yes/No), and Reminder Sent (Yes/No).
  3. Google Service Account credentials or OAuth setup to allow n8n to read and write to your Sheet.
  4. A Slack Bot Token with permissions to chat:write and users:read (to send DMs).
  5. A Gmail account connected to n8n for the email fallback functionality.

Pro tip: Create a dedicated "Interview Tracking" Slack channel and invite your bot. You can configure the workflow to also post a summary of sent reminders to this channel for full team visibility.

Quick Setup Guide

You can have this automation running in under 15 minutes.

  1. Download the template using the button above and import it into your n8n workspace.
  2. Prepare your Google Sheet with the required columns and share it with the service account email from your n8n Google Sheets credential.
  3. Configure the credentials in n8n for Google Sheets, Slack, and Gmail.
  4. Map your data by updating the node fields to match your Google Sheet column names (e.g., 'interviewer_email', 'slack_id').
  5. Customize the message templates in the Slack and Gmail nodes to match your company's tone.
  6. Set the schedule on the Schedule Trigger node (e.g., "0 18 * * 1-5" for 6 PM on weekdays).
  7. Activate the workflow and test it with a single row in your sheet.
