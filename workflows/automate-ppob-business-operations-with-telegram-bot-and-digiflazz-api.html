What This Workflow Does

This automation transforms your PPOB (Payment Point Online Bank) business operations by creating a fully-functional Telegram bot integrated with Digiflazz API. It handles the complete customer journey - from product discovery and price checking to transaction processing and receipt delivery - without manual intervention.

The system eliminates repetitive tasks like checking product availability, calculating prices, processing transactions, and sending confirmations. Business owners gain real-time visibility into sales, profits, and inventory while customers enjoy instant, 24/7 access to digital payment services through a familiar messaging interface.

How It Works

1. Customer Interaction via Telegram

The Telegram bot greets users with a menu of available PPOB services. Customers can browse products, check prices, and initiate transactions through simple text commands or interactive buttons.

2. Real-time Product Verification

When a customer selects a product, the workflow queries Digiflazz API to verify current availability and pricing. This ensures customers always see accurate, up-to-date information before purchasing.

3. Transaction Processing

After payment confirmation, the system automatically submits the transaction to Digiflazz, processes the order, and retrieves the digital product (token, voucher code, etc.). All steps happen in seconds without manual intervention.

4. Automated Receipt Delivery

The bot sends a formatted receipt to the customer via Telegram, including transaction details, product information, and customer support contacts. Simultaneously, it updates your sales records and balance sheets.

What You'll Need

  1. An active Digiflazz account with API access enabled
  2. A Telegram bot token from BotFather
  3. n8n instance (cloud or self-hosted)
  4. Basic understanding of PPOB product codes
  5. Payment processing method (can integrate with existing system)

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure your Digiflazz API credentials in the HTTP Request nodes
  3. Set up your Telegram bot token in the Telegram trigger node
  4. Customize product codes and pricing in the Function nodes
  5. Test with small transactions before going live
