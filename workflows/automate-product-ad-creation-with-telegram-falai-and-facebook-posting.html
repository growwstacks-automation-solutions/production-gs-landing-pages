Automate Product Ad Creation from Telegram to Facebook with AI

Turn a product photo sent via Telegram into a published Facebook ad in minutes. AI analyzes, generates images, writes captions, and posts—fully automated.

What This Workflow Does

This workflow solves a major time sink for marketers and e-commerce businesses: manually creating and publishing social media ads. The process typically involves editing a product photo, writing a creative brief, generating an image with AI, crafting a caption, logging into Facebook, and finally posting—a repetitive cycle that can take hours each week.

This automation condenses all those steps into a single, seamless flow triggered by a simple Telegram message. Send a product photo to your dedicated bot, and the system takes over: AI analyzes the image, creates a professional ad prompt, generates a marketing-ready visual via Fal.AI, writes an engaging caption, and publishes it to your Facebook page—all with human approval checkpoints along the way.

The result is a dramatic reduction in operational friction. Campaigns launch faster, creative resources are freed for strategy, and social media presence remains consistent without the manual grind. It’s designed for agility, allowing businesses to capitalize on trends or promotions instantly.

How It Works

The workflow is logically divided into four distinct zones, each handling a critical stage of the ad creation pipeline.

1. Product Image Analysis (Green Zone)

The automation is triggered when a user sends a product image to a configured Telegram bot. n8n retrieves the image file via the Telegram API. An OpenAI Vision node then analyzes the photo, identifying key attributes like color, material, shape, and context. This structured description is passed to an AI agent to prepare for the next stage.

2. Generate Ad Image Prompt (Red Zone)

An AI agent uses the product description and optional campaign context (e.g., "10.10 Sale") to craft a professional, detailed prompt suitable for an image generation model. This prompt is sent back to the user on Telegram for confirmation, ensuring creative control before any resources are used.

3. Create Ad Image via Fal.AI (Yellow Zone)

Once approved, the prompt and original image URL are sent to Fal.AI's image generation API. The workflow polls for completion and retrieves the high-quality generated ad image. This final visual is sent to Telegram for the user's final review and approval.

4. Write Caption & Publish (Blue Zone)

The approved ad image is analyzed again by AI to write a compelling, platform-optimized Facebook/Instagram caption. The user confirms the text. Finally, the workflow uses the Facebook Graph API to upload the image and post the caption to the specified page, completing the automated publication.

What You'll Need

  1. An n8n instance (self-hosted or Cloud account).
  2. A Telegram Bot Token (created via @BotFather).
  3. A Fal.AI API key for image generation.
  4. A Facebook Page Access Token with pages_manage_posts permission.
  5. An OpenAI API key for image analysis and text generation.
  6. Basic familiarity with n8n credentials and node configuration.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n workspace.
  2. Configure Credentials: In n8n's credentials management, set up keys for Telegram, Fal.AI, Facebook Graph API, and OpenAI.
  3. Connect the Telegram Trigger: Paste your bot token into the Telegram trigger node and set up the webhook.
  4. Set Fal.AI Endpoint: In the HTTP Request node, ensure the URL points to your desired Fal.AI model (e.g., https://fal.run/fal-ai/nano-banana) and authentication uses your Fal.AI key.
  5. Test the Flow: Send a product photo to your Telegram bot. Monitor the execution and approve prompts at the designated steps.
  6. Go Live: Once testing is successful, activate the workflow for continuous use.

Pro tip: Use the Telegram confirmation steps as quality gates. This maintains human oversight for brand voice and creative direction while automating the heavy lifting.
