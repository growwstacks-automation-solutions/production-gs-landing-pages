What This Workflow Does

This n8n automation transforms the tedious process of quiz creation into a seamless, automated system. Instead of manually reading documents, formulating questions, and building quizzes, this workflow does everything automatically. When someone uploads a document (PDF, Word, etc.) through a Jotform, the system extracts the text, sends it to Google Gemini AI for analysis, generates multiple-choice questions with correct answers, saves everything to Google Sheets for record-keeping, and finally creates a fully functional Jotform quiz ready for sharing.

The workflow solves a critical pain point for educators, corporate trainers, and content creators who regularly need to assess comprehension. Creating quality quizzes can take hours per document—this automation reduces that to minutes while maintaining educational value and accuracy.

Beyond basic question generation, the system ensures questions are contextually relevant, properly formatted, and ready for immediate use. The automated pipeline from document to published quiz eliminates manual steps while maintaining complete control over the final output.

How It Works

1. Document Submission & Trigger

The process begins when a user submits a document through a Jotform. The form captures the document file, desired quiz title, and number of questions needed. This submission triggers the entire automation workflow, ensuring no manual intervention is required to start the quiz creation process.

2. Document Processing & Text Extraction

n8n retrieves the uploaded document and uses the Extract from File node to parse and extract all text content. This works with various formats including PDFs, Word documents, and text files. The extracted text is cleaned and prepared for AI analysis, maintaining the original structure and key information.

3. AI-Powered Question Generation

The document text, along with the quiz title and question count, is sent to Google Gemini AI. The AI agent follows specific instructions to analyze the content, identify key concepts, and generate appropriate multiple-choice questions with four options each. It also identifies and marks the correct answer for each question.

4. Data Structuring & Validation

The AI-generated questions are formatted into structured JSON data. A Structured Output Parser node validates the format and ensures consistency. The data is then split into individual question items for further processing and record-keeping.

5. Google Sheets Record Keeping

Each generated question, along with all its options and the correct answer, is appended as a new row in a designated Google Sheet. This creates a searchable database of all generated questions, allowing for review, editing, and reuse across multiple quizzes.

6. Jotform Quiz Creation

The workflow dynamically constructs the API request body for Jotform, converting the AI-generated questions into the proper form fields. An HTTP Request node calls the Jotform API to create a brand-new, fully functional quiz form with all questions and options properly formatted.

7. Output & Distribution

The final output provides the direct link to the newly created Jotform quiz. This link can be immediately shared with students, trainees, or assessment participants. The quiz is ready to accept submissions and collect responses automatically.

Pro tip: Add an Email or Slack node after the HTTP Request to automatically send the quiz link back to the person who submitted the document. This creates a complete hands-off system where users submit documents and receive ready-to-use quiz links automatically.

What You'll Need

  1. Jotform Account & API Key: Required for both triggering the workflow and creating the final quiz forms via API.
  2. Google Gemini API Key: To power the AI question generation capabilities.
  3. Google Sheets Access: OAuth2 or API key credentials for saving generated questions.
  4. Source Jotform: A form with file upload, text field for quiz title, and number field for question count.
  5. n8n Instance: Self-hosted or cloud version of n8n to run the workflow.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure Jotform credentials for both the trigger node and the HTTP Request node that creates the quiz.
  3. Set up Google Gemini API in the AI Agent node with your API key and preferred model settings.
  4. Connect Google Sheets with OAuth2 or API key authentication for the spreadsheet node.
  5. Create your source Jotform with file upload, quiz title, and question count fields.
  6. Test the workflow by submitting a sample document through your Jotform.
  7. Customize question instructions in the AI Agent node to match your specific quiz requirements.
