What This Workflow Does

This automation solves the tedious manual process of updating property values in your YNAB budget. Real estate investors and homeowners often struggle to maintain accurate asset valuations in their financial tracking systems. The workflow pulls current market data from Rentcast.io's API, updates your YNAB accounts accordingly, and sends Discord notifications about value changes.

By automating this process, you eliminate the risk of outdated valuations skewing your net worth calculations. The system runs on a schedule you define (weekly, monthly, etc.), ensuring your financial picture remains current without any manual intervention. Discord alerts provide immediate visibility into valuation changes across your portfolio.

YNAB asset account showing updated property value from Rentcast.io

How It Works

1. Fetch property data from Rentcast.io

The workflow starts by querying Rentcast.io's API with your property address or ID. It retrieves the current estimated market value along with supporting data like comparable sales and rental rates if available.

2. Process valuation changes

The system compares the new valuation with your last recorded value in YNAB. Significant changes trigger the alert logic while minor fluctuations may be logged without notification based on your thresholds.

Discord alert showing the property value change notification

3. Update YNAB asset values

Using YNAB's API, the workflow locates your real estate asset account and updates the balance to reflect the current market value. The transaction includes notes about the data source and valuation date for audit purposes.

4. Send Discord notifications

For any meaningful value changes, the system posts a formatted message to your designated Discord channel. The alert includes the property details, old/new values, percentage change, and links to relevant reports.
