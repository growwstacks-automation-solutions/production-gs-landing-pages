What This Workflow Does

This automation solves the common challenge of collecting customer reviews for eCommerce stores by leveraging WhatsApp's high open rates. When a WooCommerce order is marked as completed, the workflow automatically sends a personalized WhatsApp message through Rapiwa requesting the customer to leave a review.

The system eliminates manual follow-ups while maintaining a personal touch. It waits until the order is fulfilled (not just paid) to ensure customers have actually received and experienced the product before asking for feedback. This timing significantly increases review quality and response rates compared to email requests.

How It Works

1. Order completion trigger

The workflow monitors your WooCommerce store for order status changes. When an order transitions to "completed" status, the automation is triggered and pulls all relevant order details.

2. Customer data validation

The system checks that the customer has a valid WhatsApp number registered with their order. If no number exists or it's invalid, the workflow can be configured to either skip that customer or fall back to email.

3. Personalized message generation

Using order details (customer name, product purchased, order date), the workflow constructs a personalized WhatsApp message with a friendly tone and direct link to your review platform.

4. WhatsApp delivery via Rapiwa

The message is sent through Rapiwa's WhatsApp Business API integration, ensuring reliable delivery and proper formatting. The template includes options for customers to respond directly via WhatsApp if they have questions.
