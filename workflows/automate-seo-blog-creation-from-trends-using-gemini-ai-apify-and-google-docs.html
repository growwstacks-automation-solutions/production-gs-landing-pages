What This Workflow Does

This n8n workflow automates the entire process of creating SEO-optimized blog content from trending topics. It solves the constant challenge content creators face: finding relevant topics, researching keywords, writing engaging articles, and formatting them for publication—all while maintaining SEO best practices.

The system pulls real-time trending search data using Apify's Google Trends scraper, processes it through Google's Gemini AI to generate well-structured articles complete with meta descriptions and headings, creates custom images via KIE AI's Nano Banana API, and automatically formats everything into a Google Doc ready for review and publishing.

Instead of spending 5-10 hours per article on research, writing, and formatting, this automation delivers publish-ready content in minutes. It's particularly valuable for digital marketing agencies, SEO specialists, and content teams needing to maintain consistent publishing schedules while targeting current search trends.

How It Works

The workflow follows a logical sequence that mimics what a skilled content creator would do manually, but at scale and with perfect consistency.

1. Trend Discovery & Research

The workflow starts by connecting to Apify's Google Trends scraper to fetch currently trending search topics in your target region. You can configure it for specific countries, timeframes, and search categories. The system filters out irrelevant topics and selects the most promising subjects based on search volume and relevance.

2. AI Content Generation

Each selected trend is sent to Google's Gemini AI with a carefully crafted prompt that includes SEO best practices, target keywords, tone guidelines, and structural requirements. The AI generates a complete article including: compelling title, meta description, H1/H2/H3 headings, and comprehensive body content optimized for both readers and search engines.

3. Content Refinement

An optional refinement step allows the AI to review and improve its own output. This second pass enhances readability, ensures proper keyword placement, and adds engaging elements like questions, examples, and calls-to-action that increase dwell time and engagement metrics.

4. Visual Asset Creation

Using the generated article title as inspiration, the workflow calls KIE AI's Nano Banana API to create a custom featured image. You can specify style preferences, dimensions, and branding elements. The image is automatically optimized for web and social media sharing.

5. Document Assembly & Delivery

Finally, all components—title, meta description, headings, body content, and featured image—are automatically inserted into a Google Doc using proper formatting. The document is shared with your team for final review, or can be scheduled for immediate publication if you're using automated publishing workflows.

What You'll Need

  1. Apify Account & API Token for accessing Google Trends data (free tier available)
  2. Google Cloud Project with Docs API and Gemini AI API enabled
  3. Google Service Account Credentials with access to your target Google Drive folder
  4. KIE AI Nano Banana API Key for image generation (alternative: DALL-E or Stable Diffusion APIs)
  5. n8n Instance (cloud or self-hosted) with webhook capabilities for scheduling

Pro tip: Start with Apify's free tier (10k monthly credits) and Google's free Gemini API tier (60 requests per minute). This gives you enough capacity to test and run the workflow before committing to paid plans.

Quick Setup Guide

  1. Import the template into your n8n instance using the download button above
  2. Configure Apify node with your API token and set your target country/region
  3. Set up Google OAuth2 in the Google Docs node to connect to your Drive account
  4. Add your Gemini API key and customize the content generation prompts if needed
  5. Connect KIE AI with your API token for image generation
  6. Test with a single run to verify all connections work properly
  7. Schedule the workflow to run daily, weekly, or based on your content calendar
