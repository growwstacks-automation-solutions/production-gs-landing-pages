What This Workflow Does

This automation solution solves the time-consuming problem of manually checking website performance metrics. It automatically analyzes your site using Google's PageSpeed Insights API, compares results against competitors, and delivers actionable insights through Gemini AI analysis - all delivered via Discord notifications.

The workflow eliminates the need for manual testing while providing deeper analysis than standard tools. It tracks key metrics like Core Web Vitals, First Contentful Paint, and Time to Interactive, then uses AI to highlight optimization opportunities in plain language.

PageSpeed Insights API integration configured in the n8n workflow

How It Works

1. Performance Data Collection

The workflow triggers on a schedule (daily/weekly) to fetch performance data from PageSpeed Insights API for your specified URLs. It captures both mobile and desktop metrics.

2. Competitive Benchmarking

Optionally compares your site against competitor URLs you configure, highlighting relative performance differences across key metrics.

Gemini AI analyzing the performance data to generate insights

3. AI-Powered Analysis

Gemini processes the raw metrics, identifying trends, prioritizing issues, and suggesting optimization strategies in easy-to-understand language.

4. Notification Delivery

Formatted reports get sent to your Discord channel (or other configured notification platform) with visual highlights of critical issues.

PageSpeed Insights measures both lab data (controlled testing) and field data (real-user metrics). Key metrics include Core Web Vitals (LCP, FID, CLS), First Contentful Paint, Speed Index, and Time to Interactive.

These metrics directly impact user experience and SEO rankings. For instance, a slow LCP (Largest Contentful Paint) means visitors wait longer to see meaningful content, increasing bounce rates. The workflow tracks all these automatically.

  • Mobile and desktop performance separately
