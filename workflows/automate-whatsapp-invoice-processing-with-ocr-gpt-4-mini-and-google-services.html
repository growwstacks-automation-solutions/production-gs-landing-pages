What This Workflow Does

Manually processing invoices from WhatsApp is a tedious, error-prone task that drains valuable time from business owners, freelancers, and accountants. This workflow automates the entire process: it listens for new WhatsApp messages, detects invoice attachments, extracts text using OCR, summarizes key details with AI, logs everything to Google Sheets, stores the original file in Google Drive, and sends a confirmation back to the sender—all without you lifting a finger.

Imagine a client sends an invoice photo via WhatsApp. Within seconds, the data is captured, a clean summary is generated, and a record is created in your master spreadsheet. How It Works

The workflow is built in n8n and connects several powerful services in a logical sequence.

1. WhatsApp Trigger

The automation starts when a new message arrives in your connected WhatsApp Business account. It checks for media attachments (images or PDFs) that are likely invoices.

2. File Download & OCR Processing

The attached invoice file is downloaded. It's then sent to an OCR (Optical Character Recognition) service, which scans the document and converts all visible text into machine-readable data.

3. AI-Powered Summarization

The raw extracted text is sent to an AI model (GPT-4 Mini). The AI identifies and extracts key fields: vendor name, invoice date, total amount, due date, and line items. It outputs a concise, structured summary.

4. Data Logging to Google Sheets

The structured invoice data (vendor, date, amount, due date, summary) is appended as a new row in a predefined Google Sheet. This creates a searchable, sortable master log of all processed invoices.

5. File Storage in Google Drive

The original invoice file is uploaded to a specific folder in your Google Drive, named logically (e.g., "Invoices/2024/Vendor_Name_Date.pdf"). This ensures safe, organized archiving.

6. Confirmation Reply

Finally, a confirmation message is sent back via WhatsApp to the original sender (or to an internal chat), acknowledging receipt and summarizing the action taken (e.g., "Invoice from [Vendor] for [Amount] logged and filed").

What You'll Need

  1. An n8n instance: Either n8n.cloud account or self-hosted setup.
  2. WhatsApp Business API Access: A Meta Developer account and configured WhatsApp Business API to receive messages.
  3. OCR.Space API Key: A free or paid account from OCR.space for text extraction.
  4. OpenAI API Key: For using the GPT-4 Mini model to summarize invoice data.
  5. Google Cloud Project: With Google Sheets and Google Drive APIs enabled, and service account credentials.
  6. A Google Sheet & Drive Folder: Pre-created for logging and storing the invoices.

Pro tip: Use the free tiers of OCR.space and OpenAI to test the workflow. For production, consider paid plans for higher volume and reliability.

Quick Setup Guide

Get this automation running in your environment in about 15 minutes.

  1. Download & Import: Click the "Download Template" button above to get the JSON file. In your n8n dashboard, go to Workflows > Import from File and select the downloaded JSON.
  2. Configure WhatsApp Trigger: In the first node, set up the WhatsApp trigger with your Business API credentials and webhook verification.
  3. Set Up OCR & AI Services: In the HTTP Request nodes, add your OCR.space and OpenAI API keys. Test the connection with a sample invoice image.
  4. Connect Google Services: Authenticate the Google Sheets and Google Drive nodes with your service account JSON key. Specify the target Sheet ID and Drive folder ID.
  5. Test the Flow: Send a test invoice image to your WhatsApp number. Verify the data appears in your Sheet and the file is saved in Drive.
  6. Activate & Monitor: Turn the workflow on. Monitor the executions tab initially to ensure everything runs smoothly.
