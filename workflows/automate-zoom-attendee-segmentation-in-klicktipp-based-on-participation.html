What This Workflow Does

This automation solves the manual headache of transferring webinar attendees from Zoom to your email marketing platform. After each Zoom webinar ends, the workflow automatically captures participant data, calculates how long each person attended, and transfers them to KlickTipp with appropriate segmentation tags.

Instead of downloading CSV files and manually uploading contacts, this integration happens instantly. Attendees who stayed for 90%+ of your webinar get tagged differently than those who left early, enabling you to send targeted follow-up sequences that match their demonstrated interest level.

The system validates webhook security, excludes internal users like hosts, and routes participants based on webinar topic—perfect for running multiple webinar series with automated, topic-specific follow-ups.

How It Works

1. Zoom Webhook Trigger

When a Zoom webinar ends, Zoom sends a meeting.ended webhook event to n8n. The workflow first validates this webhook using HMAC security to ensure it's genuinely from Zoom.

2. Participant Data Retrieval

After a brief wait to ensure Zoom's data is ready, the workflow fetches the complete participant list from Zoom's API, including each person's name, email, and exact attendance duration.

3. Data Processing & Filtering

The participant list is split into individual records, and internal users (hosts, co-hosts) are filtered out. The workflow then checks the webinar topic to determine which KlickTipp list and tags to apply.

4. Attendance Calculation & Segmentation

For each attendee, the system calculates their attendance percentage by comparing their duration against the total meeting time. Those with ≥90% attendance get "fully attended" tags; others get general attendance tags.

5. KlickTipp Integration

Each participant is subscribed to KlickTipp with mapped fields (name, email, webinar details) and receives appropriate tags based on their attendance level and webinar topic.

What You'll Need

  1. Zoom account with API access enabled (Business, Education, or Enterprise plan)
  2. KlickTipp account with API credentials
  3. n8n instance (self-hosted or cloud)
  4. Webhook endpoint accessible from Zoom (if self-hosting n8n)
  5. Custom fields in KlickTipp for webinar data (meeting topic, duration, join URL)
  6. Tags in KlickTipp for attendance segmentation (attended, attended fully, not attended)

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance.
  2. Configure Zoom OAuth2 credentials in the Zoom webhook and HTTP Request nodes with appropriate scopes (meeting:read:meeting and meeting:read:list_past_participants).
  3. Set up KlickTipp API credentials in the KlickTipp community nodes.
  4. Update the Switch node with your specific webinar topics and corresponding KlickTipp tags.
  5. Test with a completed webinar by triggering the webhook or using n8n's manual trigger.
  6. Verify in KlickTipp that attendees appear with correct tags and custom field data.

Pro tip: Create separate email sequences in KlickTipp for "fully attended" vs "partially attended" segments. Send your premium offer to highly engaged attendees and nurturing content to those who left early.
