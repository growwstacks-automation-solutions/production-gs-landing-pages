What This Workflow Does

This automation solves the time‑consuming, manual process of researching, writing, and publishing blog content. It pulls trending keywords from Google Trends, uses AI (GPT‑4) to generate a full article around that topic, finds a relevant stock image from Pexels, adds a watermark and SEO metadata, and publishes the complete post directly to your WordPress site.

The result is a consistently updated blog that ranks for current search trends, without requiring a writer to spend hours on each piece. It’s ideal for content marketing teams, SEO agencies, and businesses that need to maintain an active publication schedule but lack the resources for manual content creation.

Beyond just publishing, the workflow logs keywords into Google Sheets for tracking, applies brand consistency through watermarks, and ensures each article is structured for readability and search engine performance.

How It Works

Step 1: Fetch Trending Keywords

The workflow starts by querying Google Trends via an HTTP request. It retrieves an XML feed of currently popular search terms in your chosen region or category, then converts that data into JSON for easy processing.

Step 2: Filter & Select a Keyword

The list of keywords is filtered to remove irrelevant or overly broad terms. One keyword is randomly selected (or you can configure priority rules) and logged into a Google Sheet for historical tracking.

Step 3: Generate AI Content

Using OpenAI’s GPT‑4 (or another AI model), the system creates a blog article based on the selected keyword. A structured output parser ensures the text is clean, well‑formatted, and includes headings, paragraphs, and a conclusion.

Step 4: Source & Watermark an Image

The workflow searches Pexels for a stock image matching the article’s topic. It downloads the image, applies a custom watermark (your logo or brand text), and adds alt text and caption metadata for SEO.

Step 5: Publish to WordPress

The final step assembles the AI‑generated text and watermarked image into a WordPress post. It sets the title, categories, tags, featured image, and publication status, then pushes the article live via the WordPress REST API.

What You'll Need

  1. OpenAI Account: API access for GPT‑4 or another model to generate article content.
  2. Pexels API Key: Free key from Pexels.com to search and download stock images.
  3. Google Account: With Sheets access to log keywords and track performance.
  4. WordPress Site: Self‑hosted or WordPress.com with REST API enabled and credentials ready.
  5. n8n Instance: Either cloud‑hosted (n8n.cloud) or self‑hosted to run the workflow.

Pro tip: Adjust the AI prompt to match your industry’s tone—technical, conversational, or authoritative. You can also replace Pexels with Unsplash or your own media library if preferred.

Quick Setup Guide

  1. Download the template: Click the download button above to get the JSON file.
  2. Import into n8n: In your n8n dashboard, go to Workflows → Import and select the downloaded file.
  3. Configure credentials: Set up API keys for OpenAI, Pexels, Google Sheets, and WordPress in n8n’s credentials section.
  4. Adjust settings: Modify the Google Trends region, keyword filters, AI prompt, watermark text, and WordPress categories to fit your needs.
  5. Test and activate: Run the workflow once manually to verify each step works, then enable the schedule trigger for automatic execution.
