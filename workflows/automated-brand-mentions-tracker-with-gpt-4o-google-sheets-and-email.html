How It Works

The automation runs on a scheduled basis (daily by default) and follows a structured process to gather, analyze, and report AI brand intelligence.

1. Configuration & Query Execution

The workflow begins by loading your predefined configuration: your brand name, list of competitors, and specific queries to ask the AI. These queries should reflect what potential customers might ask about your industry, products, or services. The system then sends each query to the OpenAI ChatGPT API (GPT-4o) via HTTP requests.

2. AI Response Analysis

Each AI response is processed through a custom code node that performs three critical checks: identifies mentions of your brand (case-insensitive), detects any listed competitors, and extracts the core response content. This analysis happens automatically, with the system handling variations in how brands might be referenced.

3. Data Logging & Organization

All results—timestamp, query, AI response, brand mention status, competitor mentions, and any errors—are appended to a designated Google Sheet. This creates a searchable historical database of how AI has represented your brand over time, enabling trend analysis and pattern recognition.

4. Report Generation & Delivery

The system compiles a comprehensive HTML email report summarizing total queries processed, brand mention frequency, competitor mentions (with breakdowns), and any issues encountered. This report includes both summary statistics and a detailed query-by-query breakdown, delivered automatically to your specified recipients.

Pro tip: Start with 5-10 core queries that represent your most valuable customer questions. Monitor for a week, then expand based on what you learn about how AI discusses your industry.
