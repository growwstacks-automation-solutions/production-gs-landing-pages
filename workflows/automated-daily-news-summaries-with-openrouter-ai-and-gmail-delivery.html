What This Workflow Does

This automation solution solves the problem of information overload by automatically collecting, analyzing, and summarizing the latest news relevant to your interests. Instead of manually scanning multiple news sources each day, you receive a professionally formatted email digest containing only the most important developments.

The system combines AI-powered analysis with reliable email delivery to create a hands-free news monitoring service. Businesses use this to track industry trends, competitors, and market changes without dedicating staff time to manual research.

How It Works

1. Scheduled Trigger

The workflow begins automatically at your preferred time each day, ensuring fresh content arrives when you need it.

2. Dynamic News Gathering

A custom query builder creates targeted search terms based on your interests and the current date, ensuring relevant results.

3. AI-Powered Analysis

OpenRouter AI evaluates articles from multiple sources, identifies key points, and generates concise summaries while maintaining context.

4. Professional Formatting

The raw AI output gets transformed into a clean HTML email with proper formatting, styling, and organization.

5. Reliable Delivery

Gmail's API ensures your personalized news digest arrives in your inbox exactly when scheduled.

What You'll Need

  1. OpenRouter account (for AI processing)
  2. Tavily API key (for news search)
  3. Google account with Gmail enabled
  4. n8n instance (cloud or self-hosted)

Pro tip: Start with free tiers of OpenRouter and Tavily to test the workflow before committing to paid plans.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure your OpenRouter and Tavily API credentials
  3. Set up Gmail OAuth2 authentication
  4. Customize the news topics in the query builder node
  5. Adjust the schedule to your preferred delivery time
  6. Test with your email address before going live
