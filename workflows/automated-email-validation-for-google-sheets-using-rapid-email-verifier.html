What This Workflow Does

This automation solves a critical problem for businesses that rely on email lists: wasted marketing spend and damaged sender reputation from sending to invalid addresses. Manual email verification is tedious, error-prone, and doesn't scale.

The workflow automatically validates email addresses stored in Google Sheets using the Rapid Email Verifier API. It checks new entries hourly, processes only unverified emails to avoid duplicates, updates your spreadsheet with validation results, and maintains clean, deliverable contact lists without manual intervention.

Perfect for marketing teams cleaning campaign lists, sales teams qualifying leads, or any business maintaining customer databases. It transforms raw, potentially dirty email data into verified, actionable contact information.

How It Works

Step 1: Monitor Google Sheets for New Entries

The Google Sheets trigger watches your spreadsheet automatically every hour. It detects new rows added with email addresses and prepares them for verification.

Step 2: Smart Filtering to Prevent Duplicates

A filter node checks which emails haven't been verified yet by looking at the "Email Verified" column. Only new or unverified emails proceed, saving API calls and preventing redundant processing.

Step 3: Batch Processing for Reliability

The Split in Batches node processes emails one by one, respecting API rate limits and ensuring each verification receives proper attention. This maintains system stability even with large lists.

Step 4: API Verification

Each email is sent to the Rapid Email Verifier service via HTTP Request. The API returns validation status (valid/invalid/unknown), risk score, and additional details about the email's deliverability.

Step 5: Update Original Spreadsheet

Using the Serial Number as a match key, the workflow updates your Google Sheet with verification results. You get a clean, organized spreadsheet showing which contacts are worth pursuing.

What You'll Need

  1. Google Sheets account with a spreadsheet containing columns: SrNo, Name, Email, Email Verified
  2. Google Sheets credentials configured in n8n for both trigger and update operations
  3. Internet connection for API access (no additional API key required for Rapid Email Verifier's free tier)
  4. n8n instance (cloud or self-hosted) to run the workflow
  5. Basic spreadsheet organization - consistent column structure ensures smooth automation

Quick Setup Guide

  1. Download the template using the button above and import into your n8n instance
  2. Configure Google Sheets connection by adding your credentials to both the trigger and update nodes
  3. Select your spreadsheet and specify the worksheet containing your email data
  4. Test the workflow manually first by adding a few test email rows to your sheet
  5. Activate the workflow and set it to run hourly for continuous validation
  6. Monitor results in your spreadsheet's "Email Verified" column
  7. Segment your list based on validation results for targeted follow-up actions
