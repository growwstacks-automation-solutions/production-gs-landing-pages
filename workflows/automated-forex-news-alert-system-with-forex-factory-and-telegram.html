What This Workflow Does

This automation solves a critical problem for forex traders: the delay between an economic data release and your ability to act on it. Manual monitoring of sites like Forex Factory is time-consuming and prone to missed opportunities. High-impact news like Non-Farm Payrolls or CPI releases can move currency pairs hundreds of pips in seconds.

The workflow automatically captures "Actual" data releases from Forex Factory as soon as they are published, processes the information, and sends a clear alert to your Telegram chat. It filters for only numerically measurable events (ignoring speeches), extracts the currency, impact level, and compares Actual vs. Forecast values to determine if the news is positive or negative for the currency.

By automating this process, you shift from reactive to proactive trading. You get a structured signal in your pocket within seconds of a release, giving you a tangible edge in executing trades based on fresh economic data.

How It Works

The workflow is built with logical steps to ensure reliability and accuracy, handling the inherent delays of web data publication.

Step 1: Trigger on New Events

The process starts by monitoring for new news release events. It acts as the trigger, ensuring the automation only runs when there's new data to process, saving resources.

Step 2: Filter & Extract Details

The workflow filters out non-numerical events (like central bank speeches) and extracts key details: the affected currency (e.g., USD, EUR), the impact level (High/Medium), the precise release time, and the direct link to the news page on Forex Factory.

Step 3: Wait & Scrape Actual Data

It then pauses for 10 seconds—a crucial step to allow the "Actual" value to be posted on the Forex Factory page after the release time. It then uses a web scraping tool to fetch the page content. If the Actual value isn't found, it waits another 5 seconds and retries, ensuring robustness.

Step 4: Process & Compare Values

The scraper extracts both the Forecast and Actual values. The workflow cleans the data, removing characters like '%', 'K', or 'B' and converts them into comparable numbers. It then performs the core logic: comparing Actual to Forecast.

Step 5: Generate & Send Alert

Based on the comparison, the news is routed. If the Actual is lower than Forecast (and lower is better for that indicator), it goes to a "positive for currency" branch. Otherwise, it's negative. A formatted message containing all key details is then sent instantly to your specified Telegram chat or channel.

Pro tip: Use the Telegram message as a trigger for further automation. Connect it to a trading platform API (like MetaTrader) to automatically open positions based on the news signal, creating a fully automated trading execution system.

What You'll Need

  1. A running n8n instance: This could be n8n.cloud, a self-hosted n8n, or the desktop app.
  2. Telegram Bot Token & Chat ID: Create a bot via @BotFather on Telegram and obtain your bot token and the numeric ID of the chat/channel where alerts should be sent.
  3. Web Scraping Service API Key (Airtop): The template uses Airtop to reliably scrape the Actual value from Forex Factory. You'll need a free or paid Airtop account and API key.
  4. Google Calendar Credentials (Optional for trigger): If using the Google Calendar trigger method as shown in some versions, you'll need OAuth credentials set up.
  5. Basic understanding of forex news: Knowing which indicators (e.g., CPI, Employment Change) are important for which currencies will help you customize and filter the alerts effectively.

Quick Setup Guide

You can have this system live in under 15 minutes by following these steps.

  1. Download & Import: Click the "Download Template" button above to get the JSON file. In your n8n instance, go to Workflows, click "Import from file," and select the downloaded JSON.
  2. Configure Telegram Node: Open the "Telegram" node in the workflow. Add your Bot Token (from @BotFather) and the Chat ID where you want alerts sent. Send a test message to verify the connection.
  3. Set Up Airtop Node: Find the "Airtop" node. You'll need to create an account on Airtop, generate an API key, and enter it in the node's credentials section in n8n.
  4. Adjust Trigger & Filters (Optional): Review the trigger node. You may want to adjust the schedule or the filtering logic—for example, to monitor only for "High" impact news or specific currencies like USD and EUR.
  5. Test & Activate: Turn the workflow on and trigger it manually or wait for the next scheduled run. Monitor the execution in n8n to see data flow through each step and ensure an alert arrives in your Telegram.

Pro tip: For production use, consider adding an error-handling branch. If the scraping fails multiple times, have the workflow send you a "Failure Alert" via a different channel (like email) so you know the system needs attention.
