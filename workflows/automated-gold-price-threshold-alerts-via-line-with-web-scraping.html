What This Workflow Does

Manually tracking gold prices is time-consuming and prone to missed opportunities. This automation solves that by continuously monitoring gold prices from trusted financial sources and sending you instant LINE notifications when prices cross your predefined thresholds. Whether you're an investor waiting for the right buying opportunity, a trader monitoring exit points, or a business tracking commodity costs, this workflow ensures you're always informed without constant manual checking.

The system operates on a scheduled basis—checking prices every 6 hours (or any interval you choose)—extracting current prices via web scraping, comparing them against your target levels, and triggering alerts only when action is needed. This transforms reactive price monitoring into proactive strategy execution, giving you more time for analysis while ensuring you never miss critical market movements.

How It Works

1. Scheduled Price Checking

The workflow activates automatically at your chosen intervals using n8n's schedule trigger. You can set it to check prices hourly, daily, or any frequency that matches your trading strategy.

2. Web Data Extraction

It connects to trusted gold price websites (like Gold Traders Association or financial portals) using HTTP requests, then extracts the precise current price using CSS selectors that target the specific data points you need.

3. Threshold Comparison

The extracted price is converted to a numeric format and compared against your predefined threshold values. You can set multiple thresholds for buying, selling, or alert levels.

4. Smart Notification Delivery

Only when prices exceed your thresholds does the system send a formatted LINE message containing the current price, threshold breached, and timestamp. This prevents notification fatigue while ensuring critical alerts get through.

What You'll Need

  1. n8n instance (cloud or self-hosted) to run the workflow
  2. LINE Developer Account for API access tokens
  3. Target website URL with current gold prices
  4. Basic CSS selector knowledge to identify price elements (or we can help)
  5. Threshold values for buying/selling/alert levels

Pro tip: Start with conservative thresholds (3-5% from current price) to avoid excessive alerts during normal market volatility. You can adjust based on your risk tolerance and market conditions.

Quick Setup Guide

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure the schedule trigger to your preferred checking frequency
  3. Update the HTTP Request node with your target gold price website URL
  4. Adjust CSS selectors in the HTML Extract node to match the price elements
  5. Set your threshold values in the If node (e.g., 52,300 THB for buying alerts)
  6. Connect LINE API with your access token and user ID in the Send Message node
  7. Test the workflow manually to verify data extraction and notification delivery
