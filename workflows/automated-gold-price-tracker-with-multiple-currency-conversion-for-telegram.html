Automated Gold Price Tracker with Telegram Alerts

Get real-time gold prices converted to your local currency with automated Telegram notifications every 15 minutes

What This Workflow Does

Manual gold price tracking is time-consuming and prone to missed opportunities. This automation solves that by continuously monitoring global gold prices, converting them to your preferred currency, and delivering formatted updates directly to your Telegram channel or chat.

The workflow fetches live gold prices from reliable financial APIs every 15 minutes, performs currency conversion calculations, creates organized tables showing prices for different gold purities (24k, 21k, 18k), and sends professional-looking alerts. This eliminates hours of manual monitoring while ensuring you never miss important price movements.

How It Works

Step 1: Scheduled Price Fetching

A schedule trigger runs every 15 minutes to initiate the workflow. The system makes HTTP requests to trusted gold price APIs to retrieve the most current market data, ensuring you get accurate, real-time information without manual intervention.

Step 2: Currency Conversion Processing

The workflow takes the base gold price (typically in USD) and converts it to your local currency using current exchange rates. It calculates prices for different gold purities based on standard conversion formulas, providing comprehensive pricing data in a single update.

Step 3: Data Formatting and Organization

Using n8n's data processing nodes, the raw price data is transformed into clean, readable formats. The system creates structured tables and formatted messages that present the information clearly, making it easy to understand at a glance.

Step 4: Telegram Notification Delivery

The formatted price update is automatically sent to your specified Telegram chat or channel. The message includes all relevant pricing information, currency conversions, and timestamps, delivered directly to where you and your team communicate.

Pro tip: Configure the workflow to send alerts only when prices move beyond your specified thresholds to avoid notification fatigue while still catching significant market movements.

What You'll Need

  1. A running n8n instance (cloud or self-hosted)
  2. Telegram account with a bot token and chat ID
  3. Access to a gold price API (free or paid options available)
  4. Currency conversion API key (optional, for advanced conversions)
  5. Basic understanding of your local currency code (like USD, EUR, GBP)

Quick Setup Guide

Follow these steps to implement this gold price tracker in your n8n environment:

  1. Download and import the JSON template into your n8n instance
  2. Configure the schedule trigger to your preferred frequency (default: 15 minutes)
  3. Set up your gold price API credentials in the HTTP Request node
  4. Enter your local currency code for automatic conversion
  5. Configure your Telegram bot token and destination chat ID
  6. Test the workflow with a manual execution
  7. Activate the workflow for automated operation

Pro tip: Start with a free gold price API for testing, then upgrade to a premium service for more reliable data and higher request limits as your needs grow.
