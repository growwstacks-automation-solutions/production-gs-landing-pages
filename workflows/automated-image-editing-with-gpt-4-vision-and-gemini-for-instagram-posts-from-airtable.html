What This Workflow Does

This automation solves the time-consuming process of manually editing and posting images to Instagram. Content creators and marketing teams often spend hours each week selecting, editing, and scheduling visual content. This workflow eliminates that manual work by leveraging AI to analyze, enhance, and post images automatically.

The system connects Airtable (where you store your raw images and content briefs) with GPT-4 Vision for image analysis, Google Gemini for AI-powered editing, and Instagram's API for automated posting. It handles the entire process from initial image selection to final publication with built-in error handling and status tracking.

How It Works

1. Trigger from Airtable

The workflow starts when a new image record is added to your Airtable base or when you flag a record for processing. The webhook passes the record ID to initiate the automation.

2. Image Analysis with GPT-4 Vision

GPT-4 Vision examines the image to identify key elements like composition, colors, and subjects. It generates an optimized editing brief based on your predefined parameters (style, lighting, etc.) and the image's actual content.

3. AI Editing with Gemini

Google Gemini receives the analysis and creates enhanced versions of the original image according to the specifications. You can configure multiple variations with different styles.

4. Upload Back to Airtable

The edited images are saved back to Airtable with metadata about the changes made, creating a searchable archive of all processed images.

5. Instagram Posting

The workflow uses Instagram's Graph API to publish the final image along with your prepared caption and hashtags. Status is updated in Airtable upon successful posting.

What You'll Need

  1. Airtable account with a configured base
  2. OpenAI API key for GPT-4 Vision access
  3. Google Gemini API credentials
  4. Meta Developer account with Instagram Graph API permissions
  5. Instagram Business account connected to a Facebook Page
  6. n8n instance (cloud or self-hosted)

Quick Setup Guide

  1. Import the JSON template into your n8n instance
  2. Configure your Airtable base with required fields (Status, Picture, Core Subject, etc.)
  3. Add your API keys in the credential nodes
  4. Update all Airtable base/table IDs in the workflow
  5. Set your Instagram Business Account ID
  6. Test with sample images before going live

Pro tip: Start with a small test Airtable base and Instagram test account to verify everything works before connecting to production systems.
