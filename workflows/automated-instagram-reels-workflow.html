What This Workflow Does

This workflow completely automates your Instagram Reels content pipeline, from content storage in Google Drive to scheduled posting on Instagram. It solves the time-consuming manual process of creating, captioning, and posting Reels by handling everything automatically.

The system supports two content creation paths: scheduled posting of pre-created content from a Google Drive folder, and automatic processing of newly uploaded videos. It generates AI-powered captions, logs all content in Airtable for tracking, and handles Instagram's strict API requirements for reliable posting.

How It Works

1. Content Collection

The workflow either picks a random video from your "Random video mover" Google Drive folder on a schedule, or detects newly uploaded videos in your "n8n reels automation" folder.

2. AI Caption Generation

Using Google Gemini AI, the system creates engaging captions with relevant hashtags based on the video filename and your brand voice. This ensures each post has optimized text without manual effort.

3. Airtable Logging

All posted content is recorded in Airtable with the video name, generated caption, posting date, and Instagram URL for easy reference and performance tracking.

4. Instagram Posting

The workflow handles Instagram's strict API requirements by temporarily storing the file locally and serving it through a webhook, ensuring successful uploads every time.
