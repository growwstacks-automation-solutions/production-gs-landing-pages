What This Workflow Does

This automation solves the common problem of maintaining duplicate task lists in both Google Sheets and Monday.com. The workflow continuously monitors a designated Google Sheet for new tasks (marked with "Added = No") and automatically creates corresponding items in your Monday.com board.

Once the task is successfully created in Monday.com, the workflow updates the Google Sheet to mark it as "Added = Yes", creating a reliable two-way sync that eliminates manual data entry errors and saves significant time for project teams.

How It Works

Step 1: Monitoring Google Sheets

The workflow starts by querying your specified Google Sheet, looking for rows where the "Added" column is set to "No". It reads all relevant task details including title, description, assignee, due date, and any custom fields you've configured.

Step 2: Creating Monday.com Items

For each new task found, the workflow uses Monday.com's API to create a corresponding item in your specified board and group. It maps all the fields from your Sheet to the appropriate Monday.com columns.

Step 3: Updating Status

After successful creation in Monday.com, the workflow updates the original Google Sheet row to mark it as "Added = Yes", preventing duplicate entries while maintaining a complete audit trail.
