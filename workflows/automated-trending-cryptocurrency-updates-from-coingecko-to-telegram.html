What This Workflow Does

Staying ahead in cryptocurrency requires constant monitoring of market sentiment and emerging trends. Manual tracking is time-consuming and inconsistent. This automation solves that by connecting CoinGecko's trending data directly to your Telegram community.

The workflow automatically fetches the top 7 most searched cryptocurrencies from CoinGecko's public API, formats the data into a clean, readable message, and posts it to your designated Telegram group or channel. It runs on a schedule you control, ensuring your community receives timely updates without any manual intervention.

Perfect for crypto influencers, community managers, trading groups, or anyone who wants to provide consistent value to their audience with zero daily effort. The template is completely free and requires no paid API keys or coding knowledge to set up.

How It Works

The automation follows a straightforward pipeline that transforms raw API data into actionable community updates.

1. Trigger & Schedule

The workflow can be triggered manually for testing or set to run automatically using a Cron node. The default schedule is twice daily (8:30 AM & 8:30 PM IST), but you can customize this to any frequency that suits your audience.

2. Fetch Trending Data

An HTTP Request node calls CoinGecko's free public API endpoint (/api/v3/search/trending). This returns real-time data on which coins are generating the most search interest across their platform—a reliable indicator of retail trader attention.

3. Process & Format

A Function node processes the JSON response, extracting key information: coin name, symbol, market cap rank, and current price (when available). It then structures this data into a clean, formatted message suitable for Telegram, often including emojis and clear formatting for readability.

4. Deliver to Telegram

The Telegram node sends the formatted message to your specified chat ID (group or channel). You only need to configure your Telegram Bot Token once—the same bot can serve multiple communities with different messaging schedules or formats.

Pro tip: Add a "Filter" node before the Telegram step to only send alerts for coins that meet specific criteria, like being in the top 100 by market cap or having a minimum price change percentage. This prevents spam about obscure, low-volume tokens.

Quick Setup Guide

You can have this automation running in under 10 minutes:

  1. Download & Import: Click the download button above to get the JSON file, then import it into your n8n instance.
  2. Configure Telegram: In the Telegram node, add your Bot Token (from @BotFather) and replace the placeholder Chat ID with your actual group/channel ID.
  3. Test Manually: Click "Execute Workflow" to test the connection and message format. Verify the output appears correctly in your Telegram chat.
  4. Set Schedule (Optional): Adjust the Cron trigger node to your preferred timing—consider time zones when your audience is most active.
  5. Activate & Forget: Toggle the workflow to "Active" and it will run automatically according to your schedule.

Pro tip: To find your Telegram Chat ID, add @RawDataBot to your group/channel and send "/start". It will reply with the chat ID you need for configuration.
