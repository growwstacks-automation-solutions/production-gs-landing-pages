What This Workflow Does

Manually monitoring Twitter/X for breaking news, competitor moves, and industry trends consumes valuable hours each week. This intelligent automation solves that by providing real-time social media intelligence delivered directly to your team's communication channels.

The workflow automatically scrapes targeted Twitter accounts and keyword searches, processes content through Google's Gemini 2.5 Pro AI to filter noise and extract insights, then formats and delivers curated intelligence digests to WhatsApp groups. It transforms endless social media scrolling into structured, actionable business intelligence.

Beyond basic monitoring, the system identifies emerging trends before they go mainstream, analyzes sentiment around specific topics, and provides context that helps your team make faster, better-informed decisions. The automation handles rate limiting, error recovery, and data formatting so you receive clean, relevant information without technical overhead.

How It Works

Step 1: Twitter Data Collection

The workflow connects to TwitterAPI.io to scrape tweets from specified accounts and search queries. It handles authentication, rate limiting, and pagination to gather comprehensive data without hitting API limits or getting blocked.

Step 2: AI-Powered Analysis & Filtering

Collected tweets are sent to Gemini 2.5 Pro via OpenRouter. The AI analyzes content relevance, summarizes lengthy threads, identifies key trends, filters out spam/irrelevant posts, and extracts actionable insights tailored to your business interests.

Step 3: Intelligence Compilation

The workflow structures the AI-processed information into digestible formats, grouping related tweets, adding context, and prioritizing information based on relevance and urgency before preparing it for delivery.

Step 4: WhatsApp Delivery

Using the Evolution API community node, the formatted intelligence digest is sent to specified WhatsApp groups. The system includes error handling for delivery failures and confirmation mechanisms to ensure your team receives critical updates.

Quick Setup Guide

  1. Install Evolution API Node: In your n8n Settings → Community Nodes, install "n8n-nodes-evolution-api" and restart n8n.
  2. Import the Workflow: Download the JSON template above and import it into your n8n instance.
  3. Configure Twitter Access: Add your TwitterAPI.io credentials to the HTTP Request nodes and specify target accounts/keywords.
  4. Set Up AI Processing: Add your OpenRouter API key to the Basic LLM Chain node and adjust the prompt for your specific intelligence needs.
  5. Configure WhatsApp: Set up Evolution API credentials and use the included "Fetch Groups" node to find your target WhatsApp group ID.
  6. Test & Schedule: Run a manual execution to verify everything works, then schedule the workflow for daily or hourly execution based on your needs.
