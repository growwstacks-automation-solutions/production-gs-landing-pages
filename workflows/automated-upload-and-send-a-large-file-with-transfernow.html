What This Workflow Does

This automation solves the common business challenge of securely transferring large files between teams, clients, or partners. Traditional email attachments often fail with files over 25MB, forcing users to manually upload to cloud storage or use clunky FTP solutions.

The workflow creates a branded web form where recipients can upload files up to 5GB directly to your TransferNow account. The system automatically processes the file, generates a secure download link, and sends it to your specified recipients - all without manual intervention.

How It Works

1. User submits file via web form

The workflow starts with a customizable web form that can be embedded on your website or shared via direct link. Users simply drag-and-drop their file or select it from their device.

2. File uploads to TransferNow

Once submitted, the file automatically uploads to your TransferNow account using API authentication. The workflow handles all the technical aspects of the transfer in the background.

3. Download link generated

TransferNow processes the file and generates a secure download link with customizable expiration settings. You can set the link to expire after a certain number of downloads or days.

4. Notification sent to recipients

The workflow automatically sends the download link via email or other notification channels you configure. You can customize the message with your branding and instructions.

Pro tip: Add a Google Sheets integration to log all file transfers with timestamps, sender info, and download activity for compliance tracking.
