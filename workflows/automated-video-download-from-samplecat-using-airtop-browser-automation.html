What This Workflow Does

Manually downloading videos or files from websites is a repetitive, time-consuming task that often blocks marketing teams, researchers, and content creators. This workflow automates the entire process of visiting Sample.cat, locating a specific video file, and downloading it—all without any human intervention.

By leveraging Airtop.ai's browser automation capabilities within an n8n workflow, you can schedule this process to run daily, weekly, or on-demand. It's perfect for businesses that need to regularly archive promotional videos, collect training datasets, or backup user-generated content from web platforms.

The automation handles the browser session, navigation, button clicking, waiting for file processing, and final download—reducing what could be a 15-minute manual task to a 30-second automated process that runs reliably in the background.

How It Works

This n8n workflow orchestrates a series of steps between manual initiation and browser automation to complete the file download process.

Step 1: Manual Trigger & Session Start

The workflow begins with a manual trigger node, allowing you to run it on-demand or schedule it. It then initiates a new browser session using Airtop.ai, creating a clean, isolated environment for the automation to run.

Step 2: Navigation & Interaction

The browser navigates to the specified Sample.cat URL containing the downloadable media. Using Airtop's interaction capabilities, it locates and clicks the exact download button for the target video (e.g., "SD 640x360 (Seawater, drone view video, 30 FPS)").

Step 3: Wait & File Verification

The workflow includes a built-in wait period (10 seconds in the template) to allow the website to process the download request and make the file available. It then queries the session to check for downloadable files and confirms their availability status.

Step 4: Download & Cleanup

Once the file is confirmed as available, the workflow downloads it using its unique ID. Finally, it terminates the browser session to free up resources, ensuring no lingering processes consume memory or bandwidth.

Pro tip: Extend this workflow by adding nodes to immediately transfer downloaded files to cloud storage like Google Drive or Dropbox, or trigger notifications in Slack/Teams when downloads complete.

Quick Setup Guide

Import and configure this workflow in under 10 minutes:

  1. Download the template using the button above and import it into your n8n instance.
  2. Get your Airtop API key from the Airtop.ai portal and add it to the "Session" node credentials.
  3. Update the target URL in the "Window" node to point to your specific Sample.cat video page.
  4. Adjust the button selector in the "Interaction" node if your target video has a different label.
  5. Test the workflow manually using the trigger node, then schedule it for regular execution.
