What This Workflow Does

This automation transforms Shopify order cancellations into new sales opportunities by instantly sending personalized WhatsApp messages with alternative product offers. When a customer cancels an order, the workflow checks if they have a WhatsApp number, then automatically delivers a tailored upsell message within minutes.

By combining Shopify's cancellation events with Rapiwa's WhatsApp integration and Google Sheets for tracking, you create a seamless recovery funnel that works 24/7. The system logs all interactions in a spreadsheet so you can measure conversion rates and optimize your offers over time.

How It Works

1. Shopify detects order cancellation

The workflow triggers whenever Shopify marks an order as canceled. It immediately pulls the customer's contact information and order details.

2. Verify WhatsApp availability

Rapiwa checks if the customer's phone number is registered with WhatsApp. This prevents wasted effort on unreachable contacts.

3. Send personalized upsell message

The system sends a WhatsApp message with alternatives to the canceled product, complementary items, or limited-time discounts. Messages include the customer's name and order details for personalization.

4. Log interactions in Google Sheets

Every outreach attempt and customer response gets recorded in a Google Sheet with timestamps. This creates an audit trail and provides data for performance analysis.

Pro tip: Test different message templates and offers to identify what converts best. Start with a 15% discount on similar products, then refine based on response data.
