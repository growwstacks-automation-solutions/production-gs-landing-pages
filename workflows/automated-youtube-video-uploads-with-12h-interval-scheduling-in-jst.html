What This Workflow Does

This automation solves the tedious problem of manually uploading and scheduling multiple YouTube videos. Content creators, educators, and businesses often produce content in batches but need to space out publications for optimal audience engagement. Manually calculating time zones, setting publish times, and managing playlists consumes hours each week.

The workflow automatically processes a folder of video files, extracts metadata from filenames, calculates precise 12-hour intervals in Japan Standard Time (UTC+9), and schedules each upload with proper playlist assignment. It handles the entire pipeline from file discovery to YouTube publication, eliminating human error in timezone conversions and ensuring consistent publishing cadences.

Beyond basic uploading, it implements robust error handling with retry logic, maintains video sequencing through day numbering, and provides customizable buffers between uploads. This transforms a multi-hour manual process into a fully automated system that runs reliably in the background.

How It Works

1. File Discovery & Sorting

The workflow scans a specified directory for all MP4 files, extracts day numbers from filenames (like "day01", "day02"), and sorts them sequentially. This ensures educational series or multi-part content publishes in the correct order regardless of file system organization.

2. Schedule Calculation

Using pure UTC conversions to avoid daylight saving errors, the system calculates the next rounded hour in JST plus a configurable buffer. Each video receives a publish time staggered by its order × 12 hours, creating perfect spacing for global audience reach.

3. Batch Processing Loop

The workflow iterates through each sorted video file, loading it from disk and preparing metadata. This loop structure allows processing hundreds of videos in a single run while maintaining individual error isolation—if one upload fails, others continue.

4. YouTube API Integration

Each video uploads to YouTube with private visibility initially, using the calculated UTC publish time. The workflow automatically sets titles, descriptions, and category metadata extracted from filename patterns or configuration files.

5. Playlist Management

After successful upload, each video automatically adds to a specified YouTube playlist. This keeps series organized and improves viewer retention through YouTube's "Up Next" recommendations.

Pro tip: Use descriptive filenames like "course_a1_day05_introduction.mp4" to automatically generate better YouTube titles and descriptions through simple text parsing in the workflow.

What You'll Need

  1. n8n instance (self-hosted or cloud) with access to file system where videos are stored
  2. YouTube API credentials with upload permissions and OAuth 2.0 configured
  3. Video files organized in a consistent directory structure with clear naming conventions
  4. Target YouTube playlist ID where videos should be added after upload
  5. Basic understanding of timezone conversions and YouTube's publishing options

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure YouTube credentials in the OAuth2 node with proper upload scopes
  3. Set your video directory path in the "Execute Command" node to point to your MP4 files
  4. Adjust timezone settings if needed (default is JST/UTC+9) and modify the 12-hour interval as required
  5. Test with one video first to verify upload, scheduling, and playlist addition work correctly
  6. Run the full batch once testing succeeds—the workflow handles everything automatically

Important: Always test with videos set to "private" first to avoid accidental premature publication. The workflow includes safety buffers, but verification is crucial.
