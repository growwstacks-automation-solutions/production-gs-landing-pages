What This Workflow Does

Manual expense tracking is a time-consuming, error-prone chore that steals focus from core business activities. This automation solves that by creating a seamless, chat-based logging system. Instead of fumbling with receipts or spreadsheet rows, you simply send a quick message to your personal Telegram bot—like "5 usd for coffee with my card"—and the entire process is handled automatically.

The workflow captures the message, uses AI to intelligently parse the amount, category, currency, and payment method, and then neatly logs a structured record into your Google Sheet. It’s designed for freelancers, consultants, small business owners, and teams who need a frictionless way to maintain accurate financial records without administrative overhead.

How It Works

The automation follows a clear, logical sequence to transform a casual message into structured financial data.

1. Trigger via Telegram

You send a message to your dedicated Telegram bot. The workflow is instantly triggered, capturing the raw text and your user ID. This step replaces manual data entry with a simple, mobile-friendly action you can do anywhere.

2. AI-Powered Parsing

The message is sent to an AI model (like OpenAI or Anthropic). The AI is prompted to extract key details: the numeric amount, the currency (USD, EUR, etc.), a relevant category (e.g., Meals, Travel, Software), a description, and the payment method. This natural language understanding is the core intelligence of the system.

3. Data Enrichment & Validation

The workflow adds the current date and timestamp. It can optionally call a currency exchange API to convert foreign amounts to your home currency, calculating both the original and converted values. Basic validation ensures the extracted data is sensible before proceeding.

4. Logging to Google Sheets

The finalized, structured data—including all extracted fields, conversion rates, and user ID—is appended as a new row in your predefined Google Sheet. Your spreadsheet becomes a real-time, queryable database of all expenses.

Pro tip: Structure your Google Sheet with clear column headers (date, amount, category, description, currency, etc.) before importing the workflow. This ensures clean, consistent data from day one.

What You'll Need

  1. A Telegram account and a Telegram Bot token (created via @BotFather).
  2. A Google account with a Google Sheet prepared with the correct column headers.
  3. Access to an AI model API (OpenAI, Anthropic, Google Gemini, etc.) with an API key.
  4. An n8n instance (cloud or self-hosted) to import and run the workflow.
  5. (Optional) A currency exchange API key for automatic currency conversion.

Quick Setup Guide

You can have this automation running in under 15 minutes by following these steps.

  1. Prepare Your Google Sheet: Create a new Sheet. Name the first row columns: date, amount, category, description, user_id, payment_method, currency, exchange_rate, amount_converted. Copy the Sheet ID from the URL.
  2. Create Your Telegram Bot: Message @BotFather on Telegram, use the /newbot command, and securely save the API token it provides. Message @userinfobot to get your personal Chat ID.
  3. Import & Configure the Workflow: Download the template and import it into your n8n workspace. Add credentials for Google Sheets, Telegram, and your chosen AI model in n8n's settings.
  4. Connect the Dots: Paste your Chat ID into the 'Telegram Trigger' node. Paste your Google Sheet ID into the 'Append to Sheet' node. Review the AI prompt to ensure it matches your desired categories.
  5. Activate & Test: Turn the workflow on. Send a test message like "12.50 eur for lunch" to your bot. Check your Google Sheet for the new, perfectly parsed row.
