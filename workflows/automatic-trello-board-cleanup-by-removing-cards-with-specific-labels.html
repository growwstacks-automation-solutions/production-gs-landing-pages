What This Workflow Does

Trello boards can quickly become cluttered with completed tasks, outdated ideas, and cards that no longer serve a purpose. Manually reviewing and deleting these cards is time-consuming and often gets postponed, leading to visual noise that distracts teams from current priorities.

This automation solves that problem by systematically scanning your Trello board for any cards labeled "Mark for Deletion" and removing them automatically. It transforms a manual weekly chore into a hands-off process that runs on schedule, ensuring your boards remain focused and organized without any ongoing effort from your team.

How It Works

The workflow follows a logical sequence to identify and remove outdated cards while maintaining complete control over what gets deleted.

1. Board Identification

The workflow starts by connecting to your Trello account and identifying the specific board you want to clean. You can specify the board by URL or ID, making it easy to target exactly the right workspace.

2. Card Retrieval

It retrieves all cards from the board, including their labels, descriptions, due dates, and other metadata. This comprehensive data collection ensures the automation has full context before making any decisions.

3. Label Filtering

The system scans each card's labels looking for the specific "Mark for Deletion" indicator. You can customize this label name to match your team's terminology, such as "Archive," "Done," or "Remove."

4. Safe Deletion

Cards matching the deletion criteria are permanently removed from the board. The workflow can be configured to send notifications about what was deleted or move cards to an archive instead of permanent deletion for added safety.

Pro tip: Before enabling full automation, run the workflow in test mode first. This will show you which cards would be deleted without actually removing them, giving you confidence in the system.

Quick Setup Guide

Follow these steps to implement this automation in under 15 minutes:

1. Download the template using the button above and save the JSON file to your computer.

2. In n8n, create new credentials for Trello using your API key and token from trello.com/app-key.

3. Import the workflow via the n8n workspace import function and select the downloaded JSON file.

4. Configure the "Get Board" node by entering your Trello board URL (e.g., https://trello.com/b/yourboardname).

5. Test the workflow by manually triggering it and checking which cards it identifies for deletion.

6. Schedule it to run weekly using n8n's schedule trigger for hands-off automation.

Pro tip: Create a board rule in Trello that automatically adds your deletion label to cards moved to a "Completed" list. This creates a seamless flow where finishing a task eventually triggers its automatic cleanup.
