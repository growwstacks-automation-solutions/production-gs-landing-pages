What This Workflow Does

This automation solves a critical problem for e-commerce businesses: shipping packages to incorrect addresses. When customers mistype their address, forget apartment numbers, or provide incomplete information, it leads to failed deliveries, increased return rates, and frustrated customers. Manual address checking is time-consuming and error-prone, especially during peak seasons.

The workflow automatically validates and corrects shipping addresses in Billbee orders as soon as they're imported. It connects Billbee with professional address validation services to verify address accuracy, fix common errors, and ensure packages are shipped to deliverable locations. This happens in real-time, before orders reach your warehouse team, preventing costly shipping mistakes.

By automating this process, businesses reduce failed delivery attempts by up to 80%, cut return shipping costs, and improve customer satisfaction. Warehouse teams save 2-5 minutes per order that would otherwise be spent manually verifying addresses or contacting customers for clarification.

How It Works

1. Order Trigger

The workflow activates when a new order is imported into Billbee. A webhook captures the order details, including the shipping address and order ID, and initiates the validation process immediately.

2. Address Extraction

The system extracts the shipping address components—street, city, postal code, country—from the Billbee order data. It prepares this information for validation while maintaining the original order context.

3. Validation API Call

The address is sent to a professional validation service (like Endereco, SmartyStreets, or Loqate). These services compare the address against official postal databases, correct spelling errors, add missing components, and verify deliverability.

4. Conditional Processing

Based on the validation results: valid addresses are automatically updated in Billbee; problematic addresses are flagged for manual review. The system can also exclude certain delivery types like pickup stations if configured.

5. Order Update & Tracking

Validated addresses update the original Billbee order. The workflow adds tags like "Address Corrected" or "Validation Error" to track which orders were processed automatically versus those needing attention.
