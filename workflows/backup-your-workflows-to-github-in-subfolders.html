Automated GitHub Workflow Backup

Securely backup your automation workflows to GitHub with organized subfolders

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the critical problem of losing valuable workflow configurations by providing automated backups to GitHub. Unlike manual exports, this solution:

  • Automatically exports all workflows on a schedule
  • Organizes them into subfolders based on tags
  • Maintains version history through GitHub commits
  • Provides disaster recovery capabilities

The workflow checks GitHub for existing files, updates changed workflows, creates new ones, and ignores identical versions - all while maintaining proper folder structure.

How It Works

1. Workflow Export

The system uses n8n's API node to export all workflow configurations in JSON format.

2. GitHub Integration

Connects to your GitHub repository using OAuth credentials to securely push changes.

3. Folder Organization

A Set node analyzes workflow tags and organizes them into corresponding subfolders.

4. Version Control

Each update creates a new commit in GitHub, maintaining complete version history.
