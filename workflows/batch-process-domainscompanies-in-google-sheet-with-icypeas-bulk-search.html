What This Workflow Does

Manual domain and company research is time-consuming and error-prone. Sales teams spend hours verifying company information, marketing teams struggle with lead list enrichment, and researchers face data inconsistency across sources. This creates bottlenecks in outreach campaigns, delays market analysis, and results in missed opportunities due to outdated or incomplete information.

This automation solves these challenges by connecting Google Sheets directly to the Icypeas API. It automatically processes lists of domains or company names, verifies their details, enriches them with additional intelligence, and returns structured data back to your spreadsheet. What used to take hours of manual research now happens automatically in minutes, ensuring your team works with accurate, up-to-date company information.

How It Works

Step 1: Trigger from Google Sheets

The workflow starts by reading your prepared Google Sheet containing domains or company names. It processes each row individually, ensuring no data is missed during batch operations.

Step 2: Connect to Icypeas API

Each domain/company name is sent to the Icypeas API using your account credentials. The system authenticates securely and prepares the data request for verification and enrichment.

Step 3: Process Domain Intelligence

Icypeas analyzes each entry, verifying company details, checking domain registration, and enriching the data with additional intelligence like company size, industry, and contact information.

Step 4: Return Enriched Data

The verified and enriched data is structured and written back to your Google Sheet, creating a comprehensive company database ready for sales, marketing, or research use.

What You'll Need

  1. A Google Sheets document formatted with domain names or company names in a specific column
  2. An active Icypeas account with API access (API Key, API Secret, and User ID)
  3. n8n instance (cloud or self-hosted) to run the workflow
  4. Basic understanding of spreadsheet organization and data structure

Pro tip: Prepare your Google Sheet with clear column headers before running the automation. Include columns for the raw domain/company input and leave adjacent columns empty for the enriched data to be populated automatically.

Quick Setup Guide

  1. Download the template JSON file using the button above
  2. Import it into your n8n instance (Settings → Import Workflow)
  3. Configure the Google Sheets node with your spreadsheet ID and range
  4. Add your Icypeas API credentials to the HTTP Request node
  5. Test with a small sample of domains before processing your entire list
  6. Schedule the workflow to run automatically or trigger it manually as needed
