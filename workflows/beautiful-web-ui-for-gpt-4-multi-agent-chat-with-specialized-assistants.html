What This Workflow Does

This template solves a common problem in AI automation: how to give non-technical users access to powerful AI systems. Most businesses deploy GPT-4 or similar models through APIs or complex dashboards that require training. This workflow provides a beautiful, ready-to-use web interface that anyone can use immediately.

It creates a professional chat portal where users can type questions and select which specialized AI agent should respond—General Assistant, Database Query Agent, Web Search Agent, or RAG (Retrieval-Augmented Generation) Agent. Each agent is configured for specific tasks, making the AI system more accurate and useful than a single general-purpose chatbot.

The interface handles the entire user experience: real-time responses with loading animations, color-coded agent identification, dark/light mode preferences, and error handling. All without requiring any frontend development from your team.

How It Works

The system operates through two connected n8n workflows that separate the user interface from AI processing.

1. Interface Delivery Workflow

When a user visits your designated URL, a GET webhook triggers a Code node containing the complete HTML/CSS/JavaScript interface. This is served directly to their browser via a Respond to Webhook node. The interface loads instantly with all interactive elements ready.

2. AI Processing Workflow

When users send messages, the interface POSTs data to a separate webhook endpoint. A Switch node routes the request based on the selected agent type (general, database, web, or rag). Each route connects to its corresponding AI Agent node configured with appropriate tools, memory, and system prompts.

3. Response Handling

After the AI processes the query, a Code node formats the response consistently. The Respond to Webhook node sends it back to the interface, which displays it with the appropriate agent badge and smooth animation. The entire process maintains conversation history and context for follow-up questions.

Quick Setup Guide

Follow these steps to deploy your AI chat interface in under 30 minutes:

  1. Download and import the JSON template into your n8n instance.
  2. Configure the Interface Workflow: Update the webhook path to something memorable like /ai-chat.
  3. Configure the Processing Workflow: Add your AI API keys to the Agent nodes and adjust system prompts for each agent type.
  4. Test the connection: Visit your n8n URL followed by the webhook path (e.g., https://your-n8n.com/webhook/ai-chat).
  5. Customize the interface: Modify the CSS variables in the Code node to match your brand colors.
  6. Add security: Implement authentication if needed, and restrict webhook access as appropriate for your use case.
