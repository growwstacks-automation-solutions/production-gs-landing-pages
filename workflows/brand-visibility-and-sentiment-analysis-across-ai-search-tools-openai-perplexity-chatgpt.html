What This Workflow Does

In today's AI-first world, how your brand appears in ChatGPT, Perplexity, and other AI search tools directly impacts customer perception and business opportunities. This automation solves the manual, time-consuming process of checking multiple AI platforms by automatically sending your brand-related prompts to OpenAI, Perplexity, and ChatGPT, then analyzing the responses for sentiment and visibility.

The workflow systematically audits your brand's presence across different AI systems, categorizes responses by emotional tone (positive, neutral, negative), identifies whether your brand is mentioned, and creates a brand hierarchy showing how AI ranks you against competitors. All results are automatically logged to Google Sheets for easy analysis and reporting.

This transforms what would be hours of manual research into a fully automated process that runs on schedule, providing consistent, comparable data about how AI systems perceive and present your brand to users.

How It Works

1. Prompt Collection & Distribution

The workflow begins by reading brand-related prompts from a Google Sheet or manual input. These prompts (like "What are the best [industry] companies?" or "Tell me about [your brand]") are then sent simultaneously to multiple AI platforms through their respective APIs.

2. Multi-Platform AI Querying

Each prompt is sent to OpenAI (for baseline LLM knowledge), Perplexity (for answers with citations), and optionally to ChatGPT via web interface automation. This multi-platform approach ensures you get a comprehensive view of how different AI systems respond to the same questions about your brand.

3. Response Analysis & Sentiment Scoring

AI-powered analysis categorizes each response by basic polarity (positive/neutral/negative), emotion category (joy, sadness, anger, fear, disgust, surprise), and extracts brand hierarchy information showing how your brand is positioned relative to competitors in AI-generated answers.

4. Consolidated Reporting

All processed data—including the original prompt, AI source, response text, sentiment scores, brand mention flag, and brand hierarchy—is structured and appended to your Google Sheets output. A simplified branch also shows how to push individual responses to separate sheets for focused analysis.

What You'll Need

  1. OpenAI API access (GPT-4 or later models as configured in the template)
  2. Perplexity API access with appropriate credits for querying
  3. Google Sheets account with OAuth connected to n8n
  4. n8n instance (cloud or self-hosted) to run the workflow
  5. Optional: APIfy account/token if using the ChatGPT web actor path

Pro tip: Start with a small set of 5-10 key brand prompts to test the workflow and understand costs before scaling to comprehensive monitoring. Different AI APIs have varying pricing structures.

Quick Setup Guide

  1. Import the template into your n8n instance using the downloaded JSON file.
  2. Connect your Google Sheets credentials via OAuth and create two tabs: "Input" with a "Prompt" column and "Output" with columns for all response data.
  3. Add your OpenAI and Perplexity API credentials to the respective nodes in the workflow.
  4. Configure the optional ChatGPT path if using APIfy (note terms of service considerations).
  5. Test with the manual trigger using a few sample prompts to verify all connections work correctly.
  6. Schedule the workflow to run daily or weekly for ongoing brand monitoring.
