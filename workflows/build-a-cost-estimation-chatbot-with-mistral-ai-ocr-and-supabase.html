What This Workflow Does

This AI-powered cost estimation chatbot transforms how businesses handle pricing and quoting. Instead of manual calculations or static forms, it provides a conversational interface where customers can get accurate estimates instantly—whether they're asking questions, uploading invoices, or configuring complex products.

The system combines Mistral AI's reasoning capabilities with OCR document analysis and Supabase database queries. It can read information from uploaded documents (like invoices or RFQs), ask clarifying questions conversationally, perform complex calculations with unit conversions, and deliver transparent, step-by-step cost breakdowns. This eliminates manual data entry, reduces calculation errors, and provides 24/7 quoting capability without human intervention.

For businesses dealing with fabrics, tiles, manufacturing components, or any configurable products, this automation handles the entire quoting process from initial inquiry to final price delivery, with optional escalation to human agents via Gmail when needed.

How It Works

Step 1: Conversation Initiation

The workflow triggers when a user sends a message through a chat interface (n8n Chat UI) or webhook from your website. The Mistral AI model engages in natural conversation to understand what the user needs—whether it's a price quote, product configuration, or document analysis.

Step 2: Document Processing (If Applicable)

If the user uploads a document (invoice, receipt, RFQ), the OCR Extract node automatically reads and converts it into structured data. Key information like measurements, quantities, product codes, and specifications are extracted without manual data entry.

Step 3: Data Retrieval & Validation

The system queries Supabase to retrieve historical data, customer information, or previously processed documents. It compares extracted data with existing records, identifies missing information, and asks conversational questions to complete the dataset needed for accurate calculations.

Step 4: Intelligent Calculation

Using the Calculator node with predefined MATERIAL_COSTS and PROCESSING_COSTS formulas, the system performs all necessary computations. It handles unit conversions (GSM to sqm, area calculations, weight conversions), applies pricing rules, and generates detailed cost breakdowns.

Step 5: Conversational Delivery & Escalation

The AI explains the calculation process step-by-step to the user, building trust through transparency. If complex support is needed or an order confirmation is requested, the system can automatically escalate via Gmail to human team members with all relevant context.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. Mistral AI API key for the conversational intelligence layer
  3. Supabase account or similar database for storing and retrieving historical data
  4. Gmail/Google Workspace account for escalation and notification emails (optional)
  5. Your pricing rules and cost formulas defined in a clear, structured format
  6. Sample documents (invoices, RFQs) to train and test the OCR extraction accuracy

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance
  2. Configure credentials for Mistral AI, Supabase, and Gmail in n8n's credential management
  3. Customize the Calculator node with your specific MATERIAL_COSTS and PROCESSING_COSTS formulas
  4. Set up your Supabase tables to match the expected data structure for customer and product information
  5. Test with sample conversations and documents to ensure accurate extraction and calculation
  6. Deploy the webhook or chat interface on your website or customer portal
  7. Monitor initial interactions and refine the AI prompts based on real user questions

Pro tip: Start with a limited set of products or services to refine the system before expanding to your full catalog. Document edge cases and unusual requests to continuously improve the AI's understanding and accuracy.
