What This Workflow Does

This n8n automation template creates a powerful AI-powered chat agent for your Shopify store. It solves the problem of slow, limited customer support by providing an instantly available assistant that can answer product questions, guide purchases, and even initiate checkout—all within a chat interface.

By integrating Shopify's Model Context Protocol (MCP) with Thesys C1, the workflow moves beyond simple text responses. The AI agent can fetch live catalog data, inventory status, and product details, then present interactive UI elements like buttons, forms, and product cards directly in the chat. This transforms passive browsing into an engaging, guided shopping experience.

The result is a seamless, automated customer service layer that operates 24/7, reduces support ticket volume, and directly contributes to higher conversion rates by assisting customers at the moment of intent.

How It Works

The workflow is built as a responsive, event-driven system that connects customer inquiries with your store's live data and intelligent UI generation.

1. Customer Interaction Trigger

A visitor sends a message through the embedded web chat widget on your Shopify store. This triggers the n8n workflow via a webhook, capturing the user's query and context.

2. AI Agent Processing

The workflow routes the message to an AI Agent node (like OpenAI). The agent analyzes the intent—whether it's a product question, purchase request, or support issue.

3. Live Data Fetch via Shopify MCP

For product-related queries, the agent calls the Shopify MCP server. This protocol allows secure, real-time access to your store's product catalog, inventory, and customer data, ensuring the AI has accurate, up-to-date information to base its response on.

4. Interactive UI Generation with Thesys C1

Instead of returning plain text, the AI's response is sent to Thesys C1. This middleware interprets the response and generates a UI schema—specifying interactive elements like "Add to Cart" buttons, product image carousels, or checkout forms.

5. Response Delivery & Rendering

The enriched response (data + UI schema) is sent back through the webhook to the chat widget. The frontend (using Thesys's GenUI SDK) renders the interactive components, allowing the customer to take immediate action without leaving the conversation.

Pro tip: Use this workflow to handle common pre-sale questions like "Do you have this in size XL?" or "What's the return policy?". It frees your team to handle complex, high-value customer issues.

Quick Setup Guide

Follow these steps to import and configure the template in your n8n environment.

  1. Download the template using the button above and save the JSON file.
  2. In your n8n dashboard, go to Workflows → Import from file and select the downloaded JSON.
  3. Configure the Webhook node: Copy the unique webhook URL provided by n8n. You'll use this to connect your chat widget frontend.
  4. Set up your credentials: In the workflow, add your Thesys API Key and Shopify MCP Server URL to the respective nodes.
  5. Connect your AI model: Configure the AI Agent node with your preferred LLM provider (OpenAI, Anthropic, etc.).
  6. Embed the chat widget: Use the provided demo frontend code or integrate the webhook with your preferred chat interface (like a custom React component).
  7. Activate the workflow and test it by sending a message like "What products do you have?" through your chat interface.

Pro tip: Start by testing with a limited product catalog or a development store to ensure the MCP connection is working correctly before going live with customers.
