What This Workflow Does

This automation solves a critical problem for businesses using AI: generic chatbots that lack specific knowledge about your products, services, or internal processes. Without access to your data, AI models provide generic, often inaccurate answers that frustrate users and increase support costs.

The workflow builds an embeddable AI chatbot powered by an InfraNodus knowledge graph—a visual representation of your data that shows relationships between concepts. It enables the AI to retrieve context-aware information from your knowledge base before generating a response, resulting in precise, relevant answers that reflect your actual business knowledge.

Unlike complex RAG (Retrieval-Augmented Generation) setups requiring vector databases and embeddings, this solution simplifies the process. You import data from multiple sources (PDFs, websites, CSVs, etc.) into InfraNodus, then connect it to n8n to create a fully functional, embeddable chatbot that understands your specific domain.

InfraNodus knowledge graph visualization showing how topics connect and cluster within your data

How It Works

Step 1: Build Your Knowledge Base

Create a new graph in InfraNodus and import your data. You can add content manually or automatically from text files, PDFs, websites, CSV spreadsheets, Google search results, YouTube transcripts, RSS feeds, or even AI-generated content. The system structures this information into a knowledge graph that identifies main concepts and their relationships.

Step 2: Configure the n8n Automation

The downloaded template contains an HTTP Request node configured to communicate with InfraNodus's AI response API. You add your InfraNodus API key and specify the name of your knowledge graph. The workflow also includes an n8n Form Trigger that creates an embeddable web interface for users to ask questions.

Step 3: Query and Response Generation

When a user submits a question through the web form, n8n sends it to InfraNodus. The system queries your knowledge graph to retrieve relevant concepts and contextual information, then uses this enhanced context to generate a precise AI response. The answer is returned through the same web interface or can be routed to other applications.

Step 4: Continuous Improvement

InfraNodus provides visual analytics showing gaps in your knowledge graph. You can regularly import new content to fill these gaps, monitor unanswered questions, and iteratively improve your chatbot's knowledge base over time without retraining complex AI models.

What You'll Need

  1. n8n instance (cloud or self-hosted) to run the automation workflow.
  2. InfraNodus account with API access (free 14-day trial available).
  3. Knowledge materials in any format: text documents, PDF manuals, website URLs, CSV data, or existing FAQ content.
  4. Web hosting or internal platform where you'll embed the chatbot interface (optional but recommended).

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance.
  2. Sign up for InfraNodus and create your first knowledge graph by importing data from your chosen sources.
  3. Obtain your InfraNodus API key from the account settings and add it to the HTTP Request node in the workflow.
  4. Enter your graph name in the same HTTP Request node configuration.
  5. Activate the workflow and test it using the generated web form URL.
  6. Embed the form on your website or internal tools using the provided embed code or URL.
