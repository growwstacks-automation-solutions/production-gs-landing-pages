What This Workflow Does

This automation solves the problem of manual YouTube research, which can consume hours of valuable time. Instead of watching lengthy videos to extract information, this MCP (Model Context Protocol) server enables AI agents to search for relevant YouTube content and download transcripts automatically.

The workflow connects to YouTube through a professional scraping service, searches based on user queries, retrieves video metadata, and extracts complete transcripts. This transforms unstructured video content into structured text that AI systems can analyze, summarize, and integrate into research workflows.

Businesses use this to monitor competitor channels, track industry trends, gather customer feedback from review videos, and build knowledge bases from educational content—all without manual video consumption.

How It Works

1. MCP Server Trigger

The workflow starts with an MCP server trigger that listens for requests from compatible clients like Claude Desktop. When a research query arrives, it routes the request to the appropriate tool.

2. YouTube Search Tool

This component uses Apify's scraping service to search YouTube based on the query. It returns up to 10 relevant videos with titles, descriptions, URLs, and metadata, providing a curated list for further analysis.

3. Transcript Extraction Tool

For selected video URLs, this tool downloads complete subtitles and transcripts. It handles multiple video formats, extracts timestamps, and structures the text for easy processing by AI agents.

4. Usage Reporting

The workflow includes monitoring that tracks API consumption and usage limits, helping you manage costs and prevent service interruptions in production environments.

Quick Setup Guide

  1. Download the template JSON file using the button above
  2. Import it into your n8n instance (Settings → Workflows → Import from file)
  3. Configure the Apify nodes with your API credentials from apify.com
  4. Set up the MCP server trigger following n8n's documentation for your client
  5. Test with sample queries like "what is MCP?" or "latest AI developments"
  6. Connect your AI agent and begin researching YouTube content automatically
