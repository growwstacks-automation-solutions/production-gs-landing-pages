What This Workflow Does

This automation template transforms WhatsApp into an intelligent sales channel by creating an AI-powered chatbot that can answer customer questions about your products. The chatbot leverages a product catalog stored in a vector database to provide accurate, context-aware responses to customer inquiries.

Unlike basic chatbots that follow rigid scripts, this solution understands natural language questions, retrieves relevant product information, and generates helpful responses. It handles the entire conversation flow from initial greeting to final response, while maintaining context throughout the interaction.

How It Works

1. Product Catalog Setup

The workflow begins by importing your product catalog and processing it into a vector store. This creates a searchable knowledge base that the AI can query to find relevant product information.

2. Message Reception

When a customer sends a message via WhatsApp, the workflow captures it and checks if it's a text message (filtering out media and other non-text content).

3. AI Processing

The customer's message is sent to an AI agent which analyzes the intent and queries the product catalog vector store to find the most relevant information.

4. Response Generation

The AI composes a natural-sounding response based on the retrieved product information and sends it back to the customer via WhatsApp.

What You'll Need

  1. A WhatsApp Business Account
  2. OpenAI API key for the AI processing
  3. Your product catalog in a compatible format (CSV, JSON, or accessible via API)
  4. Basic understanding of n8n to configure the workflow

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure your WhatsApp credentials in the WhatsApp node
  3. Add your OpenAI API key to the AI agent node
  4. Connect your product catalog (either upload a file or connect to your product API)
  5. Test the workflow with sample messages before activating
