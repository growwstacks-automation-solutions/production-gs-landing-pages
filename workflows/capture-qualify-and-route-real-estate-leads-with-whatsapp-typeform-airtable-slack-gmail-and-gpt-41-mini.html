What This Workflow Does

Real estate teams are drowning in leads but struggling with manual follow-up. This n8n-powered automation solves that by creating a seamless, intelligent pipeline that captures inquiries from WhatsApp and website forms, processes them with AI, eliminates duplicates, assigns them fairly to agents, logs everything to a central CRM, and provides weekly performance reports.

The system replaces chaotic spreadsheets, missed messages, and uneven workload distribution. It ensures every lead receives immediate, professional attention while giving managers complete visibility into their pipeline. Designed for scalability, it can easily incorporate additional channels like Facebook Lead Ads without disrupting the core logic.

How It Works

1. Multi-Channel Lead Intake

The workflow triggers when a new lead arrives via a WhatsApp message or a submitted website form (like Typeform). It captures all initial details provided by the prospect.

2. AI-Powered Conversation & Qualification

An AI agent engages with incomplete inquiries to extract missing critical information—budget, location, property type, and timeline. It validates intent and structures the data into a consistent format for processing.

3. Smart Deduplication

Before proceeding, the system checks for duplicates both within the current execution and against existing records in your CRM (Airtable). It uses phone numbers and emails to prevent agents from contacting the same person twice, saving time and improving client experience.

4. Property-Based Routing & Fair Assignment

Leads are categorized by property interest (e.g., single-family, condo). A round-robin logic then assigns the lead to the next available agent in the rotation for that category, ensuring perfectly even distribution and eliminating manual bias.

5. Centralized CRM Logging & Instant Notifications

All lead data, assignment details, and timestamps are automatically written to your Airtable base. Simultaneously, notifications are sent via Slack and/or email to alert the assigned agent, enabling them to respond within minutes.

6. Automated Weekly Reporting

A scheduled sub-workflow runs weekly to compile key metrics: total leads received, duplicates filtered, assignments per agent, and source performance. This report provides actionable insights without manual data gathering.

What You'll Need

  1. An n8n instance (Cloud or self-hosted).
  2. API credentials for WhatsApp Business Cloud.
  3. An OpenAI API key for GPT-4.1-mini lead qualification.
  4. An Airtable base with two tables (Leads and Duplicates) set up with the specified column structure.
  5. A Slack workspace (for agent notifications).
  6. A Gmail account or SMTP service (for email alerts).

Pro tip: Before connecting live channels, test the workflow using mock data in the "Schedule Trigger" or "Manual Trigger" node to verify your Airtable structure and notification logic work correctly.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n workspace.
  2. Configure Credentials: In the n8n credentials panel, set up connections for WhatsApp, OpenAI, Airtable, Gmail, and Slack using your API keys and access tokens.
  3. Update Airtable Details: In the Airtable nodes, replace the placeholder Base ID and Table Names with your actual Airtable base and table identifiers.
  4. Customize Agent Pool: In the "Set" node for agent assignment, update the list of agent names/emails to match your team.
  5. Test & Activate: Use the "Manual Trigger" to run a test with sample data. Check that a record appears in Airtable and notifications are sent. Then, activate the main triggers (WhatsApp/Typeform) and publish the workflow.
