Check phishing URL reputation with VirusTotal and log to Google Sheets

Automatically analyze URLs for phishing attempts and malicious activity. This n8n workflow checks URL reputation using VirusTotal's API and logs results to Google Sheets for security auditing and compliance reporting.

What This Workflow Does

This automation solution addresses the growing threat of phishing attacks by automatically checking URLs against VirusTotal's comprehensive threat database. It eliminates manual security reviews while providing auditable logs of all scanned links in Google Sheets. The workflow helps organizations comply with security policies requiring documented review of external links while reducing the workload on IT teams.

By automating URL reputation checks, businesses can scan 100% of incoming links rather than relying on spot checks. The system flags suspicious URLs with detection rates from multiple antivirus engines, community reputation scores, and categorization tags. This provides security teams with actionable intelligence to block phishing attempts before they reach employees or customers.

How It Works

1. URL Input

The workflow accepts URLs from various sources including email attachments, form submissions, or security alerts. It can process single URLs or batches from spreadsheets.

2. VirusTotal API Check

Each URL is sent to VirusTotal's API which checks against 70+ security vendors. The system returns detailed reputation data including detection rates and threat categorization.

3. Risk Assessment

The workflow evaluates the VirusTotal results against your configured risk thresholds. URLs with high detection rates or phishing tags are flagged for immediate review.

4. Google Sheets Logging

All scan results are recorded in Google Sheets with timestamps, including the URL, detection details, and risk classification. This creates an auditable security log.
