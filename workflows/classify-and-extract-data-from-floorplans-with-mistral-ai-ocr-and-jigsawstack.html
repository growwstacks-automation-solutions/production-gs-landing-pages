What This Workflow Does

This automation solves a critical bottleneck for real estate, construction, and property management businesses: manually reviewing uploaded floorplan files to determine if they're valid, then painstakingly extracting room measurements. The workflow automatically classifies uploaded files (PDFs or images) as legitimate floorplans or irrelevant documents, filtering out junk uploads instantly.

For valid floorplans, it performs intelligent optical character recognition (OCR) using Mistral AI to extract room dimensions, square footage, and other structural data. The output is clean, structured JSON and user-friendly HTML tables—transforming visual information into actionable data without manual transcription.

How It Works

Step 1: File Upload & Initial Routing

The workflow triggers when a user uploads a file via webhook or form submission. It immediately routes PDFs and images (JPG, PNG) to specialized processing paths while rejecting unsupported formats.

Step 2: Two-Layer Quality Classification

First, heuristic checks analyze file characteristics. Then, AI classification determines if the content represents an actual floorplan versus unrelated documents. A confidence score decides whether to proceed or flag for manual review.

Step 3: Intelligent Data Extraction

Valid floorplans are processed through Mistral AI's OCR engine, which identifies text, measurements, and spatial relationships. The AI is constrained to output data in a strict JSON schema, ensuring consistency.

Step 4: Data Enrichment & Aggregation

Beyond simple extraction, the workflow calculates derived metrics like total wall surface area, aggregates measurements by floor level, and categorizes rooms by type (bedroom, bathroom, kitchen).

Step 5: Structured Output Delivery

Results are delivered as both machine-readable JSON for system integration and human-readable HTML tables for immediate review, completing the transformation from visual file to structured data.
