AI Gmail Classifier with Telegram Alerts

What This Workflow Does

Email overload is a universal business problem. Important client messages get buried under newsletters, promotions, and internal chatter, leading to missed deadlines and frustrated customers. This AI-powered workflow solves that by automatically reading, categorizing, and alerting you about every email that hits your Gmail inbox.

Using OpenAI's GPT-4o mini, the system intelligently classifies each incoming email into categories like High Priority (urgent client requests), Work Related (routine correspondence), Promotions (marketing emails), or Other. It then applies corresponding Gmail labels for easy filtering and sends a concise summary directly to your Telegram—so you're instantly informed without ever opening your inbox.

This isn't just about sorting emails; it's about creating a smart notification system that ensures critical communications receive immediate attention while non-essential emails are archived appropriately. For consultants, agencies, and busy professionals, this can save 5-10 hours per week of manual email management.

How It Works

The workflow follows a logical five-step process that transforms raw emails into actionable, categorized alerts.

1. Trigger: New Email Detection

The Gmail Trigger node polls your inbox at regular intervals (configurable from 1 minute to 1 hour). When a new email arrives, it extracts the sender, subject, body content, and metadata, then passes this data downstream for processing.

2. Classification: AI-Powered Sorting

The Classification Agent (powered by OpenAI via LangChain) analyzes the email content using a carefully crafted system prompt. It determines whether the email is urgent work, routine communication, promotional content, or miscellaneous. The output is structured JSON that downstream nodes can easily use.

3. Labeling: Automatic Gmail Organization

Based on the AI classification, the workflow applies the corresponding Gmail label. High Priority emails are also starred for visual prominence. These labels must exist in your Gmail beforehand, creating a consistent organizational system that works across all your devices.

4. Notification Generation: Smart Summaries

Another AI Agent node transforms the classified email into a concise, readable Telegram message. It extracts the key details—sender, subject, and most relevant snippet—and formats them for quick mobile consumption, saving you from reading the entire email.

5. Delivery: Instant Telegram Alert

The Telegram node sends the formatted notification to your specified chat or channel. You'll receive immediate alerts for high-priority items while less urgent emails are logged silently with their labels applied.

Pro tip: For maximum efficiency, create Telegram channels for different email categories (e.g., #urgent-emails, #work-updates) and route notifications accordingly using conditional logic in n8n.

Quick Setup Guide

Get this workflow running in under 15 minutes with these steps:

  1. Import the template into your n8n instance using the downloaded JSON file.
  2. Configure Gmail credentials in the Gmail Trigger node using OAuth2 authentication.
  3. Add your OpenAI API key to both AI Agent nodes in the credentials section.
  4. Create a Telegram bot via @BotFather, get your bot token, and find your chat ID using @userinfobot.
  5. Enter your Telegram credentials in the Telegram node with the bot token and chat ID.
  6. Create the three Gmail labels ("High Priority," "Work Related," "Promotions") in your Gmail account if they don't exist.
  7. Test the workflow by sending yourself a test email and verifying the classification and notification.

Important: The Gmail Trigger uses polling, not webhooks. Set an appropriate interval (5-15 minutes is recommended for most businesses) to balance responsiveness with API rate limits.
