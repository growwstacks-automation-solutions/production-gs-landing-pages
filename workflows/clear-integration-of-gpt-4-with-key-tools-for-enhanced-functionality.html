Turn discovery call forms into polished, personalized proposals in seconds. Automate your entire proposal workflow from capture to delivery.

What This Workflow Does

This automation solves one of the most time-consuming tasks for service businesses: writing personalized proposals. Instead of spending hours researching, drafting, and formatting documents for each prospect, this workflow creates professional, tailored proposals in under 30 seconds.

The system captures detailed prospect information through a Typeform discovery questionnaire, uses GPT-4 to analyze their needs and write compelling proposal content, then automatically generates a polished PandaDoc document complete with pricing tables, project timelines, and personalized recommendations.

It eliminates manual data entry, ensures consistency across all proposals, and allows you to respond to prospects faster than competitors—increasing conversion rates while reclaiming valuable business development time.

How It Works

1. Prospect Submission

A potential client fills out your Typeform discovery questionnaire with details about their company, challenges, budget, and project requirements. The workflow validates that all essential fields (email, company name) are completed before proceeding.

2. Intelligent Template Selection

Based on the prospect's budget and project complexity, the workflow automatically selects the appropriate proposal template. Quick Quote templates are used for simple projects under $2,500, while Standard Proposal templates handle more complex or higher-value engagements.

3. AI-Powered Content Creation

GPT-4 analyzes the prospect's challenges and writes personalized proposal content that directly addresses their pain points. It generates specific benefits with measurable metrics (time savings, cost savings), creates realistic 4-phase project timelines, and crafts a compelling challenge summary.

4. Document Generation & Delivery

The system populates your PandaDoc template with all generated content, including personalized fields, pricing tables, and expiration dates. Once the document is created, you receive a Slack notification with a direct link to review and send to the prospect.

What You'll Need

  1. OpenAI API account (costs approximately $0.05-0.15 per proposal)
  2. Typeform account (free tier works perfectly)
  3. PandaDoc account with API access
  4. Slack workspace for notifications
  5. n8n instance (cloud or self-hosted)

Pro tip: Start with a semi-automated approach. Have the AI draft the proposal and send it to you via Slack for review before it goes to the client. This maintains quality control while still saving 80% of the time.

Quick Setup Guide

  1. Import the template into your n8n instance using the downloaded JSON file.
  2. Add credentials for OpenAI, Typeform, and PandaDoc in their respective nodes.
  3. Create your Typeform with the recommended discovery questions (company name, industry, challenges, budget range, etc.).
  4. Set up PandaDoc templates with the placeholder tokens mentioned in the workflow description.
  5. Configure the Settings node with your company information, template IDs, and Slack webhook URL.
  6. Test the workflow by submitting a test Typeform response and verifying the proposal generates correctly.
