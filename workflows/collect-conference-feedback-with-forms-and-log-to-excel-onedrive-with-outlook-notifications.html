What This Workflow Does

This n8n workflow automates the entire conference feedback lifecycle, eliminating manual data collection and entry. It captures attendee responses from Microsoft Forms, organizes them in an Excel spreadsheet stored on OneDrive, and sends real-time notifications via Outlook when new feedback arrives.

The system transforms chaotic post-event feedback into structured, actionable data. Event organizers gain immediate visibility into attendee satisfaction without waiting for manual compilation. All feedback is timestamped, categorized, and securely stored for future analysis and reporting.

How It Works

1. Feedback Submission

Attendees complete a Microsoft Form with conference evaluation questions. The form can include rating scales, multiple choice, and open-ended questions tailored to your event's specific needs.

2. Automated Data Capture

The workflow triggers immediately when a form is submitted. It extracts all response data including timestamps, attendee information (if collected), and detailed feedback across all question types.

3. Excel Logging

Each submission appends a new row to a designated Excel file in OneDrive. The system maintains data integrity by mapping form fields to specific spreadsheet columns, ensuring consistent formatting.

4. Notification Alert

Outlook sends an email notification to designated organizers containing key feedback highlights and a direct link to the updated Excel file. Critical feedback can trigger immediate alerts while routine responses generate periodic summaries.

Pro tip: Add conditional logic to flag exceptionally positive or negative feedback for urgent review, helping you address critical issues while memories are fresh.
