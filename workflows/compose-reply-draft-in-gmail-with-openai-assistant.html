Automate Gmail Replies with OpenAI Assistant

Free n8n workflow that uses AI to compose professional email reply drafts automatically, saving hours of manual typing each week.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the time-consuming problem of manually responding to repetitive emails. For businesses receiving customer inquiries, support tickets, or standard business communications, crafting individual replies can consume hours each day. This workflow automatically generates professional, context-aware draft replies using OpenAI's Assistant, then places them directly into your Gmail drafts connected to the original email threads.

The system works by monitoring your Gmail for messages with a specific label (like "AI-Reply"), processing the email content through OpenAI to generate an appropriate response, formatting it as a proper email draft, and attaching it to the original conversation. This creates a seamless workflow where employees can review and send AI-assisted replies in seconds rather than composing them from scratch.

How It Works

Step 1: Trigger on Labeled Emails

The workflow runs at regular intervals (default: every minute) checking for new emails with your designated trigger label. This label acts as a signal that an email needs an AI-generated response.

Step 2: Process Content with OpenAI

The email's subject, body, and sender information are sent to your configured OpenAI Assistant. The AI analyzes the content, understands the context and intent, then generates a professional draft reply.

Step 3: Format and Create Draft

The AI's text response is converted to proper HTML email format and encoded according to RFC standards. n8n then uses the Gmail API to create a draft reply attached to the original email thread, maintaining the conversation context.

Step 4: Clean Up and Prevent Loops

Finally, the trigger label is removed from the email to prevent the workflow from processing the same message repeatedly. This leaves you with a ready-to-review draft in your Gmail drafts folder.

Pro tip: You can enhance this workflow by adding your company knowledge base to the OpenAI Assistant, allowing it to generate replies with specific product information, pricing details, or policy references automatically.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. Gmail account with API access enabled
  3. OpenAI API key with Assistant capabilities
  4. A configured OpenAI Assistant (you can create one specifically for email responses)
  5. A Gmail label to use as the trigger (e.g., "AI-Reply")

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance.
  2. Configure credentials for both Gmail and OpenAI in n8n's credential management.
  3. Create a label in your Gmail account (name it something like "AI-Reply").
  4. Update the workflow nodes to use your new label in both the trigger and cleanup steps.
  5. Select your OpenAI Assistant in the Assistant node configuration.
  6. Adjust the schedule trigger if you want to check for emails more or less frequently.
  7. Activate the workflow and test by applying your trigger label to an email.
