What This Workflow Does

This automation solves a critical problem for sales and marketing teams: incomplete contact data. When leads come in with just a name and email, your team wastes valuable time researching each person manually before they can engage effectively.

The workflow automatically transforms these basic leads into fully enriched HubSpot contacts by matching the person in Apollo's database, pulling their latest LinkedIn activity, analyzing the findings with GPT-4o, and upserting clean, structured data back into HubSpot. What used to take 10-15 minutes of manual research per lead now happens automatically in seconds.

For SDRs, this means inbound demo requests arrive pre-enriched with job titles, company information, and recent professional activity. For marketing teams, it enables highly segmented email campaigns based on actual professional interests rather than guesswork.

How It Works

The automation follows a sophisticated multi-step process that ensures data accuracy and relevance:

1. Data Cleaning & Preparation

The workflow begins by normalizing and deduplicating email addresses using a JavaScript Code node. This ensures consistent data formatting before enrichment begins, preventing errors in downstream processes.

2. Apollo Person Matching

The system queries Apollo's /people/match endpoint to find the exact person matching the provided name and email. If no match is found, the workflow intelligently skips to the next contact, avoiding wasted API calls.

3. LinkedIn Activity Fetch

Using RapidAPI's Fresh-LinkedIn-Profile-Data service, the workflow grabs up to three original posts from the last 30 days. This provides current professional insights rather than stale profile data.

4. AI Analysis & Summarization

Three LangChain nodes process the collected data through GPT-4o-mini, using structured and auto-fixing output parsers to produce a strict JSON block containing job title, location, professional summaries, and key insights.

5. HubSpot Contact Update

Every data point (plus five custom contact properties) maps directly into the HubSpot contact record, creating a complete professional profile ready for sales engagement.

What You'll Need

  1. Apollo.io API Key: For the HTTP Request node (set in x-api-key header)
  2. RapidAPI Key: For Fresh-LinkedIn-Profile-Data service (x-rapidapi-key header)
  3. OpenAI API Key: For the three LangChain nodes using GPT-4o-mini
  4. HubSpot OAuth2 Connection: For updating contact records
  5. Custom HubSpot Properties: Five custom contact properties should be created in HubSpot for the additional data points

Pro tip: Before running this workflow at scale, test with a small batch of contacts to ensure your API rate limits and custom field mappings work correctly. Consider adding error handling for cases where Apollo or LinkedIn data isn't available.
