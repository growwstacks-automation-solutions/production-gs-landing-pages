What This Workflow Does

This automation template solves a common business problem: extracting meaningful information from visual content without manual effort. It uses the GROQ LLaVA V1.5 7B API—a multimodal AI model with advanced vision capabilities—to analyze images and generate descriptive text automatically.

Imagine receiving product photos, screenshots, scanned documents, or marketing images and instantly getting AI-generated descriptions. This workflow connects Telegram as an input channel, allowing users to send images via chat, then processes them through GROQ's fast inference engine, and returns detailed textual analysis. It transforms visual data into actionable insights, saving hours of manual review and enabling scalable content analysis.

The value isn't just in converting images to text—it's in automating entire workflows around visual content. Businesses can use this for cataloging inventory, analyzing customer-submitted screenshots for support tickets, extracting data from infographics, generating accessibility descriptions for digital content, or monitoring social media visuals for brand sentiment.

How It Works

Step 1: Image Submission via Telegram

The workflow starts when a user sends an image to a configured Telegram bot. The Telegram Trigger node captures the image file and passes it to the next step. This provides a simple, mobile-friendly interface for teams, customers, or field staff to submit visual content.

Step 2: AI Vision Processing with GROQ

The image is sent to the GROQ LLaVA V1.5 7B API via an HTTP Request node. GROQ's specialized hardware delivers fast inference, analyzing the image's content, objects, context, and any visible text. The model returns a comprehensive textual description of what it "sees" in the image.

Step 3: Data Structuring & Output

The raw AI response is processed using the Edit Fields (Set) node to format the output into clean, structured text. You can customize this to extract specific elements—like object counts, color descriptions, or sentiment—depending on your business needs.

Step 4: Integration & Action

The generated text can then be routed to any connected system: saved to a database, sent to a CRM, posted to a Slack channel, appended to a spreadsheet, or used to trigger further automations. This turns visual input into structured data that fuels downstream processes.

Quick Setup Guide

1. Download the template JSON file and import it into your n8n workspace.

2. Create a Telegram bot via BotFather (@BotFather) and obtain your API token.

3. Sign up for GROQ API access and get your API key.

4. In the workflow, configure the Telegram Trigger node with your bot token.

5. Configure the HTTP Request node with your GROQ API key and the LLaVA model endpoint.

6. Adjust the Set node to format the output text as needed.

7. Activate the workflow and send an image to your Telegram bot to test.

8. Connect additional nodes to store or forward the generated text to your business systems.
