Automate LinkedIn Video Downloads to MP4 & Google Drive

What This Workflow Does

This automation solves a common pain point for marketers, content creators, and social media managers: manually downloading LinkedIn videos for repurposing, analysis, or archiving. The process typically involves finding the video, using various online tools, converting formats, saving files locally, and then manually uploading to cloud storage—a 15-20 minute task per video that becomes unsustainable at scale.

The workflow automatically converts any LinkedIn video URL into a downloadable MP4 file using the LinkedIn Video Downloader API via RapidAPI. It then uploads the converted file directly to your Google Drive with public sharing permissions enabled, and finally logs both the original LinkedIn URL and the new Google Drive link in a Google Sheets spreadsheet for tracking and organization.

Pro tip: Use this workflow to build a competitive intelligence library. Automatically download competitor videos, store them in organized Google Drive folders, and analyze their content strategy over time.

How It Works

Step 1: Capture LinkedIn Video URL

The workflow starts with an n8n web form or manual trigger where you paste the LinkedIn video URL. This could be triggered by new posts in a monitoring tool, scheduled searches, or manual input for one-off downloads.

Step 2: Convert to MP4 via RapidAPI

The LinkedIn Video Downloader API processes the URL and returns downloadable MP4 links. The workflow handles authentication with RapidAPI and manages API rate limits to ensure reliable conversions.

Step 3: Download & Upload to Google Drive

Once the MP4 link is obtained, the workflow downloads the file and immediately uploads it to your specified Google Drive folder. It sets appropriate sharing permissions so team members can access the content.

Step 4: Log Everything in Google Sheets

Finally, the workflow creates a detailed log entry in Google Sheets including the original URL, Google Drive link, timestamp, and status. Failed downloads are marked with "N/A" for easy troubleshooting.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. RapidAPI account with access to the LinkedIn Video Downloader API (free tier available)
  3. Google Workspace account with Drive and Sheets API access enabled
  4. Google Cloud Project with OAuth 2.0 credentials for Drive and Sheets integration
  5. Basic understanding of API keys and OAuth authentication flows

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance
  2. Configure RapidAPI credentials in the HTTP Request node with your API key from the LinkedIn Video Downloader service
  3. Set up Google Drive connection by authenticating with your Google account and specifying the target folder ID
  4. Connect Google Sheets and specify the spreadsheet ID and worksheet name for logging
  5. Test with a sample LinkedIn video URL to verify the entire flow works correctly
  6. Set up triggers – either schedule regular runs or connect to a form for manual triggering
