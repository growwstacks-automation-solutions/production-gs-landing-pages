What This Workflow Does

Manually downloading videos from Pinterest, converting them to MP4, saving them somewhere safe, and then sending them to someone is a tedious, time-consuming process. This workflow eliminates all those manual steps.

It automates the entire pipeline: from the moment a Pinterest video URL is submitted via a simple form, to the instant an MP4 download link lands in the requester's email. The video is processed via a reliable API, securely stored in your Google Drive with a shareable link, and delivered automatically. This turns a 10–15 minute manual task into a fully automated, background process that completes in seconds.

How It Works

The workflow follows a logical, step-by-step sequence to ensure reliability and deliver a polished result.

1. Capture the Request

The process starts with an n8n Form Trigger node. A user submits a Pinterest video URL and their email address through a simple web form. This triggers the workflow and passes the necessary data to the next step.

2. Fetch the Video Data

An HTTP Request node sends the submitted URL to the Pinterest Video Downloader API (hosted on RapidAPI). This service processes the Pinterest link and returns the data needed to download the video as an MP4 file.

3. Allow for Processing Time

A Wait node introduces a brief pause. This gives the external API sufficient time to fully process the video and generate a stable, downloadable MP4 file link, preventing errors from trying to download too quickly.

4. Download the MP4 File

The HTTP Downloader node retrieves the MP4 file from the URL provided by the API. The video file is now captured within the workflow and ready for the next stage.

5. Store in Cloud Storage

The Upload to Google Drive node saves the downloaded MP4 file directly to a specified folder in your Google Drive. This provides secure, organized, and accessible cloud storage.

6. Generate a Shareable Link

The Set Permissions Google Drive node configures the uploaded file to have "public" or "anyone with link" viewing permissions. This action automatically creates a shareable URL for the video.

7. Deliver to the User

Finally, a Send Email node composes and sends an email to the address provided in the initial form. The email contains a clear message and the direct, shareable Google Drive link to the downloaded MP4 file, completing the request.

Pro tip: Use this workflow as a client-facing service. Share the form link with clients or team members so they can request Pinterest videos themselves, and you only manage the automated output.

  2. A RapidAPI account and a subscription/key for the Pinterest Video Downloader API.
  3. A Google Cloud Project with the Google Drive API enabled, and OAuth 2.0 credentials set up in n8n.
  4. An Email account configured in n8n (like Gmail/ SMTP) for sending notifications.
  5. A basic understanding of how to paste API keys and OAuth credentials into n8n node settings.

Quick Setup Guide

Get this automation running in your own environment in just a few minutes.

  1. Download & Import: Click the "Download Template" button above. In your n8n instance, go to Workflows > Import from File and select the downloaded JSON file.
  2. Configure the API: In the HTTP Request node, replace the placeholder with your own RapidAPI key from the Pinterest Video Downloader API service page.
  3. Connect Google Drive: Click the "Google Drive" nodes and authenticate n8n with your Google account. Select the specific folder where downloaded videos should be saved.
  4. Set Up Email: Configure the Send Email node with your preferred email service (like Gmail or an SMTP server). Customize the email subject and body template.
  5. Activate & Test: Turn the workflow on. Use the n8n Form Trigger node's unique URL to submit a test Pinterest video link and your email. Check your inbox for the automated delivery.
