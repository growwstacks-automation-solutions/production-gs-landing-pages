What This Workflow Does

This automation solves the common e-commerce challenge of creating engaging, SEO-friendly content for product pages and blogs. Manually writing unique descriptions for dozens or hundreds of products is time-consuming and often leads to inconsistent quality.

The workflow automatically pulls product data from your Shopify store, processes it through AI (DeepSeek via OpenRouter) to generate professional blog content, then saves the results to Google Sheets and publishes drafts to your Shopify blog - all while sending email notifications at each step.

How It Works

Step 1: Fetch Shopify Product Data

The workflow begins by connecting to your Shopify store's API to retrieve product titles, descriptions, and metadata. You can configure how many products to process in each run.

Step 2: Log Raw Data to Google Sheets

Before processing, the original product data is saved to a Google Sheet for record-keeping and version control. This creates an audit trail of your source material.

Step 3: AI Content Generation

The product information is sent to DeepSeek's AI (via OpenRouter) which generates SEO-optimized blog posts with proper headings, bullet points, and formatting. A custom code node ensures clean JSON output.

Step 4: Shopify Blog Integration

The generated content is uploaded to your specified Shopify blog as drafts, ready for your review. The workflow includes your blog ID and handles all API connections.

Step 5: Email Notifications

At each major step, the workflow sends email notifications via Gmail to keep you informed about the automation's progress and results.

What You'll Need

  1. Shopify store with admin API access
  2. Google Sheets with proper column structure
  3. OpenRouter API key (or alternative AI provider)
  4. Gmail account for notifications
  5. Basic understanding of n8n workflow configuration

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure the Variables node with your Shopify store name and blog ID
  3. Connect the Google Sheets nodes to your spreadsheet
  4. Set up your OpenRouter API credentials in the Chat Model node
  5. Test with a small batch of products before full deployment
