What This Workflow Does

This automation solves a common problem for busy professionals, marketers, and founders: maintaining a consistent, high-quality LinkedIn presence without spending hours writing posts. It turns raw ideas, links, or notes sent via Telegram into professionally crafted LinkedIn content using Google's Gemini AI.

The workflow intelligently processes your input—whether it's a URL, a topic suggestion, or direct content—gathers additional context via web search or scraping, generates a LinkedIn-optimized post, and sends it for your approval via Telegram before publishing. It logs everything in Google Sheets for tracking and analysis.

Essentially, it acts as your personal content assistant, handling research, drafting, formatting, and scheduling, while you retain final approval control. This cuts content creation time by 80–90% while ensuring every post meets your professional standards.

How It Works

The workflow follows a logical, multi-step process to ensure quality and relevance.

1. Input & Authorization

You send a message to a dedicated Telegram bot. The workflow first verifies you're an authorized user, then classifies your input—URL, topic, or direct text—and extracts key elements for processing.

2. Content Enrichment

Based on the input type, the system gathers additional information. For URLs, it scrapes the article content. For topics, it performs a real-time web search to pull in the latest insights. This step ensures the AI has rich context to work with.

3. AI Generation & Formatting

Google Gemini AI synthesizes the gathered information and generates a LinkedIn post. It structures the content with engaging hooks, mobile-friendly paragraphs, strategic calls-to-action, and relevant hashtags, all within LinkedIn's character limit.

4. Approval & Publishing

The draft post is sent back to you on Telegram with action buttons (Approve, Edit, Reject). Upon approval, it's automatically published to your LinkedIn profile. If edits are requested, the AI rewrites based on your feedback.

What You'll Need

  1. A self-hosted n8n instance (required for community nodes).
  2. Telegram Bot Token (from BotFather).
  3. Google Gemini API key.
  4. Firecrawl API key (for web scraping).
  5. Brave Search API subscription key.
  6. LinkedIn Developer App credentials.
  7. Google Sheets (for logging).
  8. Installation of three community nodes: @brave/n8n-nodes-brave-search, @mendable/n8n-nodes-firecrawl, and @n8n/n8n-nodes-langchain.

Pro tip: Start with the free tiers for Gemini, Firecrawl, and Brave Search to test the workflow. They offer enough capacity for personal or light business use.

Quick Setup Guide

Follow these steps to get your automated LinkedIn content pipeline running in under 30 minutes.

  1. Install Community Nodes: In your n8n instance, install the three required community nodes and restart n8n.
  2. Create APIs & Credentials: Set up accounts and obtain API keys for Telegram, Google Gemini, Firecrawl, Brave Search, LinkedIn, and Google Sheets as outlined in the detailed guide.
  3. Import & Configure: Import the downloaded JSON template into n8n. Connect all credential nodes using your new API keys.
  4. Customize & Test: Update the "Authorized Telegram Users" node with your Telegram ID. Create a Google Sheet for logging and paste its ID into the Google Sheets node. Send a test message to your Telegram bot.
