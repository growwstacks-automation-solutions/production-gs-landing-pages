What This Workflow Does

This automation solves a critical content marketing challenge: producing consistent, high-quality video content from existing written material. Manual video creation is time-consuming, expensive, and difficult to scale. This workflow completely automates the transformation of blog articles or any website content into polished Instagram Reels.

The system takes a URL as input and produces a fully composed video with AI-generated background clips, professional voiceover narration, animated subtitles, background music, and proper source attribution. It handles the entire pipeline from content extraction to social media publishing, turning hours of manual work into a fully automated process that runs in minutes.

For content teams and digital marketers, this represents a massive efficiency gain. Instead of creating one video per week manually, you can generate multiple videos daily from your existing content library, dramatically increasing your social media presence and content reach without additional creative resources.

How It Works

1. Content Extraction & Summarization

The workflow begins by scraping the target webpage using Firecrawl API, extracting clean markdown content. An LLM via OpenRouter then condenses the article into digestible talking points limited to 1,000 characters, ensuring the video remains concise and engaging.

2. Script & Video Prompt Generation

A second LLM processes the summarized content to create structured JSON containing 3-5 specific video prompts, narration text, and a social media caption. This structured approach ensures visual coherence between what's shown and what's said throughout the video.

3. AI Video & Audio Creation

Google's Veo API generates 8-second background clips in 9:16 format (perfect for Reels) for each prompt. Simultaneously, OpenAI's text-to-speech converts the narration to professional voiceover, while Openverse API fetches royalty-free background music that matches the content tone.

4. Composition & Publishing

All assets upload to JsonCut's storage, where they're merged with auto-generated subtitles, smooth transitions, branding overlays, and source attribution. The final video then publishes directly to Instagram via Blotato with the generated caption, completing the automation cycle.

What You'll Need

  1. Self-hosted n8n instance (required for community nodes)
  2. Firecrawl API account for web scraping
  3. OpenRouter API key for LLM access (GPT-4 Mini used in template)
  4. Google Gemini API credentials for Veo video generation
  5. OpenAI account for text-to-speech functionality
  6. JsonCut account for video composition and hosting
  7. Blotato account connected to an Instagram Business profile
  8. Brand logo URL for overlay customization

Pro tip: Start with Google Veo's free tier (10 requests/day) to test the workflow before scaling. Each complete video uses 3-5 Veo requests, allowing 2-3 videos daily on the free plan.

Quick Setup Guide

  1. Install these community nodes in your n8n instance: @mendable/n8n-nodes-firecrawl, n8n-nodes-jsoncut, and @blotato/n8n-nodes-blotato.
  2. Configure all API credentials in n8n's credential manager with appropriate access levels.
  3. Update the Blotato node with your Instagram Business account ID.
  4. Replace the branding overlay image URL in the JsonCut node with your logo.
  5. Test the workflow by triggering it with a blog article URL via the chat trigger.
  6. Monitor the first run and adjust video prompts or narration style as needed.
