What This Workflow Does

This automation solves the time-consuming challenge of creating consistent short-form video content. Instead of spending hours scripting, filming, and editing, you simply send a text prompt to Telegram. The workflow automatically generates a polished one-minute video using Google's Veo 3 AI, manages all assets in Google Drive, and delivers the final product back to you.

It's designed for content creators, marketing teams, and businesses that need to produce high volumes of social media clips, promotional videos, or educational content without sacrificing quality or consistency. The system handles the entire production pipeline, from initial idea to final video, with zero manual intervention.

How It Works

1. Trigger via Telegram Message

You or your team members send a specific command with a video topic to a Telegram bot. For example: "/makevideo Create a motivational clip about productivity." This message immediately activates the workflow without any login or interface navigation.

2. AI Video Generation with Veo 3

The system processes your prompt and uses Google's Veo 3 to generate coherent, visually consistent video scenes. The AI maintains proper pacing, story flow, and visual style across the entire one-minute duration, creating content that feels professionally produced.

3. Automated Asset Management

All generated video clips are automatically uploaded to a designated Google Drive folder with public sharing links enabled. This ensures Fal.ai can reliably access and process the clips during the final assembly phase without permission issues.

4. Final Assembly & Delivery

Fal.ai merges the video segments, applies any configured enhancements, and produces the final one-minute video. The completed file is sent directly back to you via Telegram, ready for immediate use or further distribution.

What You'll Need

  1. An active n8n instance (cloud or self-hosted)
  2. Telegram bot token (create via @BotFather)
  3. Google Cloud credentials with Drive API enabled
  4. Fal.ai API key for video processing
  5. Access to Google's Veo 3 or similar AI video service

Pro tip: Start with a specific content niche (like "weekly tips" or "product highlights") to refine your prompts before scaling to broader topics. Consistent prompt structure yields more consistent video output.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n workspace.
  2. Configure the Telegram trigger node with your bot token and set the command listeners.
  3. Add your Google Service Account credentials to the Google Drive nodes and specify your target folder ID.
  4. Enter your Fal.ai API key in the HTTP Request nodes that call their video processing endpoints.
  5. Test the workflow by sending a simple prompt to your Telegram bot and verify the end-to-end process.
