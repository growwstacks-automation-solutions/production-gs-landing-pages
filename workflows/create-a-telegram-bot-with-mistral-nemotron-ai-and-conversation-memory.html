How It Works

1. Message Reception & Classification

The Telegram trigger node captures all incoming messages. The workflow immediately classifies them into three categories: commands (like /start or /clear), regular text messages for AI processing, and non-text content (photos, videos, etc.) that receives a polite error response.

2. Conversation History Management

For each user, the workflow maintains a conversation history of the last 10 messages. This memory is stored and retrieved with every interaction, allowing the AI to reference previous context. The system automatically trims old messages to keep the context relevant and manageable.

3. AI Processing & Response Generation

Text messages are formatted with the conversation history and sent to your chosen AI API. The workflow prepares the request with appropriate parameters (temperature, token limits, system prompts) and sends it to the AI service. You can customize this for any provider—OpenAI, Anthropic, Google, etc.

4. Response Delivery & Memory Update

The AI's response is sent back to the user through Telegram. Simultaneously, the conversation history is updated with both the user's latest message and the AI's response, maintaining continuity for future interactions.

5. Command Handling

Special commands like /start provide a welcome message introducing the bot's capabilities, while /clear resets the conversation history for that user, allowing fresh starts when needed.

Pro tip: Start with a lower-cost AI model like GPT-3.5 or Gemini Flash for testing, then upgrade to more powerful models once you validate the conversation patterns and volume.

Quick Setup Guide

Follow these steps to get your AI Telegram bot running in under 30 minutes:

  1. Download and Import: Download the template JSON file and import it into your n8n instance.
  2. Configure Telegram Credentials: Add your Telegram bot token in the credentials section and test the connection.
  3. Set Up AI API Credentials: Add your chosen AI provider's API key (Bearer token authentication).
  4. Customize AI Parameters (Optional): Adjust the temperature, response length, or system prompt in the "Prepare API Request" node if needed.
  5. Activate the Workflow: Turn on the workflow and note the webhook URL generated by n8n.
  6. Set Telegram Webhook: Use the Telegram API to set the webhook URL for your bot.
  7. Test the Bot: Send a message to your bot on Telegram and verify it responds correctly.

Pro tip: Use the /clear command during testing to reset conversation history and test fresh interactions. Monitor the first 100 conversations to refine your AI prompt for better responses.
