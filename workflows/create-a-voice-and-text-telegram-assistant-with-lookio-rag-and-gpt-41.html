Build a smart Telegram bot that answers questions using your documents—handles both voice and text messages.

What This Workflow Does

This automation solves the problem of slow, inaccurate answers from internal knowledge bases. Instead of employees searching through documents or asking colleagues, they can simply message a Telegram bot—via text or voice—and get precise, source‑backed answers instantly.

The workflow uses Retrieval‑Augmented Generation (RAG) powered by Lookio to ground responses in your uploaded documents (PDFs, manuals, FAQs). An AI agent (GPT‑4.1) decides when to fetch knowledge and returns concise, Telegram‑friendly replies. A privacy filter lets you restrict access to specific users for safe testing.

How It Works

Step 1: Trigger & Routing

The Telegram Trigger node listens for incoming messages. A Message Router detects whether the input is text or a voice file. Voice files are fetched via the Get Audio File node.

Step 2: Transcription

Voice messages are transcribed using the Mistral API (or another provider). The transcript—or original text—is normalized into a consistent format for processing.

Step 3: Agent Reasoning

The AI Agent (GPT‑4.1‑mini) runs with a system prompt instructing it to call the Query Knowledge Base tool when domain‑specific knowledge is needed. This separates reasoning from retrieval.

Step 4: Knowledge Retrieval

The Lookio tool queries your uploaded documents via its API, returning relevant snippets. The agent incorporates this information into its final answer.

Step 5: Response Delivery

The agent’s output is sent back to the user via Telegram Answer, completing the conversation loop.

What You'll Need

  1. An n8n instance (cloud or self‑hosted)
  2. A Telegram Bot token added in n8n credentials
  3. A Lookio account, API Key, and Assistant ID
  4. An LLM provider account (OpenAI or equivalent) for the OpenAI Chat Model node
  5. A Mistral API key (or other transcription provider) for voice transcription

Quick Setup Guide

  1. Create a Lookio assistant: Sign up to Lookio, upload your documents, and create an assistant.
  2. Add credentials in n8n: Configure Telegram API, OpenAI, and Mistral Cloud credentials.
  3. Configure Lookio tool: In the Query Knowledge Base node, replace the API key and Assistant ID placeholders.
  4. Set Telegram privacy (optional): Edit the Myself? If node to restrict access to your username.
  5. Adjust transcription (optional): Swap the Mistral transcribe node for another provider if needed.
  6. Connect LLM: Add your OpenAI API key in the OpenAI Chat Model node.
  7. Activate workflow: Activate and test by messaging your bot in Telegram.
