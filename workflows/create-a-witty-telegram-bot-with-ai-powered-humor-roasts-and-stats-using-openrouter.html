What This Workflow Does

This n8n workflow automates the creation of an engaging Telegram bot that uses OpenRouter's AI models to generate witty responses, humorous roasts, and track conversation statistics. The bot maintains context across conversations and adapts its humor style based on user interactions.

How It Works

1. Telegram Message Trigger

The workflow starts when users send messages in Telegram.

2. AI Processing via OpenRouter

Messages are routed to OpenRouter's API for AI-powered response generation.

3. Contextual Humor Generation

The AI analyzes message context and generates appropriate humorous or roast-style responses.

4. Response Delivery

The bot sends the crafted response back to the Telegram chat.

5. Statistics Tracking

All interactions are logged for analytics.

Who This Is For

Community managers, content creators, and marketers looking to boost engagement in Telegram groups. Businesses running Telegram channels can use this to increase user interaction rates.
