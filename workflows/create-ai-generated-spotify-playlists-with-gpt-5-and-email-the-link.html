What This Workflow Does

Manually creating the perfect playlist for an event, campaign, or customer gift is time-consuming and often hit-or-miss. This automation solves that by leveraging artificial intelligence to do the creative heavy lifting. When a user submits a simple form with details about an occasion—like a wedding, product launch, or corporate retreat—the workflow transforms that description into a professionally curated Spotify playlist and delivers it directly to the recipient's inbox.

The process eliminates hours of song searching, ensures the music matches the specific vibe and audience, and provides a seamless, "wow-factor" experience for the end-user. It's perfect for event planners, marketers, hospitality businesses, or anyone looking to add a personalized, automated touch to their services.

How It Works

The workflow is a smart sequence that connects a user request to AI analysis, music platform action, and final delivery.

1. Form Trigger & Data Collection

The automation starts when a user submits a web form. This form captures essential context: the type of event (e.g., "energetic tech conference after-party"), guest demographics, musical preferences, mood, and the recipient's email address. This structured data becomes the creative brief for the AI.

2. AI-Powered Song Curation

The event details are sent to OpenAI's GPT-5 model. The AI acts as an expert music curator, analyzing the context to generate a list of 18–32 song recommendations. It provides each song with the correct title and artist, and even creates a catchy, relevant name for the playlist itself. The AI responds in the user's language, enabling global use.

3. Spotify Playlist Creation & Population

The workflow authenticates with the Spotify API and searches for each recommended song. For every successful match, it retrieves the unique Spotify track ID. It then creates a new private or public playlist on the specified Spotify account and adds all the found tracks in the recommended order.

4. Email Notification & Delivery

Finally, the system sends a beautifully formatted email to the user. This email includes the custom playlist name, a direct "Open in Spotify" link, and instructions. The user can immediately listen, share, or modify the playlist, completing a fully automated, personalized experience.

What You'll Need

  1. OpenAI API Key: Required to access the GPT-5 model for generating intelligent song recommendations based on your prompts.
  2. Spotify Developer Credentials: A Client ID and Client Secret from the Spotify Developer Dashboard to search songs, create playlists, and add tracks via their API.
  3. SMTP Email Credentials: Access to an email service (like Gmail, SendGrid, or your company SMTP) to send the final playlist link to users. You'll need server, port, username, and password details.
  4. n8n Instance: A running instance of n8n, either self-hosted or via n8n.cloud, to import and execute this workflow.
  5. Web Form (Optional but Recommended): A way to trigger the workflow. This could be an n8n Form Trigger node, a form from tools like Typeform or Google Forms, or data passed from your own application.

Quick Setup Guide

Get your automated playlist generator up and running in less than 30 minutes by following these steps.

  1. Download & Import: Click the "Download Template" button above to get the JSON file. In your n8n instance, go to Workflows, click "Import from file," and select the downloaded JSON.
  2. Configure API Credentials: Locate the "OpenAI," "Spotify," and "Email Send" nodes in the workflow. Open each one and enter your respective API keys and account credentials in the designated authentication fields.
  3. Set Up the Trigger: Configure the Webhook or Form Trigger node. Copy the unique URL it provides. This is the endpoint your form will submit data to.
  4. Connect Your Form: Point your existing form tool (e.g., Typeform, JotForm) to send a POST request to the webhook URL with the required fields: event details, preferences, and email address.
  5. Test & Activate: Toggle the workflow to "Active." Submit a test form entry with sample data. Check that you receive an email with a link to a newly created Spotify playlist containing the AI-recommended songs.
