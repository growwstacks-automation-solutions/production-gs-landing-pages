What This Workflow Does

This automation transforms WhatsApp into a powerful AI-powered communication channel for your business. It eliminates manual message handling while providing intelligent, instant responses to customers 24/7.

The workflow connects Whapi.Cloud's WhatsApp API with OpenAI's GPT-3.5 to create a sophisticated chatbot that understands natural language, handles complex queries, and manages group communications - all without writing code.

How It Works

1. Message Reception

Whapi.Cloud captures incoming WhatsApp messages via webhook and forwards them to your n8n workflow. This happens in real-time with minimal latency.

2. Command Processing

The workflow analyzes each message to determine if it's a:

  • AI chat request (/ai prefix)
  • Numeric command (1-9 for menu options)
  • Help request
  • Group management command

3. Response Generation

Based on the command type, the workflow either:

  • Queries OpenAI for AI-powered responses
  • Pulls pre-defined responses from your database
  • Executes group management functions
  • Returns help instructions

What You'll Need

  1. Whapi.Cloud account with active WhatsApp number
  2. OpenAI API key (optional for AI features)
  3. n8n instance (cloud or self-hosted)
  4. Basic understanding of webhooks

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure Whapi.Cloud credentials in the HTTP Request node
  3. Set up the webhook URL in your Whapi.Cloud dashboard
  4. Add your OpenAI API key if using AI features
  5. Test with your WhatsApp number

Pro tip: Start with basic commands before enabling AI features. Test thoroughly with your team before going live with customers.
