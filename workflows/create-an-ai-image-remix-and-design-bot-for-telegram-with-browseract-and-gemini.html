What This Workflow Does

This automation turns your Telegram bot into a powerful creative partner. It solves the problem of manual image creation and design brainstorming by providing instant AI-powered visual generation directly within your Telegram chats. Whether you're a content creator needing fresh visuals, a marketer seeking inspiration, or a community manager wanting interactive experiences, this workflow delivers automated creative assistance.

The system intelligently handles multiple creative tasks: it can engage in conversational chat, scrape trending design prompts from websites using BrowserAct for inspiration, or perform deep "remixes" of uploaded photos by analyzing composition and regenerating entirely new images with sophisticated prompt engineering. This eliminates hours of manual design work and provides on-demand creative capabilities to your audience or team.

How It Works

Step 1: Message Reception & Traffic Routing

The workflow begins with a Telegram Trigger node that captures all incoming messages and interactions. It immediately splits the traffic based on interaction type—new text messages go one path, while button clicks (like "Regenerate" or "Next") follow another. This intelligent routing ensures smooth user experiences and maintains conversation state.

Step 2: Intent Classification with AI

An AI Agent analyzes text inputs to determine user intent. It distinguishes between casual conversation ("Chat" mode) and creative requests ("Start" mode for inspiration). This natural language understanding allows users to interact naturally without learning specific commands, making the bot accessible to everyone.

Step 3: Inspiration Gathering via BrowserAct

When creative mode is triggered, BrowserAct automatically scrapes trending image prompts from design inspiration sites like PromptHero. It navigates websites, extracts relevant prompts and metadata, and saves this inspiration to a Google Sheet for tracking and future reference, creating a constantly updated creative database.

Step 4: Advanced Image Analysis

If users upload images, an AI Vision Agent (powered by OpenRouter/Gemini) performs detailed visual forensics. It analyzes lighting, composition, subjects, colors, and artistic style, then generates comprehensive descriptions that capture the essence of the original image for intelligent remixing.

Step 5: Master Prompt Engineering

Specialized AI Agents expand the scraped prompts or image descriptions using the "Rule of Multiplication" technique. They transform simple inputs into massive, detailed generation prompts that include artistic styles, technical specifications, mood descriptors, and compositional guidance for optimal AI image generation.

Step 6: Image Generation & Delivery

Google Gemini generates the final images based on the engineered prompts. The results are sent back to Telegram with interactive buttons allowing users to "Regenerate" variations or move to the "Next" idea, creating an engaging, iterative creative process within the chat interface.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n instance using the workflow import function.
  2. Configure Credentials: Set up connections for Telegram, Google Sheets, BrowserAct, Google Gemini, and OpenRouter in n8n's credentials management.
  3. Prepare BrowserAct: Ensure you have the specific "Image Remix & Design Bot" template saved in your BrowserAct account and note the workflow ID.
  4. Create Google Sheet: Set up a spreadsheet with four tabs: PromptHero, Current State, UserImage, and Current Image with appropriate column headers.
  5. Connect Sheet: Update all Google Sheets nodes in the workflow with your spreadsheet ID and ensure proper column mappings.
  6. Test Telegram Bot: Activate your bot, test basic functionality, and configure any desired welcome messages or commands.
  7. Activate Workflow: Turn on the workflow in n8n and test with sample messages and image uploads to verify all connections work properly.

Pro tip: Start with a private Telegram group for testing before deploying to public channels. This allows you to refine prompts, adjust image quality settings, and ensure the bot responds appropriately to various inputs without public errors.
