How It Works

The automation follows a logical sequence that mimics a professional animator's workflow but executes it programmatically through API connections.

1. Text Prompt Processing

The workflow begins with your creative input—a text description of the scene you want to animate. The system accepts natural language prompts with optional artistic parameters (like "--s 500" for style or "--niji 6" for anime style). This flexibility allows both simple descriptions and detailed artistic direction.

2. Midjourney Image Generation

Your prompt is sent to Midjourney via PiAPI, which acts as a professional gateway service. The system handles authentication, queue management, and quality optimization automatically. Once the image generation task is submitted, the workflow monitors its status and retrieves the high-quality static illustration when complete.

3. Kling API Animation

The generated image is then passed to Kling's video synthesis API, along with a motion prompt describing how elements should move (like "butterflies flutter around them"). Kling analyzes the image and applies intelligent motion based on the scene composition and your instructions, creating a seamless animated sequence.

4. Quality Assurance & Delivery

The final animated video is processed, formatted appropriately for your target platforms (social media dimensions, optimal file size), and delivered to your specified destination—whether that's cloud storage, a content management system, or directly to social media scheduling tools.

Pro tip: For consistent brand styling, create a library of successful prompts with specific artistic parameters (like color palettes, lighting, and composition rules) that you can reuse across different projects.

Quick Setup Guide

You can have this automation running in under 15 minutes with these simple steps:

  1. Import the template: Download the JSON file above and import it into your n8n instance via the "Import from File" option.
  2. Configure PiAPI credentials: Locate the Midjourney and Kling nodes in the workflow. In each, add your PiAPI X-API-Key from your account dashboard.
  3. Test with sample prompts: Run the workflow once with the included example prompts to verify everything connects properly.
  4. Customize for your needs: Modify the prompt nodes with your specific content requirements and adjust output settings for your preferred platforms.
  5. Add triggers & destinations: Connect the workflow to your content calendar or CMS, and set where finished videos should be saved.

Pro tip: Start by generating 3-5 variations of the same concept with slightly different prompts to understand how parameter changes affect the final output. This helps you develop your own "prompt recipe book" for consistent results.
