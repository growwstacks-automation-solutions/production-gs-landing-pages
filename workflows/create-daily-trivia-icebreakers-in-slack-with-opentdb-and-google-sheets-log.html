Automate Daily Trivia Icebreakers in Slack

Boost team engagement with automated daily trivia questions. Post to Slack and log everything in Google Sheets for future reuse.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the common problem of maintaining team engagement in remote or hybrid environments. Manual icebreakers require consistent effort and often fall off schedule. This workflow automatically delivers a fresh trivia question to your Slack channel every day, providing a lightweight, fun moment that encourages interaction without demanding time.

Beyond just posting, it builds a valuable asset: a logged trivia database in Google Sheets. Every question, its category, difficulty, and answers are stored. Over time, this creates a reusable repository for quizzes, training content, or community events, turning a simple daily habit into a strategic knowledge base.

How It Works

Step 1: Schedule Trigger Initiates

A schedule node runs once per day at your chosen time (e.g., 9 AM). This ensures consistency without manual intervention.

Step 2: Random Difficulty Selection

A Set node randomly picks a difficulty level—easy, medium, or hard—to keep variety and cater to different team preferences.

Step 3: Fetch Question from OpenTDB

A Switch node routes to the appropriate OpenTDB API request based on the selected difficulty. The API returns a random multiple-choice question with category, correct answer, and incorrect options.

Step 4: Data Normalization

Each branch processes the API response into standardized fields: timestamp, date, difficulty, category, question text, correct answer, incorrect answers, and formatted message components.

Step 5: Merge and Distribution

A Merge node combines all branches into a single data stream. Slack then posts a nicely formatted trivia message to your designated channel. Simultaneously, Google Sheets appends the full data as a new row, building your archive.

  1. A Slack workspace with a channel designated for icebreakers
  2. Slack OAuth2 credentials configured in n8n
  3. A Google Sheets spreadsheet with a sheet ready for logging
  4. Google Sheets API credentials set up in n8n
  5. An n8n instance (cloud or self-hosted) to run the workflow

Quick Setup Guide

Import the template JSON into your n8n workspace. Then follow these steps:

  1. Connect your Slack credentials and select the target channel.
  2. Connect your Google Sheets credentials and specify the spreadsheet and sheet name.
  3. Adjust the schedule trigger to match your preferred time and frequency.
  4. Test the workflow manually once to ensure all connections work.
  5. Activate the workflow and let it run automatically.

Pro tip: Customize the Slack message format by adding emojis, hints, or team-specific references to make it feel more personal and engaging.
