Automate Multi-Speaker Podcasts with Google Sheets & ElevenLabs

Transform spreadsheet scripts into professional, multi-voice podcast episodes automatically. Save hours of editing and production time.

What This Workflow Does

This automation solves the time-intensive challenge of producing multi-speaker podcast content. Instead of coordinating multiple voice actors, scheduling recording sessions, and manually editing audio files, this workflow transforms a simple Google Sheets script into a fully-produced podcast episode using ElevenLabs' advanced AI voice technology.

The system reads dialogue from structured spreadsheet rows—where each line includes speaker assignments, voice IDs, and text—and generates natural-sounding conversations with distinct character voices. It handles everything from script parsing to audio generation and final file organization in Google Drive, creating broadcast-ready content in minutes rather than days.

Beyond basic text-to-speech, this workflow supports expressive delivery with emotional cues, natural pauses, and even non-verbal audio elements like [laughter] or [applause] markers, resulting in engaging, professional-quality audio content that maintains listener interest.

How It Works

1. Script Management in Google Sheets

Your team collaborates on podcast scripts directly in Google Sheets. Each row represents a line of dialogue with columns for speaker name (optional), specific ElevenLabs voice ID, and the actual text to be spoken. This familiar spreadsheet interface makes script writing and revisions accessible to everyone, not just technical team members.

2. Data Transformation & Formatting

The workflow extracts the spreadsheet data and restructures it into the precise JSON format required by the ElevenLabs Text-to-Dialogue API. This includes organizing dialogue sequences, assigning voice parameters, and preparing any audio effect markers embedded in the script.

3. AI Voice Generation via ElevenLabs

The formatted script is sent to ElevenLabs' v3 API, which generates a cohesive audio file featuring multiple distinct voices conversing naturally. The AI handles timing, emotional inflection, and character differentiation, creating a seamless conversation from individual lines.

4. Automated File Storage & Organization

The generated MP3 file is automatically uploaded to a designated Google Drive folder with a timestamped filename. Metadata can be added, and notifications can be configured to alert your team when new episodes are ready for review or distribution.

Quick Setup Guide

Follow these steps to implement this podcast automation in your n8n environment:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Duplicate the Google Sheets template and populate it with your podcast script, ensuring proper column formatting
  3. Configure ElevenLabs credentials in n8n using HTTP Header Auth with your xi-api-key
  4. Set up Google OAuth connections for both Sheets (to read scripts) and Drive (to save audio files)
  5. Update node parameters with your specific Sheet ID, Drive folder ID, and preferred voice settings
  6. Test with a short script to verify audio quality and file storage before processing full episodes
  7. Schedule or trigger the workflow based on your production calendar (manual, scheduled, or webhook)
