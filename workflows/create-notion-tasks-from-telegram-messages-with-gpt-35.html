What This Workflow Does

This automation solves the problem of losing track of important tasks mentioned in Telegram chats or voice messages. Instead of manually copying information between apps, the workflow uses AI to intelligently convert messages into properly formatted Notion tasks automatically.

The system listens for Telegram messages (both text and voice notes), processes them through GPT-3.5 to extract actionable tasks, then creates well-structured items in your Notion task database with all relevant details. It handles everything from simple "remind me to..." messages to complex multi-step instructions.

How It Works

1. Telegram message trigger

The workflow activates when you send a message to your Telegram bot or in a designated channel. It works with both text messages and voice notes.

2. AI-powered message processing

GPT-3.5 analyzes the message content to identify task components: action items, due dates, priorities, and relevant tags. For voice notes, it first converts speech to text.

3. Structured task creation

The system formats the extracted information into Notion's database structure, applying your preferred task properties and ensuring consistent formatting.

4. Notion database update

The new task gets added to your specified Notion database with all identified details automatically populated in the correct fields.
