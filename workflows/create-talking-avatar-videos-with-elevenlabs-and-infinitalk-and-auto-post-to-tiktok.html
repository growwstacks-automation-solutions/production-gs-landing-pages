What This Workflow Does

This automation solves the time-consuming process of creating engaging video content for social media. Instead of hiring actors, recording voiceovers, and manually editing videos, this workflow transforms a simple text script and a static image into a fully animated talking avatar video, then publishes it directly to TikTok.

The system eliminates 5–10 hours of manual work per video, enabling businesses and creators to produce consistent, high-quality content daily. It's perfect for weather forecasts, product announcements, educational snippets, news briefs, or any scenario where a personalized video presenter adds value but filming real people isn't practical or scalable.

How It Works

1. Text-to-Speech Generation

The workflow starts by sending your text script and selected voice profile (like "Alice") to ElevenLabs via Fal.ai. It generates studio-quality audio, polls until completion, and retrieves the audio file URL.

2. Avatar Animation

The static image URL and generated audio are sent to Infinitalk (also via Fal.ai). This AI model animates the avatar's lips and facial expressions to match the audio, guided by a prompt like "You are a girl giving a weather forecast and you must be expressive."

3. Title Optimization & Publishing

While the video renders, OpenAI GPT-4o-mini generates an engaging TikTok title under 60 characters. The finished video is downloaded and uploaded to Postiz, which schedules and publishes it to your connected TikTok account automatically.

  5. Static Image & Script – Base avatar image and text content for the video.

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n workspace.
  2. Create credentials in n8n for Fal.ai (HTTP Header Auth with Authorization header), OpenAI, and Postiz.
  3. In the Postiz node, replace "XXX" with your actual TikTok integration ID from your Postiz dashboard.
  4. Modify the "Set text input" node with your script and preferred voice.
  5. Update the "Set Video Params" node with your image URL and avatar expression prompt.
  6. Test the workflow with a short script to verify all API connections work.
  7. Schedule the workflow to run daily or trigger it via webhook for on-demand video creation.

Pro tip: Use different avatar images and voice profiles for various content series. Create a library of branded avatars for different topics or products to maintain visual consistency while scaling production.
