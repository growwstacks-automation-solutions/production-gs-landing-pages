Create Viral AI Celebrity Selfies & Upload to Instagram

Automate AI-generated viral celebrity selfie creation using Nano Banana Pro via RunPod, generate captions with Google Gemini, and post to Instagram via Postiz.

What This Workflow Does

This workflow solves the time-intensive challenge of creating engaging, viral-ready social media content manually. It automates the entire pipeline from idea to publication: generating AI-powered celebrity-style selfies, crafting platform-optimized captions, and scheduling posts to Instagram.

For content creators, marketers, and agencies, manual content creation can consume hours per post—concepting, designing, writing, and scheduling. This automation reduces that process to minutes, handling image generation via Nano Banana Pro on RunPod, intelligent caption creation with Google Gemini, asset storage in Google Drive, and final distribution through Postiz to Instagram.

The business value is immense: scalable content production, consistent quality, and the ability to experiment with viral formats without increasing operational overhead. It turns a creative bottleneck into a strategic advantage.

How It Works

1. Form Submission Trigger

The workflow starts when a user submits a simple web form with three inputs: the source image URL, a custom prompt describing the desired celebrity transformation, and the output aspect ratio. This makes the system accessible to non-technical team members or even customers.

2. AI Image Generation via RunPod

The submitted data is sent to the RunPod API, which runs the Nano Banana Pro Edit model. This advanced AI model transforms the input image according to the prompt, creating a high-quality, celebrity-style selfie. The workflow periodically checks the generation status until completion.

3. Intelligent Caption & Hashtag Creation

Once the image is ready, it's analyzed by Google Gemini (PaLM). The AI generates an engaging, viral-optimized Instagram caption and relevant hashtags, ensuring the content feels human-written and platform-native—avoiding obvious AI language that can reduce engagement.

4. Asset Backup & Social Media Publishing

The final image is uploaded to a designated Google Drive folder for backup and organization. Simultaneously, the image and caption are sent to Postiz, which schedules the post for publication on your Instagram account. Conditional logic and error handling ensure reliable execution throughout.

What You'll Need

  1. RunPod API credentials – for accessing the Nano Banana Pro Edit image generation model.
  2. Google Gemini (PaLM) API key – for generating intelligent, viral-ready captions and hashtags.
  3. Postiz API credentials – to connect and publish content directly to your Instagram account.
  4. Google Drive OAuth2 setup – for secure image backup and asset management.
  5. An active n8n instance – either self-hosted or cloud-based, to run the automation.

Quick Setup Guide

  1. Import the template – Download the JSON file and import it into your n8n instance via the workflow import function.
  2. Configure credentials – Set up the four required API connections (RunPod, Google Gemini, Postiz, Google Drive) in n8n's credentials management.
  3. Update node settings – Verify the Form Trigger node fields, check the RunPod API endpoints, confirm your Google Drive folder ID, and replace the placeholder Postiz integration ID.
  4. Test with sample data – Use the pinned test data in the "On form submission" node to simulate a form entry and trigger the workflow.
  5. Activate and monitor – Turn on the workflow, submit a real form entry, and watch the execution flow through each stage in n8n's workflow view.

Pro tip: Start with a private Instagram account or a test social media profile for initial runs. This lets you verify image quality, caption tone, and posting accuracy before automating your main brand channel.
