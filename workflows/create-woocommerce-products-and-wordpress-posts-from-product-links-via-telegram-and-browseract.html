What This Workflow Does

This automation solves a major bottleneck for e-commerce businesses: manually creating product listings and content. When you find a product you want to sell—whether from AliExpress, Amazon, or any supplier—you simply send the link to a Telegram bot. The workflow then handles everything else.

It automatically scrapes product details (title, price, description, images), uses AI to generate compelling sales copy and blog articles, optimizes everything for SEO, and publishes directly to your WooCommerce store and WordPress site. What used to take 30-60 minutes per product now happens in seconds, enabling you to scale your catalog exponentially.

How It Works

1. Product Link Detection via Telegram

You or your team members send product links to a dedicated Telegram bot. The workflow monitors these messages and uses an AI agent to distinguish between casual chat and actionable product links.

2. Automated Data Extraction with BrowserAct

When a valid product link is detected, BrowserAct executes a background browsing session to scrape all relevant product information: title, price, description, images, specifications, and customer reviews.

3. AI-Powered Content Creation

The raw product data is processed by AI (using Google Gemini or OpenAI) acting as a "Senior Copywriter." It generates a high-converting WooCommerce product description with HTML formatting, drafts an engaging blog post review, and creates SEO-friendly filenames and alt-text for all images.

4. Multi-Platform Publishing

The workflow simultaneously creates the product in WooCommerce (with proper categories, pricing, and inventory settings) and publishes the AI-written blog article to WordPress, linking back to the product for maximum SEO benefit.

5. Image Optimization Loop

Finally, the system loops through every product image, updating them in WooCommerce with the newly generated SEO metadata to ensure maximum visibility in image search results.

What You'll Need

  1. WooCommerce store with API access (Consumer Key & Secret)
  2. WordPress site with Application Password configured
  3. Telegram account and bot token (created via BotFather)
  4. BrowserAct account with the "WordPress & WooCommerce Product Management" template saved
  5. AI API access (Google Gemini, OpenAI, or OpenRouter)
  6. n8n instance (cloud or self-hosted) to run the workflow

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance.
  2. Configure credentials for all connected services in n8n's credentials management.
  3. Set up BrowserAct by saving the required template and noting your API key and workflow ID.
  4. Create your Telegram bot via @BotFather and add the token to n8n.
  5. Test the workflow by sending a product link to your Telegram bot and watching it appear in your store and blog.
  6. Customize AI prompts to match your brand voice and content requirements.
