What This Workflow Does

This automation solves a major pain point for YouTube creators and social media marketers: the time-consuming process of repurposing long-form content into engaging Shorts. Manually transcribing videos, identifying key moments, and writing punchy scripts can take hours per video. This workflow eliminates that manual labor.

When you send a YouTube video URL to a Telegram bot, the automation automatically retrieves the transcript, uses Google Gemini AI to analyze the content and generate a script optimized for YouTube Shorts format, then sends the polished script back to you via Telegram. It turns what was a multi-hour task into a 60-second interaction from your phone.

The system is particularly valuable for creators managing multiple channels, agencies producing content for clients, or anyone looking to maintain a consistent Shorts posting schedule without creative burnout.

How It Works

The automation follows a clean, four-step process that mimics how a human editor would create Shorts scripts, but at machine speed.

1. URL Reception via Telegram

The workflow is triggered when you send a YouTube video URL to your configured Telegram bot. This mobile-first approach means you can capture content ideas anywhere—during your commute, while watching other creators, or in planning meetings.

2. Automatic Transcript Extraction

Using the Supadata community node, the system fetches the complete transcript of the YouTube video. This includes not just the words but often timestamps and speaker identification, providing rich material for the AI to work with.

3. AI-Powered Script Generation

Google Gemini analyzes the transcript, identifying key themes, compelling hooks, and natural breaking points. It then generates a script specifically formatted for YouTube Shorts—complete with attention-grabbing opening, concise middle, and clear call-to-action—all within the platform's time constraints.

4. Delivery Back to Telegram

The finished script is sent directly back to your Telegram chat, formatted for easy copying. Some implementations can even include suggested hashtags, tone notes, or alternative opening lines based on the content analysis.

Pro tip: For even better results, train the AI with examples of your most successful Shorts scripts. The system can learn your preferred style, recurring phrases, and what resonates with your specific audience.

  4. Supadata API access for transcript extraction
  5. Basic familiarity with n8n for credential setup and workflow activation

Quick Setup Guide

Getting this automation running takes about 20 minutes if you have your API keys ready. Follow these steps to deploy it in your n8n environment.

  1. Install the community node: In your n8n settings, go to Community Nodes and install "n8n-nodes-supadata"
  2. Configure credentials: Add your Telegram bot token, Gemini API key, and Supadata credentials to n8n's credential manager
  3. Import the template: Download the JSON file from this page and import it into your n8n workspace
  4. Connect the nodes: Assign the correct credentials to each node in the workflow
  5. Test and activate: Send a YouTube URL to your Telegram bot and verify you receive a script back

Pro tip: Start with a few test videos you're familiar with to evaluate the AI's script quality. Adjust the prompt in the Gemini node to better match your voice before running it on important content.
