What This Workflow Does

This n8n template solves the time‑consuming challenge of producing consistent YouTube content. Instead of manually scripting, filming, editing, and uploading videos, you can automate the entire pipeline from a simple text prompt stored in Google Sheets.

The workflow reads a prompt, uses the MagicHour API to generate an AI video, optionally mixes in background music, creates SEO‑friendly metadata with Gemini, uploads the video to YouTube, and logs all results back to your spreadsheet. It’s designed for creators, marketers, and startups who need to scale video production without increasing overhead.

Whether you’re producing daily AI Shorts, product explainers, or repurposing blog posts into video format, this automation turns a batch of text ideas into published YouTube videos—hands‑free.

How It Works

1. Trigger & Prompt Reading

The workflow can be triggered by a new row in Google Sheets or a scheduled cron job (e.g., daily at 9 AM). It reads the text prompt and any optional background music URL from the sheet.

2. AI Video Generation

The prompt is sent to the MagicHour API, which generates a video based on the text description. The workflow polls the API until the render is complete, ensuring reliable delivery.

3. Audio Mixing (Optional)

If a background music URL is provided, the MediaFX community node can blend the audio track with the generated video, creating a polished final product.

4. Metadata Creation

Google Gemini analyzes the prompt and generates an engaging YouTube title, description, and relevant tags—optimizing the video for search and viewer engagement.

5. YouTube Upload

The finished video file, along with its metadata, is uploaded to your YouTube channel via the YouTube API. The video can be set as public, private, or unlisted based on your configuration.

6. Results Logging

Finally, the workflow updates the original Google Sheets row with the YouTube URL, generated metadata, and a status of “Done,” creating a complete audit trail.

What You'll Need

  1. MagicHour API Key – for text‑to‑video generation.
  2. Google Gemini API Key – for prompt parsing and metadata creation.
  3. YouTube OAuth2 Credentials – to upload videos to your channel.
  4. Google Sheets OAuth2 Access – to read prompts and write back results.
  5. Optional: MediaFX Community Node – if you want to mix background audio automatically.

Pro tip: Start with a small batch of 5–10 prompts to test the quality and settings before scaling to hundreds of videos. Adjust the MagicHour parameters (style, duration, aspect ratio) to match your brand’s visual identity.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure credentials for MagicHour, Gemini, YouTube, and Google Sheets in n8n’s credentials panel.
  3. Prepare your Google Sheet with columns for Prompt, Background Music URL (optional), Status, YouTube URL, and Metadata.
  4. Set the trigger – either a Google Sheets trigger for new rows or a Schedule trigger for daily automation.
  5. Test with one prompt and verify the video is generated, uploaded, and logged correctly.
  6. Scale up by adding more prompts to your sheet and letting the automation run on schedule.
