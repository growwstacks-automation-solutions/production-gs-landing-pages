What This Workflow Does

In today's digital landscape, your brand is being discussed across dozens of platforms every hour. Missing these conversations means missing customer feedback, PR risks, and competitive intelligence. Manual monitoring is impossible at scale.

This automation solves that by acting as your 24/7 digital listening post. It systematically searches Reddit, LinkedIn, X (Twitter), and Instagram for mentions of your defined keywords—your brand name, products, competitors, or industry terms. It captures every post and comment, stores them in a structured database, uses AI to analyze sentiment and emerging themes, and delivers a comprehensive intelligence report straight to your inbox.

You move from reactive firefighting to proactive brand management, with data-driven insights to guide marketing, product development, and customer service decisions.

How It Works

The automation follows a logical pipeline to transform raw social data into actionable intelligence.

1. Keyword Loading & Platform Search

The workflow starts by reading your monitored keywords from a central database. For each keyword, it queries the AnySite API endpoints for Reddit, LinkedIn, X, and Instagram simultaneously, searching for recent posts containing those terms.

2. Data Deduplication & Storage

Before saving anything, it checks if a post already exists in your historical database to avoid duplicates. New posts are stored with rich metadata: platform, post content, engagement metrics, author info, and direct links.

3. Comment Enrichment

For platforms like Reddit and LinkedIn, the workflow fetches the full conversation thread—all comments and replies. These are formatted and attached to the main post record, providing context for deeper analysis.

4. AI-Powered Analysis

Here's where the magic happens. An AI agent (using OpenAI's GPT) reviews all collected posts. It categorizes sentiment, identifies key themes, flags potential crises, highlights high-engagement content, and summarizes the day's conversation in natural language.

5. Report Generation & Delivery

The AI creates a polished HTML report with executive summary, sentiment charts, top posts, and recommendations. This report is automatically sent via Gmail to your designated team members, ready for their morning review.

Pro tip: Schedule this workflow to run overnight. Your team wakes up to a complete brand intelligence briefing without spending a minute on manual research.

What You'll Need

  1. An n8n instance (self-hosted or n8n.cloud account)
  2. AnySite API key for accessing social media data
  3. OpenAI API key (GPT-4o or later recommended) for AI analysis
  4. Gmail account connected via OAuth2 for sending reports
  5. Your brand keywords list (brand names, product names, competitor names, industry terms)

Quick Setup Guide

Get your brand monitoring system running in under 30 minutes:

  1. Import the template into your n8n instance using the downloaded JSON file.
  2. Configure credentials in n8n for AnySite API (add your access token), OpenAI API, and Gmail OAuth2.
  3. Create two Data Tables in n8n: "Brand Monitoring Words" (for keywords) and "Brand Monitoring Posts" (for collected data).
  4. Add your keywords to the "Brand Monitoring Words" table—one keyword per row.
  5. Test the workflow manually with one keyword to verify data collection and email delivery.
  6. Set the schedule trigger to run daily (or as frequently as needed) and activate the workflow.

Pro tip: Start with 5-10 core keywords. Once you see the value, expand to include product features, executive names, and competitor terms for comprehensive coverage.
