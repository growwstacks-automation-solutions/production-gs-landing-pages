Automate Multilingual News Curation with Google Gemini

AI-powered workflow to translate and organize RSS news into Google Sheets with Slack alerts

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automated solution revolutionizes how businesses monitor international news by combining AI-powered translation with efficient content organization. The workflow fetches updates from RSS feeds in multiple languages, processes them through Google Gemini for accurate translation and summarization, then stores the curated content in Google Sheets while sending Slack notifications for immediate awareness.

By automating what would normally be hours of manual work, this system enables real-time monitoring of global news sources without language barriers. The structured output in Google Sheets allows for easy analysis, trend spotting, and reporting - making it invaluable for international businesses, research teams, and media monitoring services.

How It Works

1. RSS Feed Processing

The workflow begins by fetching the latest updates from configured RSS feeds. You can specify multiple sources and set filters to only process articles matching your keywords or categories.

2. AI-Powered Translation

Using Google Gemini's advanced language models, the system translates content while preserving context and nuance. The workflow can handle multiple target languages simultaneously (English, Chinese, Korean in this template).

3. Content Organization

Translated articles are structured with metadata (title, summary, source URL) and stored in Google Sheets with separate columns for each language version, creating an easily searchable knowledge base.

4. Team Notification

Slack notifications deliver translated highlights to your team channels, ensuring immediate awareness of important developments without needing to check the spreadsheet manually.

What You'll Need

  1. Google Gemini API access
  2. Google Sheets with prepared columns (title, summary, location, en, zh, ko, url)
  3. Slack workspace with appropriate channel access
  4. RSS feed URLs you want to monitor
  5. n8n instance (cloud or self-hosted)

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance
  3. Configure your RSS feed URLs in the RSS Read node
  4. Set up Google Gemini API credentials
  5. Connect your Google Sheets document
  6. Configure Slack webhook for notifications
  7. Test with a manual trigger before scheduling

Pro tip: For better translation quality, customize the AI prompts in the Google Gemini nodes to include industry-specific terminology.
