What This Workflow Does

This automation solves the time-consuming challenge of manually gathering and distributing tech news updates across teams. Instead of employees wasting hours scanning multiple news sites each morning, this workflow automatically collects, summarizes, and delivers relevant tech updates directly to your Slack or Telegram channels.

The system combines BrowserAct's web scraping capabilities with OpenRouter's AI summarization to create concise, valuable daily briefings. It eliminates information overload by filtering noise and delivering only the most relevant updates tailored to your team's interests.

How It Works

1. Automated News Collection

BrowserAct visits predefined tech news sources at scheduled intervals, extracting article headlines and content. The workflow can monitor multiple sites simultaneously, ensuring comprehensive coverage without manual browsing.

2. AI-Powered Summarization

OpenRouter processes the collected articles, identifying key points and generating executive summaries. This step transforms raw information into actionable insights, saving readers hours of reading time.

3. Smart Filtering

The system applies customizable filters to prioritize news matching your team's focus areas. Keywords, sentiment analysis, and relevance scoring ensure only the most valuable updates proceed to delivery.

4. Multi-Channel Distribution

Finalized briefings automatically post to designated Slack channels or Telegram groups at optimal times. The formatted messages include headlines, summaries, and direct links for deeper reading.

What You'll Need

  1. n8n instance (cloud or self-hosted)
  2. BrowserAct credentials
  3. OpenRouter API key
  4. Slack webhook URL or Telegram bot token
  5. List of target news sources to monitor

Pro tip: Start with 3-5 high-quality news sources, then expand as you refine your filters. Too many sources initially can create noise.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure BrowserAct nodes with your target news URLs
  3. Set up OpenRouter API connection with your preferred AI model
  4. Add your Slack/Telegram credentials to the delivery nodes
  5. Test with a small set of articles to verify formatting
  6. Schedule the workflow to run daily at your preferred time
