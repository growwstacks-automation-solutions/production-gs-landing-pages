Customer Insights with Qdrant & AI

Automatically cluster and analyze customer reviews using vector embeddings and AI to uncover actionable themes and sentiment patterns.

What This Workflow Does

Businesses drown in customer feedback—reviews, surveys, support tickets—but manually analyzing thousands of comments is slow and inconsistent. This workflow automates the entire process: it scrapes reviews (like Trustpilot), converts them into semantic vector embeddings using AI, clusters similar feedback together with Qdrant, and then uses an AI agent to generate clear insights for each cluster.

Instead of spending weeks reading reviews, marketers and product teams get automated reports highlighting recurring positives (e.g., "fast shipping," "helpful support") and negatives (e.g., "complicated setup," "price increases"). You can compare your clusters against competitors, track sentiment over time, and turn raw feedback into strategic decisions.

The template is part of a series exploring vector‑embedding clustering for business intelligence. It demonstrates how AI can structure unstructured data at scale, saving time and revealing patterns humans might miss.

How It Works

Step 1: Collect Reviews

The workflow starts with an HTTP Request node that scrapes customer reviews from Trustpilot (or any other source). It fetches raw text, dates, ratings, and other metadata. This data is cleaned and prepared for embedding.

Step 2: Create Vector Embeddings

Each review is sent to an embedding model (like OpenAI's) to convert the text into a numerical vector that captures its semantic meaning. These vectors are stored in a Qdrant collection, tagged with metadata for later retrieval.

Step 3: Cluster Similar Reviews

A Python Code node runs a clustering algorithm (like K‑means or DBSCAN) on the stored vectors. Qdrant returns points grouped by similarity, automatically identifying themes—for example, all reviews mentioning "delivery speed" cluster together.

Step 4: Generate AI Insights

The workflow loops through each cluster, fetches the original review texts, and feeds them to an AI agent (OpenAI Chat Model) with a prompt like "Summarize the common themes and sentiment in these reviews." The AI outputs concise insights for each group.

Step 5: Output to Spreadsheet

The final insights—along with raw cluster data—are saved to a Google Spreadsheet using the Google Sheets node. This creates a live report that teams can use for analysis, presentation, or further dashboard integration.
