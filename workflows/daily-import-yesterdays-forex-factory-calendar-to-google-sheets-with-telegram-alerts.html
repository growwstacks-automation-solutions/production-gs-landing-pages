What This Workflow Does

For forex traders and analysts, staying updated with economic calendar events is crucial but time-consuming. Manually checking Forex Factory every day, recording data, and monitoring for high-impact events consumes valuable trading hours and introduces human error.

This automation solves that problem by running daily on a schedule to fetch yesterday's complete Forex Factory calendar via Rapid API. It extracts key data fields including currency, time, impact level, actual vs. forecast values, and market indicators, then organizes this information systematically for analysis and historical tracking.

The workflow doesn't just collect data—it intelligently processes it. High and medium impact events trigger immediate Telegram alerts to your mobile device, while all data gets saved to designated Google Sheets for easy access, sharing, and integration with your existing trading analysis tools.

How It Works

1. Daily Scheduled Trigger

The workflow activates automatically each day using n8n's schedule trigger. You configure the specific time (typically before market open) to ensure you have fresh data ready for your trading session.

2. API Data Fetch from Forex Factory

Using the Rapid API connection, the workflow requests yesterday's complete economic calendar. It handles authentication, error recovery, and data parsing to extract structured information from the API response.

3. Data Processing and Filtering

The raw API data gets cleaned and organized. The workflow separates events by impact level (high, medium, low), converts timestamps to your preferred timezone, and structures the data for easy consumption.

4. Google Sheets Integration

Processed data automatically appends to your connected Google Sheets. The workflow creates separate sheets or tabs for different impact levels, maintaining a clean, historical record that grows daily without manual intervention.

5. Intelligent Alert System

High and medium impact events trigger customized Telegram notifications sent directly to your phone or trading group. These alerts include key details like currency, event, actual vs. forecast values, and potential market implications.

Who This Is For

This automation is ideal for retail forex traders who want to systematize their market analysis, trading analysts who need reliable historical data for strategy development, investment teams requiring shared economic calendar tracking, and financial educators who demonstrate market reactions to economic events.

It's particularly valuable for traders who manage multiple currency pairs, those who trade around specific economic releases, and anyone who has missed trading opportunities because they weren't alerted to important calendar events in time.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. Rapid API account with access to a Forex Factory calendar API (multiple providers available)
  3. Google Sheets document with write permissions configured via Google Service Account or OAuth
  4. Telegram bot token and chat ID for receiving alerts (free to set up via BotFather)
  5. Basic understanding

Pro tip: Set your schedule trigger to run 30 minutes before your primary trading session begins. This gives you fresh data without last-minute rushing, and allows time to review high-impact events before markets react.

Quick Setup Guide

Follow these steps to implement this automation in under 15 minutes:

  1. Download and import the JSON template into your n8n instance
  2. Create your Google Sheet with "High Impact" and "All Events" tabs (or customize tab names in the workflow)
  3. Set up Rapid API credentials in the HTTP Request node (you'll need your API key from the provider)
  4. Configure Telegram bot by adding your bot token and chat ID to the Telegram node
  5. Adjust the schedule to run at your preferred time (default is 6 AM UTC)
  6. Test and activate the workflow with a manual run before enabling the schedule
