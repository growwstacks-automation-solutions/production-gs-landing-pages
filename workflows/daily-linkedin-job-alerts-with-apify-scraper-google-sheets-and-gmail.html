What This Workflow Does

Job hunting is time-consuming and repetitive. Manually checking LinkedIn for new postings every day drains hours that could be spent on applications and interviews. This automation solves that by turning your passive search into an active, automated system.

The workflow runs daily to scrape LinkedIn for jobs matching your specific keywords, location, and other criteria. It captures job titles, companies, locations, salaries (when available), and application links. This data is then automatically saved to a Google Sheet for tracking and simultaneously emailed to you in a clean, formatted digest.

Instead of browsing multiple times daily, you receive one consolidated email with all relevant new opportunities. This ensures you never miss a posting while reclaiming 5-10 hours per week previously spent on manual searching.

How It Works

The automation follows a logical sequence that mimics what a human would do, but with perfect consistency and speed.

1. Scheduled Trigger

Each day at noon (configurable), the workflow automatically activates. This scheduled trigger ensures fresh job data arrives consistently without manual intervention.

2. LinkedIn Data Scraping via Apify

The workflow sends a request to Apify's LinkedIn Jobs Scraper actor. You configure search parameters like keywords (using Boolean operators like AND/NOT), location, date posted, and result limits. Apify handles the complex scraping and returns structured job data.

3. Data Processing & Formatting

A code node processes the raw data, extracting key fields and building HTML for email presentation. It handles edge cases like missing salary information or work arrangement details (remote/hybrid/onsite) with conditional logic.

4. Google Sheets Storage

Processed job data is appended to your designated Google Sheet. Each entry includes timestamp, job title, company, location, salary range, and direct application link. This creates a searchable historical database of all opportunities.

5. Email Delivery via Gmail

The formatted job listings are sent to your email with a customized subject line including the current date. The email presents each opportunity clearly with all relevant details and direct links for immediate application.

What You'll Need

  1. Apify Account: Free tier available for limited scraping. Required to access the LinkedIn Jobs Scraper actor.
  2. Google Cloud Project: With Google Sheets API and Gmail API enabled to connect n8n to your Google services.
  3. Google Sheets Document: Pre-created with headers for job data storage (title, company, location, salary, link, date).
  4. n8n Instance: Either self-hosted or cloud version with community nodes installed, including the Apify node.
  5. Clear Search Criteria: Defined keywords, location preferences, and other filters for targeted results.

Pro tip: Start with broad keywords, then refine based on results. Use LinkedIn's own search to test parameters before automating. Boolean operators (AND, OR, NOT, parentheses) dramatically improve result relevance.

Quick Setup Guide

Implement this automation in under 30 minutes with these straightforward steps.

  1. Download and Import: Download the template file and import it into your n8n instance via the workflow import function.
  2. Configure Apify Node: Add your Apify API token and customize the JSON search parameters (keywords, location, limit, date_posted).
  3. Connect Google Services: Set up OAuth credentials for Google Sheets and Gmail in n8n, then connect to your specific spreadsheet and email address.
  4. Adjust Schedule: Modify the schedule trigger to your preferred time (consider time zones if searching in multiple locations).
  5. Test and Activate: Run the workflow once manually to verify data flow, then activate the schedule for daily automation.
