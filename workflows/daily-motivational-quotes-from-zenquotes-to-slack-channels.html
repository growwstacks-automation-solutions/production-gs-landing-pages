What This Workflow Does

Maintaining team morale and fostering a positive work environment is crucial for productivity and retention, but busy managers often lack the time to consistently provide inspiration. This automated workflow solves that problem by delivering a fresh, thought-provoking motivational quote to your team's Slack channel every single morning.

The system connects the free ZenQuotes API with your Slack workspace, creating a seamless pipeline of inspiration that requires zero ongoing maintenance. Every day at 8 AM (or your chosen time), it fetches a new quote, formats it for maximum impact, and posts it directly to your designated channel—keeping your team motivated without adding to anyone's workload.

Beyond simple automation, this workflow demonstrates how small, consistent gestures can significantly impact workplace culture. It shows your team you care about their mindset and creates a daily touchpoint that reinforces company values and shared purpose.

How It Works

This elegant automation follows a straightforward four-step process that runs completely autonomously once configured.

1. Scheduled Trigger

A Cron node acts as the workflow's internal clock, configured to execute daily at 8:00 AM EST. This time-based trigger ensures consistent, reliable delivery without manual intervention. You can easily adjust the schedule to match your team's timezone or preferred delivery time.

2. Fetch Fresh Quote

The workflow sends an HTTP Request to the ZenQuotes.io API, which returns a random motivational quote with author attribution. The API requires no authentication or API keys, making setup incredibly simple while providing unlimited, high-quality content for team inspiration.

3. Format for Impact

A Code node processes the raw API response, extracting the quote text and author, then structures it into an engaging Slack message format. This step adds emojis, proper formatting, and sometimes additional context to make the quote more impactful and visually appealing in your team's channel.

4. Deliver to Slack

The formatted message is sent to your chosen Slack channel via the Slack node. The integration uses Slack's API to post as your app, maintaining a consistent presence. You can customize which channel receives the quotes, whether it's your general channel, a dedicated #inspiration channel, or department-specific spaces.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. Slack workspace where you have permission to install apps
  3. Slack app credentials (Bot User OAuth Token) with chat:write and channels:read permissions
  4. Target Slack channel (default is #general, but any channel works)
  5. 5 minutes for initial setup and configuration

Pro tip: Create a dedicated #daily-inspiration or #team-culture channel for these quotes to keep your main channels focused while giving team members who want daily motivation a dedicated space to engage.

Quick Setup Guide

Follow these steps to implement this automation in your workspace:

  1. Download the template using the button above and import it into your n8n instance.
  2. Create a Slack app at api.slack.com/apps with chat:write and channels:read OAuth scopes.
  3. Install to workspace and copy the Bot User OAuth Token into the Slack node credentials.
  4. Update the channel name in the Slack node if you want to post somewhere other than #general.
  5. Adjust the schedule in the Cron node if 8 AM EST doesn't match your team's timezone.
  6. Activate the workflow and test with a manual trigger to ensure everything works.
  7. Monitor for a few days to see engagement and consider customizing the message format based on team feedback.
