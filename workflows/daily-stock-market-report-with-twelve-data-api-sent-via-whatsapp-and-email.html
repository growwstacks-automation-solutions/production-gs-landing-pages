What This Workflow Does

Manually tracking daily stock market movements is time-consuming and prone to human error. Financial analysts, investment teams, and individual traders spend hours each day collecting data, calculating gains/losses, and distributing reports—delaying critical decision-making.

This n8n workflow automates the entire process. It connects to the Twelve Data API to fetch real-time prices for the top 100 stocks, automatically identifies the top 5 gainers and losers based on percentage change, formats professional reports, and delivers them instantly via WhatsApp and email to your designated recipients. The workflow runs on a daily schedule (Monday-Friday at market close), ensuring your team receives consistent, accurate market intelligence without manual effort.

The workflow canvas showing data flow from Twelve Data API through processing nodes to WhatsApp and email outputs

How It Works

1. Scheduled Trigger

The workflow activates automatically every weekday at 5:00 PM IST (market close time). You can customize this schedule to match your preferred reporting time or market hours.

2. Data Fetching

It connects to the Twelve Data API using your API key and retrieves real-time price data for your configured list of stock symbols (default: top 100 stocks). The API returns current prices, previous close, volume, and percentage changes.

3. Data Processing

The workflow sorts all stocks by their daily percentage change, identifies the top 5 gainers and top 5 losers, and calculates key metrics like total market movement and sector performance.

4. Report Formatting

It creates two formatted reports: a concise version for WhatsApp (with emojis and key highlights) and a detailed version for email (with tables, charts, and analysis). Both include timestamps and actionable insights.

5. Multi-Channel Delivery

The workflow simultaneously sends the WhatsApp alert to your business WhatsApp number's contact list and emails the detailed report to your specified distribution list. Delivery confirmations are logged for tracking.

What You'll Need

  1. Twelve Data API key (free tier available with limited calls, paid plans for higher volume)
  2. n8n instance (self-hosted or n8n.cloud account)
  3. WhatsApp Business API access or WhatsApp-enabled messaging service
  4. Email service credentials (Gmail, SMTP, or email service provider)
  5. List of stock symbols you want to track (default includes top 100, customizable)
  6. Recipient lists for WhatsApp and email distribution

Quick Setup Guide

Follow these steps to implement this automation in under 30 minutes:

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure the Twelve Data node with your API key and adjust the stock symbols if needed.
  3. Set up the WhatsApp node with your business account credentials and test recipient.
  4. Configure the email node with your SMTP details and primary distribution list.
  5. Adjust the schedule trigger to match your preferred reporting time (default: 5 PM IST weekdays).
  6. Execute a test run with a smaller symbol list to verify data flow and alert delivery.
  7. Activate the workflow and monitor the first few automated reports for accuracy.
