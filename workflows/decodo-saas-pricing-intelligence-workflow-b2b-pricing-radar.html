What This Workflow Does

This automation solves a critical business problem: keeping up with competitor pricing in the fast-moving SaaS market. Manual price tracking is slow, inconsistent, and prone to errors, leaving businesses reacting to market changes instead of anticipating them.

The workflow creates a complete pricing intelligence pipeline. It starts by scraping live competitor pricing pages using Decodo's managed web scraping API, which handles JavaScript rendering and anti-bot protection. The raw HTML is then cleaned and processed by an AI agent that extracts structured pricing data—plan names, prices, billing periods, and key features—transforming messy web content into organized JSON.

Finally, the system generates an executive-ready pricing brief in Google Docs, converts it to PDF, and shares it directly to a designated Slack channel. This turns what was traditionally a days-long manual research process into a minutes-long automated operation, giving your team timely, consistent intelligence for strategic decisions.

How It Works

1. Manual Trigger & Configuration

The workflow begins with a manual trigger, ensuring intentional execution when you need fresh intelligence—ideal for monitoring specific pricing updates, launches, or quarterly reviews. You configure the target competitor URL and report metadata.

2. Decodo Pricing Page Scraping

Decodo renders the full pricing page using real browser emulation, capturing dynamic content, tier cards, modals, and JavaScript-heavy elements that traditional scraping misses. This ensures accurate representation of the actual customer experience.

3. HTML Normalization & Cleaning

Custom code removes scripts, styles, and irrelevant page elements, leaving clean, readable text optimized for AI processing. This step dramatically improves the accuracy of subsequent data extraction.

4. AI-Powered Plan Extraction

An LLM agent analyzes the cleaned text and extracts structured pricing data into a consistent JSON schema. It identifies plan hierarchies, pricing models (monthly/annual), feature lists, and positioning notes with human-like understanding but machine consistency.

5. Executive Brief Generation

The structured data transforms into a professional markdown brief featuring plan comparison tables, key differentiators, observations, and suggested next actions—formatted for immediate executive consumption.

6. Document Publishing & Distribution

The brief creates as a Google Doc in your designated Drive folder, converts to PDF for easy sharing, and uploads directly to your Slack #pricing-intel channel (or similar), ensuring instant team visibility without manual forwarding.

What You'll Need

  1. Decodo account with API access (Advanced Scraping API plan recommended)
  2. OpenAI or compatible LLM credentials for the plan extraction agent
  3. Google Workspace with Drive API enabled and a designated folder for reports
  4. Slack workspace with appropriate permissions for file uploads to your chosen channel
  5. n8n instance (cloud or self-hosted) with HTTP Request, Code, and Set nodes available

Pro tip: Start with 2-3 key competitors rather than trying to monitor everyone at once. Focus on direct competitors with similar target customers and feature sets for the most actionable insights.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance.
  2. Configure the Decodo node with your API credentials and test with a sample pricing URL.
  3. Set up your LLM connection (OpenAI, Anthropic, or local model) in the Agent node.
  4. Connect Google Drive and specify the folder where pricing briefs should be created.
  5. Add your Slack credentials and specify the channel for PDF distribution.
  6. Test the complete flow with a known competitor URL and verify the output in Slack.
  7. Schedule or trigger manually based on your intelligence needs—weekly for fast-moving markets, monthly for stable ones.
