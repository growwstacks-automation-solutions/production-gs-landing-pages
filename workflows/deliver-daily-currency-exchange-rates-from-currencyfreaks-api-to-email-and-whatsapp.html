Automated workflow to deliver daily multi-currency exchange rate updates via API to email and WhatsApp

What This Workflow Does

This automated n8n workflow solves the challenge of manually tracking currency exchange rates for businesses operating internationally. It delivers daily multi-currency exchange rate updates from the CurrencyFreaks API directly to your email and WhatsApp, ensuring your team always has access to accurate, up-to-date rates.

For businesses dealing with international transactions, suppliers, or customers, currency fluctuations can significantly impact costs and pricing. This workflow eliminates the need for manual rate checking, reducing errors and saving valuable time. The system fetches the latest exchange rates, formats them for clarity, and distributes them through both email and WhatsApp for maximum accessibility.

How It Works

1. API Rate Fetching

The workflow begins by connecting to the CurrencyFreaks API to retrieve the latest exchange rates for your specified currencies. It handles authentication and processes the JSON response to extract the relevant rate data.

2. Data Formatting

The raw API data is transformed into a clean, readable format. The system calculates percentage changes from previous rates, highlights significant movements, and organizes the data for easy scanning.

3. Multi-Channel Distribution

The formatted rates are simultaneously sent via email to designated recipients and as WhatsApp messages to specified numbers. This dual delivery ensures team members can access the information through their preferred channel.

4. Error Handling

The workflow includes robust error handling to manage API outages or communication failures. If rates can't be fetched, it will retry according to your configuration and send alerts if persistent issues occur.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. CurrencyFreaks API credentials (free tier available)
  3. Email service configured in n8n (SMTP or service like SendGrid)
  4. WhatsApp Business account or Twilio configured for WhatsApp
  5. List of currencies to track and recipient contacts

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure your CurrencyFreaks API credentials in the HTTP Request node
  3. Set up your email service in the Email node
  4. Configure WhatsApp integration in the WhatsApp node
  5. Customize the currency list and recipient information
  6. Test the workflow and schedule it to run daily
