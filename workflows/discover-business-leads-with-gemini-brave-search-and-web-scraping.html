What This Workflow Does

Manual lead generation is a tedious, time-consuming bottleneck for sales and marketing teams. This AI-powered workflow automates the initial prospecting phase, transforming hours of web searching into minutes of automated discovery.

Given a business type and location (e.g., "artisan bakery in Portland"), the system intelligently uses multiple search strategies to identify potential companies, find their websites, and perform quick relevance checks. It outputs a structured JSON list of prospects, ready for your CRM or outreach tools.

Think of it as a dedicated research assistant that never sleeps, scanning the web for businesses that match your ideal customer profile and delivering validated contact points.

How It Works

The workflow is built around a central AI agent that orchestrates several specialized tools to find and validate leads.

Step 1: Input & Analysis

You provide the target criteria: business type, location, desired number of prospects, and optional keywords. The AI agent interprets this request and formulates an intelligent search strategy.

Step 2: Multi-Source Discovery

The agent uses Brave Search for broad, privacy-focused web results and Google Gemini Search for context-aware discovery, finding businesses mentioned across articles, directories, and local listings.

Step 3: Validation & Enrichment

For each potential lead, the system performs quick checks. It may use Brave Local Search to confirm location or the Jina AI Web Scraper to rapidly scan a website's content for keywords, ensuring relevance before adding it to the final list.

Step 4: Structured Output

The workflow compiles findings into a clean JSON array containing business names and website URLs. This data can be instantly passed to another workflow for enrichment or imported into your sales pipeline.

Pro tip: This workflow is designed as a "tool" for larger AI agents. You can call it from a master agent that handles lead qualification, outreach sequencing, or CRM updates, creating a fully automated sales development system.

  2. Google Gemini API credentials (for the AI chat model and search tool).

Quick Setup Guide

Get this automation running in your n8n instance in under 15 minutes.

  1. Download & Import: Click the "Download Template" button and import the JSON file into your n8n workspace.
  2. Configure Credentials: Set up the required API credentials for Google Gemini, Brave Search, and Jina AI in n8n's credentials management.
  3. Assign Credentials to Nodes: Link the new credentials to the corresponding nodes in the workflow.
  4. Test the Trigger: Use the "Execute Workflow Trigger" node to manually test with sample inputs (business type, location).
  5. Integrate: Connect the output to your CRM, spreadsheet, or a follow-up automation workflow.
