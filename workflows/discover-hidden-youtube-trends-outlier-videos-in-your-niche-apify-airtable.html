What This Workflow Does

This automation solves the tedious, time-consuming process of manually researching YouTube trends in your niche. Content creators, marketers, and YouTube automation teams spend hours each week browsing search results, watching videos, and trying to spot emerging patterns. This workflow automates that entire process.

It systematically scrapes YouTube search results for your target keywords using Apify, cleans and summarizes video content with AI, and stores everything in an organized Airtable database. You get weekly updates on outlier videos—those with unusual growth, engagement, or content patterns—that signal hidden trends before they become mainstream.

How It Works

Step 1: Keyword Retrieval & Scheduling

A scheduled trigger runs weekly (or at your chosen frequency) and retrieves your predefined keyword list. These keywords represent the niche topics you want to monitor. The workflow automatically adjusts date filters to capture recent videos.

Step 2: Apify YouTube Scraping

For each keyword, the workflow calls Apify's YouTube scraper to collect videos filtered by relevance. It waits for the dataset completion, ensuring all relevant videos are captured. This replaces manual browsing across multiple search queries.

Step 3: Data Processing & AI Summarization

Each video's metadata (title, views, engagement, upload date) is extracted. The workflow then cleans video scripts and uses AI to summarize key themes, sentiment, and talking points. This transforms raw data into actionable insights.

Step 4: Airtable Database Upload

All processed data—video metadata, summaries, and trend indicators—is uploaded to your Airtable base. Records are categorized by keyword, date, and outlier status, creating a searchable, visual database for analysis.

Pro tip: Start with 5-10 niche-specific keywords. Monitor results for 2-3 weeks, then refine your list based on which keywords yield the most relevant outlier videos.

Quick Setup Guide

  1. Import the template: Download the JSON file and import into your n8n workspace.
  2. Configure the schedule: Set your preferred frequency (weekly recommended) in the Schedule Trigger node.
  3. Add your keywords: Replace the sample keywords in the "Setup Keywords" node with your niche-specific terms.
  4. Connect Apify: Insert your Apify API token in all HTTP Request nodes' URL fields.
  5. Set up Airtable: Connect your Airtable credentials and map fields to your base structure.
  6. Test and run: Execute once manually to verify data flow, then activate the schedule.

Pro tip: For initial testing, limit to 2-3 keywords and a 1-day date filter. This reduces API calls and lets you verify the workflow quickly before scaling up.
