What This Workflow Does

This automation creates an intelligent Telegram chatbot that answers questions by searching through your documents using Google Gemini AI and Supabase vector search. It eliminates manual document lookup and provides instant, accurate responses to customer or employee queries.

The system handles the complete workflow: receiving Telegram messages, processing questions with AI, searching document embeddings in Supabase, generating natural language answers, and responding conversationally. It's particularly valuable for customer support, internal knowledge bases, and educational content delivery.

How It Works

1. Document Processing

Your documents are first processed into vector embeddings using Google Gemini's text embedding capabilities. These embeddings capture semantic meaning and are stored in Supabase's vector database for fast similarity searches.

2. Telegram Message Handling

When a user sends a question via Telegram, the workflow triggers. The message content is analyzed to determine intent and extract key concepts for document search.

3. Vector Search Execution

The system queries Supabase's vector database to find document sections most relevant to the question. This semantic search goes beyond keyword matching to understand contextual meaning.

4. AI Answer Generation

Gemini AI synthesizes the retrieved document content into a natural language response tailored to the specific question. The response maintains brand voice and includes citations when appropriate.

What You'll Need

Quick Setup Guide

  1. Import the workflow JSON into your n8n instance

  6. Test with sample questions and refine response prompts
