What This Workflow Does

This automation solves the tedious, manual process of creating and managing shortened URLs for marketing campaigns, social media, and customer communications. Instead of copying long URLs into Dub.co's interface repeatedly, this workflow automatically generates branded short links using your custom domain, applies consistent tracking parameters, and even updates existing links if the slug is already taken.

The template connects directly to Dub.co's API, allowing you to programmatically create short links as part of larger automation sequences. Whether you're generating links for email newsletters, social media posts, or internal documentation, this workflow ensures every link is consistent, trackable, and professionally branded.

How It Works

The workflow follows a logical sequence to validate input, check for existing links, and create or update shortened URLs with your preferred settings.

1. Input & Validation

The workflow starts by receiving a long URL through a manual trigger or from another automation. It validates the URL format and extracts key parameters like your desired custom slug, project selection, and any UTM tracking parameters you want to include automatically.

2. API Authentication & Configuration

The "API Auth" node securely connects to your Dub.co account using your API key. You configure your default project, custom domain preferences, and other settings like link expiration or password protection if needed. These settings are stored securely and applied consistently to every link.

3. Link Creation & Management

The workflow checks if your desired short slug already exists in your Dub.co account. If it does, it can update the existing link with the new destination URL—perfect for when you need to fix broken links or redirect to updated content. If the slug is available, it creates a new short link with all your configured settings.

4. Output & Integration

The final shortened URL is output in a clean format, ready to be used in your emails, social media posts, or other marketing materials. The workflow can also log the creation details to a spreadsheet or database for tracking purposes.

Quick Setup Guide

Follow these steps to get your URL shortening automation running in under 15 minutes:

  1. Download the template using the button above and save the JSON file to your computer.
  2. Import into n8n by going to your n8n dashboard, clicking "Workflows" → "Import from file" and selecting the downloaded JSON.
  3. Configure the API Auth node by double-clicking it and entering your Dub.co API key. Set your default project slug and any other preferences.
  4. Test the workflow by clicking "Execute Workflow" and providing a sample long URL. Verify that a short link is created in your Dub.co account.
  5. Connect to triggers by replacing the manual trigger with webhooks, schedule triggers, or connections to other apps like Google Sheets or your CRM.
