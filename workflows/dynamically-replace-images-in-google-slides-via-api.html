Automate visual updates in presentations without manual editing

What This Workflow Does

This workflow solves the tedious process of manually updating images in Google Slides presentations. It creates an API endpoint that automatically replaces specified images in your slides with new versions while maintaining all formatting and positioning.

Common use cases include updating client logos in pitch decks, refreshing product screenshots in sales materials, or rotating background images in templates. The automation saves marketing and sales teams hours of manual work while ensuring brand consistency across all presentations.

How It Works

1. Setup Key Identifiers

Add unique alt text identifiers to images in your Google Slides that you want to replace programmatically. These act as reference points for the automation.

2. API Endpoint Creation

The workflow establishes a secure API endpoint that accepts presentation IDs, image keys, and new image URLs as parameters.

3. Image Replacement Logic

When triggered, the workflow uses the Google Slides API to locate images by their alt text keys and replaces them with the new images while preserving all formatting.
