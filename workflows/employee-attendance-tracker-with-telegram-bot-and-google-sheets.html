What This Workflow Does

Manual employee attendance tracking is a time-consuming, error-prone administrative task. This automation solves that by creating a simple, digital check-in system using tools your team already uses. Employees interact with a Telegram bot on their phone or computer to log their start and end times. Each action is instantly recorded in a Google Sheet, creating a clean, timestamped log.

The workflow validates the employee, prevents duplicate entries for the same day, and can even show the employee their current attendance status. This replaces paper sign-in sheets, shared spreadsheets that get messed up, and clunky dedicated time-clock software, providing a streamlined, accurate, and accessible system for managers and HR.

How It Works

The automation follows a logical sequence to ensure accuracy and ease of use.

1. Employee Initiates Action

An employee opens the Telegram bot and selects an option like "Check-in" or "Check-out" from a menu. This sends a trigger to the n8n workflow.

2. Identity & Validation

The workflow checks the employee's Telegram username against a pre-approved list in a Google Sheets "Employee" tab. This ensures only authorized staff can log attendance.

3. Duplicate Check

The system checks the "Attendance" sheet to see if the employee has already performed that action (check-in/out) for the current day, preventing double entries.

4. Data Logging

If valid, a new row is added to the Google Sheets "Attendance" tab with the employee ID, date, timestamp, and action type (In/Out).

5. Confirmation

The Telegram bot instantly replies to the employee with a confirmation message (e.g., "Check-in recorded at 9:05 AM").

Pro tip: Use the provided Google Sheets template as your starting point. It has the correct column structure ("Employee" and "Attendance" sheets) so the workflow works immediately after you populate your team list.

What You'll Need

  1. An n8n instance (Cloud or Self-Hosted).
  2. A Telegram Bot Token created via BotFather on Telegram.
  3. A Google Sheets spreadsheet (use the provided template).
  4. Google Service Account credentials to allow n8n to write to your Sheet.
  5. A list of employees with their Telegram usernames to populate the "Employee" sheet.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n workspace.
  2. Copy the Google Sheet: Use the provided template link to create your own copy. Add your employee data to the "Employee" sheet.
  3. Configure Credentials: In n8n, set up credentials for Telegram (using your bot token) and Google Sheets (using your service account JSON key).
  4. Update Node Settings: In the workflow, update the Google Sheets nodes to point to your new spreadsheet's ID and sheet names.
  5. Activate & Test: Turn the workflow on and send a "/start" command to your Telegram bot. Test check-in/out with a sample employee.
