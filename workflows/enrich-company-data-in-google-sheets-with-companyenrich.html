Automatically enhance your prospect lists with valuable firmographic data without leaving your spreadsheet

What This Workflow Does

This n8n workflow solves the tedious manual process of researching company details by automatically enriching domain names in your Google Sheets with comprehensive firmographic data from CompanyEnrich. Instead of spending hours looking up individual companies on LinkedIn, Crunchbase, or corporate websites, this automation pulls verified data directly into your spreadsheet.

The workflow reads domain names from a specified Google Sheet column, queries the CompanyEnrich API for each domain, then writes back enriched data like industry classification, employee count, estimated revenue, technology stack, and social media profiles. This transforms basic contact lists into powerful targeting datasets for sales, marketing, and recruitment teams.

How It Works

1. Trigger from Google Sheets

The workflow starts by reading domain names from a specified column in your Google Sheet. You can configure it to run on a schedule (daily/weekly) or trigger manually when you add new prospects.

2. API Query to CompanyEnrich

Each domain is sent to the CompanyEnrich API which returns structured firmographic data. The workflow handles API authentication and rate limiting automatically.

3. Data Processing & Mapping

Key data points are extracted from the API response and formatted for spreadsheet use. The workflow can handle partial matches and missing data gracefully.

4. Write Back to Google Sheets

Enriched data is written to adjacent columns in your original sheet, maintaining row alignment with the source domains. You control which fields get written and their column positions.

Pro tip: Create a separate 'Enriched' tab in your sheet to preserve original data and track enrichment history over time.
