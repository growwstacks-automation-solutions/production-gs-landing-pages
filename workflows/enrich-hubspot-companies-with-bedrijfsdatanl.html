What This Workflow Does

Manual company research is a major time-sink for sales and marketing teams. This automation solves that by instantly enriching HubSpot company records with accurate, verified data from the Bedrijfsdata.nl database—the most comprehensive source for Dutch business information.

Whenever a company property changes in HubSpot (like a domain being added), the workflow triggers, fetches real-time data including KvK registration, LinkedIn URL, phone numbers, employee count, industry classification, and even SEO and location data, then updates the HubSpot record automatically. This turns basic leads into fully profiled accounts ready for targeted engagement.

How It Works

1. Trigger on HubSpot Changes

The workflow listens for specific events in HubSpot, typically a company.propertyChange webhook on fields like "domain" or "website." When a change is detected, it captures the company ID and relevant details to process.

2. Fetch Enrichment Data from Bedrijfsdata.nl

Using the company domain or name, the workflow queries the Bedrijfsdata.nl API. This returns over 100 data points including firmographics, contact details, technographics, and business descriptions.

3. Smart Data Matching & Fallback

If an exact match isn't found, the workflow can use fuzzy matching on existing data or fallback to a Bedrijfsdata ID if previously stored. Error handling routes failed lookups for manual review.

4. Update HubSpot Company Profile

The verified data is mapped to corresponding custom properties in HubSpot and the company record is updated. The workflow can be configured to update specific fields or enrich all available data points.

What You'll Need

  1. An active n8n instance (cloud or self-hosted).
  2. HubSpot account with admin access to create a private app and webhooks.
  3. Bedrijfsdata.nl developer account (free trial available with 500 credits).
  4. The Bedrijfsdata.nl verified community node installed in your n8n environment.
  5. Custom company properties set up in HubSpot to receive the enriched data.

Pro tip: Start by enriching your existing high-value accounts in HubSpot using a batch process before enabling real-time enrichment for new leads. This immediately improves your existing data quality.

Quick Setup Guide

  1. Import the template: Download the JSON file and import it into your n8n workspace.
  2. Configure credentials: Set up OAuth2 for HubSpot and add your Bedrijfsdata.nl API key in the respective nodes.
  3. Set up HubSpot webhook: Create a private app in HubSpot with company read/write scopes and configure a webhook for company.propertyChange on the domain field.
  4. Map custom properties: Adjust the "Update a company" node to match your HubSpot custom field internal names.
  5. Test and activate: Update a test company in HubSpot and verify the enrichment works, then activate the workflow.
