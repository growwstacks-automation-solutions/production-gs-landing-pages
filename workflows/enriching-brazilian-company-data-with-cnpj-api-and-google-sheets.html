What This Workflow Does

This automation solves a critical problem for businesses operating in Brazil: manually researching company information is time-consuming, error-prone, and doesn't scale. The workflow automatically enriches your Brazilian company database by fetching comprehensive CNPJ (Cadastro Nacional da Pessoa Jurídica) information from Brazil's official MinhaReceita.org API and updating your Google Sheets spreadsheet.

Instead of spending hours looking up individual companies, you can process hundreds of records automatically. The system reads CNPJ numbers from your spreadsheet, retrieves 47+ fields of official company data including address, registration status, partners, and economic activity codes, then writes everything back to your sheet. It even sends Telegram notifications when processing is complete.

How It Works

1. Read CNPJ Numbers from Google Sheets

The workflow starts by connecting to your Google Sheets spreadsheet and reading the CNPJ column. It intelligently filters out records that already have data (checking the razao_social field) to avoid duplicate processing and save API calls.

2. Fetch Company Data from MinhaReceita.org API

Each valid CNPJ number is sent to the MinhaReceita.org API, which returns comprehensive official data from Brazil's Federal Revenue Service. This includes company name, address, tax status, partners, economic activity codes, and registration dates.

3. Process Data in Batches

To handle large datasets efficiently, the workflow processes records in configurable batches (default: 100 records). This prevents timeouts and allows for progress tracking. Failed records are logged for review without stopping the entire process.

4. Update Google Sheets with Enriched Data

The retrieved company information is mapped to corresponding columns in your spreadsheet. The system can populate any combination of 47+ available fields, from basic contact information to complex partnership structures and tax regime details.

5. Send Completion Notifications

Once all records are processed, the workflow sends a Telegram notification with summary statistics: how many records were processed, successful updates, and any failures that need attention.

Pro tip: Configure the batch size based on your typical dataset. For occasional small updates, use 50-100 records per batch. For large initial imports, you might process 200+ records with longer timeout settings.

Quick Setup Guide

  1. Prepare your spreadsheet: Create a Google Sheet with a "cnpj" column containing Brazilian company registration numbers (numbers only, no punctuation). Add columns for the data fields you want to populate.
  2. Configure Google Sheets credentials: In n8n, set up OAuth2 credentials for Google Sheets following the platform's documentation.
  3. Import the workflow: Download the template and import it into your n8n instance using the import function.
  4. Update configuration nodes: Replace the placeholder Google Sheets ID with your actual spreadsheet ID and configure the sheet name and range.
  5. Test with a small batch: Run the workflow with 5-10 records first to ensure everything works correctly before processing larger datasets.
  6. Schedule regular updates: Once verified, set up a schedule (weekly or monthly) to keep your company data current.

Pro tip: Start with just a few key fields like company name, address, and registration status. Once you're comfortable with the workflow, add more specialized fields like partner information or economic activity codes.
