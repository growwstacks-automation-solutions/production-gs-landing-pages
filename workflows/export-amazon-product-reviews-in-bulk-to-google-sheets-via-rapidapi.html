Export Amazon Product Reviews to Google Sheets

Automatically collect up to 700 recent and top-rated Amazon reviews per product into a spreadsheet for analysis, reporting, and competitive intelligence.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Manual Amazon review collection is time-consuming and inconsistent. Brand managers, marketplace sellers, and agencies need structured customer feedback data to make informed decisions about product improvements, marketing strategies, and competitive positioning.

This automation solves that problem by systematically pulling up to 700 reviews per Amazon product—including both recent feedback and top-rated opinions—and organizing them directly into a Google Sheets tab of your choice. It transforms scattered customer opinions into actionable business intelligence.

The workflow handles the entire process: extracting product identifiers, making API calls with proper throttling, filtering empty responses, and structuring the data for immediate analysis. What used to take hours of manual copying and pasting now happens automatically with a single trigger.

How It Works

1. Data Collection Trigger

The workflow starts with an n8n Form Trigger that collects essential information: brand name, product/model name, Amazon product URL, and the destination Google Sheets tab URL. This form can be shared with team members or scheduled to run automatically for regular updates.

2. ASIN Extraction & Request Preparation

A Code node extracts the Amazon Standard Identification Number (ASIN) from the product URL, then builds up to 70 API requests (10 pages for each star rating from 1 to 5 stars). This ensures comprehensive coverage of both positive and critical feedback.

3. Controlled API Execution

The Split-in-Batches node processes the request list systematically, with Wait nodes providing necessary throttling to respect API rate limits. Each batch makes HTTP requests to RapidAPI's Real-Time Amazon Data service to fetch review data.

4. Data Validation & Processing

An IF node filters out empty or error responses, ensuring only valid review data proceeds. The Split Out node then breaks review arrays into individual records, preparing each review for insertion into your spreadsheet.

5. Google Sheets Integration

Finally, the Google Sheets node appends each review to your specified tab, including review text, rating, date, reviewer name, and other metadata. The loop continues until all pages are processed, creating a complete dataset.

What You'll Need

  1. RapidAPI Account: Free tier provides ~100 monthly calls (enough for 1-2 products). Upgrade for larger volumes.
  2. Google Sheets Access: A Google account with Sheets access and OAuth2 permissions granted to n8n.
  3. n8n Instance: Self-hosted n8n, n8n.cloud, or the desktop app to run the workflow.
  4. Amazon Product URLs: The specific product pages you want to monitor (ASIN will be extracted automatically).
  5. Destination Spreadsheet: A Google Sheet with appropriate column headers for review data.

Quick Setup Guide

  1. Download & Import: Download the JSON template and import it into your n8n instance via the workflow import function.
  2. Configure Credentials: Add your RapidAPI X-RapidAPI-Key as an HTTP Header Auth credential in n8n. Connect your Google Sheets OAuth2 credentials.
  3. Prepare Spreadsheet: Create or identify your destination Google Sheet and ensure sharing settings allow editing via the API ("Anyone with the link can edit" works).
  4. Test with One Product: Execute the workflow manually with a single Amazon product URL to verify the connection and data flow.
  5. Schedule or Share: Set up a schedule for regular updates or share the form trigger URL with team members for on-demand collection.
  6. Monitor & Adjust: Check the first few runs to ensure data quality, then expand to additional products as needed.

Pro tip: Start with your own products first to validate the workflow, then expand to competitors. Consider adding a sentiment analysis step using AI services once the basic data collection is working reliably.
