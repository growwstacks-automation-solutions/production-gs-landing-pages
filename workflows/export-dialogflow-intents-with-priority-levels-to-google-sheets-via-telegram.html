Export Dialogflow Intents to Google Sheets via Telegram

Automatically export intent names and priority levels with visual indicators when triggered by Telegram message

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the challenge of manually tracking Dialogflow intent configurations by automatically exporting intent names and their priority levels to a Google Sheets spreadsheet whenever triggered by an authorized Telegram message.

The workflow includes visual indicators (color-coded emojis) for different priority levels, making it easy to scan and analyze intent hierarchies at a glance. This is particularly valuable for teams managing complex conversational flows across multiple languages or business units.

How It Works

1. Telegram Trigger

An authorized user sends a specific command (like "backup") via Telegram. The workflow validates the user ID before proceeding.

2. Dialogflow API Call

The workflow queries your Dialogflow agent using the Dialogflow API to retrieve all intents and their metadata.

3. Priority Mapping

Each intent's priority value is mapped to a visual indicator:

  • 🔴 Highest priority
  • 🟠 High priority
  • 🔵 Normal priority
  • 🟢 Low priority
  • 🚫 Ignored/disabled

4. Google Sheets Update

The processed data is appended to a Google Sheet with timestamp, including intent name, priority number, visual indicator, and description.

What You'll Need

  1. A Dialogflow agent with configured intents
  2. Google Service Account credentials for Dialogflow API access
  3. A Telegram bot token
  4. Google Sheets document with write permissions

Quick Setup Guide

  1. Import the JSON template into your n8n instance
  2. Configure Telegram node with your bot token
  3. Set up Dialogflow credentials in the API node
  4. Connect to your Google Sheets document
  5. Test with your Telegram bot

Pro tip: Schedule regular backups by combining this workflow with a cron trigger, or trigger before major agent updates.
