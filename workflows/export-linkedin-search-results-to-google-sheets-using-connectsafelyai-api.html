What This Workflow Does

This n8n workflow solves the tedious process of manually copying LinkedIn search results into spreadsheets for sales prospecting or recruiting. By connecting LinkedIn (via ConnectSafely.ai's compliant API) with Google Sheets, it automatically transfers profile data from your searches into an organized spreadsheet.

The automation handles the entire process: executing your saved LinkedIn search, extracting relevant profile details, formatting the data, and appending it to your designated Google Sheet. This eliminates hours of manual copying and pasting while ensuring data consistency and reducing human error.

How It Works

1. Configure Your LinkedIn Search

First, perform your target search on LinkedIn using their advanced filters (location, industry, job title, etc.). The workflow will capture these exact parameters each time it runs.

2. ConnectSafely.ai API Integration

The workflow uses ConnectSafely.ai's API to securely access LinkedIn data without violating terms of service. You'll need an API key (included in setup instructions).

3. Data Transformation

The raw API response is processed to extract key fields like name, headline, company, location, and profile URL. You can customize which fields to include.

4. Google Sheets Export

Processed data is appended to your specified Google Sheet with timestamps. The workflow can create new sheets or append to existing ones based on your configuration.
