What This Workflow Does

Manual Amazon product research is time-consuming and error-prone. This n8n automation template solves that by automatically scraping Amazon search results using the Olostep API and organizing the data directly into Google Sheets.

Whether you're analyzing competitors, tracking pricing trends, sourcing products for dropshipping, or building a product database, this workflow eliminates hours of manual copying and pasting. Simply enter a search query (like "wireless headphones" or "yoga mats"), and the system scrapes multiple pages of Amazon results, extracts product titles and URLs, cleans the data, and saves it to your spreadsheet.

The automation handles pagination, URL normalization, and error checking—giving you clean, structured data ready for analysis in minutes instead of days.

How It Works

1. Search Trigger & Pagination Setup

The workflow begins with a form or manual trigger where you input your Amazon search query. The system automatically generates pagination for multiple result pages (typically 1-10 pages) to capture comprehensive product listings.

2. Olostep API Scraping

For each page number, the workflow calls the Olostep API with your search query. Olostep handles the complex scraping logic, bypassing Amazon's anti-bot measures and returning structured JSON data containing product titles and URLs.

3. Data Processing & Cleaning

The raw JSON response is parsed and split into individual product items. Relative URLs are converted to full Amazon product links, and data is normalized for consistency. An IF node filters out invalid or non-product URLs to ensure data quality.

4. Google Sheets Integration

Each validated product record is automatically appended to your designated Google Sheet. The template creates columns for product titles and URLs, with timestamps for when the data was collected.

5. Rate Management & Error Handling

Built-in wait steps prevent API rate limiting, and error handling ensures the workflow continues even if individual pages fail. This makes the scraping process reliable for large-scale research projects.
