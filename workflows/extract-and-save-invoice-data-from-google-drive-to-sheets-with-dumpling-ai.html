Extract Invoice Data from Google Drive to Sheets with Dumpling AI

Automatically read invoice PDFs, extract key details using AI, and save structured data to Google Sheets—eliminating manual entry.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Manual invoice processing is a tedious, error-prone bottleneck for finance teams. This workflow solves that by automating the entire extraction pipeline. It watches a designated Google Drive folder for new invoice PDFs, uses Dumpling AI to intelligently parse the document, and saves the extracted data—vendor details, dates, line items, totals—into a structured Google Sheet.

The automation handles the entire flow: file detection, AI-based data extraction, formatting, and database entry. It transforms unstructured PDFs into actionable financial records without human intervention, saving hours per week and ensuring data consistency.

This is particularly valuable for businesses that receive invoices digitally from multiple vendors, e-commerce operations with high-volume supplier invoices, or accounting departments that need to consolidate invoice data for reporting and payment scheduling.

How It Works

Step 1: Trigger on New Invoice Upload

The workflow starts with a Google Drive trigger node monitoring a specific folder. When a new invoice PDF is uploaded, the workflow automatically captures the file and proceeds.

Step 2: Download and Prepare the File

The invoice file is downloaded from Google Drive and converted into a Base64 format suitable for API transmission. This ensures the AI service can process the document correctly.

Step 3: AI Extraction via Dumpling AI

The Base64 file is sent to Dumpling AI's extract-document endpoint with a tailored prompt instructing it to locate key invoice fields: order number, document date, PO number, sold-to/ship-to addresses, and line-item details (model, description, quantity, unit price, total).

Step 4: Parse and Structure the Response

A Code node processes Dumpling AI's JSON response, extracting the structured data and preparing it for insertion into Google Sheets. The items array is split into individual rows using a Split Out node.

Step 5: Save to Google Sheets

Each invoice line item, along with its header metadata, is appended to a preformatted Google Sheet. The sheet acts as a centralized invoice database, ready for analysis, payment processing, or integration with accounting software.

What You'll Need

  1. A Google Drive folder where invoices are uploaded.
  2. A Google Sheet with columns for: Order number, Document Date, Po_number, Sold to name, Sold to address, Ship to name, Ship to address, Model, Description, Quantity, Unity price, Total price.
  3. Dumpling AI account and API key (sign up at dumplingai.com).
  4. n8n instance (cloud or self-hosted) with Google Drive and Google Sheets credentials configured.
  5. Basic understanding of n8n workflow import and credential setup.

Quick Setup Guide

1. Import the template JSON into your n8n workspace.

2. Configure the Google Drive trigger node with your target folder ID.

3. Set up Google Sheets node with your sheet ID and sheet name (default: Sheet1).

4. Add your Dumpling AI API key to the HTTP Request node's Authorization header.

5. Test by uploading an invoice PDF to your Google Drive folder—the workflow should automatically extract data and populate your sheet.

6. Customize the Dumpling AI prompt if your invoices contain additional fields or different formats.

Pro tip: For high-volume processing, add a filter node to ignore duplicate files or non-invoice documents. You can also extend the workflow to send notifications via Slack or email when extraction completes.
