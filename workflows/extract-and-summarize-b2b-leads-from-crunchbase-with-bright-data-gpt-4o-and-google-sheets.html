What This Workflow Does

Manual B2B lead generation from platforms like Crunchbase is time-consuming and inconsistent. Sales teams spend hours researching companies, copying data, and trying to extract meaningful insights from lengthy descriptions. This workflow automates the entire process, transforming raw Crunchbase data into structured, AI-summarized leads ready for sales outreach.

The pipeline extracts company information using Bright Data's web scraping capabilities, processes the unstructured text through GPT-4o to identify key details like funding rounds, industry tags, and growth signals, then organizes everything into Google Sheets for immediate use by sales and marketing teams. It solves the problem of scaling lead discovery while maintaining data quality and contextual understanding.

How It Works

1. Data Extraction from Crunchbase

The workflow begins by accessing Crunchbase company pages through Bright Data's Web Unlocker, which handles authentication and anti-scraping measures. It captures the raw HTML content containing company descriptions, funding information, leadership details, and industry classifications.

2. Text Processing & AI Analysis

Raw markdown and HTML content is converted to clean text, then processed through two AI chains. A basic LLM chain extracts structured fields (company name, location, founding year), while a summarization chain creates executive overviews highlighting growth potential and market position.

3. Data Structuring & Storage

Extracted data is formatted into consistent JSON structures with validated fields. The workflow appends this information to a designated Google Sheet, creating organized columns for each data point while maintaining the AI-generated summaries for quick review.

4. Notification & Archiving

Finally, the system sends structured webhook notifications to configured endpoints (like Slack or CRM systems) and saves both raw and processed data files locally for audit trails and further analysis.

What You'll Need

  1. A Bright Data account with Web Unlocker API access for Crunchbase scraping
  2. An OpenAI API key with GPT-4o access for text processing and summarization
  3. Google Sheets credentials configured in n8n for data storage
  4. A running n8n instance (cloud or self-hosted) to execute the workflow
  5. Target Crunchbase URLs or search criteria for companies you want to analyze

Quick Setup Guide

Follow these steps to implement this lead discovery pipeline in your environment:

  1. Download and import the JSON template into your n8n workspace
  2. Configure credentials for Bright Data (Header Authentication), OpenAI, and Google Sheets
  3. Set target URLs in the "Set URL, Filename and Bright Data Zone" node with your Crunchbase company pages
  4. Adjust AI prompts in the LLM chains if you need different extraction fields or summary formats
  5. Test with a single company to verify data extraction quality before scaling to multiple targets
  6. Schedule execution or trigger manually based on your lead generation needs

Pro tip: Start with 5-10 target companies to validate data quality before scaling. Adjust the AI summarization prompt to emphasize factors most important to your sales team, like recent funding rounds or specific technology stacks.
