What This Workflow Does

Manual resume processing is a time-consuming bottleneck in recruitment. HR teams waste hours extracting information from PDFs, copying data into spreadsheets, and creating standardized candidate profiles. This workflow automates the entire process from resume submission to structured data extraction and formatted document creation.

Candidates upload their resumes through a Telegram bot, which triggers an automated pipeline that extracts text from PDFs, uses AI to identify and structure key information (employment history, skills, education), and generates a clean, consistently formatted PDF using Gotenberg. The result is searchable candidate data ready for your ATS and professional-looking documents for your records.

How It Works

1. Resume Submission via Telegram

Candidates send their resume PDFs to your Telegram bot. The workflow authenticates the user based on chat ID to ensure only authorized submissions are processed.

2. PDF Text Extraction

The uploaded PDF is converted to plain text, preserving all content including complex formatting, tables, and bullet points that traditional parsers might miss.

3. AI-Powered Data Parsing

An AI chain with OpenAI Chat Model and Structured Output Parser analyzes the text, identifying sections like work experience, education, skills, and contact information, then outputs this as structured JSON data.

4. HTML Formatting

Each extracted data section is formatted into clean HTML with consistent styling, creating a professional template for the final document.

5. PDF Generation with Gotenberg

The formatted HTML is sent to Gotenberg (or alternative PDF services) to generate a polished, print-ready PDF with proper margins, fonts, and layout.

Pro tip: You can replace Gotenberg with other PDF generation services like PDFMonkey or ApiTemplate if you prefer cloud-based solutions over self-hosted options.

Who This Is For

This automation is ideal for recruitment agencies, corporate HR departments, startups hiring frequently, and any business that processes more than 10 resumes per week. It's particularly valuable for technical recruiting where skill matching is crucial, and for companies wanting to improve candidate experience with faster response times.

What You'll Need

  1. A Telegram bot token (create via @BotFather)
  2. n8n instance (cloud or self-hosted)
  3. OpenAI API key for AI processing
  4. Gotenberg installation or alternative PDF service API key
  5. Basic understanding of JSON for customizing output schema

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Create your Telegram bot and add credentials to the Telegram trigger node
  3. Set your chat ID in the authentication node for security
  4. Configure your OpenAI API key in the AI chain nodes
  5. Adjust the JSON schema in the Structured Output Parser to match your data needs
  6. Point the HTTP Request node to your Gotenberg instance or alternative PDF service
  7. Test by uploading a sample resume to your Telegram bot
