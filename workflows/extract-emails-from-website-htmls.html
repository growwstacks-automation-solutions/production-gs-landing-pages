What This Workflow Does

Manual email collection is one of the biggest time-wasters in sales and marketing. Teams spend hours visiting websites, searching for contact pages, and copying email addresses—only to end up with inconsistent, error-prone lists that don't scale.

This n8n workflow automates the entire process. It acts as a custom email scraping API that takes a website URL, extracts all valid email addresses from the HTML, and returns them in a structured format. You can trigger it manually for one-off research or connect it to a list of target sites for bulk extraction.

The solution transforms email discovery from a manual chore into a scalable, repeatable process. Whether you're building prospect lists for outbound campaigns, enriching existing contact data, or monitoring competitor websites for new hiring contacts, this automation delivers clean, actionable email data directly into your systems.

How It Works

The workflow uses n8n's powerful HTTP and data transformation nodes to create a reliable email extraction engine.

Step 1: Website Fetching

The workflow starts by receiving a website URL via webhook or manual input. It uses the HTTP Request node to fetch the complete HTML content of the target page, handling redirects and connection issues automatically.

Step 2: Email Pattern Matching

Once the HTML is retrieved, the workflow applies regular expression (regex) patterns to identify all strings that match standard email address formats. This catches emails in various formats including mailto: links, plain text within paragraphs, and contact form fields.

Step 3: Data Cleaning & Validation

Raw matches often include false positives or malformed addresses. The workflow filters out invalid formats, removes duplicates, and can optionally validate domains through additional checks to ensure you only get deliverable business emails.

Step 4: Structured Output

The final step formats the extracted emails into a clean JSON or CSV structure, ready for export to your CRM, spreadsheet, or email marketing platform. The workflow can also log the source URL and extraction timestamp for tracking purposes.

Quick Setup Guide

  1. Download & Import: Click the Download Template button above to get the JSON file. In your n8n instance, go to Workflows → Import from File and select the downloaded template.
  2. Configure Webhook (Optional): If you want to trigger the workflow via API, note the Webhook URL generated by n8n. You'll use this endpoint to send website URLs for processing.
  3. Test with a Single URL: Run the workflow manually and enter a test website URL (like a company's contact page) in the designated field. Execute to see email extraction in action.
  4. Adjust Extraction Patterns: Review the regex patterns in the workflow. You may want to tweak them based on the specific email formats common in your target websites.
  5. Connect Your Destination: Add a node at the end to send extracted emails to your preferred destination—like Google Sheets, a database, or your CRM.
  6. Schedule or Automate: Set the workflow to run on a schedule for regular list updates, or connect it to a spreadsheet of target URLs for batch processing.
