What This Workflow Does

Manual invoice processing is a time-consuming, error-prone task that drains productivity from finance teams and freelancers. This workflow solves that problem by automating the entire detection, extraction, and notification process.

When an invoice arrives in your Gmail inbox, the workflow automatically identifies it using AI, extracts critical payment information (amount, due date, vendor, invoice number), and sends an immediate Slack notification to your finance team or yourself. This eliminates manual sorting, reduces payment delays, and ensures accurate data capture without human intervention.

The system works with various invoice formats—PDF attachments, plain text emails, HTML emails—making it suitable for real-world business environments where vendors use different communication styles.

How It Works

Step 1: Scheduled Gmail Monitoring

The workflow runs on a schedule (default: hourly) and checks your Gmail inbox for unread emails. It filters for emails likely to contain invoices based on subject lines, sender patterns, or attachment types.

Step 2: AI Classification & Data Extraction

Email content is sent to OpenAI's GPT-4o model, which determines whether the email contains an invoice. If confirmed, the AI extracts structured data: payment amount, due date, vendor name, invoice number, and any special notes.

Step 3: Slack Notification Delivery

The extracted data is formatted into a clear Slack message and sent to a designated channel or user. The notification includes all critical details and optionally includes actionable buttons or links for quick processing.

Step 4: Non-Invoice Email Handling

If the AI determines the email is not invoice-related, the workflow simply ignores it, ensuring only relevant communications trigger notifications.

What You'll Need

  1. A Gmail account with OAuth2 access enabled
  2. A Slack workspace with OAuth2 permissions for sending messages
  3. An OpenAI API key (GPT-4o access)
  4. An n8n instance (cloud or self-hosted) to run the workflow
  5. Basic understanding of workflow configuration (credentials setup)

Pro tip: Test the AI extraction with a few sample invoices first to ensure it correctly identifies your specific vendor formats before running the workflow at scale.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n instance via the workflow import function.
  2. Configure Credentials: Set up Gmail (OAuth2), Slack (OAuth2), and OpenAI (API key) credentials in n8n's credentials section.
  3. Adjust Schedule: Modify the Schedule Trigger node to match your preferred monitoring frequency (hourly, daily, etc.).
  4. Customize Slack Message: Edit the Slack node to change notification formatting, channel destination, or add additional fields.
  5. Test with Sample Emails: Send test invoice emails to your Gmail and verify the workflow correctly detects, extracts, and notifies.
  6. Activate the Workflow: Turn on the workflow in n8n and monitor initial executions to ensure proper functioning.
