What This Workflow Does

Manual LinkedIn research is slow, inconsistent, and unscalable. Sales teams spend hours copying company details from profiles, investors miss funding updates, and market researchers struggle with data volume. This automation solves that by extracting structured company intelligence automatically.

The workflow takes a LinkedIn company URL and an authenticated Airtop profile, then returns enriched data including company identity (name, tagline, headquarters), scale (employee count brackets), business classification (automation agency status, AI implementation level), and funding profile (latest round, total raised, investors). This turns unstructured public information into actionable business intelligence ready for CRM enrichment, lead scoring, or investment analysis.

How It Works

Step 1: Input & Session Creation

The workflow accepts a LinkedIn company page URL and your Airtop profile credentials. It creates an authenticated browser session using Airtop, simulating a human visit to the LinkedIn page without triggering anti-bot detection.

Step 2: Data Extraction & Navigation

The Airtop session navigates to the specified company profile, executes predefined queries to locate key sections (About, Employees, Funding), and extracts text content, metadata, and numerical data from the page structure.

Step 3: Classification & Structuring

Extracted raw text is processed through logic nodes that classify company type (automation agency yes/no), determine AI implementation level based on keywords, assign technical sophistication tiers, and calculate employee size brackets.

Step 4: Output & Delivery

The structured JSON output includes all enriched fields, formatted for direct import into CRM systems like Salesforce or HubSpot, databases, or analytics dashboards. The workflow can trigger follow-up actions like lead prioritization or alert notifications.

What You'll Need

  1. Airtop API Key: From your Airtop account portal, required for browser automation sessions.
  2. LinkedIn-Authenticated Airtop Profile: A browser profile within Airtop that has logged into LinkedIn to access public company pages.
  3. n8n Instance: Self-hosted n8n or n8n.cloud account to import and run the workflow.
  4. Target Company URLs: List of LinkedIn company page URLs you want to enrich (can be fed from CSV, database, or CRM).
  5. Destination System: CRM, database, or spreadsheet where enriched data will be stored for business use.

Pro tip: Create a dedicated LinkedIn account for your Airtop profile to avoid mixing personal and business browsing activity. Use a business email address and limit extraction to publicly available data only to maintain compliance.

Quick Setup Guide

  1. Download & Import: Download the JSON template and import into your n8n instance via the workflow import function.
  2. Configure Airtop Credentials: Add your Airtop API key and select your LinkedIn-authenticated browser profile in the Airtop node settings.
  3. Test with Sample URL: Run the workflow with one LinkedIn company URL (e.g., a known company) to verify extraction accuracy and output structure.
  4. Connect to Data Source: Modify the trigger or input node to accept URLs from your CSV file, database query, or CRM export.
  5. Set Up Destination: Add nodes to send enriched data to your CRM (via API), database (via insert operation), or Google Sheets.
  6. Schedule or Trigger: Configure the workflow to run on schedule (daily/weekly) or trigger via webhook when new companies are added to your list.
  7. Monitor & Optimize: Review extracted data for consistency, adjust classification logic if needed, and add error handling for failed extractions.
