What This Workflow Does

Manual LinkedIn profile research is one of the most time-consuming tasks in recruitment and sales prospecting. Copy-pasting information from profiles into your CRM or ATS is not only slow but prone to errors and inconsistencies. This automation solves that problem by programmatically extracting structured data from any public LinkedIn profile.

The workflow uses Airtop for browser automation to visit LinkedIn profiles in a real browser session, then applies AI parsing to intelligently extract and categorize information. What you get is clean, structured JSON data ready for import into your systems—complete with names, positions, experience history, skills, education, and more.

Beyond simple data transfer, this automation enables strategic talent pooling, competitive intelligence gathering, and lead enrichment at scale. It turns hours of manual research into minutes of automated processing, allowing your team to focus on relationship-building rather than data entry.

How It Works

1. Trigger & Input

The workflow starts when you provide a LinkedIn profile URL. This can come from a form submission, a spreadsheet import, or another automation that identifies potential candidates or leads. You also specify which Airtop browser profile to use (pre-configured with LinkedIn access).

2. Browser Automation

Airtop opens the LinkedIn profile in a real browser, simulating human navigation. This approach respects LinkedIn's structure and handles JavaScript-rendered content that simple scraping might miss. The browser waits for the page to fully load before proceeding.

3. AI-Powered Parsing

Once the profile is loaded, an AI model analyzes the page content to identify and extract structured information. The AI understands context—distinguishing between current and past positions, identifying skill categories, and parsing complex experience descriptions into clean data points.

4. Data Structuring & Output

The extracted information is organized into a consistent JSON format with standardized fields. This structured data is then ready for the next steps in your process—whether that's importing into your ATS, enriching existing CRM records, or triggering follow-up communications.

What You'll Need

  1. Airtop Account: Free tier available. You'll need an API key and a configured browser profile logged into LinkedIn.
  2. n8n Instance: Self-hosted or cloud. The workflow is compatible with any n8n version that supports HTTP requests and code nodes.
  3. AI Model Access: OpenAI API key or compatible LLM endpoint for parsing the profile content.
  4. Destination System: Your ATS, CRM, or database where the extracted data will be stored (optional for testing).

Quick Setup Guide

Follow these steps to get the automation running in your environment:

  1. Import the Template: Download the JSON file and import it into your n8n instance via the workflow import function.
  2. Configure Credentials: Add your Airtop API key and AI model credentials to n8n's credential management system.
  3. Set Up Airtop Profile: In the Airtop dashboard, create a browser profile and log into LinkedIn once to establish a session.
  4. Test with Sample URLs: Run the workflow with a few LinkedIn profile URLs to verify extraction works correctly.
  5. Connect to Your Systems: Modify the output nodes to send data to your ATS, CRM, or database using their respective APIs.
  6. Schedule or Trigger: Set up the workflow to run on a schedule for batch processing or trigger it from other automations.

Pro tip: Start with a small batch of 10-20 profiles to fine-tune the AI parsing prompts before scaling up. Different industries may require slight adjustments to extract the most relevant information for your use case.
