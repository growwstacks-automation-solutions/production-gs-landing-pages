What This Workflow Does

This automation solves the tedious process of manually extracting text from images and documents stored in Google Drive. It automatically detects new image uploads, extracts text using OCR technology, generates concise AI summaries, and organizes the results in Google Sheets - complete with email notifications.

Businesses dealing with scanned documents, receipts, or handwritten notes can save dozens of hours per month by automating this previously manual process. The workflow maintains full control over your data while leveraging powerful AI capabilities.

How It Works

1. File Detection

The workflow monitors a designated Google Drive folder for new image uploads, triggering automatically when files are added.

2. Text Extraction

Each image is processed through OCR.space's API to extract all text content, supporting multiple languages including Japanese.

3. AI Summarization

The extracted text is sent to OpenRouter's AI models to generate concise summaries, capturing key information from lengthy documents.

4. Data Organization

Results are automatically logged in Google Sheets with timestamps, original filenames, and both raw and summarized text.

5. Notification

Stakeholders receive email alerts via Gmail when processing completes, with links to the updated spreadsheet.

Pro tip: For sensitive documents, you can replace the cloud OCR service with a self-hosted Tesseract OCR instance for enhanced data privacy.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
