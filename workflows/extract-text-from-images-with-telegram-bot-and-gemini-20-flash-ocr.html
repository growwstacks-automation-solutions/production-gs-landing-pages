What This Workflow Does

This automation transforms your Telegram messenger into a powerful Optical Character Recognition (OCR) assistant. Instead of manually typing text from images, screenshots, documents, or signs, you simply send the image to your Telegram bot. Within seconds, the extracted text is sent back to you, ready to use.

The workflow leverages Google's Gemini 2.0 Flash AI model, which provides superior accuracy compared to traditional OCR tools—especially for handwritten notes, complex layouts, or low-quality images. This eliminates hours of tedious data entry and makes information capture as simple as taking a photo.

Businesses use this automation to digitize meeting notes, process receipts and invoices, capture text from whiteboards, extract data from screenshots, and save important information from physical documents without manual transcription.

How It Works

The automation follows a clean, efficient process that requires no technical knowledge to use once set up.

Step 1: Image Reception via Telegram

When you send any image to your configured Telegram bot, the workflow triggers automatically. It supports photos, screenshots, document scans, and even images with text overlays.

Step 2: Image Processing & Analysis

The workflow extracts the image file from Telegram and prepares it for analysis. It handles various image formats and optimizes them for the AI processing stage.

Step 3: AI-Powered Text Extraction

The image is sent to Google's Gemini Vision API, where advanced AI models analyze the visual content, identify text regions, and extract readable text with high accuracy, including understanding context and formatting.

Step 4: Results Delivery

The extracted text is formatted and sent back to you as a Telegram message. You can then copy, save, or further process the text as needed.

Pro tip: Use this workflow with a dedicated Telegram channel for team collaboration. Multiple team members can send images, and everyone receives the extracted text, creating a shared knowledge base.

What You'll Need

  1. Telegram Account & Bot: A Telegram account and a bot created via BotFather (free, takes 2 minutes).
  2. Google Gemini API Key: A free API key from Google AI Studio with access to the Gemini Vision model.
  3. n8n Instance: Either n8n.cloud (hosted) or a self-hosted n8n installation.
  4. Basic Configuration Access: Ability to add credentials to your n8n instance for Telegram and Google APIs.

Quick Setup Guide

Get this automation running in less than 5 minutes with these simple steps:

  1. Download the template using the button above and import it into your n8n instance.
  2. Create Telegram credentials by talking to @BotFather on Telegram, creating a new bot, and copying the API token.
  3. Get Google Gemini API key from Google AI Studio (free tier available) and enable the Gemini Vision API.
  4. Configure credentials in n8n by adding both the Telegram bot token and Google API key to your n8n credentials.
  5. Activate the workflow and test by sending an image to your Telegram bot.

Pro tip: For business use, create a dedicated Telegram channel and add your bot as an admin. Team members can then post images to the channel, and everyone receives the extracted text, facilitating collaboration.
