What This Workflow Does

For eCommerce researchers, affiliate marketers, and price comparison analysts, manually scouring websites for the best products is a tedious, time-consuming chore. This automation solves that by turning a product listing URL into a neatly ranked spreadsheet of top picks in minutes.

The workflow intelligently crawls any eCommerce page (like Amazon, eBay, or niche stores), takes visual screenshots, and uses GPT-4o's vision capabilities to "read" the page and extract key details: product name, price, review count, and estimated delivery date. It then ranks up to three best-value options based on your criteria and logs everything into a Google Sheet for easy review and sharing.

This isn't simple web scraping—it's AI-powered visual analysis that works on dynamically loaded pages and complex layouts where traditional scraping fails. You get consistent, structured data without writing a single line of code.

How It Works

The automation follows a clear, logical sequence to transform a URL into actionable insights.

1. URL Submission & Trigger

The process starts when you submit a product listing URL through a simple web form trigger in n8n. This could be automated to run on a schedule or triggered from another app, but the manual form provides full control for one-off research.

2. Web Crawling & Screenshot Capture

The Dumpling AI node visits the provided URL, renders the page fully (including JavaScript-loaded content), and captures screenshots. This visual approach is key—it bypasses anti-scraping protections and works on any site a human can browse.

3. AI-Powered Visual Analysis

Each screenshot is sent to the OpenAI GPT-4o model with a carefully crafted prompt. The AI acts as a virtual researcher, identifying individual product cards, reading the text, and extracting the specified fields: product name, price, number of reviews, and any free delivery promise.

4. Data Parsing & Structuring

The raw, unstructured text from the AI is passed through a Code node. This node cleans the data, ensures consistent formatting (e.g., converting prices to numbers), and structures it into a clear JSON object ready for the spreadsheet.

5. Google Sheets Logging

The structured product data is appended as a new row in a pre-configured Google Sheet. Each run creates a timestamped entry, building a searchable history of your research over time.

6. Notification & Delivery

Finally, a Gmail node sends you an email with a direct link to the updated spreadsheet row. You're immediately notified that the research is complete and can review the findings.

What You'll Need

  1. An n8n instance. You can use the n8n Cloud free tier or self-host it.
  2. A Dumpling AI account and API token. This service handles the browser automation and screenshot capture.
  3. An OpenAI API key with access to the GPT-4o model. This is used for the visual analysis of screenshots.
  4. A Google Cloud Project with the Google Sheets API enabled and service account credentials (a JSON key file). This allows n8n to write to your spreadsheet securely.
  5. A Google Sheet set up with the columns: Timestamp, Product Name, Price, Review Count, Free Delivery Date, and Source URL.

Pro tip: Use a dedicated Google Service Account for automation, not your personal Google account. It's more secure and separates automation activities from your personal data.

Quick Setup Guide

Import and configure this workflow in under 15 minutes.

  1. Download the template using the button above. You'll get a .json file.
  2. In your n8n dashboard, click Import from File and select the downloaded JSON.
  3. Configure the credentials: Click the Dumpling AI node and add your API token. Do the same for the OpenAI node with your API key.
  4. Set up Google Sheets: In the Google Sheets node, authenticate with your service account JSON key. Paste the ID of your prepared spreadsheet and specify the worksheet name.
  5. Test the workflow: Click "Execute Workflow" on the Webhook node to generate a test URL. Open that URL in your browser and submit a product page link (e.g., an Amazon search results page).
  6. Check the results: The workflow will run. Check your Google Sheet and email for the new entry containing the AI-extracted product data.
