What This Workflow Does

This automation solves the tedious, error-prone task of manually typing transaction details from receipts, invoices, and bills. When someone sends a photo of a document to your Telegram bot, the workflow automatically downloads the image, sends it to the unli.dev Vision API for analysis, extracts key transaction data (like date, amount, vendor, items), and returns clean, Markdown-formatted text—all within seconds.

Instead of employees or clients emailing attachments or typing numbers into spreadsheets, they simply snap a picture and message your bot. The AI handles the rest, converting visual information into structured data ready for your accounting software, CRM, or database. This cuts processing time by up to 90% and virtually eliminates typos.

The system is particularly valuable for businesses handling expense reports, client invoices, supplier bills, or any situation where paper or digital receipts need to be digitized and cataloged. It transforms a manual, frustrating process into a seamless, automated pipeline.

How It Works

The workflow follows a logical sequence to ensure reliable extraction and delivery of transaction data.

1. Telegram Trigger

The automation starts when a user sends an image to your configured Telegram bot. The workflow monitors for new messages containing photos or documents.

2. Image Download & Preparation

The image file is downloaded from Telegram's servers and converted into a format suitable for AI analysis, typically base64 encoding, while preserving quality for accurate OCR.

3. AI Vision Processing

The prepared image is sent to the unli.dev Vision API with specific instructions to extract transaction details. The AI identifies text, understands context (like what's a date vs. a price), and structures the information.

4. Data Structuring & Validation

Raw extracted text is parsed and organized into consistent fields: transaction date, total amount, tax, vendor name, line items, and payment method. Basic validation checks ensure numbers are formatted correctly.

5. Response & Delivery

The cleaned, structured data is sent back to the user via Telegram in an easy-to-read Markdown format. Optionally, it can be simultaneously saved to Google Sheets, Airtable, or your database.

Pro tip: Configure your AI prompt specifically for your most common receipt types (restaurant, retail, services) to improve accuracy. Include examples of desired output format in your system instructions.

Quick Setup Guide

Get this automation running in your environment in under 15 minutes:

  1. Import the Template: Download the JSON file above and import it into your n8n instance via the workflow import function.
  2. Configure Telegram Credentials: Create a bot via @BotFather on Telegram, obtain your token, and add it to the Telegram trigger node in n8n.
  3. Set Up AI Vision API: Sign up for unli.dev, generate an API key, and add it to the HTTP Request node that calls their vision endpoint.
  4. Customize the Prompt: Adjust the system prompt in the workflow to specify exactly what transaction data you want extracted (e.g., "Extract date, total, vendor, and line items as Markdown").
  5. Test & Deploy: Send a test receipt image to your Telegram bot. Verify the extracted output matches your expectations, then activate the workflow.

Pro tip: Start with a small set of test images (different receipt formats) to fine-tune your AI prompt before full deployment. This improves accuracy for your specific use case.
