What This Workflow Does

Manual SEO audits are time-consuming and error-prone. This n8n workflow automates the entire process of extracting URLs from a website's sitemap.xml file—the foundational step for any technical SEO analysis. Instead of manually downloading and parsing XML files, this workflow systematically crawls sitemaps (including sitemap indexes that contain multiple sub-sitemaps), extracts every URL, and outputs them in a structured format ready for analysis.

The automation handles the complete pipeline: fetching the sitemap, parsing XML data, splitting entries, and converting results into usable files. Whether you're auditing your own site, monitoring competitors, or building a content inventory, this workflow transforms hours of manual work into a fully automated process that runs on schedule or demand.

Beyond basic extraction, this template serves as a foundation for more advanced SEO automation. You can easily extend it to check URL indexing status, monitor for new pages, analyze content gaps, or integrate with SEO platforms like Google Search Console, SEMrush, or Ahrefs for comprehensive site health monitoring.

How It Works

The workflow follows a logical sequence to ensure reliable extraction regardless of sitemap complexity.

1. Manual Trigger & URL Configuration

The workflow begins with a manual trigger for testing, which can be replaced with a schedule trigger for regular audits. A "Set" node defines the target website's base domain, which automatically constructs the sitemap URL (e.g., https://example.com/sitemap.xml).

2. Sitemap Fetching & Parsing

An HTTP Request node fetches the sitemap.xml file from the specified domain. The workflow then uses an XML node to parse the raw XML into structured JSON data, making individual URLs and sitemap entries accessible for processing.

3. Handling Sitemap Indexes

Many large websites use sitemap indexes that reference multiple sub-sitemaps. The workflow intelligently detects these indexes, splits them into individual sitemap entries, and recursively fetches each sub-sitemap to ensure no URLs are missed.

4. URL Extraction & Output

For each sitemap (main or sub-sitemap), the workflow extracts all URL entries using a Split Out node, isolating the loc (location) tags that contain the actual page URLs. Finally, a Convert to File node outputs the complete URL list as a downloadable JSON or text file.

Pro tip: Replace the Convert to File node with a Google Sheets node to automatically append extracted URLs to a spreadsheet for ongoing tracking and analysis with your SEO team.

Quick Setup Guide

Get this workflow running in your n8n environment in under 5 minutes:

  1. Import the template: Click the Download Template button above, then import the JSON file into your n8n instance via the workflow editor.
  2. Configure the domain: In the "Set URL" node, replace the example domain with your target website's base URL.
  3. Test the extraction: Click "Execute Workflow" to run a manual test and verify URLs are being extracted correctly.
  4. Adjust for complexity: For sites with very large sitemaps, consider increasing the timeout in the HTTP Request nodes from the default 10 seconds.
  5. Add your destination: Replace the Convert to File node with your preferred output (Google Sheets, database, or other storage).
  6. Schedule automation: Replace the manual trigger with a Schedule Trigger node to run weekly or monthly audits automatically.
