Extract YouTube Video Metadata and Save to Google Docs using RapidAPI

Automatically capture video details like views, engagement metrics, and descriptions. This n8n workflow saves structured data to Google Docs for team collaboration and analysis.

Download Template JSON · Zapier compatible · Free

What This Workflow Does

This automation solves the tedious manual process of copying YouTube video details into documentation. Marketing teams, content creators, and analysts often need to track video performance metrics, descriptions, and metadata for reporting and strategy purposes. Manually collecting this data is time-consuming and prone to errors.

The workflow connects to YouTube via RapidAPI to extract comprehensive video metadata including view counts, engagement metrics, descriptions, and publishing details. It then formats this information into a structured document in Google Docs, creating a searchable archive that teams can reference, annotate, and analyze.

How It Works

Step 1: Trigger Setup

The workflow can be triggered manually for specific videos or set to run automatically on a schedule. You provide the YouTube video URLs or IDs that need tracking.

Step 2: API Data Extraction

Using RapidAPI's YouTube Metadata API endpoint, the workflow fetches all available video details. This includes both public-facing information and analytics data if your API key has sufficient permissions.

Step 3: Data Processing

The raw API response is parsed to extract the most useful fields. The workflow can filter, transform, or enrich the data as needed - for example, calculating engagement rates from raw view and like counts.

Step 4: Google Docs Creation

A new Google Doc is created (or an existing one updated) with the video metadata formatted for readability. The template includes headings, tables, and consistent styling to make the information easily scannable.

What You'll Need

  1. A RapidAPI account with access to YouTube Metadata API
  2. Google Workspace account with Docs API access
  3. n8n instance (cloud or self-hosted)
  4. YouTube video URLs or channel to monitor

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance
  3. Configure RapidAPI credentials in the HTTP Request node
  4. Set up Google Docs authentication
  5. Test with sample video URLs
  6. Schedule automatic runs if needed
