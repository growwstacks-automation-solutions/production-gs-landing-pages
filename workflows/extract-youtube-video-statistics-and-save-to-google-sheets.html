What This Workflow Does

This automation solves the tedious, time-consuming process of manually tracking YouTube video performance metrics. Content creators, marketers, and agencies often waste hours each week copying data from YouTube Studio to spreadsheets for analysis and reporting.

The workflow automatically extracts comprehensive video statistics including views, watch time, likes, comments, and subscriber changes, then organizes this data in Google Sheets with timestamps. This creates a searchable historical record that enables trend analysis, performance benchmarking, and data-driven content decisions.

How It Works

1. YouTube API Connection

The workflow authenticates with YouTube's API using your Google account credentials. This grants secure access to your channel's analytics data without sharing passwords.

2. Video Selection

You can configure the workflow to track all videos in your channel or specify particular video IDs to monitor. The template includes options for both approaches.

3. Data Extraction

For each selected video, the workflow retrieves current statistics including view counts, engagement metrics, and audience retention data. The template captures over 15 key performance indicators.

4. Google Sheets Integration

Extracted data is automatically appended to your specified Google Sheet with a timestamp. The template creates properly formatted columns for each metric and includes error handling for API limits.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. YouTube channel owner or editor access
  3. Google Cloud Platform project with YouTube Data API enabled
  4. Google Sheets document with write permissions
  5. Basic understanding of API credentials setup

Pro tip: Create a dedicated "YouTube Analytics" folder in Google Drive to keep your data organized. Share this folder with team members who need access to the reports.

Quick Setup Guide

  1. Download the JSON workflow file
  2. Import into your n8n instance
  3. Configure YouTube API credentials in the HTTP Request node
  4. Set up your Google Sheets connection
  5. Specify which videos to track (all or selected)
  6. Set your preferred execution schedule (daily recommended)
  7. Test with 1-2 videos before full deployment
