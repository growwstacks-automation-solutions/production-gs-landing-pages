What This Workflow Does

For cryptocurrency traders and investors, staying on top of market movements is time-consuming and emotionally draining. Manually checking prices, order books, and trade history across multiple platforms creates decision fatigue and can cause missed opportunities during volatile periods.

This automation solves that by connecting directly to the MEXC exchange API and delivering real-time spot market data directly to your Telegram. It fetches latest prices, 24-hour statistics, order book depth, recent trades, and candlestick data—then optionally uses GPT-4.1-mini to format this complex data into clear, actionable summaries. You get instant market intelligence without constantly watching charts.

The workflow acts as your personal market analyst, working 24/7 to monitor specific trading pairs and deliver insights exactly when you need them. It transforms raw API data into Telegram-ready messages that help you make faster, more informed trading decisions.

How It Works

1. Telegram Command Trigger

The workflow starts when you send a cryptocurrency pair (like "BTCUSDT") to your Telegram bot. A Telegram Trigger node listens for these messages and passes the command to the automation.

2. User Authentication & Session Management

Before processing, the workflow verifies your Telegram ID against an authorized list. It then generates a unique session ID from your chat ID, creating lightweight memory for your interaction without storing sensitive data.

3. MEXC API Data Collection

Multiple HTTP Request nodes query the MEXC REST v3 API simultaneously for different data types: current ticker prices, 24-hour statistics, order book depth (customizable levels), best bid/ask snapshots, candlestick patterns, and recent trade history.

4. Data Processing & AI Enhancement

Calculator nodes compute spreads, averages, and mid-prices from the raw data. If enabled, GPT-4.1-mini then transforms complex JSON outputs into human-readable summaries, highlighting key trends and anomalies in plain language.

5. Telegram Delivery with Smart Formatting

A Message Splitter ensures outputs never exceed Telegram's 4000-character limit. The final structured report—grouped into Price, Statistics, Order Book, and Trades sections—is delivered back to your Telegram as clean, readable messages.

What You'll Need

  1. Telegram Bot Token from @BotFather for message sending/receiving
  2. Your Telegram User ID for authentication (get from @userinfobot)
  3. n8n Instance (cloud or self-hosted) to run the workflow
  4. OpenAI API Key (optional) for GPT-4.1-mini formatting enhancement
  5. MEXC API Key (optional, public endpoints work without authentication)

Pro tip: Start with public MEXC API endpoints (no key needed) to test the workflow. Only add your API key if you need higher rate limits for frequent queries.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n instance using the "Import from File" option.
  2. Configure Telegram: Create a bot via @BotFather, copy the token, and add it to the Telegram Trigger and Telegram nodes in the workflow.
  3. Set Authentication: Add your Telegram User ID to the User Authentication node's allowed list.
  4. Add OpenAI (Optional): If using GPT formatting, add your OpenAI API key to the AI node credentials.
  5. Test & Activate: Turn on the workflow and send a trading pair like "BTCUSDT" to your bot. You should receive structured market data within seconds.
