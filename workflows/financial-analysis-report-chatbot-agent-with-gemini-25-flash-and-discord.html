What This Workflow Does

Financial teams and investors spend countless hours researching stocks, sectors, and market trends. This manual process is time-consuming, inconsistent, and delays decision-making. The Financial Analysis Report Chatbot Agent solves this by automating the entire research workflow.

Simply ask a question about any ticker, sector, or investment thesis (like "NVDA long-term outlook?" or "Gold ETF short-term drivers?") through a chat interface. The system uses Google Gemini 2.5 Flash AI to analyze the query, generate a structured financial report with key insights, drivers, risks, and metrics, then automatically posts a formatted summary to your designated Discord channel. This creates a real-time knowledge hub where your entire team can access AI-powered financial insights instantly.

How It Works

The automation follows a sophisticated pipeline that transforms casual questions into professional-grade financial analysis.

Step 1: Question Capture

The workflow starts with a Chat Trigger that captures user questions. This can be configured for public or private chat access, allowing team members to submit queries naturally through a familiar interface.

Step 2: Context Management

A Conversation Memory node maintains context from previous interactions, ensuring the AI understands the conversation flow and can reference earlier questions or clarifications when generating analysis.

Step 3: AI Processing

The Connect Gemini node interfaces with Google's powerful Gemini 2.5 Flash model, configured with optimal parameters for financial analysis. The Agent node applies a specialized system prompt that structures the AI's thinking around investment analysis frameworks.

Step 4: Structured Output

A Structured Output Parser enforces a consistent JSON schema, ensuring every report includes essential elements like the core investment thesis, executive summary, key drivers, risk factors, relevant metrics, and actionable next steps.

Step 5: Formatting & Delivery

A Code node transforms the structured data into Discord-ready Markdown with proper formatting, emojis, and disclaimers. Finally, a Discord Webhook posts the polished report to your designated channel, making insights immediately available to your entire team.

Pro tip: For maximum value, configure this workflow to run on a schedule (using a Cron trigger) to provide regular market updates, or connect it to news APIs to trigger analysis automatically when significant market events occur.

Quick Setup Guide

Getting started with this powerful financial analysis automation takes just a few simple steps.

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure Google Gemini by adding your API key to the Connect Gemini node and selecting your preferred model (gemini-2.5-flash-lite is recommended for cost-effectiveness).
  3. Set up Discord by creating a webhook in your desired channel and pasting the URL into the Discord Webhook node.
  4. Test the workflow by activating it and using the built-in chat interface to ask a sample question about a familiar stock or sector.
  5. Customize the analysis by modifying the system prompt in the Agent node to match your specific investment philosophy and reporting requirements.

Pro tip: Start with simple queries about well-known stocks to verify the output quality, then gradually expand to more complex sector analyses and comparative assessments as you refine the AI's parameters.
