What This Workflow Does

Manual financial reconciliation is tedious, error-prone, and consumes valuable hours each week. This workflow functions as an AI-powered Financial Controller that automatically monitors your Gmail inbox for invoices, receipts, and bills, extracts the critical data using artificial intelligence, and logs it systematically to Google Sheets for review and approval.

Unlike simple email parsers, this intelligent system uses a "Guardrail" AI agent to filter out non-financial emails (like newsletters or personal correspondence) before processing, ensuring only actual financial transactions are captured. It handles both attached PDFs and data within email bodies, validates information, assigns appropriate accounting categories, and creates a clean, searchable financial record without manual intervention.

How It Works

1. Intelligent Email Monitoring

The workflow continuously watches a designated Gmail label or your entire inbox for new emails. It can be configured to monitor specific senders (like regular vendors) or scan all incoming messages for financial content.

2. AI Guardrail Filtering

Before processing any email, a Gemini-powered AI agent analyzes the email content to determine if it contains a valid financial transaction. This prevents newsletters, promotional emails, or personal correspondence from triggering the workflow unnecessarily.

3. Smart Data Extraction

For emails with attachments (typically PDF invoices), a GPT-4o AI agent extracts structured data from the document using OCR. For emails without attachments, another AI agent extracts financial details directly from the email body text.

4. Validation & Business Logic

Code nodes validate extracted data—checking for missing fields, invalid amounts, or formatting issues. The system automatically assigns General Ledger categories (e.g., "Uber" → "Travel & Entertainment") and sets approval statuses based on configurable amount thresholds.

5. Synchronization & Notification

Validated data is logged to Google Sheets with timestamps and source information. A confirmation email can be sent, while errors are logged to a separate error sheet for review. The system maintains a complete audit trail of all processed transactions.

What You'll Need

  1. n8n instance (version 1.0 or higher) – either self-hosted or n8n.cloud
  2. Gmail account with access to the inbox you want to monitor
  3. Google Sheets – a spreadsheet to receive the extracted data
  4. OpenAI API key for GPT-4o document processing (attached PDFs)
  5. Google Gemini API key for email content analysis and guardrail filtering
  6. Basic understanding of how to import and configure n8n workflows

Pro tip: Start by monitoring a specific Gmail label like "Invoices" rather than your entire inbox. Have vendors send invoices to a dedicated email address or use filters to auto-label them for cleaner automation.

Quick Setup Guide

  1. Prepare your Google Sheet: Copy this template to your Drive (contains Invoices, Error Logs, and Success Metrics tabs).
  2. Import the workflow: Download the JSON file above and import it into your n8n instance via the "Import from File" option.
  3. Configure settings: Open the "Configuration: User Settings" node and paste your Google Sheet ID (from the URL) and admin email for error notifications.
  4. Set up credentials: Connect your Gmail, Google Sheets, OpenAI, and Google Gemini accounts in n8n's credentials management.
  5. Test and activate: Run the workflow once manually to ensure connections work, then activate the trigger for continuous monitoring.
