Find Facebook Hidden Interests via Telegram

Automate market research by extracting interests from Telegram messages and fetching related Facebook data.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves a common market research problem: manually monitoring community chats to understand audience interests and then researching those interests on social platforms. Businesses, marketers, and researchers spend hours scrolling through Telegram groups, noting topics, and then searching Facebook for related data.

The workflow automates this entire process. It listens for specific interest hashtags in a Telegram chat, extracts the interest keywords, queries the Facebook Graph API for related information, and organizes the results into a structured spreadsheet. This turns a multi-hour manual task into a real-time automated insight pipeline.

The value is clear: continuous, accurate market intelligence without manual labor. You gain insights into what your community is talking about and how those topics perform on Facebook—essential for content strategy, product development, and competitive analysis.

How It Works

Step 1: Telegram Trigger

The workflow starts by monitoring a designated Telegram chat. It listens for new messages that begin with a specific hashtag (like #interest). This trigger ensures only relevant messages are processed, filtering out general conversation.

Step 2: Message Validation & Extraction

When a qualifying message arrives, the workflow checks it against configured criteria (chat ID, hashtag format). It then extracts the message content, isolating the interest keyword from the rest of the text.

Step 3: Facebook Graph API Query

The extracted interest keyword is passed to the Facebook Graph API. The API searches for public data related to that interest—such as pages, groups, posts, or demographic insights—returning structured JSON results.

Step 4: Data Structuring & Table Creation

The API results are parsed and organized into a table format. Key fields like interest name, related Facebook entities, engagement metrics, and demographic indicators are extracted and mapped.

Step 5: Spreadsheet Generation & Delivery

The structured table is converted into a CSV spreadsheet file. This file is then sent back to the original Telegram chat, providing the research results directly to the requester in a usable format.

Pro tip: Configure the Telegram trigger to monitor multiple chats or use different hashtags for different research campaigns. This allows parallel tracking of various interest categories.

Quick Setup Guide

  1. Import the workflow: Download the JSON template and import it into your n8n instance via the workflow import function.
  2. Configure Telegram credentials: In the Telegram trigger node, add your bot token and set the webhook ID. Specify the chat ID to monitor.
  3. Set up Facebook API access: In the Facebook Graph API node, add your access token. Adjust the query parameters if needed (fields, limit).
  4. Adjust filtering logic: Modify the IF node conditions to match your desired hashtag format and chat validation rules.
  5. Test the workflow: Send a test message with the hashtag to your Telegram chat. Verify that the interest is extracted, Facebook data is retrieved, and the spreadsheet is returned.
  6. Activate the workflow: Once testing succeeds, activate the workflow. It will now run automatically whenever qualifying messages arrive.

Pro tip: Use n8n's schedule trigger to run periodic summary reports, compiling all extracted interests over a week into a master dashboard.
