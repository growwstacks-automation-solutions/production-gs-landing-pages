What This Workflow Does

Manual LinkedIn prospecting is slow, limited, and repetitive. Sales teams, recruiters, and marketers spend hours searching for specific professionals, copying profile details, and organizing data—only to hit LinkedIn's search restrictions and deal with inconsistent information.

This workflow automates the entire discovery process. It uses Google's powerful Custom Search API to find LinkedIn profiles matching your exact criteria—job titles, industries, locations—then extracts key information and saves everything to Airtable in a structured format. You get a clean, searchable database of prospects without manual effort.

The system handles pagination, deduplication, and rate limiting, ensuring reliable operation while maximizing the number of profiles discovered. It transforms a days-long research task into a minutes-long automated process.

How It Works

1. Keyword Configuration

You define target search terms like "Software Engineer React" or "Marketing Director London" in a simple configuration node. The workflow supports multiple keywords and location modifiers.

2. Google Search Execution

The workflow calls the Google Custom Search API with your keywords. Google returns LinkedIn profile URLs, titles, and snippets from its indexed database, bypassing LinkedIn's native search limitations.

3. Data Extraction & Parsing

Each search result is processed to extract the profile URL, headline, description, and search snippet. The system cleans and structures this data for consistent storage.

4. Airtable Integration

Extracted prospect information is automatically saved to your Airtable base. The workflow checks for duplicates before creating new records, maintaining data integrity.

5. Pagination & Scaling

For comprehensive results, the workflow automatically processes multiple pages of Google search results, with built-in delays to respect API rate limits.

What You'll Need

  1. Google Cloud Account with Custom Search API enabled (free tier available)
  2. Airtable Account (free plan works) with a base prepared for prospect data
  3. n8n Instance (cloud or self-hosted) to run the automation
  4. Search Engine ID from Google Programmable Search Engine configured to search the web
  5. API Key from Google Cloud for authentication

Pro tip: Start with 5-10 specific keywords to test the workflow before scaling to hundreds. Monitor your Google API usage to stay within free tier limits.

Quick Setup Guide

Follow these steps to get your LinkedIn prospecting automation running in under 30 minutes:

  1. Import the template into your n8n instance using the downloaded JSON file.
  2. Configure Google Credentials in n8n with your API key and Search Engine ID.
  3. Set up Airtable connection with your personal access token, base ID, and table name.
  4. Modify search keywords in the configuration node to match your target audience.
  5. Test with a single keyword to verify data flows correctly to Airtable.
  6. Schedule the workflow to run daily or weekly for continuous prospect discovery.
