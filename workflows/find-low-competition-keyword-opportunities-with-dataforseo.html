Find Low-Competition Keyword Opportunities with DataForSEO

Automate competitor keyword research to discover high-opportunity, low-competition keywords your business can actually rank for.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Manual keyword research is time-consuming and often misses the hidden opportunities that drive real SEO growth. This automated workflow solves that by systematically analyzing competitor domains to uncover every keyword they rank for, then evaluating each one for difficulty, search intent, and trend data.

Instead of guessing which keywords to target, you get a data-backed list of proven opportunities where competitors are already ranking but competition remains manageable. The workflow pulls from DataForSEO's comprehensive database, processes the data through multiple API endpoints, and delivers actionable insights directly to Google Sheets—ready for your content team to execute.

This approach transforms SEO from a reactive guessing game into a proactive, scalable strategy. You'll identify content gaps, monitor competitor movements, and build a keyword portfolio based on actual search performance rather than theoretical volume.

How It Works

The workflow follows a logical sequence to transform raw domain data into prioritized keyword opportunities.

Step 1: Read Competitor Domains from Google Sheets

The automation begins by reading your prepared Google Sheet containing competitor domains, along with their target location and language settings. This allows for geo-specific analysis—crucial for local SEO or international markets.

Step 2: Fetch All Ranking Keywords

For each domain, the workflow calls DataForSEO's "Keywords For Site" API to retrieve every keyword the domain currently ranks for. This reveals their complete organic search footprint, including long-tail terms you might not find through traditional keyword tools.

Step 3: Analyze Keyword Difficulty & Metrics

Each discovered keyword is then passed through DataForSEO's "Bulk Keyword Difficulty" API, which returns a 0-100 difficulty score, search volume, trend data (monthly, quarterly, yearly), and search intent classification. This step filters out highly competitive terms that would be unrealistic to target.

Step 4: Enrich with Backlink & SERP Data

The workflow combines difficulty metrics with additional SERP features like average backlink counts for top-ranking pages. This helps identify "low-hanging fruit"—keywords where the top results have relatively few backlinks, suggesting you could compete with quality content.

Step 5: Output Structured Results

All processed data is written back to a dedicated "keywords_opportunities" sheet, organized by domain with columns for keyword, volume, difficulty, intent, trends, and opportunity score. The output is immediately actionable for content planning and SEO strategy.

What You'll Need

  1. DataForSEO API credentials (Basic Auth) – Sign up for their free tier to get started.
  2. Google Sheets with input columns: seed domain, location_name, language_name, and result limit.
  3. An n8n instance (cloud or self-hosted) with HTTP Request and Google Sheets nodes configured.
  4. Basic understanding of SEO metrics and keyword research principles.

Pro tip: Start with 3-5 competitor domains in your niche to validate the workflow before scaling to larger analyses. Focus on domains with similar authority to yours for the most actionable insights.

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance.
  2. Create your input Google Sheet with columns: seed (domain), location_name (e.g., "United States"), language_name (e.g., "English"), limit (e.g., 100).
  3. Configure the DataForSEO node with your API credentials from their dashboard.
  4. Set up the Google Sheets nodes to read from your input sheet and write to your output sheet.
  5. Test with a single domain to verify the connection and data flow.
  6. Schedule the workflow to run monthly for ongoing competitive intelligence.
