Form with Dynamic Dropdown Field

Automatically update dropdown options from external data sources in your forms

Download Template JSON · n8n compatible · Free

What This Workflow Does

This n8n workflow solves the common problem of static form dropdowns that quickly become outdated. Traditional forms require manual updates whenever dropdown options change, creating administrative overhead and potential errors.

The template creates a form with a dynamic dropdown field that automatically pulls its options from an external data source like a database, spreadsheet, or API. Whenever the source data changes, the form dropdown updates instantly without any manual intervention.

How It Works

1. Data Source Connection

The workflow connects to your data source (Google Sheets, Airtable, database, etc.) and fetches the current values that should appear in the dropdown.

2. Form Generation

Using n8n's form builder capabilities, the workflow creates a customizable form with your dynamic dropdown field populated with the fetched data.

3. Real-time Updates

Whenever the form is accessed, the workflow checks the data source for updates and refreshes the dropdown options automatically.

Who This Is For

This template benefits businesses that need to collect information through forms where dropdown options frequently change. Common use cases include:

  • Product order forms with changing inventory
  • Event registration with dynamic ticket types
  • HR systems with updated department lists
  • Service booking forms with changing availability

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. Access to your data source (API credentials or file access)
  3. Basic familiarity with n8n workflows
  4. A form platform that supports dynamic fields (or webhook integration)

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure the data source node with your connection details
  3. Customize the form fields and appearance
  4. Set up the webhook or embed the form in your website
  5. Test with sample data to verify automatic updates
