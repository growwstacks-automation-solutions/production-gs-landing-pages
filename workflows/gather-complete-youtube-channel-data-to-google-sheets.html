What This Workflow Does

Manually researching YouTube channels for competitive analysis, influencer partnerships, or market intelligence is a tedious, time-consuming task. This free n8n automation template solves that by connecting directly to the YouTube Data API, fetching comprehensive channel information, and organizing it neatly into a Google Sheet—all with a single click.

The workflow extracts critical data points including the channel's title, description, custom URL, subscriber count, total view count, number of uploaded videos, country, keywords, thumbnail URLs, published date, and unique channel ID. This turns hours of manual copying and pasting into a fully automated, reliable process that ensures data accuracy and consistency.

Whether you're a marketing agency tracking competitor growth, a brand vetting potential influencer partners, or a content creator analyzing niche trends, this automation provides the structured data you need to make informed decisions without the manual grunt work.

How It Works

The automation follows a logical, step-by-step process to gather, validate, and store channel data.

1. Read Channel URLs from Google Sheets

The workflow starts by connecting to your designated Google Sheet. It reads a list of YouTube channel URLs from a "Channel URLs" tab. It only processes rows marked with a "Ready" status, allowing you to control which channels are analyzed.

2. Determine URL Type & Route Accordingly

A "Switch" node intelligently detects whether each input is a full YouTube channel URL (e.g., youtube.com/c/ChannelName) or a custom URL handle (e.g., @ChannelName). It routes each URL to the appropriate API request path within the YouTube Data API.

3. Fetch Data via YouTube API

Using authenticated HTTP Request nodes, the workflow calls the YouTube API's `channels.list` endpoint. It requests the specific statistics and snippet data for the channel, including the vital metrics needed for analysis.

4. Parse & Transform the Response

The raw JSON data from the API is parsed and transformed. The workflow extracts the relevant fields, formats dates and numbers consistently, and prepares a clean dataset for writing back to your spreadsheet.

5. Write Results & Update Status

The final step writes the fetched channel data—title, stats, description, etc.—back into the same row in your Google Sheet. It then updates the row's status to "Finished" for successful fetches or "Error" if the API call fails, providing clear visibility into the process.

Pro tip: Use the "Split In Batches" node if you have a large list of channels to avoid hitting YouTube API rate limits. Process channels in groups of 50-100 with a short pause between batches.

Quick Setup Guide

Get this automation running in your n8n environment in under 15 minutes.

  1. Download the Template: Click the "Download Template" button above to get the JSON file.
  2. Import into n8n: In your n8n dashboard, go to Workflows > Import from File and select the downloaded JSON.
  3. Duplicate the Google Sheet: Make a copy of the linked template sheet in your own Google Drive.
  4. Configure API Credentials: In the "HTTP Request" nodes, replace the placeholder API key with your own YouTube Data API key.
  5. Connect Google Sheets: Use OAuth2 or a service account to connect the "Google Sheets" nodes to your copied spreadsheet.
  6. Add Channel URLs: Paste the YouTube channel URLs you want to analyze into the "Channel URLs" tab of your sheet and set their status to "Ready".
  7. Execute Workflow: Click "Execute Workflow" in n8n. The automation will run, fetch the data, and populate your sheet.

Pro tip: For recurring analysis, add an "n8n Schedule Trigger" node to the start of the workflow. Set it to run weekly or monthly to keep your channel data continuously updated.
