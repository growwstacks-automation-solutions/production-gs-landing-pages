What This Workflow Does

This automation solves the time-consuming process of manually creating music playlists. Instead of spending hours searching for similar songs and building cohesive playlists, you can simply send a song recommendation to a Telegram bot and instantly receive a fully curated Spotify playlist.

The workflow acts as your personal AI DJ: it takes a single song input, uses AI to analyze musical patterns and find similar tracks, creates a new Spotify playlist with your username and the prompt, sends you the link immediately, and then populates the playlist with 25+ carefully matched songs in the background. It's perfect for music discovery, content creation, or simply expanding your music library with minimal effort.

How It Works

Step 1: Trigger via Telegram

You send a message to your Telegram bot with a song name and artist (e.g., "Daft Punk - One More Time"). The workflow immediately captures this input and begins processing.

Step 2: Instant Playlist Creation

Before any AI processing happens, the workflow creates an empty playlist on your Spotify account named with your username and the song prompt. It then sends the Spotify playlist link back to your Telegram chat so you have immediate feedback.

Step 3: AI Music Analysis

The song prompt is sent to an AI agent (via OpenRouter) configured as a "Greatest DJ." The AI analyzes the musical characteristics and generates a JSON list of approximately 25 similar tracks and artists, considering factors like genre, mood, tempo, and musical style.

Step 4: Automated Spotify Search & Population

The workflow splits the AI-generated list into individual items and loops through each one. For each track, it searches Spotify for the specific song and artist combination. If found, it adds the track to your playlist. The system includes intelligent error handling to skip tracks that can't be found and implements rate limiting (1-second delays between adds) to respect Spotify API limits.

Pro tip: You can easily swap the OpenRouter AI model for OpenAI GPT-4o, Anthropic Claude, or other LLMs by changing one node setting. This flexibility lets you choose the AI that best understands musical context for your needs.

  3. Telegram Bot Token created via @BotFather
  4. OpenRouter API Key (or alternative AI service credentials)
  5. Basic understanding of how to set up API credentials in n8n

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance.
  2. Configure Telegram credentials in both the Trigger and Send Message nodes using your bot token.
  3. Set up Spotify OAuth2 credentials in the Create Playlist, Search, and Add Item nodes with your Client ID and Secret.
  4. Add your OpenRouter API key to the Chat Model node (or switch to your preferred AI provider).
  5. Test the workflow by activating it and sending a song to your Telegram bot.
  6. Customize the AI prompt if you want specific playlist criteria (mood, era, genre restrictions).
