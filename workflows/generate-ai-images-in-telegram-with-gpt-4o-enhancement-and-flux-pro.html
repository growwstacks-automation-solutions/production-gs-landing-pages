What This Workflow Does

This automation solves the problem of manual image creation by providing an on-demand AI image generation service through Telegram. Businesses spend significant time and money on graphic design for marketing, social media, and product visuals. This workflow automates that process entirely.

When a user sends a text description to your Telegram bot, the system enhances the prompt using GPT-4o for better results, generates a high-quality image using the Flux Pro AI model, tracks usage per user with daily limits to control costs, logs all requests in Google Sheets for analytics, and delivers the final image back to the user with an AI-generated caption.

The solution is particularly valuable for content creators, marketing teams, e-commerce businesses, and community managers who need rapid visual content without design software skills or budget for professional designers.

How It Works

1. Telegram Message Trigger

The workflow activates when a user sends any message to your configured Telegram bot. The system captures the user's ID, message text, and chat details to begin the image generation process.

2. Usage Limit Check

Before processing, the workflow checks Google Sheets to count how many images this user has generated today. If they've exceeded the daily quota (default: 5), the system sends a friendly limit message and stops processing.

3. Prompt Enhancement with GPT-4o

The user's simple description (like "astronaut cat in space") is sent to GPT-4o, which expands it into a detailed, descriptive prompt optimized for image generation models, resulting in significantly better visual output.

4. AI Image Generation via Flux Pro

The enhanced prompt is sent to the AI/ML API's Flux Pro model, which generates a high-resolution image. This commercial-grade model produces professional-quality visuals suitable for business use.

5. Automated Caption Creation

Another AI call creates a descriptive caption for the generated image, adding context and making the output more engaging when shared or used in marketing materials.

6. Delivery and Logging

The final image is sent back to the user in Telegram. Simultaneously, the system logs the user ID, date, original query, and image URL in Google Sheets for tracking, analytics, and quota management.

Pro tip: Set different daily limits for different user groups by modifying the quota logic. Consider offering higher limits to premium subscribers or team members while keeping basic users at the default level.

What You'll Need

  1. Telegram Bot Token: Created through @BotFather on Telegram with basic configuration.
  2. Google Sheets Access: A Google account with Sheets API enabled, either through OAuth2 or service account authentication.
  3. AI/ML API Key: Account with AIMLAPI.com for accessing the Flux Pro image generation model and GPT-4o for prompt enhancement.
  4. n8n Instance: Self-hosted n8n or n8n.cloud account with the HTTP Request, Google Sheets, and Telegram nodes available.
  5. Google Sheet Prepared: A spreadsheet with columns for user_id, date, query, and result_url to log all image generations.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n instance using the "Import from File" option in the workflow editor.
  2. Configure Telegram Credentials: Add your bot token to the Telegram trigger node. Test that it can receive messages.
  3. Set Up Google Sheets: Connect your Google account in the credentials manager. Update the spreadsheet ID in the Google Sheets nodes to point to your prepared sheet.
  4. Add AI API Keys: Enter your AI/ML API credentials in the HTTP Request nodes for both GPT-4o enhancement and Flux Pro image generation.
  5. Adjust Daily Limits: Modify the "Set Daily Limit" node if you want to change the default 5 images per user per day.
  6. Test Thoroughly: Send a test message to your bot, verify the image generation works, and check that data logs correctly to Google Sheets.

Pro tip: Before going live, test with various prompt types to ensure your GPT-4o enhancement produces good results for your specific use case. You may want to customize the enhancement instructions for better alignment with your brand or industry.
