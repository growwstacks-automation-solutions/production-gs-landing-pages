What This Workflow Does

This automation solves the challenge of creating custom, royalty-free music for business content without musical expertise or expensive software. Instead of searching through generic stock music libraries or hiring composers, you can generate perfectly tailored audio in seconds.

The workflow connects ElevenLabs' advanced AI music generation with Google Drive's organizational power. You describe the music you need—whether it's "upbeat electronic background for a tech product video" or "calm acoustic guitar for a meditation app"—and receive a professionally generated MP3 file automatically saved to your designated Google Drive folder with a shareable link.

This eliminates manual file management, ensures consistent audio branding across projects, and provides an auditable archive of all generated music. Content teams, marketers, and creators can focus on their core work while maintaining high-quality audio production standards.

How It Works

Step 1: Text Description Input

The workflow starts with a simple web form where users enter their music requirements. This could be as detailed as "90s synthwave with driving bass and atmospheric pads" or as simple as "happy background music." The system captures this prompt along with any metadata like project name or intended use.

Step 2: AI Music Generation

The text prompt is sent to ElevenLabs' Music API, which analyzes the description and generates a unique audio composition. The AI considers musical theory, emotional tone, instrumentation, and structural elements to create a professional-quality MP3 file matching the request.

Step 3: File Processing & Organization

Once generated, the audio file is processed with proper naming conventions, metadata tagging, and quality verification. The system automatically creates appropriate folder structures in Google Drive based on project, date, or music type, ensuring organized storage from day one.

Step 4: Cloud Storage & Distribution

The final MP3 is uploaded to the designated Google Drive folder with correct permissions and sharing settings. A confirmation page provides the user with direct download links, embed codes, and file information for immediate use in their projects.

What You'll Need

  1. ElevenLabs API Key: A free or paid ElevenLabs account with Music API access
  2. Google Drive Account: Any Google Workspace or personal Google Drive with available storage
  3. n8n Instance: Self-hosted n8n or n8n.cloud account
  4. Web Form Access: Basic understanding of how to access and use the generated web form URL

Pro tip: Create a standardized naming convention for your music prompts (e.g., "ProjectName_Genre_Mood_Duration") to maintain consistency across generated files and make searching your Google Drive archive much easier.

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n instance
  2. Configure the ElevenLabs node with your API key from your account settings
  3. Set up Google Drive credentials in the upload node, selecting the destination folder
  4. Test the workflow with a simple prompt like "calm piano music"
  5. Activate the workflow and use the provided web form URL for your team
  6. Bookmark the Google Drive folder where all music will be automatically saved
