Automatically combine daily task progress and sales opportunities into intelligent summaries delivered to Slack, Email, and Google Drive.

What This Workflow Does

Manually compiling end-of-day reports is a time-consuming, error-prone task that pulls managers away from strategic work. This automation solves that by seamlessly connecting your project management (ClickUp) and sales pipeline (GoHighLevel) to deliver a unified, AI-powered performance snapshot every weekday.

The workflow triggers automatically at 6 PM, fetches all completed tasks and won opportunities, merges the data, and uses Azure OpenAI GPT-4 to generate insightful summaries. It then formats and distributes these reports across three channels: a concise Markdown version to your team's Slack channel, a polished HTML version via email to leadership, and a full-text version archived in Google Drive for historical reference.

How It Works

1. Scheduled Data Collection

A schedule trigger (cron) activates the workflow every weekday evening. It simultaneously queries your ClickUp workspace for tasks marked "complete" today and your GoHighLevel account for opportunities marked "won" today.

2. Data Merging & Transformation

The raw data from both platforms is cleaned and merged into a single, structured dataset. This creates a unified view where you can see, for example, that completing "Marketing Campaign Setup" tasks correlated with three new won deals in "Digital Services."

3. AI-Powered Analysis & Summarization

The consolidated data is sent to an AI model (like GPT-4) with a custom prompt designed to extract business insights. The AI doesn't just list data; it identifies key wins, potential blockers, trends, and suggests actionable next steps based on the day's activities.

4. Multi-Channel Distribution

The generated summary is then formatted for different audiences and sent out in parallel: a bullet-point version to Slack for immediate team discussion, a formal report via email for stakeholders, and a raw text file saved to a designated Google Drive folder.

Who This Is For

This template is ideal for operations managers, sales team leads, agency owners, and client success directors who need to track daily progress across both execution and revenue. If you spend more than 30 minutes at the end of each day copying data from different apps into a report, this automation will give you that time back with superior insights.

What You'll Need

  1. ClickUp Account: With OAuth2 access to read tasks from your specific Spaces, Folders, and Lists.
  2. GoHighLevel Account: With API credentials to fetch opportunity/contact data.
  3. Azure OpenAI API Key: Or access to another compatible LLM (OpenAI, Anthropic, etc.) for the analysis step.
  4. Slack Workspace: With a bot token and the channel ID where reports should be posted.
  5. Email SMTP Credentials: (Gmail, Outlook, etc.) or a transactional email service to send the HTML reports.
  6. Google Drive Access: With OAuth2 permissions to create and save files to a specific folder.

Pro tip: Test the workflow manually first with a small date range. Verify the data pulled from ClickUp and GoHighLevel is correct before letting it run automatically. This ensures your AI summaries are based on accurate information.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n instance.
  2. Configure Credentials: Set up the connection nodes for ClickUp, GoHighLevel, Azure OpenAI, Slack, your Email service, and Google Drive using your API keys and OAuth2 credentials.
  3. Set Your IDs: In the ClickUp node, specify your Team ID, Space ID, Folder ID, and List ID. In the Slack node, enter your target channel ID.
  4. Customize Recipients: In the Email node, set the "From" address and the "To" recipients (you can add multiple emails for leadership distribution).
  5. Adjust Schedule (Optional): The default cron is set for 6 PM Monday-Friday. Modify the Schedule Trigger node if you need reports at a different time or on different days.
  6. Test & Activate: Run the workflow once manually to ensure all connections work and the report looks good. Then activate it for automated, daily execution.
