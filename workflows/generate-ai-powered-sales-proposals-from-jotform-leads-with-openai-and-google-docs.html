What This Workflow Does

Manual proposal writing is a time-consuming bottleneck for sales teams. Every new lead requires researching their needs, crafting a tailored document, and sending it promptly—often taking hours per proposal. This workflow solves that by automating the entire process.

When a potential client submits information via a JotForm (like a contact form, consultation request, or quote inquiry), this automation instantly triggers. It uses OpenAI to analyze the lead's data and generate a professionally written, personalized sales proposal. The proposal is then formatted in Google Docs, converted to PDF, and emailed directly to the lead via Gmail—all within minutes.

The result: sales teams can respond to leads instantly with high-quality proposals, dramatically increasing conversion rates while freeing up hours each week for more strategic activities.

How It Works

Step 1: Capture Lead Data

The workflow starts when a new submission arrives in your JotForm. It captures all form fields—company name, contact details, project requirements, budget range, and any custom questions you've set up.

Step 2: AI Proposal Generation

The lead data is sent to OpenAI (GPT-4 or similar model) with a tailored prompt that instructs the AI to create a sales proposal. The AI incorporates your company's tone, service offerings, and pricing templates to produce a draft that feels personal and professional.

Step 3: Document Creation & Formatting

The AI-generated text is inserted into a Google Docs template. This template includes your branding, logo, standard sections, and formatting. The workflow populates the document with the proposal content and dynamically names it (e.g., "Company Name | Proposal").

Step 4: PDF Conversion & Delivery

The Google Doc is automatically converted to a PDF for easy sharing and printing. Then, using Gmail, the PDF is attached and sent to the lead's email address from the form. The email includes a personalized message and follow-up instructions.

What You'll Need

  1. A JotForm account with an active form collecting lead information.
  2. An OpenAI API key (access to GPT-4 or similar models).
  3. Google Drive and Google Docs access with OAuth2 credentials.
  4. A Gmail account for sending proposals (or Google Workspace).
  5. An n8n instance (cloud or self-hosted) to run the workflow.
  6. A basic proposal template in Google Docs to serve as the formatting foundation.

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n workspace.
  2. Configure the JotForm trigger node with your form ID and API key.
  3. Set up the OpenAI node with your API key and adjust the prompt if needed.
  4. Connect the Google Drive and Google Docs nodes using OAuth2.
  5. Link the Gmail node with your sending account credentials.
  6. Test the workflow with a sample form submission to verify the output.
  7. Activate the workflow and start automating proposals instantly.
