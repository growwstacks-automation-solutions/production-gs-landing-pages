What This Workflow Does

This n8n workflow automates the process of discovering high-value keywords for your Amazon FBA products by leveraging Amazon's own Completion API. The API powers Amazon's search suggestions, revealing what real customers are actually searching for on the platform.

Manual keyword research is time-consuming and often misses valuable long-tail opportunities. This workflow systematically queries Amazon's autocomplete to generate hundreds of relevant keywords in minutes, giving you data-driven insights to optimize your product listings and PPC campaigns.

Example of keyword suggestions from Amazon's autocomplete API

How It Works

1. Input your seed keywords

The workflow starts with your initial product keywords or ASINs. These serve as the foundation for discovering related search terms that actual Amazon shoppers are using.

2. Query Amazon's Completion API

The workflow makes intelligent API calls to Amazon's autocomplete service, simulating how real customers search on Amazon. It systematically tests different keyword combinations and variations.

3. Process and organize results

Raw API responses are cleaned and organized into a structured list of keywords. The workflow removes duplicates and irrelevant terms, focusing on commercially valuable search phrases.

The automated workflow processes and organizes keyword data

4. Output ready-to-use keywords

Final results are formatted for immediate use in your Amazon Seller Central campaigns or product listings. You can export to CSV or connect directly to other tools.

Pro tip: Combine this workflow with competitor ASIN analysis to uncover keywords your competitors are ranking for but you might be missing.
