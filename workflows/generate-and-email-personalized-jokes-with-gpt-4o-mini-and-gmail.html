Generate & Email Personalized Jokes with GPT-4o-mini & Gmail

Automate the creation and delivery of unique, funny content to engage your audience—no manual writing required.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation turns a simple form submission into a personalized joke delivered via email. It’s perfect for adding a touch of humor to customer onboarding, team morale emails, or even lighthearted marketing campaigns. Instead of spending time crafting individual jokes, you set up this workflow once and let AI handle the creative work.

The process starts when someone submits a form with names or topics. The workflow passes this input to GPT-4o-mini, which generates a unique, context-aware joke. Then, it automatically sends that joke via Gmail to a designated recipient. You can customize the form fields, joke tone, and email formatting to match your brand or audience.

Beyond entertainment, this template demonstrates a powerful pattern: using AI to generate personalized content and distribute it automatically. It saves hours of manual effort, ensures consistency, and can be adapted for many other personalized communication needs.

How It Works

Step 1: Form Trigger

The workflow is triggered by a form submission. You can embed an n8n Form Trigger node on your website or use a third-party form tool. The submitted data (like a person's name or a topic) is captured and passed to the next step.

Step 2: Set Output Language

A Set node allows you to manually define the language for the joke output (e.g., English, Spanish). This ensures the AI generates jokes in the correct language for your audience.

Step 3: OpenAI Message Model

The OpenAI node takes the form input and language setting, then crafts a personalized joke. GPT-4o-mini is optimized for quick, creative text generation at a low cost. The prompt is engineered to produce humorous, relevant content based on the provided names or objects.

Step 4: Gmail Send Message

The generated joke is formatted into an email and sent automatically via Gmail. You configure the sender, recipient, subject line, and any additional email styling. The workflow ensures each email is unique and personally addressed.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. OpenAI API key (for GPT-4o-mini access)
  3. Gmail account with OAuth2 credentials configured in n8n
  4. A form or data source to trigger the workflow (n8n Form Trigger, Google Forms, etc.)
  5. Basic understanding of n8n node configuration

Quick Setup Guide

1. Download the template JSON file from this page.

2. Import it into your n8n workspace via the Import from File function.

3. Configure the OpenAI node with your API key and adjust the prompt if desired.

4. Set up the Gmail node with your OAuth2 credentials and specify the recipient email.

5. Test the workflow by submitting a sample form input and verifying the email output.

6. Deploy the workflow as a webhook or schedule it to run automatically.

Pro tip: You can replace the Form Trigger with any data source—like a CRM update, a new Slack message, or a scheduled timer—to trigger joke generation based on different events.
