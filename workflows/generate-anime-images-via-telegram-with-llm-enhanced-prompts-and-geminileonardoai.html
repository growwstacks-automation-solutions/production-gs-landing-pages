What This Workflow Does

This automation transforms simple text ideas into professional-quality anime-style images through Telegram. It solves the challenge of quickly generating custom anime visuals without artistic skills by combining:

  • Telegram's mobile-friendly interface for prompt submission
  • LLM-powered prompt enhancement for better image results
  • Choice of Gemini (free) or Leonardo.AI (premium) image generation
  • Automatic delivery of multiple image variations back to Telegram
Sample output showing detailed anime character generation from text prompts

How It Works

1. Telegram Trigger

Users send a text message to your Telegram bot with their image idea (e.g., "cyberpunk samurai with neon katana").

2. Prompt Enhancement

The workflow uses an LLM (DeepSeek via OpenRouter by default) to expand the simple idea into detailed anime-ready prompts with style descriptors.

3. Image Generation

Each enhanced prompt gets sent to either Gemini (Google's AI) or Leonardo.AI (specialized anime models) to generate visual interpretations.

4. Result Delivery

The system converts AI outputs into image files and sends them back through Telegram, typically producing 4 variations per prompt.

Pro tip: For consistent character designs, include specific descriptors like hair color, eye style, and clothing details in your initial prompt.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure Telegram bot credentials in the "Configure Me!" nodes
  3. Set up your preferred LLM provider (default: OpenRouter)
  4. Choose image generator (Gemini or Leonardo.AI) and add API keys
  5. Adjust number of images per prompt in the counter node if needed
  6. Activate workflow and test with simple Telegram messages
