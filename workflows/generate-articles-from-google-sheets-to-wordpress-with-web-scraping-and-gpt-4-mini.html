What This Workflow Does

This automation solves the time-consuming challenge of manual content creation by building a complete pipeline from idea to publication. Instead of spending hours researching, writing, and publishing articles, this workflow monitors a Google Sheet for new topics, automatically researches source material, generates original content using AI, and publishes drafts to WordPress—all while tracking progress back to your spreadsheet.

For content marketers, bloggers, and publishers, this means scaling content production without scaling your team. The system handles the repetitive work while you focus on strategy and quality control. It's particularly valuable for agencies managing multiple clients, SEO specialists creating keyword-focused content, and news publishers automating article aggregation.

How It Works

1. Sheet Monitoring & Topic Detection

The workflow continuously monitors your Google Sheet for rows marked "New" in the status column. When a new topic is detected, it extracts the article subject and any source URLs provided for reference material.

2. Content Research & Web Scraping

Using the HTTP Request node, the system fetches content from provided source URLs. This ensures the AI has current, relevant material to reference rather than generating content from scratch without context.

3. AI-Powered Article Generation

Dual AI agents process the scraped content: one summarizes and analyzes the source material, while the other creates an original, SEO-optimized article following your specified tone, structure, and formatting guidelines.

4. WordPress Draft Creation

The generated article is automatically formatted and published as a draft in your WordPress site, complete with proper categories, tags, and metadata. The system can be configured to publish immediately or wait for editorial review.

5. Status Tracking & Reporting

Every step is tracked back to your Google Sheet, updating status columns and providing direct links to created drafts. This creates full visibility into your content pipeline's performance.

What You'll Need

  1. Google Sheets API access with OAuth2 credentials configured
  2. OpenAI API key or compatible AI service access
  3. WordPress REST API credentials with appropriate permissions
  4. A Google Sheet with columns for: Topic, Source URL, Flow Status, Publish Status, and Publish Link
  5. n8n instance (self-hosted or cloud) with HTTP Request, Google Sheets, and OpenAI/LangChain nodes available

Quick Setup Guide

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure credentials for Google Sheets, OpenAI, and WordPress in n8n's credential management
  3. Update the Google Sheet ID in all Sheet nodes to point to your actual spreadsheet
  4. Adjust column mappings to match your sheet's exact column names and structure
  5. Test with a single row marked "New" to verify the complete pipeline works end-to-end
  6. Customize AI prompts in the Article Creator node to match your desired writing style and format
  7. Set up scheduling to run the workflow at your preferred frequency (hourly, daily, etc.)
