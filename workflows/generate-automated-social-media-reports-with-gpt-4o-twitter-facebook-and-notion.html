What This Workflow Does

Manual social media reporting consumes valuable marketing hours each week—collecting data from multiple platforms, formatting spreadsheets, and trying to extract meaningful insights from raw numbers. This automated workflow eliminates that drudgery by creating a complete, AI-enhanced reporting system that runs on autopilot.

The system connects to Twitter (X) and Facebook APIs to pull real-time engagement metrics, follower counts, and post performance data. It then processes this information through GPT-4o to generate human-readable insights, trends, and recommendations. Finally, it distributes polished reports to your team via Notion databases, Slack notifications, and formatted email summaries—all without manual intervention.

How It Works

1. Data Collection from Social Platforms

The workflow starts by fetching the latest metrics from both Twitter and Facebook APIs simultaneously. It retrieves follower growth, engagement rates, reach, impressions, and top-performing content from your specified accounts and time periods.

2. Data Validation & Normalization

Before analysis, the system validates that complete datasets were received from both platforms and normalizes the metrics into a consistent JSON structure. This ensures apples-to-apples comparisons and prevents reporting errors from missing data.

3. AI-Powered Insight Generation

The normalized data flows into Azure OpenAI's GPT-4o model, which analyzes trends, identifies performance patterns, highlights anomalies, and generates professional HTML reports complete with tables, emojis, and actionable recommendations.

4. Centralized Storage in Notion

All processed analytics are automatically saved to a designated Notion "Growth Chart" database, creating a historical record for trend analysis and making insights accessible to your entire marketing team in one centralized location.

5. Multi-Channel Distribution

The system simultaneously sends formatted email reports to stakeholders, posts concise Slack updates comparing platform performance, and logs any errors to Google Sheets for monitoring—ensuring the right people get the right information through their preferred channels.

Who This Is For

This automation is ideal for marketing teams, social media managers, digital agencies, and growth-focused businesses that need consistent, insightful reporting without the manual effort. It's particularly valuable for teams managing multiple brand accounts, agencies serving multiple clients, and data-driven organizations that want to combine social metrics with other business intelligence.

What You'll Need

  1. Twitter API credentials with access to public metrics for your accounts
  2. Facebook Graph API access token for page insights
  3. Azure OpenAI API key for GPT-4o report generation
  4. Notion API integration with write access to your database
  5. Gmail OAuth2 setup for email distribution
  6. Slack Bot Token with chat:write permissions
  7. Google Sheets access (optional, for error logging)

Pro tip: Start with weekly reporting cadence, then adjust based on your needs. The workflow can run daily, weekly, or monthly—whatever provides optimal insights without overwhelming your team with data.

Quick Setup Guide

  1. Download the template and import it into your n8n instance
  2. Connect all required API credentials in their respective nodes
  3. Configure your Facebook Page ID and Twitter handle in the API nodes
  4. Update the Notion database ID to match your "Growth Chart" structure
  5. Add recipient emails in the Gmail node and specify your Slack channel
  6. Test the workflow manually to verify data flow and formatting
  7. Schedule it to run automatically (weekly recommended)

Automated reports handle data collection and basic analysis, freeing humans for strategic interpretation. The best approach combines AI-generated insights with human context about business goals, campaign nuances, and market conditions that algorithms can't fully understand.
