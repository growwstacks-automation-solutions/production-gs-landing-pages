What This Workflow Does

Creating fresh, engaging social media content daily is a constant challenge for marketers and content creators. This workflow solves that problem by automating the entire process—from discovering trending topics to generating ready-to-post captions.

The automation runs daily at noon, pulling search phrases from Google Sheets, fetching real-time Google autocomplete suggestions via Dumpling AI, and transforming them into polished captions using GPT-4o. The final captions are saved back to Google Sheets for easy scheduling and review.

Instead of spending hours brainstorming content, you get a daily batch of trend-based captions that resonate with what people are actually searching for right now.

How It Works

1. Daily Schedule Trigger

The workflow activates automatically every day at 12 PM, ensuring you receive fresh content suggestions without manual intervention.

2. Fetch Base Keywords

It pulls your predefined search phrases from a Google Sheet. These serve as starting points for discovering related trending topics through autocomplete.

3. Get Autocomplete Suggestions

Using Dumpling AI's API, the workflow queries Google's autocomplete feature for each base phrase, returning what real users are searching for right now.

4. Process and Format Data

The raw autocomplete suggestions are formatted into a clean array, preparing them for individual processing through the AI caption generator.

5. Generate AI Captions

Each autocomplete suggestion is sent to GPT-4o with specific instructions to create engaging, human-sounding captions under 280 characters with appropriate tone and calls-to-action.

6. Save to Google Sheets

The original search suggestions and their corresponding AI-generated captions are saved to a separate Google Sheet tab, creating an organized content library for scheduling.

Quick Setup Guide

Follow these steps to get this automation running in your n8n instance:

  1. Download the template using the button above and import it into your n8n workspace
  2. Configure Google Sheets nodes by connecting your Google account and specifying the spreadsheet IDs for input and output
  3. Set up Dumpling AI credentials by adding your API key to the HTTP Request node
  4. Connect OpenAI by adding your API key to the AI node and adjusting the prompt if needed
  5. Test the workflow manually first to ensure all connections work properly
  6. Activate the schedule to run daily at your preferred time (default is 12 PM)
  7. Monitor the output in your Google Sheets and refine your base search phrases based on results
