How It Works

The workflow follows a sophisticated four-stage pipeline that mimics how expert researchers operate, but at machine speed and scale.

1. Query Optimization & Generation

Google Gemini AI analyzes your initial research topic and generates three optimized search queries designed to capture different perspectives and comprehensive coverage. This ensures you're not just getting surface-level information but diving deep into the subject from multiple angles.

2. Multi-Source Research Execution

Tavily's advanced search engine executes all three queries simultaneously, gathering information from credible sources across the web. It doesn't just return links—it provides AI-generated answers with source citations, saving you from visiting dozens of websites manually.

3. Intelligent Report Synthesis

Another Google Gemini instance consolidates the gathered information, eliminates duplicates, identifies key insights, and structures everything into a coherent HTML report. The system understands context, prioritizes important findings, and creates logical sections with proper formatting.

4. Professional Delivery

The final HTML report is automatically sent via Gmail to your specified recipients. The email includes proper formatting, clear subject lines, and can be scheduled to run at regular intervals for ongoing monitoring.

Quick Setup Guide

Follow these steps to get your automated research system running in under 30 minutes:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure API credentials in all three Google Gemini nodes and the Tavily Search node
  3. Set up Gmail authentication in the "Send Email" node with your sender account
  4. Update recipient email addresses in the email node settings
  5. Customize your default query in the "Edit Fields" node (currently set to "n8nとdifyの違い" for testing)
  6. Test with a simple topic to ensure all connections work properly before scaling up
  7. Schedule regular runs using n8n's scheduler for ongoing research needs
