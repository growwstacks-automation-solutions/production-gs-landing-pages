What This Workflow Does

This automation workflow solves a critical challenge for bakeries, caterers, and event planners: converting customer cake design requests into visual concepts quickly and professionally. Traditionally, creating custom cake designs requires skilled artists, multiple revisions, and significant time investment. This n8n template automates the entire process using cutting-edge AI technology.

The workflow takes customer descriptions (like "rainbow unicorn birthday cake with glitter" or "elegant wedding cake with fresh flowers") and generates high-quality visual mockups. It combines OpenAI's GPT for understanding natural language requests with Replicate's Flux Schnell model for state-of-the-art image generation. The result is an automated design assistant that can produce multiple cake concepts in minutes instead of hours.

The n8n workflow interface showing how customer requests are processed through OpenAI GPT and transformed into visual cake designs

How It Works

Step 1: Customer Input Processing

The workflow begins by receiving customer cake design requests. This could come from various sources like web forms, email inquiries, or CRM systems. The input captures details about cake type, occasion, preferred colors, themes, and any specific decorative elements the customer wants.

Step 2: AI-Prompt Refinement with OpenAI GPT

OpenAI GPT analyzes the customer's description and refines it into a detailed, optimized prompt for image generation. This step ensures the AI image model receives clear, specific instructions that will produce the best possible results. GPT adds relevant details about cake styling, lighting, and composition that might be missing from the customer's initial request.

Step 3: Image Generation with Replicate Flux Schnell

The refined prompt is sent to Replicate's Flux Schnell model, which generates high-resolution cake design images. This model specializes in creating photorealistic food images with accurate textures, colors, and lighting. The generated images show the cake from multiple angles with professional food photography quality.

Step 4: Delivery to Customer or Team

Finally, the generated cake design images are automatically delivered to the appropriate recipient. This could be the customer (for approval), the bakery's design team (for refinement), or directly into order management systems. The workflow can also store the images for future reference and catalog building.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. OpenAI API account with GPT access
  3. Replicate account with API credentials
  4. A source for customer requests (webhook, form, email, CRM)
  5. Destination for generated images (cloud storage, email, CMS)

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n instance
  2. Configure your OpenAI and Replicate API credentials in the respective nodes
  3. Set up your trigger node (webhook, email, or form) to receive customer requests
  4. Configure the output nodes to deliver generated images to your preferred destination
  5. Test the workflow with sample cake descriptions to ensure proper image generation
  6. Deploy the workflow and integrate it with your customer-facing systems

Pro tip: Start with specific customer descriptions that include colors, themes, and occasion details for best results. The more detail provided, the more accurate the generated images will be.
