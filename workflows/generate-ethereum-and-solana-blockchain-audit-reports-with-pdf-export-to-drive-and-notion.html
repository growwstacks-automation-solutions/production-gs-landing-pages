What This Workflow Does

This enterprise-grade automation transforms how financial institutions, crypto businesses, and compliance teams handle blockchain transaction monitoring. Instead of manually checking wallets and spreadsheets, this system automatically tracks transactions across Ethereum and Solana networks, analyzes them for risk factors, generates professional audit reports as PDFs, and stores everything in organized Google Drive folders while updating a central Notion database.

The workflow solves the critical compliance challenge of maintaining accurate, timely audit trails for cryptocurrency transactions—a requirement for regulated businesses dealing with digital assets. It eliminates hours of manual work while providing standardized documentation that meets regulatory scrutiny.

How It Works

1. Multi-Chain Transaction Monitoring

The system connects to Ethereum via Alchemy API and Solana through RPC endpoints, continuously monitoring specified wallet addresses for new transactions. It captures transaction details including amounts, timestamps, counterparties, and gas fees.

2. Intelligent Risk Analysis & Scoring

Each transaction undergoes automated risk assessment using customizable parameters. The system evaluates factors like transaction size, frequency, counterparty history, and network patterns to generate a risk score (0-100) and flag suspicious activity.

3. Professional PDF Report Generation

Using APITemplate.io, the workflow creates standardized audit reports with company branding, transaction details, risk assessments, and compliance summaries. These professional PDFs include timestamps, unique identifiers, and all necessary documentation for regulatory purposes.

4. Automated Document Management

Generated reports are automatically uploaded to Google Drive with intelligent folder organization (by date, wallet address, and risk level). Simultaneously, a comprehensive audit entry is created in Notion, linking to the PDF and including all transaction metadata for easy search and retrieval.

5. Multi-Channel Alert System

When high-risk transactions are detected or audit cycles complete, the system sends detailed notifications to compliance teams via email, including summary data and direct links to the full reports in Drive and Notion.

What You'll Need

  1. Alchemy API key for Ethereum network access
  2. Solana RPC endpoint (public or private)
  3. APITemplate.io account with PDF template configured
  4. Google Service Account with Drive API access
  5. Notion integration token and database ID
  6. SMTP credentials for email notifications
  7. Etherscan API key (optional, for enhanced verification)

Pro tip: Start with a test wallet containing minimal funds to verify the automation works correctly before connecting production wallets. This allows you to fine-tune risk parameters without exposing significant assets.

Quick Setup Guide

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure blockchain connections by adding your Alchemy and Solana RPC credentials
  3. Set up APITemplate.io with your audit report design template
  4. Connect Google Drive and specify the target folder structure
  5. Link your Notion database and map the field structure
  6. Add wallet addresses you want to monitor for transactions
  7. Test with a small transaction to verify the complete flow from detection to documentation
