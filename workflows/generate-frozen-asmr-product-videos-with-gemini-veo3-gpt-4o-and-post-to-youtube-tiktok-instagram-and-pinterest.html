What This Workflow Does

This automation solves the massive time drain of creating engaging video content for social media. Instead of spending hours filming, editing, and posting, this workflow automatically transforms simple product images into viral-ready frozen ASMR videos using cutting-edge AI.

The system pulls product data from Google Sheets, applies stunning frozen visual effects via Gemini AI, generates authentic ASMR audio and video with Veo3, crafts platform-optimized captions using GPT-4o, and publishes simultaneously to four major social platforms. What used to take a content team 10-20 hours per week now runs completely autonomously.

How It Works

1. Data Trigger & Image Processing

A scheduled trigger checks your Google Sheet for new products. It downloads the product image and sends it to Google's Gemini AI with specific instructions to apply a "frozen" or "ice effect" visual transformation, creating a unique, eye-catching visual.

2. Video Generation & Enhancement

The enhanced image is uploaded to a temporary hosting service (ImgBB) to create a publicly accessible URL. This URL is then sent to Veo3 (via Kie.ai) which generates a 10-12 second ASMR-style video complete with realistic ice cracking sounds and subtle visual animations that captivate viewers.

3. AI-Powered Content Creation

GPT-4o analyzes the product and generated video to create platform-specific titles, descriptions, and hashtags. It tailors the messaging for YouTube's search-focused audience, TikTok's trend-driven community, Instagram's visual platform, and Pinterest's inspiration-seeking users.

4. Multi-Platform Publishing & Tracking

The workflow simultaneously uploads the final video to all four platforms using their respective APIs. Once published, it updates the Google Sheet status from "processing" to "uploaded" and sends a Telegram notification to your team with performance links.

What You'll Need

  1. Google AI API key for Gemini image processing (available through Google AI Studio)
  2. Kie.ai account for Veo3 video generation (offers realistic ASMR video creation)
  3. ImgBB API key for temporary image hosting (free tier available)
  4. OpenAI API key for GPT-4o caption generation
  5. Upload-post.com account with connected TikTok, Instagram, and Pinterest profiles
  6. YouTube channel with OAuth 2.0 access enabled
  7. Google Sheet with product data formatted with columns for topic, image_url, and status
  8. Telegram bot and chat ID for notifications (optional but recommended)

Quick Setup Guide

Follow these steps to implement this automation in your business:

  1. Download the template using the button above and import it into your n8n instance
  2. Configure API credentials by replacing all placeholder keys in the workflow nodes with your actual API keys
  3. Set up your Google Sheet with the required columns: topic, image_url, and status (leave status empty for new products)
  4. Connect social accounts through upload-post.com for TikTok, Instagram, and Pinterest, and set up YouTube OAuth
  5. Test with one product by adding a single row to your sheet and triggering the workflow manually
  6. Schedule automation once testing is successful—set the schedule trigger to run daily or weekly based on your content needs
  7. Monitor and optimize by reviewing performance metrics and adjusting AI prompts for better results

Pro tip: Create a "prompt library" document with variations for different product categories. For example, fashion items might use different frozen effect descriptions than electronics. This helps the AI generate more appropriate content for each product type.
