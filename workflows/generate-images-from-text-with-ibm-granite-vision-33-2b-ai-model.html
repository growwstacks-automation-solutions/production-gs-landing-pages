What This Workflow Does

This n8n workflow automates AI-powered image generation using IBM's Granite Vision 3.3 2B model. It transforms text descriptions into visual content automatically, handling the entire process from API request to final image delivery. Businesses spend thousands on graphic design and stock photography—this workflow eliminates those costs by generating custom images on demand.

The automation connects to Replicate's API hosting of the Granite Vision model, sends your text prompts with customizable parameters like creativity level and image style, monitors the generation process, and delivers the finished image URL. It's like having a 24/7 graphic designer who works instantly from written instructions.

How It Works

1. Trigger & API Setup

The workflow starts with a manual trigger or can be scheduled automatically. It securely loads your Replicate API credentials, keeping sensitive information protected within the workflow environment rather than hard-coded in multiple places.

2. Image Generation Request

Your text prompt is sent to IBM's Granite Vision model via HTTP request. You can control parameters like creativity (temperature), randomness (seed), and output dimensions to match your brand requirements. The model understands complex descriptions and generates corresponding visuals.

3. Automated Polling & Status Monitoring

Since AI image generation takes seconds to complete, the workflow automatically checks the job status every few seconds. It uses conditional logic to wait patiently until the image is ready, then proceeds to the next step—no manual checking required.

4. Result Processing & Delivery

Once generation completes, the workflow extracts the image URL, metadata, and generation metrics. The output is structured cleanly for immediate use in other systems or direct download.

Pro tip: Connect this workflow to your content calendar or CMS to automatically generate featured images when new articles are published. The AI can read article summaries and create relevant visuals without human intervention.
