What This Workflow Does

This n8n workflow automates the entire process of creating professional interior design moodboards from a simple client brief. Instead of spending hours manually searching for inspiration images, compiling visuals, and designing presentation layouts, this system does it all automatically. Clients submit their design preferences through a form, and within minutes, they receive a beautifully formatted, two-page PDF moodboard delivered directly to their inbox.

The workflow combines multiple AI technologies: Claude and Gemini agents analyze the design brief and generate detailed image prompts, Hugging Face's FLUX model creates high-quality interior visuals, and automated systems compile everything into a polished, brand-consistent presentation. This eliminates the most time-consuming parts of the design consultation process while delivering superior visual results that help clients visualize concepts before any physical work begins.

For interior designers, architects, and design studios, this represents a fundamental shift in how client presentations are created. What traditionally took 6-8 hours of manual work now happens automatically, freeing up creative professionals to focus on higher-value tasks like client relationships, project management, and detailed design development.

How It Works

The automation follows a sophisticated seven-stage process that transforms a text description into a complete visual presentation.

1. Client Brief Collection

The process begins with a simple web form where clients describe their vision. They provide a project title, detailed description (including colors, materials, mood, and lighting preferences), and contact email. This form can be embedded on your website or sent directly to clients.

2. AI-Prompt Refinement

Claude Sonnet 4 (via OpenRouter) analyzes the client's brief and expands it into 12 detailed image prompts (300-500 words each). The AI considers architectural styles, material textures, color psychology, and spatial relationships to create comprehensive visual descriptions that will generate coherent images.

3. Visual Generation Loop

Each of the 12 prompts is sent sequentially to Hugging Face's FLUX.1-schnell model, which generates high-quality interior design images. The system creates 11 individual element shots (furniture, materials, details) plus one comprehensive 3D-rendered room view that integrates all design elements.

4. Secure Storage & Sharing

Generated images are automatically uploaded to your Nextcloud instance (or any cloud storage) and made available via public shareable URLs. This creates a permanent archive of all generated assets while ensuring clients can view them without needing login credentials.

5. Professional Layout Creation

Google Gemini 2.5 Pro receives all 12 image URLs and the original design brief, then generates a two-page HTML moodboard. Page 1 features a visually striking collage with the main 3D render prominently displayed. Page 2 includes an administrative summary with design overview, color palette, material specifications, and project details.

6. PDF Conversion

The HTML moodboard is converted to a print-ready PDF using the Gotenberg service, ensuring proper A4 dimensions, page breaks, and high-resolution image rendering. The system handles all formatting complexities automatically.

7. Automated Delivery

The final PDF is emailed directly to the client with a personalized message. The entire workflow—from brief submission to delivered moodboard—typically completes in 5-8 minutes, compared to days in traditional design processes.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. OpenRouter API key for accessing Claude Sonnet 4 (approximately $0.01 per moodboard)
  3. Google Gemini API key from Google AI Studio (free tier available)
  4. Hugging Face account with API access to FLUX.1-schnell or similar image generation model
  5. Nextcloud instance or alternative cloud storage (Dropbox, Google Drive via n8n nodes)
  6. Gotenberg PDF service (self-hosted via Docker or cloud service)
  7. Email service (Gmail SMTP or professional email service provider)
  8. Basic understanding of n8n node configuration and API key management

Pro tip: Start with the free tiers of each service to test the workflow. OpenRouter offers initial credits, Google Gemini has a generous free tier, and Hugging Face provides limited free inference. You can upgrade as your volume increases.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance using the download button above.
  2. Configure credentials for each service node (OpenRouter, Google Gemini, Hugging Face, Nextcloud, Gmail).
