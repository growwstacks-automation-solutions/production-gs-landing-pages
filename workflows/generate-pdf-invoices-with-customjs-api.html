What This Workflow Does

This n8n workflow automates the entire PDF invoice generation process, transforming raw billing data into professional, branded invoices ready for distribution. It eliminates the tedious manual work of creating invoices while ensuring accuracy and consistency across all documents.

The system captures invoice details through a webhook or API connection, processes the data with CustomJS for dynamic calculations and formatting, then generates a polished PDF with your company branding. The finished invoice can be automatically emailed to clients, saved to cloud storage, or forwarded to your accounting system.

Complete n8n workflow for automated PDF invoice generation

How It Works

1. Data Collection

The workflow begins by receiving invoice data through a webhook or API call. This typically includes client details, line items, taxes, and payment terms. The data can come from your CRM, accounting software, eCommerce platform, or custom database.

2. Data Processing

CustomJS nodes transform the raw data into the required invoice structure. This stage handles calculations (subtotals, taxes, discounts), applies business rules, and formats all values consistently. Conditional logic can apply different rates or templates based on client type or location.

CustomJS nodes processing invoice data with dynamic calculations

3. PDF Generation

The processed data feeds into the PDF creation node where CustomJS generates the final document. This includes applying your company branding, logo, colors, and legal requirements. The template supports dynamic elements like due dates, payment terms, and personalized messages.

4. Output & Delivery

Completed invoices can be routed through multiple channels simultaneously. Common outputs include email attachments, cloud storage (Google Drive, Dropbox), accounting software integrations, or direct printing for physical mail.

What You'll Need

  2. Access to your invoice data source (API, webhook, or database)
  3. Your company branding assets (logo, colors, legal text)
  4. Basic understanding of JavaScript for CustomJS adjustments
  5. Destination setup for invoices (email SMTP, cloud storage, etc.)

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance (Settings → Workflows → Import)
  3. Configure the trigger node with your data source
  4. Update CustomJS nodes with your business logic
  5. Adjust the PDF template with your branding
  6. Set up output destinations (email, storage, etc.)
  7. Test with sample data and activate the workflow
