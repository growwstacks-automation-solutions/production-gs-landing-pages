Generate Personalized Sales Follow-Ups from Fireflies Transcripts

Automatically create AI-powered follow-up messages from meeting transcripts. Save hours on sales outreach and improve client engagement.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Sales teams spend hours manually crafting follow-up emails after client meetings. This process is time-consuming, often delayed, and lacks personalization—leading to missed opportunities and weaker engagement.

This workflow solves that by automating the entire follow-up generation process. It triggers when a new appointment is created in Google Calendar, fetches the meeting transcript from Fireflies, uses Google Gemini AI to analyze the conversation and create 12 personalized follow-up messages, then stores them neatly in Google Drive for easy access and sharing.

The result is immediate, context-rich follow-ups that reference specific topics discussed, address client pain points, and guide the conversation toward conversion—all without manual effort.

How It Works

Step 1: Trigger on New Appointment

The workflow starts automatically when a new appointment is added to Google Calendar. It captures the guest's email, meeting times, and status.

Step 2: Extract Guest Details

An Edit Fields node processes the calendar event to isolate key information: first name, last name, email, and appointment details needed for the next steps.

Step 3: Fetch Meeting Transcript

A GraphQL node queries Fireflies using the guest's email to retrieve the full meeting transcript, including sentences, participant names, and summary.

Step 4: Filter Empty Records

A Filter node checks if a transcript exists. If no record is found in Fireflies, the workflow skips further processing to avoid errors.

Step 5: Generate AI Follow-Ups

The AI Agent node (powered by Google Gemini) analyzes the transcript and creates 12 personalized follow-up messages. These messages are conversational, concise, and directly reference topics and pain points mentioned during the call.

Step 6: Store in Google Drive

A Google Drive node saves the AI-generated messages in a designated folder, named after the guest for easy reference. The team can review, edit, or send these messages directly.

What You'll Need

  1. An n8n instance (self-hosted or cloud).
  2. Google Gemini API credentials.
  3. Google Drive account with folder access permissions.
  4. Fireflies API credentials with GraphQL access.
  5. Google Calendar with appointments that include guest email addresses.

Quick Setup Guide

  1. Download the Template: Use the download button above to get the JSON workflow file.
  2. Import into n8n: Open your n8n instance and import the JSON file.
  3. Connect Credentials: In the Google Calendar, GraphQL, AI Agent, and Google Drive nodes, select or create credentials for each service.
  4. Verify Extraction: Check the Edit Fields node extracts the required guest details correctly.
  5. Customize AI Prompt: Update the AI Agent node instructions to match your desired message tone, length, and structure.
  6. Set Drive Folder: Configure the Google Drive node to save messages in your preferred folder.
  7. Activate Workflow: Save and activate the workflow. It will now run automatically after new calendar appointments.

Pro tip: For best results, customize the AI prompt to include your company's specific sales frameworks, common value propositions, and preferred communication style. This makes the generated messages align perfectly with your brand voice.
