AI Portfolio Generator: Create Upwork Copy & Screenshots

Transform any website URL into a complete portfolio entry with professional screenshots and AI-generated Upwork descriptions—automatically.

Download Template JSON · n8n compatible · Free

What This Workflow Does

For freelancers, agencies, and web professionals, documenting past work is essential but time-consuming. Manually capturing screenshots, writing project descriptions, and organizing portfolio entries can take hours per project. This automation solves that problem by turning any website URL into a complete, professional portfolio entry in minutes.

The workflow automatically analyzes website content, captures multiple screenshot types (hero, full-page, mobile), generates compelling Upwork-style project descriptions with relevant skills, and saves everything to Google Drive with a complete log in Sheets. It handles JavaScript-rendered sites, retries failed captures, and maintains consistent quality across all portfolio entries.

How It Works

1. URL Submission & Analysis

Users submit a website URL through a simple web form. The workflow uses Firecrawl to render JavaScript-heavy sites (React, Vue, SPAs) and extract meaningful content structure, identifying key sections beyond navigation and footers.

2. Intelligent Screenshot Capture

ScreenshotOne API captures four professional image types: hero section (1920×1080 @2x), full-page scroll, individual content sections, and mobile view (375×812 @2x). The system includes retry logic for failed captures and filters invalid results.

3. AI Content Generation

OpenAI's GPT-4o-mini analyzes the website to generate a complete Upwork portfolio entry including a project title (max 50 chars), your role, a compelling description (max 600 chars), and five relevant technical skills tailored to the project.

4. Storage & Notification

All screenshots save to a designated Google Drive folder with organized naming. A complete log with metadata, analysis, and AI-generated content appends to Google Sheets. Finally, a Telegram notification delivers all links and the portfolio copy for immediate use.

Who This Is For

This automation is ideal for freelancers building their Upwork profiles, agencies documenting client work at scale, web developers showcasing their projects, and anyone who needs consistent, high-quality website documentation. It's particularly valuable for professionals with multiple past projects to document or those who regularly update their portfolios with new work.

What You'll Need

  1. n8n instance (cloud or self-hosted)
  2. Firecrawl API key (500 pages/month free tier)
  3. ScreenshotOne API key (100 screenshots/month free tier)
  4. OpenAI API credentials (pay-as-you-go)
  5. Google Workspace account for Drive and Sheets (15GB free)
  6. Telegram bot token & chat ID for notifications (free)

Pro tip: Test the workflow with simple HTML sites first before trying complex single-page applications. This helps verify all connections work before tackling JavaScript-heavy sites that require Firecrawl's rendering capabilities.

Quick Setup Guide

  1. Import the downloaded JSON template into your n8n instance
  2. Add your Firecrawl API key to the HTTP Request node
  3. Configure ScreenshotOne credentials with your API key
  4. Connect OpenAI credentials (default uses GPT-4o-mini for cost efficiency)
  5. Set up Google Drive connection and specify your target folder ID
  6. Connect Google Sheets and specify your spreadsheet ID
  7. Configure Telegram bot with your token and chat ID
  8. Activate the workflow and share the generated webhook URL as your submission form
