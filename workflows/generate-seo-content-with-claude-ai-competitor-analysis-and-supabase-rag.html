What This Workflow Does

This automation solves the time-consuming challenge of creating SEO-optimized content at scale. Instead of manually researching competitors, analyzing search intent, and writing meta tags for each keyword, this workflow automates the entire process using AI, competitor data scraping, and your business-specific information.

The system processes batches of keywords, performs Google searches to identify top competitors, scrapes their content structure, then uses Claude AI to generate personalized meta titles, descriptions, H1 headings, and comprehensive content briefs. It connects to your Supabase database to ensure content accuracy and relevance to your actual business offerings.

What typically takes content marketers 2-4 hours per keyword (research, analysis, writing) becomes a fully automated process that handles dozens or hundreds of keywords simultaneously, maintaining consistent quality while saving 80-90% of the manual effort.

How It Works

Step 1: Data Input and Configuration

The workflow starts by reading your Google Sheets configuration containing client information, target keywords, and content parameters. It filters valid keywords and prepares them for batch processing, ensuring only relevant targets move forward.

Step 2: Competitor Research and Analysis

For each keyword, the system performs Google searches via Apify to identify the top 10 ranking pages. It then scrapes the first 5 competitor websites using Firecrawl, extracting their heading structures (H1-H6), meta tags, and content organization patterns.

Step 3: AI-Powered Content Generation

Claude AI analyzes the competitor data alongside your business information from Supabase. It generates optimized meta titles (65 characters max), compelling meta descriptions (165 characters max), and user-focused H1 headings (70 characters max) that outperform competitor content.

Step 4: Comprehensive Content Brief Creation

The AI creates detailed content briefs including search intent analysis, MECE page structure recommendations, rich media suggestions, and writing guidelines with detail level scoring. These briefs provide everything a writer needs to create high-ranking content.

Step 5: Data Integration and Updates

All generated content is structured and written back to your Google Sheets, preserving existing data while adding the new SEO elements. The system continues processing remaining keywords in batches until complete.

What You'll Need

  1. Google Sheets API access for reading configuration and updating results
  2. Anthropic API key for Claude AI content generation
  3. OpenAI API key for embeddings and vector search functionality
  4. Apify API key for Google search results retrieval
  5. Firecrawl API key for competitor website scraping
  6. Supabase account (optional but recommended for vector database and business data integration)
  7. Template spreadsheet configured with your client information and target keywords

Pro tip: Start with a small batch of 5-10 keywords to test the workflow before processing large lists. This helps you verify API configurations and adjust prompts for optimal output quality without consuming excessive API credits.

Quick Setup Guide

  1. Copy the template Google Sheets spreadsheet and configure it with your client information and target keywords
  2. Import the n8n workflow JSON file into your n8n instance
  3. Configure all required API credentials in n8n's credential management system
  4. Test the workflow by sending your Google Sheets URL to the chat trigger
  5. Verify that all sheets are readable and that the first keyword processes correctly
  6. Once confirmed, activate the workflow for full batch processing
  7. Monitor the Google Sheets for updates as content is generated and written back
