What This Workflow Does

This automation solves the creative bottleneck of producing original audio content. Instead of hiring musicians, lyricists, and audio engineers, you can generate complete songs from a simple text prompt in any chat interface. The workflow transforms ideas like "upbeat pop song about summer adventures" or "melancholic piano ballad for a documentary" into structured lyrics and a finished music track.

It bridges two powerful AI services: OpenAI for intelligent, genre-appropriate lyric writing and Fal.ai's minimax-music model for professional-grade audio synthesis. The process handles the entire creative pipeline—from prompt interpretation and lyric structuring to music generation, status polling, and delivery—returning a shareable audio URL alongside the lyrics.

How It Works

1. Chat Trigger & Prompt Interpretation

The workflow activates via a chat message in your n8n interface. The user's text prompt is captured and prepared for processing. This initial step ensures the automation is user-friendly and accessible directly from communication platforms.

2. AI Lyric Generation with OpenAI

The prompt is sent to an OpenAI model (like GPT-4) configured with a specialized system instruction. This AI agent acts as a songwriter, crafting original lyrics with a defined structure—verses, choruses, and bridges—that match the requested theme, genre, and emotional tone. It enforces a minimum length for substantial output.

3. Music Synthesis Queue with Fal.ai

The generated lyrics are then submitted to Fal.ai's music generation API. This step packages the lyrics with the style parameters from the original prompt, queuing a job for the AI to compose matching instrumentation, melody, and vocal synthesis.

4. Asynchronous Status Polling

Since music generation can take time, the workflow intelligently polls the Fal.ai queue at intervals. It checks the job status, waiting patiently until the track is fully rendered. This robust error-handling and waiting logic prevents timeouts.

5. Result Delivery

Once the audio file is ready, the workflow fetches the final track URL. It then formats and delivers both the complete lyrics and the direct link to the generated song back to the user in the chat, closing the loop on a seamless creative experience.

What You'll Need

  1. An active n8n instance (cloud or self-hosted) with chat trigger capabilities enabled.
  2. An OpenAI API account with credits and access to GPT models.
  3. A Fal.ai account with an API key and access to the minimax-music model.
  4. Basic understanding of how to add API credentials within n8n.

Quick Setup Guide

  1. Import: Download the JSON template and import it into your n8n workspace.
  2. Credentials: Create and assign your OpenAI API credentials to the "OpenAI Chat Model" node.
  3. More Credentials: Create an HTTP Header Auth credential for Fal.ai (Header: "Authorization", Value: "Key YOUR_KEY") and assign it to the three Fal.ai nodes.
  4. Activate: Turn the workflow on. The chat trigger will become available in your n8n chat panel.
  5. Test: Send a message like "Create a motivational synthwave track for a productivity app" and watch the magic happen.

Pro tip: For best results, use descriptive prompts that include genre, mood, tempo, and key themes. Example: "A hopeful, acoustic folk song about new beginnings, with gentle guitar and soft vocals."
