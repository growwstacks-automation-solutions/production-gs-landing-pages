What This Workflow Does

This n8n workflow automates professional video creation using OpenAI's Sora 2 AI model, eliminating watermarks and complex setup. It transforms text descriptions or uploaded images into high-quality, ready-to-use videos for marketing campaigns, social media content, product demonstrations, or educational materials.

The system handles the entire video generation pipeline: from receiving user requests through a web form, processing prompts and images, submitting to the Sora 2 API via Kie.AI, monitoring generation status, and delivering the final video—all without manual intervention. It supports both text-to-video and image-to-video generation with customizable aspect ratios and quality settings.

Businesses can use this to scale video content production, create personalized marketing materials, generate training videos, or prototype visual concepts rapidly. The automation reduces video production time from days to minutes while maintaining professional quality suitable for public-facing content.

How It Works

Step 1: User Submission

A web form collects video requirements including description, aspect ratio (16:9, 9:16, 1:1), quality settings, and optional image uploads. Users can specify detailed prompts for Sora 2 to interpret.

Step 2: Media Processing

If an image is uploaded, the workflow automatically uploads it to ImgBB to generate a public URL for Sora 2's image-to-video feature. The system determines whether to use text-to-video or image-to-video based on input.

Step 3: API Request to Sora 2

The workflow sends a properly formatted request to the Sora 2 API via Kie.AI with authentication headers, prompt details, and generation parameters. It handles API rate limits and authentication automatically.

Step 4: Status Polling & Completion

After submission, the workflow waits 30 seconds then checks generation status. If not ready, it loops back every 30 seconds until completion. This automated polling ensures you get your video as soon as it's ready.

Step 5: Video Delivery

Once generation completes, the workflow downloads the watermark-free MP4 file and can deliver it via Telegram notification, email, or save to cloud storage. The final output is production-ready video content.

Quick Setup Guide

Follow these steps to implement this AI video generation system in your n8n instance:

  1. Import the template by downloading the JSON file and importing it into your n8n workspace
  2. Configure Kie.AI credentials in n8n using HTTP Header Auth with your API key
  3. Set up ImgBB integration by adding your free API key to the image upload node
  4. Customize the web form to match your branding and collect necessary information
  5. Test with a simple prompt like "A cat playing piano in a jazz club" to verify the pipeline works
  6. Configure delivery method - update Telegram chat IDs or switch to email/cloud storage
  7. Deploy and activate the workflow, then share the web form URL with your team

Pro tip: Start with shorter videos (15-30 seconds) to minimize generation costs while testing. Use specific, descriptive prompts with style references (e.g., "cinematic," "animated explainer," "product demo style") for best results.
