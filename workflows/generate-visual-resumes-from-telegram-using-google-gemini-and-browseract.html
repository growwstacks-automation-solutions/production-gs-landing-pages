What This Workflow Does

This automation transforms unstructured resume information received via Telegram messages into professionally designed visual resumes. Candidates simply send their details through Telegram, and the system automatically processes the information using Google Gemini's AI capabilities before generating polished visual layouts with BrowserAct.

The workflow solves three major recruitment pain points: eliminating manual resume formatting labor, ensuring consistent visual quality across all applications, and enabling candidates to submit information through convenient messaging platforms rather than complex forms. Recruiters receive standardized, visually appealing resumes ready for evaluation.

How It Works

1. Telegram Message Processing

The workflow monitors specified Telegram channels or direct messages for resume submissions. When a candidate sends their information, the system captures the complete message thread including any attached files or links.

2. AI-Powered Data Extraction

Google Gemini analyzes the unstructured text, identifying and categorizing resume components (work experience, education, skills etc.). The AI normalizes dates, corrects formatting inconsistencies, and structures the data into standardized JSON format.

3. Visual Template Generation

BrowserAct takes the structured data and generates professional visual resumes using predefined templates. The system can produce multiple format options (infographic, timeline, skills-focused) based on the candidate's career profile.

4. Delivery & Notification

The final visual resume gets sent back to the candidate via Telegram while also being stored in your recruitment system. Hiring managers receive notifications with download links to the formatted resumes.
