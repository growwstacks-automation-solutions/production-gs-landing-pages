What This Workflow Does

Manual cold email reporting is a time-consuming, error-prone process that steals valuable hours from sales teams each week. This automation solves that problem by automatically generating comprehensive performance reports for your ManyReach email campaigns.

The workflow runs every Monday, fetching completed campaigns from ManyReach, analyzing them with GPT-4.1 AI, creating professional Google Docs reports, and sharing them directly to your Slack channel. It transforms raw campaign data into actionable insights without any manual intervention.

How It Works

1. Scheduled Data Collection

Every Monday morning, the workflow automatically retrieves up to 100 campaigns from your ManyReach account via API. It filters for only active, completed campaigns to ensure you're analyzing relevant data.

2. Campaign Processing Loop

Each qualifying campaign is processed individually. The workflow fetches detailed campaign data, then sends it to GPT-4.1 for intelligent analysis. The AI evaluates performance against industry benchmarks and generates insights.

3. AI-Powered Analysis

GPT-4.1 creates comprehensive reports including executive summaries, performance metrics (open rates, click rates, reply rates), email content analysis, configuration insights, and actionable recommendations for improvement.

4. Report Generation & Distribution

The AI-generated markdown is converted to HTML, formatted for Google Docs, and uploaded to your connected Drive account. A link to the document is then automatically posted to your designated Slack channel with the campaign name for easy team access.

Pro tip: Configure the workflow to run on Friday afternoons instead of Monday mornings to give your team performance insights before weekend planning sessions.

What You'll Need

  1. ManyReach account with API access (HTTP Query Auth credentials)
  2. OpenAI API key for GPT-4.1 analysis capabilities
  3. Google Drive account with OAuth2 setup for document creation
  4. Slack workspace with OAuth2 access to post to your desired channel
  5. n8n instance (cloud or self-hosted) to run the workflow

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance
  2. Configure credentials for ManyReach, OpenAI, Google Drive, and Slack in n8n's credentials management
  3. Update the Slack channel in the "Send a Doc Link" node to match your team's #manyreach or #reports channel
  4. Adjust the schedule trigger if you prefer a different day or time for report generation
  5. Test with a single campaign by temporarily modifying the filter to process one campaign and verify the output
  6. Activate the workflow and let it run automatically every week

Pro tip: Start by testing with historical campaign data to see the quality of AI-generated insights before running on live campaigns.
