What This Workflow Does

This automation acts as your autonomous Chief Solution Officer for YouTube content strategy. Instead of spending hours manually watching competitor videos and reading comments, this workflow systematically scrapes YouTube data—including video descriptions and the top 20-50 comments—to identify what your target audience actually cares about.

It goes beyond basic analytics by analyzing the "Decision Confusion" and "Operational Pain Points" expressed in comments, then generates five strategic content ideas that address genuine audience needs rather than generic topics. The insights and content briefs are automatically organized and saved in your Notion workspace, creating a living research database that evolves with your niche.

The system uses a "Sage + Ruler" AI persona to ensure outputs are professional, objective, and focused on high-level business logic. This transforms content creation from guesswork to a data-driven process that consistently delivers value to your audience.

How It Works

1. Target & Configure

You define your niche or specific search keywords in the initial configuration node. This could be a broad topic like "digital marketing" or a specific tool like "Notion tutorials." The workflow uses this as the starting point for all subsequent research.

2. Data Extraction with Firecrawl

Firecrawl searches YouTube for the most relevant videos based on your keywords and scrapes comprehensive data including titles, descriptions, view counts, engagement metrics, and—most importantly—the top comments. This bypasses surface-level analytics to capture actual audience sentiment.

3. AI Analysis & Pattern Recognition

OpenAI processes the scraped data to identify recurring themes, frustrations, and unanswered questions. It looks for patterns in what the audience is complaining about, confused by, or requesting more information on—the real "voice of the customer" that most creators miss.

4. Strategic Content Generation

The AI converts these insights into five actionable content briefs with unique strategic angles. Each brief includes the core problem to address, the audience's specific pain points, competitive gaps to exploit, and suggested positioning that differentiates your content.

5. Automated Notion Organization

All research findings and generated content ideas are automatically structured and saved into your Notion database. This creates a searchable, sortable repository of audience insights and content opportunities that grows with each workflow execution.

What You'll Need

  1. Firecrawl API Key – Sign up at firecrawl.dev for web scraping capabilities
  2. OpenAI API Key – GPT-4o recommended for best analysis results
  3. Notion Integration – Create an internal integration and share your database with it
  4. Two Notion Databases – One for "Audience Insights" and one for "Content Ideas"
  5. n8n Instance – Self-hosted or cloud version with webhook capabilities

Quick Setup Guide

Follow these steps to implement this automation in under 30 minutes:

  1. Import the template – Download the JSON file above and import it into your n8n instance
  2. Configure Firecrawl – Add your API key to the HTTP Request nodes' Authorization headers
  3. Connect OpenAI – Add your API credentials to the OpenAI Chat Model node
  4. Set up Notion – Create two databases (Insights & Ideas) and connect via the Notion node
  5. Define your niche – Enter your target keywords in the "Set Niche" configuration node
  6. Test and refine – Run the workflow with a specific search term and adjust parameters as needed

Important: Start with a narrow niche focus for your first test. Broad topics like "marketing" will return too much data. Try "B2B SaaS content marketing" or "Notion for project management" for more actionable insights.
