What This Workflow Does

This automation solves the biggest bottleneck for YouTube creators: script writing. Instead of spending hours researching and drafting, you provide a single video topic, and the workflow generates two complete, research-backed scripts—one optimized for YouTube Shorts (under 60 seconds) and another for long-form videos (8+ minutes).

The system uses Google Gemini AI for creative writing and Tavily AI for real-time web research, ensuring your scripts contain current information and trends. Both scripts are automatically saved to Google Docs, ready for your review and filming. This effectively multiplies your content output while maintaining quality and relevance.

How It Works

The workflow operates through parallel processing paths after receiving your topic submission:

1. Topic Submission & Initial Processing

You submit a video topic through a connected form. The workflow captures this input and prepares it for both script generation paths simultaneously.

2. Shorts Script Generation Path

This branch performs a quick web search via Tavily AI to gather key facts, then immediately uses a Gemini AI agent to write a concise, attention-grabbing script structured for YouTube Shorts' fast-paced format.

3. Long-Form Script Generation Path

This branch conducts more comprehensive research, uses Gemini AI to create a detailed outline with sections and talking points, then generates a complete long-form script based on that structured outline.

4. Document Creation & Delivery

Both scripts are automatically created as separate Google Docs in your designated Drive folder, with clear titles indicating format and topic for easy organization.

What You'll Need

  1. An active n8n instance (cloud or self-hosted)
  2. A form trigger (built-in n8n form or external form connected via webhook)
  3. API key for Tavily AI (for web research)
  4. API key for Google Gemini AI (for script generation)
  5. Google OAuth2 credentials configured in n8n (for Google Docs integration)
  6. A Google Drive folder for script storage

Quick Setup Guide

Follow these steps to implement this automation in your n8n environment:

  1. Import the template using the download button above
  2. Configure your form trigger to accept a "topic" field
  3. Add your Tavily API key to both Tavily nodes in the credentials section
  4. Add your Google Gemini API key to the AI Agent and Create Outline nodes
  5. Set up Google OAuth2 credentials for all Google Docs nodes
  6. Test with a sample topic and check your Google Drive for the generated scripts
  7. Activate the workflow and connect it to your production form
