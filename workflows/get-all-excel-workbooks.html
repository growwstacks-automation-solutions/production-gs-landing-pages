What This Workflow Does

This automation solves the common business challenge of collecting Excel workbooks from multiple sources. Instead of manually downloading attachments from emails, cloud storage, or form submissions, this workflow automatically gathers all relevant Excel files into one centralized location.

The template handles the entire collection process - identifying Excel files across different platforms, standardizing file names if needed, and organizing them in your preferred destination. It eliminates hours of repetitive manual work while ensuring no files are missed during the collection process.

How It Works

Step 1: Source Identification

The workflow scans configured sources (email accounts, cloud folders, form submissions) to identify all Excel workbook files. It recognizes both .xlsx and .xls formats across different platforms.

Step 2: File Collection

Identified Excel files are downloaded from their original locations while maintaining metadata like creation dates. The system can handle password-protected files if credentials are provided.

Step 3: Organization

Collected workbooks are organized based on your preferences - either maintaining original folder structures or creating a new standardized organization system. Files can be renamed consistently if desired.

Step 4: Notification

The workflow sends a summary notification listing all collected files, their sources, and any processing notes. This creates an audit trail for the collection process.

Who This Is For

This workflow benefits any business or department that regularly receives Excel files from multiple sources. Finance teams collecting departmental budgets, HR departments gathering timesheets, and operations teams consolidating inventory reports will find this particularly valuable.

It's especially useful for organizations where multiple stakeholders submit spreadsheet data - such as franchise locations reporting sales, field teams submitting expense reports, or partners providing performance metrics.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. Access credentials for your Excel file sources (email accounts, cloud storage, etc.)
  3. A destination folder for the collected files
  4. Basic familiarity with n8n workflow configuration

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure the source nodes with your specific email accounts or cloud storage connections
  3. Set your preferred destination folder in the storage node
  4. Adjust file naming and organization preferences if needed
  5. Test with sample files before activating the full workflow
