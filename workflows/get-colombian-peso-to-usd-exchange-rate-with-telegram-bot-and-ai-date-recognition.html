What This Workflow Does

This automation solves a common problem for businesses and individuals dealing with Colombian financial transactions: quickly and accurately retrieving the official TRM (Tasa Representativa del Mercado) – the Colombian Peso to US Dollar exchange rate – for any specific historical date.

Instead of manually searching government websites or spreadsheets, users can simply send a text or voice message to a Telegram bot (e.g., "What was the rate on March 15?"). The workflow uses AI to understand the date request, fetches the official rate from the Colombian government's open data API, and delivers the answer instantly. If data isn't available for the exact date, it intelligently searches backwards to find the most recent available rate.

This eliminates manual errors, saves hours of administrative time each month, and ensures financial records, invoices, and reports are based on accurate, auditable official data.

How It Works

The workflow is a smart sequence that connects user-friendly messaging with reliable financial data.

Step 1: User Request via Telegram

A user sends a message to your dedicated Telegram bot. This can be a text phrase like "TRM for last Friday" or a voice message. The Telegram Trigger node captures this input.

Step 2: AI-Powered Date Extraction

The message is sent to an AI model (like OpenAI) with a specific prompt to extract and format the date mentioned. The AI understands natural language and returns a standardized date (YYYY-MM-DD).

Step 3: Date Validation & Logic

The workflow checks if the extracted date is in the future. If it is, the bot immediately replies that future rates are not available. This is handled by an "If" node.

Step 4: API Query for the TRM

Using the HTTP Request node, the workflow queries the official Colombian government data API (datos.gov.co) with the specific date. It requests the TRM value for that day.

Step 5: Result Handling & Fallback Search

If data is found, the bot formats and sends the rate to the user. If no data exists for that date (e.g., a weekend or holiday), a "Loop" node activates to search up to 10 previous days until it finds the most recent available official rate, ensuring a useful response.

Step 6: Final Delivery

The Telegram node sends a clear, formatted reply to the user with the date and the corresponding TRM value, completing the interaction in seconds.

Quick Setup Guide

Get this automation running in your environment in under 15 minutes.

  1. Download the Template: Click the "Download Template" button on this page to get the JSON file.
  2. Import into n8n: In your n8n dashboard, go to Workflows > Import from File and select the downloaded JSON.
  3. Configure Credentials: Set up the required credentials for Telegram Bot and OpenAI in n8n's Credentials section.
  4. Activate the Workflow: Turn on the workflow trigger. Your bot is now listening on Telegram.
  5. Test: Message your bot with a date query like "exchange rate for January 10, 2024" to see it in action.
