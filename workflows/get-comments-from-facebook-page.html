What This Workflow Does

Managing social media engagement manually is time-consuming and inefficient. This workflow solves that by automatically collecting every comment from your Facebook Page posts, transforming raw social interactions into clean, structured data ready for analysis or further automation.

Instead of constantly checking your page or missing valuable customer feedback, this automation runs on a schedule you define, fetching recent posts and their associated comments. It outputs organized data including comment text, author information, timestamps, and post context—everything you need to understand audience sentiment, respond to inquiries promptly, and measure content performance.

The structured output enables seamless integration with other business systems, turning social media conversations into actionable business intelligence without manual data entry or spreadsheet management.

How It Works

The workflow follows a logical sequence to ensure reliable data collection and processing.

Step 1: Authentication & Configuration

The workflow begins by establishing a secure connection to the Facebook Graph API using your access token with appropriate permissions. You configure your Page ID and specify how many recent posts to fetch, balancing completeness with processing time.

Step 2: Post Retrieval

The system fetches the specified number of recent posts from your Facebook Page, including essential metadata like post ID, message, creation time, and engagement metrics. This creates the foundation for comment collection.

Step 3: Comment Collection

For each retrieved post, the workflow queries Facebook's API to gather all associated comments. It handles pagination automatically, ensuring no comments are missed even on highly active posts.

Step 4: Data Structuring

Raw comment data is transformed into a consistent format, extracting key fields like comment ID, message, author name and ID, creation timestamp, and like count. The data is enriched with post context for meaningful analysis.

Step 5: Output & Integration

Structured comment data is output in a clean format ready for immediate use. The workflow can be extended to send this data to databases, spreadsheets, CRM systems, or team collaboration tools for real-time action.

Pro tip: Schedule this workflow to run during off-peak hours to avoid API rate limits while ensuring you capture all daily engagement. Start with once-daily execution and increase frequency as needed.

What You'll Need

  1. Facebook Developer Account: Access to the Facebook Developer portal to create an app and generate access tokens.
  2. Facebook Page Access: Admin or editor access to the Facebook Page you want to monitor.
  3. API Permissions: The pages_read_engagement and pages_read_user_content permissions granted to your Facebook app.
  4. n8n Instance: A running n8n installation (self-hosted or cloud) where you can import and execute workflows.
  5. Page ID: Your Facebook Page's numeric ID, easily obtainable from your Page settings.

Quick Setup Guide

Follow these steps to get your Facebook comment automation running in minutes:

  1. Download the template using the button above and save the JSON file to your computer.
  2. Import into n8n: In your n8n instance, navigate to Workflows → Import from file and select the downloaded JSON.
  3. Configure Facebook connection: In the "Facebook Graph API" node, add your access token with the required permissions.
  4. Set your Page ID: Update the "Page ID" parameter in the workflow with your actual Facebook Page ID.
  5. Adjust post count: Modify the "Set Number of Latest Posts to Fetch" node based on how many posts you want to scan for comments.
  6. Test execution: Click "Execute Workflow" to run a test and verify comments are being collected correctly.
  7. Schedule automation: Add a Schedule Trigger node to run the workflow automatically at your desired frequency (daily, hourly, etc.).

Pro tip: Start with a small number of posts (5-10) during testing to ensure everything works correctly before scaling up to collect comments from more historical posts.
