What This Workflow Does

Manual stock research consumes hours each week, often relying on outdated data and subjective analysis. This creates missed opportunities, delayed reactions to market shifts, and portfolio decisions based on incomplete information rather than comprehensive, real-time intelligence.

This n8n automation template solves this by connecting directly to Danelfin's AI analytics API, automatically pulling real-time stock rankings, sector performance metrics, and industry-level insights. It transforms raw data into actionable investment signals, delivering explainable AI recommendations directly to your preferred tools—whether that's a spreadsheet, portfolio management software, or alert system—ensuring you always operate with the latest market intelligence.

How It Works

1. API Authentication & Data Fetching

The workflow begins by securely authenticating with Danelfin's API using your credentials. It then makes parallel requests to three key endpoints: stock rankings, sector analysis, and industry performance data, collecting the most current AI-generated insights available.

2. Data Processing & Normalization

Raw API responses are processed to extract key metrics—AI scores, price targets, sentiment indicators, and performance rankings. The workflow normalizes this data into a consistent format, calculating derived values like score changes, sector comparisons, and trend indicators.

3. Decision Logic & Filtering

Based on your configured criteria (minimum AI score, sector focus, volatility thresholds), the workflow filters stocks and sectors. It identifies top opportunities, flags deteriorating positions, and highlights significant ranking changes that warrant attention.

4. Output & Distribution

Processed insights are formatted and delivered to your chosen destinations. This could include updating a Google Sheets portfolio tracker, sending Slack alerts for urgent opportunities, creating Notion research pages, or triggering webhooks to other financial systems.

Quick Setup Guide

  1. Download & Import: Click the download button above and import the JSON file into your n8n instance.
  2. Configure API Credentials: Add your Danelfin API key to the HTTP Request nodes in the "Authentication" section.
  3. Set Your Parameters: Adjust the filter nodes to match your investment strategy—minimum AI scores, preferred sectors, alert thresholds.
  4. Connect Outputs: Link the final nodes to your desired destinations (Google Sheets, Slack, email, etc.) and test with sample data.
  5. Schedule & Deploy: Set the trigger schedule (daily, hourly, or real-time via webhook) and activate the workflow.

Pro tip: Start with a conservative schedule (once daily) to monitor performance before moving to more frequent updates. Create a separate "alert" channel in Slack or Teams for high-priority signals to avoid notification fatigue.
