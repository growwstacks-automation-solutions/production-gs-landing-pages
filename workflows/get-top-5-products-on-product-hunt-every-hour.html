What This Workflow Does

This n8n workflow solves the problem of manually checking Product Hunt for trending new products. Many businesses and teams want to stay updated on the latest tools and innovations but don't have time to constantly monitor the site. This automation delivers the top 5 products directly to your Discord server every hour.

The workflow eliminates the tedious process of copying product details manually, ensuring your team gets curated updates without any effort. It's particularly valuable for product managers, marketers, and tech teams who need to stay ahead of industry trends and discover useful new tools.

How It Works

1. Scheduled Trigger

The workflow starts with a cron trigger set to run every hour. This ensures you get fresh product updates throughout the day as rankings change.

2. Fetch Product Hunt Data

Using Product Hunt's API, the workflow retrieves the current top 5 products. It captures key details like product name, description, maker information, and upvote count.

3. Format for Discord

The raw API data gets transformed into clean, readable messages with proper formatting for Discord. Each product appears as a rich embed with relevant links.

4. Post to Discord Channel

Finally, the workflow sends the formatted product updates to your specified Discord channel using a webhook. Team members can immediately see and discuss the latest trending products.
