Gmail Email Classifier with GPT-4 Auto-Generated Draft Replies

What This Workflow Does

Every day, business owners and professionals waste hours sorting through emails, deciding what's urgent, what can wait, and crafting responses to common inquiries. This intelligent automation solves that problem by combining Gmail integration with advanced AI to create a smart email assistant that works 24/7.

The workflow automatically monitors your Gmail inbox, classifies incoming messages into meaningful categories like High Priority, Sales Inquiry, Customer Support, and Finance/Billing, then uses GPT-4 to generate contextually appropriate draft replies. The drafts are saved in Gmail for your review and sending, ensuring you maintain control while dramatically reducing response time.

How It Works

The automation follows a logical sequence that mimics how a human assistant would process emails, but with AI-powered precision and consistency.

Step 1: Trigger on New Emails

The workflow activates whenever a new email arrives in your specified Gmail inbox. It captures the full email content, sender information, subject line, and any attachments.

Step 2: AI-Powered Classification

Using a text classifier (with optional GPT-4 enhancement), the system analyzes the email content and assigns it to predefined categories. You can customize these categories based on your business needs—common ones include Urgent Action Required, Sales Lead, General Inquiry, Billing Question, or Newsletter/Subscription.

Step 3: Apply Gmail Labels

Once classified, the workflow automatically applies the corresponding Gmail label to the message. This organizes your inbox visually and enables you to create filters, priorities, and follow-up systems based on these labels.

Step 4: Generate Draft Reply with GPT-4

Based on the classification category, GPT-4 generates a professional, context-aware draft reply. The AI considers the original email's tone, content, and intent to create a response that's helpful, appropriate, and ready for your personal touch.

Step 5: Save Draft & Optional Notifications

The generated draft is saved directly in your Gmail drafts folder. Optionally, you can configure notifications to Slack, Teams, or other channels for urgent categories, ensuring time-sensitive messages get immediate attention.

What You'll Need

  1. A Gmail account or Google Workspace account with API access enabled
  2. An OpenAI API key with GPT-4 access (or GPT-3.5-turbo for cost-effective operation)
  3. An n8n instance (cloud or self-hosted) with the Gmail and OpenAI nodes installed
  4. Basic understanding of how to create Gmail labels for organization
  5. 10-15 minutes for initial setup and configuration

Pro tip: Start with 3-5 classification categories that match your most common email types. You can always add more later as you refine the system. Too many categories initially can reduce accuracy.

Quick Setup Guide

Getting this automation running in your environment takes just a few simple steps:

  1. Import the template: Download the JSON file above and import it into your n8n instance via the "Import from File" option.
  2. Configure credentials: Set up your Gmail OAuth credentials and OpenAI API key in n8n's credential management system.
  3. Customize categories: Update the classification categories in the "Text Classifier" node to match your business needs.
  4. Set Gmail label IDs: Replace the placeholder label IDs with your actual Gmail label IDs (find these via Gmail API or n8n's Gmail node).
  5. Test with sample emails: Run the workflow manually with a few test emails to verify classification accuracy and draft quality.
  6. Activate the trigger: Enable the Gmail trigger to run automatically on new emails.
