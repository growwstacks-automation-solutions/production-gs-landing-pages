What This Workflow Does

This n8n workflow automates the process of extracting keyword suggestions from Google's autocomplete feature. Instead of manually typing partial queries and recording suggestions, the workflow systematically generates variations, queries Google, and compiles the results into an organized dataset.

For SEO specialists and content marketers, this solves the time-consuming task of uncovering long-tail keyword opportunities. The automated approach ensures comprehensive coverage of search variations while eliminating human error in data collection. The results can be exported directly to your preferred keyword research tool or content planning system.

The workflow showing keyword extraction from Google autocomplete results

How It Works

1. Seed keyword input

The workflow starts with your core keywords or topics. These serve as the foundation for generating autocomplete queries. You can input multiple seed keywords at once for batch processing.

2. Query generation

The system automatically creates partial search queries by adding letters of the alphabet and common modifiers (like "how to", "best", etc.) to your seed keywords. This ensures comprehensive coverage of possible autocomplete variations.

3. Google API requests

The workflow makes controlled requests to Google's search interface with each generated query, mimicking human typing behavior to stay within acceptable usage limits while collecting the autocomplete suggestions.

4. Data processing

Raw suggestions are cleaned, deduplicated, and organized by relevance. The workflow can categorize keywords by type (questions, comparisons, location-based, etc.) based on linguistic patterns.

5. Output delivery

Final keyword lists are formatted for your preferred output - whether that's a spreadsheet, database, or direct integration with SEO tools like Ahrefs or SEMrush.

Pro tip: Combine this with your ranking tracking to identify autocomplete keywords you're not currently ranking for - these represent quick-win content opportunities.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. Google API credentials (optional but recommended for higher volumes)
  3. Basic understanding of keyword research principles
  4. Your core seed keywords or topics to research
  5. Output destination (Google Sheets, Airtable, etc.)

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance (Settings → Workflows → Import)
  3. Configure your seed keywords in the "Keywords" node
  4. Set your preferred output method (CSV, Google Sheets, etc.)
  5. Adjust query delay timing if needed (default is 1-2 seconds between requests)
  6. Run the workflow and monitor initial results
  7. Schedule regular runs to keep your keyword database fresh
