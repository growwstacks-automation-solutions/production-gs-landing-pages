What This Workflow Does

Manual lead generation from Google Maps is tedious and inefficient. Sales teams and marketing agencies spend hours searching for local businesses, copying contact details into spreadsheets, and sending repetitive emails. This process is error-prone, doesn't scale, and takes valuable time away from actual selling.

This n8n workflow automates the entire Google Maps prospecting pipeline. It scrapes business contacts (emails, phone numbers, websites, social media) for any location and category, cleans and structures the data, saves everything to an Airtable database, and sends personalized follow-up emails via Gmail. What used to take days now happens automatically with a single click.

The system is particularly valuable for B2B service providers, marketing agencies targeting local businesses, freelancers offering specialized services, and sales teams building territory-based prospect lists. It turns manual research into a scalable, repeatable system that works 24/7.

How It Works

The automation follows a logical six-step process that mimics what a human researcher would do, but with perfect consistency and at scale.

Step 1: Trigger the Workflow

The workflow starts with a manual trigger that you can run on-demand. For production use, you can replace this with a scheduled trigger (to run weekly/daily) or a webhook (to integrate with other systems). This flexibility lets you control when and how leads are collected based on your business needs.

Step 2: Scrape Google Maps with Apify

The workflow sends a request to Apify's Google Maps Email Leads Scraper. You specify search parameters like location, business category, and number of results. Apify handles the complex scraping process, respecting Google's terms while collecting comprehensive business data including emails, phone numbers, websites, LinkedIn/Facebook links, ratings, and reviews.

Step 3: Wait for Processing

A brief wait node ensures Apify has enough time to complete the scraping and return all data. This prevents the workflow from moving forward with incomplete information. The delay is configurable based on the number of results requested.

Step 4: Clean and Structure Data

Raw scraped data is messy and inconsistent. The Set node transforms it into a clean, standardized format with proper field names that match your Airtable columns. This step ensures "Company Name," "Email," "Phone," "Website," and other fields are properly extracted and formatted for reliable database storage.

Step 5: Save to Airtable Database

Each cleaned lead record is saved to your Airtable base. The workflow maps each field to the corresponding Airtable column, creating a searchable, filterable database of prospects. Airtable's flexibility allows you to add custom fields for tracking outreach status, notes, and follow-up dates.

Step 6: Send Personalized Email

For each new lead saved to Airtable, the workflow triggers a personalized email via Gmail. The email template dynamically inserts the business name, location, and other details from the Google Maps listing, making each message feel custom-written rather than generic spam.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure Apify credentials by adding your API token to the HTTP Request node. Get this from your Apify account settings.
  3. Set up Airtable connection using a Personal Access Token. Create this at airtable.com/create/tokens with access to your base.
  4. Connect Gmail using OAuth2 authentication within n8n's credential manager.
  5. Update search parameters in the Apify node body to target your desired location and business category.
  6. Customize the email template with your messaging, value proposition, and call-to-action.
  7. Test with a manual trigger to verify all connections work and data flows correctly through each step.
