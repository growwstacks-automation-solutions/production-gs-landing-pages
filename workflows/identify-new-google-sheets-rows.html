What This Workflow Does

Manually checking a Google Sheet for new entries is a repetitive, error-prone task that wastes valuable time. This automation solves that by continuously monitoring your spreadsheet, identifying only the rows that haven't been processed, and then triggering any follow-up action you need.

The workflow uses a smart tracking system: it looks for a dedicated column (like "Processed" or "Status") to distinguish between new and handled data. When it finds a new row—such as a fresh lead from a form, a new order, or an updated inventory item—it can instantly notify your team, update another system, or begin a multi-step data enrichment process.

This turns your static spreadsheet into a dynamic, event-driven data hub. It's perfect for businesses using Google Sheets as a simple database, collaborative input tool, or staging area before data moves into more specialized software.

How It Works

The automation runs on a schedule you set, combining reliability with flexibility.

Step 1: Scheduled Check

An Interval trigger runs the workflow automatically every few minutes, hours, or days. This replaces you manually opening the sheet to look for changes.

Step 2: Fetch All Sheet Data

The workflow connects to your specified Google Sheet and retrieves all current rows. It securely accesses the data using your Google account credentials set up in n8n.

Step 3: Filter for New Rows

This is the core logic. The workflow examines a designated status column (e.g., "Processed"). Rows where this column is empty (or marked "New") are identified as unprocessed. Rows already marked with a timestamp or "Done" are intelligently skipped.

Step 4: Process Each New Item

Each new row is passed individually to the next steps. This could be sending an email, creating a task in your project management tool, adding a contact to your CRM, or performing data calculations.

Step 5: Update Status & Log

Once a row is successfully processed, the workflow writes back to the "Processed" column in the same sheet—usually with a timestamp. This creates a clear audit trail, prevents the same row from being handled twice, and lets your team see what's been completed at a glance.

What You'll Need

  1. A Google Sheets spreadsheet with your data. Ensure it has a unique identifier column (like "ID" or "Email") and a dedicated status column (like "Processed") to track what's been handled.
  2. A free or paid n8n instance. You can use n8n.cloud for a hosted version or self-host n8n on your own server.
  3. Google account credentials configured in n8n to allow access to your Sheets. (n8n provides a straightforward OAuth setup guide).
  4. Clear knowledge of what should happen to new data (e.g., "Send a welcome email," "Create a Asana task").

Pro tip: Name your status column clearly, like "Automation_Status." Avoid using columns that might be edited manually for other purposes to prevent the workflow from getting confused.

Quick Setup Guide

Get this automation running in your environment in under 15 minutes.

  1. Download the Template: Click the "Download Template" button above to save the JSON file to your computer.
  2. Import to n8n: In your n8n dashboard, go to Workflows > Import from File and select the downloaded JSON file.
  3. Configure Google Sheets Node: Click the "Google Sheets" node. Authenticate with your Google account if you haven't already, then specify the exact Spreadsheet ID and Worksheet name you want to monitor.
  4. Set Your Status Column: In the "If" node, adjust the condition to match the name of your status column (default is "Processed"). The workflow checks if this column is empty.
  5. Add Your Action: Replace the placeholder "No Operation" node with your desired action—like sending an email via Gmail, posting to Slack, or adding a row to Airtable.
  6. Set the Schedule: Configure the "Interval" node to run as often as you need (e.g., every 10 minutes for leads, once a day for reports).
  7. Activate & Test: Turn the workflow ON, add a new row to your test sheet, and watch the automation process it and mark it complete.
