What This Workflow Does

This automation solves the challenge of manually analyzing Instagram visual content at scale. Marketing teams often spend hours reviewing competitor posts or tracking brand mentions, but this workflow automates the entire process from data collection to AI-powered analysis.

The system scrapes recent Instagram posts for any specified account using Apify, extracts the images, sends them to OpenAI for visual analysis, and organizes the insights in Google Sheets. This creates a searchable database of visual trends, object detection results, and content patterns that would be impossible to track manually.

How It Works

1. Instagram Data Collection

The workflow begins by using Apify's Instagram scraper to collect recent posts from specified accounts. It handles authentication, rate limiting, and data extraction automatically, pulling both the image URLs and associated metadata.

2. Image Processing

Each image is downloaded and prepared for analysis. The workflow handles different post types (single images, carousels, videos) and ensures proper formatting for the AI analysis step.

3. OpenAI Visual Analysis

Images are sent to OpenAI's visual analysis API which detects objects, scenes, colors, text, and aesthetic qualities. The AI generates detailed descriptions and scores each image on various dimensions that you can customize.

4. Google Sheets Integration

All analysis results are structured and written to Google Sheets with timestamps. The workflow creates different tabs for raw data, summary statistics, and trend analysis that updates automatically with each run.

Who This Is For

This workflow is ideal for social media managers, content strategists, and competitive intelligence teams. Brands that rely heavily on visual storytelling (fashion, food, travel, etc.) will find particular value in systematically analyzing Instagram content.

Digital agencies managing multiple client accounts can scale their reporting capabilities, while ecommerce businesses can track how their products appear in user-generated content.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. Apify account with Instagram scraper access
  3. OpenAI API key with vision capabilities
  4. Google Sheets connected to your n8n
  5. Target Instagram usernames to analyze

Quick Setup Guide

  1. Download the JSON template and import into your n8n instance
  2. Configure the Apify node with your account credentials
  3. Add your OpenAI API key in the AI analysis node
  4. Connect to Google Sheets and specify your target spreadsheet
  5. Add the Instagram usernames you want to track
  6. Test with 1-2 accounts before scaling up
