How It Works

1. Command Input

Users send voice or text commands via Telegram specifying their lead criteria. The system understands natural language requests including industry, job titles, technologies used, and geographic filters.

2. AI Processing

The workflow analyzes the request using NLP to extract key parameters, then determines the most relevant data sources to query based on the criteria provided.

3. Data Collection

The system automatically scrapes and aggregates data from configured sources like LinkedIn, company websites, and industry directories, applying filters to match the request.

4. Report Generation

Collected data is organized into a structured format with contact details, company information, and relevant insights. For research requests, the AI generates comparative analysis.

5. Delivery

The final output is delivered back through Telegram as an organized message with attachments. The workflow can also push data directly to CRMs like Salesforce or HubSpot.
