LINE Messages with GPT: Save Notes & Extract Namecard Data

A free n8n workflow template that processes incoming LINE messages, uses AI to extract contact info from namecard images, saves notes to Microsoft Teams, creates tasks in Microsoft To Do, and stores images in OneDrive—fully automated.

What This Workflow Does

This automation acts as a personal assistant for your LINE messaging. It listens for incoming messages—whether text notes, images, or namecard photos—and intelligently routes them to the right place. Text notes get saved to a Microsoft Teams channel for team visibility. Images are uploaded to OneDrive for safekeeping. Most importantly, photos of business cards are analyzed by AI to extract contact details like name, company, email, and phone number, which are then automatically created as a follow-up task in Microsoft To Do.

It solves the chaos of managing communication across multiple channels. Instead of manually typing out contact info from a namecard photo or copying notes between apps, this workflow does it instantly and accurately. It ensures no lead or important detail gets lost in a busy chat stream, turning casual LINE conversations into structured, actionable data.

How It Works

The workflow is triggered the moment a message arrives in your connected LINE channel. It uses a smart router to determine the message type and takes appropriate action.

Step 1: Message Reception & Classification

A LINE Webhook node captures the incoming message. A Switch node then analyzes the content: text starting with "T" is flagged as a task, plain text is treated as a note, and images are sent for further analysis.

Step 2: AI-Powered Image Processing

If the message contains an image, it's sent to an AI model (via OpenRouter) to determine if it's a namecard, a handwritten note, or another type of image. For namecards, the AI extracts structured contact data.

Step 3: Data Routing & Storage

Based on the classification: Tasks are created in Microsoft To Do with the extracted details. Notes are posted to a designated Microsoft Teams channel. All images are uploaded to a specific OneDrive folder, renamed for easy reference.

Step 4: User Confirmation

Finally, a reply is sent back through LINE confirming the action taken (e.g., "[Task Created]", "[Note Saved]"), closing the loop with the user.

What You'll Need

  1. A LINE Developer account and a Messaging API channel to generate webhook credentials.
  2. Access to Microsoft 365 (Teams, To Do, OneDrive) with API permissions enabled.
  3. An OpenRouter.ai account or access to another AI/vision API (like OpenAI GPT-4o) for image analysis.
  4. An n8n instance (cloud or self-hosted) to deploy the workflow.
  5. Basic understanding of webhooks and API authentication.

Pro tip: Use a dedicated LINE channel or bot for this automation, not your personal chat, to keep things organized and test safely.

Quick Setup Guide

  1. Import the Template: Download the JSON file above and import it into your n8n instance.
  2. Configure LINE: In the LINE Developers Console, create a webhook and paste the URL from the "LINE Webhook" node. Verify the connection.
  3. Connect Microsoft Services: Add credentials for Microsoft To Do, Teams, and OneDrive nodes in n8n. Specify the target Teams channel and OneDrive folder path.
  4. Set Up AI: Add your OpenRouter.ai API key to the "OpenRouter Chat Model" node. You can adjust the AI prompt for namecard extraction if needed.
  5. Test & Activate: Send a test message (text and an image of a namecard) to your LINE bot. Check that tasks appear in To Do, notes in Teams, and images in OneDrive. Activate the workflow.
