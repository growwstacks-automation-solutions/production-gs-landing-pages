AI Food Logging: LINE + OpenAI Vision to Google Sheets

Automate food calorie tracking. Send food images via LINE bot, analyze with AI vision, and log nutritional data to spreadsheets instantly.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Manual food logging is time-consuming and often inaccurate. This automation solves that by creating a seamless pipeline where users simply snap a picture of their meal and send it to a LINE bot. The AI analyzes the image, estimates calories and nutritional content, and automatically logs everything to Google Sheets—creating a searchable, analyzable food diary without any manual data entry.

For wellness coaches, healthcare providers, or corporate wellness programs, this represents a transformative efficiency gain. Instead of clients struggling with food tracking apps or handwritten logs, they get an effortless experience that actually improves compliance and data quality. The automation handles the technical complexity while providing clean, structured data for analysis and reporting.

How It Works

1. User Interaction via LINE

The workflow starts when a user sends a message or food image to your connected LINE bot. The LINE webhook triggers the automation, capturing the user's message and any attached images for processing.

2. Image Processing & AI Analysis

If an image is detected, the workflow downloads it from LINE's servers and sends it to OpenAI's Vision model. The AI identifies food items, estimates portion sizes, and calculates approximate calorie values based on visual analysis.

3. Data Structuring & Validation

The raw AI response is parsed and structured into consistent JSON format. The workflow includes validation steps to ensure food items were actually detected and reasonable calorie estimates were generated before proceeding.

4. Google Sheets Logging

Validated food data—including timestamp, user ID, food items, estimated calories, and any additional notes—is appended to a designated Google Sheet. This creates a permanent, organized record accessible for analysis or reporting.

5. User Confirmation

Finally, the workflow sends a confirmation message back through LINE, informing the user that their food has been logged and providing the calorie estimate. This closes the loop with immediate feedback.

What You'll Need

  1. A LINE Developer account with Messaging API channel configured
  2. OpenAI API credentials with access to the Vision model
  3. Google Cloud Platform account with Sheets API enabled
  4. A Google Sheet prepared with appropriate column headers
  5. n8n instance (cloud or self-hosted) to run the workflow
  6. Webhook URL from n8n to configure in LINE Developer console

Quick Setup Guide

Follow these steps to implement this food logging automation in your environment:

  1. Download and import the template JSON file into your n8n instance
  2. Configure LINE credentials in the webhook and HTTP Request nodes with your channel access token
  3. Add your OpenAI API key to the OpenAI Vision node settings
  4. Connect Google Sheets by authenticating and specifying your target spreadsheet ID
  5. Test the workflow by sending a food image to your LINE bot and verifying the Sheets entry
  6. Activate the workflow and share your LINE bot QR code with users

Pro tip: Start with a small group of test users to refine the AI's food recognition accuracy for your specific cuisine types before rolling out broadly.
