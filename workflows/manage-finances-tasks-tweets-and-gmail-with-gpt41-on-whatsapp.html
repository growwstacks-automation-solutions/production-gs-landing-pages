What This Workflow Does

This workflow transforms your WhatsApp into a versatile AI-powered command center. Instead of juggling multiple apps, you can send a simple text or voice message to handle daily tasks: log an expense, create a reminder, post a tweet, search your inbox, or get real-time information. It bridges the convenience of chat with the power of automation.

The system intelligently routes your request. Say "spent $45 on lunch" and it logs to Google Sheets. Ask "remind me to call John tomorrow" and it creates a Google Task. Command "tweet: Just launched our new feature!" and it publishes to X. It understands intent and executes the correct action, saving you countless clicks and mental context switches every day.

How It Works

1. Message Reception & Voice Processing

A webhook via Green-API captures incoming WhatsApp messages. If you send a voice note, it's automatically transcribed to text using OpenAI's Whisper model, making the assistant fully voice-capable.

2. Intent Recognition & Routing

The core AI agent (GPT‑4.1) analyzes the message to determine your goal. It classifies the request into categories: finance logging, task management, social posting, email search, or general Q&A. Strict routing rules ensure one clear action per command.

3. Action Execution

Based on the classified intent, a specific sub-agent or tool is triggered. For finances, it parses amounts and categories, then appends a row to Google Sheets. For tasks, it interacts with Google Tasks API. For tweets, it formats and posts via X API. Each action is performed with the appropriate permissions and safeguards.

4. Response Delivery

The result is sent back as a clear reply within WhatsApp. If the original input was a voice message, the reply can also be converted to speech using text-to-speech, creating a natural conversational loop.

What You'll Need

  1. A Green-API account to connect WhatsApp webhooks.
  2. An OpenAI API key for GPT‑4.1, Whisper transcription, and optional TTS.
  3. Google Sheets and Google Tasks access (with appropriate OAuth scopes).
  4. Twitter/X API credentials for posting.
  5. Gmail API access (read-only for searching).
  6. SerpAPI or similar for real-time web searches (weather, news, etc.).
  7. An n8n instance (cloud or self-hosted) to run the workflow.

Pro tip: Start with a single function, like expense tracking, to test the flow. Once comfortable, enable additional tools one by one. Use the built-in routing logic to prevent unintended actions.

Quick Setup Guide

  1. Import the template: Download the JSON file and import it into your n8n workspace.
  2. Configure credentials: Set up the required API keys and OAuth connections for each service (Green-API, OpenAI, Google, Twitter, etc.).
  3. Set the webhook: Point your Green-API webhook to the n8n webhook node URL to receive WhatsApp messages.
  4. Test with a simple command: Send a message like "add task: Review report" to verify the Google Tasks integration works.
  5. Refine routing prompts: Optionally adjust the AI agent's system prompt to better match your terminology and common requests.
  6. Go live: Once testing is successful, activate the workflow and start using your WhatsApp assistant daily.
