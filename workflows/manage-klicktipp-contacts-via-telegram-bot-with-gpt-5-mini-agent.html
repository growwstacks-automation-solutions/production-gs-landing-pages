What This Workflow Does

This automation solves a common problem for marketing and sales teams: CRM data is trapped in complex interfaces that require training to use effectively. Team members waste time switching between communication apps (like Telegram) and their contact management system (KlickTipp) to look up information or update records.

The workflow bridges this gap by connecting a Telegram bot to KlickTipp through an intelligent AI agent. Users can simply type natural language commands in Telegram—such as "Show me the contact with email john@example.com" or "Tag all customers from Berlin as 'Local Leads'"—and the AI interprets these requests, executes the appropriate KlickTipp API operations, and returns the results directly in the chat.

This creates a mobile-first, conversational interface for your contact database that reduces training overhead, minimizes manual data entry errors, and keeps your CRM updated based on real-time conversations.

How It Works

1. Telegram Message Trigger

The workflow starts when a user sends a message to your configured Telegram bot. This trigger captures the message content and chat metadata, then passes it to the AI agent for interpretation.

2. AI Agent Interpretation

An AI model (GPT-5 mini, Claude, or OpenAI) analyzes the natural language message using a specialized prompt that understands KlickTipp operations. The agent determines the user's intent—whether they want to search, update, tag, or create contacts—and maps this to specific API calls.

3. KlickTipp API Execution

Based on the AI's interpretation, the workflow executes the corresponding KlickTipp operation. This could be searching contacts by email, listing contacts with specific tags, updating contact fields, applying new tags, or creating opt-in processes.

4. Response Generation & Delivery

The results from KlickTipp are formatted into a readable response by the AI agent, then sent back to the user through the same Telegram chat. The conversation state is maintained using a memory buffer, allowing for follow-up questions without repeating context.

What You'll Need

  1. A Telegram bot token from BotFather
  2. KlickTipp API credentials with appropriate permissions
  3. An AI provider account (OpenAI, Anthropic Claude, or compatible LLM)
  4. n8n instance (cloud or self-hosted) with internet access
  5. Basic understanding of webhooks and API authentication

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance.
  2. Configure the Telegram trigger node with your bot token from BotFather.
  3. Set up the AI model node with your API key (OpenAI/Claude) and adjust the system prompt if needed.
  4. Connect all KlickTipp nodes using your API credentials from the KlickTipp dashboard.
  5. Test with simple queries like "Find contact with email test@example.com" before expanding to more complex operations.
  6. Activate the workflow and start using your Telegram bot for contact management.

Pro tip: Start with read-only operations (contact searches) during testing before enabling write operations (tagging, updates). Implement a simple approval mechanism for destructive actions until you're confident in the AI's interpretation accuracy.
