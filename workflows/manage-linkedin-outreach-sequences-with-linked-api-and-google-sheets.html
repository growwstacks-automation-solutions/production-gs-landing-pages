What This Workflow Does

Manual LinkedIn prospecting is time-consuming and inconsistent. Sales teams spend hours each day sending connection requests, tracking acceptances, and following up—often losing leads in spreadsheets or forgetting timely follow-ups.

This n8n workflow automates your entire LinkedIn outreach sequence. It sends personalized connection requests to targeted prospects, monitors when they accept, delivers your pre-written message sequence, and tracks every interaction in Google Sheets. The system respects LinkedIn's daily limits, includes smart delays between actions, and marks leads as "no response" after appropriate timeouts.

You get a fully automated prospecting machine that works 24/7, ensures no lead falls through the cracks, and provides complete visibility into your outreach performance through automated reporting.

How It Works

1. Lead Import & Status Check

The workflow reads leads from your Google Sheet where status is marked as "NEW." It checks daily connection limits to ensure compliance with LinkedIn's guidelines before proceeding with any outreach.

2. Connection Request Phase

Using Linked API, the system sends personalized connection requests to each new lead. Each request includes a customized note based on the prospect's profile information, increasing acceptance rates.

3. Acceptance Monitoring

The workflow periodically checks which connection requests were accepted, expired, or declined. Accepted connections trigger the next phase, while expired requests are marked appropriately in your tracking sheet.

4. Message Sequence Delivery

Once a connection is accepted, the system waits a configured delay (typically 24 hours) then sends your first follow-up message. Up to three messages can be configured with increasing delays between each.

5. Response Tracking & Follow-up

The system monitors for replies to your messages. If no reply is received within a configured timeframe, leads are marked as "NO_RESPONSE" in Google Sheets, allowing for future re-engagement campaigns.

6. Complete Activity Logging

Every action—connection sent, acceptance status, messages delivered, replies received—is timestamped and logged in your Google Sheet, creating a complete audit trail of your outreach efforts.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. Linked API account with valid API credentials for LinkedIn automation
  3. Google Sheets with OAuth2 access configured in n8n
  4. Google Sheet template containing your lead list with columns for status, messages, and timestamps
  5. Pre-written message templates (3 variations recommended) for your follow-up sequence

Pro tip: Start with conservative daily limits (20-25 connection requests) and gradually increase as you establish consistent sending patterns. Personalize connection notes with specific references to the prospect's recent activity or shared connections for higher acceptance rates.

Quick Setup Guide

Follow these steps to implement this LinkedIn outreach automation in under 30 minutes:

  1. Download the template using the button above and import it into your n8n instance.
  2. Copy the Google Sheet template from the provided link in the workflow description and customize the column headers if needed.
  3. Configure credentials in n8n for Google Sheets (OAuth2) and Linked API (API key).
  4. Update the Config node with your Google Sheet URL and adjust timing parameters based on your outreach strategy.
  5. Add your leads to the Google Sheet with status "NEW" and populate your message templates.
