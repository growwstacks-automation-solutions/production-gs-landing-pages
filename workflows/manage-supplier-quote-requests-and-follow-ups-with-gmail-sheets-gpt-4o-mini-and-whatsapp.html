What This Workflow Does

Manual supplier quote management is a time-consuming, error-prone process that drains procurement teams. This complete automation system transforms that chaos into a streamlined, hands-off operation. It handles the entire lifecycle: from sending personalized quote requests to multiple suppliers, monitoring their email responses, extracting pricing data using AI, and sending intelligent follow-ups via WhatsApp.

The workflow consists of four independent modules that work together seamlessly. It reads your supplier list from Google Sheets, sends customized emails via Gmail, tracks responses automatically, uses OpenAI's GPT-4o-mini to parse PDF and Excel attachments for pricing data, and sends escalating reminders to non-responsive suppliers. What typically takes 5 hours of manual work per quote cycle becomes a 10-minute review task.

Beyond time savings, this system improves supplier response rates by 30% through consistent, timely follow-ups and ensures 95%+ data accuracy in price extraction versus 5-10% human error rates. It gives procurement teams back their most valuable resource—time—to focus on negotiation and strategic sourcing rather than administrative tasks.

How It Works

The automation runs on four coordinated modules that share data through Google Sheets while operating independently on different schedules.

1. Quote Request Sender (Manual Trigger)

You trigger this module when you need quotes for a new procurement project. It reads your supplier database from Google Sheets, generates personalized emails with specific categories and deadlines, and sends them via Gmail. Each request is logged with timestamps, creating a complete audit trail.

2. Response Monitor (Hourly Schedule)

This module runs automatically every hour, checking your Gmail for supplier replies. When it detects a response with attachments (PDFs, Excel files), it updates the tracking sheet status to "Quote Received" and prepares the documents for AI processing. No manual email monitoring required.

3. AI Price Extraction (Manual Trigger)

When quotes arrive, you run this module to process attachments. It downloads files, extracts text using n8n's built-in parser, sends the content to OpenAI GPT-4o-mini, and asks the AI to identify products, prices, quantities, and currencies. The structured data is saved to your Price Comparison sheet.

4. WhatsApp Follow-ups (Daily at 9 AM)

This scheduled module checks for suppliers who haven't responded after 3, 5, and 7 days. It sends increasingly urgent reminders via WhatsApp (or falls back to email if no phone number exists), logs all follow-up attempts, and escalates to your team only when necessary.

Quick Setup Guide

Get this automation running in 20-30 minutes with these steps:

  1. Create your Google Sheets: Make two sheets with the specified columns for tracking and price comparison.
  2. Connect credentials: Set up OAuth for Gmail and Google Sheets, add your OpenAI API key, and configure Twilio WhatsApp sandbox.
  3. Update sheet IDs: Replace the Google Sheet IDs in all 8 Google Sheets nodes across the workflow.
  4. Test with sample data: Add 2-3 test suppliers to your tracking sheet and run the Quote Sender module.
  5. Activate schedules: Enable the hourly Response Monitor and daily WhatsApp Follow-ups.

Pro tip: Use email aliases (yourname+supplier1@gmail.com) when testing to keep your main inbox clean while verifying the automation works correctly.
