What This Workflow Does

This automation solves the time-intensive, often inconsistent process of conducting market research and writing business case studies. For entrepreneurs, consultants, analysts, and product managers, turning a business idea into a validated, structured report can take weeks of manual effort.

The workflow automates the entire pipeline. You provide a simple prompt like “Analyze the market opportunity for electric scooter rentals in Lisbon.” The system then scopes the research, performs live web searches for the latest data, analyzes competitive landscapes and trends using multiple AI models, and synthesizes everything into a professional 1500-word business case study—automatically saved to Google Docs.

It transforms vague curiosity into actionable, data-supported intelligence, enabling faster and more confident strategic decisions.

How It Works

1. Chat Trigger & Research Scoping

The workflow starts with a user prompt sent via a built-in chat interface. A GPT-4o node breaks down the query into structured components: industry, geography, target audience, key trends, and potential challenges. This creates a clear research framework.

2. Live Web Research with Perplexity

Using the structured framework, the Perplexity Sonar node performs deep, live web research. It retrieves current industry statistics, consumer trend reports, competitor information, regulatory updates, and market forecasts—all from real-time sources.

3. Analysis & Synthesis with Claude

The raw research data is passed to Anthropic’s Claude Sonnet model. Claude excels at nuanced reasoning and long-form writing. It analyzes the gathered information, identifies key insights, assesses risks and opportunities, and structures the findings into a logical narrative.

4. Report Generation & Export

The final synthesized analysis is formatted into a comprehensive business case study with standard sections: Executive Summary, Market Overview, Opportunity Analysis, Competitive Landscape, Risks & Challenges, Strategic Recommendations, and Conclusion. This completed report is then appended to a designated Google Doc, ready for sharing or further editing.

What You'll Need

  1. A self-hosted n8n instance (this workflow uses community nodes).
  2. API keys for OpenAI (GPT-4o), Anthropic (Claude), and Perplexity.
  3. A Google Cloud project with the Google Docs API enabled and appropriate credentials.
  4. A target Google Doc ID where the final reports will be saved.
  5. Basic familiarity with n8n to configure credentials and import the JSON template.

Quick Setup Guide

  1. Download the template using the button above and save the JSON file.
  2. In your n8n instance, go to Workflows → Import from File and select the downloaded JSON.
  3. Configure credentials for OpenAI, Anthropic, Perplexity, and Google Docs in n8n’s credentials panel.
  4. In the workflow, update the “Google Docs” node with your target document ID.
  5. Test the trigger by activating the workflow and sending a test prompt via the Chat Trigger node.
  6. Monitor the execution and check your Google Doc for the generated business case study.
