What This Workflow Does

Manual competitor ad spying is time-consuming, inconsistent, and often misses strategic patterns. This automation solves that by building a complete Facebook Ad Intelligence pipeline. It continuously collects live ads from specified Facebook Ad Library URLs, filters for high-signal advertisers, classifies ads by format (video, image, or text), and applies advanced AI analysis to extract strategy, positioning, and reusable creative angles.

The result is a reliable, hands-off system that turns competitor ads into structured, searchable intelligence stored in Google Sheets. Instead of folders of screenshots, you get a living database of insights—deep strategic summaries, repurposed ad copy, and visual analysis—enabling evidence-based creative decisions and faster iteration.

How It Works

The workflow orchestrates several powerful tools into a seamless intelligence-gathering machine.

Step 1: Ad Data Collection with Apify

The Apify node automatically scrapes your target Facebook Ad Library URLs, retrieving raw ad snapshots including copy, headlines, media URLs, page metadata, and engagement signals. This replaces manual browsing and screenshotting.

Step 2: Signal Filtering

A filter node ensures only ads from serious advertisers move forward—for example, filtering for pages with more than 5,000 likes. This removes noise and focuses analysis on competitors with proven traction and budget.

Step 3: Creative Type Detection & Routing

A switch node classifies each ad as video, image, or text-only. Each format is routed to its own specialized processing path, ensuring the correct AI model and analysis method is applied for maximum insight accuracy.

Step 4: AI-Powered Creative Analysis

Image ads are converted to binary and analyzed visually by GPT-4o. Video ads are downloaded and analyzed frame-by-frame using Google Gemini. Text ads are summarized and rewritten directly. The AI extracts strategic summaries, emotional hooks, value propositions, and suggests repurposed copy.

Step 5: Structured Intelligence Storage

All insights—summary, rewritten copy, prompts, and source data—are automatically appended to a Google Sheet, deduplicated by ad archive ID. Each row becomes a reusable asset for your creative team, building a centralized, searchable intelligence hub.

What You'll Need

  1. An Apify account with access to the Facebook Ad Library scraping actor.
  2. OpenAI API key with text and image analysis access (GPT-4o).
  3. Google Gemini API access for video analysis capabilities.
  4. Google Sheets with OAuth access and edit permissions on your target spreadsheet.
  5. Dropbox account (optional) for OAuth access if you wish to store and reuse video assets.
  6. n8n instance, either self-hosted or cloud.

Quick Setup Guide

Import and configure this template in under 15 minutes to start gathering intelligence.

  1. Download and Import: Click the download button above to get the JSON file. In your n8n instance, go to Workflows > Import from File and select the downloaded template.
  2. Configure Credentials: Set up credentials in n8n for Apify, OpenAI, Google Gemini, and Google Sheets (and Dropbox if used).
  3. Set Your Targets: In the Apify node, enter the specific Facebook Ad Library URLs you want to monitor. You can add multiple URLs for different competitors or keywords.
  4. Adjust Filters (Optional): Modify the filter node criteria (e.g., minimum page likes) to match your signal quality threshold.
  5. Connect Your Sheet: In the Google Sheets node, specify the exact spreadsheet and worksheet where insights should be stored. The workflow will create the necessary columns if they don't exist.
  6. Test and Schedule: Run the workflow once manually to ensure all connections work. Then, set a schedule (e.g., daily or weekly) to automate continuous intelligence gathering.

Pro tip: Start by monitoring 2-3 of your closest competitors. Let the system run for a week to build an initial insights database before expanding your target list. Review the Google Sheet weekly to identify emerging patterns.
