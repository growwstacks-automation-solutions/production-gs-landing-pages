What This Workflow Does

This enterprise-grade automation system solves the time-consuming challenge of manually tracking competitor activity on Instagram. It automatically:

  • Fetches latest posts from competitor accounts you specify
  • Calculates engagement metrics and performance trends
  • Generates AI-powered strategic insights using Claude 3.5
  • Delivers alerts via WhatsApp and email with key findings
  • Maintains historical logs in Google Sheets for trend analysis

By automating what would normally take hours of manual work each week, this workflow gives you real-time competitive intelligence without constant manual checking.

How It Works

1. Scheduled Data Collection

The workflow runs daily (or on your preferred schedule) to fetch the last 10 posts from each competitor account via Instagram's Graph API.

2. Performance Analysis

Each post's engagement metrics are analyzed to calculate:

  • Average engagement rates
  • Performance trends (improving/declining)
  • Content type effectiveness

3. AI Insight Generation

Claude 3.5 processes the collected data to identify:

  • Emerging content themes
  • Successful engagement strategies
  • Potential opportunities for your own content

4. Multi-Channel Alerting

Key findings are delivered through:

  • Concise WhatsApp alerts via Twilio for immediate visibility
  • Detailed email reports with metrics and AI analysis

5. Historical Logging

All data is recorded in Google Sheets for long-term trend analysis and benchmarking.

What You'll Need

  1. Instagram Business account with API access
  2. Google Sheets with Competitors and AlertsLog tabs
  3. Claude AI API credentials (claude-3-5-sonnet model)
  4. Twilio account for WhatsApp alerts
  5. Email/SMTP service for detailed reports
  6. n8n instance (v1.50+ recommended)

Quick Setup Guide

  1. Import the workflow into your n8n instance
  2. Configure credentials for all connected services
  3. Update spreadsheet IDs with your Google Sheets
  4. Add competitor accounts to the Competitors sheet
  5. Set alert recipients for WhatsApp and email
  6. Test with manual execution to verify all steps
  7. Schedule daily runs or trigger via webhook
