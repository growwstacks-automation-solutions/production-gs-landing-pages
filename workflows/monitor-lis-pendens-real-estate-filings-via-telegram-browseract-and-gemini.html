What This Workflow Does

This automation transforms your Telegram bot into a powerful real estate monitoring tool that tracks Lis Pendens filings (pre-foreclosure notices) across county records. When you request property data via Telegram, the system:

  • Scrapes public records using BrowserAct automation
  • Processes messy legal documents with Gemini AI
  • Delivers clean, structured property reports directly to your chat

Real estate investors save 10+ hours per week by eliminating manual searches while gaining earlier access to distressed property leads than competitors relying on traditional MLS listings.

How It Works

1. Telegram Request Processing

The workflow starts when you message your Telegram bot with search parameters like "Check filings for Miami-Dade county last 7 days." An AI agent interprets natural language requests into structured queries.

2. Automated Records Search

BrowserAct executes background searches across county clerk websites, handling CAPTCHAs and complex interfaces that normally require manual navigation. It extracts all matching Lis Pendens filings.

3. AI Document Processing

Gemini AI analyzes the raw legal text to identify key details: property addresses, filing dates, plaintiff banks, and attorney contacts. It removes boilerplate language and formats a clean report.

4. Telegram Delivery

The system splits large results into digestible messages and delivers them to your Telegram chat with download links for full documents. You can request specific property details via follow-up messages.

Pro tip: Set this workflow to run automatically each morning to get fresh filings before competitors.
