What This Workflow Does

For parents, educational consultants, and institutions, tracking school admission announcements is a stressful, time-consuming task. Manually checking multiple websites daily leads to missed deadlines and last-minute rushes. This automation solves that problem completely.

The workflow automatically monitors your list of school websites for admission updates. It uses web scraping to fetch content, Google Gemini AI to intelligently analyze whether admissions have opened for your target year and class, and sends you an immediate email alert when they do. It runs on a daily schedule, working 24/7 so you don't have to.

How It Works

1. Scheduled Trigger & School List

A Cron node triggers the workflow daily. It reads from your configured list of schools and their specific admission page URLs.

2. Web Content Fetching & Cleaning

For each school, an HTTP Request node fetches the latest HTML from the admission page. The content is cleaned to remove irrelevant code, focusing on the textual information about admissions.

3. AI-Powered Admission Detection

The cleaned HTML is sent to Google Gemini with a precise prompt like: "Analyze this school webpage content. Determine if pre-nursery admissions for the 2026-27 academic year have been announced. Respond only with 'YES' or 'NO' with a brief reason." The AI understands context, not just keywords.

4. Conditional Alerting

An IF node checks the AI's response. If it's "YES" with confirmation, the workflow proceeds to notification. If "NO", it moves to the next school without alerting you.

5. Instant Email Notification

When admissions are detected, the Send Email node triggers, sending you a detailed alert with the school name, the detected information, and a direct link to the admission page so you can act immediately.

Who This Is For

This automation is ideal for parents applying to multiple schools for their children, educational consultants managing client applications, tutoring centers informing their students, and school admission offices monitoring competitor announcements. Anyone who needs to stay ahead of admission deadlines across multiple institutions will benefit.

What You'll Need

  1. An n8n instance (cloud or self-hosted) to run the workflow.
  2. A Google Gemini API key (free tier available) for the AI analysis.
  3. SMTP email credentials (Gmail, Outlook, or your company email server) to send alerts.
  4. Your list of target schools with their specific admission page URLs.

Quick Setup Guide

  1. Import the template into your n8n workspace using the downloaded JSON file.
  2. Configure the Gemini node by adding your API key and adjusting the prompt if tracking a different class/year.
  3. Set up the email node with your SMTP details, "From" address, and "To" recipient email.
  4. Update the "Shortlisted Schools" node with your actual list of schools and their admission page URLs.
  5. Activate the workflow and test with one school to ensure alerts work correctly.

Pro tip: Start by monitoring just 2-3 schools to verify the AI correctly interprets their website content. Adjust the prompt slightly if needed for specific wording patterns before scaling to your full list.
