What This Workflow Does

This n8n automation continuously monitors any Ethereum wallet address for changes in its USDT (ERC-20) token balance. It checks Etherscan's blockchain data at regular intervals and sends instant Telegram notifications whenever the balance increases or decreases.

For crypto businesses, traders, and financial managers, manually checking wallet balances wastes valuable time and risks missing critical transactions. This workflow eliminates that hassle with automated monitoring and real-time alerts.

The complete workflow showing Etherscan API integration and Telegram notification setup

How It Works

1. Regular Etherscan API checks

The workflow queries Etherscan's API at your configured interval (default: 15 minutes) to fetch the latest USDT balance for your specified wallet address.

2. Balance change detection

Each API response is compared against the previous balance. The system detects both incoming deposits and outgoing transfers.

3. Telegram notification triggers

When a balance change is detected, the workflow formats a detailed alert message and sends it to your Telegram account via a bot.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure your Etherscan API key in the HTTP Request node
  3. Add your Telegram bot token and chat ID in the Telegram node
  4. Set your wallet address in the Etherscan API parameters
  5. Adjust the polling interval if needed (default: 15 minutes)
  6. Activate the workflow and test with a small USDT transfer
