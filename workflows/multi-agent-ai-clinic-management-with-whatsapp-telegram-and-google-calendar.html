What This Workflow Does

Healthcare clinics juggle patient communication, appointment scheduling, reminders, internal task management, and media processing—all manually. This creates bottlenecks, missed appointments, slow responses, and staff burnout.

This n8n workflow solves that by automating the entire clinic management lifecycle using AI agents. It connects WhatsApp for patient interactions, Telegram for staff commands, Google Calendar for appointments, and Google Tasks for internal reminders. The system intelligently routes messages, processes audio/images, sends proactive confirmations, and escalates complex cases to human staff.

You get a 24/7 digital assistant that handles inquiries, schedules appointments, confirms bookings, manages rescheduling, creates shopping lists for staff, and keeps everything synchronized across platforms—freeing your team to focus on patient care.

How It Works

1. WhatsApp Patient Interaction Flow

Patients message your clinic via WhatsApp. The workflow captures these via Evolution API webhook, classifies the content (text, image, audio), processes media using OpenAI models, and routes the request to a specialized AI agent. The agent responds appropriately, formats the reply for WhatsApp, and sends it back.

2. Telegram Staff Management Flow

Staff send commands via Telegram (e.g., “reschedule appointment for John tomorrow”). The workflow interprets the request, updates Google Calendar via MCP client, adds tasks to Google Tasks, and sends a confirmation back to the staff member.

3. Automated Appointment Reminders

A daily scheduled trigger scans Google Calendar for next-day appointments. For each, it sends a WhatsApp confirmation message to the patient. If the patient replies “cancel” or “reschedule,” the workflow routes that to the appropriate agent to update the calendar.

4. Media Processing & Escalation

Audio messages are downloaded, converted, and transcribed using OpenAI Whisper. Images are analyzed with OpenAI Vision to extract text. If content is sensitive or urgent, the workflow escalates the conversation to a human staff member for manual handling.

What You'll Need

  1. n8n instance (cloud or self-hosted) version 1.88.0 or higher.
  2. Evolution API credentials for WhatsApp integration.
  3. Telegram Bot token for staff command channel.
  4. Google Calendar & Google Tasks OAuth2 access for appointment and task management.
  5. OpenAI API key (or OpenRouter) for AI agents, Whisper, and Vision.
  6. PostgreSQL database (or alternative) for persistent chat memory.
  7. Clinic-specific configuration: operating hours, escalation rules, cancellation policies.

Quick Setup Guide

Follow these steps to deploy this automation in your clinic:

  1. Import the workflow: In n8n, go to Workflows → Import from File and upload the downloaded JSON.
  2. Configure credentials: Set up Evolution API, Telegram Bot, Google Calendar/Tasks OAuth2, OpenAI/OpenRouter, and PostgreSQL connections.
  3. Replace placeholder variables: Update instance names, WhatsApp numbers, server URLs, API keys, and calendar IDs in the workflow nodes.
  4. Customize AI prompts: Adjust system prompts to match your clinic’s tone, services, hours, and escalation rules.
  5. Test thoroughly: Simulate patient WhatsApp messages, staff Telegram commands, and verify daily reminders work.
  6. Activate & monitor: Turn on the workflow, monitor logs, and refine based on real‑world usage.

Pro tip: Start with a small pilot—automate just appointment reminders first. Once stable, expand to full patient interaction and staff command handling.
