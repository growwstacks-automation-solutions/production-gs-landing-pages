What This Workflow Does

This automation solves the time-intensive problem of creating consistent, high-quality content for SEO blogs and email newsletters. Manual content creation requires hours of research, writing, editing, and formatting—tasks that become repetitive and scale poorly with business growth.

The workflow employs a team of four specialized AI agents that collaborate sequentially. The Research Agent gathers factual information and credible sources. The Outline Agent structures the content logically. The Writer Agent produces engaging, Medium-style prose. Finally, the Editor Agent polishes the text and optimizes it for search engines. Based on your input, the system automatically routes output to either blog format (with DALL-E generated featured images) or newsletter format, then saves everything to Airtable or Google Sheets for management.

How It Works

Step 1: Topic Input & Research

You submit a content topic through a form or trigger. The Research Agent uses OpenRouter (with Grok AI) to search for relevant facts, statistics, and authoritative sources, building a foundation of credible information.

Step 2: Outline Generation

The Outline Agent analyzes the research and creates a logical structure with headings, subheadings, and key points. This ensures the content follows best practices for readability and SEO hierarchy.

Step 3: Content Writing

The Writer Agent, powered by Google Gemini, transforms the outline into engaging, human-like prose. It adopts a conversational yet professional tone suitable for blog readers or newsletter subscribers.

Step 4: Editing & SEO Optimization

The Editor Agent reviews the draft, improves flow, checks grammar, and inserts SEO keywords naturally. It ensures the content meets both quality standards and search engine requirements.

Step 5: Formatting & Distribution

The system branches based on your selection: blog posts receive a DALL-E generated featured image and are saved to Airtable; newsletters are formatted for email and saved to Google Sheets. Optional Telegram notifications alert you when content is ready.

What You'll Need

  1. OpenRouter API key – for accessing Grok AI model used by the Research Agent.
  2. Google Gemini API key – powers the Writer Agent for high-quality prose generation.
  3. OpenAI API key – enables DALL-E image generation for blog post featured images.
  4. Airtable account – stores finalized blog posts with metadata for easy management.
  5. Google Sheets account – serves as a content calendar and repository for newsletter drafts.
  6. Telegram bot token (optional) – sends notifications when content generation completes.

Quick Setup Guide

  1. Import the downloaded JSON workflow file into your n8n instance.
  2. Add your OpenRouter, Google Gemini, and OpenAI API credentials to the respective language model nodes.
  3. Configure your Airtable base ID and table ID in the "Save Blog to Airtable" node.
  4. Set your Google Sheets document ID in the "Save Newsletter to Google Sheets" node.
  5. If using notifications, add your Telegram bot token and chat ID to the Telegram node.
  6. Test the workflow by submitting a sample topic through the form trigger and verify output appears in your Airtable/Sheets.

Pro tip: Start with simple topics like "benefits of remote work" or "intro to cryptocurrency" to see how the agents collaborate. Once confident, expand to complex subjects requiring deeper research.
