How It Works

1. Conversational Interface

The workflow begins with a chat trigger that receives user messages. A "Music Producer Agent" powered by Google Gemini engages users conversationally, asking questions about their desired song title, musical style, lyrical content, and elements to avoid. This natural interaction makes the system accessible to anyone, regardless of technical or musical knowledge.

2. Intelligent Tool Selection

The AI agent dynamically chooses between different tools based on user needs. For original lyrics, it calls the "Songwriter" tool to generate creative content. For existing songs, it uses the "Search songs" tool to find and adapt lyrics. This adaptive intelligence ensures the system provides exactly what each user requires.

3. Data Validation & Formatting

Collected data passes through multiple validation layers including JSON format checking, parsing, and structure fixing. This robust error handling cleans malformed AI responses and ensures consistent data flow to downstream processes, dramatically reducing failure rates in production environments.

4. AI Music Generation

The formatted song parameters are sent to the Kei AI API (which accesses Suno's music generation capabilities) to create the actual audio track. The workflow uses webhook-based resume functionality to handle the asynchronous nature of AI music generation, which can take several minutes to complete.

5. Result Processing & Storage

Once generation completes, the system retrieves the finished songs, downloads each audio file, renames them with timestamps for organization, and automatically uploads them to a specified Google Drive folder. This complete hands-off process ensures generated content is immediately available for use.

What You'll Need

  1. Google Gemini API key for the conversational AI components
  2. Gemini Search API access for lyric search functionality
  3. Kei AI Bearer Token to access Suno music generation capabilities
  4. Google Drive account with OAuth2 authentication configured
  5. n8n instance (cloud or self-hosted) with webhook capabilities
  6. Publicly accessible URL for webhook callbacks if using asynchronous processing

Pro tip: Start with simple song requests to test the system before moving to complex compositions. The AI handles everything from pop songs to cinematic scores, but simpler requests typically complete faster and more reliably during initial setup.

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance
  2. Configure API credentials in the Gemini Chat Model nodes and HTTP Request nodes
  3. Set up Google Drive OAuth2 in the upload node and verify your target folder ID
  4. Configure webhook settings in the Wait node and update your Kei AI callback URL
  5. Test with sample requests like "Write a pop song about summer" or "Create an upbeat electronic track for a product launch"
  6. Customize prompts and parameters to match your specific brand voice and musical preferences
