What This Workflow Does

This automation solves the time-consuming challenge of creating extended video content from existing clips. Instead of manually editing or reshooting scenes, this workflow uses AI to analyze your starting video and generate logical scene extensions that continue the narrative.

The system takes a control panel approach using Google Sheets, where you specify your starting video, how many scenes to extend, aspect ratio preferences, and narrative guidance. It then automatically processes each scene: analyzing the current clip, extracting the last frame, generating a prompt for the next scene, creating the new video with Veo3, and stitching everything together into a seamless final video.

For content creators, marketers, and businesses producing regular video content, this represents a 90% reduction in production time and costs. What used to require hours of editing and coordination can now be automated with consistent, high-quality results.

How It Works

1. Input Setup & Control Panel

A Google Sheet serves as your command center. You enter the starting video URL, number of scenes to generate, aspect ratio, model preference (V3 or V3 Fast), narrative theme, and specific scene instructions. The workflow checks for rows marked "For Production" and processes them automatically.

2. Scene Analysis & Understanding

The workflow sends your current video to File.AI's Video Understanding API, which analyzes the last frame, audio, and scene details. This creates a detailed context description that informs what should happen next in the narrative sequence.

3. AI Prompt Generation

An AI Agent node combines the video analysis with your narrative theme and scene instructions to create a precise prompt for the next scene. This ensures the generated content maintains visual consistency while advancing the story logically.

4. Frame Extraction & Scene Generation

The last frame is extracted from the current video and used as a reference image. This is sent to Key.AI (Veo3 Fast) along with the generated prompt to create the next scene. The system waits for generation completion and retrieves the new clip URL.

5. Looping & Final Assembly

The workflow loops through each requested scene, incrementally building your narrative chain. Once all scenes are generated, File.AI's Merge Videos API stitches everything together into a final MP4. The control sheet is updated with the final video URL and marked "Done."

What You'll Need

  1. n8n instance (cloud or self-hosted)
  2. Google Sheets account for the control panel
  3. File.AI API access for video analysis and merging
  4. Key.AI API access for Veo3 video generation
  5. Starting video clips in MP4 format
  6. Clear narrative instructions for scene progression

Pro tip: Start with simple, well-lit scenes with clear subjects. AI generates best results when the starting video has good lighting, consistent framing, and unambiguous visual elements. Complex action sequences or rapidly changing scenes may require more specific instructions.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance.
  2. Create your Google Sheet with the required columns: Video URL, Scene Count, Aspect Ratio, Model, Narrative Theme, Special Requests, and Status.
  3. Configure API credentials for File.AI and Key.AI in the respective n8n nodes.
  4. Set up the Google Sheets node with your sheet ID and appropriate ranges.
  5. Test with a single scene extension using a simple starting video to verify the workflow.
  6. Adjust narrative instructions based on initial results to refine scene coherence.
  7. Schedule or trigger the workflow based on your content production needs.
