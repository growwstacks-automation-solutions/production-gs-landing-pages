What This Workflow Does

Manual news monitoring and sentiment analysis consume valuable hours that could be spent on strategic decision-making. This workflow automates the entire process using a sophisticated two-agent AI system. It transforms a simple query into comprehensive market intelligence by researching current information and analyzing its emotional tone.

The system operates like having a dedicated research team and analysis department working in perfect harmony. The first agent gathers relevant information from across the web, while the second specializes in understanding the emotional context of that information. This division of labor ensures both breadth of research and depth of insight.

Businesses use this automation to track brand mentions, monitor competitors, identify emerging trends, and understand market sentiment without manual effort. It turns scattered information into structured, actionable intelligence that informs marketing strategies, product development, and competitive positioning.

How It Works

The workflow follows a logical sequence where each agent has a specialized role, ensuring high-quality results through focused task execution.

Step 1: Chat Trigger Initiation

The process begins when you submit a query through n8n's chat interface. This could be a company name, product term, industry topic, or any subject requiring market intelligence. The system treats this input as the research starting point.

Step 2: Research Agent Investigation

The Research Agent receives your query and begins its investigation using Google Gemini as its reasoning engine. It has access to two specialized tools: a web_search function powered by your self-hosted SearXNG instance for real-time information gathering, and a get_current_date tool for temporal context. The agent compiles relevant findings into a concise summary.

Step 3: Handoff to Sentiment Analysis Agent

Once research is complete, the findings are automatically passed to the Sentiment Analysis Agent. This seamless handoff ensures no information is lost between stages and maintains context throughout the analysis process.

Step 4: Emotional Tone Analysis

The Sentiment Analysis Agent evaluates the compiled information to determine its emotional tone—positive, negative, or neutral. It considers contextual clues, language patterns, and comparative references to provide nuanced sentiment assessment rather than simple keyword matching.

Step 5: Final Report Delivery

The completed analysis returns through the chat interface, presenting both the researched information and its sentiment assessment. This delivers comprehensive intelligence in a single, easily digestible format ready for business decision-making.

Quick Setup Guide

Follow these steps to implement this automation in your n8n environment and begin receiving automated market intelligence.

  1. Download and Import: Download the template file and import it into your n8n instance using the workflow import function.
  2. Configure LLM Credentials: Set up your Google Gemini API credentials in the LLM nodes. The workflow is pre-configured for Gemini but can adapt to other models.
  3. Connect SearXNG: Configure the web search node with your SearXNG instance URL to enable real-time information gathering.
  4. Customize Agent Prompts: Adjust the system prompts for both agents to match your specific research focus and analysis requirements.
  5. Test with Sample Queries: Use the chat interface to test the workflow with sample topics and verify proper information flow between agents.
  6. Schedule Execution: Set up a schedule trigger to run the workflow automatically at intervals matching your monitoring needs (hourly, daily, weekly).

Pro tip: Start with broad industry terms before narrowing to specific brands. This helps calibrate the system's sensitivity and ensures you capture relevant context before focusing on precise targets.
