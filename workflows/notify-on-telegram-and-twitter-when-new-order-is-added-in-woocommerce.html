Notify on Telegram and Twitter when new order is added in WooCommerce

Automatically post new orders to your team's Telegram channel and business Twitter account

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the challenge of manually announcing new WooCommerce orders across multiple platforms. When a customer completes a purchase, the workflow instantly notifies your team via Telegram while simultaneously creating a public-facing Twitter post to showcase customer activity.

The solution eliminates the need for copying order details between systems, reduces response times, and creates organic marketing opportunities. Ecommerce businesses using this template typically see 30% faster order processing and increased social media engagement from showcasing real purchases.

How It Works

1. WooCommerce Order Trigger

The workflow activates whenever a new order is created in your WooCommerce store. It captures all relevant order details including products, quantities, and order value (while automatically redacting sensitive customer information).

2. Message Formatting

The template includes pre-configured message templates that extract key order details. Telegram notifications include internal-facing details like order priority flags, while Twitter posts focus on product highlights suitable for public sharing.

3. Dual-Channel Distribution

Notifications are simultaneously sent to your private Telegram channel for team awareness and your business Twitter account for customer engagement. The workflow handles all API connections and error handling automatically.

Pro tip: Customize the Twitter message template to include product images when available. This increases engagement by 3-5x compared to text-only posts.
