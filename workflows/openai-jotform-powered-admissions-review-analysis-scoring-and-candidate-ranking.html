What This Workflow Does

This automation transforms the college admissions process by combining Jotform application intake with OpenAI's AI analysis capabilities. It handles the entire admissions pipeline from initial application through to final candidate ranking.

The system automatically analyzes essays for writing quality, authenticity, and red flags while scoring applicants across five key dimensions: academic strength, extracurricular profile, personal qualities, institutional fit, and diversity contribution. Top candidates are flagged for immediate review while all applications are logged with detailed analytics.

How It Works

1. Application Intake

Prospective students submit complete applications through a customized Jotform that captures all required materials including essays, transcripts, and activities lists.

2. AI Essay Analysis

OpenAI's models evaluate each essay for grammar, vocabulary, narrative coherence, authenticity (detecting AI-generated content), and specific red flags like plagiarism indicators.

3. Holistic Scoring

Each applicant receives a composite score (0-100) based on weighted evaluation of academic credentials, extracurriculars, personal qualities, institutional fit, and diversity factors.

4. Smart Routing

Applications are automatically categorized and routed:

  • Strong Admit (85-100): Fast-tracked with immediate alerts
  • Committee Review (65-84): Detailed analysis for human review
  • Standard Review (<65): Processed with verification

Quick Setup Guide

  1. Create your Jotform application form with sections for essays, academics, and activities
  2. Import the workflow JSON into your n8n instance
  3. Configure OpenAI API credentials in both analysis nodes
  4. Customize scoring weights and email templates
  5. Connect your Google Sheet for applicant tracking
  6. Test with sample applications before going live

Pro tip: Start with a small batch of applications to calibrate your scoring thresholds before processing your full applicant pool.
