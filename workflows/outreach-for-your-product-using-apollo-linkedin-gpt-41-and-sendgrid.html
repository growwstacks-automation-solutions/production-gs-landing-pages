What This Workflow Does

Manual, spray-and-pray cold emailing is dead. Today's buyers expect relevance and personalization. This automated workflow transforms your outreach from generic blasts to hyper-targeted conversations that convert.

The system creates a complete AI‑driven sales engine: it scrapes qualified leads from Apollo.io, enriches each profile with LinkedIn career history and recent posts, uses GPT‑4.1 to craft emails that reference specific pain points, and dispatches them through SendGrid with full tracking. It also logs everything in Supabase to prevent duplicates and measure performance.

Instead of spending hours researching and writing individual emails, your sales team gets a steady stream of warm, personalized conversations while focusing on closing deals.

How It Works

1. Lead Acquisition & Storage

The workflow begins by scraping up to 500 targeted leads from Apollo.io using Apify HTTP nodes—no hard‑coded API keys required. Each lead's email and LinkedIn URL are immediately stored in Supabase with a timestamp, creating a clean, queryable database and eliminating duplicate outreach.

2. LinkedIn Profile Enrichment

For each stored lead, the system calls the RapidAPI LinkedIn endpoint to fetch detailed professional data: current role, past experience, skills, and their five most recent posts or articles. This creates a rich profile that goes far beyond basic contact information.

3. AI‑Driven Personalization

Two specialized AI agents work in sequence. First, the Product Description Agent analyzes the lead's profile against your product to identify specific pain points and potential value propositions. Then, the Story Agent—configured with your brand voice and outreach rules—generates a completely unique email that feels handwritten, referencing their career journey or recent content.

4. Email Assembly & Delivery

The HTML Modifier node ensures consistent branding and formatting while generating a compelling subject line. Finally, SendGrid delivers the email at scale while maintaining high deliverability (unlike Gmail nodes which often land in spam). Send status is logged back to Supabase for complete campaign tracking.

Pro tip: Use the conditional nodes to filter out leads from irrelevant industries or seniority levels before the AI generation step. This saves API credits and increases your overall reply rate.

Quick Setup Guide

Follow these steps to deploy this automation in under 30 minutes:

  1. Import the template: Download the JSON file above and import it into your n8n instance.
  2. Configure credentials: In n8n's credential manager, set up connections for Apollo.io, RapidAPI, OpenAI, SendGrid, and Supabase using your API keys.
  3. Adjust lead filters: Modify the Apollo HTTP node parameters to target your ideal customer profile (industry, company size, job title).
  4. Customize AI prompts: Edit the Product Description and Story Agent nodes to reflect your product's unique value proposition and desired brand tone.
  5. Test with a small batch: Run the workflow manually with 5‑10 leads to verify email quality and deliverability.
  6. Schedule automation: Set the workflow to trigger daily or weekly for continuous lead generation.

Security reminder: Never hard‑code API keys in HTTP nodes. Always use n8n's built‑in credential management for secure storage and rotation.
