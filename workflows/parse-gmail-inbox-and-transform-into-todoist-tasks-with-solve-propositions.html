What This Workflow Does

If your Gmail inbox is overflowing with emails that require action, this automation is your solution. It intelligently scans your unread emails, uses AI to analyze the content, and automatically creates structured Todoist tasks with clear action items and suggested responses.

The workflow transforms email chaos into organized action. Instead of manually reading through threads to identify what needs to be done, the AI extracts key information, proposes next steps, and creates ready-to-complete tasks. This is particularly valuable for teams sharing inboxes or individuals managing multiple projects through email.

Beyond simple task creation, the system maintains context by linking tasks back to original emails, ensuring you can always reference the full conversation. It also handles edge cases like existing tasks and provides intelligent filtering to prevent duplicate work.

How It Works

The automation follows a logical sequence to ensure reliable email-to-task conversion while maintaining data integrity.

1. Email Trigger & Filtering

The workflow activates on new unread emails in your Gmail inbox. You can configure it to run on a schedule (every 15 minutes, hourly) or use Gmail's IMAP trigger for real-time processing. Initial filters exclude automated emails, newsletters, and other non-actionable messages.

2. Existing Task Check

Before creating new tasks, the system checks your Todoist projects for existing tasks with similar titles. This prevents duplicate entries and ensures you're only creating tasks for genuinely new requests that haven't been addressed yet.

3. AI Analysis & Task Creation

For new actionable emails, the AI analyzes the entire thread and generates three key outputs: a concise summary of the email content, specific proposed actions to be taken, and a suggested reply template. This structured data then creates a comprehensive Todoist task with all necessary context.

4. Task Management & Cleanup

The system monitors task completion status and can automatically handle follow-up actions. If an email gets unstarred but the corresponding task remains open, the workflow can close the task, maintaining synchronization between your email and task management systems.

Pro tip: Start with a dedicated label or folder in Gmail for testing. Process emails from this folder first to refine the AI prompts before applying the workflow to your main inbox.

Quick Setup Guide

Import and configure this workflow in under 15 minutes with these steps:

  1. Download the template using the button above and import it into your n8n instance
  2. Configure Gmail credentials in the Email Trigger node using OAuth or app password
  3. Add your Todoist API token in the Todoist node (found in Todoist Settings → Integrations)
  4. Set up OpenAI credentials with your API key for the AI analysis component
  5. Test with a single email first by using the Manual Trigger node before enabling automated runs
  6. Adjust AI prompts in the OpenAI node to match your specific task formatting preferences
  7. Activate the workflow and monitor the first few executions to ensure proper task creation
