Bulk Email Verification with Icypeas

Automate email address validation at scale. Verify hundreds of contacts in minutes using Icypeas API and Google Sheets.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves a critical problem for sales, marketing, and recruitment teams: wasted time and resources contacting invalid email addresses. Manual email verification is tedious, error-prone, and doesn't scale. This workflow automates the entire process, connecting your Google Sheets contact lists directly to Icypeas' professional verification API.

The system reads contact data (first names, last names, company domains) from your spreadsheet, sends it to Icypeas for real-time validation, and returns accurate verification results. You get immediate insight into which contacts are valid, which should be removed, and which need further investigation—all without manual checking.

How It Works

1. Trigger and Data Input

The workflow starts with a manual trigger or scheduled automation. It connects to your Google Sheets document containing contact information. The system reads rows containing first names, last names, and company domains—the essential data needed for email discovery and verification.

2. Icypeas API Connection

Each contact record is sent to the Icypeas API using secure authentication (API key, secret, and user ID). The service performs comprehensive checks including syntax validation, domain existence, MX record verification, and mailbox detection to determine email deliverability.

3. Bulk Processing and Results

The workflow processes contacts in batches for efficiency, handling API rate limits automatically. Verification results are returned with status codes (valid, invalid, risky, unknown) and confidence scores. The system can process hundreds of contacts in minutes versus hours of manual work.

4. Output and Integration

Verified results are written back to your Google Sheets or exported to your preferred destination (CRM, database, or notification system). You get a clean, validated contact list ready for marketing campaigns, sales outreach, or recruitment efforts.

What You'll Need

  1. Icypeas Account: Active subscription with API credentials (key, secret, user ID)
  2. Google Sheets Document: Formatted with columns for first name, last name, and company/domain
  3. n8n Instance: Self-hosted or cloud version of n8n
  4. Contact Data: Minimum dataset of contacts to verify (can start with just 10-20 for testing)
  5. Basic Understanding: Familiarity with spreadsheet management and API concepts

Pro tip: Start with a small test batch (10-20 contacts) to verify your setup before processing large lists. This helps you understand the verification results and adjust your data formatting if needed.

Quick Setup Guide

  1. Download and Import: Download the template file and import it into your n8n instance
  2. Configure Google Sheets: Connect your Google account and select the spreadsheet with contact data
  3. Set Up Icypeas: Enter your API credentials in the HTTP Request node settings
  4. Map Data Fields: Ensure your spreadsheet columns match the expected input format
  5. Test Execution: Run the workflow with a small batch to verify everything works
  6. Schedule Automation: Set up regular runs (daily/weekly) for ongoing list hygiene
