Personal Finance Assistant: Query Notion Data via Telegram & WhatsApp with Claude AI

Get instant answers about your finances from your Notion database using natural language queries through popular messaging apps.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves a common problem for individuals and professionals who track their finances in Notion but struggle to quickly access insights. Instead of manually searching through databases or creating complex views, you can now ask natural language questions through Telegram or WhatsApp and get instant, accurate answers.

The workflow acts as your personal AI-powered accountant. When you send a query like "What were my total expenses in March?" or "Show me my biggest spending categories this quarter," the system intelligently interprets your question, identifies the correct Notion database, applies appropriate filters, retrieves the data, and returns a clear summary directly to your phone.

This eliminates the friction between having financial data and actually using it to make decisions. Whether you're checking your budget while shopping, reviewing expenses during a meeting, or planning investments on the go, your financial insights are always just a message away.

How It Works

Step 1: Message Reception

The workflow listens for incoming messages on either Telegram (via a bot you create) or WhatsApp Business. When you send a financial query, the message is captured and prepared for processing.

Step 2: AI Interpretation

Claude AI (through OpenRouter) analyzes your natural language question to understand what financial data you're asking for. It identifies key elements like time periods, categories, amounts, and comparison requests.

Step 3: Notion Database Query

Based on the AI's understanding, the workflow constructs a precise query for your Notion finance database. It selects the correct database, applies filters for dates/categories, and retrieves only the relevant records.

Step 4: Data Processing & Summarization

The raw data from Notion is processed—totals are calculated, categories are grouped, trends are identified. The AI then creates a human-readable summary that directly answers your original question.

Step 5: Response Delivery

The final answer is sent back through the same messaging channel (Telegram or WhatsApp), providing you with immediate financial insight without ever leaving your favorite chat app.

What You'll Need

  1. Notion Account with a personal finance database (the template works with the popular "Personal Finance System" Notion template)
  2. Notion Integration Token created through Notion's developer settings
  3. Telegram Bot Token and Chat ID (created via BotFather) OR WhatsApp Business Account with API access
  4. OpenRouter API Key for accessing Claude AI capabilities
  5. n8n Instance (self-hosted or cloud version) to run the workflow
  6. Basic Technical Comfort with copying API keys and following setup instructions

Pro tip: Start with Telegram for testing—it's easier to set up than WhatsApp Business and provides the same core functionality. Once you've validated the workflow works with your data, you can add WhatsApp support.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Duplicate the Personal Finance System template in your Notion workspace if you haven't already
  3. Create a Notion integration and share your finance database with it
  4. Set up a Telegram bot via @BotFather and note your token and chat ID
  5. Get an OpenRouter API key from their website (free tier available)
  6. Configure all credentials in the n8n workflow nodes
  7. Test with simple queries like "What are my expenses this month?" before moving to complex questions
