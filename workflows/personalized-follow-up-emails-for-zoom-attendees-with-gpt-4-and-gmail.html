What This Workflow Does

After every Zoom meeting, manually sending follow-up emails to each participant is time-consuming and often generic. This workflow solves that problem by automatically generating and sending personalized, AI-crafted follow-up emails to every attendee immediately after the meeting ends.

It captures the meeting end event via Zoom webhook, extracts participant names and emails, analyzes discussion context (if transcript available), and uses GPT-4 to draft professional, context-aware emails that recap key points, assign action items, and thank participants specifically. These emails are then sent via Gmail, ensuring timely, valuable communication that strengthens relationships and clarifies outcomes.

How It Works

Step 1: Zoom Meeting Ends

A Zoom webhook triggers the workflow when a meeting concludes. The payload includes participant list, meeting details, and optionally a transcript.

Step 2: Data Normalization

The workflow processes the incoming data, extracting participant names, email addresses, and key discussion points from the transcript or meeting metadata.

Step 3: AI Email Drafting

GPT-4 analyzes the extracted context and generates personalized email drafts for each participant. These drafts include meeting recap, specific thank-you notes, and clear next steps.

Step 4: Automated Email Delivery

Gmail sends the finalized, personalized emails to each participant automatically. The workflow can include scheduling delays if immediate sending isn't desired.

What You'll Need

  1. A Zoom account with webhook capabilities enabled for the meeting.ended event.
  2. A Gmail account connected via OAuth for sending emails.
  3. An OpenAI API key for GPT-4 access (or alternative AI service).
  4. Basic n8n setup to import and run the workflow.
  5. Meeting participant email addresses available in Zoom payload (standard in most setups).

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n instance.
  2. Configure the Zoom webhook node: Enable meeting.ended event in your Zoom app settings and paste the workflow's webhook URL.
  3. Set up Gmail OAuth connection in the Gmail node within n8n.
  4. Add your OpenAI API key in the AI node configuration.
  5. Test the workflow with a completed Zoom meeting to verify email generation and sending.
  6. Activate the workflow for automatic execution after each meeting.

Pro tip: Customize the AI prompt in the workflow to match your company's tone, include specific branding elements, or emphasize certain meeting outcomes for more tailored results.
