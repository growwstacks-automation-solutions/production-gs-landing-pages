Personalized Outreach for Lawyers with LinkedIn Scraping & GPT-4o

Automate LinkedIn lead generation for legal professionals. Scrape profiles, enrich with AI, and send personalized outreach messages—all without manual work.

What This Workflow Does

Manual LinkedIn outreach for lawyers is time-consuming and ineffective. Searching for relevant profiles, reading their bios, and crafting personalized messages takes hours per week—and often results in generic, low-response outreach.

This automation solves that by combining Google Programmable Search, AI enrichment, and automated messaging. It finds solo lawyers and small firm founders on LinkedIn, extracts their profile details, uses GPT-4o to generate context-aware messages, and stores everything in Google Sheets for organized outreach. You can target up to 1,000 prospects daily without touching LinkedIn’s API or using browser automation.

The workflow is designed specifically for legal tech founders, marketing consultants serving law firms, and agencies doing outbound legal outreach. It replaces expensive LinkedIn Sales Navigator subscriptions and manual research with a scalable, AI-powered system.

How It Works

Step 1: Profile Discovery via Google Search

The workflow uses Google Programmable Search Engine (free API) to find LinkedIn profiles matching your criteria—like “solo lawyer” or “legal firm founder.” It returns name, title, profile URL, and a snippet for each result.

Step 2: Data Storage in Google Sheets

Each discovered profile is immediately written into a Google Sheets spreadsheet. This creates a clean, centralized database of raw leads before any AI processing begins.

Step 3: AI Enrichment with OpenRouter Sonar Pro

For each profile, the workflow uses OpenRouter’s Sonar Pro model to research external content about that lawyer—recent cases, publications, or news mentions. This adds depth beyond the LinkedIn snippet.

Step 4: Personalized Message Generation with GPT-4o

GPT-4o (via OpenAI or OpenRouter) crafts a one-line personalized outreach message referencing the lawyer’s specialization, recent work, or unique profile details. The tone is adjustable—formal, casual, or consultative.

Step 5: Final Output to Google Sheets

The generated message is appended to the same row in Google Sheets, creating a ready-to-use outreach list. You can then connect this sheet to your CRM or email tool for sending.

What You'll Need

  1. Google Programmable Search API – Enable the Custom Search API on Google Cloud, create a search engine, and obtain your API key and CX ID.
  2. Google Sheets – Create a sheet with columns: Name, Title, Profile URL, Outreach Message. Share it with your OAuth-connected account.
  3. OpenRouter Account – Sign up at openrouter.ai, fund with at least $5, and generate an API key. Use the model “perplexity/sonar-pro” for real-time research.
  4. GPT-4o Access – Either your OpenAI API key or route GPT-4o through OpenRouter.
  5. n8n Instance – A running n8n environment (cloud or self-hosted) to import and execute the workflow.

Quick Setup Guide

Estimated setup time: 15–20 minutes.

  1. Import the Template – Download the JSON file and import it into your n8n workspace.
  2. Configure Google Search – Replace the placeholder API key and CX ID in the “Find Lawyer Profiles” node with your credentials.
  3. Connect Google Sheets – Authenticate the Google Sheets node and point it to your pre‑created spreadsheet.
  4. Set Up OpenRouter – Add your OpenRouter API key in the “Enrich Profile with AI” node.
  5. Adjust the AI Prompt – Modify the GPT‑4o prompt in the “Generate Personalized Message” node to match your desired tone and messaging style.
  6. Test with a Small Batch – Run the workflow once with a limited search query to verify outputs look correct.
  7. Schedule Regular Execution – Add a Schedule Trigger node to run this daily or weekly for continuous lead generation.

Pro tip: Change the Google search query to target other professional niches—like “founder AND therapist site:linkedin.com/in”—to adapt this workflow for accountants, consultants, or any LinkedIn‑listed profession.
