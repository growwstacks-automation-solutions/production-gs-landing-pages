What This Workflow Does

This automation solves the problem of generic, repetitive music recommendations. Instead of suggesting songs based solely on listening history, it creates a truly personalized experience by considering your current context—where you are, how you feel, the weather outside, and even your astrological sign.

The workflow acts as a personal music curator that analyzes multiple data points to recommend Taiwan indie music you might not discover otherwise. It bridges the gap between emotional state, environmental conditions, and musical discovery, creating recommendations that feel thoughtfully crafted rather than algorithmically generated.

For businesses in music, lifestyle, or content curation, this represents a powerful tool to increase engagement by delivering uniquely relevant content that adapts to users' daily lives, not just their past behavior.

How It Works

The automation follows a logical sequence that transforms basic user inputs into deeply personalized music recommendations.

1. Input Collection

The workflow begins by gathering key information: your city (for local weather), current mood, birthday (for zodiac calculation), and musical preferences. This establishes the foundation for personalization.

2. Context Enrichment

It then fetches real-time weather data for your location and calculates your zodiac sign based on your birthday. These external data points add layers of context that most music services ignore.

3. AI Analysis & Recommendation

An AI model (like GPT-4) processes all collected data—weather, mood, zodiac, location—to recommend a Taiwan indie song that matches this unique combination. The AI explains why the song fits, highlighting thematic connections.

4. Music Retrieval & Delivery

The workflow searches Spotify for the recommended song, retrieves the track link, and compiles everything into a beautifully formatted output ready for display in your preferred platform.

What You'll Need

  1. n8n instance (cloud or self-hosted)
  2. OpenAI API key for AI analysis and recommendation generation
  3. Spotify Developer account with API credentials for music search
  4. Weather API access (like OpenWeatherMap) for local conditions
  5. Basic understanding of how to set API credentials in n8n nodes

Pro tip: Test with different mood and location combinations to see how dramatically recommendations change. A "melancholy" mood with "rainy" weather in Taipei will suggest completely different artists than "energetic" with "sunny" in Kaohsiung.

Quick Setup Guide

Import and configure this workflow in under 15 minutes to start delivering personalized music discoveries.

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure API credentials for OpenAI, Spotify, and your chosen weather service in their respective nodes.
  3. Customize the input node with default values for testing, or connect it to a form/webhook for live user input.
  4. Test the workflow manually with different cities, moods, and birthdays to verify recommendations feel personalized.
  5. Connect output nodes to your delivery channels—Discord, email, web app, or social media auto-poster.
