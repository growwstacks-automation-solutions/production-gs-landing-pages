Plant Care Reminder with OpenWeather, Sheets and Telegram

Automate watering schedules and plant maintenance tracking with weather-aware reminders

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the challenge of managing watering schedules and fertilization routines for plant collections. By connecting Google Sheets, Telegram, and OpenWeather API, it eliminates manual tracking and guesswork from plant care.

The system checks each plant's specific needs against weather conditions, sends personalized reminders via Telegram, and maintains a complete care log in Google Sheets. It's particularly valuable for plant nurseries, botanical gardens, or anyone managing multiple plants with different care requirements.

How It Works

1. Daily Schedule Trigger

The workflow activates automatically each day to check which plants need attention.

2. Vacation Mode Check

First verifies if vacation mode is active in the settings sheet before proceeding with reminders.

3. Plant Data Retrieval

Pulls all plant records from Google Sheets including last watered date, frequency, and location data.

4. Weather Integration

For outdoor plants, checks OpenWeather API to adjust watering recommendations based on recent and forecasted rain.

5. Telegram Notifications

Sends personalized care reminders via Telegram with one-tap completion confirmation.

6. Log Updates

When tasks are marked complete, automatically updates the Google Sheets log with timestamps.
