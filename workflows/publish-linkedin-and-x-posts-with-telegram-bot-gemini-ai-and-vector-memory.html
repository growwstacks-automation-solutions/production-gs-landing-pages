What This Workflow Does

Managing multiple social media accounts is a constant drain on time and creative energy. Ideas get lost in notes apps, drafting posts is repetitive, and maintaining a consistent posting schedule feels like a part-time job. This automation solves that by creating a seamless pipeline from idea to publication.

This workflow gives you a personal social media assistant via Telegram. You can send it text, voice messages, images, or documents (like PDFs or presentations). The bot uses Google's Gemini AI to understand your content, draft a professional post tailored for LinkedIn or X, and saves key context to a vector memory so it learns about you and your brand over time. Crucially, it always sends you a draft for approval before publishing, keeping you in control.

How It Works

The automation connects Telegram, AI, and your social platforms into a intelligent, self-improving system.

1. Content Submission via Telegram

You interact with a Telegram bot just like messaging a friend. Send a text idea, a voice note explaining a concept, an image, or a document. The bot receives this payload and prepares it for processing.

2. AI-Powered Understanding & Drafting

The workflow uses Google's Gemini AI to analyze your submission. It extracts key themes, checks against stored context in its vector memory (your bio, company info, past topics), and generates a platform-appropriate draft. The AI ensures the tone and format fit LinkedIn's professional style or X's concise nature.

3. Human-in-the-Loop Approval

The drafted post is sent back to you in Telegram for review. You can approve, request edits, or reject it. This step guarantees quality and prevents unwanted automated posts.

4. Automated Publishing

Upon your approval, the workflow uses the official LinkedIn and X APIs to publish the post directly to your accounts. It handles authentication, formatting, and any media attachments seamlessly.

5. Context Learning (Vector Memory)

Relevant information from your interactions is embedded and saved to a vector database. This means the bot gradually learns about your expertise, products, and communication style, making future drafts more accurate and personalized.

Pro tip: Use the vector memory to store your professional bio, company mission, and key product details. This teaches the AI your "voice," so every generated draft sounds authentically like you.

It's perfect if you frequently have ideas on the go, value consistency in your social presence, and want to leverage AI without losing the human touch. Agencies managing multiple client accounts can also adapt this system to streamline their content approval and publishing pipelines.

What You'll Need

  1. A self-hosted n8n instance (this template uses community nodes).
  2. Google AI Studio API Key for Gemini AI capabilities.
  3. Telegram Bot Token created via @BotFather on Telegram.
  4. LinkedIn API Credentials (requires a developer account and app setup).
  5. X (Twitter) API Credentials (from the X developer portal).

Quick Setup Guide

Follow these steps to get your automated social media assistant running.

  1. Import the Template: Download the JSON file and import it into your n8n instance.
  2. Configure Telegram: Create a bot via @BotFather, get the token, and add it to the "Telegram Trigger" and "Telegram" nodes.
  3. Set Up AI: Get a Google AI Studio API key and add it to the "Google Gemini Chat Model," "Describe Document," and "Describe Audio" nodes.
  4. Connect Social Accounts: Create apps in the LinkedIn and X developer portals. Add your OAuth credentials to the respective "Create Post" nodes.
  5. Test & Deploy: Activate the workflow, send a message to your Telegram bot, and follow the approval flow. Start with a test post to ensure everything works.
