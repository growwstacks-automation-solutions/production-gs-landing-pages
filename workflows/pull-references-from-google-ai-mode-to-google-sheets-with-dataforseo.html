What This Workflow Does

Google's AI Mode (Search Generative Experience) is changing how people find information. Instead of just listing websites, it provides summarized answers with cited sources. For SEO professionals and marketers, knowing which websites Google's AI considers authoritative for your target keywords is invaluable competitive intelligence.

This n8n workflow automates the collection of these source references. It queries the DataForSEO SERP API for specified keywords, extracts the domains, URLs, and titles that Google's AI cites, and logs them systematically into Google Sheets. This transforms what would be hours of manual research into a scheduled, automated process that delivers structured data ready for analysis.

Whether you're monitoring your own brand's visibility in AI answers, tracking which competitors are being cited, or identifying authoritative domains for link-building opportunities, this workflow provides the data foundation you need to adapt your SEO strategy for the age of generative search.

How It Works

1. Scheduled Trigger

The workflow begins with a Schedule Trigger node, which you can configure to run daily, weekly, or monthly based on your monitoring needs. This ensures your source tracking happens automatically without manual intervention.

2. DataForSEO API Query

For each target keyword, the workflow sends a request to the DataForSEO SERP API configured to return Google AI Mode results. The API provides structured data including the AI-generated answer and all source references cited within it.

3. Data Extraction & Cleaning

n8n processes the API response to extract source domains, URLs, titles, and snippet text. Function nodes clean and format this data, removing duplicates and ensuring consistent formatting for analysis.

4. Google Sheets Integration

The cleaned data is appended to a Google Sheet with columns for Source Domain, URL, Title, Snippet Text, Keyword, and Timestamp. This creates a historical record you can sort, filter, and analyze over time.

Pro tip: Add a "Sentiment Analysis" node between steps 3 and 4 to automatically categorize whether cited sources mention your brand/competitors positively, negatively, or neutrally.

What You'll Need

  1. An active n8n instance (cloud or self-hosted)
  2. A DataForSEO account with API credits (they offer a free tier for testing)
  3. A Google Sheet with columns: Source Domain, URL, Title, Snippet Text, Keyword, Date
  4. Google Sheets credentials configured in n8n
  5. Your target keywords list (can be stored in the workflow or a separate sheet)

Quick Setup Guide

Follow these steps to implement this automation in under 15 minutes:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure DataForSEO credentials in the HTTP Request node with your API login and key
  3. Update the keyword list in the Function node or connect to a Google Sheet containing your target keywords
  4. Connect your Google Sheet by setting up OAuth2 credentials and specifying your spreadsheet ID
  5. Test with one keyword to ensure data flows correctly from API to sheet
  6. Adjust the schedule based on how frequently you need updates (weekly is recommended for most use cases)
  7. Run the workflow and verify data appears in your Google Sheet

Pro tip: Start with 5-10 core keywords to validate the setup before scaling to hundreds. Monitor your DataForSEO API usage to avoid unexpected costs.
