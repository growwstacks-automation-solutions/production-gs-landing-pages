Automate Namecheap Dynamic DNS Updates

What This Workflow Does

If you host a website, email server, VPN, or any service from a location with a dynamic public IP address, you face a constant challenge: your internet service provider can change your IP at any time. When that happens, your domain name points to the wrong address, causing immediate downtime for your services.

This free n8n automation solves that problem permanently. It periodically checks your current public IP address, compares it with the last known IP, and if a change is detected, automatically updates your Namecheap Dynamic DNS records. This ensures your domain and subdomains always resolve to the correct, current IP address—without any manual intervention.

How It Works

The workflow follows a logical sequence to ensure reliable, accurate updates.

1. Schedule & IP Detection

A scheduler node triggers the workflow at regular intervals (e.g., every 10 minutes). The first step queries a public IP service to retrieve your current external IP address.

2. Change Detection

The workflow compares the newly detected IP with the IP stored from the previous run. If they match, the workflow stops—no unnecessary API calls are made.

3. DNS Record Update

If the IP has changed, the workflow uses the Namecheap API to update the A records for your specified domain and subdomains. It sends the new IP address securely to Namecheap's DDNS endpoint.

4. Confirmation & Logging

The workflow captures the API response to confirm success, logs the change (optionally to a file or database), and can send a notification via email, Slack, or another channel to inform you of the update.

Pro tip: Set the scheduler to run frequently (every 5-10 minutes) to minimize potential downtime. Most residential IPs don't change more than once a day, but frequent checks ensure near-instant updates when they do.

What You'll Need

  1. A running n8n instance (cloud, self-hosted, or desktop).
  2. A Namecheap domain with Dynamic DNS enabled in your account.
  3. Your Namecheap DDNS password (found in your Namecheap account under Advanced DNS).
  4. The subdomains you want to update (e.g., www, mail, vpn, home).
  5. Basic understanding of DNS A records (what they are and how they work).

Quick Setup Guide

Import and configure this template in under 10 minutes.

  1. Download the template using the button above and import it into your n8n workspace.
  2. Open the "yourdomain.com" node and replace the placeholder with your actual domain name.
  3. Enter your Namecheap DDNS password in the HTTP Request node's authentication section.
  4. Open the "subdomains" node and edit the list to match your specific subdomains (one per line).
  5. Adjust the schedule to your preferred frequency (default is every 10 minutes).
  6. Test the workflow by executing it once manually and checking your Namecheap DNS records.
  7. Activate the workflow and let it run automatically forever.
