What This Workflow Does

This automation solves the visibility gap that occurs when new lists are created in Affinity but team members don't immediately notice. In relationship-driven businesses like venture capital, private equity, and enterprise sales, new lists often represent emerging opportunities or strategic groupings that require prompt attention.

The workflow monitors Affinity for newly created lists and sends real-time notifications to designated team members or channels. Unlike manual checking, this ensures no important list creation goes unnoticed, whether it's a new investor syndicate, sales territory grouping, or partnership target list.

How It Works

1. List creation detection

The workflow polls Affinity's API at regular intervals (configurable from 5 minutes to 1 hour) to check for newly created lists. It compares against the last known state to identify only new additions.

2. Data enrichment

When a new list is detected, the workflow retrieves additional details like the creator's name, list description, initial member count, and any tags or custom fields applied to the list.

3. Notification delivery

The enriched list data is formatted into a clean notification and delivered to your preferred channel(s). The template includes options for email, Slack, Microsoft Teams, or webhook destinations.

4. Logging and error handling

All notifications are logged with timestamps for audit purposes. The workflow includes automatic retry logic for failed deliveries and alerts for persistent issues.

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance
  3. Configure the Affinity API connection with your credentials
  4. Set your notification preferences (channels, message format)
  5. Adjust the polling interval if needed (default is 15 minutes)
  6. Test by creating a new list in Affinity and verifying the notification
  7. Activate the workflow

Pro tip: Add list name filters to only receive notifications for lists containing keywords like "investor", "target", or "priority" to reduce noise.
