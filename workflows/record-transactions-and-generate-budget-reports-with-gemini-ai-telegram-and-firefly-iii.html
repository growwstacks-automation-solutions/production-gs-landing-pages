What This Workflow Does

Manual financial tracking is time-consuming and error-prone. This automation solves that by creating a seamless pipeline from transaction capture to ledger entry. Instead of manually entering receipts or bank statements, you simply send a photo to a Telegram bot. Gemini AI extracts the relevant details, categorizes the transaction based on your preferences, and posts it directly to your Firefly III instance.

The system also handles budget reporting on demand. When you send the word "Report" via Telegram, it queries Firefly III for all budgets from the beginning of the month, calculates remaining amounts, and sends you a clean CSV file. This transforms financial management from a monthly chore into a real-time, interactive process that takes seconds instead of hours.

How It Works

1. Transaction Capture via Telegram

You send a transaction screenshot, receipt photo, or bank statement image to your dedicated Telegram bot. The bot accepts the image and passes it to the next stage. This creates a frictionless entry point—no special apps, no manual typing, just a photo from your camera roll.

2. AI-Powered Data Extraction

Gemini AI analyzes the image based on your configured preferences: asset account IDs, expense categories, and description formats. It extracts merchant names, amounts, dates, and categories with high accuracy, transforming visual data into structured transaction information ready for your ledger.

3. Automated Ledger Posting

The structured transaction data is formatted into a proper API request and posted to your Firefly III instance using OAuth2 authentication. The system handles all the API complexities, ensuring transactions appear correctly categorized in your financial dashboard without manual intervention.

4. On-Demand Budget Reporting

When you request a report, the workflow queries Firefly III for all budgets between the first of the month and the current date. It calculates remaining amounts, formats the data into a clean CSV, and sends it back through Telegram. You get instant visibility into your financial standing without logging into multiple systems.

Pro tip: Start with your most frequent transaction types (groceries, dining, subscriptions) when configuring categories. The AI will learn from your patterns and become more accurate over time, reducing manual corrections.

Quick Setup Guide

1. Create your Telegram bot by messaging @BotFather and save the API token provided.

2. Set up OAuth in Firefly III by creating a new client in the administration panel, using n8n's redirect URL from the OAuth2 credential creator.

3. Configure credentials in n8n: Add your Telegram bot token, Firefly OAuth details, and Gemini API key to their respective credential sections.

4. Import the workflow template you downloaded from this page into your n8n instance.

5. Update node configurations: Set the correct Firefly base URL in each HTTP node and customize the AI system message with your preferred categories and account IDs.

6. Test with a sample receipt by sending an image to your bot and verifying it appears correctly in Firefly III.
