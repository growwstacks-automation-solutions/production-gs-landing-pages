What This Workflow Does

This automation solves the critical problem of undetected workflow failures in n8n. When running production automations, errors can occur at any time - API changes, authentication issues, or data validation problems. Without real-time alerts, these failures might go unnoticed for hours, potentially impacting business operations and customer experiences.

The template monitors your n8n workflows and immediately sends detailed error reports to a Telegram channel when failures occur. Each alert includes the workflow name, error message, timestamp, and execution context - giving your team everything needed to diagnose and resolve issues quickly.

Sample Telegram notification showing workflow error details

How It Works

1. Error Detection

The workflow triggers whenever an n8n execution fails. It captures the complete error context including the specific node that failed, error codes, and any available stack traces.

2. Alert Formatting

Error details are formatted into a clear Telegram message with emoji indicators for severity levels. The template includes clickable links back to the n8n editor for quick access to the failing workflow.

3. Notification Delivery

Formatted alerts are sent to your designated Telegram channel where team members receive push notifications. The system supports both private chats and group channels for collaborative troubleshooting.

Pro tip: Create a dedicated Telegram channel for automation alerts and pin important messages about ongoing incidents.
