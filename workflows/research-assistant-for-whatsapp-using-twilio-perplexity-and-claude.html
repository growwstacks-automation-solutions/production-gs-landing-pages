What This Workflow Does

This automation transforms WhatsApp into a powerful research assistant that combines real-time web research (via Perplexity) with advanced AI analysis (via Claude). Users simply send a research question to your WhatsApp number, and receive back a comprehensive report with key findings, sources, and synthesized insights - all formatted for easy reading on mobile.

The system eliminates hours of manual research by automatically gathering information from multiple authoritative sources, cross-referencing facts, and presenting the most relevant data in a structured format. Unlike basic chatbots, it provides properly cited answers with verifiable sources, making the information trustworthy for business decisions.

How It Works

1. WhatsApp Request

Users message your Twilio-connected WhatsApp number with their research question. The message triggers the workflow and passes the query to the next steps.

2. Perplexity Research

The system sends the question to Perplexity AI, which performs real-time web searches across academic, news, and industry sources. Perplexity returns a preliminary answer with citations and key points.

3. Claude Analysis

The research findings are processed by Claude AI, which synthesizes the information into a coherent report. Claude adds executive summaries, identifies trends, and formats the response for clarity.

4. WhatsApp Delivery

Twilio sends the final research report back to the user via WhatsApp, including source citations and follow-up questions. The message can include formatted text, bullet points, and links.

Pro tip: Configure Claude to structure responses with "Key Findings", "Supporting Data", and "Recommended Actions" sections for maximum business impact.
