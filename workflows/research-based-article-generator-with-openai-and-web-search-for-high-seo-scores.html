What This Workflow Does

Creating high-quality, SEO-optimized long-form content is time-consuming. Manual research, sourcing recent data, structuring outlines, and writing drafts can take days per article. This workflow automates the entire process, turning a simple topic and keyword input into a fully researched, outlined, and drafted article ready for human editing.

It solves the problem of content scalability for marketing teams, SEO agencies, and founders who need to produce authoritative, data-backed content consistently without sacrificing quality or burning out their writers.

How It Works

The automation follows a logical, multi-stage process that mimics how a skilled content researcher and writer would work.

1. Input & Keyword Refinement

You provide a domain, target keywords, and desired language via a simple form. The workflow first refines these keywords using AI to ensure they are relevant and search-optimized.

2. Research & Citation Gathering

The system performs automated web searches to find recent articles and authoritative sources on the topic. It identifies key statistics, quotes, and references that will add credibility to the final piece.

3. Master Outline Creation

Based on the research, an AI model synthesizes a comprehensive article outline, breaking down the topic into logical sections, subheadings, and key points to cover.

4. Section-by-Section Drafting

The workflow loops through each outline section. For each, it generates specific search queries, fetches fresh web results, summarizes findings, and then writes the section using an advanced AI model, incorporating the research directly.

5. Final Assembly & Delivery

All drafted sections are aggregated into a clean, formatted Markdown document. Optional delivery nodes can send the draft via email or Telegram, or save it directly to your CMS.

What You'll Need

  1. An n8n instance (cloud or self-hosted) with outbound internet access.
  2. An OpenAI API key with access to the Responses API and web_search_preview capability.
  3. (Optional) Credentials for delivery channels like Email or Telegram if you want to receive drafts automatically.
  4. Basic familiarity with n8n to import the JSON and configure API credentials.

Pro tip: Start by using the workflow to generate drafts for your existing content ideas. Use the human-in-the-loop review step (which you can enable) to maintain quality control and brand voice before any automated publishing.

Quick Setup Guide

  1. Download & Import: Click the download button above to get the JSON file. In your n8n instance, go to Workflows → Import from file and select the downloaded template.
  2. Add Credentials: In the n8n credentials panel, add your OpenAI API key. The workflow nodes are pre-configured to use this credential.
  3. Configure Models: Locate the "LLM Params" node in the workflow. Set your preferred simple_model (for research tasks) and advanced_model (for writing tasks) according to your OpenAI plan and needs.
  4. Test Run: Trigger the workflow manually with a test domain and keyword. Check the execution for any errors and verify the output format.
  5. Customize & Deploy: Adjust prompts in the "Section Prompts" node to match your brand tone. Set up a schedule trigger or webhook to run the automation based on your content calendar.
