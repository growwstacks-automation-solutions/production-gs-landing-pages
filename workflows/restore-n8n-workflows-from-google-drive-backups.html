What This Workflow Does

This automation solves the time-consuming challenge of manually restoring multiple n8n workflows from backup files stored in Google Drive. When migrating your n8n instance to a new server or recovering from system failures, manually importing dozens of individual workflow files can take hours of repetitive work.

The template automatically retrieves your workflow backup files from Google Drive and imports them into your n8n instance in the correct order. It handles authentication, file retrieval, and workflow restoration as a single automated process, reducing what could be a half-day manual task to just minutes of automated execution.

How It Works

1. Connect to Google Drive

The workflow authenticates with your Google Drive account using secure OAuth credentials. It accesses the specific folder where your n8n workflow backups are stored.

2. Retrieve Backup Files

It scans the designated backup folder and retrieves all JSON workflow files, sorting them by modification date to ensure proper restoration order.

3. Prepare n8n Instance

The workflow verifies your target n8n instance is accessible and prepares it for the import process, checking available resources and permissions.

4. Import Workflows

Each workflow file is imported sequentially into your n8n instance, with progress tracking and error handling for each individual workflow restoration.

What You'll Need

  1. An active n8n instance (cloud or self-hosted)
  2. Google Drive account with workflow backups
  3. OAuth credentials for Google Drive API
  4. Admin access to your n8n instance
  5. Backup files in standard n8n JSON format

Quick Setup Guide

  1. Download and import the template JSON into your n8n instance
  2. Configure Google Drive OAuth credentials in the first node
  3. Set the path to your workflow backup folder in Google Drive
  4. Enter your target n8n instance URL and credentials
  5. Test with a single workflow file before full restoration
  6. Schedule regular backups to maintain restore points

Pro tip: Create a dedicated Google Drive folder just for n8n backups and set up automated weekly backups to ensure you always have recent restore points available.
