What This Workflow Does

This automation solves the constant struggle of maintaining an active, professional LinkedIn presence. Instead of manually writing posts or repurposing content, it automatically monitors your blog (or any website), detects new articles, uses AI to create LinkedIn-optimized summaries, sends them for human approval, and publishes them directly to your LinkedIn account.

The system keeps a "human in the loop" using gotoHuman, where team members can edit AI drafts, request regenerations, or adjust prompts before anything goes live. This ensures brand consistency while eliminating 90% of the manual work involved in social media content creation.

Businesses using this workflow typically save 5-10 hours per week on content creation while doubling their LinkedIn posting frequency and improving engagement through consistent, timely content distribution.

How It Works

1. Content Discovery & Scraping

The workflow starts by checking your target blog URL daily (configurable). It fetches the latest articles, filters for new publications from the last 24 hours, and extracts the full content including titles, bodies, and publication dates.

2. AI Processing & Summarization

Each article is sent to GPT-4o (or your preferred LLM) to create a concise, engaging summary. The AI is instructed to write in your brand voice, highlight key takeaways, and format the content for LinkedIn's professional audience.

3. Human Review & Approval

The AI-generated draft is sent to gotoHuman, where designated team members receive a notification. They can edit the post directly, ask the AI to regenerate with different instructions, or approve it as-is. This step ensures quality control and brand alignment.

4. Automated Publishing

Once approved, the post is automatically published to your LinkedIn company page or personal profile. The workflow handles authentication, formatting, scheduling, and can even add relevant hashtags or track performance metrics.

What You'll Need

  1. n8n instance (cloud or self-hosted)
  2. OpenAI API key for GPT-4o access
  3. LinkedIn Developer Account with posting permissions
  4. gotoHuman account for human review workflow
  5. Blog/website URL to monitor for new content

Pro tip: Before going live, test with a private LinkedIn account or use LinkedIn's "draft" feature to review posts before they become visible to your audience.

Quick Setup Guide

  1. Install the gotoHuman node in your n8n instance before importing this template.
  2. Import the workflow JSON using n8n's import functionality.
  3. Configure credentials for OpenAI, LinkedIn, and gotoHuman in n8n's credentials management.
  4. Set up your gotoHuman template using ID "sMxevC9tSAgdfWsr6XIW" or create a custom review template.
  5. Update the blog URL in the HTTP Request node to monitor your specific website.
  6. Test with one article before enabling the daily schedule trigger.
