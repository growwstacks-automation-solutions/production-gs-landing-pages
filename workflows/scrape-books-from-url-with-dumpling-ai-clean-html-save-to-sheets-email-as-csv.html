What This Workflow Does

This automation solves the tedious, manual process of copying product information from websites. It's specifically built for book listings but can be adapted for any catalog-style webpage. The workflow watches a Google Sheet for new URLs, uses Dumpling AI to scrape the HTML content, extracts book titles and prices using CSS selectors, sorts the data by price, converts it to a clean CSV file, and emails that file to you—all without any manual intervention.

For virtual assistants, researchers, and data analysts, this represents a massive time savings. Instead of spending hours copying and pasting data, you get structured, sorted information delivered directly to your inbox. The automation handles the entire pipeline from data collection to delivery, ensuring consistency and eliminating human error in data entry.

The system is particularly valuable for competitive analysis, price monitoring, market research, or building product databases. Whether you're tracking book prices across retailers, compiling research materials, or monitoring inventory changes, this workflow transforms what was previously a manual chore into a fully automated process.

How It Works

The automation follows a logical sequence that mimics how a human would collect and process this data, but with perfect consistency and at scale.

1. Trigger from Google Sheets

The workflow starts when a new URL is added to a designated Google Sheet. This could be a book listing page from any retailer or catalog website. The Google Sheets trigger node monitors a specific column and passes new URLs to the next step.

2. AI-Powered Web Scraping

Each URL is sent to Dumpling AI via an HTTP request. Dumpling AI handles the complexities of web scraping—rendering JavaScript, navigating dynamic content, and returning clean HTML. This is far more reliable than traditional scraping methods that break when websites change their layout.

3. Data Extraction with CSS Selectors

The cleaned HTML is processed through n8n's HTML node using CSS selectors. First, it extracts individual book containers (using selectors like ".row > li"), then parses each container for specific fields: book titles (from "h3 > a" elements) and prices (from ".price_color" elements).

4. Sorting and Formatting

Extracted data is sorted in descending order by price, ensuring the most expensive items appear first. The Sort node handles this automatically, though you can modify the sorting logic to prioritize other criteria like alphabetical order or publication date.

5. CSV Conversion and Delivery

The sorted data is converted to CSV format using n8n's Convert node, creating a clean, spreadsheet-ready file. Finally, the Gmail node attaches this CSV and sends it to your specified email address, completing the automated pipeline from URL to delivered data.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. Google Sheets account with a spreadsheet containing URLs to scrape
  3. Dumpling AI API key (sign up at app.dumplingai.com for scraping credits)
  4. Gmail account or SMTP credentials for sending emails
  5. Basic understanding of CSS selectors to adapt to different website structures
  6. Target websites that allow automated access (check robots.txt and terms of service)

Pro tip: Start with a test URL to verify your CSS selectors work correctly before automating at scale. Use browser developer tools (F12) to inspect elements and find the right selectors for your target website.

Quick Setup Guide

Follow these steps to implement this automation in your n8n environment:

  1. Import the template by downloading the JSON file and importing it into your n8n instance
  2. Configure Google Sheets by connecting your account and specifying the spreadsheet and column containing URLs
  3. Set up Dumpling AI by creating an account, generating an API key, and adding it to the HTTP Request node
  4. Adjust CSS selectors in the HTML nodes to match your target website's structure
  5. Connect Gmail by adding your email credentials and specifying the recipient address
  6. Test with one URL to ensure the entire pipeline works before automating at scale
  7. Activate the workflow and add URLs to your Google Sheet to trigger automatic processing

Always check a website's robots.txt file and terms of service before scraping.
