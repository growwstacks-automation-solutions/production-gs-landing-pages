What This Workflow Does

Manual TikTok influencer research is time-consuming and inconsistent. Marketing teams spend hours scrolling through profiles, manually recording follower counts, engagement rates, and contact information—only to have that data become outdated within weeks. This n8n automation solves that problem by systematically scraping TikTok influencer profiles using Bright Data's reliable API and organizing everything into Google Sheets.

The workflow accepts TikTok profile URLs through a simple web form, sends them to Bright Data for processing, monitors the scraping status, retrieves comprehensive profile data, and automatically populates a structured spreadsheet. What used to take hours of manual work now happens automatically in minutes, giving marketing teams accurate, up-to-date data for campaign planning and outreach.

Beyond basic scraping, this template includes intelligent error handling, retry logic for failed requests, and status monitoring—ensuring complete data collection even when dealing with thousands of profiles. The result is a scalable system for influencer discovery, competitive analysis, and market intelligence that grows with your business needs.

How It Works

1. Profile Submission & API Trigger

Users submit TikTok profile URLs through a customizable web form. The workflow captures these URLs and sends them to Bright Data's TikTok dataset API with your specified parameters—including country targeting, data fields to extract, and processing preferences.

2. Intelligent Status Polling

Since TikTok data scraping isn't instantaneous, the workflow implements smart polling. It checks the Bright Data API every 30 seconds to monitor scraping progress, automatically retrying if data isn't ready yet. This ensures reliable completion without manual intervention.

3. Data Extraction & Validation

Once Bright Data returns the scraped information, the workflow extracts and validates key metrics: follower counts, engagement rates, verification status, biography content, profile links, and performance data. It cleans and formats this data for optimal spreadsheet organization.

4. Automated Google Sheets Population

The validated data automatically populates your Google Sheets spreadsheet with structured columns for each metric. The template includes timestamp tracking, status indicators, and error messages—creating a complete audit trail of your influencer research activities.

Quick Setup Guide

  1. Import the workflow: Copy the JSON template and import it into your n8n instance under Workflows → Add workflow → Import from JSON.
  2. Configure Bright Data: Create HTTP Request credentials in n8n with your Bright Data API key as a Bearer Token. Verify access to the TikTok dataset (gd_l1villgoiiidt09ci).
  3. Prepare Google Sheets: Create a new spreadsheet named "TikTok Influencer Data" with a sheet tab called "TikTok profile by url." Set up Google Sheets OAuth2 credentials in n8n.
  4. Update workflow settings: Replace the placeholder Google Sheet ID with your actual document ID. Update the Bright Data API key in the HTTP Request node.
  5. Test with sample profiles: Use the generated webhook URL to submit 2-3 TikTok profile URLs. Monitor execution and verify data appears correctly in your spreadsheet.
  6. Activate for production: Once testing succeeds, share the form URL with your team and begin scaling your influencer research.
