What This Workflow Does

Every professional wastes precious time digging through Outlook trying to find specific emails. Traditional search requires remembering exact keywords, dates, or senders—and often returns irrelevant results or misses what you need entirely.

This workflow solves that problem by bringing AI-powered natural language search to your Outlook mailbox. Instead of constructing complex search queries, you simply ask in plain English: "Find the invoice from Acme Corp last week" or "Show me feedback emails about the new product launch." The workflow uses GPT-4o to understand your intent, translates it into effective search parameters, queries your Outlook account, and returns ranked results with relevance scores and direct links to each email.

Beyond basic search, it intelligently ranks results by relevance, handles temporal context ("last month," "Q2"), understands synonyms and related concepts, and can be customized to prioritize certain senders or topics. It transforms email from a storage system into an instantly searchable knowledge base.

How It Works

Step 1: Natural Language Processing

You input your search query in everyday language. The workflow sends this to GPT-4o, which analyzes the query to extract key components: subject matter, timeframes, senders/recipients, urgency indicators, and document types. The AI then constructs an optimized search query that Outlook's API can understand.

Step 2: Outlook API Integration

The translated query connects to Microsoft Outlook via secure OAuth authentication. The workflow searches across your specified folders (default is entire mailbox) with appropriate filters for date ranges, sender domains, and content keywords. It retrieves a configurable number of results (default 5, adjustable up to 50).

Step 3: Relevance Scoring & Ranking

Each retrieved email is analyzed against your original query using semantic similarity scoring. The workflow evaluates how closely each email matches the intent of your search, not just keyword matches. Results are then sorted by relevance score, with the most likely matches appearing first.

Step 4: Formatted Output

The final output presents a clean table showing sender, subject, date, relevance score (0-100%), and a direct link to open each email in Outlook. This structured format lets you quickly identify the exact email you need without opening multiple messages.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance.
  2. Configure OpenAI credentials in n8n with your API key.
  3. Set up Microsoft Outlook OAuth2 connection—n8n will guide you through Microsoft's authentication flow.
  4. Adjust search parameters if desired (result limit, folder scope, date ranges).
  5. Test with a simple query like "emails from yesterday" to verify everything works.
  6. Deploy as a workflow or trigger manually when needed.
