What This Workflow Does

This automation solves the time-consuming problem of manually tracking economic calendar events that impact forex markets. It automatically imports Forex Factory calendar events into Google Calendar while applying your customized filters to only include relevant market-moving announcements.

Traders typically spend hours each week copying event details, setting reminders, and cleaning up outdated entries. This workflow eliminates that manual work while ensuring your trading calendar stays focused on the events that matter to your strategy. The automated cleanup feature removes past events and prevents duplicate entries that can clutter your calendar.

How It Works

1. Event Retrieval

The workflow first fetches the latest events from Forex Factory's economic calendar API. It pulls all available event data including timestamps, currency impact, and expected values.

2. Filter Application

Your configured filters are applied to select only relevant events. This can include impact level (high/medium), specific currency pairs, event types (central bank decisions, employment data, etc.), and time windows.

3. Google Calendar Integration

Filtered events are formatted and added to your specified Google Calendar. Each event includes key details like previous/expected values, historical context, and trading implications.

4. Automated Cleanup

The workflow periodically scans your calendar to remove past events and prevent duplicates. Optional archiving preserves historical event data for later analysis.

Pro tip: Set up separate calendars for different currency pairs or impact levels to quickly visualize market-moving events at a glance.
