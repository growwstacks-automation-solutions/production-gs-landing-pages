Get instant team alerts when orders come in - no manual checking required

What This Workflow Does

This automation instantly notifies your Mattermost team channel whenever a new order is placed in WooCommerce. It eliminates the need for manual order monitoring and ensures your fulfillment, customer service, and management teams see orders the moment they come in.

The workflow captures all essential order details - customer information, products ordered, payment method, and shipping details - and formats them into a clean Mattermost message. You can customize which fields to include based on your team's needs.

How It Works

1. WooCommerce order trigger

The workflow monitors WooCommerce for new orders using the WooCommerce API. When an order is created, it immediately pulls all relevant order data.

2. Data processing

Order information is parsed and formatted for readability. The workflow can filter orders based on your criteria (like order value or product type) before sending notifications.

3. Mattermost notification

A formatted message containing the order details is sent to your specified Mattermost channel. The message can include @mentions for specific team members when certain conditions are met.

What You'll Need

  1. A running n8n instance (cloud or self-hosted)
  2. WooCommerce store with API access
  3. Mattermost instance with webhook permissions
  4. Basic understanding of n8n workflows

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure WooCommerce API credentials in the trigger node
  3. Set up your Mattermost webhook URL in the message node
  4. Customize the message format if needed
  5. Activate the workflow

Pro tip: Add conditional logic to route high-value orders to a dedicated VIP channel or @mention specific team members for rush orders.
