Automate Daily Currency Exchange Rate Updates

Fetch live rates from CurrencyFreaks API and email them automatically every morning. Stay informed without manual work.

What This Workflow Does

For businesses and individuals dealing with international finances, staying updated on currency exchange rates is crucial but time-consuming. Manually checking rates every morning is inefficient and prone to being forgotten. This automation solves that by delivering fresh exchange rates directly to your inbox automatically.

The workflow connects to the CurrencyFreaks API to fetch real-time exchange rates for your preferred currencies (like USD, EUR, GBP, PKR, INR, BDT), formats them into a clear, readable email, and sends it via Gmail on a schedule you set. It turns a daily chore into a zero-effort background process, ensuring you always have the latest financial data when you start your day.

How It Works

1. Scheduled Trigger

The workflow begins with a Schedule Trigger node configured to run daily at your specified time (e.g., 8 AM). This acts as the automation's clock, initiating the process without any manual intervention.

2. Configure API & Currency Preferences

Set Variables nodes store your CurrencyFreaks API key and the specific currency pairs you want to track. This keeps sensitive credentials secure and allows easy customization without editing the workflow logic.

3. Fetch Live Exchange Rates

An HTTP Request node calls the CurrencyFreaks API endpoint, passing your API key and preferred currencies. It retrieves the latest exchange rate data, including timestamps and conversion values.

4. Format & Prepare Email

The raw API data is processed and formatted into a clean HTML email template. Set nodes define the recipient email address and a clear subject line (e.g., "Daily Exchange Rates - [Date]").

5. Send via Gmail

A Gmail node using OAuth2 authentication sends the formatted email. The message includes the date, base currency, and a table of exchange rates for easy reference.

Pro tip: Extend this workflow by adding a "Code" node to calculate percentage changes from the previous day's rates, giving you instant insight into market movements.

This automation is ideal for freelancers and contractors billing international clients, e-commerce businesses with global customers, import/export companies managing procurement costs, accounting teams needing daily rates for bookkeeping, and investors tracking forex markets. What You'll Need

  1. A CurrencyFreaks API key (free tier available)
  2. A Gmail account with OAuth2 access configured in n8n
  3. n8n instance (cloud or self-hosted)
  4. List of currency codes you want to track (e.g., USD, EUR, GBP)
  5. Email address(es) to receive the daily updates

Quick Setup Guide

Follow these steps to implement this automation in under 10 minutes:

  1. Download the template using the button above and import it into your n8n workspace.
  2. Obtain your API key from CurrencyFreaks (sign up for a free account).
  3. Configure the "Set Variables" nodes with your API key and preferred currency codes.
  4. Set up Gmail OAuth2 credentials in n8n if you haven't already.
  5. Update the recipient email and customize the email subject if desired.
  6. Adjust the schedule trigger to your preferred daily time.
  7. Test the workflow with the "Execute Workflow" button to ensure everything works.
  8. Activate the workflow to run automatically on your schedule.
