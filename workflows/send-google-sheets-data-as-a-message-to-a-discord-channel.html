Automate Google Sheets to Discord Notifications

Send instant messages to Discord channels when new rows are added or updated in Google Sheets. Keep your team synchronized with real-time data alerts.

What This Workflow Does

This automation solves the common problem of teams missing important updates in shared Google Sheets. When new data is added—like a new customer lead, project task update, or inventory change—someone has to manually check the sheet and notify others. This workflow eliminates that manual step by automatically sending a formatted message to a designated Discord channel whenever a row is added or modified.

The workflow monitors your Google Sheet continuously, detects changes, extracts the relevant data, formats it into a clear Discord message, and posts it instantly. This ensures everyone in your Discord team gets notified simultaneously, reducing oversight errors and speeding up response times.

How It Works

Step 1: Monitor Google Sheets Changes

The workflow starts with a Google Sheets node configured to watch a specific spreadsheet and range. It triggers whenever a new row appears or an existing row is edited, capturing the updated data.

Step 2: Process and Format Data

Using a code node, the raw sheet data is transformed into a readable format suitable for Discord. You can customize this step to include column headers, highlight specific values, or add contextual notes.

Step 3: Send to Discord Channel

The formatted message is passed to a Discord node, which posts it to your chosen channel via a webhook. The message appears as a regular Discord notification with all the relevant details.

Pro tip: Configure the workflow to run on a schedule (e.g., every 5 minutes) for near-real-time alerts, or trigger it immediately upon change for instant notifications.

What You'll Need

  1. A Google account with access to the Sheets you want to monitor
  2. A Discord account and a channel where you want notifications posted
  3. A Discord webhook URL for that channel (easy to set up in Discord settings)
  4. n8n instance (cloud or self-hosted) to run the workflow
  5. Basic understanding of your Google Sheets structure (columns, sheet name)

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n workspace.
  2. Configure the Google Sheets node with your spreadsheet ID and range.
  3. Set up Google credentials in n8n (OAuth with Sheets API scope).
  4. Configure the Discord node with your webhook URL and channel details.
  5. Test the workflow by adding a new row to your sheet—check Discord for the message.
  6. Activate the workflow to run automatically on changes or schedule.
