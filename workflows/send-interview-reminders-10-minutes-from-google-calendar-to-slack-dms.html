Automatically ping interviewers via Slack DMs 10 minutes before scheduled interviews

What This Workflow Does

This automation solves the common problem of interviewers forgetting or joining late to scheduled interviews. It monitors your Google Calendar for upcoming interviews and automatically sends a Slack direct message to each interviewer 10 minutes before the session starts.

The system eliminates manual reminder tasks for HR teams while ensuring better interview punctuality. Companies using this workflow report 30% fewer late arrivals and more professional candidate experiences. The reminders include key details like candidate name, position, and meeting link pulled directly from calendar events.

How It Works

1. Calendar Monitoring

The workflow checks your designated Google Calendar every 5 minutes for upcoming interviews. It scans event titles, descriptions, and participant lists to identify valid interview sessions.

2. Time-Based Trigger

When an interview is detected within the 10-minute reminder window, the system activates. It accounts for time zones and only sends reminders for future events, never past ones.

3. Participant Identification

The workflow extracts all participant emails from the calendar event, excluding the candidate (identified by "@" in email). This ensures only interviewers receive reminders.

4. Slack DM Delivery

For each identified interviewer, the system sends a personalized Slack direct message with interview details. The message includes a "Join Now" button linking directly to the meeting.

Pro tip: Create a dedicated "Interviews" calendar in Google Calendar to keep scheduling separate from other meetings.
