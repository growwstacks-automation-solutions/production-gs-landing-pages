What This Workflow Does

This automation solves the problem of constantly checking multiple platforms for booking updates. Professionals using Cal.com or Calendly often miss important appointment changes buried in email notifications or have to manually check their calendars throughout the day.

The workflow monitors both scheduling platforms in real-time, then sends beautifully formatted WhatsApp messages for every booking activity. It transforms raw booking data into mobile-friendly notifications with all relevant details clearly presented.

How It Works

1. Booking Detection

The workflow continuously monitors both Cal.com and Calendly for new, changed, or canceled appointments through their API connections.

2. Data Aggregation

Booking details from both platforms are standardized into a common format, ensuring consistent notifications regardless of which service was used.

3. AI Message Formatting

Google Gemini transforms the raw booking data into human-readable WhatsApp messages with proper spacing, emojis, and emphasis on key details.

4. Instant Notification

The formatted message is immediately sent via WhatsApp Business API to your specified number, typically arriving within seconds of the booking activity.

Pro tip: For teams, you can modify the workflow to notify multiple WhatsApp numbers when bookings involve specific team members or meeting types.

What You'll Need

  1. Active Cal.com and/or Calendly account with API access
  2. WhatsApp Business API credentials
  3. Google API key for Gemini
  4. Webhook endpoints configured for both calendar services
  5. Basic understanding of n8n workflow configuration

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Connect your Cal.com and/or Calendly accounts in the trigger nodes
  3. Configure the WhatsApp node with your Business API credentials
  4. Add your Google Gemini API key to the AI processing node
  5. Test with sample bookings to verify notification formatting
  6. Activate the workflow for live monitoring
