What This Workflow Does

This intelligent automation solves the time-consuming problem of manually sorting and responding to incoming emails. Instead of spending hours each day reading and categorizing messages, this workflow uses AI to instantly analyze email content, determine the type of inquiry, and send appropriate, personalized responses automatically.

The system can distinguish between different request types—such as guest post submissions, YouTube review requests, or course inquiries—and route each to the correct response template. It not only sends replies but also updates your CRM, applies Gmail labels, and marks emails as processed, creating a complete hands-off email management system.

How It Works

1. Email Trigger & Filtering

The workflow monitors your Gmail inbox hourly for new messages. It filters out internal emails and already-replied threads, ensuring only genuine new inquiries are processed.

2. AI-Powered Classification

Using LangChain's Text Classifier (enhanced optionally with Google Gemini), the system analyzes email content and categorizes it into predefined types like "Guest Post," "YouTube Review," or "Udemy Course Inquiry."

3. Automated Response Generation

Based on the classification, the workflow selects and sends a professionally crafted HTML email response tailored to that specific inquiry type, maintaining your brand voice and providing relevant information.

4. CRM & Follow-up Actions

The sender is automatically added to your Brevo contact list for future marketing, the email is labeled in Gmail for organization, and the message is marked as read to prevent duplicate processing.

Who This Is For

This automation is ideal for agencies, consultants, SaaS companies, content creators, and educational platforms that receive regular inbound inquiries. If you find yourself sending similar responses to common questions about your services, pricing, partnerships, or content submissions, this workflow will save you significant time. What You'll Need

  1. A Gmail account with OAuth2 access configured
  2. LangChain Text Classifier node credentials
  3. Google Gemini API account (optional for enhanced classification)
  4. SMTP credentials (from your email service provider)
  5. Brevo (formerly Sendinblue) account and API key
  6. An n8n instance (cloud or self-hosted)

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure Gmail OAuth2 credentials in the trigger node
  3. Set up your LangChain and Google Gemini API connections
  4. Add your SMTP server details for sending automated responses
  5. Connect your Brevo account for CRM integration
  6. Test with sample emails and adjust response templates as needed
  7. Activate the workflow and let it handle incoming emails automatically

Pro tip: Start by testing the workflow with a separate Gmail label or folder before applying it to your main inbox. This allows you to verify classification accuracy and response quality without affecting real customer communications.
