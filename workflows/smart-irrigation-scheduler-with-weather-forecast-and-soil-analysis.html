Automate garden and farm watering using real-time weather data and soil conditions to optimize water usage and plant health.

What This Workflow Does

Manual irrigation scheduling wastes water and often harms plants through overwatering or underwatering. This smart automation solves that by creating intelligent watering decisions based on actual environmental conditions rather than fixed timers.

The workflow analyzes weather forecasts, calculates evapotranspiration rates, considers soil types and plant requirements, then determines exactly when and how much to water. It automatically skips watering when rain is expected, adjusts for temperature changes, and optimizes for different zones with varying needs—from delicate flowers to drought-resistant shrubs.

For businesses managing landscapes, farms, or commercial greenhouses, this represents significant water savings, reduced labor costs, and healthier plants that thrive under optimized conditions.

How It Works

1. Data Collection & Analysis

The system first gathers current weather conditions and 5-day forecasts from weather APIs. It simultaneously pulls soil moisture data and calculates evapotranspiration—the combined water loss from soil evaporation and plant transpiration.

2. Multi-Zone Processing

Each irrigation zone is processed independently based on its specific characteristics: plant type (vegetables, grass, shrubs), soil composition (clay, loam, sand), and historical watering patterns. The system maintains individual moisture profiles for every zone.

3. Decision Logic Execution

Advanced logic evaluates multiple factors: expected rainfall in the next 24 hours, current soil moisture deficit, plant-specific water requirements, temperature adjustments, and optimal watering times based on wind conditions. Watering is only scheduled when all conditions indicate necessity.

4. Action & Notification

Approved watering schedules trigger IoT valve controllers, log decisions to Google Sheets for audit trails, and send notifications via Slack. The system provides full transparency into why each decision was made.

Quick Setup Guide

Follow these steps to implement this smart irrigation system:

  1. Import the template into your n8n instance using the downloaded JSON file.
  2. Configure weather API credentials in the OpenWeatherMap nodes with your location coordinates.
  3. Define your irrigation zones in the "Define Irrigation Zones" node with plant types, soil characteristics, and water thresholds.
  4. Connect your IoT controller by updating the HTTP Request node endpoint to match your valve control system.
  5. Set up logging by connecting Google Sheets or your preferred database for decision tracking.
  6. Test with manual trigger to verify weather data retrieval and decision logic before enabling scheduled execution.
  7. Activate the schedule to run daily (or as needed) for automated irrigation management.
