Spotify Discover Weekly Archive

Automatically save your weekly music discoveries to a permanent playlist. Never lose a favorite track again.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Spotify's Discover Weekly playlist is a magical feature that delivers 30 new songs every Monday based on your listening habits. The problem? These tracks disappear the following week, replaced by a fresh batch. If you don't manually save songs you love, they're gone forever.

This n8n automation solves that problem permanently. It automatically copies all tracks from your Discover Weekly playlist to a separate archive playlist you create, checking for duplicates so your archive stays clean and organized. Once set up, it runs on a schedule (weekly is recommended), preserving your musical discoveries without any ongoing effort.

The workflow represents a perfect example of "set-and-forget" automation—solving a recurring personal productivity task that would otherwise require manual attention every single week.

How It Works

The automation follows a logical sequence to ensure reliable, duplicate-free archiving.

Step 1: Scheduled Trigger

A schedule node initiates the workflow automatically every Monday (or your preferred frequency). This ensures the archiving happens consistently without you remembering to run it manually.

Step 2: Fetch Discover Weekly Tracks

The workflow connects to your Spotify account via OAuth and retrieves all tracks from your current Discover Weekly playlist. It handles authentication securely and can access any playlist you own or follow.

Step 3: Check Existing Archive

Before adding new tracks, the workflow fetches all songs already in your designated archive playlist. This prevents adding duplicates and keeps your archive clean and unique.

Step 4: Filter & Compare

Using n8n's logic nodes, the system compares the new Discover Weekly tracks against what's already in your archive. Only tracks that don't already exist in the archive proceed to the next step.

Step 5: Add to Archive Playlist

The filtered list of new tracks is added to your archive playlist through Spotify's API. The workflow handles any API rate limits and errors gracefully, ensuring all eligible tracks are preserved.

Step 6: Notification & Logging

Optionally, the workflow can send you a notification (email, Slack, etc.) confirming how many tracks were archived. It also logs the operation for troubleshooting and peace of mind.

Pro tip: Set the schedule to run early Tuesday morning to ensure you capture the full Discover Weekly playlist after it updates on Monday.

What You'll Need

  1. A Spotify Premium account (required for API access)
  2. An n8n instance (cloud or self-hosted)
  3. A manually created Spotify playlist to serve as your archive destination
  4. Spotify Developer credentials (Client ID and Client Secret)
  5. Basic understanding of how to create and configure OAuth apps (guided in setup)

Note: The archive playlist must be created manually first—the workflow adds tracks to it but doesn't create the playlist itself.

Quick Setup Guide

Follow these steps to implement this automation in under 15 minutes:

  1. Download the template using the button above and import it into your n8n instance.
  2. Create a Spotify Developer application at developer.spotify.com to get your Client ID and Client Secret.
  3. Set up OAuth credentials in n8n's Spotify node, following the documentation links included in the workflow.
  4. Create your archive playlist in Spotify and note its playlist ID (found in the share link).
  5. Configure the workflow variables with your Discover Weekly playlist ID and archive playlist ID.
  6. Set your preferred schedule (weekly recommended) and activate the workflow.
  7. Test with manual trigger first to ensure tracks are copying correctly before enabling the schedule.
