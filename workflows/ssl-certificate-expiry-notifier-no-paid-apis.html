What This Workflow Does

Website security breaches often start with something as simple as an expired SSL certificate. When certificates lapse, visitors see alarming security warnings, search engines penalize your site, and customer trust evaporates instantly. Manual monitoring across multiple domains is tedious, error-prone, and often forgotten until it's too late.

This automation solves that problem by providing a complete, zero-cost SSL certificate monitoring system. It automatically checks expiry dates for all your domains daily, identifies certificates expiring within 7 days, and sends immediate email alerts to your team. The entire system runs on free APIs and open-source tools, eliminating expensive monthly subscriptions while providing enterprise-grade monitoring.

Whether you manage 5 websites or 500, this workflow ensures you never face the business disruption of an expired SSL certificate again. It transforms a critical security task from a manual chore into a reliable, automated process that protects your online presence 24/7.

How It Works

The workflow follows a logical sequence that runs automatically each day, checking all your domains and alerting you only when action is needed.

1. Daily Scheduled Trigger

The workflow activates automatically every morning at 8 AM using n8n's schedule trigger. This consistent daily check ensures you have at least 7 days' notice for any expiring certificates, providing ample time for renewal.

2. Read Domain List from Google Sheets

It connects to your Google Sheet containing all domains to monitor. This centralized list makes management simple—just add or remove URLs in the spreadsheet, and the automation immediately adjusts. No code changes required.

3. Check SSL Certificate Details

For each domain, the workflow calls the free ssl-checker.io API to retrieve current certificate information: valid from date, valid until date, issuer, and days remaining. This free API provides accurate, up-to-date SSL data without rate limits for reasonable usage.

4. Identify Certificates Needing Attention

A conditional filter identifies certificates with 7 or fewer days remaining. This threshold is configurable—you can adjust it to 14, 30, or any number of days based on your renewal process timing.

5. Send Email Alerts

When expiring certificates are detected, the system sends formatted HTML email alerts to your designated team members. The email clearly lists which domains are affected, their expiry dates, and days remaining, enabling immediate action.

6. Update Monitoring Log

Finally, the workflow updates your Google Sheet with the latest check results, creating an audit trail of all monitoring activity. This historical data helps identify patterns and proves compliance with security monitoring requirements.

What You'll Need

  1. n8n instance – Either self-hosted or n8n.cloud account
  2. Google Sheets access – With OAuth2 credentials configured in n8n
  3. Email SMTP credentials – For sending alerts (Gmail, SendGrid, etc.)
  4. List of domains to monitor – In a Google Sheet with a "URL" column
  5. 10-15 minutes setup time – To configure connections and test

Pro tip: Create a dedicated "Security Alerts" email distribution list rather than using individual addresses. This ensures alerts never get missed if someone is out of office, and creates a permanent record of all certificate warnings for compliance purposes.

Quick Setup Guide

Getting this automation running takes about 15 minutes. Follow these steps to deploy it in your environment.

  1. Download and import the JSON template into your n8n instance
  2. Connect Google Sheets using OAuth2 credentials and select your domain list spreadsheet
  3. Configure SMTP email with your email service credentials and recipient addresses
  4. Adjust the alert threshold if you prefer warnings earlier than 7 days (optional)
  5. Test with one domain to verify the complete flow works correctly
  6. Activate the workflow and let it run automatically every day

The workflow includes detailed sticky notes explaining each node's configuration, making setup straightforward even if you're new to n8n. Once active, it requires zero maintenance—just update your domain list in Google Sheets as needed.
