What This Workflow Does

This automation solves the tedious problem of manually collecting prospect information from LinkedIn and transferring it to your database. Sales and recruiting teams spend hours copying names, titles, companies, and contact details—a process prone to errors and inconsistencies.

The workflow automatically runs a LinkedIn Profile Scraper phantom through Phantombuster, extracts structured profile data, and saves it directly to your Airtable base. What used to take 5-10 minutes per profile now happens automatically in the background, allowing your team to focus on engagement rather than data entry.

Beyond LinkedIn, this template serves as a blueprint for any web scraping-to-database automation. You can adapt it to collect product data, monitor prices, gather reviews, or track industry news—any scenario where you need to move online information into structured business records.

How It Works

Step 1: Trigger the Phantom

The workflow starts by checking for completed phantom runs in your Phantombuster account. You configure which LinkedIn profiles to scrape (by URL, search results, or company pages) through Phantombuster's interface before the automation runs.

Step 2: Extract and Structure Data

Phantombuster returns raw profile data including name, headline, current position, company, location, and sometimes contact information. The workflow processes this data, cleaning and formatting it into consistent fields ready for database storage.

Step 3: Transform with Set Node

A Set node maps the scraped data to your preferred field structure. You can rename fields, combine values (like first + last name), add timestamps, or calculate derived values (like seniority level based on title).

Step 4: Save to Airtable

The structured data appends to your specified Airtable base and table. Each profile becomes a new record with all extracted fields populated. The automation can include error handling to skip duplicates or flag incomplete records.

What You'll Need

  1. A Phantombuster account with the LinkedIn Profile Scraper phantom configured
  2. An Airtable account with a base and table ready to receive prospect data
  3. n8n instance (cloud or self-hosted) with Phantombuster and Airtable nodes installed
  4. API keys for both Phantombuster and Airtable added to your n8n credentials
  5. Clear understanding of which LinkedIn profiles or searches you want to scrape

Quick Setup Guide

  1. Download the template and import it into your n8n instance
  2. Add your Phantombuster and Airtable API credentials to n8n
  3. Configure the Phantombuster node with your specific agent ID
  4. Update the Airtable node with your base ID, table name, and field mappings
  5. Test with a single profile URL to verify the end-to-end flow
  6. Schedule the workflow to run daily or weekly for continuous data collection
