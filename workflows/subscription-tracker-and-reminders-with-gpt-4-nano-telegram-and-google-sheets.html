Subscription Tracker & Reminders with GPT-4 Nano, Telegram & Google Sheets

Automate subscription billing reminders and data updates via Telegram. Runs daily to check due subscriptions, send reminders, and process user messages using AI.

What This Workflow Does

Manual subscription tracking is a tedious, error-prone task that consumes hours each week. Businesses forget to send reminders, miss renewal dates, and struggle to update billing information consistently. This automation solves that by creating a fully automated subscription management system.

The workflow runs daily at 8:00 AM to scan your Google Sheets subscription database. It identifies upcoming due subscriptions, formats personalized reminders, and sends them directly to users via Telegram. Simultaneously, it processes incoming Telegram messages—users can ask to add, update, or retrieve their subscription info—using GPT-4 nano to understand natural language requests and update the spreadsheet accordingly.

This transforms subscription management from a manual administrative burden into an automated, conversational system that improves accuracy, saves time, and enhances customer experience.

How It Works

Step 1: Daily Subscription Check

A schedule trigger initiates the workflow every morning. It connects to your Google Sheets and reads all subscription rows, filtering those with due dates within the next 3–7 days.

Step 2: Reminder Generation & Delivery

For each upcoming subscription, the workflow formats a clear reminder message including amount, due date, and payment method. It then sends this message via Telegram bot directly to the user's chat.

Step 3: User Message Processing

When users send Telegram messages like "Update my plan to $50 monthly" or "When is my next bill?", GPT-4 nano interprets the request, extracts relevant data, and updates the Google Sheets accordingly. The workflow then replies with confirmation.

Step 4: Data Synchronization

All changes—whether from automated checks or user requests—are synchronized back to Google Sheets, maintaining a clean, up-to-date subscription database without manual entry.

What You'll Need

  1. A Google Sheets spreadsheet with subscription data (columns for user, amount, due date, status)
  2. OpenAI API key with access to GPT-4 nano
  3. Telegram bot token created via BotFather
  4. n8n instance (cloud or self-hosted)
  5. Basic understanding of connecting APIs (credentials setup)

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n workspace.
  2. Configure the Google Sheets node with your spreadsheet ID and OAuth credentials.
  3. Add your OpenAI API key in the GPT-4 nano node settings.
  4. Insert your Telegram bot token and set up the webhook URL.
  5. Adjust the schedule trigger to your preferred reminder timing (default 8:00 AM daily).
  6. Test with a few sample subscriptions to verify reminder formatting and delivery.
  7. Activate the workflow and monitor initial runs for any configuration adjustments.

Pro tip: Start with a small test group of subscriptions before rolling out to all customers. This allows you to fine-tune message formatting and timing based on actual user feedback.
