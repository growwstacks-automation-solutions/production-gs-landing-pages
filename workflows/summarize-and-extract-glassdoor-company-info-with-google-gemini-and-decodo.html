What This Workflow Does

Manual Glassdoor research is time-consuming and inconsistent. Recruiters, investors, and market analysts spend hours copying data from company profiles, reading through hundreds of reviews, and trying to spot trends in employee sentiment. This process is not only slow but also prone to human bias and oversight.

This n8n workflow automates the entire Glassdoor intelligence pipeline. It extracts raw company data using the Decode API, processes it through Google Gemini AI to identify key insights and sentiment patterns, and delivers structured, actionable reports in JSON format. What used to take days of manual research now happens in minutes with consistent, data-driven results.

The automation transforms scattered employee feedback into organized intelligence—tracking company culture shifts, identifying competitive advantages, and providing executive summaries that inform hiring decisions, investment research, and market positioning strategies.

How It Works

The workflow follows a logical pipeline from data collection to insight delivery:

1. Trigger & Input Configuration

The process begins with a manual trigger or scheduled execution. You provide the Glassdoor company URL and geographic location parameters. The workflow validates inputs and prepares for API calls.

2. Data Extraction via Decode API

The Decode API node fetches comprehensive company data from Glassdoor, including overall ratings, detailed review breakdowns, salary information, interview experiences, CEO approval ratings, and frequently asked questions. This replaces manual browsing and copy-pasting.

3. AI-Powered Analysis with Google Gemini

Raw Glassdoor data passes to Google Gemini AI, which performs natural language processing on employee reviews. The AI identifies common themes in pros and cons, detects sentiment trends, extracts cultural indicators, and highlights recurring compensation or management issues.

4. Structured Data Transformation

The workflow uses n8n's data transformation nodes to organize AI insights into consistent categories: company overview, rating summaries, key strengths/weaknesses, cultural indicators, and hiring recommendations. This creates a standardized analysis framework.

5. Report Generation & Export

Final insights compile into comprehensive JSON reports with executive summaries. The workflow can save files locally, send to cloud storage, or push to business intelligence platforms. Optional email notifications alert stakeholders when analysis completes.

Pro tip: Schedule this workflow to run weekly for ongoing competitor monitoring. Track how employee sentiment changes after major company announcements or quarterly earnings reports.

Quick Setup Guide

Get this automation running in under 15 minutes:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure credentials for both Decode API and Google Gemini in n8n's credential management
  3. Set your target companies by updating the "Set Input Fields" node with Glassdoor URLs and locations
  4. Test the workflow with a single company to verify data extraction and AI processing
  5. Configure output destinations - choose where reports should be saved (local file, Google Drive, database)
  6. Schedule execution for regular automated analysis (daily, weekly, or monthly)
  7. Review sample output and customize the AI prompt if you need different insight categories

Implementation note: Start with 2-3 companies to establish baseline performance before scaling to dozens of targets. Monitor API usage costs, especially for Gemini processing of large review volumes.
