YouTube Transcript Summarization with Google Gemini

Automatically summarize YouTube videos in any language and save to Google Docs

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the challenge of manually transcribing and summarizing YouTube videos for content repurposing. It eliminates hours of manual work by automatically extracting transcripts, generating concise summaries in any language using Google Gemini's AI, and storing the results in Google Docs.

The workflow is particularly valuable for content teams, educators, and marketers who need to quickly repurpose video content for blogs, social media, or internal documentation. By handling multilingual summarization, it enables global content strategies without requiring human translators.

How It Works

1. Video URL Input

The workflow starts by accepting a YouTube video URL. This can be triggered manually or automatically from a content calendar or RSS feed.

2. Transcript Extraction

Using the YouTube API, the workflow retrieves the complete transcript with timestamps and speaker information when available.

3. AI Summarization

Google Gemini processes the transcript, generating a concise summary in the specified target language while preserving key points and context.

4. Google Docs Storage

The final summary is formatted and appended to a designated Google Doc, ready for review or immediate publishing.

  4. n8n instance (self-hosted or cloud)

Quick Setup Guide

  1. Import the JSON template into your n8n instance
  2. Configure YouTube API credentials in the HTTP Request node
  3. Set up Google Gemini API connection
  4. Specify your target Google Doc in the Google Docs node
  5. Test with sample YouTube URLs and adjust summary length as needed

Pro tip: For multilingual teams, add a language detection step to automatically route videos to the appropriate summarization model.
