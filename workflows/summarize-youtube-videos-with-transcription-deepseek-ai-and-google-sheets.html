YouTube Video Summarizer with AI

Automatically transcribe and summarize YouTube videos using DeepSeek AI and save to Google Sheets

Download Template JSON · n8n compatible · Free

What This Workflow Does

Tired of manually watching long YouTube videos just to extract the main points? This automation solves that problem by instantly turning any video into an actionable, AI-generated summary stored in Google Sheets.

The workflow reads video URLs from your Google Sheet, generates accurate transcripts using Supadata.ai (with 100 free uses/month), then creates smart summaries using DeepSeek AI including: a short summary, key points, and main topics. Everything automatically updates your spreadsheet.

How It Works

1. Input YouTube URLs

Simply add YouTube video links to your Google Sheet. The automation monitors a "Status" column - set to "pending" to trigger processing.

2. Automatic Transcription

The workflow extracts the video transcript using Supadata.ai's API, which handles speech recognition and timestamp alignment.

3. AI Summarization

DeepSeek AI analyzes the transcript to generate three outputs: a concise summary paragraph, bullet points of key ideas, and identified main topics.

4. Results Storage

All outputs are automatically written back to your Google Sheet, with status updated to "complete". The structured format makes insights easy to search and reference.

Quick Setup Guide

  1. Create a Google Sheet with columns for URL, Status, Transcript, and Summary
  2. Import this workflow template into your n8n instance
  3. Connect your Google Sheets and API credentials
  4. Add YouTube URLs and set status to "pending" to trigger processing
  5. Monitor the "done" column for completion status

Pro tip: Add a "Topics" column to your sheet and use n8n's AI node to auto-tag videos by category for better organization.
