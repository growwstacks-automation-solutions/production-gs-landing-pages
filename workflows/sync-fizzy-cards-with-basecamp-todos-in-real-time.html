Sync Fizzy Cards with Basecamp Todos in Real Time

Automatically keep your visual card system and structured todo lists perfectly aligned—no manual updates needed.

What This Workflow Does

Teams often use multiple project management tools for different purposes—Fizzy for visual card-based collaboration and Basecamp for structured client-facing todo lists. Without automation, team members must manually duplicate updates between platforms, leading to missed tasks, inconsistent status reporting, and wasted administrative time.

This workflow solves that by creating a real-time bridge between Fizzy and Basecamp. Whenever a card is created, assigned, completed, or reopened in Fizzy, the corresponding Basecamp todo instantly reflects the change. Card tags automatically map to todolists, and team assignments sync via email matching. The result is a unified project management system that maintains both platforms' strengths without manual overhead.

How It Works

Step 1: Receive Fizzy Webhook Events

Fizzy sends instant notifications to n8n whenever card events occur—published, assigned, unassigned, closed, reopened, or postponed. The webhook captures all relevant card data including title, description, tags, assignees, and status.

Step 2: Match to Basecamp Project

The workflow identifies the corresponding Basecamp project by matching the Fizzy board name exactly. This ensures each board syncs to the correct project, supporting multiple parallel workflows for different teams or clients.

Step 3: Map Tags to Todolists

Fizzy card tags determine which Basecamp todolist receives the todo. If a matching todolist doesn't exist, the automation creates it automatically, preserving your organizational structure across platforms.

Step 4: Create or Update Todos

Based on the event type, the workflow either creates a new todo, updates an existing todo's assignees, or marks it complete/incomplete. Each todo includes a reference to the original Fizzy card ID for tracking consistency.

Step 5: Sync Assignees by Email

Team members are matched between platforms using their email addresses. When a card is assigned in Fizzy, the corresponding Basecamp todo is assigned to the same person, maintaining responsibility alignment.

Pro tip: Ensure team members use consistent email addresses in both Fizzy and Basecamp for seamless assignee matching. The first tag on a Fizzy card determines the Basecamp todolist—use clear, consistent tagging conventions.

What You'll Need

  1. An n8n instance (self-hosted or cloud)
  2. Basecamp account with project access and Todos tool enabled
  3. Fizzy account with board management permissions
  4. Matching project/board names in both platforms
  5. Consistent email addresses for team members across both systems
  6. Basecamp OAuth2 credentials (Client ID and Secret from launchpad.37signals.com)

Quick Setup Guide

  1. Install Basecamp community node: Run npm install n8n-nodes-basecamp or install via n8n Community Nodes interface.
  2. Configure Basecamp credentials: Create integration at launchpad.37signals.com, copy Client ID and Secret, add OAuth2 credentials in n8n.
  3. Set your Basecamp Account ID: Find it in your Basecamp URL and update the "Set Basecamp account ID" node in the workflow.
  4. Prepare Basecamp project: Create a project with matching name to your Fizzy board, enable Todos tool.
  5. Configure Fizzy webhook: Open your Fizzy board, click globe icon for webhook settings, paste URL from "Receive Fizzy webhook" node, enable all event types.
  6. Activate workflow: Turn on the workflow in n8n and test with a card creation in Fizzy to verify Basecamp todo appears.

Pro tip: Each Fizzy board requires its own webhook configuration. For multiple boards, duplicate the workflow and adjust the board name matching logic.
