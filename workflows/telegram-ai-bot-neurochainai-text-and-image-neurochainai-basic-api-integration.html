What This Workflow Does

This automation template solves the problem of manual, time-consuming customer interactions by creating an intelligent Telegram bot powered by NeurochainAI. Businesses often struggle to provide instant, 24/7 support while managing costs—this workflow bridges that gap by automating both text-based conversations and image generation requests.

The solution connects Telegram's massive messaging platform with NeurochainAI's advanced inference capabilities, allowing customers to ask questions, get detailed responses, and even request AI-generated images—all through a familiar chat interface. This transforms your Telegram channel from a simple communication tool into a powerful automated assistant that handles routine inquiries, qualifies leads, and delivers visual content on demand.

How It Works

1. Telegram Message Trigger

The workflow starts when a user sends a message to your Telegram bot. The Telegram Trigger node listens for new messages and captures the user's input, whether it's a text question or a request for an image.

2. Message Analysis & Routing

A Switch node analyzes the message content to determine the user's intent. Text-based queries are routed to NeurochainAI's text inference engine, while image generation requests (containing keywords like "generate," "create," or "image") are directed to the image processing branch.

3. AI Processing

For text queries, the HTTP Request node sends the message to NeurochainAI's API with your configured parameters, receiving intelligent, context-aware responses. For image requests, it calls NeurochainAI's Flux model with the user's description to generate custom visuals.

4. Response Delivery

The processed response—whether text answer or generated image—is formatted and sent back to the user through the Telegram node, completing the conversation loop instantly without human intervention.

What You'll Need

  1. A Telegram Bot Token (created via @BotFather)
  2. NeurochainAI API Key with sufficient credits
  3. n8n instance (cloud or self-hosted)
  4. Basic understanding of your customer's common questions
  5. List of image generation scenarios relevant to your business

Pro tip: Start by mapping out your 10 most frequent customer questions. Configure the bot to handle these perfectly first, then expand to more complex scenarios. This ensures immediate value and user satisfaction.

Quick Setup Guide

  1. Download and import the JSON template into your n8n workspace
  2. Create credentials for Telegram Bot Token in n8n's credentials management
  3. Add your NeurochainAI API Key as HTTP Request authentication
  4. Test with simple queries to verify the connection works
  5. Customize the Switch node logic to match your specific use cases
  6. Activate the workflow and start your Telegram bot
