Telegram bot: extract & store TikTok and LinkedIn data to Google Sheets with Dumpling AI

Automate social media profile data collection through a Telegram interface with AI-powered extraction

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the tedious manual process of collecting and organizing social media profile data. Marketing teams waste hours copying metrics from TikTok and LinkedIn into spreadsheets, while recruiters struggle to track candidate social footprints. The workflow creates a Telegram bot interface that triggers AI-powered data extraction from any public profile URL.

Dumpling AI parses complex social media pages into structured data points, which automatically populate designated Google Sheets. The system maintains data consistency, eliminates copy-paste errors, and enables real-time team collaboration on the collected insights. Historical tracking becomes effortless as the sheet grows with each new data request.

How It Works

1. Telegram Bot Trigger

Team members send profile URLs via Telegram message to your dedicated bot. The bot accepts TikTok and LinkedIn links in any chat format - private messages, groups, or channels.

2. Dumpling AI Processing

The workflow passes each URL to Dumpling AI's extraction API. Their machine learning models analyze profile pages to identify key data points like follower counts, engagement metrics, job history, and skills.

3. Google Sheets Integration

Structured data returns from Dumpling AI formatted for direct insertion into your Google Sheet. The workflow appends new rows with timestamps, maintaining a searchable historical record of all collected profiles.

Pro tip: Create separate sheets for TikTok influencers and LinkedIn professionals to enable platform-specific analysis while maintaining one master database.

What You'll Need

  1. Active n8n instance (self-hosted or cloud)
  2. Telegram bot token from @BotFather
  3. Dumpling AI API key (free tier available)
  4. Google Sheets with edit permissions
  5. List of target profile URLs (optional starting dataset)

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure Telegram bot credentials in the trigger node
  3. Add your Dumpling AI API key to the HTTP request node
  4. Connect to Google Sheets and specify your target spreadsheet ID
  5. Test by sending profile links to your Telegram bot
  6. Deploy the workflow for team access
