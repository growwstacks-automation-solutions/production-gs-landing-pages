What This Workflow Does

This n8n workflow transforms your Telegram messenger into a professional interior design studio powered by artificial intelligence. Instead of spending hours manually cataloging furniture or creating design mockups, you can simply send photos through Telegram and let the automation handle everything.

The system intelligently analyzes uploaded images to identify furniture pieces, room characteristics, and design elements. It maintains organized databases of your inventory and spaces, then uses advanced AI to generate stunning interior designs tailored to your specific requirements. Whether you're a professional designer managing client projects or a homeowner planning renovations, this automation turns complex design tasks into simple conversations.

By combining Telegram's accessibility with powerful AI capabilities, you get a design assistant available 24/7 that understands your style preferences, remembers your furniture catalog, and produces professional-quality visualizations in seconds rather than days.

How It Works

1. Image Upload & Analysis

Users send photos of furniture or rooms through Telegram. The workflow automatically detects whether it's a furniture piece or a room space, then routes it to the appropriate processing path. For furniture, it extracts style, materials, dimensions, and era. For rooms, it analyzes size, lighting, architectural features, and existing design elements.

2. Intelligent Database Management

The system maintains three interconnected databases: a furniture catalog with detailed product information, a room documentation database with space characteristics, and a design portfolio storing all AI-generated concepts. Each entry includes searchable metadata, professional descriptions, and image references for easy retrieval.

3. AI-Powered Design Generation

When a design request comes in, the workflow combines room data with relevant catalog items and uses Gemini AI to create photorealistic interior designs. It can modify existing designs based on feedback, maintain style consistency across iterations, and generate multiple variations for client selection.

4. Telegram Integration & Delivery

All interactions happen seamlessly within Telegram. Users receive catalog confirmations, room analyses, and generated designs directly in their chat. The system supports natural language requests like "create a modern living room design using my catalog items" or "show me alternative layouts for this bedroom."

Quick Setup Guide

Follow these steps to implement your AI interior design assistant:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure Telegram bot by creating a bot via BotFather and adding your token to the Telegram trigger node
  3. Set up API credentials for OpenAI and Google Gemini in n8n's credentials management
  4. Connect Supabase by creating a new project and adding your connection details to the database nodes
  5. Test the workflow by sending a furniture photo to your Telegram bot - you should receive a catalog entry confirmation
  6. Customize prompts in the AI nodes to match your specific design style preferences and terminology
  7. Activate the workflow and start designing through simple Telegram messages

Important: For production use, set up proper error handling and consider adding user authentication if multiple team members will access the system. The template includes basic error notifications but can be extended for enterprise requirements.
