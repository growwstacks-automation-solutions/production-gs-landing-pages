Telegram to Spotify with OpenAI Automation

Control music playback through Telegram messages using AI-powered search. Perfect for office environments, retail spaces, and collaborative music systems.

What This Workflow Does

This automation solves a common problem in shared environments: how to let multiple people control music playback without sharing account credentials or interrupting workflow. Instead of requiring someone to manually search for songs on Spotify, team members can simply describe what they want to hear in a Telegram message.

The system uses OpenAI's natural language processing to interpret vague song descriptions—even when users don't remember exact titles or artists—and automatically finds and plays the closest match on Spotify. This creates a seamless, collaborative music experience that enhances workplace atmosphere without compromising security or productivity.

How It Works

The automation follows a sophisticated sequence that transforms simple Telegram messages into perfect music selections:

1. Message Reception & Processing

When a user sends a message to your Telegram bot (like "play that chill indie song from last week"), the workflow captures the text and prepares it for AI analysis. The system identifies the user and validates they have permission to control music.

2. AI-Powered Music Interpretation

OpenAI analyzes the natural language description and extracts key elements: genre, mood, era, instrumentation, and any remembered lyrics or artist hints. The AI generates an optimized search query that Spotify can understand, dramatically increasing match accuracy.

3. Spotify Search & Validation

The refined query searches Spotify's extensive library. The workflow evaluates results based on popularity, release date, and relevance to the original request. If no suitable match is found, the system sends a helpful message back to Telegram asking for clarification.

4. Playback Control & Confirmation

The selected track is added to the queue on your active Spotify device. The system ensures playback starts correctly, fetches the currently playing track information, and sends a confirmation message to Telegram with song details—creating a satisfying feedback loop.

Pro tip: For office environments, create a dedicated Telegram group for music requests. This allows team members to see what others are playing and creates a collaborative music culture without overwhelming any single person with control responsibilities.

What You'll Need

  1. Active n8n instance (cloud or self-hosted) with the workflow enabled
  2. OpenAI API key with access to appropriate models
  3. Telegram bot token created through BotFather
  4. Spotify Premium account with OAuth2 API access configured
  5. Active Spotify device (computer, speaker, or phone with Spotify open)
  6. Basic understanding of webhook URLs for Telegram configuration

Quick Setup Guide

Follow these steps to implement this automation in your environment:

  1. Download and import the template JSON file into your n8n instance
  2. Configure Telegram node with your bot token and set up the webhook
  3. Add your OpenAI API key to the AI node with appropriate model settings
  4. Authenticate Spotify using OAuth2—this creates a secure connection without storing passwords
  5. Test the workflow by sending a Telegram message to your bot while Spotify is active on a device
  6. Refine AI prompts based on your team's music preferences and description styles
  7. Set permissions by adjusting Telegram user ID filters if you want to restrict control

Pro tip: Start with a small group of test users before rolling out to your entire team. This allows you to refine the AI's interpretation of your specific music culture and identify any edge cases in song selection before widespread adoption.
