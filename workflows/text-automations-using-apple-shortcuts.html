What This Workflow Does

This automation solves a common productivity bottleneck: manually rewriting, translating, or polishing text across different apps. Instead of copying text, opening a translation tool or AI chat, pasting, waiting, and copying back, this workflow lets you process text instantly from anywhere on your Mac.

By connecting Apple Shortcuts to n8n, you create a personal text‑processing assistant. Select any text—an email draft, a social media post, a report paragraph—trigger a keyboard shortcut, and the processed result replaces your selection immediately. The workflow handles multiple request types through a single webhook, routing each to the appropriate AI prompt for translation, grammar correction, summarization, or expansion.

The business value is clear: reduced context‑switching, faster content creation, and consistent output quality. Writers, marketers, support agents, and non‑native speakers can polish communications in seconds, not minutes, directly within their workflow.

How It Works

The system is a elegant bridge between your local device and cloud‑powered AI, using n8n as the processing engine.

Step 1: Trigger from Apple Shortcuts

You install a pre‑configured Shortcut on your Mac or iOS device and assign a keyboard shortcut (e.g., Cmd+Shift+T). When you select text and press the keys, the Shortcut captures the text and sends it—along with a “query type”—to your n8n webhook URL via a POST request.

Step 2: Webhook Receives & Routes

The n8n workflow starts with a Webhook node that listens for incoming requests. It extracts the text content and the query type (e.g., “translate to Spanish,” “correct grammar,” “make shorter”). A Switch node then routes the request to the appropriate branch based on the query type.

Step 3: AI Processing

Each branch contains an OpenAI (or other AI service) node configured with a specific prompt tailored to the task. For example, the translation branch instructs the model to translate the input text into the target language while preserving meaning and tone. The grammar‑correction branch asks for corrections without altering the original intent.

Step 4: Response & Replacement

Once the AI returns the processed text, n8n’s “Respond to Webhook” node sends the result back to the Apple Shortcut. The Shortcut then takes the returned text and replaces the originally selected text in your active application, completing the loop in under a few seconds.

Pro tip: You can extend this template to add your own custom query types—like “convert to bullet points,” “rewrite in formal tone,” or “extract key dates”—by duplicating a branch and modifying the AI prompt.

Quick Setup Guide

Follow these steps to get your text automation running in under 10 minutes:

  1. Import the workflow: In your n8n dashboard, click “Import from File” and upload the downloaded JSON template.
  2. Add your OpenAI credentials: In each OpenAI node, add your API key. You can use the same key for all branches.
  3. Activate the webhook: Click the “Webhook” node and copy the unique URL it provides (e.g., https://your‑n8n.cloud/webhook/...​).
  4. Configure the Shortcut: On your Mac, open the downloaded Shortcut file. In the second step, replace the placeholder webhook URL with your copied n8n webhook URL.
  5. Set a keyboard shortcut: In Shortcuts app, open the shortcut details and assign a global keyboard shortcut (e.g., Cmd+Shift+1).
  6. Enable scripting: On your Mac, go to System Settings > Privacy & Security > Advanced, and check “Allow Running Scripts.”
  7. Test it: Select text in any app, press your keyboard shortcut, and watch it transform instantly.

Pro tip: Duplicate the Shortcut and create separate versions for different query types, each with its own keyboard shortcut. For example, use Cmd+Shift+1 for translation and Cmd+Shift+2 for grammar check for even faster access.
