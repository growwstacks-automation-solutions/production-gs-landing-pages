AI Image Generator Telegram Bot with Google Gemini

What This Workflow Does

This automation solves a common business problem: the need for quick, custom visual content without the time and cost of manual design work. Many teams struggle to create illustrations for social media, blog posts, presentations, or internal communications, often resorting to generic stock photos or lengthy design requests.

The workflow creates an intelligent Telegram bot that acts as your team's personal AI artist. When someone sends a text description like "a futuristic office scene," the bot automatically enhances this basic request into a detailed professional prompt, generates a high-quality image using Google's Gemini 2.0 Flash model, and delivers the result back to the chat within seconds.

This transforms a multi-day design process into a 30-second conversation, empowering your entire team to create professional visuals on demand while maintaining consistent quality through automated prompt engineering.

How It Works

Step 1: Message Reception

The Telegram Trigger node listens for incoming messages from users. When someone sends a text description to your bot, the workflow immediately captures it and prepares it for processing.

Step 2: AI Prompt Enhancement

Google Gemini 2.5 Flash Lite analyzes the basic user request and expands it into a detailed, professional prompt. It adds artistic details, lighting specifications, composition guidance, style parameters, and quality requirements—turning "a cat" into "a majestic Siamese cat on a velvet cushion in cinematic lighting."

Step 3: Structured Processing

The enhanced prompt is formatted using JSON schema to ensure consistency. This structured approach guarantees that every generated image follows the same quality standards and includes all necessary parameters for optimal results.

Step 4: Image Generation

The formatted prompt is sent to Google Gemini 2.0 Flash image generation API. This advanced AI model creates a high-quality visual based on the detailed specifications, typically within 10-15 seconds.

Step 5: Delivery

The generated image is converted to a file format and sent back to the user via Telegram. The entire process—from basic text request to delivered image—happens automatically without any manual intervention.

What You'll Need

  1. Telegram Bot Token: Create a bot via @BotFather on Telegram to get your API credentials.
  2. Google AI Studio Account: Access to Gemini API services with an API key.
  3. n8n Instance: Self-hosted n8n (this workflow uses community nodes).
  4. Basic Understanding: Familiarity with connecting APIs in n8n or willingness to follow our setup guide.

Pro tip: Start with a private Telegram group for your team to test the bot before making it publicly available. This allows you to refine the prompt enhancement settings based on your specific brand voice and visual style requirements.

Quick Setup Guide

  1. Import the Template: Download the JSON file above and import it into your n8n instance.
  2. Configure Telegram: Create your bot via @BotFather and add the token to n8n's Telegram credentials.
  3. Set Up Google Gemini: Get your API key from Google AI Studio and add it to n8n's Google Gemini credentials.
  4. Test the Connection: Send a test message to your bot to verify the entire workflow functions correctly.
  5. Customize Prompts: Adjust the prompt enhancement instructions to match your brand's visual style preferences.
  6. Deploy: Activate the workflow and share your bot with your team or audience.
