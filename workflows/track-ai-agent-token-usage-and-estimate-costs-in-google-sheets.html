What This Workflow Does

As businesses increasingly deploy AI agents for customer service, content generation, data analysis, and automation, managing associated costs becomes critical. AI providers charge based on token usage, but tracking this consumption across multiple agents and projects is complex and often overlooked until invoices arrive.

This n8n automation template solves this problem by systematically extracting token usage metadata from AI agent executions, processing the data, and logging it into Google Sheets with automatic cost calculations. It provides real-time visibility into AI spending, enables budget forecasting, and helps identify optimization opportunities.

The workflow taps into execution metadata that standard AI APIs often don't surface comprehensively, particularly for tool calls and complex agent interactions. It works seamlessly with major providers like OpenAI, Google Gemini, and Anthropic, though it can be adapted for other LLM services.

How It Works

Step 1: AI Agent Execution & Metadata Capture

The workflow begins when your AI agent completes its task. Instead of just returning the final output, the system captures comprehensive execution metadata including prompt tokens, completion tokens, total tokens, and specific tool call consumption.

Step 2: Sub-workflow Token Calculation

A dedicated sub-workflow is triggered to analyze the execution data. This component extracts token counts from the workflow execution logs, normalizes data across different AI provider formats, and prepares it for cost calculation based on your specific pricing tier.

Step 3: Google Sheets Integration & Cost Estimation

The processed data is sent to Google Sheets where pre-configured formulas automatically calculate costs based on your provider's pricing (e.g., OpenAI's GPT-4 pricing, Gemini Pro rates). The spreadsheet maintains historical records, enabling trend analysis and budget forecasting.

Pro tip: Call the token calculation sub-workflow after all other agent branches complete to ensure comprehensive data capture. Place it at the end of your main agent workflow for accurate reporting.

Quick Setup Guide

  1. Download and Import: Download the JSON template file and import it into your n8n instance via the workflow import function.
  2. Configure Credentials: Set up credentials for your AI provider(s), Google Sheets API, and n8n instance in the n8n credentials management.
  3. Customize AI Agent: Replace the example AI agent in the template with your actual agent workflow while preserving the token calculation sub-workflow call.
  4. Set Up Google Sheet: Duplicate the provided Google Sheets template, note the Sheet ID, and update the workflow node with your specific spreadsheet details.
  5. Test Execution: Run a test execution with your AI agent, verify data flows to Google Sheets, and check cost calculations match your provider pricing.
  6. Deploy and Schedule: Activate the workflow and set appropriate triggers or schedules based on your agent usage patterns.
