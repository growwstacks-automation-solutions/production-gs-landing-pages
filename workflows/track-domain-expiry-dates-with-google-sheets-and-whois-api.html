What This Workflow Does

Forget manual domain checking. This automation solves the critical problem of domain portfolio management by eliminating the risk of unexpected website downtime. Every expired domain means lost traffic, revenue, and brand damage—especially for SEO-dependent businesses where rankings can plummet overnight.

The workflow automatically reads your domain list from Google Sheets, queries the WHOIS API for real-time DNS SOA records, extracts expiry timestamps, converts them to readable dates, and updates your tracking sheet—all without manual intervention. It processes domains sequentially with built-in delays to respect API rate limits, making it production-ready for portfolios of any size.

How It Works

1. Configuration & Domain Retrieval

The workflow starts by loading your Google Sheet ID and worksheet name. It reads all domains listed in your "Websites" column, creating a clean dataset for processing. This setup ensures you can use your existing domain tracking spreadsheet without reformatting.

2. WHOIS API Query

For each domain, the system sends a secure request to the WHOIS API (via RapidAPI) to fetch DNS SOA records. These records contain the authoritative expiry timestamp directly from the domain registry, providing the most accurate expiration data available.

3. Data Processing & Formatting

The raw expiry timestamp is extracted and converted into human-readable DD-MM-YYYY format. The workflow also parses out the expiry month name and year as separate values, giving you multiple data points for filtering and reporting in your spreadsheet.

4. Sheet Update & Loop Control

Processed data is written back to the corresponding row in your Google Sheet, matching domains using the "Websites" column. A 30-second pause between domains prevents API throttling while maintaining steady progress through your entire portfolio.

Pro tip: Schedule this workflow to run monthly. This gives you 60-90 days notice before any domain expires, plenty of time for renewal while avoiding last-minute panic or premium redemption fees.

Quick Setup Guide

  1. Import the workflow: Copy the JSON template, open n8n, go to Workflows → Import from JSON, paste and import.
  2. Configure Google Sheets: Create OAuth2 credentials in n8n for Google Sheets, then paste your Sheet ID and worksheet name into the "Set Sheet Configuration" node.
  3. Set up WHOIS API: Get your RapidAPI key from their dashboard and add it to the HTTP Request node headers.
  4. Test with one domain: Run the workflow manually with a single domain in your sheet to verify data flows correctly.
  5. Schedule automation: Set the workflow to trigger monthly using n8n's schedule node for hands-off operation.
