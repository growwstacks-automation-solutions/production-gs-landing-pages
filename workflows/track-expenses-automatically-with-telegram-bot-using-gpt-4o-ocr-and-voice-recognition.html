What This Workflow Does

Manual expense tracking is tedious, error-prone, and often forgotten. This automation solves that by turning Telegram into a smart, AI-powered expense assistant. Whether you're a freelancer managing receipts, a business owner tracking costs, or a team submitting expenses, this workflow eliminates manual data entry.

The bot accepts inputs in any form: a photo of a receipt, a PDF invoice, a voice message like "spent $45 on client lunch," or a simple text. It uses AI (GPT-4o) to understand the context, OCR to extract text from images, and voice recognition to transcribe audio. It then automatically categorizes the expense, logs it with the correct date and amount, and stores it in your chosen format.

Beyond logging, you can query your expenses naturally. Ask "How much did I spend on groceries last month?" or "Show my business travel costs," and the bot provides instant summaries and comparisons. It's like having a personal bookkeeper available 24/7 via a chat you already use.

How It Works

The workflow is built on n8n and connects several powerful services into a seamless, intelligent system.

Step 1: Message Reception & Privacy Lock

The Telegram trigger node listens for all incoming messages. A privacy module ensures only the first authorized user (you) can access the bot, rejecting any other attempts to keep your financial data secure.

Step 2: Input Processing & Routing

The workflow intelligently routes different message types. Text and voice messages go directly to the AI for interpretation. Photos and documents are first sent through an OCR service to extract any text, which is then passed to the AI.

Step 3: AI-Powered Understanding & Categorization

This is the core. Using GPT-4o via OpenRouter, the system understands natural language. It identifies the amount, currency, vendor, date, and most relevant category (e.g., Groceries, Transportation, Software). It can handle multiple languages and ambiguous descriptions.

Step 4: Data Storage & Organization

The structured expense data is saved to a JSON storage system (like Airtable or a simple database) organized by month. Each entry gets a unique ID, timestamp, and full details for accurate record-keeping and reporting.

Step 5: Query Handling & Response

When you ask a question, the Expense Assistant agent queries the stored data, performs calculations (totals, category breakdowns, month-over-month comparisons), and formats a clear, conversational reply sent back through Telegram.

What You'll Need

  1. A Telegram Bot Token: Created for free via @BotFather on Telegram. This allows your workflow to send and receive messages.
  2. An OpenRouter API Key: Provides access to GPT-4o and other AI models for understanding and processing expenses. A free tier is available to start.
  3. An Ainoflow API Key (or alternative): For the OCR, voice transcription, and JSON storage services used in the template. You can also substitute with other services like Google Cloud Vision or OpenAI Whisper.
  4. An n8n Instance: You can use the free, self-hosted version of n8n or n8n.cloud to run the workflow.
  5. A Storage Destination: The template stores data via Ainoflow's JSON storage, but you can easily modify it to save to Google Sheets, Airtable, a PostgreSQL database, or any other platform you prefer.

Pro tip: Start with the free tiers of OpenRouter and Ainoflow to test the workflow fully. You can process dozens of expenses per month at no cost before needing to upgrade.

Quick Setup Guide

Follow these steps to import and configure your personal expense tracker bot in under 30 minutes.

  1. Download & Import: Click the download button above to get the JSON file. In your n8n instance, go to Workflows > Import from File and select the downloaded template.
  2. Configure Telegram Credentials: In the "Telegram Trigger" node, create a new credential using the bot token from @BotFather. Test the connection.
  3. Set Up AI Services: In the "OpenRouter" and "Ainoflow" nodes, add your API keys as credentials. For Ainoflow, use the Bearer Token auth method.
  4. Customize Categories (Optional): Review the "Expense Assistant" agent node. You can modify the list of expense categories in the system prompt to match your personal or business needs.
  5. Activate & Test: Turn the workflow on. Message your new Telegram bot with "/start" to claim ownership. Then, send a test expense like a photo of a receipt or a voice message saying "bought coffee for $5."
  6. Connect Your Storage (Optional): To change where data is saved, modify the "JsonStorageMcp" tool node or replace it with a node for Google Sheets, Airtable, etc.
