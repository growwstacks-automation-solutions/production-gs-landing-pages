What This Workflow Does

Forex traders rely on economic news releases to make timely decisions. Manual monitoring of Forex Factory is slow, error-prone, and misses opportunities. This automation solves that by instantly capturing actual data releases, comparing them against forecasts, and alerting you via Telegram when a release is positive or negative for a currency.

The workflow also logs the news details and live price from MyFxBook into Google Sheets, creating a historical database for analysis. This turns volatile news events into structured, actionable data, enabling systematic trading based on objective economic indicators rather than gut feeling.

It’s designed for traders who want to act before the market reacts—capturing the moment when GDP, inflation, or employment numbers deviate from expectations, and turning that information into profit.

How It Works

Step 1: Trigger on News Release

The workflow starts when a new Forex Factory news event is published. It filters for events with a numerical forecast value (GDP, CPI, unemployment), ignoring speeches or meetings that can’t be measured.

Step 2: Extract & Wait for Actual Data

It extracts currency, impact level, release date, and the news page link. Then it waits 10 seconds for the actual value to appear on the page, ensuring you get the final number, not a placeholder.

Step 3: Scrape & Compare Values

Using Airtop, the workflow scrapes the actual and forecast values from the news page. It removes non-numeric characters (%, K, M, B, T) and converts them to numbers. If the actual is lower than forecast (and lower is better), it sends the data to the True branch for positive alerts.

Step 4: Send Telegram Alert

A formatted message is sent to your Telegram chat with the currency, impact, actual vs forecast difference, and a link to the news. This alert can be used as a trigger to open a trading position in MetaTrader.

Step 5: Log Data to Google Sheets

The workflow fetches the current live price for the affected currency pair from MyFxBook and records the news details, actual/forecast values, and price into a Google Sheets spreadsheet for historical analysis.

What You'll Need

  1. Google Calendar credentials – to trigger on scheduled news events.
  2. Airtop API key – for scraping actual values from Forex Factory pages.
  3. Telegram Bot API token & Chat ID – to send alerts to your Telegram channel or group.
  4. Google Sheets credentials – with Google Drive API enabled in Google Cloud Console.
  5. MyFxBook account – to fetch live prices for affected currency pairs.
  6. A Google Sheets file structured with columns for currency, impact, actual, forecast, price, and timestamp.

Pro tip: Create your Google Sheets file exactly as shown in the template example. This ensures data logs correctly and you can later build dashboards or charts from the sheet.

  1. Import the template into your n8n instance via the JSON download.

  6. Monitor & refine – adjust wait times or filtering based on your trading strategy.
