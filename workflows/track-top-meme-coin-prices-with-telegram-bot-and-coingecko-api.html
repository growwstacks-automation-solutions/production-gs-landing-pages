What This Workflow Does

This automation solves the problem of constantly checking meme coin prices manually. Crypto traders and community managers can instantly fetch current prices for top meme coins through a simple Telegram command, saving time while staying informed about volatile markets.

The workflow listens for Telegram commands, fetches real-time data from CoinGecko's API, formats the response clearly, and sends it back to the user - all automatically. It's particularly valuable during periods of high volatility when meme coins can swing 20-50% in hours.

How It Works

1. Telegram Command Trigger

The workflow activates when users send "/memecoin" in a Telegram chat with your bot. The trigger captures the chat ID and message details to route the response correctly.

2. CoinGecko API Request

The system queries CoinGecko's API for the top 5 meme coins by market cap, retrieving current prices in USD along with coin identifiers for deeper research links.

3. Data Processing

A function node cleans and formats the API response into a readable message with emoji indicators, proper spacing, and direct links to each coin's CoinGecko page.

4. Telegram Response

The formatted message is sent back to the original Telegram chat, providing users with an instant, neatly organized price update without leaving the messaging app.
