What This Workflow Does

This n8n automation solves the challenge of manually tracking YouTube trends across different countries and languages. Content creators, marketers, and agencies often struggle to identify regional content preferences and emerging trends that could inform their video strategy. The workflow automatically pulls trending YouTube data through RapidAPI, organizes it by specified countries and languages, and logs everything in Google Sheets for easy analysis.

By automating this process, you save hours each week that would otherwise be spent manually checking YouTube's trending pages across multiple regions. The system captures valuable metadata including view counts, engagement metrics, and content categories, giving you data-driven insights to optimize your video content for different markets.

How It Works

1. Configure target countries and languages

The workflow starts by letting you specify which countries and languages you want to track. You can monitor multiple regions simultaneously, making it ideal for creators with international audiences or agencies managing clients in different markets.

2. Fetch trending data via RapidAPI

n8n connects to RapidAPI's YouTube trends endpoint to retrieve up-to-date information about the most popular videos in your selected regions. The API provides comprehensive data including view counts, engagement metrics, and content categories.

3. Process and organize the data

The workflow cleans and structures the raw API data, extracting the most valuable metrics and adding timestamps for trend analysis over time. It normalizes metrics across regions for easier comparison.

4. Update Google Sheets automatically

All processed data gets logged in a Google Sheet with separate tabs for different regions. The structured format makes it easy to create pivot tables, charts, or connect to data visualization tools.
