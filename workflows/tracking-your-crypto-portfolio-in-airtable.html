Tracking Your Crypto Portfolio in Airtable

Automatically sync your cryptocurrency investments with Airtable. This n8n workflow updates your portfolio values, calculates ROI, and creates visual dashboards - saving hours of manual tracking each month.

What This Workflow Does

Managing cryptocurrency investments manually is time-consuming and prone to errors. This automation connects your crypto holdings to Airtable, automatically fetching current prices and updating your portfolio value. It eliminates the need for manual spreadsheet updates while giving you a powerful visual dashboard to track performance.

The workflow pulls data from cryptocurrency APIs (like CoinGecko or CoinMarketCap), calculates your current portfolio value based on your holdings, and updates your Airtable base on a schedule you set. You can track ROI, allocation percentages, and historical performance - all automatically updated without lifting a finger.

The workflow connects crypto APIs to Airtable with automated data processing

How It Works

1. Fetch current crypto prices

The workflow connects to cryptocurrency APIs to get real-time or delayed price data for all coins in your portfolio. It handles API authentication and processes the raw data into a standardized format.

2. Calculate portfolio values

Using your recorded holdings (stored in Airtable), the workflow multiplies coin quantities by current prices to calculate the current USD value of each position and your total portfolio value.

3. Update Airtable records

The workflow updates your Airtable base with the new values, preserving historical data while refreshing current prices. It can also calculate metrics like ROI, daily/weekly changes, and allocation percentages.

4. Schedule automatic updates

Configure the workflow to run on your preferred schedule - daily, hourly, or weekly. Each run refreshes all values automatically, keeping your dashboard current without manual intervention.

Pro tip: Add a "Last Updated" field in Airtable to track when each price refresh occurred. The workflow can automatically timestamp each update.

What You'll Need

  1. An n8n instance (cloud or self-hosted)
  2. An Airtable account with a base for your crypto portfolio
  3. API keys for CoinGecko, CoinMarketCap, or your preferred crypto data source
  4. A list of your cryptocurrency holdings (coins and quantities)

Quick Setup Guide

  1. Download the template file and import it into your n8n instance
  2. Connect your Airtable account in the workflow settings
  3. Configure your crypto API connection (CoinGecko/CoinMarketCap)
  4. Map your Airtable fields to the workflow data structure
  5. Set your preferred update schedule (daily recommended)
  6. Test the workflow with a small subset of your portfolio
  7. Activate the workflow for automatic updates
