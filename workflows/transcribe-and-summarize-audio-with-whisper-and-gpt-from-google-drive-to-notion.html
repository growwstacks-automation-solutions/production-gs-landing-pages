What This Workflow Does

This automation solves the time-consuming problem of manually transcribing and summarizing audio recordings. Whether you're dealing with meeting recordings, interviews, lectures, or podcast episodes, this workflow automatically processes audio files uploaded to Google Drive, converts them to text using OpenAI's Whisper model, generates concise summaries with GPT-4, and stores everything neatly in your Notion workspace.

For researchers, content creators, students, and professionals, this represents a massive efficiency gain. Instead of spending hours listening and typing, you get searchable transcripts and actionable summaries delivered automatically to your knowledge base. The workflow handles the entire pipeline from file detection to intelligent summarization without any manual intervention.

How It Works

Step 1: Google Drive Trigger

The workflow monitors a specific Google Drive folder for new audio file uploads. When an MP3, WAV, or other audio file appears, it automatically triggers the automation and downloads the file for processing.

Step 2: Audio Transcription with Whisper

The downloaded audio file is sent to OpenAI's Whisper API, which converts speech to text with high accuracy. This creates a complete transcript of the recording, including speaker differentiation when possible.

Step 3: Intelligent Summarization with GPT-4

The raw transcript is passed to GPT-4 with a custom prompt that extracts key points, action items, and creates a structured summary. You can customize the prompt for different use cases—meeting minutes, lecture notes, or interview highlights.

Step 4: Notion Integration

The final transcript and summary are formatted and added as a new page in your specified Notion database. Metadata like file name, upload date, and duration are included for easy organization and retrieval.

What You'll Need

  1. A Google Drive account with audio files to process
  2. A Notion account and database to receive transcripts
  3. OpenAI API access (for Whisper and GPT-4)
  4. n8n instance (self-hosted version required for community nodes)
  5. Basic understanding of API key management

Pro tip: Create separate Google Drive folders for different types of recordings (meetings, interviews, lectures) and configure different summary prompts for each to get context-appropriate outputs in Notion.

Quick Setup Guide

  1. Download the template and import it into your n8n instance
  2. Connect your Google Drive account and select the folder to monitor
  3. Add your OpenAI API key to both the Whisper and GPT-4 nodes

  5. Connect your Notion account and select the target database
  6. Test with a sample audio file and activate the workflow
