What This Workflow Does

This automation solves the time-consuming problem of manually transcribing audio content and extracting key insights. Content creators, researchers, and professionals often waste hours listening to recordings and typing notes, delaying content production and knowledge sharing.

The workflow automatically monitors a Google Drive folder for new audio files, sends them to OpenAI's Whisper API for accurate transcription, processes the text through GPT-4 to generate concise summaries, and creates organized pages in Notion with both the full transcript and summary. This turns hours of manual work into a fully automated process that runs in the background.

How It Works

Step 1: Audio File Detection

The workflow continuously monitors a specified Google Drive folder. When a new audio file (MP3, WAV, M4A) is uploaded, it automatically triggers the automation process without any manual intervention.

Step 2: AI-Powered Transcription

The audio file is sent to OpenAI's Whisper API, which converts speech to text with high accuracy. The system handles various accents, background noise, and technical terminology, producing a clean transcript ready for processing.

Step 3: Intelligent Summarization

The raw transcript is analyzed by GPT-4, which identifies key points, action items, decisions, and insights. The AI generates a concise summary that captures the essence of the conversation while maintaining context and important details.

Step 4: Organized Storage in Notion

A new page is created in your specified Notion database containing the audio file name, full transcript, AI summary, timestamp, and any relevant metadata. This creates a searchable knowledge base of all your audio content.

  3. A Notion account with a database ready to receive transcribed content
  4. n8n instance (cloud or self-hosted) to run the automation

Quick Setup Guide

  1. Download the template file and import it into your n8n instance
  2. Connect your Google Drive account and specify the folder to monitor
  3. Add your OpenAI API key and configure the Whisper and GPT-4 nodes
  4. Connect your Notion account and select the target database
  5. Test the workflow with a sample audio file and activate the trigger
  6. Upload audio files to your Google Drive folder and watch automation happen

Pro tip: Create separate Notion databases for different content types (interviews, meetings, lectures) and use workflow branches to route transcripts accordingly. This keeps your knowledge base organized from day one.
