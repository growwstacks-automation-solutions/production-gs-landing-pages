What This Workflow Does

This automation solves a major pain point for content creators, marketers, and small businesses: the time-consuming process of creating and distributing video content. Manually producing short videos for platforms like YouTube Shorts and TikTok requires design skills, editing software, and hours of work per clip.

The workflow completely automates this pipeline. You provide a static image and a text prompt (like "a serene mountain landscape at sunset"), and the system uses the MiniMax Hailuo 02 AI model to generate a 6-10 second animated video. It then automatically uploads the finished video to both YouTube and TikTok, and logs all the details in a Google Sheet for tracking.

This turns what was a multi-hour, multi-step manual process into a fully hands-off operation that can run on a schedule, processing dozens of videos while you focus on strategy and engagement.

How It Works

1. Trigger & Data Collection

The workflow starts either manually or on a schedule (e.g., every 5 minutes). It checks a designated Google Sheet for new rows where the "VIDEO" column is empty. Each row contains an image URL and a text prompt describing the desired video.

2. AI Video Generation

For each pending task, the system sends the image and prompt to the MiniMax Hailuo 02 API. This advanced AI model analyzes the image, interprets the prompt, and generates a short, animated video. The workflow polls the API every 60 seconds until the video is ready.

3. Video Processing & Storage

Once generation is complete, the workflow retrieves the video URL and uploads the file to Google Drive for secure storage and backup. This creates a permanent archive of all generated content.

4. Multi-Platform Publishing

The system then uses the Upload-Post.com API to publish the video to both YouTube and TikTok simultaneously. It generates an SEO-optimized title using AI and includes relevant metadata. The Google Sheet is automatically updated with the video URL and platform links.

Pro tip: Use a simple Google Form connected to your input sheet. Team members or clients can submit image links and video ideas directly, triggering the automation without any technical involvement.

What You'll Need

  1. MiniMax API Key: For accessing the Hailuo 02 video generation model (available through their platform).
  2. Upload-Post.com Account: Free tier offers 10 uploads/month; paid plans unlock TikTok uploads and higher limits.
  3. Google Sheets: A spreadsheet with columns for Image URL, Prompt, Video URL, and YouTube Link.
  4. Google Drive: For storing generated video files (connected via Google Sheets node).
  5. n8n Instance: Self-hosted or cloud version to run the workflow.

Quick Setup Guide

Follow these steps to implement this automation in under 30 minutes:

  1. Download the template using the button above and import it into your n8n instance.
  2. Create your Google Sheet with columns: Image URL, Prompt, Video URL, YouTube Link, Status.
  3. Connect the Google Sheets node to your spreadsheet and set the trigger interval.
  4. Add your API keys to the MiniMax and Upload-Post.com nodes in the workflow.
  5. Test with one row by adding an image URL and prompt, then activate the workflow.
  6. Scale up by adding multiple rows to your sheet and letting the automation run on schedule.

Pro tip: Start with the free tier of Upload-Post.com to test YouTube uploads. Upgrade to a paid plan when ready to add TikTok automation and increase your monthly video quota.
