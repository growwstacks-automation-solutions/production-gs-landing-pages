What This Workflow Does

This automation solves a common business problem: creating professional, consistent headshots for team members, clients, or personal branding without the time and expense of traditional photography. Many professionals rely on casual selfies or outdated photos for LinkedIn profiles, company websites, and marketing materials, which undermines credibility and brand perception.

The workflow automatically transforms user-submitted photos into studio-quality headshots using advanced AI image generation. It accepts input via Telegram (perfect for mobile users) or manual URL submission, processes images through the Nano Banana Pro AI model with specific prompts for professional appearance, and delivers polished results to both Google Drive for organization and FTP for immediate web use.

Beyond simple image editing, this automation includes intelligent polling to wait for AI processing completion, secure access controls, and dual storage destinations—creating a complete, hands-off solution for businesses, HR departments, agencies, and individuals who need professional imagery at scale.

How It Works

1. Image Submission & Validation

The workflow begins when a user sends a photo through a Telegram bot or provides an image URL. It validates the sender's authorization and prepares the image for processing, extracting metadata and creating a unique filename based on timestamp to prevent conflicts.

2. AI Processing & Enhancement

The image is sent to the Nano Banana Pro AI model via API with carefully crafted prompts specifying professional attire, studio lighting, clean backgrounds, and natural positioning. The AI analyzes facial features, lighting conditions, and composition to generate a realistic headshot that maintains likeness while elevating professional appearance.

3. Status Polling & Completion Check

Since AI image generation takes variable time, the workflow includes intelligent polling that repeatedly checks the API for completion status. This ensures reliable processing even during peak loads and prevents failed runs due to timeouts, a common pitfall in automated image workflows.

4. Result Retrieval & Distribution

Once processing completes, the enhanced headshot is downloaded and simultaneously uploaded to Google Drive (for organized, cloud-accessible storage) and FTP (for immediate website or application use). The workflow generates shareable links and can notify users of completion.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. Telegram Bot Token from BotFather for chat-based image submission
  3. Fal.ai API Key for accessing the Nano Banana Pro image generation model
  4. Google Drive credentials with folder access for organized storage
  5. FTP server details (host, credentials, path) for web-accessible storage
  6. Basic understanding of API authentication to configure the HTTP request nodes

Pro tip: Before running at scale, test with various input image qualities (well-lit vs. low-light, close-up vs. full-body) to understand how the AI handles different scenarios. Adjust prompts accordingly for your specific use case.

Quick Setup Guide

1. Download and import the JSON template into your n8n instance using the import functionality.

2. Configure credentials in the n8n credentials panel for Telegram, Google Drive, FTP, and Fal.ai API access.

3. Update node parameters including the specific Telegram user ID in the sanitization node, FTP paths, and Google Drive folder ID.

4. Test the workflow manually by triggering with a sample image URL before connecting the Telegram webhook.

5. Activate the workflow and set up the Telegram webhook URL to enable chat-based submissions.

6. Monitor initial runs to ensure proper image handling and adjust AI prompts if needed for your desired output style.
