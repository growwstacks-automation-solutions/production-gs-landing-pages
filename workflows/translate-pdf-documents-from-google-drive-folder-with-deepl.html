What This Workflow Does

This automation solution eliminates the manual process of translating business documents by automatically converting all PDF files in a specified Google Drive folder to your target language using DeepL's advanced AI translation. It's designed for businesses that regularly deal with multilingual documents, international clients, or global teams.

The workflow monitors your designated Google Drive folder for new PDF uploads, processes them through DeepL's translation engine, and saves the translated versions back to your Drive. This creates a seamless pipeline for document localization without any manual intervention, saving hours of repetitive work while ensuring consistent translation quality.

The automated workflow steps for translating PDF documents from Google Drive

How It Works

1. Folder Monitoring

The workflow continuously monitors your specified Google Drive folder for new PDF files. You can set up multiple folders for different languages or departments if needed.

2. Document Processing

When a new PDF is detected, the system extracts the text content while preserving the original formatting. The workflow handles various PDF types including scanned documents (using OCR when necessary).

3. AI Translation

The extracted text is sent to DeepL's translation API, which converts it to your specified target language with industry-leading accuracy. You can customize terminology preferences for consistent business vocabulary.

4. File Reconstruction

The translated text is reconstructed into a new PDF document that maintains the original layout and formatting as closely as possible.

5. Delivery & Organization

The translated document is saved back to your Google Drive, either in the same folder or a designated translations subfolder. The workflow can optionally notify team members via email or Slack when translations are complete.

What You'll Need

  1. A Google Workspace account with Drive access
  2. DeepL API credentials (free tier available)
  3. n8n instance (self-hosted or cloud)
  4. A designated Google Drive folder for documents to translate
  5. Target language(s) selected

Pro tip: Create separate folders for different languages to keep translations organized. The workflow can be duplicated and modified for each language pair you need.

Quick Setup Guide

  1. Download the template file and import it to your n8n instance
  2. Connect your Google Drive account in the credentials section
  3. Enter your DeepL API key in the designated node
  4. Specify your source folder ID and target language
  5. Test with a sample PDF document
  6. Activate the workflow for continuous monitoring
