What This Workflow Does

This automation solves the problem of LinkedIn inbox overload by intelligently processing incoming messages using AI. Professionals waste hours weekly manually sorting through LinkedIn notifications and InMails - this workflow automates that process while ensuring important communications never get missed.

The system analyzes each message, determines its importance, categorizes it appropriately, stores the information in Notion for tracking, and alerts you via Slack when immediate action is needed. It handles the tedious work so you can focus on meaningful conversations.

How It Works

1. Email Collection

The workflow starts by collecting LinkedIn emails from your Gmail inbox using a specific label you define (like "LinkedIn"). This ensures only relevant messages get processed.

2. Initial Filtering

Two filters run immediately to remove noise: First, keeping only invitations and direct messages (not notifications). Second, removing common automated message patterns.

3. AI Analysis

Each message gets sent to an AI model (OpenAI or compatible) which returns structured data including action priority, relevance score, sales likelihood, summary, and optional reply draft.

4. Quality Control

A quality gate ensures only high-signal messages proceed, filtering out low-relevance communications based on configurable thresholds.

5. Notion Integration

All processed messages get logged in Notion as tickets, creating a searchable database of your LinkedIn interactions with full context.

6. Slack Alerts

For messages marked as requiring quick response, you receive a Slack DM with key details so you never miss time-sensitive opportunities.
