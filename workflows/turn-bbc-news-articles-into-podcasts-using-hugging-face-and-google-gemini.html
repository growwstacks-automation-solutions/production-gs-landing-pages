Turn BBC News Articles into Podcasts with AI

Automatically convert breaking news into engaging audio content using Google Gemini and Hugging Face text-to-speech.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves a common challenge for content creators, educators, and news enthusiasts: transforming written news into accessible audio format. Manually creating podcasts from articles involves hours of research, scriptwriting, recording, and editing. This workflow eliminates that manual effort entirely.

The system automatically fetches the latest BBC News articles, analyzes their content for podcast suitability, converts selected articles into professionally structured podcast scripts using Google Gemini's AI, and generates natural-sounding audio using Hugging Face's text-to-speech models. What used to take 3-4 hours of work now happens automatically in minutes.

How It Works

Step 1: Fetch BBC News Content

The workflow begins by scraping the BBC News website for the latest articles. It identifies headlines, summaries, and article URLs, preparing them for content analysis.

Step 2: Classify News for Podcast Suitability

Using Google Gemini's classification capabilities, the system evaluates each article based on factors like topic relevance, narrative potential, and audience interest. Only the most suitable articles proceed to the next stage.

Step 3: Extract Full Article Content

For selected articles, the workflow fetches the complete text content, including details, quotes, and contextual information needed for a comprehensive podcast episode.

Step 4: Generate Podcast Script with AI

Google Gemini transforms the article text into a conversational podcast script. It adds natural introductions, transitions, summaries, and concluding remarks—formatting the content specifically for audio consumption.

Step 5: Convert Script to Speech

The final script is sent to Hugging Face's text-to-speech API, which generates high-quality audio files with natural inflection and pacing. The output is ready-to-publish podcast audio.

Who This Is For

This template is ideal for content creators who want to establish daily news podcasts without the production overhead, educators creating audio learning materials from current events, businesses wanting to republish industry news in audio format for their audience, and individuals who prefer consuming news through podcasts rather than reading.

What You'll Need

  1. A running n8n instance (self-hosted or cloud)
  2. Google Gemini API credentials (free tier available)
  3. Hugging Face account with API access token
  4. Basic understanding of n8n's interface (no coding required)
  5. Storage destination for generated audio files (local or cloud)

Pro tip: Start with the free tiers of both Gemini and Hugging Face—they offer sufficient capacity for regular podcast production and let you test the workflow before any financial commitment.

Quick Setup Guide

  1. Download the template JSON file using the button above
  2. Import it into your n8n instance via the workflow import function
  3. Configure the HTTP Request nodes with your Hugging Face token
  4. Set up the Google Gemini nodes with your API credentials
  5. Test with a single article to verify audio output quality
  6. Schedule the workflow to run daily or as needed
  7. Connect the output to your podcast hosting platform or storage
