What This Workflow Does

This automation solves a critical security challenge: verifying user identity with high confidence while maintaining a smooth user experience. Manual verification processes are slow, error-prone, and don't scale. This workflow automates a two-factor authentication (2FA) system that combines voice calls and email verification to create a robust security layer.

When a user needs to verify their identity—whether during sign-up, a sensitive transaction, or password reset—the system automatically initiates a text-to-speech phone call through ClickSend. The user receives a spoken verification code, then enters it into a form. Upon successful voice verification, a second code is sent via email for additional confirmation. This dual-channel approach significantly reduces fraud risk while keeping the process user-friendly.

How It Works

1. User Trigger & Data Collection

The workflow starts when a user submits a form with their phone number, email, and name. This trigger can come from your website, application, or any system requiring verification.

2. Voice Code Generation & Call

A unique verification code is generated and formatted for clear speech. The workflow then uses the ClickSend API to place an automated voice call that reads the code to the user's phone number.

3. First Verification Checkpoint

The user is prompted to enter the code they heard. An "If" node compares their input with the generated code. If correct, the process continues; if incorrect, they receive a failure message and can retry.

4. Email Verification Layer

After passing the voice check, a second unique code is generated and sent to the user's email via SMTP. This adds an additional security layer confirming access to both their phone and email.

5. Final Verification & Success

The user enters the email code. A final check validates it. Successful verification triggers a confirmation message, completing the secure authentication process.

Quick Setup Guide

Follow these steps to implement this two-factor authentication system in under 30 minutes:

  1. Download & Import: Click the download button above and import the JSON file into your n8n instance.
  2. Configure ClickSend: Create a Basic Auth credential in n8n using your ClickSend username and API key as the password.
  3. Set Up Email: Add your SMTP credentials (host, port, user, pass) in the "Send Email" node.
  4. Customize Messages: Adjust the verification codes, voice messages, and email content to match your brand.
  5. Test Thoroughly: Run the workflow with your phone number and email to ensure both voice and email delivery work correctly.
  6. Connect to Your App: Replace the form trigger with a webhook trigger, then call it from your application when verification is needed.
