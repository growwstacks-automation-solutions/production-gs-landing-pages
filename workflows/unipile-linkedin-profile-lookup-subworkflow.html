What This Workflow Does

This subworkflow automates LinkedIn profile lookups using Unipile's API within n8n, saving recruiters and sales teams hours of manual research. It takes a LinkedIn profile URL or name as input and returns structured professional data including work history, education, skills, and recommendations.

The automation solves the time-consuming process of manually reviewing profiles and copying data into CRMs or ATS systems. By integrating with Unipile, it handles LinkedIn's API complexities and rate limits while ensuring compliance with their terms of service.

How It Works

1. Input Trigger

The workflow accepts LinkedIn profile URLs or names from your existing automation (like a candidate tracking system or lead list).

2. Unipile API Connection

Unipile authenticates with LinkedIn and fetches the full profile data, handling all API complexities behind the scenes.

3. Data Extraction

The workflow parses the profile to extract key professional details like current position, company, job history, education, and skills.

4. Output Formatting

Data is structured into consistent fields that can be directly used in your CRM, ATS, or other business systems.
