What This Workflow Does

This automation solution transforms how teams validate and assess workflow templates by combining AI analysis with automated reporting. It addresses the time-consuming manual review process that often creates bottlenecks in workflow development and publishing pipelines.

The workflow automatically retrieves JSON templates from Google Drive, validates their structure, evaluates quality against multiple criteria using GPT-4o, generates detailed assessment reports, archives them to Drive, and notifies stakeholders via Slack and email - creating a complete validation cycle without manual intervention.

How It Works

1. File Retrieval

Connects to Google Drive to fetch all workflow JSON files from a designated folder, collecting metadata like creation dates and owners.

2. Validation Processing

Each workflow undergoes sequential validation including JSON parsing, structure verification, and AI-powered quality assessment against eight key criteria.

3. AI Analysis

GPT-4o evaluates workflows for documentation quality, security practices, node naming standards, visual presentation, and overall usefulness.

4. Reporting

Generates structured reports in both JSON and human-readable formats, highlighting strengths and improvement areas for each workflow.

5. Notification

Sends validation results via Slack messages and formatted emails with direct links to reports stored in Google Drive.
