What This Workflow Does

This automation solves a critical challenge for businesses using HighLevel for CRM - ensuring mailing addresses are deliverable before initiating direct mail campaigns. By integrating with Lob's address verification API, the workflow automatically checks each new contact's address against USPS databases.

The solution eliminates undeliverable mail costs, improves campaign success rates, and maintains cleaner CRM data by flagging problematic addresses for follow-up. The entire process runs in the background without manual intervention.

How It Works

1. New Contact Trigger

The workflow activates when a new contact is added in HighLevel, capturing all relevant address fields.

2. Address Validation

The contact's address is sent to Lob's verification API which checks against authoritative postal databases.

3. Deliverability Check

Lob returns a deliverability score and standardized address format if corrections were corrections needed.

4. CRM Update

The workflow updates HighLevel with the verification status and any address corrections from Lob.
