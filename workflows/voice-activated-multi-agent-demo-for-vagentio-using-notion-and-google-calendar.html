How It Works

The workflow is built on a supervisor-agent pattern, which is more reliable and scalable than a single, monolithic AI bot.

Step 1: Voice Input & Transcription

You use the Vagent.io mobile app to record a voice message. The app transcribes your speech to text and sends it, along with a unique session ID, to a webhook endpoint in your n8n workflow. This maintains conversation continuity.

Step 2: AI Supervisor Routing

A main "Supervisor" AI agent receives the transcribed message. It analyzes the intent using the conversation history (linked by the session ID) and decides which specialized "tool" (sub-agent workflow) is needed to fulfill the request—like updating Notion or managing Google Calendar.

Step 3: Specialized Agent Execution

The Supervisor calls the appropriate sub-agent. Each sub-agent has its own isolated chat memory and is an expert at one specific task. For example, the "Google Calendar Agent" knows how to check availability, create events, and format responses.

Step 4: Action & Response Formatting

The sub-agent performs the action via API calls (e.g., creates a calendar event). It then formats the result in clear Markdown and returns it to the Supervisor with a special prompt to prevent re-interpretation.

Step 5: Final Output & Voice Response

The Supervisor compiles the response, separating any draft text (like meeting notes) from the spoken summary. The final formatted output is sent back to the Vagent app, which reads the spoken part aloud and displays the full text.

Pro tip: Start by testing the workflow using the built-in chat trigger before connecting the voice app. This lets you debug the AI logic and API connections with simple text inputs.

What You'll Need

  1. A running n8n instance (cloud or self-hosted) to import and execute the workflow.
  2. OpenAI API key (or another supported LLM) to power the AI agents.
  3. Google Cloud Project with Calendar API enabled and OAuth 2.0 credentials.
  4. Notion account and an internal integration token with access to the pages/databases you want to manage.
  5. Vagent.io mobile app (free) installed on your iOS or Android device.
  6. A public URL for your n8n webhook (using n8n.cloud, ngrok, or a similar tunneling service if testing locally).

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n instance using the "Import from File" option.
  2. Configure Credentials: Set up the credential nodes for OpenAI, Google Calendar (OAuth2), and Notion. Follow the sticky note instructions on the canvas.
  3. Update IDs & Details: Replace placeholder IDs in the workflow with your actual Google Calendar ID and Notion Database/Page IDs.
  4. Test with Chat Trigger: Temporarily disable nodes after the "Tools Agent" and run the workflow using the "Chat Trigger" node. Verify it can understand and execute simple commands.
  5. Connect Vagent App: Copy the webhook URL from the "Webhook" node. In the Vagent.io app settings, paste your OpenAI API key, the webhook URL, and the password you set for header authentication.
  6. Go Live: Re-enable all nodes. Tap the microphone in the Vagent app and start giving voice commands to manage your calendar and Notion.
