Web Content to Telegram Publisher with AI Enhancement

Automatically fetch, enhance, and publish web articles to your Telegram channel—with AI rewriting and branded image watermarks.

What This Workflow Does

Manually curating and publishing content to Telegram channels is time-consuming and inconsistent. This automation solves that by creating a fully hands-off content pipeline that turns any webpage into engaging Telegram posts.

The workflow automatically fetches articles from a specified website, uses AI to rewrite or summarize the content for better engagement, applies your brand's watermark to images, and publishes everything directly to your Telegram channel. Unlike RSS-based solutions, it directly parses HTML, giving you more control and flexibility over source selection.

This system transforms hours of manual work into a scheduled background process, ensuring your channel stays active with high-quality content even when you're focused on other business priorities.

How It Works

1. Scheduled Trigger & Source Fetching

The workflow starts on a schedule you define (daily, hourly, etc.). It then makes an HTTP request to your target webpage and retrieves the full HTML content.

2. Link Extraction & Duplicate Prevention

It parses the HTML to extract article links, then checks a Google Sheet to see which links have already been processed. New links are added to the sheet to prevent reposting.

3. Article Processing & AI Enhancement

For each new article, the workflow fetches the full text, cleans the HTML formatting, and sends it to an AI agent (like OpenAI or Gemini) for customization—rewriting, summarizing, or adjusting tone.

4. Image Watermarking & Telegram Publishing

Any associated image is fetched, your watermark (logo or text) is applied automatically, and the final package—enhanced text plus watermarked image—is posted to your configured Telegram channel.

What You'll Need

  1. A running n8n instance (cloud or self-hosted)
  2. A Telegram bot token and channel ID (create via @BotFather)
  3. A Google Sheet with edit permissions to track processed links
  4. Access to an AI model provider (OpenAI, Anthropic, Google Gemini, etc.)
  5. The URL of the website you want to monitor for new content
  6. Your watermark image or text specifications

Pro tip: Start with a single reliable website source to test the parsing logic before adding multiple sources. Some websites have complex HTML structures that might require minor adjustments to the extraction nodes.

Quick Setup Guide

  1. Import the template: Download the JSON file above and import it into your n8n instance.
  2. Configure the Google Sheet node: Create a new sheet with columns for URLs and timestamps, then share edit access with your n8n service account.
  3. Set your source URL: In the HTTP Request node, enter the website you want to monitor for new articles.
  4. Connect your AI provider: Add your API key to the AI Agent node and customize the prompt for how you want content enhanced.
  5. Setup Telegram: Add your bot token and channel ID to the Telegram node.
  6. Customize the watermark: Adjust the Edit Image node with your logo, text, position, and opacity preferences.
  7. Test and schedule: Run the workflow once manually to verify everything works, then set the Schedule Trigger to your desired frequency.
