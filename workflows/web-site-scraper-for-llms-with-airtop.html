What This Workflow Does

This automation solves the challenge of manually collecting large volumes of web content for training language models. It recursively scrapes content from multiple linked pages, saving hundreds of hours compared to manual data collection.

The workflow starts with seed URLs, extracts content from each page, identifies new links to follow based on your criteria, and continues this process to a specified depth. All collected content is organized in Google Sheets for easy access and analysis.

How It Works

1. URL Input

The workflow reads starting URLs from a Google Sheet where you specify your initial targets and depth parameters.

2. Content Extraction

Using Airtop's API, each page is scraped to extract the main content while ignoring navigation and boilerplate elements.

3. Link Discovery

New links are extracted from each page and filtered based on your criteria (domain patterns, keywords etc).

4. Recursive Processing

The workflow repeats the extraction process for each new link until reaching your specified depth limit.

5. Data Organization

All collected content is saved to Google Sheets with source URLs and metadata for easy reference.
