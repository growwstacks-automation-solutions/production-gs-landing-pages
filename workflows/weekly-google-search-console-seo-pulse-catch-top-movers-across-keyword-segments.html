What This Workflow Does

This automation solves the challenge of manually tracking keyword performance changes in Google Search Console. SEO professionals often waste hours each week compiling spreadsheets to identify ranking fluctuations, missing critical opportunities to capitalize on emerging trends or address sudden drops.

The workflow automatically compares your current week's search performance against the previous period, highlighting significant changes across key segments. It categorizes keywords by type (branded, non-branded, commercial intent) and surfaces the top movers in each category, giving you actionable insights without manual data crunching.

How It Works

1. Data Collection

The workflow connects to your Google Search Console account via API and pulls performance data for your selected property. It retrieves metrics like clicks, impressions, CTR, and average position for all queries during the current and previous periods.

2. Keyword Segmentation

Queries are automatically categorized into segments based on your configured rules. Typical segments include branded terms (containing your company/product names), non-branded discovery terms, and commercial intent keywords (containing buying signals).

3. Change Analysis

The system calculates week-over-week changes for each query across all metrics. It applies statistical significance filters to highlight only meaningful changes, ignoring normal search volatility.

4. Reporting

Top performers and significant movers from each segment are compiled into an easy-to-scan report. The output can be delivered via email, Slack, or integrated directly into your SEO dashboard.
