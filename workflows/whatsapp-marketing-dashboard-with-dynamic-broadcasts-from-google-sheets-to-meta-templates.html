What This Workflow Does

This workflow solves the complex challenge of running sophisticated WhatsApp marketing campaigns without being limited by off-the-shelf tools. Manual WhatsApp marketing is time-consuming, error-prone, and lacks personalization—sending the same generic message to everyone hurts engagement.

The automation provides a complete backend solution that connects your contact database in Google Sheets with Meta's approved WhatsApp templates. It enables dynamic, personalized broadcasts where each message can include the recipient's name, unique links, and even specific images pulled from your spreadsheet. This transforms bulk messaging into tailored conversations at scale.

Beyond just sending messages, the system maintains perfect synchronization between your approved templates and your sending dashboard, ensuring you're always using the latest approved content. It acts as a robust API backend for any custom front-end interface you want to build.

How It Works

The workflow operates through three independent but connected systems that work together seamlessly.

1. Automated Template Sync

A scheduled trigger periodically fetches all your approved WhatsApp message templates directly from your Meta Business Account. It clears and updates an n8n Data Table, ensuring your available template list is always current. This eliminates manual checking and prevents using outdated templates.

2. Front-end API Endpoint

A dedicated webhook acts as an API for your marketing dashboard. When your front-end calls this endpoint, it returns a clean JSON list of all available templates from the Data Table. This powers dropdown menus in your interface, letting users select which template to send.

3. Dynamic Broadcast Engine

The main webhook listens for broadcast requests containing the template name. It looks up the template structure, fetches all contacts from Google Sheets, then for each contact, dynamically constructs a personalized API request. A Code node merges contact data (name, user ID, image URL) into the template before sending via WhatsApp Business API.

Quick Setup Guide

Follow these steps to implement this WhatsApp marketing automation in your business:

  1. Prepare your Google Sheet: Create columns for Phone Number, Full Name, and any dynamic content like Marketing Image URL or User ID.
  2. Set up Meta credentials: Create an HTTP Header Auth credential in n8n using your permanent WhatsApp Business API token.
  3. Create n8n Data Table: Name it "WhatsApp Templates" with columns: template_name, language_code, components_structure, template_id, status, category.
  4. Import the workflow: Download and import the JSON template into your n8n instance.
  5. Configure node credentials: Update the HTTP Request nodes with your Phone Number ID and WABA ID. Connect the Google Sheets node to your contact spreadsheet.
  6. Initial template sync: Manually trigger the "Sync Meta Templates" flow once to populate your Data Table.
  7. Build your front-end: Create a simple interface that calls the API endpoint for template list and triggers broadcasts.
  8. Test and activate: Send test messages to yourself, then activate all workflow parts for production use.
