WhatsApp Group Summary Automation

What This Workflow Does

Managing multiple WhatsApp groups creates information overload. Important messages get buried, audio clips remain unheard, and daily summaries require manual effort. This automation solves that by creating a structured system that captures every message—text or audio—stores it in Google Sheets, and delivers AI-powered summaries each morning.

The workflow distinguishes itself with a control panel: you enable or disable summaries per group with a single checkbox. It’s built for teams, communities, and educational groups who need to stay informed without wasting hours scrolling through chats. The system handles transcription, categorization, and delivery automatically, turning chaotic group communication into actionable insights.

How It Works

Step 1: Message Capture

A webhook receives messages from WhatsApp via Evolution API. The workflow checks if the message originates from a group (not individual chat) and routes it based on type: text messages go directly to storage; audio messages are transcribed using OpenAI before storage.

Step 2: Organized Storage

Every message is logged in Google Sheets with columns for group name, sender, timestamp, and content. The system automatically creates a control panel tab—if a new group appears, it adds a row with a checkbox allowing you to toggle summaries for that group.

Step 3: Daily Summary Generation

Scheduled daily at 8 AM (configurable), the workflow collects all messages from the previous day, groups them by chat, checks if the group is enabled in the control panel, and sends the batch to OpenAI GPT for summarization.

Step 4: Formatted Delivery

The AI-generated summary is formatted cleanly and sent back into the WhatsApp group via Evolution API, providing participants with a digest of yesterday’s key discussions without requiring anyone to manually compile it.

Quick Setup Guide

  1. Download the template and import into your automation platform.
  2. Configure the webhook connection to your Evolution API setup.
  3. Make a copy of the provided Google Sheets template and connect credentials.
  4. Add your OpenAI API key in the workflow settings.
  5. Adjust the schedule trigger to your preferred summary time.
  6. Test with a single group message to verify capture and storage.

Pro tip: Start with one group enabled. Verify the summary quality, then gradually add more groups. Customize the AI prompt to emphasize topics relevant to your context (e.g., “highlight decisions and action items”).
