What This Workflow Does

Manual Instagram outreach is time-consuming and difficult to scale. Researching each profile, analyzing their content, and crafting personalized messages can take 5-10 minutes per creator. For campaigns targeting hundreds of profiles, this becomes impossible to manage effectively while maintaining quality.

This automation solves that problem by connecting Instagram data scraping (via Apify) with AI-powered message generation (via OpenAI) and centralized tracking (via Google Sheets). It reads a list of Instagram handles, fetches their latest post and bio information, analyzes the content with AI vision, and generates context-aware direct messages that reference specific elements from each profile. The result is scalable outreach that feels personal and relevant.

Pro tip: This workflow uses community nodes that require self-hosted n8n. Both Apify and OpenAI offer free credits that can cover initial testing runs.

How It Works

1. Data Collection from Google Sheets

The workflow starts by reading a list of Instagram usernames from a Google Sheet. Each row represents a target profile for outreach. The sheet also tracks status columns for whether data has been scraped, messages generated, and outreach sent.

2. Instagram Profile Scraping with Apify

For each Instagram handle, the workflow triggers Apify's Instagram Post Scraper to fetch the latest post data, including images, captions, engagement metrics, and bio information. This provides the raw material for personalization.

3. AI Analysis with OpenAI Vision

The image URL from the newest post is sent to OpenAI's Vision API for analysis. The AI identifies themes, objects, emotions, and context within the visual content, generating insights that inform the personalized message.

4. Personalized Message Generation

Combining the AI analysis with profile metadata (bio, follower count, engagement), the workflow uses OpenAI's language model to craft a unique direct message. The prompt ensures references to specific post content and alignment with the creator's apparent interests.

5. Results Logging Back to Sheets

The generated message, along with scraped data and status flags, is written back to the Google Sheet. This creates a complete audit trail and enables manual review before sending, or integration with sending tools.

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance.
  2. Duplicate the Google Sheet template and update all Google Sheet nodes with your new sheet ID.
  3. Add your Instagram handles to the first column of the sheet.
  4. Configure credentials for Google Sheets, Apify, and OpenAI in n8n.
  5. Create and configure the Apify Instagram Post Scraper task as referenced in the workflow.
  6. Test with 2-3 profiles to verify data flow and message quality before scaling.
  7. Review generated messages in the sheet and adjust AI prompts if needed for your use case.

Pro tip: Start with a small batch of profiles to optimize your AI prompt before scaling. The message tone and structure can be adjusted in the OpenAI node to match your brand voice and campaign goals.
